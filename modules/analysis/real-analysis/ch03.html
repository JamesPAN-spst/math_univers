Lebesgue 积分

01简单函数积分与非负函数积分

🎯 为什么研究这个问题

有了 σ-代数和测度之后，自然的下一步是构造与测度"兼容"的积分——使得 \(\int \mathbf{1}_A\, d\mu = \mu(A)\) 自然成立。Riemann 积分沿定义域切分（竖条矩形）的策略在高度不连续函数面前失败；Lebesgue 另辟蹊径，改为"沿值域分层"。

  • 量子力学：可观测量的期望值 \(\langle A \rangle = \int a\, dE_a\) 依赖谱测度上的积分——这正是 Lebesgue "按值域切分"思想在算子理论中的体现；
  • 概率论：随机变量的期望 \(\mathbb{E}[X] = \int X\, dP\) 就是 Lebesgue 积分，其构造保证了"先积简单随机变量、再取极限"的路径合法；
  • 数值方法：蒙特卡洛积分的理论基础正是 Lebesgue 积分——大数定律将期望（积分）与样本均值联系起来。

Lebesgue 积分的"自底向上"三步构造——简单函数 → 非负可测函数（取 \(\sup\)）→ 一般可测函数（\(f=f^+-f^-\)）——不仅优雅，而且天然保证了积分与单调极限可交换（MCT），这是后续全部收敛定理的基石。

🔍 抽象化方法
  • 值域分层原理——\(\int f\, d\mu\) 不切定义域，而问"\(f\) 取值在 \([y, y+dy)\) 的点集有多大测度"，将求面积升级为测度运算。
  • 三步构造法——① 简单函数 \(\to\) ② 非负可测函数（取 \(\sup\)）\(\to\) ③ 一般可测函数（\(f = f^+ - f^-\)）。每一步只依赖前一步，层层推进。
  • 函子观点——积分是从可测函数空间到 \(\overline{\mathbb{R}}\) 的正线性泛函，保持单调性与极限交换；这正是 Daniell 积分与 Riesz 表示定理的抽象起点。
📐 公式推导 / 模型建立

简单函数积分

设 \(\varphi = \sum_{i=1}^n a_i \mathbf{1}_{A_i}\) 为标准表示（\(A_i\) 两两不交），定义

\[\int \varphi\, d\mu = \sum_{i=1}^n a_i \, \mu(A_i)\]

此定义与表示方式无关：将两种表示取公共细分即可验证。

非负可测函数积分

对 \(f \geq 0\) 可测，定义

\[\int f\, d\mu = \sup\bigl\{\int \varphi\, d\mu : 0 \leq \varphi \leq f,\;\varphi\text{ 简单}\bigr\}\]

关键逼近定理（简单函数逼近）保证存在简单函数单调递增序列 \(\varphi_n \nearrow f\)，从而 \(\int f = \lim \int \varphi_n\)。

一般可测函数积分

将 \(f\) 分解为正部与负部：\(f^+ = \max(f,0)\)，\(f^- = \max(-f,0)\)。当 \(\int f^+\) 和 \(\int f^-\) 至少一个有限时，定义

\[\int f\, d\mu = \int f^+\, d\mu - \int f^-\, d\mu\]

\(f\) 可积（\(f \in L^1\)）\(\Leftrightarrow \int |f|\, d\mu < \infty\)，即正部负部都可积。

🌐 实际应用

概率论中的期望——随机变量 \(X\) 的期望 \(E[X] = \int_\Omega X\, dP\) 就是关于概率测度的 Lebesgue 积分；简单函数积分对应离散随机变量的加权求和，非负函数积分推广到连续分布。

物理中的质量与电荷——密度函数 \(\rho\) 对测度 \(\mu\) 的积分 \(\int_\Omega \rho\, d\mu\) 统一了连续与奇异分布（如点电荷 \(\delta\)），使物理建模无需区分"求和"与"积分"。

信号处理中的能量——信号 \(f(t)\) 的能量 \(E = \int |f|^2\, dt\) 正是 Lebesgue 积分；简单函数逼近对应数字采样量化，非负积分保证了能量的良定义性。

✏️ 例题分析与易错点

例题：Dirichlet 函数的 Lebesgue 积分

计算 \(f = \mathbf{1}_{\mathbb{Q} \cap [0,1]}\) 关于 Lebesgue 测度的积分。

解：\(f\) 本身是简单函数（取值 0 和 1），直接套定义：

\[\int f\, d\lambda = 1 \cdot \lambda(\mathbb{Q} \cap [0,1]) + 0 \cdot \lambda([0,1]\setminus\mathbb{Q}) = 1 \cdot 0 = 0\]

此函数不 Riemann 可积（上下 Darboux 和分别为 1 和 0），但 Lebesgue 可积且积分为 0——Lebesgue 积分的值域分层直接"看穿"了有理数集的零测本质。

常见错误

  • ❌ 认为积分定义中取的是"极限"而非上确界——实际是 \(\sup\) 而非 \(\lim\)；单调极限只是实现上确界的一种方式。 ✅ 非负函数积分 \(= \sup\{\int \varphi : 0 \le \varphi \le f\}\)。
  • ❌ 混淆"\(\int f^+\) 或 \(\int f^-\) 有限"与"\(f \in L^1\)"——前者只要求至少一个有限（积分可定义但可能 = \(\pm\infty\)），后者要求两个都有限。 ✅ \(L^1 \Leftrightarrow \int |f| < \infty\)。
  • ❌ 忽视简单函数表示的唯一性问题——不同表示给出同一积分值需要用公共细分论证。 ✅ 先取公共细分 \(\{A_i \cap B_j\}\) 再验证。

02积分的基本性质

🎯 为什么研究这个问题

积分作为从函数到数值的映射，必须满足一些基本的"合理性"要求：将各部分之和还原为整体的线性性，"更大的函数有更大积分"的单调性，以及零测集上的修改不影响积分值。这些性质不仅符合物理直觉，更是后续泛函分析理论的起点。

  • 概率论：Chebyshev（Markov）不等式用期望（积分）反向控制尾概率（水平集测度），是大数定律和集中不等式的基石；
  • 泛函分析：积分作为正线性泛函的性质直接导出 Riesz 表示定理——连接抽象泛函与具体测度；
  • 信号处理：信号能量 \(\int |f|^2\) 的三角不等式 \(|\int f| \leq \int |f|\) 保证了滤波运算的稳定性。

这些看似初等的性质共同赋予积分以"正线性泛函"和"格上的单调映射"双重身份——前者是 \(L^p\) 对偶理论的根基，后者是 Chebyshev 不等式和 Markov 链估计的逻辑起点。零测集上的不变性更为 \(L^p\) 空间的等价类概念铺平了道路。

🔍 抽象化方法
  • 正线性泛函——积分 \(I(f) = \int f\, d\mu\) 是可测函数空间上的正线性泛函：\(I(\alpha f + \beta g) = \alpha I(f) + \beta I(g)\)，\(f \ge 0 \Rightarrow I(f) \ge 0\)。
  • 格结构——单调性将积分嵌入可测函数的偏序格中；三角不等式 \(|I(f)| \le I(|f|)\) 是将积分提升为范数的关键。
  • 从点态到测度态——Chebyshev 不等式本质上是用积分（全局信息）控制水平集测度（局部信息），这是概率与分析衔接的桥梁。
📐 公式推导 / 模型建立

线性性

对可积函数 \(f, g\) 和常数 \(\alpha, \beta\)：

\[\int (\alpha f + \beta g)\, d\mu = \alpha \int f\, d\mu + \beta \int g\, d\mu\]

先对简单函数验证（直接展开定义），再由单调逼近传递到一般非负可测函数，最后用 \(f = f^+ - f^-\) 推广到一般情形。

单调性与三角不等式

\(f \leq g\) a.e. \(\Rightarrow \int f\, d\mu \leq \int g\, d\mu\)。推论：

\[\left|\int f\, d\mu\right| \leq \int |f|\, d\mu\]

Chebyshev（Markov）不等式

对 \(f \geq 0\)，\(\alpha > 0\)：

\[\mu\bigl(\{x : f(x) \geq \alpha\}\bigr) \leq \frac{1}{\alpha}\int f\, d\mu\]

证明一行：\(\alpha \cdot \mathbf{1}_{\{f \geq \alpha\}} \leq f\)，两边积分即得。

a.e. 相等不改变积分

若 \(f = g\) a.e.，则 \(\{f \neq g\}\) 零测，\(\int |f - g| = 0\)，从而 \(\int f = \int g\)。

🌐 实际应用

概率论尾估计——Chebyshev 不等式化为 \(P(|X| \geq \alpha) \leq E[|X|]/\alpha\)；取 \(g = (X - \mu)^2\) 即得方差版 \(P(|X - \mu| \geq k\sigma) \leq 1/k^2\)，是统计推断和置信区间的基础。

数值分析误差控制——线性性保证数值求积公式（如 Simpson 法则）可逐项分析误差；单调性保证正被积函数的求积结果为非负——这是积分方程数值解稳定性分析的核心。

量子力学归一化——波函数 \(\psi\) 满足 \(\int |\psi|^2 = 1\)；积分的三角不等式保证了态叠加后概率幅的可控性，Chebyshev 不等式给出粒子在特定区域外被探测到的概率上界。

✏️ 例题分析与易错点

例题：用水平集方法证明测度正

设 \(f \in L^1([0,1])\) 且 \(\int_0^1 f\, d\lambda = 3\)。证明 \(\lambda(\{x : f(x) \geq 1\}) > 0\)。

解：分解 \(f = f \cdot \mathbf{1}_{\{f \geq 1\}} + f \cdot \mathbf{1}_{\{f < 1\}}\)。后者满足 \(\int f \cdot \mathbf{1}_{\{f < 1\}} < 1 \cdot \lambda([0,1]) = 1\)。因此 \(\int f \cdot \mathbf{1}_{\{f \geq 1\}} > 3 - 1 = 2 > 0\)，故 \(\lambda(\{f \geq 1\}) > 0\)。

常见错误

  • ❌ 用 Chebyshev 不等式试图证明下界——该不等式给出的是测度的上界 \(\mu(\{f \ge \alpha\}) \le \int f / \alpha\)，不能直接反向使用。 ✅ 下界需用分解 + 互补估计的技巧。
  • ❌ 认为 \(f = g\) a.e. 意味着处处相等——零测集上可以任意不同。 ✅ 积分只"看"a.e. 行为；\(L^1\) 中的元素本质是等价类。
  • ❌ 对变号函数直接应用 Chebyshev——原始形式要求 \(f \ge 0\)。 ✅ 对一般 \(f\)，用 \(|f|\) 或 \(f^2\) 代入。

03Riemann 与 Lebesgue 积分的比较

🎯 为什么研究这个问题

"既然 Lebesgue 积分更强大，为什么还要学 Riemann 积分？"这是每个学习测度论的学生都会提出的自然疑问。答案在于两者并非对立而是包含关系——Lebesgue 积分严格扩展了 Riemann 积分，经典结果一个不丢。

  • 实变函数论：Lebesgue 判别定理精确刻画了 Riemann 可积的边界——有界函数 Riemann 可积当且仅当不连续点集零测，将直觉条件数学化；
  • Fourier 分析：Dirichlet 函数 \(\mathbf{1}_\mathbb{Q}\) 不 Riemann 可积，但 Lebesgue 积分轻松处理——Fourier 理论需要更广的可积函数类；
  • 数值计算：大量具体积分的计算仍用 Riemann 技巧（换元、分部），Lebesgue 的兼容性保证了这些结果可直接搬到测度论框架。

真正的差异在于对极限运算的稳定性：Riemann 可积函数在逐点极限下不封闭，而 Lebesgue 积分与极限的良好交换性（MCT、DCT）正是引入新积分理论的核心回报。从完备化视角看，\(L^1\) 是 Riemann 可积函数空间在 \(\|\cdot\|_1\) 范数下的自然完备化。

🔍 抽象化方法
  • 竖切 vs 横切——Riemann 沿定义域切分（竖条矩形），Lebesgue 沿值域切分（水平层）；后者的抽象层次更高，代价是需要测度论基础。
  • 振荡 vs 测度——Riemann 可积性等价于"振荡大的区间总长为零"（Darboux 条件）；Lebesgue 判别法将这一条件精确化为"不连续点集零测"。
  • 完备化视角——\(L^1\) 在 \(\|\cdot\|_1\) 范数下完备，而 Riemann 可积函数空间不是；Lebesgue 积分是 Riemann 积分的自然完备化。
📐 公式推导 / 模型建立

Lebesgue 判别定理

有界函数 \(f: [a,b] \to \mathbb{R}\) Riemann 可积 \(\Leftrightarrow\) 不连续点集 \(D_f\) 满足 \(\lambda(D_f) = 0\)。

证明思路（充分性）：将上下 Darboux 和之差 \(U(P,f) - L(P,f) = \sum (M_i - m_i)\Delta x_i\) 分为"连续区间"和"不连续区间"两部分。连续点处 \(M_i - m_i\) 可任意小；不连续点集零测，对总振荡的贡献通过精细划分可控。

兼容性定理

\(f\) 在 \([a,b]\) 上 Riemann 可积 \(\Rightarrow\) \(f\) Lebesgue 可积，且

\[\int_a^b f(x)\, dx = \int_{[a,b]} f\, d\lambda\]

经典反例

Dirichlet 函数 \(f = \mathbf{1}_\mathbb{Q}\)：\(D_f = \mathbb{R}\)，\(\lambda(D_f) > 0\)，故不 Riemann 可积。但 \(f\) 是简单函数，Lebesgue 可积且 \(\int_{[0,1]} f\, d\lambda = 0\)。

🌐 实际应用

PDE 弱解理论——偏微分方程的弱解只需 Lebesgue 可积（不要求经典可微），使得 Navier-Stokes 等方程在 Riemann 框架下无法表述的解有了严格意义。

傅里叶分析的 \(L^2\) 理论——Riemann 积分下傅里叶级数的收敛性极难处理，Lebesgue 积分 + \(L^2\) 完备性使 Parseval 等式和 Plancherel 定理得以成立，是信号处理的理论基础。

概率论的测度论基础——Kolmogorov 公理化概率论中，期望 = Lebesgue 积分。连续型与离散型随机变量的统一处理只有在 Lebesgue 框架下才自然实现。

✏️ 例题分析与易错点

例题：积分与极限不可无条件交换

设 \(f_n(x) = n \cdot \mathbf{1}_{(0,\, 1/n)}(x)\) 在 \([0,1]\) 上。每个 \(f_n\) Riemann 可积，\(\int_0^1 f_n = 1\)。但 \(f_n(x) \to 0\)（\(x > 0\)）。

解：\(\lim \int f_n = 1 \neq 0 = \int \lim f_n\)。原因：不存在可积控制函数 \(g\) 使 \(|f_n| \le g\)（DCT 条件不满足），也不满足单调递增（MCT 条件不满足）。这是"质量集中到越来越小区域"的逃逸现象。

常见错误

  • ❌ 认为逐点收敛就能交换极限与积分——必须有控制条件（DCT 的控制函数或 MCT 的单调性）。 ✅ 先验证收敛定理的前提再交换。
  • ❌ 误以为 Lebesgue 与 Riemann 积分值可能不同——在两者都定义时值恒相同。 ✅ 兼容性是 Lebesgue 理论的设计基石。
  • ❌ 把"不连续点零测"和"处处连续"混淆——有有限个不连续点的函数仍 Riemann 可积（零测集可远大于有限集）。 ✅ Lebesgue 判别法精确条件：\(\lambda(D_f) = 0\)。

04反常积分与 Lebesgue 积分

🎯 为什么研究这个问题

微积分中处理无穷区间上的积分 \(\int_0^\infty f\, dx\) 时，通常先截断再取极限。Lebesgue 积分直接在整个 \([0,\infty)\) 上定义，无需截断——但这种"直接性"以函数绝对可积为代价。两种积分何时一致、何时分歧？

  • Fourier 分析：Dirichlet 积分 \(\int_0^\infty \frac{\sin x}{x}\, dx = \frac{\pi}{2}\) 在反常 Riemann 意义下收敛，但 \(\sin x / x \notin L^1\)——Fourier 变换理论必须调和这一矛盾；
  • 概率论：期望 \(\mathbb{E}[X]\) 是 Lebesgue 积分——只承认绝对收敛意味着 Cauchy 分布没有期望，这不是缺陷而是深刻的概念性选择；
  • 分布理论：对 \(f \notin L^1\) 的情形，Schwartz 分布提供了超越 Lebesgue 可积性的推广——条件收敛的积分在此框架下重获意义。

Lebesgue 积分要求绝对可积看似是"限制"，实则是深思熟虑的设计——放弃条件收敛换来的是积分运算的完全稳定性：Fubini 定理、逐项积分、积分号下求导等强大工具都以绝对可积为前提，而条件收敛的积分无法保证这些运算的合法性。

🔍 抽象化方法
  • "绝对可积"作为准入门槛——Lebesgue 积分要求 \(\int |f| < \infty\)（\(L^1\) 条件），这排除了条件收敛，代价是损失某些积分，收获是积分运算的完全稳定性（重排、Fubini 等不受限）。
  • MCT 作为桥梁——对 \(f \ge 0\)，\(\int_{[0,\infty)} f\, d\lambda = \lim_{R \to \infty} \int_{[0,R]} f\, d\lambda\)（由单调收敛定理），因此非负情形下两种积分自动一致。
  • 分布理论的推广——当 \(f \notin L^1\) 时，仍可在 Schwartz 分布意义下定义"积分"（如 \(\sin x / x\) 的 Fourier 变换），超越了 Lebesgue 可积性的限制。
📐 公式推导 / 模型建立

反常 Riemann 积分的定义

\[\int_0^\infty f(x)\, dx \;:=\; \lim_{R \to \infty} \int_0^R f(x)\, dx\]

若极限存在（有限），称反常积分收敛。

与 Lebesgue 积分的关系

非负情形：\(f \ge 0\) 且反常 Riemann 积分收敛 \(\Rightarrow\) \(f \in L^1([0,\infty))\) 且两者相等。（MCT 保证。）

绝对收敛情形：\(\int_0^\infty |f|\, dx < \infty\)（反常 Riemann 意义）\(\Rightarrow\) \(f \in L^1\) 且 Lebesgue 积分 = 反常 Riemann 积分。

条件收敛反例：

\[\int_0^\infty \frac{\sin x}{x}\, dx = \frac{\pi}{2} \quad\text{（反常 Riemann 收敛）}\]

但 \(\int_0^\infty |\sin x / x|\, dx = +\infty\)，故 \(\sin x / x \notin L^1([0,\infty))\)。Lebesgue 积分不承认此积分。

判别准则

反常 Riemann 积分对 \(f\) 绝对收敛 \(\Leftrightarrow\) \(f \in L^1\)。此时 Lebesgue = Riemann（反常），且可自由使用 Fubini、换序等定理。

🌐 实际应用

Fourier 变换的 \(L^1\) 理论——\(\hat{f}(\xi) = \int_\mathbb{R} f(x) e^{-2\pi i \xi x}\, dx\) 要求 \(f \in L^1(\mathbb{R})\)（绝对可积）。\(\sin x / x \notin L^1\) 说明其 Fourier 变换需要在 \(L^2\) 或分布意义下定义——这推动了 Plancherel 定理和 Schwartz 分布理论的发展。

Laplace 变换与工程——控制论中 \(\mathcal{L}\{f\}(s) = \int_0^\infty f(t) e^{-st}\, dt\) 在 \(\text{Re}(s)\) 足够大时绝对收敛，此时可在 Lebesgue 框架下自由交换积分顺序（Fubini），保证了传递函数运算的严格性。

量子场论中的正规化——Feynman 积分经常条件收敛但不绝对收敛；物理学家使用维度正规化等技巧使之绝对收敛后，才能在 Lebesgue 框架下严格处理——这正是条件收敛 vs 绝对收敛之区别的物理体现。

✏️ 例题分析与易错点

例题：判断 Lebesgue 可积性

判断 \(\int_1^\infty \frac{\cos x}{x^2}\, dx\) 是否 Lebesgue 可积。

解：\(|\cos x / x^2| \le 1/x^2\)，而 \(\int_1^\infty x^{-2}\, dx = 1 < \infty\)。由比较判别法，\(\cos x / x^2 \in L^1([1,\infty))\)，Lebesgue 可积。

常见错误

  • ❌ 混淆"反常 Riemann 收敛"与"Lebesgue 可积"——\(\sin x / x\) 反常收敛但不 \(L^1\)。 ✅ Lebesgue 可积 = 绝对可积（\(\int |f| < \infty\)）。
  • ❌ 对非负函数仍担心两种积分不一致——\(f \ge 0\) 时 MCT 保证两者自动一致。 ✅ 仅变号函数的条件收敛才可能产生差异。
  • ❌ 认为 Lebesgue 积分"更小"——两种积分在都定义时值相同，Lebesgue 只是不接受条件收敛。 ✅ 绝对收敛时两者完全等价。
⚡ 快速检测
\(\int_0^{\infty} \frac{\sin x}{x} dx\) 作为反常 Riemann 积分
