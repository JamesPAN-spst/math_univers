Lebesgue 积分

01简单函数积分与非负函数积分

🎯 为什么研究这个问题

有了测度和可测函数之后，下一步是定义积分。Lebesgue 的革命性思想 是：不像 Riemann 那样沿定义域划分，而是沿值域划分——先处理 最简单的阶梯函数（简单函数），再通过单调极限扩展到所有非负可测 函数。这种"自底向上"的构造保证了积分与极限运算的良好相容性。

🔍 抽象化方法

Lebesgue 积分的核心思想是"按值域分层"：\(\int f\, d\mu\) 不是 将定义域切成小块，而是问"\(f\) 取值在 \([a, a+da)\) 的那些 \(x\) 构成多大的集合"。这一思路将"求面积"从几何操作升级为测度运算。

📐 公式推导 / 模型建立

简单函数积分：对 \(\varphi = \sum_{i=1}^n a_i \mathbf{1}_{A_i}\) （标准表示，\(A_i\) 两两不交），定义 \[\int \varphi\, d\mu = \sum_{i=1}^n a_i \, \mu(A_i)\] 非负可测函数积分：对 \(f \geq 0\) 可测，定义 \[\int f\, d\mu = \sup\left\{\int \varphi\, d\mu : 0 \leq \varphi \leq f,\; \varphi \text{ 简单}\right\}\] 一般可积函数：\(f = f^+ - f^-\)，其中 \(f^+ = \max(f, 0)\)， \(f^- = \max(-f, 0)\)。当 \(\int f^+\, d\mu\) 和 \(\int f^-\, d\mu\) 至少一个有限时，定义 \(\int f\, d\mu = \int f^+ d\mu - \int f^- d\mu\)。 \(f\) 可积（\(f \in L^1\)）\(\Leftrightarrow \int |f|\, d\mu < \infty\)。

✏️ 例题分析与易错点

例题：计算 Dirichlet 函数 \(f = \mathbf{1}_{\mathbb{Q} \cap [0,1]}\) 关于 Lebesgue 测度的积分。 解答：\(f\) 是简单函数，\(\int f\, d\lambda = 1 \cdot \lambda(\mathbb{Q} \cap [0,1]) = 1 \cdot 0 = 0\)。 注意：此函数不 Riemann 可积（上下 Darboux 和分别为 1 和 0）， 但 Lebesgue 可积且积分为 0。 易错点：忘记 Lebesgue 积分定义中取的是 \(\sup\) 而非 \(\lim\)。 积分等于所有小于 \(f\) 的简单函数积分的上确界。

02积分的基本性质

🎯 为什么研究这个问题

积分作为分析学的核心运算，需要满足一系列代数和序结构性质。 线性性和单调性是最基本的——它们保证积分是"合理的"度量函数 "总量"的方式。Chebyshev 不等式则将积分与函数水平集的测度 联系起来，是概率论中不可或缺的工具。

📐 公式推导 / 模型建立

线性性：对可积函数 \(f, g\) 和常数 \(\alpha, \beta\)， \[\int (\alpha f + \beta g)\, d\mu = \alpha \int f\, d\mu + \beta \int g\, d\mu\] 先对简单函数验证（直接由定义），再由逼近传递到一般情形。 单调性：\(f \leq g\) a.e. \(\Rightarrow \int f\, d\mu \leq \int g\, d\mu\)。 三角不等式：\(\left|\int f\, d\mu\right| \leq \int |f|\, d\mu\)。 Chebyshev(Markov) 不等式：对 \(f \geq 0\), \(\alpha > 0\)， \[\mu(\{x : f(x) \geq \alpha\}) \leq \frac{1}{\alpha} \int f\, d\mu\] 证明：\(\alpha \cdot \mathbf{1}_{\{f \geq \alpha\}} \leq f\)， 积分两边即得。 a.e. 相等 ⇒ 积分相等：若 \(f = g\) a.e.，则 \(\int f\, d\mu = \int g\, d\mu\)。

🌐 实际应用

Chebyshev 不等式在概率论中化为 \(P(|X| \geq \alpha) \leq E[|X|]/\alpha\)，广泛用于估计随机变量的尾概率。

✏️ 例题分析与易错点

例题：设 \(f \in L^1([0,1])\) 且 \(\int_0^1 f\, d\lambda = 3\)。 证明 \(\lambda(\{x : f(x) \geq 1\}) > 0\)。 解答：\(f = f \cdot \mathbf{1}_{\{f \geq 1\}} + f \cdot \mathbf{1}_{\{f < 1\}}\)。 \(\int f \cdot \mathbf{1}_{\{f < 1\}} d\lambda < 1 \cdot \lambda([0,1]) = 1\)。 故 \(\int f \cdot \mathbf{1}_{\{f \geq 1\}} d\lambda > 2 > 0\)， 因此 \(\lambda(\{f \geq 1\}) > 0\)。 易错点：试图用 Chebyshev 不等式时方向搞反。 \(\mu(\{f \geq \alpha\}) \leq \int f / \alpha\) 给出的是上界， 不能直接用来得到下界。

03Riemann 与 Lebesgue 积分的比较

🎯 为什么研究这个问题

学生自然会问：既然已经有了 Riemann 积分，为什么还要引入更复杂的 Lebesgue 积分？关键在于 Riemann 积分对极限运算不够稳定： 逐点收敛的 Riemann 可积函数序列的极限可能不 Riemann 可积。 Lebesgue 积分的优势恰恰在于它与极限运算的完美相容性。

🔍 抽象化方法

Riemann 积分"沿定义域切分"，Lebesgue 积分"沿值域切分"。 后者的抽象层次更高，代价是需要测度论基础，但收获是 强大的收敛定理（MCT, DCT）。

📐 公式推导 / 模型建立

Lebesgue 判别法：有界函数 \(f: [a,b] \to \mathbb{R}\) Riemann 可积 \(\Leftrightarrow\) \(f\) 的不连续点集 \(D_f\) 满足 \(\lambda(D_f) = 0\)。 证明思路（充分性）：设 \(\lambda(D_f) = 0\)。对划分 \(P\) 的 上下 Darboux 和 \(U(P, f) - L(P, f) = \sum (M_i - m_i) \Delta x_i\)。 在连续点附近 \(M_i - m_i\) 可任意小；不连续点集零测， 其对总振荡的贡献可以通过精细划分控制。 关系定理：\(f\) 在 \([a,b]\) 上 Riemann 可积 \(\Rightarrow\) \(f\) Lebesgue 可积，且 \[\int_a^b f(x)\, dx = \int_{[a,b]} f\, d\lambda\] 反例：\(f = \mathbf{1}_{\mathbb{Q}}\)，\(D_f = \mathbb{R}\)， \(\lambda(D_f) > 0\)（实际上 \(= \infty\)），所以不 Riemann 可积。 但 \(f\) Lebesgue 可积。

✏️ 例题分析与易错点

例题：\(f_n(x) = n \cdot \mathbf{1}_{(0, 1/n)}(x)\) 在 \([0,1]\) 上。 每个 \(f_n\) 都 Riemann 可积，且 \(\int_0^1 f_n = 1\)。但 \(f_n(x) \to 0\) 对所有 \(x > 0\)（即 a.e.）。所以 \(\lim \int f_n = 1 \neq 0 = \int \lim f_n\)。 为什么极限和积分不能交换？ 解答：因为没有可积控制函数 \(g\) 使 \(|f_n| \leq g\)（DCT 的条件 不满足）。这恰好说明收敛定理的条件是必要的。 易错点：认为"逐点收敛就能交换极限和积分"。 必须有控制条件（如 DCT 的控制函数）才行。

⚡ 快速检测
函数 \(f(x) = \begin{cases} 1 & x \in \mathbb{Q} \cap [0,1] \\ 0 & x \notin \mathbb{Q} \end{cases}\) 是否 Riemann 可积？

04反常积分与 Lebesgue 积分

🎯 为什么研究这个问题

在微积分中我们学过反常积分 \(\int_0^{\infty} f(x)\, dx\)，它通过 截断后取极限定义。Lebesgue 积分处理无穷区间上的积分有不同的方式： 它直接对整个定义域定义积分，无需截断。两种积分在什么条件下一致？ 何时反常 Riemann 积分存在但 Lebesgue 积分不存在？

📐 公式推导 / 模型建立

反常 Riemann 积分：\(\int_0^{\infty} f = \lim_{R \to \infty} \int_0^R f(x)\, dx\)。 Lebesgue 积分：\(\int_{[0,\infty)} f\, d\lambda\)，要求 \(\int |f|\, d\lambda < \infty\)。 关系：若 \(f \geq 0\) 且反常 Riemann 积分收敛，则 Lebesgue 积分存在且两者相等。 但对变号函数，可能反常 Riemann 积分存在（条件收敛）而 Lebesgue 积分不存在。经典例子： \[\int_0^{\infty} \frac{\sin x}{x}\, dx = \frac{\pi}{2}\] 此反常 Riemann 积分存在，但 \(\int_0^{\infty} |\sin x / x|\, dx = +\infty\)， 故 \(\sin(x)/x \notin L^1([0,\infty))\)。 Lebesgue 积分要求绝对可积（\(L^1\)），不承认条件收敛， 这保证了积分运算更强的稳定性。

✏️ 例题分析与易错点

例题：判断 \(\int_1^{\infty} \frac{\cos x}{x^2}\, dx\) 是否 Lebesgue 可积。 解答：\(|\cos x / x^2| \leq 1/x^2\)，而 \(\int_1^{\infty} 1/x^2\, dx = 1 < \infty\)，由比较判别法，\(\cos x / x^2 \in L^1([1,\infty))\)。 易错点：混淆"反常 Riemann 积分收敛"与"Lebesgue 可积"。 \(\sin(x)/x\) 的反常积分收敛但不 Lebesgue 可积。 Lebesgue 可积 = 绝对可积。

⚡ 快速检测
\(\int_0^{\infty} \frac{\sin x}{x} dx\) 作为反常 Riemann 积分
