01Lp 范数与 Hölder/Minkowski 不等式

🎯 为什么研究这个问题

我们希望在可积函数空间上定义"距离"，以便讨论函数的逼近和收敛。 对不同的指数 \(p\)，\(L^p\) 范数刻画了函数的不同"大小"特征。 但要使 \(\|f\|_p\) 真正成为范数，需要验证三角不等式——这就是 Minkowski 不等式。而 Hölder 不等式是证明 Minkowski 不等式的 关键工具，也是泛函分析中对偶理论的基石。

🔍 抽象化方法

\(L^p\) 空间将函数视为无穷维向量空间中的"点"。范数赋予这个 空间度量结构，使得分析学的逼近方法（如 Fourier 级数收敛、 密度论证）得以展开。

📐 公式推导 / 模型建立

\(L^p\) 范数：对 \(1 \leq p < \infty\)， \(\|f\|_p = \left(\int |f|^p\, d\mu\right)^{1/p}\)。 \(L^\infty\) 范数：\(\|f\|_\infty = \inf\{M : |f| \leq M \text{ a.e.}\}\)。 Hölder 不等式：设 \(1/p + 1/q = 1\)（\(1 < p < \infty\)），则 \[\int |fg|\, d\mu \leq \|f\|_p \cdot \|g\|_q\] 证明关键：Young 不等式 \(ab \leq a^p/p + b^q/q\)（\(a, b \geq 0\)）， 应用于 \(a = |f|/\|f\|_p\), \(b = |g|/\|g\|_q\)，积分两边得 \(\int |fg|/(\|f\|_p \|g\|_q) \leq 1/p + 1/q = 1\)。 Minkowski 不等式：\(\|f + g\|_p \leq \|f\|_p + \|g\|_p\)。 证明：\(|f+g|^p \leq |f+g|^{p-1}(|f| + |g|)\)，对右边用 Hölder： \[\|f+g\|_p^p \leq \|f+g\|_p^{p-1}(\|f\|_p + \|g\|_p)\] 除以 \(\|f+g\|_p^{p-1}\) 即得。

🌐 实际应用

概率论中，Hölder 不等式的 \(p=q=2\) 特例即 Cauchy-Schwarz， 用于证明相关系数有界 \(|\rho(X,Y)| \leq 1\)。

✏️ 例题分析与易错点

例题：设 \(f \in L^3([0,1])\), \(g \in L^{3/2}([0,1])\)。证明 \(fg \in L^1([0,1])\)。 解答：\(1/3 + 2/3 = 1\)，即 \(p = 3\), \(q = 3/2\)。 由 Hölder，\(\int |fg| \leq \|f\|_3 \|g\|_{3/2} < \infty\)。✓ 易错点：将 Hölder 不等式中的共轭指数搞错。 \(1/p + 1/q = 1\) 意味 \(q = p/(p-1)\)，不是 \(q = p\)。

02Lp 空间的完备性 (Riesz-Fischer)

🎯 为什么研究这个问题

一个范数空间不一定完备——Cauchy 列可能不收敛。但在分析学中， 完备性至关重要：它保证逼近过程有极限。Riesz-Fischer 定理证明 \(L^p\) 空间是完备的（即 Banach 空间），这使得 \(L^p\) 成为 泛函分析的基本框架。

🔍 抽象化方法

完备性的证明策略：先从 Cauchy 列中抽取快速收敛的子列 （使 \(\|f_{n_{k+1}} - f_{n_k}\|_p < 2^{-k}\)），构造 级数 \(\sum (f_{n_{k+1}} - f_{n_k})\)，由绝对收敛推出 收敛，最后证明整个 Cauchy 列收敛。

📐 公式推导 / 模型建立

定理 (Riesz-Fischer)：\(L^p(\mu)\)（\(1 \leq p \leq \infty\)）是 Banach 空间。 证明框架（\(1 \leq p < \infty\)）：设 \(\{f_n\}\) 是 \(L^p\) 中 Cauchy 列。 取子列 \(\{f_{n_k}\}\) 使 \(\|f_{n_{k+1}} - f_{n_k}\|_p < 2^{-k}\)。 令 \(g_N = \sum_{k=1}^N |f_{n_{k+1}} - f_{n_k}|\)。 由 Minkowski，\(\|g_N\|_p \leq \sum_{k=1}^N 2^{-k} < 1\)。 \(g_N \uparrow g\) 点点，由 MCT，\(g \in L^p\)。 故级数 \(f_{n_1} + \sum_{k=1}^{\infty}(f_{n_{k+1}} - f_{n_k})\) 绝对收敛 a.e.，定义其极限为 \(f\)。 再证 \(\|f_{n_k} - f\|_p \to 0\)（由 DCT，控制函数 \((|f| + g)^p\)）。 最后由 Cauchy 列性质推出 \(\|f_n - f\|_p \to 0\)。

🌐 实际应用

\(L^2\) 的完备性保证 Fourier 级数在均方意义下收敛； \(L^p\) 的完备性保证 PDE 弱解的存在性论证合法。

✏️ 例题分析与易错点

例题：\(l^p\)（可数测度空间上的 \(L^p\)）也是 Banach 空间。 验证 \(\{e_n\}\)（标准基向量 \(e_n = (0,\ldots,0,1,0,\ldots)\)） 在 \(l^2\) 中不是 Cauchy 列。 解答：\(\|e_m - e_n\|_2 = \sqrt{2}\) 对所有 \(m \neq n\)， 不趋于 0，故不是 Cauchy 列。 易错点：混淆"\(L^p\) 中 Cauchy 列收敛"与"逐点收敛"。 \(L^p\) 收敛只保证 a.e. 子列逐点收敛，全列不一定逐点收敛。

03L² 空间与 Hilbert 结构

🎯 为什么研究这个问题

在所有 \(L^p\) 空间中，\(L^2\) 最特别——它有内积结构。内积使得我们 可以定义"正交性"，从而进行正交分解、投影、Fourier 展开。 \(L^2\) 是量子力学中状态空间的数学模型，也是 Fourier 分析和 统计学中的核心空间。

🔍 抽象化方法

\(L^2\) 的内积 \(\langle f, g \rangle = \int f \bar{g}\, d\mu\) 将 函数空间赋予了欧几里得几何的结构。正交补和投影定理将 有限维线性代数推广到无穷维。

📐 公式推导 / 模型建立

内积：\(\langle f, g \rangle = \int f \bar{g}\, d\mu\)， 诱导范数 \(\|f\|_2 = \sqrt{\langle f, f \rangle}\)。 平行四边形恒等式： \[\|f + g\|_2^2 + \|f - g\|_2^2 = 2(\|f\|_2^2 + \|g\|_2^2)\] 此恒等式在 \(L^p\)（\(p \neq 2\)）中不成立——它刻画了 哪些 Banach 空间的范数来自内积。 Cauchy-Schwarz：\(|\langle f, g \rangle| \leq \|f\|_2 \|g\|_2\) （Hölder 不等式的 \(p = q = 2\) 特例）。 正交性：\(f \perp g \Leftrightarrow \langle f, g \rangle = 0\)。 Parseval 等式：若 \(\{e_n\}\) 是标准正交基，则 \[\|f\|_2^2 = \sum_n |\langle f, e_n \rangle|^2\]

🌐 实际应用

量子力学中，\(L^2(\mathbb{R}^3)\) 是粒子状态空间， \(\langle \psi, \phi \rangle\) 是概率振幅（Born 规则）。

✏️ 例题分析与易错点

例题：在 \(L^2([0, 2\pi])\) 中验证 \(e_n(x) = e^{inx}/\sqrt{2\pi}\) 构成标准正交系。 解答：\(\langle e_m, e_n \rangle = \frac{1}{2\pi} \int_0^{2\pi} e^{i(m-n)x} dx\)。 \(m = n\) 时 \(= 1\)；\(m \neq n\) 时 \(= \frac{1}{2\pi} \cdot \frac{e^{i(m-n)2\pi} - 1}{i(m-n)} = 0\)。✓ 易错点：在实值 \(L^2\) 中忘记共轭。对复值情形， 内积定义为 \(\int f \bar{g}\)，不是 \(\int fg\)。

04稠密性与可分性

🎯 为什么研究这个问题

在无穷维空间中工作时，我们希望复杂函数可以被"简单"函数逼近。 \(L^p\) 空间中的稠密性定理告诉我们：简单函数、连续函数、甚至 光滑的紧支函数都可以在 \(L^p\) 范数下逼近任意 \(L^p\) 函数。 这保证了数值逼近方法的理论合法性。

📐 公式推导 / 模型建立

稠密性链（\(1 \leq p < \infty\)）： \[C_c^{\infty}(\mathbb{R}^n) \subset C_c(\mathbb{R}^n) \subset \text{简单函数} \subset L^p(\mathbb{R}^n)\] 每一层在下一层中稠密（\(L^p\) 范数下）。 简单函数在 \(L^p\) 中稠密：由简单函数逼近定理， \(\varphi_n \uparrow |f|\)，\(|\varphi_n|^p \leq |f|^p \in L^1\)， 由 DCT，\(\|\varphi_n - |f|\|_p \to 0\)。对一般 \(f\) 分正负部处理。 连续函数稠密：利用 Lebesgue 测度的正则性， 可测集 \(A\) 的特征函数可被连续函数逼近（Lusin 定理： 几乎处处等于连续函数）。 可分性：\(L^p(\mathbb{R}^n)\)（\(1 \leq p < \infty\)）可分， 因为有理系数阶梯函数是可数稠密子集。 但 \(L^\infty\) 不可分。

🌐 实际应用

Fourier 分析中，\(L^2\) 的可分性保证存在可数标准正交基 \(\{e^{inx}\}\)，使 Fourier 级数逼近有意义。

✏️ 例题分析与易错点

例题：证明 \(L^\infty([0,1])\) 不可分。 解答：对每个 \(t \in (0,1)\)，令 \(f_t = \mathbf{1}_{[0,t]}\)。 当 \(s \neq t\) 时 \(\|f_s - f_t\|_\infty = 1\)。 开球 \(B(f_t, 1/3)\) 两两不交，有不可数多个。 可分空间中不可能有不可数多个两两不交的开球。✓ 易错点：误以为所有 \(L^p\) 都可分。\(L^\infty\) 是例外。
