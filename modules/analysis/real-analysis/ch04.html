01单调收敛定理 (MCT)

🎯 为什么研究这个问题

分析学的核心问题之一是：何时可以交换极限和积分？单调收敛定理 给出最简洁的条件——只要函数序列非负且单调递增，极限和积分就 可以交换。这是 Lebesgue 积分理论中最基础、最常用的工具， 也是证明其他收敛定理的基石。

🔍 抽象化方法

MCT 体现了 Lebesgue 积分的"从下逼近"本质：非负函数的积分 本身就是简单函数积分的上确界，而单调递增序列的上确界就是极限。

📐 公式推导 / 模型建立

定理 (MCT)：设 \(0 \leq f_1 \leq f_2 \leq \cdots\) 为可测函数序列， \(f_n \uparrow f\) 逐点，则 \[\lim_{n \to \infty} \int f_n\, d\mu = \int f\, d\mu\] （两边可以是 \(+\infty\)）。 证明框架： "\(\leq\)"：\(f_n \leq f\) 对所有 \(n\)，由单调性 \(\int f_n \leq \int f\)， 取极限得 \(\lim \int f_n \leq \int f\)。 "\(\geq\)"：对任意简单函数 \(\varphi \leq f\) 和 \(0 < c < 1\)， 令 \(E_n = \{x : f_n(x) \geq c\varphi(x)\}\)，则 \(E_n \uparrow X\)。 \[\int f_n\, d\mu \geq \int_{E_n} f_n\, d\mu \geq c \int_{E_n} \varphi\, d\mu\] 由测度的连续性，\(\int_{E_n} \varphi\, d\mu \to \int \varphi\, d\mu\)。 令 \(n \to \infty\) 再令 \(c \to 1\)，得 \(\lim \int f_n \geq \int \varphi\)。 对所有 \(\varphi \leq f\) 取 \(\sup\)，得 \(\lim \int f_n \geq \int f\)。

🌐 实际应用

MCT 的级数版本：\(\int \sum f_n = \sum \int f_n\)（\(f_n \geq 0\)）， 用于证明 \(L^p\) 空间的完备性和级数的逐项积分。

✏️ 例题分析与易错点

例题：计算 \(\int_0^1 \sum_{n=1}^{\infty} x^n\, dx\)。 解答：\(f_n(x) = x^n \geq 0\)，部分和 \(S_N = \sum_{n=1}^N x^n\) 单调递增趋向 \(x/(1-x)\)（\(0 \leq x < 1\)）。由 MCT， \(\int_0^1 \frac{x}{1-x} dx = \sum_{n=1}^{\infty} \int_0^1 x^n dx = \sum_{n=1}^{\infty} \frac{1}{n+1}\)。 注意此级数发散，所以积分为 \(+\infty\)（\(x/(1-x)\) 在 1 附近不可积）。 易错点：忘记 MCT 要求 \(f_n \geq 0\)。 对可以取负值的函数序列，MCT 不适用，需用 DCT。

02Fatou 引理

🎯 为什么研究这个问题

当函数序列不单调递增时，MCT 不再适用。Fatou 引理提供了一个 更一般的工具：对非负函数序列，积分的下极限至少不小于下极限的 积分。虽然这只是一个不等式而非等式，但在证明 DCT 和研究弱 收敛等问题中，Fatou 引理是不可或缺的。

🔍 抽象化方法

Fatou 引理可视为 MCT 的推论：\(g_n = \inf_{k \geq n} f_k\) 单调递增趋向 \(\liminf f_n\)，对 \(g_n\) 用 MCT 即得。

📐 公式推导 / 模型建立

定理 (Fatou 引理)：设 \(f_n \geq 0\) 可测，则 \[\int \liminf_{n \to \infty} f_n\, d\mu \leq \liminf_{n \to \infty} \int f_n\, d\mu\] 证明：令 \(g_n = \inf_{k \geq n} f_k\)，则： (1) \(g_n\) 可测（可测函数的 \(\inf\) 可测）； (2) \(0 \leq g_1 \leq g_2 \leq \cdots\)（递增）； (3) \(g_n \uparrow \liminf f_n\)。 由 MCT，\(\int \liminf f_n = \lim \int g_n\)。 又 \(g_n \leq f_n\) 意味 \(\int g_n \leq \int f_n\)，故 \[\lim_{n} \int g_n = \liminf_{n} \int g_n \leq \liminf_{n} \int f_n\] 不等式可以严格：例如 \(f_n = n \cdot \mathbf{1}_{(0,1/n)}\)， \(\liminf f_n = 0\) a.e.，但 \(\int f_n = 1\) 对所有 \(n\)。

✏️ 例题分析与易错点

例题：\(f_n = \mathbf{1}_{[n, n+1]}\) 在 \((\mathbb{R}, \lambda)\) 上。 验证 Fatou 引理。 解答：\(\liminf f_n(x) = 0\) 对所有 \(x\)（任意 \(x\) 只属于有限个 \([n, n+1]\)），故 \(\int \liminf f_n = 0\)。 \(\int f_n = 1\) 对所有 \(n\)，故 \(\liminf \int f_n = 1\)。 不等式 \(0 \leq 1\) 成立，但严格不等。✓ 易错点：试图将 Fatou 引理反向使用（\(\geq\)）。 Fatou 引理只给出一个方向的不等式，严格不等的反例说明 反向不成立。

03控制收敛定理 (DCT)

🎯 为什么研究这个问题

DCT 是 Lebesgue 积分理论中最实用的定理——它告诉我们：如果 函数序列逐点收敛且被一个可积函数"控制"，则极限和积分可以交换。 这正是 Riemann 积分做不到的——DCT 的威力使得 Lebesgue 积分 在分析学、概率论和 PDE 中成为标准工具。

🔍 抽象化方法

DCT 的证明是 Fatou 引理的双向应用：对 \(g + f_n\) 和 \(g - f_n\) 分别使用 Fatou 引理，利用控制函数 \(g\) 的可积性"两面夹击"。

📐 公式推导 / 模型建立

定理 (DCT)：设 \(f_n\) 可测，\(f_n \to f\) a.e.，存在 \(g \in L^1\) 使 \(|f_n| \leq g\) a.e. 对所有 \(n\)。则 \(f \in L^1\) 且 \[\lim_{n \to \infty} \int f_n\, d\mu = \int f\, d\mu\] 即 \(\int |f_n - f|\, d\mu \to 0\)。 证明：\(|f| = \lim |f_n| \leq g\)，故 \(f \in L^1\)。 \(g + f_n \geq 0\)，对 \(\{g + f_n\}\) 用 Fatou 引理： \[\int (g + f) \leq \liminf \int (g + f_n)\] 即 \(\int f \leq \liminf \int f_n\)。 \(g - f_n \geq 0\)，对 \(\{g - f_n\}\) 用 Fatou 引理： \[\int (g - f) \leq \liminf \int (g - f_n)\] 即 \(-\int f \leq -\limsup \int f_n\)，即 \(\limsup \int f_n \leq \int f\)。 合并：\(\int f \leq \liminf \int f_n \leq \limsup \int f_n \leq \int f\)。

🌐 实际应用

概率论中 DCT 对应有界收敛定理；PDE 中利用 DCT 证明 含参积分的连续性和可微性（Leibniz 积分法则的严格化）。

✏️ 例题分析与易错点

例题：计算 \(\lim_{n \to \infty} \int_0^{\infty} \frac{n \sin(x/n)}{x(1+x^2)}\, dx\)。 解答：\(f_n(x) = \frac{n \sin(x/n)}{x(1+x^2)}\)。 当 \(n \to \infty\)，\(n\sin(x/n) \to x\)，故 \(f_n(x) \to \frac{1}{1+x^2}\) 逐点。 控制函数：\(|n\sin(x/n)| \leq x\)，所以 \(|f_n(x)| \leq \frac{1}{1+x^2} = g(x) \in L^1(\mathbb{R}^+)\)。 由 DCT，\(\lim \int f_n = \int \frac{1}{1+x^2} dx = \frac{\pi}{2}\)。 易错点：找不到控制函数时草率声称可以用 DCT。 必须明确给出控制函数 \(g\) 并验证 \(g \in L^1\)。

04积分号下求导与逐项积分

🎯 为什么研究这个问题

实际分析中经常遇到含参积分 \(F(t) = \int f(x,t)\, d\mu(x)\)， 需要对参数 \(t\) 求导。Leibniz 积分法则（积分号下求导）何时合法？ 类似地，无穷级数 \(\sum f_n\) 何时可以逐项积分？这些问题都是 收敛定理的直接应用。

📐 公式推导 / 模型建立

积分号下求导 (Leibniz 规则的测度论版)： 设 \(f(x,t)\) 关于 \(t\) 可微（a.e. \(x\)），且存在 \(g \in L^1(\mu)\) 使 \(|\partial f/\partial t(x,t)| \leq g(x)\) 对所有 \(t\)。则 \[\frac{d}{dt} \int f(x,t)\, d\mu(x) = \int \frac{\partial f}{\partial t}(x,t)\, d\mu(x)\] 证明：\(\frac{F(t+h) - F(t)}{h} = \int \frac{f(x,t+h) - f(x,t)}{h} d\mu\)。 由均值定理，差商被 \(g(x)\) 控制，对 \(h \to 0\) 用 DCT 即得。 逐项积分：若 \(f_n \geq 0\) 或 \(\sum \int |f_n| < \infty\)，则 \[\int \sum_{n=1}^{\infty} f_n\, d\mu = \sum_{n=1}^{\infty} \int f_n\, d\mu\] 非负情形由 MCT 保证；绝对收敛情形由 DCT 保证 （控制函数 \(g = \sum |f_n|\) 可积）。

🌐 实际应用

含参积分求导在统计学中用于计算 Fisher 信息 \(I(\theta) = -E[\partial^2 \log f(X|\theta)/\partial\theta^2]\)； 在物理学中用于求解场论中的传播子。

✏️ 例题分析与易错点

例题：设 \(F(t) = \int_0^{\infty} e^{-tx} \frac{\sin x}{x}\, dx\)，\(t > 0\)。 求 \(F'(t)\)。 解答：\(\frac{\partial}{\partial t}(e^{-tx} \sin x/x) = -e^{-tx} \sin x\)， \(|{-e^{-tx} \sin x}| \leq e^{-t_0 x}\) 对 \(t \geq t_0 > 0\)， 且 \(e^{-t_0 x} \in L^1([0,\infty))\)。由 Leibniz 规则， \(F'(t) = -\int_0^{\infty} e^{-tx} \sin x\, dx = -\frac{1}{1+t^2}\)。 积分得 \(F(t) = -\arctan t + C\)。由 \(F(\infty) = 0\) 得 \(C = \pi/2\)，故 \(F(t) = \pi/2 - \arctan t\)， 特别 \(F(0^+) = \pi/2\)（即 Dirichlet 积分）。 易错点：忘记验证控制条件，直接在积分号下求导。 必须找到不依赖 \(t\)（或在 \(t\) 的邻域内一致）的 \(L^1\) 控制函数。
