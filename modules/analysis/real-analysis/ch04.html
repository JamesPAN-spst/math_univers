收敛定理

01单调收敛定理 (MCT)

🎯 为什么研究这个问题

分析学的核心问题之一是：什么条件下极限与积分可以交换？单调收敛定理（MCT）给出了最简洁的充分条件——非负函数列单调递增即可，不要求任何一致性或可积性控制。这一看似简单的条件蕴含了极其丰富的推论。

  • 级数求和：将 \(f_n = \sum_{k=1}^n g_k\)（\(g_k \geq 0\)）代入 MCT 立即得到逐项积分 \(\int \sum g_k = \sum \int g_k\)——这是调和分析和数值方法中的基本工具；
  • 概率论：MCT 保证了非负随机变量期望的极限交换性，是大数定律和鞅收敛定理的技术基础；
  • PDE 理论：弱解的存在性证明中，逼近解序列的积分极限合法性通常依赖 MCT 或其推论。

MCT 的深层含义是：Lebesgue 积分的"自底向上"定义（简单函数逼近 → 取 \(\sup\)）本身就是 MCT 的"内置版"——定理只是将这一构造推广为可操作的工具。掌握 MCT 即掌握了 Fatou 引理、DCT、逐项积分等整个收敛定理群的根基。

🔍 抽象化方法
  • "从下逼近"原理——非负函数积分 = 简单函数积分的 \(\sup\)；单调递增序列的 \(\sup\) = \(\lim\)，因此 \(\int \lim = \lim \int\) 几乎是定义的直接推论。
  • 测度的连续性——证明中关键一步利用了测度对单调递增集列的连续性 \(\mu(E_n) \uparrow \mu(\bigcup E_n)\)，将点态收敛传递到积分。
  • 级数版本——取 \(f_n = \sum_{k=1}^n g_k\)（\(g_k \ge 0\)），MCT 立即给出 \(\int \sum g_k = \sum \int g_k\)——这是 Tonelli 定理的一维前身。
📐 公式推导 / 模型建立

定理陈述

设 \(0 \leq f_1 \leq f_2 \leq \cdots\) 为可测函数序列，\(f_n \uparrow f\) 逐点。则

\[\lim_{n \to \infty} \int f_n\, d\mu = \int f\, d\mu\]

（两边可以是 \(+\infty\)。）

证明框架

"\(\le\)" 方向：\(f_n \le f\) 对所有 \(n\)，由单调性 \(\int f_n \le \int f\)，取极限得 \(\lim \int f_n \le \int f\)。

"\(\ge\)" 方向：对任意简单函数 \(\varphi \le f\) 和 \(0 < c < 1\)，令 \(E_n = \{x : f_n(x) \ge c\varphi(x)\}\)，则 \(E_n \uparrow X\)。

\[\int f_n\, d\mu \;\ge\; \int_{E_n} f_n\, d\mu \;\ge\; c\int_{E_n} \varphi\, d\mu\]

由测度的连续性，\(\int_{E_n} \varphi \to \int \varphi\)。令 \(n \to \infty\) 再令 \(c \to 1\)：\(\lim \int f_n \ge \int \varphi\)。对所有 \(\varphi \le f\) 取 \(\sup\)，得 \(\lim \int f_n \ge \int f\)。

🌐 实际应用

级数的逐项积分——\(\int \sum_{n=1}^\infty f_n = \sum_{n=1}^\infty \int f_n\)（\(f_n \ge 0\)），这是数值求积中"无穷和 ↔ 积分"互换的理论保障，也是 Fubini/Tonelli 定理证明的关键步骤。

\(L^p\) 空间完备性——Riesz-Fischer 定理的标准证明先用 MCT 对部分和的范数取极限，确保 Cauchy 列在 \(L^p\) 中收敛——Banach 空间结构由此得证。

概率论中的期望计算——非负随机变量 \(X\) 的期望 \(E[X] = \lim E[\min(X, n)]\) 正是 MCT 的直接应用；"截断 + 取极限"是概率论计算的标准流程。

✏️ 例题分析与易错点

例题：逐项积分幂级数

计算 \(\int_0^1 \sum_{n=1}^\infty x^n\, dx\)。

解：\(f_n(x) = x^n \ge 0\)，部分和 \(S_N = \sum_{n=1}^N x^n\) 单调递增趋向 \(x/(1-x)\)（\(0 \le x < 1\)）。由 MCT：

\[\int_0^1 \frac{x}{1-x}\, dx = \sum_{n=1}^\infty \int_0^1 x^n\, dx = \sum_{n=1}^\infty \frac{1}{n+1}\]

此级数发散，故积分为 \(+\infty\)——MCT 允许极限为无穷。

常见错误

  • ❌ 对可取负值的函数直接用 MCT——MCT 要求 \(f_n \ge 0\)（或至少 \(f_1 \in L^1\) 的推广版）。 ✅ 变号序列用 DCT 或先平移使之非负。
  • ❌ 忘记 MCT 允许 \(\int f = +\infty\)——这与 DCT（要求 \(L^1\) 控制）不同。 ✅ MCT 两边可为 \(+\infty\)，无需有限性。
  • ❌ 单调递减时也套用 MCT——递减序列需额外条件（\(f_1 \in L^1\)）才能交换。 ✅ 对 \(f_1 - f_n \ge 0\) 递增，用标准 MCT 后倒推。

02Fatou 引理

🎯 为什么研究这个问题

当函数序列不单调递增时，MCT 不再适用。Fatou 引理对任意非负可测函数序列给出一个方向的估计——虽然只是不等式而非等式，但这个"单侧"结果的力量远超预期。

  • 概率论：Fatou 引理的概率版 \(\mathbb{E}[\liminf X_n] \leq \liminf \mathbb{E}[X_n]\) 是大数定律和鞅论中的基本工具；
  • 变分法：能量泛函的下半连续性证明中，Fatou 引理保证极小化序列的积分极限满足下界——这是直接方法的核心步骤；
  • PDE 弱收敛：弱解理论中，逼近解序列的范数估计常通过 Fatou 不等式传递到极限。

Fatou 不等式的"间隙" \(\liminf \int f_n - \int \liminf f_n\) 精确度量了序列中"逃逸"的积分量——质量可以在无穷远处消散或在尖峰处集中而丢失。理解这一间隙的来源，是深入掌握收敛定理群的关键。

🔍 抽象化方法
  • MCT 的推论——令 \(g_n = \inf_{k \ge n} f_k\)，则 \(g_n\) 单调递增趋向 \(\liminf f_n\)；对 \(g_n\) 用 MCT 再利用 \(g_n \le f_n\) 即得 Fatou。
  • 反向 Fatou——若 \(f_n \le g \in L^1\)，取 \(g - f_n \ge 0\) 用标准 Fatou，得 \(\limsup \int f_n \le \int \limsup f_n\)。
  • 概率论视角——Fatou 引理的概率版：\(E[\liminf X_n] \le \liminf E[X_n]\)（\(X_n \ge 0\)），是大数定律和鞅论中常用的基本工具。
📐 公式推导 / 模型建立

定理陈述

设 \(f_n \geq 0\) 可测，则

\[\int \liminf_{n \to \infty} f_n\, d\mu \;\leq\; \liminf_{n \to \infty} \int f_n\, d\mu\]

证明

令 \(g_n = \inf_{k \ge n} f_k\)：

(1) \(g_n\) 可测（可测函数的逐点 \(\inf\) 可测）；(2) \(0 \le g_1 \le g_2 \le \cdots\)（递增）；(3) \(g_n \uparrow \liminf f_n\)。

由 MCT：\(\int \liminf f_n = \lim \int g_n\)。又 \(g_n \le f_n\) 意味 \(\int g_n \le \int f_n\)，故

\[\lim_{n} \int g_n = \liminf_{n} \int g_n \;\leq\; \liminf_{n} \int f_n\]

不等式可严格

例：\(f_n = n \cdot \mathbf{1}_{(0,1/n)}\)，\(\liminf f_n = 0\) a.e.，但 \(\int f_n = 1\) 恒成立。间隙 = 1。

🌐 实际应用

概率论弱收敛——Portmanteau 定理中，\(\liminf P_n(O) \ge P(O)\) 对开集 \(O\) 的证明核心就是 Fatou 引理对指示函数的应用；这是统计推断中分布逼近的理论基石。

变分法下半连续性——能量泛函的下半连续性 \(I(u) \le \liminf I(u_n)\) 可视为 Fatou 引理在无穷维空间中的推广，是椭圆 PDE 弱解存在性的关键步骤。

信息论与 KL 散度——KL 散度的联合下半连续性 \(D_{KL}(P\|Q) \le \liminf D_{KL}(P_n\|Q_n)\) 亦依赖 Fatou 引理，保证了机器学习中变分推断的目标函数良定义。

✏️ 例题分析与易错点

例题：验证 Fatou 不等式

\(f_n = \mathbf{1}_{[n, n+1]}\) 在 \((\mathbb{R}, \lambda)\) 上。

解：\(\liminf f_n(x) = 0\) 对所有 \(x\)（任意 \(x\) 只属于有限个 \([n,n+1]\)），故 \(\int \liminf f_n = 0\)。\(\int f_n = 1\) 对所有 \(n\)，故 \(\liminf \int f_n = 1\)。不等式 \(0 \le 1\) 成立，但严格 ✓。

常见错误

  • ❌ 试图反向使用 Fatou（\(\ge\)）——标准 Fatou 只给单方向不等式；反向需额外的上界控制函数。 ✅ 反向 Fatou 要求 \(f_n \le g \in L^1\)。
  • ❌ 忘记 \(f_n \ge 0\) 的前提——对可取负值的序列，Fatou 不成立（\(\inf_{k \ge n} f_k\) 可能是 \(-\infty\)）。 ✅ 变号时先平移或取绝对值。
  • ❌ 混淆 \(\liminf\) 和 \(\lim\)——Fatou 引理用的是 \(\liminf\)，不要求极限存在。 ✅ 若极限存在，\(\liminf = \lim\)，但一般情况只有 \(\liminf\)。

03控制收敛定理 (DCT)

🎯 为什么研究这个问题

MCT 限于非负单调序列，Fatou 只给单侧估计——日常分析计算中最需要的是一个适用于变号函数、给出精确等式的收敛定理。控制收敛定理（DCT）正是这样的工具：只要有一个可积上界函数"控制"住整个序列，极限与积分就可以交换。

  • 含参积分：DCT 直接推出积分号下求导的 Leibniz 法则——统计中 Fisher 信息、物理中传播子微分都依赖这一结果；
  • 概率论：有界收敛定理（DCT 的特例）是概率论中最常用的极限交换工具，支撑着条件期望和鞅的大量推导；
  • PDE：弱解的对偶论证中，将试探函数的逼近序列代入方程后取极限，合法性由 DCT 保证。

DCT 的本质是 Fatou 引理的"双向夹逼"：对 \(g + f_n\) 和 \(g - f_n\) 分别用 Fatou，再借 \(g \in L^1\) 消去公共项，从两侧锁定极限。收敛定理的推广链 MCT → Fatou → DCT → Vitali 层层递进，条件逐步放宽，体现了测度论处理极限问题的系统力量。

🔍 抽象化方法
  • Fatou 的双向夹逼——对 \(g + f_n \ge 0\) 和 \(g - f_n \ge 0\) 分别用 Fatou 引理，利用 \(g \in L^1\) 消去，从两侧逼近得 \(\lim \int f_n = \int f\)。
  • "控制"= 一致可积——\(|f_n| \le g \in L^1\) 等价于 \(\{f_n\}\) 一致可积，这在概率论中直接对应均匀可积鞅的收敛。
  • 推广链——MCT（非负 + 单调）→ Fatou（非负）→ DCT（变号 + 控制）→ Vitali（一致可积）——层层推进，条件逐步放宽。
📐 公式推导 / 模型建立

定理陈述

设 \(f_n\) 可测，\(f_n \to f\) a.e.，存在 \(g \in L^1\) 使 \(|f_n| \le g\) a.e. 对所有 \(n\)。则 \(f \in L^1\) 且

\[\lim_{n \to \infty} \int f_n\, d\mu = \int f\, d\mu, \qquad \int |f_n - f|\, d\mu \to 0\]

证明

\(|f| = \lim |f_n| \le g\)，故 \(f \in L^1\)。

下界：\(g + f_n \ge 0\)，Fatou 给出 \(\int(g + f) \le \liminf \int(g + f_n)\)，即 \(\int f \le \liminf \int f_n\)。

上界：\(g - f_n \ge 0\)，Fatou 给出 \(\int(g - f) \le \liminf \int(g - f_n)\)，即 \(\limsup \int f_n \le \int f\)。

合并：\(\int f \le \liminf \int f_n \le \limsup \int f_n \le \int f\)。

🌐 实际应用

含参积分的连续性——\(F(t) = \int f(x,t)\, d\mu(x)\) 在 \(|f(x,t)| \le g(x) \in L^1\) 且 \(f(\cdot, t)\) 关于 \(t\) 逐点连续时，\(F\) 连续。这是 PDE 中 Green 函数性质分析的标准工具。

概率论有界收敛定理——若 \(|X_n| \le M\)（有界）且 \(X_n \to X\) a.s.，则 \(E[X_n] \to E[X]\)。这是 DCT 在有限测度空间上的特例，广泛用于鞅收敛和停时定理。

统计学似然推断——最大似然估计的渐近理论中，Fisher 信息的计算需要在积分号下对对数似然求导——DCT 保证了导数与期望的交换合法性。

✏️ 例题分析与易错点

例题：含参积分求极限

计算 \(\displaystyle\lim_{n \to \infty} \int_0^\infty \frac{n\sin(x/n)}{x(1+x^2)}\, dx\)。

解：令 \(f_n(x) = \frac{n\sin(x/n)}{x(1+x^2)}\)。当 \(n \to \infty\)，\(n\sin(x/n) \to x\)，故 \(f_n(x) \to \frac{1}{1+x^2}\)。控制：\(|n\sin(x/n)| \le x\)，故 \(|f_n(x)| \le \frac{1}{1+x^2} = g(x) \in L^1(\mathbb{R}^+)\)。由 DCT：

\[\lim \int f_n = \int \frac{1}{1+x^2}\, dx = \frac{\pi}{2}\]

常见错误

  • ❌ 找不到控制函数就草率声称可用 DCT——必须明确给出 \(g\) 并验证 \(g \in L^1\)。 ✅ 写出 \(|f_n| \le g\) 并计算 \(\int g\)。
  • ❌ 将 DCT 用于 \(\int f_n = +\infty\) 的情形——DCT 要求控制函数可积，极限函数自动可积。 ✅ 若积分可能发散，考虑用 MCT 或 Fatou。
  • ❌ 忽略"a.e."——DCT 只要求 \(f_n \to f\) a.e.（不需处处），常忘记验证。 ✅ a.e. 收敛 + a.e. 控制即可。

04积分号下求导与逐项积分

🎯 为什么研究这个问题

含参积分 \(F(t) = \int f(x,t)\, d\mu(x)\) 在数学和应用科学中无处不在——对参数 \(t\) 求导、对函数级数逐项积分都是基本需求。微积分中"积分号下求导"只是形式操作，测度论则明确给出了合法交换的精确条件。

  • 统计学：Fisher 信息 \(I(\theta) = -\mathbb{E}\!\left[\frac{\partial^2}{\partial\theta^2}\log f(X|\theta)\right]\) 的良定义性依赖积分号下二阶求导——需要偏导被可积函数控制；
  • 物理学：Feynman 传播子对坐标的微分性质通过 Leibniz 法则分析，控制条件由被积函数的衰减性保证；
  • 数值分析：将复杂积分展开为级数后逐项积分，是特殊函数计算和 Fourier 分析中的标准方法。

无论是积分号下求导还是逐项积分，核心技术始终是找到"不依赖参数"的可积控制函数。Leibniz 法则的本质是"差商被一致控制 → DCT 保证极限交换"；逐项积分的本质是"部分和单调递增 → MCT"（非负）或"总和绝对收敛 → DCT"（变号）——收敛定理群的统一逻辑在此充分体现。

🔍 抽象化方法
  • 差商 → 导数——积分号下求导本质是将差商 \(\frac{f(x,t+h) - f(x,t)}{h}\) 的积分取极限；由均值定理给出一致控制后，DCT 保证极限与积分交换。
  • 级数 = 部分和的极限——逐项积分将 \(\int \sum f_n\) 理解为 \(\int \lim S_N\)；非负时 MCT 直接适用，变号时需 \(\sum \int |f_n| < \infty\)（绝对收敛）。
  • 高阶推广——同样的技术可推广到高阶导数、多变量参数、甚至 Fréchet 导数——关键始终是找到"不依赖参数"的可积控制。
📐 公式推导 / 模型建立

积分号下求导（Leibniz 法则）

设 \(f(x,t)\) 关于 \(t\) 可微（a.e. \(x\)），且存在 \(g \in L^1(\mu)\) 使 \(\left|\frac{\partial f}{\partial t}(x,t)\right| \le g(x)\) 对所有 \(t\)。则

\[\frac{d}{dt}\int f(x,t)\, d\mu(x) = \int \frac{\partial f}{\partial t}(x,t)\, d\mu(x)\]

证明：差商 \(\frac{F(t+h) - F(t)}{h} = \int \frac{f(x,t+h) - f(x,t)}{h}\, d\mu\)。由均值定理，差商被 \(g(x)\) 控制。对 \(h \to 0\) 用 DCT 即得。

逐项积分

若 \(f_n \ge 0\) 或 \(\sum \int |f_n| < \infty\)，则

\[\int \sum_{n=1}^\infty f_n\, d\mu = \sum_{n=1}^\infty \int f_n\, d\mu\]

非负情形：部分和 \(S_N \uparrow \sum f_n\)，MCT 保证。绝对收敛情形：\(g = \sum |f_n| \in L^1\) 为控制函数，DCT 保证。

🌐 实际应用

统计 Fisher 信息——\(I(\theta) = -E\!\left[\frac{\partial^2}{\partial\theta^2}\log f(X|\theta)\right]\) 的合法性依赖于积分号下二阶求导——需要 \(\partial^2 \log f / \partial\theta^2\) 被可积函数控制。

物理传播子——量子场论中 Feynman 传播子 \(G(x) = \int \frac{e^{ipx}}{p^2 + m^2}\, d^4p\) 对坐标 \(x\) 的微分性质通过积分号下求导分析——控制条件由 \(|p|^{-2}\) 的衰减保证。

数值级数求和——交换 \(\int\) 与 \(\sum\) 后，将复杂的积分拆为可逐项计算的简单积分是 Fourier 分析和特殊函数计算的标准方法。

✏️ 例题分析与易错点

例题：Dirichlet 积分的优雅计算

设 \(F(t) = \int_0^\infty e^{-tx}\frac{\sin x}{x}\, dx\)（\(t > 0\)）。求 \(F'(t)\)，并推导 \(\int_0^\infty \frac{\sin x}{x}\, dx = \frac{\pi}{2}\)。

解：\(\frac{\partial}{\partial t}(e^{-tx}\sin x / x) = -e^{-tx}\sin x\)。对 \(t \ge t_0 > 0\)，\(|-e^{-tx}\sin x| \le e^{-t_0 x} \in L^1\)。由 Leibniz 法则：

\[F'(t) = -\int_0^\infty e^{-tx}\sin x\, dx = -\frac{1}{1+t^2}\]

积分得 \(F(t) = -\arctan t + C\)。由 \(F(\infty) = 0\) 得 \(C = \pi/2\)，故 \(F(0^+) = \pi/2\)。

常见错误

  • ❌ 积分号下求导时忘记验证控制条件——直接交换等于没有证明。 ✅ 必须找到不依赖参数 \(t\) 的 \(L^1\) 上界。
  • ❌ 逐项积分时不区分非负与变号——非负用 MCT（无条件），变号需绝对收敛 \(\sum \int |f_n| < \infty\)。 ✅ 先判断符号再选择定理。
  • ❌ 对 \(t = 0\) 边界点直接应用 Leibniz——控制函数在 \(t = 0\) 可能失效（如 \(e^{-tx}\) 在 \(t \to 0\) 时不衰减）。 ✅ 先在 \(t > 0\) 求导，再取 \(t \to 0^+\) 的极限。
