测度空间

01σ-代数与可测空间

🎯 为什么研究这个问题

19 世纪末，Riemann 积分的局限性暴露了一个根本问题：逐点收敛的可积函数列其极限可能不再可积。Lebesgue 意识到，根源不在积分定义本身，而在于我们尚未精确回答一个更基本的问题——"哪些集合可以被赋予大小"。

  • 概率论：事件空间必须在补运算和可数并下封闭，才能保证"事件的对立面"和"可数个事件中至少一个发生"仍是合法事件——这正是 σ-代数的公理；
  • Fourier 分析：Fourier 级数的逐点收敛极限不一定连续，但必须是可测函数——σ-代数保证了极限运算不会"逃出"可测集族；
  • 遍历理论：动力系统中不变集的交、并、补仍应是不变集，σ-代数为此提供了恰当的封闭性框架。

σ-代数的深层意义在于划定"可知"与"不可知"的边界：不是所有子集都能被度量（Vitali 反例），而 σ-代数恰好框定了"行为良好"的集合族——测度论、概率论和遍历理论的公共语言由此奠基。

🔍 抽象化方法
  • 封闭性公理：σ-代数 \(\mathcal{F}\) 对补运算和可数并封闭——这确保对可测集做无穷次集合运算不会"逃出"可测集族。
  • 生成 σ-代数：\(\sigma(\mathcal{C})\) 是包含集合族 \(\mathcal{C}\) 的最小 σ-代数（所有包含 \(\mathcal{C}\) 的 σ-代数之交），体现"最经济扩充"思想。
  • Borel σ-代数：\(\mathcal{B}(\mathbb{R}) = \sigma(\text{开集})\) 连接拓扑与测度——拓扑给出"开集/连续"，σ-代数给出"可测"。
📐 公式推导 / 模型建立

1. σ-代数的三条公理

设 \(X\) 为非空集，\(\mathcal{F} \subseteq \mathcal{P}(X)\) 称为 σ-代数，若满足：

(i) \(X \in \mathcal{F}\)；(ii) \(A \in \mathcal{F} \Rightarrow A^c \in \mathcal{F}\)；(iii) \(A_1, A_2, \ldots \in \mathcal{F} \Rightarrow \bigcup_{n=1}^{\infty} A_n \in \mathcal{F}\)。

由 (ii)(iii) 推出对可数交也封闭：\(\bigcap_{n=1}^\infty A_n = \bigl(\bigcup_{n=1}^\infty A_n^c\bigr)^c \in \mathcal{F}\)。

2. Borel σ-代数与等价生成

\(\mathcal{B}(\mathbb{R}) = \sigma(\text{开集})\)。等价地可由以下任一族生成：

开区间 \(\{(a,b)\}\)、闭集、半开区间 \(\{(-\infty,a] : a \in \mathbb{R}\}\)，甚至 \(\{(-\infty,a] : a \in \mathbb{Q}\}\)。

关键：\((-\infty,a] = \bigcap_{n=1}^\infty (-\infty, a+1/n)\)（可数交），而每个开集是可数个开区间之并。

3. 可测空间与测度空间

\((X, \mathcal{F})\)：可测空间。\((X, \mathcal{F}, \mu)\)：可测空间配上测度 \(\mu: \mathcal{F} \to [0,+\infty]\)。典型例子：\((\mathbb{R}, \mathcal{B}(\mathbb{R}), m)\)（Lebesgue 测度空间）。

4. σ-代数的交与生成

任意一族 σ-代数的交仍是 σ-代数。由此 \(\sigma(\mathcal{C}) = \bigcap\{\mathcal{G} : \mathcal{G}\text{ 是 σ-代数}, \mathcal{C} \subseteq \mathcal{G}\}\) 是良定义的。若 \(\mathcal{C}_1 \subseteq \sigma(\mathcal{C}_2)\) 且 \(\mathcal{C}_2 \subseteq \sigma(\mathcal{C}_1)\)，则 \(\sigma(\mathcal{C}_1) = \sigma(\mathcal{C}_2)\)。

🌐 实际应用

概率论中的事件代数：样本空间 \(\Omega\) 上的 σ-代数 \(\mathcal{F}\) 精确刻画了"哪些事件可以被赋予概率"。抛硬币模型中 \(\Omega = \{H,T\}^\infty\)，柱集生成的 σ-代数使我们可以讨论"无穷多次出现正面"这类事件的概率。

金融数学中的信息流：σ-代数 \(\mathcal{F}_t\) 表示到时刻 \(t\) 为止市场可获得的全部信息。滤波 \(\{\mathcal{F}_t\}_{t \geq 0}\)（递增的 σ-代数族）是鞅论和随机微积分的基础——交易策略必须关于 \(\mathcal{F}_t\) 可测（不能"看到未来"）。

信号处理中的可观测量：传感器系统中，σ-代数编码了"哪些事件可以被检测"。噪声信号 \(X(\omega)\) 关于某 σ-代数 \(\mathcal{G} \subseteq \mathcal{F}\) 可测意味着 \(X\) 仅依赖于 \(\mathcal{G}\) 所携带的信息——这与条件期望 \(E[X|\mathcal{G}]\) 直接相关。

✏️ 例题分析与易错点

例题：证明 \(\{(-\infty, a] : a \in \mathbb{Q}\}\) 生成 \(\mathcal{B}(\mathbb{R})\)

第一步 — 用有理数逼近实数：对任意 \(a \in \mathbb{R}\)，取有理数列 \(q_n \downarrow a\)，则 \((-\infty, a] = \bigcap_{n=1}^\infty (-\infty, q_n]\)。

第二步 — 包含关系：\(\{(-\infty, a] : a \in \mathbb{R}\} \subseteq \sigma(\{(-\infty, q] : q \in \mathbb{Q}\})\)。反之 \(\{(-\infty, q] : q \in \mathbb{Q}\} \subseteq \{(-\infty, a] : a \in \mathbb{R}\}\)。

第三步 — 结论：两族互含于对方生成的 σ-代数中，故 \(\sigma(\{(-\infty, q] : q \in \mathbb{Q}\}) = \sigma(\{(-\infty, a] : a \in \mathbb{R}\}) = \mathcal{B}(\mathbb{R})\)。 ✔

常见错误

  • ❌ 认为 σ-代数只需对有限并封闭。
    ✅ 反例：\(\{\emptyset, \mathbb{R}\} \cup \{\text{有限集}\}\) 对有限并封闭但 \(\bigcup_{n=1}^\infty \{n\} = \mathbb{N}\) 不在其中。σ-代数要求可数并封闭。
  • ❌ 认为两个 σ-代数的并 \(\mathcal{F}_1 \cup \mathcal{F}_2\) 仍是 σ-代数。
    ✅ 并一般不封闭。例：\(X = \{1,2,3\}\)，\(\mathcal{F}_1 = \{\emptyset, \{1\}, \{2,3\}, X\}\)，\(\mathcal{F}_2 = \{\emptyset, \{2\}, \{1,3\}, X\}\)，则 \(\{1\} \cup \{2\} = \{1,2\} \notin \mathcal{F}_1 \cup \mathcal{F}_2\)。
  • ❌ 混淆 \(\sigma(\mathcal{C})\)（生成的 σ-代数）与 \(\mathcal{C}\) 本身。
    ✅ \(\sigma(\mathcal{C})\) 一般远大于 \(\mathcal{C}\)。例：\(\mathcal{C} = \{(a,b) : a < b\}\) 只有开区间，而 \(\sigma(\mathcal{C}) = \mathcal{B}(\mathbb{R})\) 包含所有 Borel 集（不可数多个）。

02测度的定义与基本性质

🎯 为什么研究这个问题

有了可测空间 \((X, \mathcal{F})\) 之后，下一步是为可测集赋予量化的"大小"。直觉上这应该推广长度、面积和概率，但仅凭有限可加性不足以保证极限运算的合法性——从有限到可数的跨越需要一条更强的公理。

  • 物理学：质量分布、电荷密度的数学表达都是测度——对任意可测区域给出一个非负数值，且"部分之和等于整体"；
  • 概率论：概率测度 \(P\) 满足 \(P(\Omega)=1\) 和 σ-可加性，保证了独立事件的概率可以逐项相加；
  • 信号处理：谱测度将频率域分解为不相交的频带，每个频带的能量之和等于总能量——σ-可加性是频谱分析的数学基石。

σ-可加性是区分"真测度"与"病态集函数"的分水岭：它使测度与可数极限运算（\(\lim\)、\(\sup\)、\(\inf\)）完美兼容，从而保证了自下/自上连续性——这正是三大收敛定理（MCT、Fatou、DCT）的技术支撑。

🔍 抽象化方法
  • 集函数的公理化：测度是从 σ-代数到 \([0,+\infty]\) 的 σ-可加集函数，\(\mu(\emptyset)=0\)。所有基本性质（单调性、次可加性、连续性）都由这两条公理推出。
  • 从有限到可数的跨越：σ-可加性是有限可加性到无穷的核心跨越——它使测度与拓扑中的可数运算（lim, sup, inf）完美兼容。
📐 公式推导 / 模型建立

1. 测度的定义

\(\mu: \mathcal{F} \to [0,+\infty]\) 满足：(M1) \(\mu(\emptyset) = 0\)；(M2) σ-可加性：若 \(\{A_n\}\) 两两不交，则

\[\mu\Bigl(\bigcup_{n=1}^\infty A_n\Bigr) = \sum_{n=1}^\infty \mu(A_n)\]

2. 基本性质推导

单调性：\(A \subseteq B\)，写 \(B = A \cup (B \setminus A)\)，由可加性 \(\mu(B) = \mu(A) + \mu(B \setminus A) \geq \mu(A)\)。

次可加性：\(\mu(\bigcup A_n) \leq \sum \mu(A_n)\)（不要求不交），用不交化 + σ-可加性 + 单调性证明。

3. 测度的连续性

自下连续（无条件）：\(A_n \uparrow A\)。令 \(B_1 = A_1\)，\(B_n = A_n \setminus A_{n-1}\)，则 \(\{B_n\}\) 两两不交且 \(\bigcup B_n = A\)：

\[\mu(A) = \sum_{n=1}^\infty \mu(B_n) = \lim_{N \to \infty} \sum_{n=1}^N \mu(B_n) = \lim_{N \to \infty} \mu(A_N)\]

自上连续（需有限性）：\(A_n \downarrow A\)，\(\mu(A_1) < \infty\) 时 \(\mu(A_n) \to \mu(A)\)。反例：计数测度，\(A_n = \{n, n+1, \ldots\}\)，\(\bigcap A_n = \emptyset\) 但 \(\mu(A_n) = \infty\)。

4. 重要特例

计数测度 \(\#(A) = |A|\)；Dirac 测度 \(\delta_x(A) = \mathbf{1}_A(x)\)；概率测度 \(P(X) = 1\)；σ-有限测度 \(X = \bigcup E_n\)，\(\mu(E_n) < \infty\)。

🌐 实际应用

求和与积分的统一：计数测度 \(\#\) 使求和统一到 Lebesgue 积分框架中——\(\int f\, d\# = \sum f(n)\)。这使得 \(L^p\) 空间理论同时涵盖 \(\ell^p\)（序列空间）和经典 \(L^p\)（函数空间）。

信号处理中的 Dirac 测度：δ-函数 \(\delta_0\) 在分布意义下是集中在原点的测度，模拟理想化脉冲。\(\int f\, d\delta_0 = f(0)\) 是采样定理（Shannon）的数学基础。在离散信号处理中，Dirac 梳（\(\sum_n \delta_n\)）连接连续与离散 Fourier 变换。

概率论中的随机变量分布：随机变量 \(X\) 的分布 \(\mu_X = P \circ X^{-1}\) 是像测度。连续型分布有密度 \(f\)（关于 Lebesgue 测度绝对连续），离散型用 Dirac 测度的加权和表示。测度的 σ-可加性保证了概率的规范性。

✏️ 例题分析与易错点

例题：递减集列的测度极限

题目：\(\mu\) 是 \((\mathbb{R}, \mathcal{B}(\mathbb{R}))\) 上的测度，\(A_n = (0, 1/n)\)，\(\mu((0,1)) = 1\)。求 \(\lim \mu(A_n)\)。

第一步 — 验证条件：\(A_n \downarrow \emptyset\)，且 \(\mu(A_1) = \mu((0,1)) = 1 < \infty\)，满足自上连续性条件。

第二步 — 应用连续性：\(\mu(A_n) \to \mu(\bigcap A_n) = \mu(\emptyset) = 0\)。 ✔

常见错误

  • ❌ 对递减集列直接用自上连续性而不检查 \(\mu(A_1) < \infty\)。
    ✅ 反例：计数测度，\(A_n = \{n, n+1,\ldots\}\)，\(\mu(A_n) = \infty\) 但 \(\mu(\bigcap A_n) = \mu(\emptyset) = 0\)。有限性假设不可省略。
  • ❌ 混淆 σ-可加性与有限可加性，认为它们等价。
    ✅ 有限可加性 + 自下连续性 ⟺ σ-可加性。单独的有限可加性严格弱于 σ-可加（存在仅有限可加的"测度"不满足连续性）。
  • ❌ 认为 \(\mu(A \cup B) = \mu(A) + \mu(B)\) 不需要不交条件。
    ✅ 正确公式：\(\mu(A \cup B) = \mu(A) + \mu(B) - \mu(A \cap B)\)（需 \(\mu(A \cap B) < \infty\)）。σ-可加性的前提是"两两不交"。

03可测函数与简单函数逼近

🎯 为什么研究这个问题

要在测度空间上定义积分，首先需要确定哪些函数"可以被积"——函数必须与底层的 σ-代数结构相容。Riemann 可积函数在逐点极限下不封闭，而 Lebesgue 理论要求一个在极限运算下稳定的函数族。

  • 量子力学：可观测量对应自伴算子，其谱分解依赖于对 Borel 可测函数的积分——不可测函数无法参与谱计算；
  • 机器学习：损失函数的期望 \(\mathbb{E}[L(h(X), Y)]\) 要求 \(L \circ h\) 可测，否则积分无意义——可测性是经验风险最小化的隐含前提；
  • 数值分析：Lebesgue 积分从简单函数出发，通过单调逼近过渡到一般函数——与数值积分"先用阶梯函数逼近"的思路完全一致。

可测函数类的真正威力在于对可数极限运算的封闭性：\(\sup\)、\(\inf\)、\(\limsup\)、\(\liminf\) 都不会把可测函数变成不可测的——这是 Lebesgue 理论相对于 Riemann 框架的核心结构优势。

🔍 抽象化方法
  • 原像视角：可测函数将"目标空间的信息结构"拉回到"源空间的 σ-代数"。验证可测性只需检查 \(\{f > a\} \in \mathcal{F}\) 对所有 \(a\)。
  • 从有限到无穷：简单函数 = 有限个指示函数的线性组合。逼近定理 = "任何非负可测函数是简单函数的单调递增极限"——体现分析学"有限逼近无穷"的核心思想。
📐 公式推导 / 模型建立

1. 可测函数的定义与等价条件

\(f: (X, \mathcal{F}) \to (\mathbb{R}, \mathcal{B}(\mathbb{R}))\) 可测 ⟺ \(\forall B \in \mathcal{B}(\mathbb{R})\)，\(f^{-1}(B) \in \mathcal{F}\)。等价条件：只需验证下列之一——\(\{f > a\}\)、\(\{f \geq a\}\)、\(\{f < a\}\)、\(\{f \leq a\}\) 对所有 \(a \in \mathbb{R}\) 属于 \(\mathcal{F}\)。

2. 运算封闭性

\(f, g\) 可测 ⟹ \(f+g\)、\(fg\)、\(f/g\)（\(g \neq 0\)）可测。关键技巧：

\[\{f + g > a\} = \bigcup_{r \in \mathbb{Q}} (\{f > r\} \cap \{g > a - r\})\]

可数并保持可测性。类似地 \(\sup_n f_n\)、\(\inf_n f_n\)、\(\limsup f_n\)、\(\liminf f_n\) 均可测——可测函数类在可数极限运算下封闭。

3. 简单函数与逼近定理

简单函数：\(\varphi = \sum_{i=1}^n a_i \mathbf{1}_{A_i}\)，\(A_i \in \mathcal{F}\)。

逼近定理：若 \(f \geq 0\) 可测，存在简单函数列 \(0 \leq \varphi_1 \leq \varphi_2 \leq \cdots\) 使 \(\varphi_n(x) \uparrow f(x)\) 对所有 \(x\)。

标准构造：\(\varphi_n(x) = k/2^n\) 当 \(k/2^n \leq f(x) < (k+1)/2^n\)，\(k = 0,\ldots,n \cdot 2^n - 1\)；\(\varphi_n(x) = n\) 当 \(f(x) \geq n\)。

4. 一般可测函数的分解

任意可测函数 \(f = f^+ - f^-\)，其中 \(f^+ = \max(f,0)\)，\(f^- = \max(-f,0)\) 均为非负可测函数，可分别用简单函数逼近。

🌐 实际应用

数字图像的数学模型：灰度图像 \(f: [0,1]^2 \to [0,255]\) 是可测函数。阈值分割 \(\{f > 128\}\) 是原像操作——可测性保证分割区域有良定义的面积（Lebesgue 测度），可用于定量分析肿瘤面积等医学指标。

量子力学中的可观测量：可观测量（自伴算子 \(A\)）的谱分解 \(A = \int \lambda\, dE_\lambda\) 依赖于谱测度 \(E\)。对 Borel 集 \(B\) 的可测性确保 \(E(B)\) 是良定义的投影算子——"测量结果落在 \(B\) 中"的事件具有概率 \(\|E(B)\psi\|^2\)。

机器学习中的损失函数：损失函数 \(\ell(y, \hat{y})\) 关于参数可测是计算期望风险 \(R(\theta) = \int \ell\, dP\) 的前提。Lebesgue 积分框架允许使用非连续的损失（如 0-1 损失），而简单函数逼近对应着经验风险的有限样本近似。

✏️ 例题分析与易错点

例题：构造 \(f(x) = x^2\) 在 \([0,2]\) 上的逼近简单函数 \(\varphi_1\)

第一步 — 参数确定：\(n = 1\) 时，值域划分步长 \(1/2^1 = 1/2\)，截断在 \(n = 1\)。

第二步 — 划分区间：\([0, 1/2)\) 对应 \(f(x) \in [0, 1/2)\)，即 \(x \in [0, 1/\sqrt{2})\)；\([1/2, 1)\) 对应 \(x \in [1/\sqrt{2}, 1)\)；\(f \geq 1\) 对应 \(x \in [1, 2]\)。

第三步 — 写出结果：\(\varphi_1 = 0 \cdot \mathbf{1}_{[0, 1/\sqrt{2})} + \tfrac{1}{2} \cdot \mathbf{1}_{[1/\sqrt{2}, 1)} + 1 \cdot \mathbf{1}_{[1, 2]}\)。 ✔

常见错误

  • ❌ 混淆"可测函数"与"连续函数"。
    ✅ Dirichlet 函数 \(\mathbf{1}_{\mathbb{Q}}\) 处处不连续但 Borel 可测（\(\mathbb{Q} \in \mathcal{B}(\mathbb{R})\)）。可测性远弱于连续性。
  • ❌ 认为可测函数的逐点极限不一定可测。
    ✅ 可测函数的逐点极限一定可测：\(\lim f_n = \liminf f_n = \sup_n \inf_{k \geq n} f_k\)，可数个 sup/inf 保持可测性。这是 Lebesgue 理论优于 Riemann 理论的核心。
  • ❌ 简单函数逼近中忘记截断（认为 \(\varphi_n\) 在 \(f\) 很大时也精确逼近）。
    ✅ 标准构造在 \(f(x) \geq n\) 时令 \(\varphi_n = n\)（截断）。逼近精度仅在 \(f < n\) 的区域为 \(1/2^n\)。需要 \(n \to \infty\) 才能处理无界函数。
⚡ 快速检测
若 \(f_n\) 均为可测函数，则 \(g(x) = \limsup_{n \to \infty} f_n(x)\) 是否可测？

04测度空间的完备化与零测集

🎯 为什么研究这个问题

实分析中大量命题以"几乎处处"成立的形式出现，但在 Borel σ-代数中，零测集的子集未必是 Borel 集——这意味着"a.e."概念可能引发可测性问题。为了让理论自洽，我们需要将所有零测子集纳入可测集族。

  • 概率论：完备概率空间保证了条件期望和随机过程正则版本的良定义性——独立性论证中经常需要处理零概率事件的子集；
  • PDE 理论：Sobolev 空间中函数的"代表元"只确定到零测集上的修改，完备化使得等价类运算不会产生不可测的残留；
  • 遍历理论：不变集的零测修改在完备测度下仍属于 σ-代数，保证了遍历分解定理的技术细节不出错。

完备化在数学中扮演"补全缺失"的角色，类似于度量空间的 Cauchy 完备化。Lebesgue σ-代数正是 Borel σ-代数关于 Lebesgue 测度的完备化——它严格包含后者，揭示了 Cantor 集等零测但不可数集合中隐藏的测度论微妙性。

🔍 抽象化方法
  • 完备化 = 补全缺失：类比度量空间的完备化（补充 Cauchy 列的极限），测度空间的完备化补充"零测集的所有子集"，使可测集族更自洽。
  • "可忽略"的严格化："几乎处处(a.e.)"= 性质在零测集之外成立。\(L^p\) 空间中的等价类 \([f] = \{g : f = g\text{ a.e.}\}\) 依赖于此概念。
📐 公式推导 / 模型建立

1. 零测集与完备测度

零测集：\(N \in \mathcal{F}\)，\(\mu(N) = 0\)。完备测度空间：若 \(\mu(N) = 0\) 且 \(E \subseteq N\)，则 \(E \in \mathcal{F}\)（且 \(\mu(E) = 0\)）。

2. 完备化构造

定义扩充的 σ-代数：

\[\overline{\mathcal{F}} = \{A \cup E : A \in \mathcal{F},\; E \subseteq N,\; N \in \mathcal{F},\; \mu(N) = 0\}\]

令 \(\overline{\mu}(A \cup E) = \mu(A)\)。可验证：\(\overline{\mathcal{F}}\) 是 σ-代数，\(\overline{\mu}\) 在其上是完备测度，且 \(\overline{\mu}|_{\mathcal{F}} = \mu\)。

3. Lebesgue σ-代数与 Borel σ-代数

\(\mathcal{L} = \overline{\mathcal{B}(\mathbb{R})}\)（Borel σ-代数关于 Lebesgue 测度的完备化）。\(\mathcal{B}(\mathbb{R}) \subsetneq \mathcal{L}\)：Cantor 集 \(C\) 的子集都 Lebesgue 可测（零测子集），但其中存在非 Borel 集（基数论证：\(|\mathcal{P}(C)| = 2^{\mathfrak{c}} > \mathfrak{c} = |\mathcal{B}(\mathbb{R})|\)）。

4. "几乎处处"的形式化

性质 \(P(x)\) 几乎处处成立 ⟺ \(\mu(\{x : \lnot P(x)\}) = 0\)。关键事实：零测集的可数并仍为零测（次可加性 \(\mu(\bigcup N_n) \leq \sum \mu(N_n) = 0\)）——因此可以同时"忽略"可数多个零测例外。

🌐 实际应用

\(L^p\) 空间的等价类：在 \(L^p\) 空间中，\(f\) 和 \(g\) 若 \(f = g\) a.e.，则视为同一元素。完备化保证了"修改零测集上的值不影响积分"这一直觉的严格性——\(\|f - g\|_p = 0 \iff f = g\) a.e.，从而 \(L^p\) 成为真正的赋范空间（而非半赋范）。

概率论中的正则条件概率：条件概率 \(P(A|X)\) 在完备概率空间中有正则版本（regular conditional probability），对零概率事件的子集也有良定义的概率赋值。Kolmogorov 配套定理的证明也依赖于完备化。

实变函数论中的 Lusin 定理：Lebesgue 可测函数在去掉任意小测度集后连续（Lusin 定理）。这里"Lebesgue 可测"而非"Borel 可测"是关键——完备化扩大了定理适用的函数类。Lusin 定理是调和分析中的基本工具。

✏️ 例题分析与易错点

例题：证明存在 Lebesgue 可测但非 Borel 的集合

第一步 — Cantor 集的性质：Cantor 集 \(C \subseteq [0,1]\) 是零测集（\(m(C) = 0\)）且不可数（\(|C| = 2^{\aleph_0} = \mathfrak{c}\)）。

第二步 — 基数论证：\(|\mathcal{B}(\mathbb{R})| = \mathfrak{c}\)（可数生成元生成的 σ-代数），而 \(|\mathcal{P}(C)| = 2^{\mathfrak{c}} > \mathfrak{c}\)。

第三步 — 结论：\(C\) 的子集中必存在非 Borel 集 \(E\)。但由完备化，\(E \subseteq C\)，\(m(C) = 0\) ⟹ \(E \in \mathcal{L}\)，\(m(E) = 0\)。✔

常见错误

  • ❌ 认为"Lebesgue 可测集 = Borel 集"。
    ✅ \(\mathcal{B}(\mathbb{R}) \subsetneq \mathcal{L}\)。Lebesgue σ-代数严格包含 Borel σ-代数——差集恰好是零测集的非 Borel 子集构成的。
  • ❌ 认为"零测集一定是可数集"。
    ✅ Cantor 集是不可数零测集（\(|C| = \mathfrak{c}\) 但 \(m(C) = 0\)）。"零测"与"可数"是不同概念：可数集必零测，但零测集不必可数。
  • ❌ 在非完备空间中对零测子集做操作而不检查可测性。
    ✅ 在 Borel σ-代数 \(\mathcal{B}(\mathbb{R})\) 中，零测集的子集不一定 Borel 可测。只有在完备化后的 \(\mathcal{L}\) 中才能安全地对零测子集赋测度。使用"a.e."性质时需明确工作在完备空间中。
