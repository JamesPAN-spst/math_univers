01σ-代数与可测空间

🎯 为什么研究这个问题

19世纪末，数学家们发现 Riemann 积分无法处理许多极限交换问题： 例如逐点收敛的函数序列，其极限函数可能不 Riemann 可积。Lebesgue 意识到，问题根源在于"哪些集合可以被赋予大小"。为此需要一套严格 的框架来指定"可测集"——这就是 σ-代数的概念。σ-代数为测度论、 概率论和现代分析提供了统一的集合论基础。

🔍 抽象化方法

核心抽象：将"可以度量大小的集合"公理化。σ-代数 \(\mathcal{F}\) 是对补运算和可数并运算封闭的集合族。这种封闭性确保我们可以对 可测集进行无穷次集合运算而不离开可测集的范畴。生成 σ-代数 \(\sigma(\mathcal{C})\) 则是"最经济地"将一族集合扩充为 σ-代数。

📐 公式推导 / 模型建立

定义：设 \(X\) 为非空集，\(\mathcal{F} \subseteq \mathcal{P}(X)\) 称为 σ-代数，若满足： (i) \(X \in \mathcal{F}\)； (ii) \(A \in \mathcal{F} \Rightarrow A^c \in \mathcal{F}\)； (iii) \(A_1, A_2, \ldots \in \mathcal{F} \Rightarrow \bigcup_{n=1}^{\infty} A_n \in \mathcal{F}\)。 由 (ii)(iii) 可导出对可数交也封闭： \[\bigcap_{n=1}^{\infty} A_n = \left(\bigcup_{n=1}^{\infty} A_n^c\right)^c \in \mathcal{F}\] Borel σ-代数：\(\mathcal{B}(\mathbb{R}) = \sigma(\{\text{开集}\})\)。 等价地，\(\mathcal{B}(\mathbb{R})\) 可由半开区间 \(\{(-\infty, a] : a \in \mathbb{R}\}\) 生成，因为每个开集是可数个开区间之并。 可测空间 \((X, \mathcal{F})\)：集合配上 σ-代数。 测度空间 \((X, \mathcal{F}, \mu)\)：可测空间配上测度 \(\mu\)。

🌐 实际应用

概率论中，样本空间 \(\Omega\) 上的事件 σ-代数 \(\mathcal{F}\) 精确刻画了"哪些事件可以被赋予概率"；金融数学中，σ-代数 \(\mathcal{F}_t\) 表示到时刻 \(t\) 为止可获得的信息。

✏️ 例题分析与易错点

例题：证明 \(\{(-\infty, a] : a \in \mathbb{Q}\}\) 生成 \(\mathcal{B}(\mathbb{R})\)。 解答：对任意 \(a \in \mathbb{R}\)，取有理数列 \(q_n \downarrow a\)，则 \((-\infty, a] = \bigcap_{n=1}^{\infty} (-\infty, q_n]\)， 属于 \(\sigma(\{(-\infty, q] : q \in \mathbb{Q}\})\)。 因此 \(\{(-\infty, a] : a \in \mathbb{R}\}\) 中所有集合都在此 σ-代数中，故两个生成的 σ-代数相同。 易错点：误以为 σ-代数只需对有限并封闭。 反例：有限并封闭的集合族 \(\{\emptyset, \mathbb{R}\} \cup \{\text{有限集}\}\) 不对可数并封闭（如 \(\bigcup \{n\} = \mathbb{N}\) 不一定在其中）。σ-代数要求的是可数并封闭。

02测度的定义与基本性质

🎯 为什么研究这个问题

有了可测空间后，下一步是给可测集赋予"大小"。直觉上，测度推广了 长度、面积、体积的概念。关键难点在于：如何确保对可数多个不相交 集合赋大小时满足"可加性"？Lebesgue 和 Carathéodory 的工作表明， σ-可加性（而非仅有限可加性）是保证极限运算合法的根本条件。

🔍 抽象化方法

测度是从集合族到 \([0, +\infty]\) 的σ-可加函数。σ-可加性是有限 可加性到无穷的推广，它确保测度与可数极限运算相容——这正是 Lebesgue 积分优于 Riemann 积分的根源。

📐 公式推导 / 模型建立

定义：测度 \(\mu: \mathcal{F} \to [0, +\infty]\) 满足： (M1) \(\mu(\emptyset) = 0\)； (M2) σ-可加性：若 \(\{A_n\}\) 两两不交，则 \[\mu\left(\bigcup_{n=1}^{\infty} A_n\right) = \sum_{n=1}^{\infty} \mu(A_n)\] 基本性质（由 σ-可加性推出）： - 单调性：\(A \subseteq B \Rightarrow \mu(A) \leq \mu(B)\) - 次可加性：\(\mu(\bigcup A_n) \leq \sum \mu(A_n)\) - 连续性（自下）：\(A_n \uparrow A \Rightarrow \mu(A_n) \to \mu(A)\) - 连续性（自上）：\(A_n \downarrow A\) 且 \(\mu(A_1) < \infty \Rightarrow \mu(A_n) \to \mu(A)\) 自下连续性证明：令 \(B_1 = A_1\), \(B_n = A_n \setminus A_{n-1}\)， 则 \(\{B_n\}\) 两两不交且 \(\bigcup B_n = A\)， \[\mu(A) = \sum_{n=1}^{\infty} \mu(B_n) = \lim_{N \to \infty} \sum_{n=1}^{N} \mu(B_n) = \lim_{N \to \infty} \mu(A_N)\]

🌐 实际应用

计数测度 \(\mu(A) = |A|\) 用于将求和统一到积分框架中； Dirac 测度 \(\delta_x(A) = \mathbf{1}_A(x)\) 是集中在一点的 概率测度，在信号处理中模拟脉冲。

✏️ 例题分析与易错点

例题：设 \(\mu\) 是 \((\mathbb{R}, \mathcal{B}(\mathbb{R}))\) 上的 测度，\(A_n = (0, 1/n)\)。若 \(\mu((0,1)) = 1\)，求 \(\lim_{n \to \infty} \mu(A_n)\)。 解答：\(A_n \downarrow \emptyset\)，且 \(\mu(A_1) = \mu((0,1)) = 1 < \infty\)， 由测度自上连续性，\(\mu(A_n) \to \mu(\emptyset) = 0\)。 易错点：忘记自上连续性需要 \(\mu(A_1) < \infty\) 的条件。 反例：\(\mu = \) 计数测度，\(A_n = \{n, n+1, \ldots\}\)， \(\mu(A_n) = \infty\) 对所有 \(n\)，但 \(\mu(\bigcap A_n) = \mu(\emptyset) = 0\)。

03可测函数与简单函数逼近

🎯 为什么研究这个问题

测度空间建立后，要在上面做"积分"，首先要确定哪些函数可以被积分。 可测函数是"与 σ-代数相容"的函数——它的原像将可测集映回可测集。 更深层的问题是：如何用最简单的函数（简单函数）来逐步逼近任意可测 函数？这个逼近定理是 Lebesgue 积分构造的关键一步。

🔍 抽象化方法

可测函数将一个可测空间上的"信息结构"拉回到另一个可测空间。 简单函数逼近定理体现了"从有限到无穷"的分析学核心思想： 先对有限情形（简单函数）定义积分，再通过单调极限过渡到一般。

📐 公式推导 / 模型建立

可测函数定义：\(f: (X, \mathcal{F}) \to (\mathbb{R}, \mathcal{B}(\mathbb{R}))\) 可测，若 \(\forall B \in \mathcal{B}(\mathbb{R})\)， \(f^{-1}(B) \in \mathcal{F}\)。等价地，只需验证 \(\{x : f(x) > a\} \in \mathcal{F}\) 对所有 \(a \in \mathbb{R}\)。 运算封闭性：若 \(f, g\) 可测，则 \(f + g\), \(fg\), \(\sup_n f_n\), \(\limsup_n f_n\) 均可测。证明关键： \[\{f + g > a\} = \bigcup_{r \in \mathbb{Q}} (\{f > r\} \cap \{g > a - r\})\] 简单函数：\(\varphi = \sum_{i=1}^{n} a_i \mathbf{1}_{A_i}\)， 其中 \(A_i \in \mathcal{F}\)，\(a_i \in \mathbb{R}\)。 逼近定理：若 \(f \geq 0\) 可测，存在简单函数序列 \(0 \leq \varphi_1 \leq \varphi_2 \leq \cdots\) 使 \(\varphi_n(x) \uparrow f(x)\) \(\forall x\)。 构造：令 \(\varphi_n(x) = \begin{cases} k/2^n & \text{if } k/2^n \leq f(x) < (k+1)/2^n,\; k = 0,1,\ldots,n2^n - 1 \\ n & \text{if } f(x) \geq n \end{cases}\)

✏️ 例题分析与易错点

例题：设 \(f(x) = x^2\) 在 \(([0,2], \mathcal{B}([0,2]))\) 上， 写出 \(\varphi_1\)（即 \(n=1\) 时的逼近简单函数）。 解答：\(n=1\) 时，划分值域为 \([0,1/2), [1/2,1), [1,3/2), [3/2,2), [2,\infty)\)。 但 \(n=1\) 意味着截断在 1，故 \(\varphi_1(x) = 0 \cdot \mathbf{1}_{[0, 1/\sqrt{2})} + \frac{1}{2} \cdot \mathbf{1}_{[1/\sqrt{2}, 1)} + 1 \cdot \mathbf{1}_{[1, 2]}\)。 易错点：混淆"可测函数"与"连续函数"。 Dirichlet 函数 \(\mathbf{1}_{\mathbb{Q}}\) 处处不连续但 Borel 可测（因为 \(\mathbb{Q} \in \mathcal{B}(\mathbb{R})\)）。

⚡ 快速检测
若 \(f_n\) 均为可测函数，则 \(g(x) = \limsup_{n \to \infty} f_n(x)\) 是否可测？

04测度空间的完备化与零测集

🎯 为什么研究这个问题

在实际分析中，我们经常遇到"几乎处处"成立的性质——即一个性质 在零测集之外成立。但 Borel σ-代数中，零测集的子集不一定是 Borel 集！这意味着"几乎处处"的概念在 Borel 框架下不够 自洽。完备化过程解决这个问题：将所有零测集的子集纳入可测 集族，使"几乎处处"操作不会导致可测性问题。

🔍 抽象化方法

完备化是将测度空间"补全"的过程，使零测集的任意子集都可测 （且测度为零）。这与度量空间的完备化精神一致——补充"缺失" 的元素使结构更自洽。

📐 公式推导 / 模型建立

零测集：\(N \in \mathcal{F}\)，\(\mu(N) = 0\)。 完备测度空间：若 \(\mu(N) = 0\) 且 \(E \subseteq N\)，则 \(E \in \mathcal{F}\)（且 \(\mu(E) = 0\)）。 完备化构造：定义 \[\overline{\mathcal{F}} = \{A \cup E : A \in \mathcal{F},\; E \subseteq N \text{ for some } N \in \mathcal{F} \text{ with } \mu(N) = 0\}\] 并令 \(\overline{\mu}(A \cup E) = \mu(A)\)。 可验证 \(\overline{\mathcal{F}}\) 是 σ-代数，\(\overline{\mu}\) 是完备测度。 Lebesgue σ-代数 \(\mathcal{L}\) 严格大于 Borel σ-代数 \(\mathcal{B}(\mathbb{R})\)。存在 Lebesgue 可测但非 Borel 的集合 （如 Cantor 集的子集）。 "几乎处处"(a.e.)：性质 \(P(x)\) 几乎处处成立，若 \(\mu(\{x : P(x) \text{ 不成立}\}) = 0\)。

🌐 实际应用

概率论中，随机变量的"几乎必然(a.s.)"性质依赖于完备概率 空间；在 \(L^p\) 空间中，函数等价类的定义依赖于将 a.e. 相等 的函数视为同一元素。

✏️ 例题分析与易错点

例题：Cantor 集 \(C\) 是零测集，\(|C| = 2^{\aleph_0}\)（不可数）。 证明存在 \(C\) 的子集不是 Borel 集但是 Lebesgue 可测的。 解答：\(|\mathcal{B}(\mathbb{R})| = 2^{\aleph_0}\)（可数生成元 生成的 σ-代数基数为连续统），而 \(|\mathcal{P}(C)| = 2^{2^{\aleph_0}} > 2^{\aleph_0}\)。因此 \(C\) 的子集中存在非 Borel 集。 但由于 \(C\) 零测且 Lebesgue 测度完备，\(C\) 的所有子集都 Lebesgue 可测（测度为 0）。 易错点：误以为"Lebesgue 可测集 = Borel 集"。实际上 \(\mathcal{B}(\mathbb{R}) \subsetneq \mathcal{L}\)。
