Radon-Nikodym 与符号测度

01符号测度与 Hahn-Jordan 分解

🎯 为什么研究这个问题

物理中电荷可正可负，概率论中两组分布之差、金融中的损益函数——这些对象自然地需要"可以取负值的测度"来刻画。从非负测度推广到符号测度是否会导致理论失控？Hahn-Jordan 分解给出了令人安心的回答。

  • 统计学：似然比检验中两个概率测度之差是符号测度，其 Jordan 分解直接给出检验的最优拒绝域（Neyman-Pearson 引理的测度论表述）；
  • 金融数学：对冲组合的损益可正可负，其分布的精确刻画需要符号测度——全变差给出了"最坏情况下的总风险"；
  • 泛函分析：\(C_0(X)\) 上的有界线性泛函由 Riesz-Markov 定理与正则符号测度一一对应——符号测度空间是 Banach 空间 \(M(X)\) 的载体。

Jordan 分解 \(\mu = \mu^+ - \mu^-\) 的核心启示是：任何符号测度都可以被"拆"回两个非负测度之差，且拆法唯一。这与实数 \(x = x^+ - x^-\) 的正负分解完全类比——Hahn 分解先将空间一分为二（正集/负集），再由此导出唯一的测度分解，使非负测度的全部理论可以"移植"到符号测度上。

🔍 抽象化方法
  • 与实数正负分解的类比——实数 \(x = x^+ - x^-\)（\(x^+ = \max(x,0)\)，\(x^- = \max(-x,0)\)）；Jordan 分解 \(\mu = \mu^+ - \mu^-\) 是这一思想在集函数空间中的推广
  • 空间的二分法——Hahn 分解将全空间 \(X = P \sqcup N\) 拆为正集和负集，使得符号测度在各半空间上退化为普通非负测度。这是一种"降维"策略
  • 范数与 Banach 空间——全变差 \(\|\mu\| = |\mu|(X)\) 赋予有限符号测度空间 \(M(X, \mathcal{F})\) 一个完备范数，使之成为 Banach 空间。\(L^1(\nu)\) 通过 \(f \mapsto f\nu\) 等距嵌入 \(M\)
📐 公式推导 / 模型建立

符号测度的定义

集函数 \(\mu: \mathcal{F} \to (-\infty, +\infty]\)（或 \([-\infty, +\infty)\)）满足：

  • \(\mu(\emptyset) = 0\)
  • σ-可加性：\(\mu\bigl(\bigsqcup_{n=1}^\infty E_n\bigr) = \sum_{n=1}^\infty \mu(E_n)\)
  • \(\mu\) 不同时取 \(+\infty\) 和 \(-\infty\)（避免 \(\infty - \infty\)）

Hahn 分解定理

存在 \(P, N \in \mathcal{F}\)，\(P \sqcup N = X\)，使得 \(\mu(E) \geq 0\) 对所有 \(E \subseteq P\)，\(\mu(E) \leq 0\) 对所有 \(E \subseteq N\)。

证明策略：取 \(\sup\{\mu(A) : A \in \mathcal{F}\}\)，构造递增序列 \(A_n\) 使 \(\mu(A_n) \to \sup\)，令 \(P = \bigcup A_n\)。关键引理：正集中的每个可测子集仍为正集（需仔细处理可数可加性）。

Jordan 分解

由 Hahn 分解定义： \[\mu^+(E) = \mu(E \cap P), \quad \mu^-(E) = -\mu(E \cap N)\] 则 \(\mu^+, \mu^- \geq 0\)，\(\mu = \mu^+ - \mu^-\)，且 \(\mu^+ \perp \mu^-\)（集中在不相交的集上）。

虽然 Hahn 分解 \((P, N)\) 不唯一（可差一个 μ-零测集），但 Jordan 分解 \((\mu^+, \mu^-)\) 是唯一的。

全变差测度

\[|\mu| = \mu^+ + \mu^-, \quad |\mu|(E) = \sup\biggl\{\sum_{i=1}^n |\mu(E_i)| : E = \bigsqcup_{i=1}^n E_i\biggr\}.\] 全变差范数 \(\|\mu\| = |\mu|(X)\) 使有限符号测度空间成为 Banach 空间。

🌐 实际应用

统计学：假设检验与似然比——在 Neyman-Pearson 框架中，检验统计量是两个概率测度之差 \(P_1 - P_0\)（符号测度）的 Hahn 分解。正集 \(P\) 对应拒绝域——在 \(P\) 上 \(P_1\) 比 \(P_0\) "更大"，故拒绝原假设。全变差 \(\|P_1 - P_0\| = 2\sup_A |P_1(A) - P_0(A)|\) 度量两个分布的最大可区分性。

物理学：电荷与电势——空间中的电荷分布可用符号测度 \(\rho\) 表示（正电荷对应 \(\rho^+\)，负电荷对应 \(\rho^-\)）。全变差 \(|\rho|(X)\) 是总电荷量。Poisson 方程 \(-\Delta \phi = \rho\) 中的右端项即符号测度；Hahn 分解给出正负电荷区域的划分。

信号处理：带通滤波器——频域中理想带通滤波器的传递函数 \(H(\omega)\) 可正可负（如 Hilbert 变换）。其对应的时域脉冲响应作为 tempered distribution 可通过符号测度逼近。全变差范数控制滤波器的稳定性。

✏️ 例题分析与易错点

例题：显式 Hahn-Jordan 分解

设 \(\mu(E) = \int_E f\,d\lambda\)，\(f(x) = x\) 在 \([-1, 1]\) 上。求 \(\mu\) 的 Hahn 分解和 Jordan 分解。

Step 1　确定正集：\(f(x) = x \geq 0\) 当 \(x \in [0, 1]\)，故取 \(P = [0, 1]\)。

Step 2　确定负集：\(f(x) = x < 0\) 当 \(x \in [-1, 0)\)，故 \(N = [-1, 0)\)。

Step 3　Jordan 分解：\(\mu^+(E) = \int_{E \cap [0,1]} x\,dx\)，\(\mu^-(E) = \int_{E \cap [-1,0)} (-x)\,dx\)。

Step 4　全变差：\(|\mu|([-1,1]) = \int_0^1 x\,dx + \int_{-1}^0 (-x)\,dx = 1/2 + 1/2 = 1\)。

常见错误

  • ❌ 认为 Hahn 分解唯一 → ✅ Hahn 分解不唯一：\(P, N\) 可差一个 μ-零集（如 \(\{0\}\) 可归入 \(P\) 或 \(N\)）；但 Jordan 分解 \(\mu^+, \mu^-\) 是唯一的
  • ❌ 将全变差 \(|\mu|(E)\) 误解为 \(|\mu(E)|\) → ✅ \(|\mu|(E) = \mu^+(E) + \mu^-(E) \geq |\mu(E)|\)，等号一般不成立；\(|\mu|\) 度量的是"总变动量"而非"净值的绝对值"
  • ❌ 忽略符号测度不能同时取 \(\pm\infty\) 的约束 → ✅ 若 \(\mu^+(X) = +\infty\) 且 \(\mu^-(X) = +\infty\)，则 \(\mu(X) = \infty - \infty\) 无意义；定义要求排除这种情况

02绝对连续与 Radon-Nikodym 定理

🎯 为什么研究这个问题

什么时候一个测度 \(\mu\) 可以用另一个测度 \(\nu\) 的"密度函数"来表示——即 \(d\mu = f\,d\nu\)？这不仅是测度论的核心问题，更是连接抽象测度与具体函数的桥梁。Radon-Nikodym 定理回答：当且仅当 \(\mu\) 关于 \(\nu\) 绝对连续时，密度函数存在。

  • 概率论：条件期望的严格定义、Girsanov 定理中的测度变换、鞅的收敛定理——全部以 Radon-Nikodym 导数为核心工具；
  • 统计学：似然比 \(L = dP_1/dP_0\) 是两个统计模型之间的 RN 导数——充分统计量、信息不等式等概念的数学基础；
  • PDE 与 Sobolev 空间：Sobolev 空间的对偶表示、弱导数的测度论解释都依赖 RN 定理。

RN 定理的深层意义在于为测度空间建立了"微积分学"——\(f = d\mu/d\nu\) 类比于 \(F'(x) = dF/dx\)，链式法则 \(\frac{d\mu}{d\lambda} = \frac{d\mu}{d\nu}\frac{d\nu}{d\lambda}\) 更加强了这一类比。von Neumann 的证明方法尤其优雅：通过 Riesz 表示定理将测度论问题转化为 Hilbert 空间的几何问题，展示了泛函分析工具在测度论中的威力。

🔍 抽象化方法
  • 测度论中的"求导"——\(f = d\mu/d\nu\) 将两个测度之间的关系编码为一个函数，类比微积分中 \(F'(x) = dF/dx\)。链式法则 \(\frac{d\mu}{d\lambda} = \frac{d\mu}{d\nu}\frac{d\nu}{d\lambda}\) 进一步加强了这一类比
  • Hilbert 空间方法——von Neumann 证法：在 \(L^2(\mu + \nu)\) 上定义泛函，用 Riesz 表示定理找到密度函数。这将测度论问题转化为 Hilbert 空间的几何问题
  • σ-有限性是"可分块"的保证——σ-有限意味着全空间可分解为可数个有限测度子空间，在每个子空间上证明 RN 定理后再"拼合"。非 σ-有限时定理可能失效
📐 公式推导 / 模型建立

绝对连续的定义

\(\mu \ll \nu\) 若 \(\nu(E) = 0 \Rightarrow \mu(E) = 0\)。

对有限测度，等价于 ε-δ 刻画：\(\forall \varepsilon > 0\)，\(\exists \delta > 0\)，\(\nu(E) < \delta \Rightarrow |\mu(E)| < \varepsilon\)。

Radon-Nikodym 定理

设 \(\mu, \nu\) 为 σ-有限测度，\(\mu \ll \nu\)。则存在 ν-a.e. 唯一的非负可测函数 \(f\) 使 \[\mu(E) = \int_E f\,d\nu, \quad \forall E \in \mathcal{F}.\] \(f\) 称为 Radon-Nikodym 导数，记 \(f = \frac{d\mu}{d\nu}\)。

证明思路（von Neumann 方法）

  1. 令 \(\lambda = \mu + \nu\)。在 \(L^2(\lambda)\) 上定义有界线性泛函 \(\phi(g) = \int g\,d\nu\)
  2. Riesz 表示定理：存在 \(h \in L^2(\lambda)\) 使 \(\int g\,d\nu = \int gh\,d\lambda\)，且 \(0 \leq h \leq 1\) λ-a.e.
  3. 令 \(A = \{h > 0\}\)。定义 \(f = (1 - h)/h\) 在 \(A\) 上，则 \(\mu(E) = \int_E f\,d\nu\)（需要仔细处理 \(\{h = 0\}\) 和 \(\{h = 1\}\) 的集合）

链式法则

若 \(\mu \ll \nu \ll \lambda\)，则 \[\frac{d\mu}{d\lambda} = \frac{d\mu}{d\nu} \cdot \frac{d\nu}{d\lambda}, \quad \lambda\text{-a.e.}\]

推广：符号测度的 RN 定理

若 \(\mu\) 为符号测度，\(\nu\) 为 σ-有限测度，\(\mu \ll \nu\)，则存在 \(f \in L^1(\nu)\) 使 \(\mu(E) = \int_E f\,d\nu\)，且 \(|f| = d|\mu|/d\nu\)。

🌐 实际应用

概率论：条件期望与鞅——条件期望 \(\mathbb{E}[X|\mathcal{G}]\) 就是测度 \(\mu(G) = \int_G X\,dP\) 关于 \(P|_\mathcal{G}\) 的 Radon-Nikodym 导数。鞅理论中，\(M_n = \mathbb{E}[X|\mathcal{F}_n]\) 是 RN 导数的序列。Doob 分解和可选停时定理都以此为基础。

金融数学：风险中性定价——Black-Scholes 框架中，风险中性测度 \(Q\) 相对于物理测度 \(P\) 的 RN 导数 \(dQ/dP = \mathcal{E}(-\theta W)_T\)（Girsanov 定理）给出了市场风险价格。衍生品价格 = 在 \(Q\) 下对折现收益取期望，本质上是用 RN 导数做测度变换。

统计推断：Neyman-Pearson 与信息论——似然比检验统计量 \(\Lambda = dP_1/dP_0\) 是两个模型下数据分布的 RN 导数。KL 散度 \(D_{\text{KL}}(P\|Q) = \int \log(dP/dQ)\,dP\) 也以 RN 导数为核心，度量两个分布的"信息距离"。

✏️ 例题分析与易错点

例题：直接读出 RN 导数

设 \(\mu(E) = \int_E 2x\,dx\) 在 \(([0,1], \mathcal{B}, \lambda)\) 上。求 \(d\mu/d\lambda\)。

Step 1　由定义，\(\mu(E) = \int_E 2x\,d\lambda\)，直接读出 \(f(x) = 2x\)。

Step 2　验证：\(f \geq 0\) 在 \([0,1]\) 上，\(\int_0^1 2x\,dx = 1\)，故 \(\mu\) 是概率测度，\(f\) 是 \(\text{Beta}(2,1)\) 密度。

例题：链式法则

设 \(\mu(E) = \int_E 3x^2\,dx\)，\(\nu(E) = \int_E 2x\,dx\)。求 \(d\mu/d\nu\)。

Step 1　\(d\mu/d\lambda = 3x^2\)，\(d\nu/d\lambda = 2x\)。

Step 2　链式法则：\(d\mu/d\nu = (d\mu/d\lambda)/(d\nu/d\lambda) = 3x^2/(2x) = 3x/2\)（\(x > 0\)）。

常见错误

  • ❌ 忘记 RN 导数仅在 a.e. 意义下唯一 → ✅ 可在零测集上任意修改 \(f\) 的值而不影响积分；唯一性是 ν-a.e. 的
  • ❌ 省略 σ-有限性条件 → ✅ 经典反例：\(\mu =\) 计数测度，\(\nu = \lambda\)（Lebesgue 测度），\(\mu \ll \lambda\) 不成立（单点集 \(\lambda\)-零测但 \(\mu\)-非零），但即使构造绝对连续的例子，σ-有限性仍是证明存在性的关键
  • ❌ 将 ε-δ 绝对连续与定义混淆 → ✅ ε-δ 版本（定量）对有限测度等价于定义（定性）；对 σ-有限但非有限测度，仅定性版本成立

03Lebesgue 分解定理

🎯 为什么研究这个问题

两个测度之间不一定存在绝对连续关系——例如 Lebesgue 测度与 Dirac 测度互不"随从"。Lebesgue 分解定理告诉我们：无需纠结于这种不兼容性，因为任何测度都可以唯一地拆为"随 \(\nu\) 变化的部分"和"与 \(\nu\) 完全无关的部分"。

  • 概率论：随机变量的分布关于 Lebesgue 测度的分解自然给出三类——连续型（有密度）、离散型、奇异连续型（如 Cantor 分布），这一分类是概率论的标准框架；
  • 信号处理：将信号的谱测度分解为绝对连续部分（宽带噪声）与奇异部分（纯频率峰），是频谱分析的基本操作；
  • 泛函分析：Lebesgue 分解是测度空间 \(M(X)\) 中的"正交分解"——类比 Hilbert 空间中向量在子空间及其正交补上的投影。

Lebesgue 分解的本质是测度论的"结构定理"：它揭示了两个测度之间关系的完整图景——要么"随之变化"（绝对连续），要么"互不相干"（奇异），每个测度恰好是这两种极端情况的唯一叠加。绝对连续和奇异是互补关系，不能同时成立（除非为零测度），这保证了分解的唯一性。

🔍 抽象化方法
  • 正交分解类比——在 Hilbert 空间中，任何向量可正交分解为子空间上的投影和正交补上的分量。Lebesgue 分解是这一思想在测度空间中的对应：\(\mu_{ac}\) 是 \(\mu\) 在 \(\nu\) "方向"上的投影，\(\mu_s\) 是"正交"于 \(\nu\) 的分量
  • Radon-Nikodym 的推论——Lebesgue 分解可从 RN 定理直接推导：令 \(\lambda = \mu + \nu\)，\(\mu \ll \lambda\)，写出 \(d\mu/d\lambda = f\)，\(d\nu/d\lambda = g\)，在 \(\{g > 0\}\) 上定义 \(\mu_{ac}\)，在 \(\{g = 0\}\) 上定义 \(\mu_s\)
  • 分解的唯一性——绝对连续和奇异是"互补"的关系：不能既绝对连续又奇异（除非为零测度），这保证了分解的唯一性
📐 公式推导 / 模型建立

定理（Lebesgue 分解）

设 \(\mu, \nu\) 为 σ-有限测度。则存在唯一的分解 \[\mu = \mu_{ac} + \mu_s\] 其中 \(\mu_{ac} \ll \nu\)（绝对连续），\(\mu_s \perp \nu\)（互相奇异）。

互相奇异的定义

\(\mu_s \perp \nu\)：存在 \(A \in \mathcal{F}\) 使得 \(\mu_s(A^c) = 0\) 且 \(\nu(A) = 0\)。即 \(\mu_s\) 和 \(\nu\) "住在"不相交的集合上。

证明思路

Step 1　令 \(\lambda = \mu + \nu\)。\(\mu \ll \lambda\) 且 \(\nu \ll \lambda\)。

Step 2　RN 定理给出 \(d\mu = f\,d\lambda\)，\(d\nu = g\,d\lambda\)（\(f, g \geq 0\)）。

Step 3　令 \(A = \{g > 0\}\)，\(B = \{g = 0\}\)。则 \(\nu(B) = \int_B g\,d\lambda = 0\)。

Step 4　定义 \(\mu_{ac}(E) = \mu(E \cap A)\)，\(\mu_s(E) = \mu(E \cap B)\)。

  • \(\mu_{ac} \ll \nu\)：若 \(\nu(E) = 0\)，则 \(g = 0\) ν-a.e. on \(E\)，进而 \(g = 0\) λ-a.e. on \(E \cap A\)，但 \(A = \{g > 0\}\)，故 \(\lambda(E \cap A) = 0\)，故 \(\mu(E \cap A) = 0\)
  • \(\mu_s \perp \nu\)：\(\mu_s(B^c) = \mu(B^c \cap B) = 0\) 且 \(\nu(B) = 0\) ✓

与 RN 导数的关系

\(\mu_{ac}\) 有 RN 导数 \(d\mu_{ac}/d\nu = f/g\)（在 \(A = \{g > 0\}\) 上定义）。\(\mu_s\) 没有关于 \(\nu\) 的密度。

🌐 实际应用

概率论：分布的三元分类——任何随机变量 \(X\) 的分布 \(\mu_X\) 相对于 Lebesgue 测度 \(\lambda\) 可分解为 \(\mu_X = \mu_{ac} + \mu_d + \mu_{sc}\)（绝对连续 + 离散 + 奇异连续）。绝对连续部分有密度函数 \(f_X\)；离散部分集中在可数点集上（PMF）；奇异连续部分（如 Cantor 分布）连续但无密度——这一分类是概率论教材中"三类分布"的严格基础。

信号处理：谱分解——平稳随机过程的谱测度 \(S\) 关于 Lebesgue 测度的 Lebesgue 分解区分了连续谱（宽带噪声）和离散谱（纯正弦分量）。谱分析算法（如 MUSIC、ESPRIT）本质上是在估计奇异部分 \(S_s\) 的支撑位置（频率检测）。

图像处理：梯度的测度论——BV（有界变差）函数 \(u\) 的梯度 \(Du\) 是向量值符号测度。其 Lebesgue 分解 \(Du = \nabla u \cdot \lambda + D_s u\) 将梯度拆为"光滑部分"和"跳跃/Cantor 部分"。边缘检测对应于定位 \(D_s u\) 的支撑集（图像的边缘线）。

✏️ 例题分析与易错点

例题：Lebesgue + Dirac 的分解

将 \(\mu = \lambda + \delta_0\)（\(\lambda\) = Lebesgue 测度，\(\delta_0\) = 原点 Dirac 测度）关于 \(\lambda\) 做 Lebesgue 分解。

Step 1　\(\lambda \ll \lambda\)（自身绝对连续），故 \(\mu_{ac} = \lambda\)。

Step 2　\(\delta_0 \perp \lambda\)：\(\delta_0\) 集中在 \(\{0\}\)，\(\lambda(\{0\}) = 0\)。故 \(\mu_s = \delta_0\)。

Step 3　验证：\(\mu = \lambda + \delta_0 = \mu_{ac} + \mu_s\) ✓。

常见错误

  • ❌ 混淆"绝对连续"和"互相奇异" → ✅ \(\mu \ll \nu\)："ν 归零则 μ 归零"（μ 到处跟随 ν）；\(\mu \perp \nu\)："μ 和 ν 住在不相交的集上"（互不相干）。二者是"极端"的两种关系
  • ❌ 认为 Cantor 函数的 Stieltjes 测度 \(\mu_C\) 绝对连续于 \(\lambda\) → ✅ \(\mu_C \perp \lambda\)（\(\mu_C\) 集中在 Cantor 集 \(C\) 上，\(\lambda(C) = 0\)）。\(\mu_C\) 虽然连续（无原子），但与 \(\lambda\) 奇异——这是奇异连续测度的典范
  • ❌ 忘记分解的唯一性 → ✅ 若 \(\mu = \mu_{ac}' + \mu_s'\) 也满足条件，则 \(\mu_{ac} - \mu_{ac}'\) 既绝对连续于 \(\nu\) 又奇异于 \(\nu\)，只能为零测度

04条件概率与贝叶斯公式的严格化

🎯 为什么研究这个问题

初等概率论中 \(P(A|B) = P(A \cap B)/P(B)\) 要求 \(P(B) > 0\)，但连续型随机变量的条件概率 \(P(A|X=x)\) 面对的是 \(P(X=x)=0\) 的困境——分母为零，公式崩塌。测度论的 Radon-Nikodym 定理为条件期望提供了不依赖"除以概率"的严格定义。

  • 贝叶斯统计：后验分布 \(\pi(\theta|x) \propto f(x|\theta)\pi(\theta)\) 在连续参数空间中的严格化、MCMC 采样的收敛保证——都建立在测度论条件概率之上；
  • 金融数学：鞅理论中的滤过和条件期望是资产定价基本定理的数学语言——信息流的精炼由子 σ-代数链 \(\mathcal{G}_1 \subseteq \mathcal{G}_2 \subseteq \cdots\) 表达；
  • 机器学习：最优预测 \(\mathbb{E}[Y|X]\) 是 \(Y\) 在 \(\sigma(X)\) 可测函数空间上的 \(L^2\) 正交投影——条件期望同时拥有测度论和几何两种解读。

条件期望 \(\mathbb{E}[X|\mathcal{G}]\) 的测度论定义统一了离散和连续情形：它是子 σ-代数 \(\mathcal{G}\) 上的 RN 导数，也是 \(L^2\) 中的正交投影。塔性质 \(\mathbb{E}[\mathbb{E}[X|\mathcal{G}]|\mathcal{H}] = \mathbb{E}[X|\mathcal{H}]\)（\(\mathcal{H} \subseteq \mathcal{G}\)）揭示了信息精炼的本质——更少的信息只能给出更粗糙的估计，但绝不引入新的误差。

🔍 抽象化方法
  • 投影算子类比——条件期望 \(\mathbb{E}[X|\mathcal{G}]\) 是 \(L^2(\Omega, \mathcal{F}, P)\) 中的正交投影到子空间 \(L^2(\Omega, \mathcal{G}, P)\)。这使条件期望同时拥有测度论（RN 导数）和 Hilbert 空间（最优逼近）两种解读
  • 从函数到测度——将 \(\mathbb{E}[X|\mathcal{G}]\) 视为子 σ-代数 \(\mathcal{G}\) 上的测度变换：定义 \(\mu(G) = \int_G X\,dP\)，则 \(\mathbb{E}[X|\mathcal{G}] = d\mu/dP|_\mathcal{G}\)。条件概率 \(P(A|\mathcal{G}) = \mathbb{E}[\mathbf{1}_A|\mathcal{G}]\) 是特例
  • 信息的精炼——子 σ-代数 \(\mathcal{G} \subseteq \mathcal{F}\) 代表"部分信息"。条件期望将全信息下的随机变量"投影"到部分信息上，保留了能被 \(\mathcal{G}\) 观测的部分。这是鞅论中滤过（filtration）概念的基础
📐 公式推导 / 模型建立

条件期望的测度论定义

\(\mathbb{E}[X|\mathcal{G}]\) 是满足以下两个条件的（ν-a.e. 唯一）随机变量 \(Y\)：

  1. \(Y\) 是 \(\mathcal{G}\)-可测的
  2. \(\int_G Y\,dP = \int_G X\,dP\)，对所有 \(G \in \mathcal{G}\)

存在性：定义符号测度 \(\mu(G) = \int_G X\,dP\) 在 \((\Omega, \mathcal{G})\) 上，\(\mu \ll P|_\mathcal{G}\)（因为 \(P(G) = 0 \Rightarrow \mu(G) = 0\)）。RN 定理给出 \(Y = d\mu/dP|_\mathcal{G}\)。

基本性质

  • 线性性：\(\mathbb{E}[aX + bY|\mathcal{G}] = a\mathbb{E}[X|\mathcal{G}] + b\mathbb{E}[Y|\mathcal{G}]\)
  • 塔性质：\(\mathcal{H} \subseteq \mathcal{G} \Rightarrow \mathbb{E}[\mathbb{E}[X|\mathcal{G}]|\mathcal{H}] = \mathbb{E}[X|\mathcal{H}]\)
  • 独立性：若 \(X\) 与 \(\mathcal{G}\) 独立，\(\mathbb{E}[X|\mathcal{G}] = \mathbb{E}[X]\)
  • 可测提取：若 \(Y\) 是 \(\mathcal{G}\)-可测的，\(\mathbb{E}[XY|\mathcal{G}] = Y\mathbb{E}[X|\mathcal{G}]\)

贝叶斯公式的 RN 表达

设 \(P_0\)（先验）和 \(P_1\)（后验）为两个概率测度，\(P_1 \ll P_0\)。令 \(L = dP_1/dP_0\)（似然比）。则对事件 \(A\)： \[P_1(A) = \mathbb{E}_{P_0}[L \cdot \mathbf{1}_A].\] 这是经典贝叶斯公式 \(P(\theta|x) = \frac{P(x|\theta)P(\theta)}{P(x)}\) 在一般测度空间上的推广。

正则条件概率

当 \(\mathcal{G} = \sigma(Y)\) 时，存在函数 \(P(A|Y = y)\)（关于 \(y\) 可测）使得 \(P(A|\mathcal{G})(\omega) = P(A|Y = Y(\omega))\) a.s.。在 Polish 空间上，这样的正则版本总是存在。

🌐 实际应用

贝叶斯统计：后验推断——给定先验 \(\pi(\theta)\) 和似然 \(f(x|\theta)\)，后验测度 \(\Pi(\cdot|x)\) 关于先验 \(\Pi_0\) 的 RN 导数为 \(d\Pi(\cdot|x)/d\Pi_0 = f(x|\theta)/m(x)\)，其中 \(m(x) = \int f(x|\theta)\,d\Pi_0\)。MCMC 采样器（Metropolis-Hastings、Gibbs）实际上是在模拟这个 RN 导数的比值。

机器学习：重要性采样与密度比估计——训练数据分布 \(Q\) 与目标分布 \(P\) 不同时，期望校正为 \(\mathbb{E}_P[f(X)] = \mathbb{E}_Q[f(X) \cdot dP/dQ(X)]\)。密度比 \(dP/dQ\) 就是 RN 导数。域适应（domain adaptation）、off-policy 强化学习的重要性权重都基于此。

金融工程：风险中性定价——衍生品定价核心公式 \(V_0 = e^{-rT}\mathbb{E}_Q[\text{payoff}]\) 中的 \(Q\) 是风险中性测度。由 Girsanov 定理，\(dQ/dP\) 编码了市场风险价格。条件期望 \(\mathbb{E}_Q[\text{payoff}|\mathcal{F}_t]\) 给出任意时刻的公允价格——这就是鞅定价理论。

✏️ 例题分析与易错点

例题：指数分布的条件期望

设 \(X \sim \text{Exp}(\lambda)\)，\(Y = \mathbf{1}_{\{X > 1\}}\)。求 \(\mathbb{E}[X|Y]\)。

Step 1　\(\sigma(Y) = \{\emptyset, \{X > 1\}, \{X \leq 1\}, \Omega\}\)——只有两个非平凡事件，故 \(\mathbb{E}[X|Y]\) 在 \(\{X > 1\}\) 和 \(\{X \leq 1\}\) 上各取常数。

Step 2　在 \(\{X > 1\}\) 上：由无记忆性，\(\mathbb{E}[X|X > 1] = 1 + 1/\lambda\)。

Step 3　在 \(\{X \leq 1\}\) 上： \[\mathbb{E}[X|X \leq 1] = \frac{\int_0^1 x\lambda e^{-\lambda x}\,dx}{1 - e^{-\lambda}} = \frac{1/\lambda - (1 + 1/\lambda)e^{-\lambda}}{1 - e^{-\lambda}}.\]

Step 4　故 \(\mathbb{E}[X|Y] = \mathbb{E}[X|X > 1] \cdot \mathbf{1}_{\{X > 1\}} + \mathbb{E}[X|X \leq 1] \cdot \mathbf{1}_{\{X \leq 1\}}\)。

常见错误

  • ❌ 对连续 \(X\) 写 \(P(A|X = x) = P(A \cap \{X=x\})/P(X=x)\)（除以零）→ ✅ 用条件密度 \(f_{Y|X}(y|x)\) 或正则条件概率的测度论定义
  • ❌ 忘记塔性质中 σ-代数的包含方向 → ✅ \(\mathbb{E}[\mathbb{E}[X|\mathcal{G}]|\mathcal{H}] = \mathbb{E}[X|\mathcal{H}]\) 要求 \(\mathcal{H} \subseteq \mathcal{G}\)（先粗化再精化 = 直接粗化）；反向不成立
  • ❌ 混淆 \(\mathbb{E}[X|Y]\)（随机变量）和 \(\mathbb{E}[X|Y=y]\)（实数）→ ✅ \(\mathbb{E}[X|Y]\) 是 \(\sigma(Y)\)-可测的随机变量；\(\mathbb{E}[X|Y=y] = g(y)\) 是确定性函数，二者通过 \(\mathbb{E}[X|Y](\omega) = g(Y(\omega))\) 关联
