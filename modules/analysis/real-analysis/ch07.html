Radon-Nikodym 与符号测度

01符号测度与 Hahn-Jordan 分解

🎯 为什么研究这个问题

在概率论和物理学中，经常遇到"可以取负值的测度"——例如带符号的 电荷分布，或两个概率测度之差。符号测度将测度的概念从非负推广到 可正可负。Hahn 分解和 Jordan 分解告诉我们：每个符号测度都可以 "分离"为正部和负部，本质上仍可回到非负测度的框架。

🔍 抽象化方法

Hahn 分解将空间一分为二（正集和负集），Jordan 分解将符号测度 写为两个非负测度之差 \(\mu = \mu^+ - \mu^-\)。这与实数分解为 正负部 \(x = x^+ - x^-\) 完全类比。

📐 公式推导 / 模型建立

符号测度：\(\mu: \mathcal{F} \to [-\infty, +\infty]\) 满足 \(\mu(\emptyset) = 0\) 和 σ-可加性（\(\mu\) 不同时取 \(+\infty\) 和 \(-\infty\)）。 Hahn 分解定理：存在 \(P, N \in \mathcal{F}\)，\(P \cup N = X\)， \(P \cap N = \emptyset\)，使得 \(\mu(E) \geq 0\) \(\forall E \subseteq P\)， \(\mu(E) \leq 0\) \(\forall E \subseteq N\)。 Jordan 分解：\(\mu = \mu^+ - \mu^-\)，其中 \(\mu^+(E) = \mu(E \cap P)\)，\(\mu^-(E) = -\mu(E \cap N)\)， 且 \(\mu^+ \perp \mu^-\)（互相奇异）。 全变差测度：\(|\mu| = \mu^+ + \mu^-\)， \[|\mu|(E) = \sup\left\{\sum_{i=1}^n |\mu(E_i)| : E = \bigsqcup E_i\right\}\]

✏️ 例题分析与易错点

例题：设 \(\mu(E) = \int_E f\, d\lambda\)，\(f(x) = x\) 在 \([-1, 1]\) 上。 求 \(\mu\) 的 Hahn 分解和 Jordan 分解。 解答：\(P = [0, 1]\)（\(f \geq 0\)），\(N = [-1, 0)\)（\(f < 0\)）。 \(\mu^+(E) = \int_{E \cap [0,1]} x\, dx\)， \(\mu^-(E) = \int_{E \cap [-1,0)} (-x)\, dx\)。 \(|\mu|([-1,1]) = \int_0^1 x\, dx + \int_{-1}^0 (-x)\, dx = 1/2 + 1/2 = 1\)。 易错点：Hahn 分解不唯一（\(P, N\) 可以差一个零测集）， 但 Jordan 分解 \(\mu^+, \mu^-\) 是唯一的。

02绝对连续与 Radon-Nikodym 定理

🎯 为什么研究这个问题

什么时候一个测度 \(\mu\) 可以用另一个测度 \(\nu\) 的"密度函数" 表示为 \(d\mu = f\, d\nu\)？直觉上，如果 \(\nu\) 认为某个集合 "大小为零"，\(\mu\) 也应该认为它大小为零——这就是绝对连续性 \(\mu \ll \nu\)。Radon-Nikodym 定理给出了精确的回答： 绝对连续等价于密度函数的存在性。

🔍 抽象化方法

Radon-Nikodym 定理是"求导"在测度论中的推广：\(f = d\mu/d\nu\) 类比实变函数的导数，将两个测度的关系编码为一个函数。

📐 公式推导 / 模型建立

绝对连续：\(\mu \ll \nu\) 若 \(\nu(E) = 0 \Rightarrow \mu(E) = 0\)。 Radon-Nikodym 定理：设 \(\mu, \nu\) 是 σ-有限测度， \(\mu \ll \nu\)。则存在 \(\nu\)-a.e. 唯一的非负可测函数 \(f\) 使得 \[\mu(E) = \int_E f\, d\nu, \quad \forall E \in \mathcal{F}\] \(f\) 称为 Radon-Nikodym 导数，记为 \(f = \frac{d\mu}{d\nu}\)。 链式法则：若 \(\mu \ll \nu \ll \lambda\)，则 \[\frac{d\mu}{d\lambda} = \frac{d\mu}{d\nu} \cdot \frac{d\nu}{d\lambda} \quad \lambda\text{-a.e.}\] σ-有限性必要：反例——计数测度 \(\mu\) 关于 Lebesgue 测度 \(\lambda\) 不满足 \(\mu \ll \lambda\)，但也说明 σ-有限性不能随意去除。

🌐 实际应用

概率论：条件期望 \(E[X|\mathcal{G}]\) 就是 \(\mu_X\) 关于 \(P|_\mathcal{G}\) 的 Radon-Nikodym 导数；贝叶斯公式中似然比 \(dP_1/dP_0\) 是两个概率测度的 Radon-Nikodym 导数。

✏️ 例题分析与易错点

例题：设 \(\mu(E) = \int_E 2x\, dx\) 在 \(([0,1], \mathcal{B}, \lambda)\) 上。 求 \(d\mu/d\lambda\)。 解答：由定义 \(\mu(E) = \int_E 2x\, d\lambda\)， 故 \(f(x) = d\mu/d\lambda = 2x\)（直接读出）。 易错点：忘记 Radon-Nikodym 导数仅在 a.e. 意义下唯一。 可以在零测集上任意修改 \(f\) 的值。

03Lebesgue 分解定理

🎯 为什么研究这个问题

给定两个测度 \(\mu\) 和 \(\nu\)，\(\mu\) 不一定绝对连续于 \(\nu\)。 Lebesgue 分解定理说：\(\mu\) 总可以唯一地分解为关于 \(\nu\) 绝对连续的部分和奇异的部分。这一定理将任意测度对之间的关系 完美刻画，是测度论的收尾性定理之一。

🔍 抽象化方法

Lebesgue 分解是"正交分解"在测度空间中的类比： \(\mu_{ac}\) 对应投影到 \(\nu\) 方向，\(\mu_s\) 对应 垂直于 \(\nu\) 的分量。

📐 公式推导 / 模型建立

定理 (Lebesgue 分解)：设 \(\mu, \nu\) 为 σ-有限测度。则 存在唯一的分解 \[\mu = \mu_{ac} + \mu_s\] 其中 \(\mu_{ac} \ll \nu\)，\(\mu_s \perp \nu\)。 互相奇异 \(\mu_s \perp \nu\)：存在 \(A\) 使 \(\mu_s(A^c) = 0\) 且 \(\nu(A) = 0\)。 证明思路：定义 \(\lambda = \mu + \nu\)。\(\mu \ll \lambda\) 且 \(\nu \ll \lambda\)，由 Radon-Nikodym， \(d\mu = f\, d\lambda\), \(d\nu = g\, d\lambda\)。 令 \(A = \{g > 0\}\)，\(B = \{g = 0\}\)。 \(\mu_{ac}(E) = \mu(E \cap A) = \int_{E \cap A} (f/g)\, d\nu\)， \(\mu_s(E) = \mu(E \cap B)\)，且 \(\nu(B) = 0\)。

✏️ 例题分析与易错点

例题：将 \(\mu = \lambda + \delta_0\)（Lebesgue 测度 + Dirac 测度） 关于 \(\lambda\) 做 Lebesgue 分解。 解答：\(\mu_{ac} = \lambda\)（绝对连续于 \(\lambda\)）， \(\mu_s = \delta_0\)（\(\delta_0 \perp \lambda\)，因为 \(\delta_0\) 集中在 \(\{0\}\)，\(\lambda(\{0\}) = 0\)）。 易错点：将 "绝对连续" 和 "互相奇异" 搞混。 \(\mu \ll \nu\)："\(\nu\) 归零则 \(\mu\) 归零"。 \(\mu \perp \nu\)："\(\mu\) 和 \(\nu\) 住在不相交的集合上"。

04条件概率与贝叶斯公式的严格化

🎯 为什么研究这个问题

初等概率论中，条件概率 \(P(A|B) = P(A \cap B)/P(B)\) 在 \(P(B) > 0\) 时有定义。但若条件是一个连续随机变量的取值 （如 \(P(A | X = x)\)），分母 \(P(X = x) = 0\)，公式失效。 Radon-Nikodym 定理为条件概率提供了严格的定义： 条件期望就是一个 Radon-Nikodym 导数。

🔍 抽象化方法

条件期望 \(E[X | \mathcal{G}]\) 是测度论中的"投影"—— 将随机变量投影到子 σ-代数上，类似于 \(L^2\) 中的正交投影。 Radon-Nikodym 导数将这一抽象概念植根于具体的测度关系。

📐 公式推导 / 模型建立

条件期望的测度论定义：\(E[X | \mathcal{G}]\) 是唯一的 \(\mathcal{G}\)-可测函数 \(Y\) 满足 \[\int_G Y\, dP = \int_G X\, dP, \quad \forall G \in \mathcal{G}\] 等价地，定义测度 \(\mu(G) = \int_G X\, dP\) 在 \(\mathcal{G}\) 上， 若 \(\mu \ll P|_\mathcal{G}\)，则 \(Y = d\mu / dP|_\mathcal{G}\)。 贝叶斯公式的 RN 表达：若 \(P_0, P_1\) 是两个概率测度， \(P_1 \ll P_0\)，令 \(L = dP_1/dP_0\)（似然比），则对事件 \(A\)： \[P_1(A) = E_{P_0}[L \cdot \mathbf{1}_A]\] 这推广了经典贝叶斯公式到一般测度空间。

🌐 实际应用

统计推断：假设检验中的 Neyman-Pearson 引理以似然比 \(dP_1/dP_0\) 为检验统计量；机器学习中的重要性采样 基于密度比 \(dP/dQ\) 校正样本权重。

✏️ 例题分析与易错点

例题：设 \(X \sim \text{Exp}(\lambda)\)，\(Y = \mathbf{1}_{\{X > 1\}}\)。 求 \(E[X | Y]\)。 解答：\(\sigma(Y) = \{\emptyset, \{X > 1\}, \{X \leq 1\}, \Omega\}\)， \(E[X | Y]\) 在 \(\{X > 1\}\) 上等于 \(E[X | X > 1] = 1 + 1/\lambda\) （无记忆性），在 \(\{X \leq 1\}\) 上等于 \(E[X | X \leq 1] = \frac{\int_0^1 x \lambda e^{-\lambda x} dx}{1 - e^{-\lambda}}\)。 易错点：试图对连续随机变量写 \(P(A | X = x)\) 时除以 \(P(X = x) = 0\)。正确做法是用条件密度或 RN 导数。
