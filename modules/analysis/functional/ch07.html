紧算子与谱论初步

01紧算子的定义与性质

🎯 为什么研究这个问题

在有限维空间中，线性算子自动将有界集映为有界集（因此列紧集）。 无穷维空间中这不再成立——一般算子不保持紧性。紧算子是介于 有限维操作和一般操作之间的"好算子"：它们保持紧性，保留了 有限维矩阵理论中许多优美的性质（如谱离散、特征值分解）。

🔍 抽象化方法

紧算子 = 有限秩算子的极限。这一刻画体现了"无穷维中 最接近有限维的对象"的思想。

📐 公式推导 / 模型建立

定义：\(T: X \to Y\) 为紧算子，若 \(T(B_X)\)（有界集的像） 在 \(Y\) 中列紧（即 \(\overline{T(B_X)}\) 紧致）。 性质： (1) 紧算子是有界的：\(\overline{T(B_X)}\) 紧 ⇒ 有界 ⇒ \(\|T\| < \infty\)。 (2) 有限秩算子紧（像空间有限维，有界闭集紧致）。 (3) 紧算子的极限紧：若 \(T_n\) 紧，\(\|T_n - T\| \to 0\)，则 \(T\) 紧。 (4) 紧算子的复合：\(T\) 紧，\(S\) 有界 ⇒ \(ST\) 和 \(TS\) 紧。 (5) 在 Hilbert 空间中：\(T\) 紧 \(\Leftrightarrow\) \(T\) 是有限秩算子的范数极限。 关键判别：若 \(X\) 无穷维 Banach 空间，恒等算子 \(I\) 不紧 （Riesz 引理：闭单位球不列紧）。

🌐 实际应用

积分算子 \((Tf)(x) = \int K(x,y) f(y) dy\) 在适当条件下紧 （Hilbert-Schmidt 算子），PDE 的 Green 函数给出紧算子。

✏️ 例题分析与易错点

例题：\(T: l^2 \to l^2\)，\(T(x_1, x_2, \ldots) = (x_1, x_2/2, x_3/3, \ldots)\)。 证明 \(T\) 紧。 解答：定义 \(T_n(x_1, \ldots) = (x_1, x_2/2, \ldots, x_n/n, 0, 0, \ldots)\)。 \(T_n\) 有限秩（像在前 \(n\) 个坐标），故紧。 \(\|T - T_n\| = \sup_{k > n} 1/k = 1/(n+1) \to 0\)。 紧算子的极限紧。✓ 易错点：恒等算子在无穷维空间中不紧！ \(I(B_X) = B_X\)，闭单位球在无穷维中不列紧。

02谱的定义与分类

🎯 为什么研究这个问题

矩阵的特征值完全决定了矩阵的结构（Jordan 形）。无穷维算子 的"特征值理论"是什么？谱 \(\sigma(T)\) 推广了特征值的概念： \(\lambda \in \sigma(T)\) 当 \(T - \lambda I\) 不可逆。但无穷维中 "不可逆"的原因比有限维更多——可能不是单射（特征值）， 也可能不是满射或逆无界。

🔍 抽象化方法

谱将算子分析转化为复分析——谱是复平面的紧子集， 预解式 \(R(\lambda, T) = (T - \lambda I)^{-1}\) 是解析的算子值函数。

📐 公式推导 / 模型建立

预解集：\(\rho(T) = \{\lambda \in \mathbb{C} : (T - \lambda I)^{-1} \in B(X)\}\)（存在 + 有界 + 定义在全空间）。 谱：\(\sigma(T) = \mathbb{C} \setminus \rho(T)\)。 谱的分类： - 点谱 \(\sigma_p(T)\)：\(T - \lambda I\) 不是单射（\(\lambda\) 是特征值） - 连续谱 \(\sigma_c(T)\)：\(T - \lambda I\) 单射，像稠密但不等于 \(Y\)，逆无界 - 剩余谱 \(\sigma_r(T)\)：\(T - \lambda I\) 单射，像不稠密 基本性质： (1) \(\sigma(T)\) 是紧集合（\(\sigma(T) \subseteq \overline{B}(0, \|T\|)\)） (2) \(\sigma(T)\) 非空（\(X\) 为复 Banach 空间时） (3) 谱半径 \(r(T) = \sup\{|\lambda| : \lambda \in \sigma(T)\} = \lim \|T^n\|^{1/n}\)

✏️ 例题分析与易错点

例题：右移算子 \(R: l^2 \to l^2\)，\(R(x_1, x_2, \ldots) = (0, x_1, x_2, \ldots)\)。 求 \(\sigma_p(R)\)。 解答：\(Rx = \lambda x \Rightarrow (0, x_1, x_2, \ldots) = (\lambda x_1, \lambda x_2, \ldots)\) \(\Rightarrow \lambda x_1 = 0\)，\(x_1 = \lambda x_2\)，\(\ldots\) 若 \(\lambda = 0\)：\(x = 0\)（非特征向量）。 若 \(\lambda \neq 0\)：\(x_1 = 0 \Rightarrow x_n = 0\) 对所有 \(n\)。 故 \(\sigma_p(R) = \emptyset\)（没有特征值！）。 但 \(\sigma(R) = \overline{B}(0, 1)\)（闭单位圆盘）。 易错点：无穷维算子可以没有特征值但谱非空。 谱不等于特征值集。

03紧自伴算子的谱定理

🎯 为什么研究这个问题

实对称矩阵可以正交对角化——这是有限维线性代数最优美的结果。 紧自伴算子的谱定理将此推广到无穷维：Hilbert 空间上的紧自伴 算子有实特征值序列趋于零，对应的特征向量构成标准正交系。 这一定理是量子力学中可观测量谱分解的数学基础。

🔍 抽象化方法

谱定理将紧自伴算子"对角化"为特征值与特征向量的组合： \(T = \sum \lambda_n \langle \cdot, e_n \rangle e_n\)。 这完美推广了矩阵的 \(A = Q\Lambda Q^T\)。

📐 公式推导 / 模型建立

定理：设 \(T: H \to H\) 为紧自伴算子，\(H\) 为可分 Hilbert 空间。则： (1) \(\sigma(T) \subseteq \mathbb{R}\)（因为自伴） (2) 不同特征值的特征向量正交 (3) \(\sigma(T) \setminus \{0\}\) 至多可数，且仅在 \(0\) 处聚集 (4) 存在标准正交系 \(\{e_n\}\) 和实数列 \(\lambda_n \to 0\) 使 \[Tx = \sum_{n=1}^{\infty} \lambda_n \langle x, e_n \rangle e_n\] 自伴算子特征值为实：\(\lambda \langle x, x\rangle = \langle Tx, x\rangle = \langle x, Tx\rangle = \overline{\lambda} \langle x, x\rangle\)， 故 \(\lambda = \overline{\lambda} \in \mathbb{R}\)。 \(\|T\|\) 或 \(-\|T\|\) 是特征值： \(\|T\| = \sup_{\|x\|=1} |\langle Tx, x\rangle|\)（自伴算子）， 取到 sup 的极值点即为特征向量。

🌐 实际应用

PCA（主成分分析）：协方差算子是紧自伴的，其特征值和特征向量 即主成分方向和方差。

✏️ 例题分析与易错点

例题：\(T: l^2 \to l^2\)，\(T(x_1, x_2, \ldots) = (x_1, x_2/2, x_3/3, \ldots)\)。 求谱。 解答：\(T\) 自伴（对角算子实系数）且紧（前面已证）。 特征值为 \(\{1/n : n \geq 1\}\)，特征向量为 \(e_n\)。 \(\sigma(T) = \{0\} \cup \{1/n : n \geq 1\}\)（\(0\) 是聚点但不是特征值， 因为 \(Tx = 0 \Rightarrow x = 0\)）。 易错点：\(0\) 可能在谱中但不是特征值（\(\ker T = \{0\}\) 但 \(T\) 不可逆——逆无界）。

04Fredholm 替代与 Sturm-Liouville 应用

🎯 为什么研究这个问题

紧算子理论的一个核心应用是 Fredholm 替代定理：对紧算子 \(T\)， 方程 \((I - T)x = y\) 要么对所有 \(y\) 有唯一解，要么齐次方程 \((I - T)x = 0\) 有非平凡解。这推广了有限维线性方程组的 Fredholm 替代（秩-零度定理），在积分方程和 Sturm-Liouville 问题中有根本性应用。

🔍 抽象化方法

Fredholm 替代体现了"紧算子使无穷维问题表现得像有限维"的精神： \(I - T\)（\(T\) 紧）的可逆性问题归结为核空间的有限维分析。

📐 公式推导 / 模型建立

定理 (Fredholm 替代)：设 \(T: X \to X\) 紧，\(\lambda \neq 0\)。则恰好一个成立： (a) \((T - \lambda I)x = 0\) 只有零解，此时 \(T - \lambda I\) 是双射 （方程 \((T - \lambda I)x = y\) 对每个 \(y\) 有唯一解）； (b) \((T - \lambda I)x = 0\) 有非零解（\(\lambda\) 是特征值）， 此时 \((T - \lambda I)x = y\) 仅当 \(y\) 满足有限个正交条件时有解。 Riesz-Schauder 定理：紧算子的非零谱点都是特征值， 其特征空间有限维，且 \(\sigma(T) \setminus \{0\}\) 至多可数。 Sturm-Liouville 问题：\(-u'' + q(x)u = \lambda u\) 在 \([0,1]\)， 边界条件 \(u(0) = u(1) = 0\)。 转化为积分方程 \(u = \lambda K u\)，\(K\) 为 Green 函数给出的 紧自伴算子，谱定理保证特征值可数、特征函数正交完备。

🌐 实际应用

振动膜的固有频率 = Laplacian 的特征值（Sturm-Liouville 推广）； 量子力学中束缚态能量 = Hamilton 算子的离散谱。

✏️ 例题分析与易错点

例题：积分方程 \(u(x) = \lambda \int_0^1 K(x,y) u(y) dy + f(x)\)， \(K(x,y) = \min(x,y)\)。 解答：\(K\) 是 \(-u'' = \lambda u\), \(u(0) = u(1) = 0\) 的 Green 函数。 特征值 \(\lambda_n = n^2\pi^2\)，特征函数 \(u_n = \sin(n\pi x)\)。 由 Fredholm 替代，当 \(\lambda \neq n^2\pi^2\) 对所有 \(n\) 时， 方程对每个 \(f\) 有唯一解。 易错点：Fredholm 替代是"二择一"——不能同时有非零核 和对所有右端项可解。
