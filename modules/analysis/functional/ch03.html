Hilbert 空间

01内积空间与 Hilbert 空间

🎯 为什么研究这个问题

Banach 空间有"长度"（范数）但缺少"角度"概念。内积将角度和 正交性引入线性空间，使得无穷维空间拥有类似欧几里得空间的几何 结构。Hilbert 空间——完备的内积空间——是量子力学的数学框架， 也是 Fourier 分析和统计学的基石。

🔍 抽象化方法

内积将"投影"和"正交分解"推广到无穷维。平行四边形恒等式 刻画了哪些 Banach 空间是 Hilbert 空间——它体现了欧几里得 几何的本质。

📐 公式推导 / 模型建立

内积：\(\langle \cdot, \cdot \rangle: H \times H \to \mathbb{K}\) 满足 共轭对称、线性（第一分量）、正定。诱导范数 \(\|x\| = \sqrt{\langle x, x \rangle}\)。 Cauchy-Schwarz 不等式：\(|\langle x, y \rangle| \leq \|x\| \cdot \|y\|\)。 等号 ⇔ \(x, y\) 线性相关。 证明：对 \(t \in \mathbb{R}\)，\(0 \leq \|x + ty\|^2 = \|x\|^2 + 2t\text{Re}\langle x,y\rangle + t^2\|y\|^2\)， 判别式 \(\leq 0\) 即得。 平行四边形恒等式： \[\|x + y\|^2 + \|x - y\|^2 = 2(\|x\|^2 + \|y\|^2)\] 直接由内积展开验证。此恒等式是内积空间的特征—— Banach 空间的范数满足此恒等式 ⇔ 范数来自内积（极化恒等式重构内积）。 Hilbert 空间 = 完备内积空间。

🌐 实际应用

量子力学中，状态 \(|\psi\rangle \in H\)，可观测量是自伴算子， 测量概率 \(= |\langle \phi | \psi \rangle|^2\)（Born 规则）。

✏️ 例题分析与易错点

例题：验证 \(l^1\) 的范数不满足平行四边形恒等式。 解答：取 \(x = (1,0,0,\ldots)\), \(y = (0,1,0,\ldots)\)。 \(\|x+y\|_1 = 2\), \(\|x-y\|_1 = 2\), \(\|x\|_1 = \|y\|_1 = 1\)。 \(4 + 4 = 8 \neq 2(1 + 1) = 4\)。不满足。✓ 故 \(l^1\) 不是 Hilbert 空间。 易错点：认为所有内积空间都是 Hilbert 空间。 不完备的内积空间不是 Hilbert 空间（如有限线性组合 按 \(L^2\) 内积构成的空间）。

02正交投影与最佳逼近

🎯 为什么研究这个问题

在 Hilbert 空间中，给定一个闭子空间 \(M\)，每个元素 \(x\) 都有 唯一的"最佳逼近" \(\hat{x} \in M\)（使 \(\|x - \hat{x}\|\) 最小）。 这不仅推广了有限维向量的投影概念，更是最小二乘法、Fourier 展开和信号滤波的理论基础。

🔍 抽象化方法

正交投影将"最佳逼近"问题转化为正交条件 \(x - \hat{x} \perp M\)。 正交分解 \(H = M \oplus M^\perp\) 是线性代数直和分解在无穷维的推广。

📐 公式推导 / 模型建立

投影定理：设 \(M\) 为 Hilbert 空间 \(H\) 的闭子空间，\(x \in H\)。 则存在唯一 \(\hat{x} \in M\) 使 \[\|x - \hat{x}\| = \inf_{m \in M} \|x - m\| = d(x, M)\] 且 \(x - \hat{x} \perp M\)（即 \(\langle x - \hat{x}, m \rangle = 0,\; \forall m \in M\)）。 证明（存在性）：令 \(d = \inf_m \|x - m\|\)，取 \(m_n\) 使 \(\|x - m_n\| \to d\)。由平行四边形恒等式： \[\|m_n - m_k\|^2 = 2\|x - m_n\|^2 + 2\|x - m_k\|^2 - 4\|x - (m_n + m_k)/2\|^2 \leq 2\|x - m_n\|^2 + 2\|x - m_k\|^2 - 4d^2 \to 0\] \(\{m_n\}\) 是 Cauchy 列，由 \(M\) 闭，收敛到 \(\hat{x} \in M\)。 正交分解：\(H = M \oplus M^\perp\)，\(M^\perp = \{x : \langle x, m \rangle = 0, \forall m \in M\}\)。 正交投影 \(P_M: H \to M\) 是有界线性算子，\(P_M^2 = P_M\), \(P_M^* = P_M\), \(\|P_M\| = 1\)。

🌐 实际应用

最小二乘回归：\(y \approx X\beta\)，\(\hat{\beta}\) 使 \(y - X\beta\) 正交于 \(X\) 的列空间，即 \(X^T(y - X\hat{\beta}) = 0\)。

✏️ 例题分析与易错点

例题：在 \(L^2([0, 2\pi])\) 中，求 \(f(x) = x\) 在 \(M = \text{span}\{1, \cos x, \sin x\}\) 上的正交投影。 解答：\(\hat{f} = a_0 + a_1 \cos x + b_1 \sin x\)， \(a_0 = \langle x, 1\rangle / \|1\|^2 = \pi\)， \(a_1 = \langle x, \cos x\rangle / \|\cos x\|^2 = 0\)（奇 × 偶 = 奇，积分为 0）， \(b_1 = \langle x, \sin x\rangle / \|\sin x\|^2 = 2\)（分部积分）。 \(\hat{f}(x) = \pi + 2\sin x\)。 易错点：投影定理需要 \(M\) 是闭子空间。 有限维子空间自动闭，但无穷维子空间可能不闭。

⚡ 快速检测

03标准正交基与 Parseval 等式

🎯 为什么研究这个问题

有限维空间有标准正交基，每个向量可以唯一地表示为基向量的 线性组合。无穷维 Hilbert 空间是否也有类似结构？标准正交基 （ONB）给出肯定回答——每个 Hilbert 空间元素可以展开为 Fourier 级数 \(x = \sum \langle x, e_n \rangle e_n\)，这是 Fourier 分析的抽象形式。

🔍 抽象化方法

标准正交基将无穷维 Hilbert 空间"坐标化"为 \(l^2\)： 映射 \(x \mapsto (\langle x, e_n \rangle)\) 是等距同构 \(H \cong l^2\)。

📐 公式推导 / 模型建立

标准正交系 (ONS)：\(\{e_\alpha\}\) 满足 \(\langle e_\alpha, e_\beta \rangle = \delta_{\alpha\beta}\)。 Bessel 不等式：\(\sum_\alpha |\langle x, e_\alpha \rangle|^2 \leq \|x\|^2\)。 证明：\(\|x - \sum_{i=1}^N c_i e_i\|^2 = \|x\|^2 - \sum |c_i|^2 \geq 0\) （取 \(c_i = \langle x, e_i \rangle\)）。 标准正交基 (ONB)：ONS \(\{e_\alpha\}\) 满足 \(\overline{\text{span}}\{e_\alpha\} = H\)。 Parseval 等式：\(\{e_n\}\) 是 ONB \(\Leftrightarrow\) \[\|x\|^2 = \sum_n |\langle x, e_n \rangle|^2, \quad \forall x \in H\] 可分 Hilbert 空间的分类：可分无穷维 Hilbert 空间都等距 同构于 \(l^2\)（有可数 ONB）。

🌐 实际应用

Fourier 分析中 \(\{e^{inx}/\sqrt{2\pi}\}\) 是 \(L^2([-\pi,\pi])\) 的 ONB， Parseval 等式即 \(\|f\|_2^2 = \sum |c_n|^2\)。

✏️ 例题分析与易错点

例题：在 \(l^2\) 中，标准基 \(e_n = (0,\ldots,0,1,0,\ldots)\) 构成 ONB。对 \(x = (1, 1/2, 1/3, \ldots) \in l^2\)， 验证 Parseval：\(\sum |\langle x, e_n \rangle|^2 = \sum 1/n^2 = \pi^2/6 = \|x\|^2\)。✓ 易错点：混淆"极大正交系"和"标准正交基"在可分和不可分情形。 在可分 Hilbert 空间中两者等价，但需要 Zorn 引理确保存在。

04可分 Hilbert 空间与 l² 同构

🎯 为什么研究这个问题

一个令人惊叹的事实：所有可分的无穷维 Hilbert 空间在等距同构 意义下都是"同一个"空间——\(l^2\)。这意味着 \(L^2([0,1])\)、 \(L^2(\mathbb{R})\)（可分）、任何其他可分 Hilbert 空间在几何上 完全相同。结构定理大大简化了 Hilbert 空间理论。

🔍 抽象化方法

ONB 给出"坐标系"：\(x \mapsto (\langle x, e_n \rangle)_{n=1}^\infty\) 是到 \(l^2\) 的等距同构。这将函数空间中的问题转化为序列空间 中的问题。

📐 公式推导 / 模型建立

定理：可分无穷维 Hilbert 空间 \(H\) 等距同构于 \(l^2\)。 证明：取 ONB \(\{e_n\}_{n=1}^\infty\)（可数，因为可分）。定义 \[U: H \to l^2, \quad Ux = (\langle x, e_n \rangle)_{n=1}^\infty\] - 像在 \(l^2\) 中（Bessel 不等式） - 等距（Parseval：\(\|Ux\|_{l^2} = \|x\|_H\)） - 线性（内积线性） - 满射（对 \((\alpha_n) \in l^2\)，\(x = \sum \alpha_n e_n\) 收敛于 \(H\)，\(Ux = (\alpha_n)\)） 故 \(U\) 是等距同构。 推论： - \(L^2([0,1]) \cong l^2 \cong L^2(\mathbb{R})\)（作为 Hilbert 空间） - 有限维：\(\dim H = n \Rightarrow H \cong \mathbb{C}^n\)（或 \(\mathbb{R}^n\)）

✏️ 例题分析与易错点

例题：\(L^2([-\pi, \pi])\) 和 \(L^2([0, 1])\) 等距同构吗？ 解答：是的。两者都是可分无穷维 Hilbert 空间，故都同构于 \(l^2\)。 具体构造：线性映射 \(f \mapsto g\)，\(g(t) = \sqrt{2\pi}\, f(2\pi t - \pi)\) 给出等距同构。 易错点：等距同构不保持具体函数的"形状"，只保持内积结构。 \(L^2\) 和 \(l^2\) "看起来"很不同，但作为 Hilbert 空间完全相同。

⚡ 快速检测
\(L^2(\mathbb{R})\) 是否可分？
