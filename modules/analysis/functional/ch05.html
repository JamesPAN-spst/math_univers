Hahn-Banach 定理

01Hahn-Banach 解析形式与延拓

🎯 为什么研究这个问题

在有限维空间中，线性泛函可以自由延拓（给基赋值即可）。但在 无穷维空间中，一个定义在子空间上的泛函能否延拓到全空间，并且 保持范数不变？Hahn-Banach 定理给出肯定回答，它是泛函分析 三大定理中用途最广的一个——保证了对偶空间"足够大"。

🔍 抽象化方法

Hahn-Banach 定理的证明使用 Zorn 引理（类比选择公理）， 体现了"极大延拓"的思想——将泛函一步步延拓到越来越大的 子空间，直到覆盖全空间。

📐 公式推导 / 模型建立

定理 (Hahn-Banach, 解析形式)：设 \(X\) 为实线性空间， \(p: X \to \mathbb{R}\) 次线性（\(p(x+y) \leq p(x)+p(y)\), \(p(\alpha x) = \alpha p(x)\)（\(\alpha \geq 0\)））， \(f: Y \to \mathbb{R}\) 线性（\(Y\) 为子空间），\(f \leq p\) on \(Y\)。 则存在线性 \(F: X \to \mathbb{R}\) 使 \(F|_Y = f\) 且 \(F \leq p\) on \(X\)。 赋范空间版本：设 \(Y \subseteq X\) 为子空间，\(f \in Y^*\)。 则存在 \(F \in X^*\) 使 \(F|_Y = f\) 且 \(\|F\|_{X^*} = \|f\|_{Y^*}\)。 （取 \(p(x) = \|f\| \cdot \|x\|\)。） 推论：对任意 \(x_0 \neq 0\)，存在 \(f \in X^*\) 使 \(f(x_0) = \|x_0\|\) 且 \(\|f\| = 1\)。 （在 \(Y = \text{span}\{x_0\}\) 上定义 \(f(\alpha x_0) = \alpha \|x_0\|\)， 然后延拓。） 这意味 \(X^*\) 分离 \(X\) 中的点。

🌐 实际应用

在优化理论中，Hahn-Banach 保证凸集可以被超平面分离（几何形式）， 是支持向量机理论的数学基础。

✏️ 例题分析与易错点

例题：在 \(c_0\) （趋于 0 的序列空间）中，\(Y = \{x : x_1 = 0\}\)， \(f(x) = x_2\) 定义在 \(Y\) 上。求 \(\|f\|\) 并延拓到 \(c_0\)。 解答：\(\|f\|_Y = \sup\{|x_2| : x \in Y, \|x\|_\infty \leq 1\} = 1\)。 延拓：\(F(x) = x_2\) 在全 \(c_0\) 上（因为 \(|x_2| \leq \|x\|_\infty\)， \(\|F\| = 1\)）。此例中延拓很自然。 易错点：Hahn-Banach 延拓不唯一（除非在 Hilbert 空间中用正交补）。 只保证存在性，不保证唯一性。

02Hahn-Banach 几何形式与凸集分离

🎯 为什么研究这个问题

Hahn-Banach 定理不仅有泛函延拓的"解析形式"，还有超平面 分离凸集的"几何形式"。几何形式直观地说：在赋范空间中， 不相交的凸集可以用一个超平面（连续线性泛函的水平集）隔开。 这一结果是凸分析、优化理论和博弈论中 minimax 定理的基石。

🔍 抽象化方法

几何 Hahn-Banach 将凸集分离推广到无穷维。超平面 \(\{x : f(x) = c\}\) 是"广义的平面"，凸集分离就是 找到这样的平面将两个凸集隔开。

📐 公式推导 / 模型建立

分离定理 (弱形式)：设 \(A, B \subseteq X\) 为非空凸集， \(A \cap B = \emptyset\)，\(A\) 开。则存在 \(f \in X^*\), \(c \in \mathbb{R}\) 使 \[f(a) < c \leq f(b), \quad \forall a \in A,\; b \in B\] 严格分离定理：若 \(A\) 紧, \(B\) 闭, \(A \cap B = \emptyset\)， 则存在 \(f \in X^*\) 和 \(c_1 < c_2\) 使 \[f(a) \leq c_1 < c_2 \leq f(b)\] 支撑超平面定理：闭凸集边界上每一点都存在支撑超平面。 推论：\(X^*\) 分离 \(X\) 中的点——若 \(x \neq y\)，则存在 \(f \in X^*\) 使 \(f(x) \neq f(y)\)。

🌐 实际应用

SVM 最大间隔分类器：数据点分属两个凸集，分离超平面 即 \(\{x : w^T x = b\}\)，最大间隔 = 两超平面间距离的最大化。

✏️ 例题分析与易错点

例题：在 \(\mathbb{R}^2\) 中，\(A = \{(x,y) : x^2 + y^2 < 1\}\)（开单位圆盘）， \(B = \{(2, 0)\}\)。找分离超平面。 解答：\(f(x, y) = x\)，\(c = 3/2\)。\(f(a) < 1 < 3/2 \leq f(b) = 2\)。✓ 实际上最佳分离是 \(x = 1\)（支撑超平面）。 易错点：两个闭凸集不相交不一定能严格分离。 例：\(A = \{(x,y) : y \geq 1/x, x > 0\}\)，\(B = \{(x,0) : x \geq 0\}\)， 不相交但无法严格分离。严格分离需要其中一个紧致。

03Hahn-Banach 的推论

🎯 为什么研究这个问题

Hahn-Banach 定理看似抽象，但其推论极为具体和有用：对偶空间 足以"看见"原空间的每个点和区分不同的元素。这些推论建立了 "对偶方法"——通过研究泛函来推断空间元素的性质——的合法性。

📐 公式推导 / 模型建立

推论 1 (范数达到性)：\(\forall x \in X\), \(\exists f \in X^*\) 使 \(f(x) = \|x\|\), \(\|f\| = 1\)。 证明：在 \(Y = \text{span}\{x\}\) 上定义 \(g(\alpha x) = \alpha \|x\|\)， \(\|g\| = 1\)，由 Hahn-Banach 延拓。 推论 2 (对偶分离点)：若 \(f(x) = f(y)\) 对所有 \(f \in X^*\)，则 \(x = y\)。 即 \(X^*\) 分离 \(X\) 中的点。 推论 3 (范数公式)： \[\|x\| = \sup\{|f(x)| : f \in X^*,\; \|f\| \leq 1\}\] 推论 4 (子空间闭包)：\(x \in \overline{Y}\)（\(Y\) 为子空间） \(\Leftrightarrow\) 对所有 \(f \in X^*\)，\(f|_Y = 0 \Rightarrow f(x) = 0\)。 证明：若 \(x \notin \overline{Y}\)，在 \(\overline{Y} \oplus \text{span}\{x\}\) 上定义 \(f = 0\) on \(\overline{Y}\), \(f(x) = 1\)，延拓即得。

✏️ 例题分析与易错点

例题：在 \(L^1([0,1])\) 中，\(\|f\|_1 = \sup\{|\int fg\, dx| : g \in L^\infty,\, \|g\|_\infty \leq 1\}\)。 验证：取 \(g = \text{sgn}(f)\)，\(\int fg = \int |f| = \|f\|_1\)。✓ 易错点：范数公式中的 sup 不一定能达到（非自反空间中）。 只保证是 sup，不保证是 max。

04对偶理论的基础应用

🎯 为什么研究这个问题

Hahn-Banach 定理和对偶理论不仅是抽象工具，还直接联系到 凸优化中的 Lagrange 对偶和最优化条件。从函数空间的角度看， "原问题"和"对偶问题"之间的间隙由分离定理控制——强对偶性 等价于超平面分离的某种"完美"实现。

📐 公式推导 / 模型建立

对偶对 \((X, X^*)\)：\(X\) 和 \(X^*\) 通过双线性配对 \(\langle x, f \rangle = f(x)\) 关联。 双极定理（陈述）：对 \(A \subseteq X\)，极集 \(A^\circ = \{f \in X^* : |f(x)| \leq 1, \forall x \in A\}\)， \((A^\circ)^\circ = \overline{\text{conv}}(A \cup \{0\})\)（弱闭凸包）。 Goldstine 定理（陈述）：\(X\) 的闭单位球在 \(X^{**}\) 的 弱*拓扑下稠密于 \(X^{**}\) 的闭单位球。 推论：\(X\) 自反 \(\Leftrightarrow\) 闭单位球弱紧。 应用于凸优化：最小化 \(f(x)\) subject to \(x \in C\)（闭凸集）， \(x_0\) 是最优解 \(\Leftrightarrow\) \(0 \in \partial f(x_0) + N_C(x_0)\) （次梯度条件），而法锥 \(N_C(x_0)\) 由支持泛函描述。

🌐 实际应用

SVM 对偶问题通过核技巧将有限维对偶推广到 RKHS（再生核 Hilbert 空间），Hahn-Banach 保证了这种推广的合法性。

✏️ 例题分析与易错点

例题：在 \(\mathbb{R}^n\) 中，\(A = \{x : \|x\|_1 \leq 1\}\)。 求极集 \(A^\circ\)。 解答：\(A^\circ = \{y \in \mathbb{R}^n : |\langle x, y \rangle| \leq 1, \forall \|x\|_1 \leq 1\}\)。 \(\sup_{\|x\|_1 \leq 1} |\langle x, y \rangle| = \|y\|_\infty\) （Hölder 对偶），故 \(A^\circ = \{y : \|y\|_\infty \leq 1\}\)。 \(l^1\) 球的极集是 \(l^\infty\) 球。✓ 易错点：极集在无穷维中更复杂，需注意弱*闭性。
