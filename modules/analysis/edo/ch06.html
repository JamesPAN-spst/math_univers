Chapitre 6

稳定性与 Lyapunov 方法

01稳定性的定义与分类

🎯 为什么研究这个问题

系统受到小扰动后是否会「回来」？这是工程和科学中最基本的问题：

  • 控制系统：卫星姿态受微小推力扰动后能否恢复？自动驾驶遇到侧风后能否保持车道？稳定性决定设计是否可行。
  • 物理平衡：倒立摆是不稳定的（任何扰动都导致倾倒），普通摆是稳定的（小角度扰动后振荡回归）——这种区别需要精确的数学刻画。
  • 不求解的判定：Lyapunov（1892）的革命性思想是直接从方程结构判定稳定性，无需求解——这使得非线性系统的稳定性分析成为可能。
🔍 抽象化方法

稳定性是一种拓扑/定性性质——关心轨线之间的定性关系而非解的具体公式：

  • \(\epsilon\)-\(\delta\) 语言：Lyapunov 稳定性的严格定义直接模仿分析学中的连续性定义——初值在 \(\delta\)-球内 → 解始终在 \(\epsilon\)-球内。
  • 稳定性层次：Lyapunov 稳定 ⊂ 渐近稳定 ⊂ 指数稳定，每一层增加更强的收敛要求。
  • 能量类比：物理系统中小球在碗底是稳定的（势能最小值），碗底的形状决定了稳定性的类型——这个直觉将被 Lyapunov 函数严格化。
📐 公式推导 / 模型建立

1. Lyapunov 稳定

设 \(\mathbf{X}^*\) 为 \(\mathbf{X}' = \mathbf{F}(\mathbf{X})\) 的平衡点（不妨设为原点）。原点称为 Lyapunov 稳定的，若

\[\forall \epsilon > 0,\ \exists \delta > 0:\ \|\mathbf{X}(0)\| < \delta \implies \|\mathbf{X}(t)\| < \epsilon,\quad \forall t \geq 0\]

直觉：初值足够近 → 解永远不会太远。但解不一定趋向平衡点（可能在附近振荡）。

2. 渐近稳定

原点称为渐近稳定的，若 Lyapunov 稳定 且 存在 \(\delta_0 > 0\) 使得

\[\|\mathbf{X}(0)\| < \delta_0 \implies \lim_{t \to \infty}\mathbf{X}(t) = \mathbf{0}\]

注意两个条件缺一不可。仅有 \(\mathbf{X}(t) \to 0\) 但不 Lyapunov 稳定的例子是存在的（尽管不常见）。

3. 指数稳定

原点称为指数稳定的，若存在 \(M > 0\)、\(\alpha > 0\) 使得

\[\|\mathbf{X}(t)\| \leq M\|\mathbf{X}(0)\|e^{-\alpha t},\quad \forall t \geq 0\]

指数衰减是最强的要求。蕴含关系：指数稳定 \(\Rightarrow\) 渐近稳定 \(\Rightarrow\) Lyapunov 稳定。反之均不成立。

4. 不稳定

不满足 Lyapunov 稳定：存在 \(\epsilon_0 > 0\) 使得对任意小的 \(\delta\)，都有初值 \(\|\mathbf{X}(0)\| < \delta\) 的解在某时刻 \(\|\mathbf{X}(t)\| \geq \epsilon_0\)。

🌐 实际应用

卫星姿态控制：自旋稳定卫星绕最大/最小惯量轴旋转是稳定的，绕中间惯量轴旋转是不稳定的（网球定理/Dzhanibekov 效应）。

电力系统：发电机受到短路故障扰动后能否恢复同步运行（暂态稳定性），取决于「临界清除时间」——这直接对应 Lyapunov 分析中的吸引域大小。

生态学：种群平衡的全局渐近稳定性意味着无论初始种群如何，长期都趋向共存状态（持久性 vs 灭绝由稳定类型决定）。

✏️ 例题分析与易错点

例题：渐近稳定但非指数稳定

判断 \(x' = -x^3\) 在 \(x = 0\) 的稳定类型。

第 1 步：求解。分离变量得 \(x(t) = \frac{x_0}{\sqrt{1 + 2x_0^2 t}}\)。

第 2 步：\(x(t) \to 0\) as \(t \to \infty\) → 渐近稳定。

第 3 步：衰减速率分析。\(|x(t)| \sim \frac{1}{\sqrt{2t}}\) 是多项式衰减，不存在 \(\alpha > 0\) 使 \(|x(t)| \leq Me^{-\alpha t}\)。

结论：\(x = 0\) 是渐近稳定但非指数稳定的。

常见易错点

  • ❌ 将渐近稳定等同于指数稳定。非线性系统的衰减可以任意慢（多项式甚至对数衰减），指数稳定是额外的强条件。
  • ❌ 忘记「Lyapunov 稳定」是「渐近稳定」的前提条件。仅有 \(\mathbf{X}(t) \to 0\) 不够——还需要解对初值的连续依赖（不能先跑远再回来）。
  • ❌ 混淆局部与全局渐近稳定。上述定义是局部的（初值足够近）。全局渐近稳定要求对所有初值都趋向平衡点。
⚡ 快速检测

02线性系统的稳定性判据

🎯 为什么研究这个问题

对于线性系统 \(\mathbf{X}' = A\mathbf{X}\)，稳定性有一个完美的判据——完全由矩阵特征值决定：

  • 特征值判据：所有特征值实部为负 \(\Leftrightarrow\) 渐近稳定。这是线性 ODE 理论最优美的结论之一。
  • Routh-Hurwitz：不直接求特征值，通过特征多项式的系数判断所有根是否在左半平面——高阶系统的实用工具。
  • 线性化的基础：由 Hartman-Grobman 定理，非线性系统在双曲平衡点的稳定性由线性化决定。因此线性判据间接适用于非线性系统。
🔍 抽象化方法

线性系统稳定性的核心是谱条件——矩阵的谱（特征值集合）完全决定系统行为：

  • 谱与指数增长：\(e^{tA}\) 的增长由 \(\max_i \mathrm{Re}(\lambda_i)\)（谱横坐标）控制。谱横坐标 < 0 \(\Leftrightarrow\) \(e^{tA} \to 0\)。
  • Jordan 形的角色：幂零部分引起多项式增长 \(t^k\)，但当 \(\mathrm{Re}(\lambda) < 0\) 时被指数衰减 \(e^{\mathrm{Re}(\lambda)t}\) 压制。边界情况（\(\mathrm{Re}(\lambda) = 0\)）时多项式增长决定稳定/不稳定。
  • Hurwitz 矩阵：所有特征值实部为负的矩阵称为 Hurwitz 矩阵，在控制论中有核心地位。
📐 公式推导 / 模型建立

1. 核心定理

常系数线性系统 \(\mathbf{X}' = A\mathbf{X}\)，解 \(\mathbf{X}(t) = e^{tA}\mathbf{X}_0\)。原点是

  • 渐近稳定（也是指数稳定）\(\Leftrightarrow\) \(\mathrm{Re}(\lambda_i) < 0\) 对所有特征值。衰减率 \(\alpha = -\max_i \mathrm{Re}(\lambda_i) > 0\)。
  • Lyapunov 稳定 \(\Leftarrow\) \(\mathrm{Re}(\lambda_i) \leq 0\) 对所有特征值，且纯虚/零特征值的 Jordan 块为 \(1 \times 1\)。
  • 不稳定 \(\Leftarrow\) 存在 \(\mathrm{Re}(\lambda_i) > 0\)，或纯虚/零特征值有 \(> 1\) 的 Jordan 块。

2. 证明要点

Jordan 块 \(J_k(\lambda)\) 对应解含 \(t^j e^{\lambda t}\)（\(j = 0, \ldots, k-1\)）。

  • \(\mathrm{Re}(\lambda) < 0\) 时：\(t^j e^{\mathrm{Re}(\lambda)t} \to 0\)（指数杀死多项式）
  • \(\mathrm{Re}(\lambda) = 0\)，\(k = 1\)：\(e^{i\omega t}\) 有界
  • \(\mathrm{Re}(\lambda) = 0\)，\(k > 1\)：\(t^j e^{i\omega t}\) 无界 → 不稳定

3. Routh-Hurwitz 判据

给定特征多项式 \(p(\lambda) = \lambda^n + a_1\lambda^{n-1} + \cdots + a_n\)，构造 Routh 表。所有特征值实部为负 \(\Leftrightarrow\) Routh 表第一列全为正。

必要条件：所有系数 \(a_i > 0\)（二阶以下也是充分条件）。

4. Lyapunov 方程判据

\(A\) 是 Hurwitz 矩阵 \(\Leftrightarrow\) 对任意正定 \(Q\)，存在唯一正定 \(P\) 满足

\[A^TP + PA = -Q\]

此时 \(V = \mathbf{X}^TP\mathbf{X}\) 是 Lyapunov 函数。这是线性系统 Lyapunov 方法的代数等价。

🌐 实际应用

控制系统设计：状态反馈 \(u = -Kx\) 使闭环矩阵 \(A - BK\) 的特征值全在左半平面（极点配置）。Routh-Hurwitz 判据用于验证设计的正确性。

电路稳定性：RLC 网络的特征多项式系数由电路参数决定。Routh 判据直接给出「电阻/电容/电感的取值范围使电路稳定」。

Nyquist 判据：频域方法，通过开环传递函数的频率响应判断闭环稳定性。与 Routh-Hurwitz 互补——一个用于参数设计，一个用于频率分析。

✏️ 例题分析与易错点

例题：特征值判据与边界情况

矩阵 \(A = \begin{pmatrix} -2 & 1 \\ 0 & -3 \end{pmatrix}\) 的系统是否渐近稳定？

第 1 步：上三角矩阵，特征值为对角元 \(\lambda_1 = -2\)，\(\lambda_2 = -3\)。

第 2 步：\(\mathrm{Re}(\lambda_i) < 0\) 对所有 \(i\) → 渐近稳定（也是指数稳定）。

第 3 步：衰减率 \(\alpha = \min(2, 3) = 2\)，\(\|\mathbf{X}(t)\| \leq Me^{-2t}\|\mathbf{X}(0)\|\)。

边界情况对比

\(A = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}\)：特征值 \(\lambda = \pm i\)（纯虚，\(1 \times 1\) 块） → Lyapunov 稳定但非渐近稳定（中心，解为周期函数）。

常见易错点

  • ❌ 将「所有特征值实部 \(\leq 0\)」等同于渐近稳定。\(\leq 0\) 只保证 Lyapunov 稳定（在 Jordan 块条件满足时），渐近稳定要求严格< 0。
  • ❌ 对纯虚特征值忘记检查 Jordan 块大小。\(\lambda = 0\) 是 \(2 \times 2\) Jordan 块时系统不稳定（解含 \(t\) 项）。
  • ❌ Routh-Hurwitz 中误以为「所有系数为正」就足够。三阶及以上还需检查 Routh 表的附加条件。
⚡ 快速检测
\(A\) 的特征值为 \(-1\) 和 \(0\)（\(0\) 是 \(1 \times 1\) Jordan 块），系统是？

03Lyapunov 直接法

🎯 为什么研究这个问题

非线性系统通常无法求解，但 Lyapunov 直接法允许我们不求解方程就判断稳定性：

  • 能量函数思想：构造一个「广义能量」\(V(\mathbf{X})\)。如果 \(V\) 沿轨线单调递减，系统就在「耗散能量」，必然趋向平衡点。
  • 通用性：适用于任意非线性系统、变系数系统、甚至无穷维系统（PDE）。是 ODE 稳定性理论中最重要、最通用的方法。
  • 设计工具：Lyapunov 方法不仅用于分析，还可以反向使用——先选 \(V\)，再设计控制使 \(\dot{V} < 0\)（Lyapunov 重构设计/backstepping）。
🔍 抽象化方法

Lyapunov 函数是广义能量，将稳定性从轨线分析转化为标量函数分析：

  • 正定性：\(V(\mathbf{0}) = 0\)，\(V(\mathbf{X}) > 0\) 对 \(\mathbf{X} \neq \mathbf{0}\)——\(V\) 的等值面是「嵌套碗」。
  • 导数的物理意义：\(\dot{V} = \nabla V \cdot \mathbf{F}\) 是 \(V\) 沿系统流的变化率。\(\dot{V} < 0\) 意味着轨线穿越等值面的方向是向内的。
  • 困难：没有通用的 Lyapunov 函数构造方法——这是一门「艺术」。但对线性系统有系统的代数方法（Lyapunov 方程）。
📐 公式推导 / 模型建立

1. Lyapunov 函数与轨线导数

\(V: U \to \mathbb{R}\)，\(V(\mathbf{0}) = 0\)，\(V(\mathbf{X}) > 0\) 对 \(\mathbf{X} \neq \mathbf{0}\)（正定）。沿系统 \(\mathbf{X}' = \mathbf{F}(\mathbf{X})\) 的导数：

\[\dot{V}(\mathbf{X}) = \nabla V(\mathbf{X}) \cdot \mathbf{F}(\mathbf{X}) = \sum_{i=1}^n \frac{\partial V}{\partial x_i}F_i(\mathbf{X})\]

注意 \(\dot{V}\) 的计算不需要求解 ODE——直接代入 \(\mathbf{F}\) 即可。

2. Lyapunov 稳定性定理

若存在正定 \(V\) 使得 \(\dot{V}(\mathbf{X}) \leq 0\)（负半定），则原点 Lyapunov 稳定。

证明思路：\(V\) 沿轨线不增 → \(V(\mathbf{X}(t)) \leq V(\mathbf{X}(0)) < V_\epsilon\) → \(\mathbf{X}(t)\) 留在 \(\{V < V_\epsilon\} \subset B_\epsilon\)。

3. 渐近稳定性定理

若 \(V\) 正定且 \(\dot{V}(\mathbf{X}) < 0\) 对所有 \(\mathbf{X} \neq \mathbf{0}\)（负定），则原点渐近稳定。

4. 线性系统的 Lyapunov 方程

对 \(\mathbf{X}' = A\mathbf{X}\)，取 \(V = \mathbf{X}^TP\mathbf{X}\)（\(P\) 正定对称），则

\[\dot{V} = \mathbf{X}^T(A^TP + PA)\mathbf{X} = -\mathbf{X}^TQ\mathbf{X}\]

其中 \(Q = -(A^TP + PA)\)。渐近稳定 \(\Leftrightarrow\) 对任意正定 \(Q\)，Lyapunov 方程 \(A^TP + PA = -Q\) 有唯一正定解 \(P\)。

🌐 实际应用

机器人控制：PD 控制器 \(\tau = -K_p q - K_d \dot{q}\) 的稳定性通过 Lyapunov 函数 \(V = \frac{1}{2}\dot{q}^TM(q)\dot{q} + \frac{1}{2}q^TK_p q\)（动能+势能）证明。\(\dot{V} = -\dot{q}^TK_d\dot{q} \leq 0\)（负半定），再用 LaSalle 原理得渐近稳定。

电力系统：暂态稳定性分析中，「等面积法则」本质上是一维 Lyapunov 函数方法。多机系统使用构造性能量函数估计临界清除角。

神经网络：Hopfield 网络的收敛性通过 Lyapunov 函数 \(V = -\frac{1}{2}x^TWx + \sum \int g^{-1}(x_i)dx_i\) 证明，保证能量单调下降到局部最小值。

✏️ 例题分析与易错点

例题：Lyapunov 函数构造与验证

用 Lyapunov 方法证明 \(x' = -x + y^2\)，\(y' = -y\) 在原点渐近稳定。

第 1 步：取 \(V = x^2 + y^2\)（正定）。

第 2 步：计算 \(\dot{V} = 2x(-x + y^2) + 2y(-y) = -2x^2 + 2xy^2 - 2y^2\)。

第 3 步：在小邻域 \(x^2 + y^2 \leq r^2\) 内估计交叉项。由 \(|y^2| \leq r^2\) 和 \(|2xy^2| \leq 2|x| \cdot r^2 \leq r(x^2 + r^2)\)，当 \(r\) 足够小时：

\[\dot{V} \leq -2x^2 + r(x^2 + r^2) - 2y^2 \leq -(2-r)(x^2 + y^2) < 0\]

结论：在 \(r < 2\) 的邻域内 \(\dot{V}\) 负定 → 渐近稳定。

常见易错点

  • ❌ \(\dot{V} \leq 0\)（负半定）只能得 Lyapunov 稳定，不能直接得渐近稳定。需要 \(\dot{V} < 0\)（负定）或使用 LaSalle 不变性原理。
  • ❌ 忘记验证 \(V\) 的正定性。\(V = x^2 - y^2\) 不是正定的（在 \(y\) 轴上为负），不能作为 Lyapunov 函数。
  • ❌ Lyapunov 方法给出充分条件而非必要条件。找不到 Lyapunov 函数不代表系统不稳定——可能只是 \(V\) 没选好。
⚡ 快速检测

04Chetaev 不稳定定理与 LaSalle 不变性原理

🎯 为什么研究这个问题

Lyapunov 直接法有两个常见困难，需要补充工具：

  • 不稳定性判定：Lyapunov 方法给出稳定的充分条件，但如何证明不稳定？Chetaev 定理提供了系统性的方法——构造一个在某区域递增的函数。
  • \(\dot{V}\) 负半定的困境：许多物理系统的耗散只在部分方向发生（如摩擦力只作用于速度方向），导致 \(\dot{V}\) 仅为负半定。LaSalle 不变性原理放宽了负定的要求，仍能得出渐近稳定。
  • 工程实用性：LaSalle 原理是控制工程中最常用的稳定性证明工具——几乎所有带阻尼的机械/电气系统都需要它。
🔍 抽象化方法
  • Chetaev 的对偶思想：如果 Lyapunov 方法是「找递减的正定函数证稳定」，Chetaev 方法就是「找递增的函数证不稳定」——轨线被强制推离平衡点。
  • LaSalle 的精炼：\(\dot{V} \leq 0\) 保证 \(V\) 单调下降且有下界(0)，故 \(V(\mathbf{X}(t)) \to c\)。轨线的极限集必须在 \(\dot{V} = 0\) 的集合上——再排除非平衡点即可。
  • 正不变集的核心角色：LaSalle 原理的关键是「\(\dot{V} = 0\) 中最大的正不变集 \(M\)」——轨线不是趋向 \(\dot{V} = 0\) 的所有点，而是只趋向那些可以一直停留的点。
📐 公式推导 / 模型建立

1. Chetaev 不稳定定理

设 \(V\) 连续，\(V(\mathbf{0}) = 0\)。若存在域 \(D\)（\(\mathbf{0} \in \partial D\)）使得在 \(D \cap B_\epsilon(\mathbf{0})\) 中：

  • \(V(\mathbf{X}) > 0\)（函数正值）
  • \(\dot{V}(\mathbf{X}) > 0\)（函数递增）

则原点不稳定。

直觉：在 \(D\) 中 \(V\) 持续增大，轨线不能回到 \(V = 0\) 的边界（即原点附近），因此被推向远方。

2. LaSalle 不变性原理

设 \(\Omega \subset \mathbb{R}^n\) 为正不变紧集（轨线从 \(\Omega\) 出发不离开），\(V \in C^1\) 在 \(\Omega\) 上满足 \(\dot{V} \leq 0\)。定义

\[E = \{\mathbf{X} \in \Omega : \dot{V}(\mathbf{X}) = 0\}\]

设 \(M\) 为 \(E\) 中最大的正不变集。则 \(\Omega\) 中的每条轨线都趋向 \(M\)：

\[\omega(\mathbf{X}_0) \subseteq M,\quad \forall \mathbf{X}_0 \in \Omega\]

3. 证明思路

  • \(\dot{V} \leq 0\) → \(V(\mathbf{X}(t))\) 单调不增且有下界 → \(V(\mathbf{X}(t)) \to c\) 存在
  • \(\omega\)-极限集 \(\omega(\mathbf{X}_0)\) 是紧不变集，在其上 \(V \equiv c\) → \(\dot{V} \equiv 0\) → \(\omega(\mathbf{X}_0) \subset E\)
  • 又 \(\omega(\mathbf{X}_0)\) 是不变集 → \(\omega(\mathbf{X}_0) \subset M\)

4. 经典应用：阻尼摆

\(\theta'' + c\theta' + \sin\theta = 0\)（\(c > 0\)）。取 \(V = \frac{1}{2}(\theta')^2 + (1 - \cos\theta)\)（动能+势能），则

\[\dot{V} = \theta'(\theta'' + \sin\theta) = -c(\theta')^2 \leq 0\]

\(\dot{V}\) 仅负半定（\(\theta' = 0\) 时为零）。\(E = \{\theta' = 0\}\)。在 \(E\) 中：\(\theta' = 0\) → \(\theta'' = 0\) → \(\sin\theta = 0\) → \(\theta = 0\)（在小邻域内）。故 \(M = \{(0,0)\}\)，LaSalle 原理 → 渐近稳定。

🌐 实际应用

机器人 PD 控制：机器人方程 \(M(q)\ddot{q} + C(q,\dot{q})\dot{q} + g(q) = \tau\)，PD 控制 \(\tau = g(q) - K_p(q - q_d) - K_d\dot{q}\)。Lyapunov 函数 \(V = \frac{1}{2}\dot{q}^TM\dot{q} + \frac{1}{2}(q-q_d)^TK_p(q-q_d)\)，\(\dot{V} = -\dot{q}^TK_d\dot{q} \leq 0\)（仅负半定）。LaSalle 原理证明 \((q, \dot{q}) \to (q_d, 0)\)。

能量耗散系统：所有含粘性阻尼的结构（建筑、桥梁、机翼）的振动衰减都需要 LaSalle 原理——因为阻尼力 \(-c\dot{x}\) 只在 \(\dot{x} \neq 0\) 时做功。

Chetaev 应用：证明倒立摆上端平衡点的不稳定性。取 \(V = \theta\theta'\)，在适当区域内 \(\dot{V} > 0\)，由 Chetaev 定理得不稳定。

✏️ 例题分析与易错点

例题：LaSalle 不变性原理

对 \(x' = y\)，\(y' = -x - y^3\)，用 LaSalle 原理证明原点渐近稳定。

第 1 步：取 \(V = \frac{1}{2}(x^2 + y^2)\)（正定）。

第 2 步：\(\dot{V} = xy + y(-x - y^3) = -y^4 \leq 0\)（仅负半定）。

第 3 步：定义 \(E = \{(x,y): y = 0\}\)。在 \(E\) 中要求轨线始终满足 \(y(t) \equiv 0\)：

  • \(y \equiv 0 \Rightarrow y' \equiv 0 \Rightarrow -x - 0 = 0 \Rightarrow x = 0\)

第 4 步：\(E\) 中最大正不变集 \(M = \{(0,0)\}\)。由 LaSalle 原理，所有轨线趋向原点 → 渐近稳定。

常见易错点

  • ❌ LaSalle 原理需要正不变紧集 \(\Omega\)。对于无界系统，需先论证轨线有界（通常利用 \(V\) 的水平集 \(\{V \leq c\}\) 是紧的）。
  • ❌ 混淆 \(E = \{\dot{V} = 0\}\) 和 \(M =\) \(E\) 中最大正不变集。轨线趋向的是 \(M\) 而非 \(E\)。例如 \(E\) 可能是一条直线，但 \(M\) 只是其中的原点。
  • ❌ 在 LaSalle 分析中忘记将方程代入检查正不变性。仅令 \(y = 0\) 不够——还需检查 \(y = 0\) 是否在系统动力学下自洽。
