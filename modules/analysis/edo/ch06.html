稳定性与 Lyapunov 方法

01稳定性的定义与分类

🎯 为什么研究这个问题

工程师设计控制系统、物理学家分析力学平衡时，最关心的问题是： 系统受到小扰动后是否会「回来」？Lyapunov 在 1892 年给出了 严格的数学定义：稳定、渐近稳定、不稳定。这些概念不依赖具体 求解方程，而是直接刻画解的长期行为。

🔍 抽象化方法

稳定性是一种「拓扑」性质：它关心的是轨线之间的定性关系（靠近/远离）， 而非解的具体公式。这种观点使得我们可以在不求解方程的情况下 判断稳定性（Lyapunov 直接法）。

📐 公式推导 / 模型建立

设 \(\mathbf{X}^*\) 为 \(\mathbf{X}' = \mathbf{F}(\mathbf{X})\) 的平衡点（不妨设为原点）。 Lyapunov 稳定：\(\forall \epsilon > 0\)，\(\exists \delta > 0\)， \(\|\mathbf{X}(0)\| < \delta \Rightarrow \|\mathbf{X}(t)\| < \epsilon\) 对所有 \(t \geq 0\)。 渐近稳定：Lyapunov 稳定 + \(\lim_{t \to \infty}\mathbf{X}(t) = \mathbf{0}\) （在某邻域内的初值）。 指数稳定：存在 \(M, \alpha > 0\) 使得 \[\|\mathbf{X}(t)\| \leq M\|\mathbf{X}(0)\|e^{-\alpha t}\] 指数稳定 \(\Rightarrow\) 渐近稳定 \(\Rightarrow\) Lyapunov 稳定。反之不成立。 不稳定：不满足 Lyapunov 稳定的条件，即存在任意小的扰动 导致轨线离开某 \(\epsilon\)-邻域。

🌐 实际应用

- 卫星姿态控制中的稳定性要求 - 电力系统暂态稳定性分析

✏️ 例题分析与易错点

例题：判断 \(x' = -x^3\) 在 \(x=0\) 的稳定类型。 解答：解为 \(x(t) = x_0/\sqrt{1+2x_0^2 t}\)。\(x(t) \to 0\)（渐近稳定）。 衰减速率：\(|x(t)| \sim 1/\sqrt{2t}\)（多项式衰减，非指数衰减）。 故 \(x=0\) 是渐近稳定但非指数稳定的。 易错点：渐近稳定不等于指数稳定。指数稳定要求衰减速度至少为 \(e^{-\alpha t}\)，多项式衰减（如上例）不够快。

02线性系统的稳定性判据

🎯 为什么研究这个问题

对于线性系统 \(\mathbf{X}' = A\mathbf{X}\)，稳定性完全由矩阵 \(A\) 的 特征值决定：所有特征值实部为负 \(\Leftrightarrow\) 渐近稳定。这个 优美的结果是线性 ODE 理论的最终结论，也是控制理论（Routh-Hurwitz 判据）的数学基础。

📐 公式推导 / 模型建立

常系数线性系统 \(\mathbf{X}' = A\mathbf{X}\)： 解为 \(\mathbf{X}(t) = e^{tA}\mathbf{X}_0\)。\(e^{tA}\) 的行为由 \(A\) 的 Jordan 形决定。 定理：\(\mathbf{X} = \mathbf{0}\) 是 - 渐近稳定 \(\Leftrightarrow\) \(\text{Re}(\lambda_i) < 0\) 对所有特征值 （此时也是指数稳定的，\(\alpha = -\max_i \text{Re}(\lambda_i)\)） - Lyapunov 稳定 \(\Leftarrow\) \(\text{Re}(\lambda_i) \leq 0\) 且纯虚特征值 对应的 Jordan 块为 \(1 \times 1\) - 不稳定 \(\Leftarrow\) 存在 \(\text{Re}(\lambda_i) > 0\)，或纯虚特征值 有 \(>1\) 的 Jordan 块 证明关键：Jordan 块 \(J_k(\lambda)\) 对应解含 \(t^j e^{\lambda t}\)（\(j = 0,\ldots,k-1\)）。 当 \(\text{Re}(\lambda) < 0\) 时 \(t^j e^{\lambda t} \to 0\)。 当 \(\text{Re}(\lambda) = 0\) 且 \(k > 1\) 时 \(t^j e^{i\omega t}\) 无界。

🌐 实际应用

- Routh-Hurwitz 判据：不直接算特征值，通过系数判断所有特征值 实部是否为负（用于高阶控制系统设计）

✏️ 例题分析与易错点

例题：矩阵 \(A = \begin{pmatrix} -2 & 1 \\ 0 & -3 \end{pmatrix}\) 的系统是否渐近稳定？ 解答：上三角矩阵特征值为对角元 \(\lambda_1=-2\)，\(\lambda_2=-3\)。 \(\text{Re}(\lambda_i) < 0\) 对所有 \(i\)，故渐近稳定（也是指数稳定的）。 衰减率 \(\alpha = \min(2,3) = 2\)：\(\|\mathbf{X}(t)\| \leq Me^{-2t}\)。 易错点：\(A = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}\)（特征值 \(\pm i\)） 是 Lyapunov 稳定但非渐近稳定（中心，解为周期函数）。

⚡ 快速检测
\(A\) 的特征值为 \(-1\) 和 \(0\)（\(0\) 是 \(1 \times 1\) Jordan 块），系统是？

03Lyapunov 直接法

🎯 为什么研究这个问题

当方程无法求解时（非线性系统），如何判断稳定性？Lyapunov 的 天才想法是：构造一个「能量函数」\(V(\mathbf{X})\)，如果它沿轨线 单调递减，系统就是稳定的——无需求解方程。这是 ODE 稳定性理论 中最重要的方法，也是控制理论中 Lyapunov 设计的基础。

🔍 抽象化方法

Lyapunov 函数是广义的「能量」：正定函数 \(V > 0\)（除原点外）， 其沿系统轨线的「耗散率」\(\dot{V} = \nabla V \cdot \mathbf{F}\) 反映系统 是否在「消耗能量」。这将稳定性从轨线分析转化为标量函数分析。

📐 公式推导 / 模型建立

Lyapunov 函数：\(V: U \to \mathbb{R}\) 在原点邻域上满足： - \(V(\mathbf{0}) = 0\)，\(V(\mathbf{X}) > 0\) 对 \(\mathbf{X} \neq \mathbf{0}\)（正定） 沿系统轨线的导数： \[\dot{V}(\mathbf{X}) = \nabla V(\mathbf{X}) \cdot \mathbf{F}(\mathbf{X}) = \sum_i \frac{\partial V}{\partial x_i} F_i(\mathbf{X})\] Lyapunov 稳定性定理：若存在正定 \(V\) 使得 \(\dot{V} \leq 0\)（负半定）， 则原点 Lyapunov 稳定。 渐近稳定性定理：若 \(V\) 正定且 \(\dot{V}\) 负定（\(\dot{V} < 0\) 对 \(\mathbf{X} \neq \mathbf{0}\)）， 则原点渐近稳定。 常见构造：对于线性系统 \(\mathbf{X}' = A\mathbf{X}\)，取 \(V = \mathbf{X}^T P\mathbf{X}\) （\(P\) 正定对称矩阵），则 \(\dot{V} = \mathbf{X}^T(A^T P + PA)\mathbf{X}\)。 \(A^T P + PA = -Q\)（\(Q\) 正定）时渐近稳定。这是 Lyapunov 方程。

🌐 实际应用

- 机器人控制中的 Lyapunov 稳定性设计 - 电力系统暂态稳定性分析中的能量函数构造

✏️ 例题分析与易错点

例题：用 Lyapunov 方法证明 \(x' = -x + y^2\)，\(y' = -y\) 在原点渐近稳定。 解答：取 \(V = x^2 + y^2\)（正定）。 \(\dot{V} = 2x(-x+y^2) + 2y(-y) = -2x^2 + 2xy^2 - 2y^2\)。 在 \(x^2+y^2 \leq r^2\) 小邻域内：\(|2xy^2| \leq 2r(x^2+y^2)/2\)（由 AM-GM）， 故当 \(r\) 足够小时 \(\dot{V} \leq -(2-r)(x^2+y^2) < 0\)。渐近稳定。 易错点：\(\dot{V} \leq 0\)（负半定）只能得出 Lyapunov 稳定， 要得到渐近稳定需要 \(\dot{V} < 0\)（负定）或使用 LaSalle 不变性原理。

04Chetaev 不稳定定理与 LaSalle 不变性原理

🎯 为什么研究这个问题

Lyapunov 直接法的两个重要补充：Chetaev 定理处理不稳定性 （构造一个在某区域递增的函数），LaSalle 不变性原理放宽了 \(\dot{V}\) 负定的要求——只需负半定加上一个不变集条件。 LaSalle 原理在工程中极为实用，因为许多物理系统的耗散恰好 只在部分方向上发生（如摩擦力只作用于速度方向）。

📐 公式推导 / 模型建立

Chetaev 不稳定定理：设 \(V\) 连续，\(V(\mathbf{0}) = 0\)。若存在域 \(D\) 使得在 \(D \cap B_\epsilon(\mathbf{0})\) 中 \(V > 0\) 且 \(\dot{V} > 0\)， 且 \(\mathbf{0} \in \partial D\)，则原点不稳定。 直觉：\(V\) 在 \(D\) 内持续增大，轨线不能回到原点附近。 LaSalle 不变性原理：设 \(\Omega\) 为正不变紧集，\(V\) 在 \(\Omega\) 上满足 \(\dot{V} \leq 0\)。令 \(E = \{\mathbf{X} \in \Omega : \dot{V}(\mathbf{X}) = 0\}\)， \(M\) 为 \(E\) 中最大的正不变集。则 \(\Omega\) 中的每条轨线趋向 \(M\)。 \[\omega(\mathbf{X}_0) \subseteq M \quad\text{for all } \mathbf{X}_0 \in \Omega\] 典型应用（阻尼摆）：\(\theta'' + c\theta' + \sin\theta = 0\)。 \(V = \frac{1}{2}(\theta')^2 + (1-\cos\theta)\)，\(\dot{V} = -c(\theta')^2 \leq 0\)（仅负半定）。 \(E = \{\theta' = 0\}\)，\(M\) 中 \(\dot{V} = 0\) 且 \(\theta'' = 0\) ⇒ \(\sin\theta = 0\) ⇒ \(\theta = 0\)。 LaSalle 原理 ⇒ \((\theta,\theta') \to (0,0)\)，渐近稳定。

🌐 实际应用

- 机器人控制中，PD 控制器的渐近稳定性证明（利用 LaSalle 原理） - 能量耗散系统（阻尼振动、电阻电路）的长期行为分析

✏️ 例题分析与易错点

例题：对 \(x' = y\)，\(y' = -x - y^3\)，用 LaSalle 原理证明原点渐近稳定。 解答：\(V = \frac{1}{2}(x^2 + y^2)\)，\(\dot{V} = xy + y(-x-y^3) = -y^4 \leq 0\)。 \(\dot{V}\) 仅负半定（\(y=0\) 时为零）。 \(E = \{y = 0\}\)。在 \(E\) 中：\(y=0 \Rightarrow y'=0 \Rightarrow -x=0 \Rightarrow x=0\)。 \(M = \{(0,0)\}\)。LaSalle 原理 ⇒ 所有轨线趋向原点，渐近稳定。 易错点：LaSalle 原理需要正不变紧集 \(\Omega\)。 对于无界系统需先论证轨线有界（通常用 \(V\) 的水平集是紧的）。
