01自治系统与平衡点

🎯 为什么研究这个问题

当方程不显含时间 \(t\)（自治系统 \(\mathbf{X}' = \mathbf{F}(\mathbf{X})\)）时， 解的行为可以在相空间中几何化地理解。平衡点是系统长期行为的 「锚点」——行星轨道、化学平衡、种群稳态都对应平衡点。 理解平衡点的类型（稳定/不稳定/鞍点）是分析非线性系统的第一步。

🔍 抽象化方法

相空间的观点将动力系统从「时间的函数」转化为「空间中的流」。 平衡点的线性化通过 Jacobian 矩阵将非线性问题局部化为线性代数问题。

📐 公式推导 / 模型建立

自治系统：\(\mathbf{X}' = \mathbf{F}(\mathbf{X})\)，\(\mathbf{F}: \mathbb{R}^n \to \mathbb{R}^n\)，不显含 \(t\)。 平衡点：\(\mathbf{X}^*\) 满足 \(\mathbf{F}(\mathbf{X}^*) = \mathbf{0}\)。 线性化：在 \(\mathbf{X}^*\) 附近令 \(\mathbf{u} = \mathbf{X} - \mathbf{X}^*\)，Taylor 展开： \[\mathbf{u}' = \mathbf{F}(\mathbf{X}^* + \mathbf{u}) \approx D\mathbf{F}(\mathbf{X}^*)\mathbf{u} = A\mathbf{u}\] 其中 \(A = D\mathbf{F}(\mathbf{X}^*) = \left(\frac{\partial F_i}{\partial x_j}\right)\) 是 Jacobian 矩阵。 分类（二维 \(A = \begin{pmatrix} a & b \\ c & d \end{pmatrix}\) 的特征值 \(\lambda_{1,2}\)）： 由 \(\text{tr}(A) = \lambda_1 + \lambda_2\) 和 \(\det(A) = \lambda_1 \lambda_2\) 决定。 在 \((\text{tr}, \det)\) 平面上可划分各种平衡点类型。

🌐 实际应用

- Lotka-Volterra 捕食者-猎物模型的共存平衡点分析 - 化学反应网络中的稳态浓度

✏️ 例题分析与易错点

例题：求系统 \(x' = x(1-y)\)，\(y' = y(x-1)\) 的平衡点和 Jacobian。 解答：\(\mathbf{F} = 0\)：\(x(1-y)=0\) 且 \(y(x-1)=0\)。 平衡点：\((0,0)\) 和 \((1,1)\)。 \(A = \begin{pmatrix} 1-y & -x \\ y & x-1 \end{pmatrix}\)。 在 \((1,1)\)：\(A = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}\)， 特征值 \(\lambda = \pm i\)（中心）。这是 Lotka-Volterra 模型的特征。 易错点：线性化在中心情况下不能确定非线性系统的定性行为 （中心可能变为焦点），需要更精细的分析。

⚡ 快速检测

02二维线性系统的分类与相图

🎯 为什么研究这个问题

二维线性系统 \(\mathbf{X}' = A\mathbf{X}\) 的所有可能行为可以通过 \(A\) 的特征值完全分类。这种分类产生了经典的相图库——鞍点、结点、 焦点、中心——它们不仅描述线性系统，也（通过 Hartman-Grobman 定理） 刻画非线性系统的局部行为。相图是直觉理解动力系统的最强工具。

📐 公式推导 / 模型建立

设 \(A\) 的特征值为 \(\lambda_1, \lambda_2\)，令 \(\tau = \text{tr}(A)\)，\(\Delta = \det(A)\)， 判别式 \(D = \tau^2 - 4\Delta\)。 分类： 1. 鞍点 (\(\Delta < 0\))：\(\lambda_1 > 0 > \lambda_2\)，不稳定 2. 稳定结点 (\(\Delta > 0\), \(\tau < 0\), \(D \geq 0\))：\(\lambda_1, \lambda_2 < 0\)， 轨线沿特征方向趋向原点 3. 不稳定结点 (\(\Delta > 0\), \(\tau > 0\), \(D \geq 0\))：\(\lambda_1, \lambda_2 > 0\) 4. 稳定焦点 (\(\Delta > 0\), \(\tau < 0\), \(D < 0\))：\(\lambda = \alpha \pm i\beta\), \(\alpha < 0\)， 轨线螺旋趋向原点 5. 不稳定焦点 (\(\tau > 0\), \(D < 0\))：螺旋远离原点 6. 中心 (\(\tau = 0\), \(\Delta > 0\))：纯虚特征值，封闭轨道 相图绘制：特征向量确定直线方向（结点/鞍点）， \(\text{Re}(\lambda)\) 的符号确定螺旋方向（焦点）， \(\text{Im}(\lambda)\) 确定旋转方向。

🌐 实际应用

- 工程中的阻尼系统设计：过阻尼(结点)、临界阻尼(退化结点)、 欠阻尼(焦点)的选择取决于应用需求

✏️ 例题分析与易错点

例题：对 \(A = \begin{pmatrix} -1 & 2 \\ -2 & -1 \end{pmatrix}\) 画出相图类型。 解答：\(\tau = -2 < 0\)，\(\Delta = 1+4 = 5 > 0\)，\(D = 4-20 = -16 < 0\)。 特征值 \(\lambda = -1 \pm 2i\)。\(\text{Re}(\lambda) < 0\)，\(\text{Im}(\lambda) \neq 0\)。 类型：稳定焦点。轨线以衰减的螺旋趋向原点。 易错点：混淆判别式 \(D = \tau^2 - 4\Delta\) 与行列式 \(\Delta\)。 \(D < 0\) 表示复特征值（焦点/中心），\(\Delta < 0\) 表示鞍点。

⚡ 快速检测
若线性系统 \(\mathbf{X}' = A\mathbf{X}\) 的特征值为 \(-3 \pm 4i\)，相图是？

03Hartman-Grobman 定理与非线性局部行为

🎯 为什么研究这个问题

Hartman-Grobman 定理是非线性动力系统理论的基石：它保证在 「双曲」平衡点（特征值实部非零）附近，非线性系统的相图与其 线性化系统拓扑等价。这意味着线性分析的结论可以推广到非线性 情形——但有一个重要例外：当特征值有零实部时（如中心）， 线性化不能确定非线性行为。

🔍 抽象化方法

Hartman-Grobman 定理建立的是「拓扑等价」而非「微分等价」： 存在保持轨线结构的同胚映射，但不一定光滑。这个区分很关键， 因为轨线的渐近方向和稳定性被保持，但曲率等量化信息可能改变。

📐 公式推导 / 模型建立

双曲平衡点：\(\mathbf{X}^*\) 称为双曲的，若 Jacobian \(A = D\mathbf{F}(\mathbf{X}^*)\) 的所有特征值的实部非零，即 \(\text{Re}(\lambda_i) \neq 0\) 对所有 \(i\)。 Hartman-Grobman 定理：若 \(\mathbf{X}^*\) 是双曲平衡点，则存在 \(\mathbf{X}^*\) 的邻域 \(U\) 和原点的邻域 \(V\)，以及同胚 \(h: U \to V\)， 使得非线性流 \(\phi_t\) 与线性流 \(e^{tA}\) 满足 \[h \circ \phi_t = e^{tA} \circ h\] 即轨线的拓扑结构（稳定性类型、轨线走向）完全由线性化决定。 非双曲情况的反例：\(x' = -y + x(x^2+y^2)\)，\(y' = x + y(x^2+y^2)\)。 线性化 \(A = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}\) 给出中心（纯虚特征值）， 但非线性系统的原点是不稳定焦点（轨线螺旋远离）。

🌐 实际应用

- 生态学中判断种群共存平衡点的稳定性类型 - 非线性控制系统的局部稳定性分析

✏️ 例题分析与易错点

例题：判断系统 \(x' = -x + y^2\)，\(y' = -2y\) 在原点附近的定性行为。 解答：\(\mathbf{F}(0,0) = (0,0)\)，Jacobian \(A = \begin{pmatrix} -1 & 0 \\ 0 & -2 \end{pmatrix}\)。 特征值 \(\lambda_1 = -1\)，\(\lambda_2 = -2\)，均为负实数（双曲）。 线性化是稳定结点。由 Hartman-Grobman 定理，非线性系统在原点附近 也呈稳定结点行为。 易错点：特征值为纯虚时（\(\text{Re}(\lambda) = 0\)），Hartman-Grobman 不适用。此时中心可能变为焦点，必须用其他方法分析。

04Poincaré-Bendixson 定理与极限环

🎯 为什么研究这个问题

在二维自治系统中，轨线除了趋向平衡点外，还可能趋向周期轨道 （极限环）。Poincaré-Bendixson 定理刻画了这种行为：如果轨线留在 有界区域内且不趋向平衡点，就必须趋近周期轨道。这个定理是二维 独有的（三维起可以出现混沌），也是排除/确认极限环存在的基本工具。

🔍 抽象化方法

Poincaré-Bendixson 反映了二维的拓扑限制：Jordan 曲线定理使得 二维封闭曲线将平面分为内外两部分，轨线无法「穿越」自身， 长期行为只能是平衡点或周期轨道，没有其他可能。

📐 公式推导 / 模型建立

Poincaré-Bendixson 定理：设 \(\omega(\mathbf{X}_0)\) 为轨线经过 \(\mathbf{X}_0\) 的 \(\omega\)-极限集。若 \(\omega(\mathbf{X}_0)\) 非空、有界、且不包含平衡点， 则 \(\omega(\mathbf{X}_0)\) 是一个周期轨道（极限环）。 实际应用框架：找一个正不变区域 \(D\)（轨线从 \(D\) 内不逃出）， 排除 \(D\) 内无平衡点 → 必有极限环。 Dulac 判别法（排除极限环）：若存在 \(B(x,y) \in C^1\) 使得 \[\frac{\partial(BF_1)}{\partial x} + \frac{\partial(BF_2)}{\partial y}\] 在单连通域 \(D\) 上不变号（且不恒为零），则 \(D\) 内无周期轨道。 理由：对任何封闭轨线 \(\gamma\)，Green 定理给出 \(\oint_\gamma B\mathbf{F} \cdot \mathbf{n}\,ds = \iint \text{div}(B\mathbf{F})\,dA \neq 0\)，矛盾。 Bendixson 负判据：\(B \equiv 1\) 的特例， 若 \(\text{div}\,\mathbf{F} = \partial F_1/\partial x + \partial F_2/\partial y\) 不变号则无极限环。

🌐 实际应用

- Van der Pol 振荡器 \(x'' - \mu(1-x^2)x' + x = 0\) 的极限环存在性 - 生物节律模型（心脏起搏、昼夜节律）中的自持振荡

✏️ 例题分析与易错点

例题：证明 \(x' = y + x(1-x^2-y^2)\)，\(y' = -x + y(1-x^2-y^2)\) 有极限环。 解答：极坐标 \(r^2 = x^2+y^2\)：\(rr' = xx'+yy'= r^2(1-r^2)\)， \(r' = r(1-r^2)\)。环形区域 \(\frac{1}{2} \leq r \leq 2\) 中： \(r = 1/2\) 时 \(r' = 1/2 \cdot 3/4 > 0\)（向外），\(r = 2\) 时 \(r' = 2(-3) < 0\)（向内）。 此区域正不变，无平衡点。Poincaré-Bendixson → 极限环 \(r = 1\)。 易错点：Poincaré-Bendixson 仅适用于 \(\mathbb{R}^2\)。 三维系统中有界轨线可以有混沌行为（如 Lorenz 系统）。
