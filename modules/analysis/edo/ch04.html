线性微分方程组

01线性方程组一般理论与基解矩阵

🎯 为什么研究这个问题

线性方程组是 ODE 理论的核心框架，原因有三：

  • 高阶化为一阶：任何 \(n\) 阶标量方程 \(y^{(n)} + p_{n-1}y^{(n-1)} + \cdots + p_0 y = g\) 都可以令 \(x_i = y^{(i-1)}\) 改写为一阶系统 \(\mathbf{X}' = A\mathbf{X} + \mathbf{B}\)。
  • 多自由度系统：耦合弹簧、电路网络、多物种生态系统天然以方程组形式出现。
  • 矩阵语言的威力：将 \(n\) 个标量方程打包为 \(\mathbf{X}' = A(t)\mathbf{X}\)，可以直接利用线性代数的全套工具（特征值、对角化、Jordan 形）来分析解的结构。
🔍 抽象化方法

方程组的解空间理论与标量情况完全平行，只是用矩阵语言重述：

  • 解空间：\(\mathbf{X}' = A(t)\mathbf{X}\) 的解集构成 \(\mathbb{R}^n\) 上的 \(n\) 维向量空间。
  • 基解矩阵：将 \(n\) 个线性无关的列向量解 \(\mathbf{X}_1, \ldots, \mathbf{X}_n\) 排成矩阵 \(\Phi = [\mathbf{X}_1 | \cdots | \mathbf{X}_n]\)，它满足矩阵 ODE \(\Phi' = A\Phi\)。
  • Abel-Liouville 公式：\(\det\Phi(t) = \det\Phi(t_0) \exp\!\int_{t_0}^t \mathrm{tr}\,A(s)\,ds\)——与标量 Wronskian 的 Abel 公式完全类比。
📐 公式推导 / 模型建立

1. 线性方程组的形式

齐次系统：\(\mathbf{X}'(t) = A(t)\mathbf{X}(t)\)，其中 \(A(t) \in \mathbb{R}^{n \times n}\)，\(\mathbf{X}(t) \in \mathbb{R}^n\)。

存在唯一性：若 \(A(t)\) 的各分量在区间 \(I\) 上连续，则 IVP \(\mathbf{X}(t_0) = \mathbf{X}_0\) 在 \(I\) 上有唯一解，且解全局存在（线性方程不爆破）。

2. 基解矩阵

设 \(\mathbf{X}_1, \ldots, \mathbf{X}_n\) 是 \(n\) 个线性无关的解，构造

\[\Phi(t) = \begin{pmatrix} | & | & & | \\ \mathbf{X}_1 & \mathbf{X}_2 & \cdots & \mathbf{X}_n \\ | & | & & | \end{pmatrix}\]

则 \(\Phi' = A\Phi\)，且 \(\det\Phi(t) \neq 0\) 对所有 \(t \in I\) 成立。通解为 \(\mathbf{X}(t) = \Phi(t)\mathbf{c}\)，\(\mathbf{c} \in \mathbb{R}^n\)。

3. Abel-Liouville 公式

\[\det\Phi(t) = \det\Phi(t_0) \cdot \exp\left(\int_{t_0}^{t} \mathrm{tr}\,A(s)\,ds\right)\]

证明思路（\(n=2\) 情况）：设 \(\Phi = (X_{ij})\)，则 \(\det\Phi = X_{11}X_{22} - X_{12}X_{21}\)。对行列式求导，利用 \(\Phi' = A\Phi\) 展开，合并后得 \((\det\Phi)' = \mathrm{tr}(A) \cdot \det\Phi\)。

推论：\(\det\Phi\) 要么恒非零，要么恒为零（指数函数无零点）。

4. 初值问题的解

\(\mathbf{X}(t_0) = \mathbf{X}_0 \Rightarrow \mathbf{c} = \Phi(t_0)^{-1}\mathbf{X}_0\)，故

\[\mathbf{X}(t) = \Phi(t)\Phi(t_0)^{-1}\mathbf{X}_0\]

矩阵 \(\Phi(t)\Phi(t_0)^{-1}\) 称为状态转移矩阵（state transition matrix），记为 \(\Psi(t, t_0)\)。

🌐 实际应用

耦合弹簧系统：两个质量块通过三根弹簧连接，\(m_1 x_1'' + (k_1+k_2)x_1 - k_2 x_2 = 0\) 等。改写为一阶系统后，基解矩阵的列描述了各个正则模态（normal mode），每个模态对应一个特征频率。

控制论的状态空间模型：\(\dot{\mathbf{x}} = A\mathbf{x} + B\mathbf{u}\)，\(\mathbf{y} = C\mathbf{x}\)。自由响应（\(\mathbf{u} = 0\)）即为 \(\mathbf{x}(t) = \Phi(t)\mathbf{x}_0\)。状态转移矩阵 \(\Psi(t, t_0)\) 是分析系统可控性、可观性的基本工具。

生态学的 Lotka-Volterra 模型：多物种竞争/共生系统在平衡点附近的线性化即为矩阵 ODE，基解矩阵描述了种群偏差随时间的演化。

✏️ 例题分析与易错点

例题：高阶方程化为方程组

将 \(y'' + 3y' + 2y = 0\) 改写为一阶系统并求基解矩阵。

第 1 步：令 \(x_1 = y\)，\(x_2 = y'\)。则 \(x_1' = x_2\)，\(x_2' = -2x_1 - 3x_2\)。

\[\mathbf{X}' = \begin{pmatrix} 0 & 1 \\ -2 & -3 \end{pmatrix}\mathbf{X}\]

第 2 步：特征方程 \(\lambda^2 + 3\lambda + 2 = 0\)，\(\lambda_1 = -1\)，\(\lambda_2 = -2\)。

第 3 步：特征向量。\(\lambda_1 = -1\)：\(\mathbf{v}_1 = (1, -1)^T\)。\(\lambda_2 = -2\)：\(\mathbf{v}_2 = (1, -2)^T\)。

基解矩阵：

\[\Phi(t) = \begin{pmatrix} e^{-t} & e^{-2t} \\ -e^{-t} & -2e^{-2t} \end{pmatrix}\]

常见易错点

  • ❌ 改写为一阶系统时，伴侣矩阵最后一行的符号弄反。标准形 \(y'' + 3y' + 2y = 0\) 对应 \(x_2' = -2x_1 - 3x_2\)（系数取负号）。
  • ❌ 混淆基解矩阵与普通的解矩阵。基解矩阵要求 \(\det\Phi \neq 0\)，即列向量线性无关。
  • ❌ 忘记特征向量是列向量，基解矩阵 \(\Phi\) 的每一列是一个解 \(\mathbf{v}_i e^{\lambda_i t}\)。

02矩阵指数的定义与性质

🎯 为什么研究这个问题

常系数系统 \(\mathbf{X}' = A\mathbf{X}\) 的解可以写成一个极其简洁的形式：

\[\mathbf{X}(t) = e^{tA}\mathbf{X}_0\]

完全类比标量方程 \(x' = ax\) 的解 \(x = e^{at}x_0\)。矩阵指数 \(e^{tA}\) 是理解常系数线性系统的核心对象：

  • 统一框架：无论 \(A\) 的特征值是实数、复数还是有重根，\(e^{tA}\) 都给出完整的解。
  • 物理学基石：量子力学的时间演化 \(U(t) = e^{-iHt/\hbar}\)、李群的指数映射 \(R(\theta) = e^{\theta\hat{n}}\) 都是矩阵指数的实例。
  • 数值方法：现代 ODE 数值方法中的指数积分子（exponential integrators）直接基于矩阵指数的高效计算。
🔍 抽象化方法

矩阵指数是连接三个数学领域的桥梁：

  • 分析 → 代数：通过幂级数 \(e^A = \sum A^k/k!\) 将指数函数从标量推广到矩阵，收敛性由范数不等式 \(\|e^A\| \leq e^{\|A\|}\) 保证。
  • 代数 → 几何：\(t \mapsto e^{tA}\) 定义了矩阵空间中的单参数群（one-parameter group），即光滑的群=同态 \((\mathbb{R}, +) \to (GL_n, \cdot)\)。
  • Lie 理论：矩阵指数是从 Lie 代数 \(\mathfrak{gl}_n\)（矩阵加法 + 括号积）到 Lie 群 \(GL_n\)（矩阵乘法）的指数映射。
📐 公式推导 / 模型建立

1. 定义与收敛性

\[e^{tA} = \sum_{k=0}^{\infty} \frac{(tA)^k}{k!} = I + tA + \frac{t^2A^2}{2!} + \frac{t^3A^3}{3!} + \cdots\]

收敛性：\(\|e^{tA}\| \leq \sum \frac{|t|^k \|A\|^k}{k!} = e^{|t|\|A\|}\)（对任意矩阵范数），故级数绝对收敛。

2. 基本性质

  • \(\frac{d}{dt}e^{tA} = Ae^{tA} = e^{tA}A\)（逐项微分，\(A\) 与自身的幂可交换）。
  • \(e^{0 \cdot A} = I\)（级数在 \(t = 0\) 退化为只剩 \(I\)）。
  • \(e^{(s+t)A} = e^{sA} e^{tA}\)（单参数群性质，因为 \(sA\) 和 \(tA\) 可交换）。
  • \((e^{tA})^{-1} = e^{-tA}\)（上条取 \(s = -t\)）。

3. 不可交换的警告

对一般矩阵 \(A, B\)：

\[e^{A+B} \neq e^A e^B \quad (\text{除非 } AB = BA)\]

反例：\(A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}\)，\(B = \begin{pmatrix} 0 & 0 \\ 1 & 0 \end{pmatrix}\)，\(AB \neq BA\)，可以验证 \(e^{A+B} \neq e^A e^B\)。

修正公式（Baker-Campbell-Hausdorff）：\(e^A e^B = e^{A + B + \frac{1}{2}[A,B] + \cdots}\)。

4. 与基解矩阵的关系

\(\Phi(t) = e^{tA}\) 是满足 \(\Phi(0) = I\) 的标准基解矩阵（fundamental matrix normalized at \(t = 0\)）。初值问题的解为

\[\mathbf{X}(t) = e^{(t - t_0)A}\mathbf{X}_0\]
🌐 实际应用

量子力学：时间演化算子 \(U(t) = e^{-iHt/\hbar}\) 是 Hamiltonian \(H\) 的矩阵指数。Schrödinger 方程 \(i\hbar |\psi\rangle' = H|\psi\rangle\) 的解为 \(|\psi(t)\rangle = U(t)|\psi(0)\rangle\)。

旋转群：三维旋转矩阵 \(R(\theta, \hat{n}) = e^{\theta[\hat{n}]_\times}\)，其中 \([\hat{n}]_\times\) 是反对称矩阵。这是机器人学、计算机图形学中表示旋转的核心工具。

控制理论：离散化 \(\dot{x} = Ax + Bu\) 得到 \(x_{k+1} = e^{A\Delta t} x_k + \int_0^{\Delta t} e^{As}B\,ds \cdot u_k\)。矩阵指数的高效计算直接影响实时控制系统的性能。

✏️ 例题分析与易错点

例题：幂零矩阵的矩阵指数

计算 \(e^{tA}\)，其中 \(A = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}\)。

第 1 步：计算 \(A\) 的幂。\(A^2 = \begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix} = 0\)。

第 2 步：级数截断。\(e^{tA} = I + tA + 0 + 0 + \cdots = \begin{pmatrix} 1 & t \\ 0 & 1 \end{pmatrix}\)。

验证：\(\frac{d}{dt}e^{tA} = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix} = A\)，而 \(Ae^{tA} = A\)。✓

常见易错点

  • ❌ 对不可交换的矩阵使用 \(e^{A+B} = e^Ae^B\)。这是最常见的错误——必须先检查 \([A, B] = 0\)。
  • ❌ 混淆 \(e^{tA}\) 的逐项微分与标量指数的微分。矩阵版本 \(\frac{d}{dt}e^{tA} = Ae^{tA}\) 之所以成立，关键是 \(A\) 与 \(A^k\) 可交换。
  • ❌ 试图对每个元素分别取指数：\(e^A \neq (e^{a_{ij}})\)（元素逐个取指数不是矩阵指数！）。

03矩阵指数的计算——对角化与 Jordan 形

🎯 为什么研究这个问题

矩阵指数的定义（无穷级数）在实际计算中几乎不可行。需要有效的计算方法：

  • 可对角化：若 \(A = PDP^{-1}\)，则 \(e^{tA} = Pe^{tD}P^{-1}\)，对角矩阵的指数只需对每个特征值取标量指数——问题化为求特征值和特征向量。
  • 不可对角化：需要 Jordan 标准形 \(A = PJP^{-1}\)，其中 Jordan 块的指数有优美的公式，涉及多项式乘以指数函数。
  • 实际意义：掌握这些计算方法是求解常系数方程组、分析系统稳定性和设计控制器的前提。
🔍 抽象化方法

计算 \(e^{tA}\) 的核心思想是谱分解——将 \(A\) 分解为简单组分，然后对每个组分分别取指数：

  • 对角化：\(A = P \cdot \mathrm{diag}(\lambda_1, \ldots, \lambda_n) \cdot P^{-1}\)。指数直接作用在对角元素上：\(e^{tD} = \mathrm{diag}(e^{\lambda_1 t}, \ldots, e^{\lambda_n t})\)。
  • Jordan 分解：\(A = P(D + N)P^{-1}\)，\(D\) 为对角部分，\(N\) 为幂零部分，且 \(DN = ND\)。因此 \(e^{t(D+N)} = e^{tD} \cdot e^{tN}\)（可交换！），幂零部分的指数是有限多项式。
  • Cayley-Hamilton 替代：利用 \(p(A) = 0\) 将 \(e^{tA}\) 表示为 \(A\) 的 \(\leq n-1\) 次多项式，系数由插值条件确定。
📐 公式推导 / 模型建立

1. 可对角化情况

若 \(A = PDP^{-1}\)，\(D = \mathrm{diag}(\lambda_1, \ldots, \lambda_n)\)，则

\[A^k = PD^kP^{-1} \implies e^{tA} = \sum \frac{t^k}{k!} PD^kP^{-1} = P\left(\sum \frac{t^k D^k}{k!}\right)P^{-1} = Pe^{tD}P^{-1}\] \[e^{tD} = \mathrm{diag}(e^{\lambda_1 t}, e^{\lambda_2 t}, \ldots, e^{\lambda_n t})\]

2. Jordan 块的矩阵指数

\(k \times k\) Jordan 块 \(J_k(\lambda) = \lambda I_k + N_k\)，其中 \(N_k\) 为上三角移位矩阵，\(N_k^k = 0\)。

由于 \(\lambda I\) 和 \(N\) 可交换，\(e^{tJ_k(\lambda)} = e^{\lambda t} \cdot e^{tN_k}\)，其中

\[e^{tN_k} = I + tN + \frac{t^2}{2!}N^2 + \cdots + \frac{t^{k-1}}{(k-1)!}N^{k-1}\]

写成矩阵形式：

\[e^{tJ_k(\lambda)} = e^{\lambda t}\begin{pmatrix} 1 & t & \frac{t^2}{2!} & \cdots & \frac{t^{k-1}}{(k-1)!} \\ 0 & 1 & t & \cdots & \frac{t^{k-2}}{(k-2)!} \\ \vdots & & \ddots & \ddots & \vdots \\ 0 & & \cdots & 1 & t \\ 0 & & \cdots & 0 & 1 \end{pmatrix}\]

3. 一般 Jordan 标准形

若 \(A = P \cdot \mathrm{diag}(J_{k_1}(\lambda_1), \ldots, J_{k_r}(\lambda_r)) \cdot P^{-1}\)，则

\[e^{tA} = P \cdot \mathrm{diag}(e^{tJ_{k_1}(\lambda_1)}, \ldots, e^{tJ_{k_r}(\lambda_r)}) \cdot P^{-1}\]

4. Cayley-Hamilton 方法

由 Cayley-Hamilton 定理 \(p(A) = 0\)，矩阵函数 \(f(A)\) 可用 \(\leq n-1\) 次多项式 \(q(A) = \alpha_0 I + \alpha_1 A + \cdots + \alpha_{n-1}A^{n-1}\) 表示。系数 \(\alpha_i(t)\) 由条件 \(f(\lambda_j) = q(\lambda_j)\)（及重根的导数条件）确定。

🌐 实际应用

耦合振动的正则模分析：多自由度振动系统 \(M\ddot{x} + Kx = 0\) 的对角化给出独立的正则模态，每个模态以特定频率独立振动。建筑抗震设计中的模态分析正是基于这一理论。

电路瞬态分析：状态空间方法中，电路的自然响应由 \(e^{tA}\) 的特征值决定——负实部特征值对应衰减，正实部对应不稳定，虚部对应振荡。

数值计算：计算 \(e^{tA}\) 是数值线性代数中的经典问题。Moler & Van Loan (2003) 的综述论文 "Nineteen Dubious Ways to Compute the Exponential of a Matrix" 总结了 19 种方法及其优缺点。

✏️ 例题分析与易错点

例题：Jordan 块的矩阵指数

求 \(e^{tA}\)，其中 \(A = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix}\)。

第 1 步：识别结构。\(A = J_2(1) = I + N\)，\(N = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}\)，\(N^2 = 0\)。

第 2 步：计算。

\[e^{tA} = e^t \cdot e^{tN} = e^t(I + tN) = e^t \begin{pmatrix} 1 & t \\ 0 & 1 \end{pmatrix} = \begin{pmatrix} e^t & te^t \\ 0 & e^t \end{pmatrix}\]

验证：\(\frac{d}{dt}\begin{pmatrix} e^t & te^t \\ 0 & e^t \end{pmatrix} = \begin{pmatrix} e^t & e^t + te^t \\ 0 & e^t \end{pmatrix} = \begin{pmatrix} 1 & 1 \\ 0 & 1 \end{pmatrix} \begin{pmatrix} e^t & te^t \\ 0 & e^t \end{pmatrix}\)。✓

常见易错点

  • ❌ 处理 Jordan 块时忘记 \(e^{\lambda t}\) 的前因子。\(e^{tJ_k(\lambda)} = e^{\lambda t} \cdot e^{tN}\)，不是 \(e^{tN}\) 单独。
  • ❌ 混淆特征值的代数重数和几何重数。只有几何重数 < 代数重数时才需要 Jordan 块处理。
  • ❌ Cayley-Hamilton 方法中忘记对重特征值使用导数条件。若 \(\lambda_0\) 是 \(m\) 重特征值，需要 \(q^{(j)}(\lambda_0) = f^{(j)}(\lambda_0)\)，\(j = 0, 1, \ldots, m-1\)。

04非齐次系统与常数变易法

🎯 为什么研究这个问题

实际系统几乎总有外部输入，齐次方程不够用：

  • 控制系统：\(\dot{x} = Ax + Bu\)，控制信号 \(u(t)\) 就是非齐次项。系统的响应 = 自由响应 + 受控响应。
  • 机械系统：外力驱动的振动 \(M\ddot{x} + C\dot{x} + Kx = F(t)\)，如地震作用、发动机激励。
  • 电路网络：独立电源驱动多端口网络，状态方程自然是非齐次的。

常数变易法将高阶标量方程的求解技巧自然推广到系统情形，给出统一的积分公式。

🔍 抽象化方法

非齐次系统的求解本质上是线性叠加原理的体现：

  • 通解结构：\(\mathbf{X} = \mathbf{X}_h + \mathbf{X}_p\)（齐次通解 + 特解），这是线性方程的基本范式。
  • 常数变易法思想：将齐次解中的常数 \(\mathbf{c}\) 替换为待定函数 \(\mathbf{u}(t)\)，利用方程确定 \(\mathbf{u}(t)\)——这是 Lagrange 方法的矩阵推广。
  • 卷积/Green 函数视角：Duhamel 公式 \(\mathbf{X}_p = \int e^{(t-s)A}\mathbf{B}(s)\,ds\) 是时不变线性系统的卷积响应，\(e^{tA}\) 扮演 Green 函数/脉冲响应矩阵的角色。
📐 公式推导 / 模型建立

1. 常数变易法推导

非齐次系统 \(\mathbf{X}' = A(t)\mathbf{X} + \mathbf{B}(t)\)，\(\Phi(t)\) 为齐次方程的基解矩阵。

设 \(\mathbf{X} = \Phi(t)\mathbf{u}(t)\)（将常数 \(\mathbf{c}\) 变为函数），代入方程：

\[\Phi'\mathbf{u} + \Phi\mathbf{u}' = A\Phi\mathbf{u} + \mathbf{B}\]

由于 \(\Phi' = A\Phi\)，左边第一项与右边第一项抵消，得到

\[\Phi(t)\mathbf{u}'(t) = \mathbf{B}(t) \implies \mathbf{u}'(t) = \Phi(t)^{-1}\mathbf{B}(t)\]

积分得

\[\mathbf{u}(t) = \int_{t_0}^{t} \Phi(s)^{-1}\mathbf{B}(s)\,ds + \mathbf{c}\]

2. 通解公式

\[\mathbf{X}(t) = \Phi(t)\mathbf{c} + \Phi(t)\int_{t_0}^{t} \Phi(s)^{-1}\mathbf{B}(s)\,ds\]

第一项是齐次通解，第二项是满足 \(\mathbf{X}(t_0) = \mathbf{0}\) 的特解。

3. Duhamel 公式（常系数）

常系数情况 \(\Phi(t) = e^{tA}\)，\(\Phi(t)\Phi(s)^{-1} = e^{(t-s)A}\)，特解简化为

\[\mathbf{X}_p(t) = \int_{t_0}^{t} e^{(t-s)A}\mathbf{B}(s)\,ds\]

这是卷积形式：系统对输入 \(\mathbf{B}(s)\) 的响应 = 脉冲响应 \(e^{(t-s)A}\) 与输入的积分。IVP 的完整解：

\[\mathbf{X}(t) = e^{(t-t_0)A}\mathbf{X}_0 + \int_{t_0}^{t} e^{(t-s)A}\mathbf{B}(s)\,ds\]

4. 变系数情况与 Peano-Baker 级数

变系数系统没有简洁的闭式解。通解仍为 \(\mathbf{X}(t) = \Phi(t,t_0)\mathbf{X}_0 + \int_{t_0}^{t}\Phi(t,s)\mathbf{B}(s)\,ds\)，其中 \(\Phi(t,t_0)\) 是状态转移矩阵，需要通过 Peano-Baker 级数或数值方法求得。

🌐 实际应用

现代控制理论：状态空间方程 \(\dot{x} = Ax + Bu\)，\(y = Cx + Du\) 的解就是 Duhamel 公式。可控性的 Kalman 判据本质上是判断 Duhamel 积分能否到达所有状态。

电路瞬态分析：多端口 RLC 网络的状态方程 \(\dot{x} = Ax + Bu_s(t)\)（\(u_s\) 为独立电源），Duhamel 公式给出完整的瞬态+稳态响应。SPICE 仿真中的瞬态分析就是数值求解这一公式。

结构动力学：多自由度结构受地震激励 \(M\ddot{u} + C\dot{u} + Ku = -M\mathbf{1}\ddot{u}_g(t)\)，Duhamel 积分在模态空间中对每个模态分别使用——这就是模态叠加法。

✏️ 例题分析与易错点

例题：Duhamel 公式求解 IVP

求 \(\mathbf{X}' = \begin{pmatrix} 1 & 0 \\ 0 & 2 \end{pmatrix}\mathbf{X} + \begin{pmatrix} 1 \\ t \end{pmatrix}\)，\(\mathbf{X}(0) = \mathbf{0}\)。

第 1 步：\(A\) 为对角矩阵，\(e^{tA} = \begin{pmatrix} e^t & 0 \\ 0 & e^{2t} \end{pmatrix}\)，\(e^{-sA} = \begin{pmatrix} e^{-s} & 0 \\ 0 & e^{-2s} \end{pmatrix}\)。

第 2 步：由 Duhamel 公式 \(\mathbf{X}(t) = \int_0^t e^{(t-s)A} \mathbf{B}(s)\,ds\)，逐分量计算：

\[x_1(t) = \int_0^t e^{t-s} \cdot 1\,ds = e^t - 1\] \[x_2(t) = \int_0^t e^{2(t-s)} \cdot s\,ds = e^{2t}\int_0^t s e^{-2s}\,ds\]

第 3 步：分部积分计算 \(\int_0^t se^{-2s}\,ds = \left[-\frac{s}{2}e^{-2s}\right]_0^t + \frac{1}{2}\int_0^t e^{-2s}\,ds = -\frac{t}{2}e^{-2t} - \frac{1}{4}(e^{-2t} - 1)\)。

最终结果：

\[\mathbf{X}(t) = \begin{pmatrix} e^t - 1 \\[4pt] \dfrac{1}{4}(e^{2t} - 1) - \dfrac{t}{2} \end{pmatrix}\]

验证：\(x_2'(t) = \frac{1}{2}e^{2t} - \frac{1}{2} = 2 \cdot \left[\frac{1}{4}(e^{2t}-1) - \frac{t}{2}\right] + t = 2x_2 + t\)。✓

常见易错点

  • ❌ Duhamel 积分中 \(e^{(t-s)A}\) 的指数是 \((t-s)\)，不是 \(t\) 也不是 \(s\)。写成 \(e^{tA}e^{-sA}\) 只在常系数（\(A\) 与自身可交换）时成立。
  • ❌ 变系数情况中 \(\Phi(t)\Phi(s)^{-1} \neq \Phi(t-s)\)。状态转移矩阵 \(\Phi(t,s)\) 不具有平移不变性。
  • ❌ 忘记分部积分的细节。Duhamel 公式常涉及 \(\int s^n e^{\alpha s}\,ds\) 类型积分，需要仔细执行分部积分。
