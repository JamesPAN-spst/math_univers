高阶线性方程

01线性方程一般理论与解空间结构

🎯 为什么研究这个问题

许多物理系统（弹簧-质点、RLC 电路、梁的弯曲）都由高阶线性 ODE 描述。这类方程的关键特征是「叠加原理」：解的线性组合仍是解。 理解齐次方程的解空间是 \(n\) 维向量空间，是求解的理论基础—— 只需找到 \(n\) 个线性无关的特解，就能写出通解。

🔍 抽象化方法

高阶线性 ODE 的解集构成线性空间，其维数等于方程的阶数。这让 我们可以用线性代数的语言来研究 ODE：基、维数、线性组合。 微分算子 \(L = a_n D^n + \cdots + a_1 D + a_0\) 是线性算子。

📐 公式推导 / 模型建立

\(n\) 阶线性方程： \[a_n(x)y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_0(x)y = g(x)\] 标准形：\(y^{(n)} + p_{n-1}(x)y^{(n-1)} + \cdots + p_0(x)y = q(x)\)。 叠加原理：齐次方程 \(L[y] = 0\) 的解集 \(V\) 是向量空间。 若 \(y_1, y_2\) 是齐次解，\(c_1 y_1 + c_2 y_2\) 也是齐次解。 维数定理：\(\dim V = n\)。 证明思路：考虑映射 \(\Phi: V \to \mathbb{R}^n\)， \(\Phi(y) = (y(x_0), y'(x_0), \ldots, y^{(n-1)}(x_0))\)。 由存在唯一性定理，\(\Phi\) 是双射，故 \(\dim V = n\)。 通解结构：非齐次方程 \(L[y] = g(x)\) 的通解为 \[y = c_1 y_1 + c_2 y_2 + \cdots + c_n y_n + y_p\] 其中 \(\{y_1, \ldots, y_n\}\) 是齐次方程的基本解组，\(y_p\) 是特解。

🌐 实际应用

- 弹簧-质点系统：\(my'' + cy' + ky = F(t)\) 描述阻尼受迫振动 - RLC 电路：\(L\ddot{q} + R\dot{q} + q/C = V(t)\)

✏️ 例题分析与易错点

例题：验证 \(y_1 = e^x\)，\(y_2 = e^{2x}\) 是 \(y'' - 3y' + 2y = 0\) 的基本解组。 解答：代入验证 \(y_1\)：\(e^x - 3e^x + 2e^x = 0\)。✓ 代入验证 \(y_2\)：\(4e^{2x} - 6e^{2x} + 2e^{2x} = 0\)。✓ 线性无关：\(e^x / e^{2x} = e^{-x}\) 非常数。✓ 通解：\(y = c_1 e^x + c_2 e^{2x}\)。 易错点：找到 \(n\) 个解不够，还需验证线性无关性。 例如 \(y_1 = e^x\) 和 \(y_2 = 3e^x\) 都是解但不构成基。

02Wronskian 行列式与 Abel 公式

🎯 为什么研究这个问题

给定若干解，如何判断它们是否线性无关？直接验证定义（某线性组合恒为零 ⇒ 系数全零）在实际中很难操作。Wronskian 行列式提供了一个强有力的 判据：它要么恒不为零（线性无关），要么恒为零（线性相关）。 Abel 公式则给出 Wronskian 的闭合表达式，无需逐个求解。

🔍 抽象化方法

Wronskian 是将 ODE 的线性无关性问题转化为行列式计算的桥梁。 Abel 公式 \(W(x) = W(x_0) \exp(-\int p_{n-1}\,dx)\) 揭示了 Wronskian 本身满足一个一阶 ODE，其解的指数结构意味着 \(W\) 要么处处非零要么处处为零。

📐 公式推导 / 模型建立

Wronskian 定义：给定 \(n\) 个函数 \(y_1, \ldots, y_n\)， \[W(y_1,\ldots,y_n)(x) = \det\begin{pmatrix} y_1 & \cdots & y_n \\ y_1' & \cdots & y_n' \\ \vdots & & \vdots \\ y_1^{(n-1)} & \cdots & y_n^{(n-1)} \end{pmatrix}\] 判定定理：若 \(y_1,\ldots,y_n\) 是齐次线性 ODE 的解，则： 线性无关 \(\Leftrightarrow\) \(W(x) \neq 0\) 对某个（等价地，每个）\(x\)。 Abel 公式（二阶情况）：对 \(y'' + p(x)y' + q(x)y = 0\)， \[W(x) = W(x_0) \exp\left(-\int_{x_0}^x p(s)\,ds\right)\] 证明：\(W = y_1 y_2' - y_1' y_2\)。求导： \(W' = y_1 y_2'' - y_1'' y_2 = y_1(-py_2'-qy_2) - (-py_1'-qy_1)y_2 = -p W\)。 解此一阶 ODE 得 Abel 公式。

🌐 实际应用

- Abel 公式可以从一个已知解 \(y_1\) 求第二个线性无关解 \(y_2\) （降阶法：\(y_2 = y_1 \int W/(y_1^2)\,dx\)）

✏️ 例题分析与易错点

例题：已知 \(y_1 = x\) 是 \(x^2 y'' - 2xy' + 2y = 0\) 的解，求 \(y_2\)。 解答：标准形 \(y'' - (2/x)y' + (2/x^2)y = 0\)，\(p(x) = -2/x\)。 Abel 公式：\(W = Ce^{\int 2/x\,dx} = Cx^2\)。 \(W = y_1 y_2' - y_1' y_2 = x y_2' - y_2 = Cx^2\)。 即 \((y_2/x)' = y_2'/x - y_2/x^2 \cdot x/x = C\)... 更直接地： \(y_2 = y_1 \int \frac{W}{y_1^2}dx = x\int \frac{Cx^2}{x^2}dx = x(Cx+D)\)。 取 \(C=1, D=0\)：\(y_2 = x^2\)。验证：\(x^2(2) - 2x(2x) + 2x^2 = 0\)。✓ 易错点：Abel 公式仅对线性 ODE 的解成立，对一般函数组无效。

⚡ 快速检测
若两个解的 Wronskian \(W(x_0) = 0\) 在某点 \(x_0\)，能否断言它们线性相关？

03常系数齐次方程与特征方程

🎯 为什么研究这个问题

当系数为常数时，\(a_n y^{(n)} + \cdots + a_0 y = 0\) 有一种优美的 代数求解方法：令 \(y = e^{\lambda x}\) 代入得到特征方程 \(a_n \lambda^n + \cdots + a_0 = 0\)。不同类型的根（实、复、重） 分别对应不同形式的基本解。这将微分方程的求解转化为代数方程。

🔍 抽象化方法

微分算子 \(D = d/dx\) 在指数函数上的作用是乘以 \(\lambda\)： \(D(e^{\lambda x}) = \lambda e^{\lambda x}\)。因此常系数线性 ODE 等价于多项式 \(p(D)\) 的核的求解，解的结构完全由 \(p(\lambda)\) 的根决定。

📐 公式推导 / 模型建立

特征方程：对 \(y'' + py' + qy = 0\)，令 \(y = e^{\lambda x}\) 得 \[\lambda^2 + p\lambda + q = 0\] 三种情况： 1. 两个不等实根 \(\lambda_1 \neq \lambda_2\)： 通解 \(y = c_1 e^{\lambda_1 x} + c_2 e^{\lambda_2 x}\) 2. 重根 \(\lambda_1 = \lambda_2 = \lambda\)： 通解 \(y = (c_1 + c_2 x)e^{\lambda x}\) （\(xe^{\lambda x}\) 的来源：\((D - \lambda)^2[xe^{\lambda x}] = 0\) 因为 \(e^{-\lambda x}(D-\lambda)[xe^{\lambda x}] = 1\)，再作用一次得 0） 3. 共轭复根 \(\lambda = \alpha \pm i\beta\)： \(e^{(\alpha+i\beta)x} = e^{\alpha x}(\cos\beta x + i\sin\beta x)\)（Euler 公式）， 取实部和虚部得实解： \[y = e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\] \(n\) 阶推广：\(k\) 重根 \(\lambda\) 贡献 \(k\) 个解 \(e^{\lambda x}, xe^{\lambda x}, \ldots, x^{k-1}e^{\lambda x}\)。

🌐 实际应用

- 阻尼振动：\(mx'' + cx' + kx = 0\)，特征根决定过阻尼/临界阻尼/欠阻尼

✏️ 例题分析与易错点

例题：求 \(y''' - 3y'' + 3y' - y = 0\) 的通解。 解答：特征方程 \(\lambda^3 - 3\lambda^2 + 3\lambda - 1 = 0\)，即 \((\lambda-1)^3 = 0\)。 三重根 \(\lambda = 1\)。 通解：\(y = (c_1 + c_2 x + c_3 x^2)e^x\)。 易错点：重根时忘记乘 \(x\) 的幂次。\(\lambda = 1\) 的三重根给出 \(e^x, xe^x, x^2 e^x\) 而非 \(e^x, e^x, e^x\)。

04非齐次方程的求解方法

🎯 为什么研究这个问题

实际物理系统通常有外力驱动（受迫振动 \(mx''+kx = F\cos\omega t\)、 非齐次电路方程等），对应非齐次 ODE \(L[y] = g(x)\)。 两种主要方法——待定系数法（高效但受限于特殊右端）和常数变易法 （普适但计算量大）——各有适用场景，是工程和物理中最常用的技术。

📐 公式推导 / 模型建立

常数变易法（\(n\) 阶）：设齐次通解为 \(y_h = \sum c_i y_i\)。 令 \(y_p = \sum u_i(x) y_i\) 并施加 \(n-1\) 个约束： \(\sum u_i' y_i = 0\)，\(\sum u_i' y_i' = 0\)，...，\(\sum u_i' y_i^{(n-2)} = 0\)。 剩余方程 \(\sum u_i' y_i^{(n-1)} = g(x)/a_n\) 与约束组成 \(n \times n\) 线性系统： \[\begin{pmatrix} y_1 & \cdots & y_n \\ \vdots & & \vdots \\ y_1^{(n-1)} & \cdots & y_n^{(n-1)} \end{pmatrix} \begin{pmatrix} u_1' \\ \vdots \\ u_n' \end{pmatrix} = \begin{pmatrix} 0 \\ \vdots \\ g/a_n \end{pmatrix}\] 由 Cramer 法则解出 \(u_i'\)，积分得 \(u_i\)。系数矩阵为 Wronskian（非零）。 待定系数法（常系数情况）：若 \(g(x) = P(x)e^{\alpha x}\)，\(P\) 为多项式， 则特解形如 \(y_p = x^s Q(x)e^{\alpha x}\)，其中 \(\deg Q = \deg P\)， \(s\) 为 \(\alpha\) 作为特征根的重数。对 \(g = e^{\alpha x}(P\cos\beta x + Q\sin\beta x)\) 类似处理。

🌐 实际应用

- 受迫振动分析中的共振现象：当驱动频率等于固有频率时， \(s \geq 1\)，特解含 \(x\) 因子使振幅线性增长

✏️ 例题分析与易错点

例题：求 \(y'' - y = e^x\) 的通解。 解答：齐次解 \(y_h = c_1 e^x + c_2 e^{-x}\)。 待定系数法：\(\alpha = 1\) 是特征根（\(s = 1\)）， 故设 \(y_p = Axe^x\)。代入：\(y_p'' = A(2e^x + xe^x)\)， \(y_p'' - y_p = 2Ae^x = e^x\)，\(A = 1/2\)。 通解：\(y = c_1 e^x + c_2 e^{-x} + \frac{1}{2}xe^x\)。 易错点：使用待定系数法时，忘记检查 \(e^{\alpha x}\) 是否已是齐次解。 若 \(\alpha\) 是 \(s\) 重特征根，必须乘 \(x^s\)。
