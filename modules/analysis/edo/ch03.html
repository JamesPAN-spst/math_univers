高阶线性方程

01线性方程一般理论与解空间结构

🎯 为什么研究这个问题

高阶线性 ODE 在物理和工程中无处不在：

  • 力学振动：弹簧-质点系统 \(my'' + cy' + ky = F(t)\) 是二阶线性 ODE，描述从汽车悬挂到建筑抗震的一切振动问题。
  • 电路分析：RLC 电路 \(L\ddot{q} + R\dot{q} + q/C = V(t)\) 与力学振动具有完全相同的数学结构。
  • 梁的弯曲：Euler-Bernoulli 方程 \(EI y^{(4)} = w(x)\) 是四阶线性 ODE。

这类方程的核心特征是叠加原理——解的线性组合仍是解。这意味着只需找到 \(n\) 个「基本解」，就能用线性代数的方法写出通解。

🔍 抽象化方法

高阶线性 ODE 的解集具有丰富的代数结构：

  • 解空间是向量空间：齐次方程 \(L[y] = 0\) 的解集构成 \(n\) 维向量空间 \(V\)。这让我们可以用线性代数的语言——基、维数、线性组合——来描述解集。
  • 微分算子：\(L = a_n D^n + a_{n-1}D^{n-1} + \cdots + a_0\)（其中 \(D = d/dx\)）是线性算子，\(\ker L = V\)。
  • 非齐次 = 齐次 + 特解：\(L[y] = g\) 的解集是 \(V\) 的一个仿射子空间 \(y_p + V\)，类似于线性方程组 \(Ax = b\) 的解结构。
📐 公式推导 / 模型建立

1. \(n\) 阶线性 ODE 的标准形式

一般形式：

\[a_n(x)y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_1(x)y' + a_0(x)y = g(x)\]

除以 \(a_n(x)\)（假设 \(a_n \neq 0\)）得标准形：

\[y^{(n)} + p_{n-1}(x)y^{(n-1)} + \cdots + p_0(x)y = q(x)\]

2. 叠加原理

对齐次方程 \(L[y] = 0\)，若 \(y_1, y_2\) 是解，则 \(\forall c_1, c_2 \in \mathbb{R}\)，\(c_1y_1 + c_2y_2\) 也是解。

证明：\(L[c_1y_1 + c_2y_2] = c_1 L[y_1] + c_2 L[y_2] = 0 + 0 = 0\)（利用 \(L\) 的线性性）。

3. 解空间的维数定理

定理：\(\dim(\ker L) = n\)。

证明思路：构造映射 \(\Phi: \ker L \to \mathbb{R}^n\)，\(\Phi(y) = (y(x_0),\, y'(x_0),\, \ldots,\, y^{(n-1)}(x_0))\)。

  • \(\Phi\) 是线性的（显然）。
  • \(\Phi\) 是单射：若 \(\Phi(y) = 0\)，则 \(y\) 满足零初值 IVP，由唯一性定理 \(y \equiv 0\)。
  • \(\Phi\) 是满射：\(\forall (a_0, \ldots, a_{n-1}) \in \mathbb{R}^n\)，由存在性定理，存在解满足这些初值。

故 \(\Phi\) 是同构，\(\dim V = n\)。

4. 通解结构

非齐次方程 \(L[y] = g(x)\) 的通解为

\[y = c_1 y_1 + c_2 y_2 + \cdots + c_n y_n + y_p\]

其中 \(\{y_1, \ldots, y_n\}\) 是齐次方程的基本解组（\(n\) 个线性无关的齐次解），\(y_p\) 是 \(L[y_p] = g\) 的任意一个特解，\(c_1, \ldots, c_n\) 为任意常数。

🌐 实际应用

弹簧-质点系统：\(my'' + cy' + ky = F(t)\)。齐次解 \(y_h\) 描述自由振动（瞬态响应，通常因阻尼衰减），特解 \(y_p\) 描述受迫振动（稳态响应）。工程上关心的通常是稳态特解。

RLC 串联电路：\(LI'' + RI' + I/C = V'(t)\)，与力学系统通过 \(L \leftrightarrow m\)，\(R \leftrightarrow c\)，\(1/C \leftrightarrow k\) 完全类比。电气工程师和力学工程师使用同一套数学工具。

量子力学：定态 Schrödinger 方程 \(-\frac{\hbar^2}{2m}\psi'' + V(x)\psi = E\psi\) 是二阶线性 ODE，其解空间结构直接决定能级和波函数。

✏️ 例题分析与易错点

例题：验证基本解组

验证 \(y_1 = e^x\)，\(y_2 = e^{2x}\) 是 \(y'' - 3y' + 2y = 0\) 的基本解组。

第 1 步：验证 \(y_1\) 是解。\(y_1'' - 3y_1' + 2y_1 = e^x - 3e^x + 2e^x = 0\)。✓

第 2 步：验证 \(y_2\) 是解。\(y_2'' - 3y_2' + 2y_2 = 4e^{2x} - 6e^{2x} + 2e^{2x} = 0\)。✓

第 3 步：验证线性无关。\(y_1/y_2 = e^{-x}\) 非常数函数，故线性无关。✓

结论：通解 \(y = c_1 e^x + c_2 e^{2x}\)。

例题 2：非齐次初值问题

求 \(y'' - 3y' + 2y = 4e^{3x}\)，\(y(0) = 1\)，\(y'(0) = 0\) 的解。

第 1 步：齐次通解 \(y_h = c_1 e^x + c_2 e^{2x}\)（由例题 1 已知）。

第 2 步：\(\alpha = 3\) 不是特征根，设 \(y_p = Ae^{3x}\)。代入得 \(9A - 9A + 2A = 4\)，\(A = 2\)。

第 3 步：通解 \(y = c_1 e^x + c_2 e^{2x} + 2e^{3x}\)。由 \(y(0) = c_1 + c_2 + 2 = 1\)，\(y'(0) = c_1 + 2c_2 + 6 = 0\)，解得 \(c_1 = 2\)，\(c_2 = -3\)。

答案：\(y = 2e^x - 3e^{2x} + 2e^{3x}\)。

常见易错点

  • ❌ 找到 \(n\) 个解就以为是基本解组。\(y_1 = e^x\) 和 \(y_2 = 3e^x\) 都是解，但 \(y_2 = 3y_1\)，线性相关，不构成基。
  • ❌ 忘记非齐次通解中要加特解。只写 \(c_1y_1 + c_2y_2\) 而遗漏 \(y_p\) 是常见错误。
  • ❌ 在解存在唯一性的论证中，忘记要求 \(a_n(x) \neq 0\)。若首项系数在某点为零，该点称为奇点（singular point），解的结构可能与正则情形完全不同。

02Wronskian 行列式与 Abel 公式

🎯 为什么研究这个问题

找到 \(n\) 个齐次解后，如何判断它们是否线性无关（即构成基本解组）？

  • 直接验证困难：需要证明 \(c_1y_1 + \cdots + c_ny_n \equiv 0 \Rightarrow c_i = 0\)，这在实际中很难操作。
  • Wronskian 判据：只需计算一个行列式 \(W(x)\)，若在某一点 \(W(x_0) \neq 0\)，则线性无关。
  • Abel 公式：给出 \(W(x)\) 的显式表达式，无需逐个求解方程。更实用的是，它还提供了从一个已知解求第二个解的降阶法。
🔍 抽象化方法

Wronskian 是连接 ODE 理论与线性代数的桥梁：

  • 行列式检测：将 \(n\) 个函数及其导数排成矩阵，行列式非零等价于线性无关——与有限维向量的线性无关判据完全类比。
  • \(W\) 自身满足 ODE：Abel 公式揭示 Wronskian 满足一阶 ODE \(W' = -p_{n-1}W\)，其解为指数函数 \(W = Ce^{-\int p_{n-1}}\)。这立即说明 \(W\) 要么处处非零，要么恒为零——不存在中间状态。
  • 降阶原理：已知一个解 \(y_1\)，用 Abel 公式可以将 \(n\) 阶方程降为 \(n-1\) 阶方程。
📐 公式推导 / 模型建立

1. Wronskian 的定义

给定 \(n\) 个（充分光滑的）函数 \(y_1, \ldots, y_n\)，定义

\[W(y_1,\ldots,y_n)(x) = \det\begin{pmatrix} y_1 & y_2 & \cdots & y_n \\ y_1' & y_2' & \cdots & y_n' \\ \vdots & \vdots & \ddots & \vdots \\ y_1^{(n-1)} & y_2^{(n-1)} & \cdots & y_n^{(n-1)} \end{pmatrix}\]

2. 线性无关的判定

定理：设 \(y_1, \ldots, y_n\) 是 \(n\) 阶齐次线性 ODE 的 \(n\) 个解，则

\(y_1, \ldots, y_n\) 线性无关 \(\Leftrightarrow\) \(W(x_0) \neq 0\) 对某个（等价地，每个）\(x_0\)。

关键：由于 Abel 公式，\(W\) 要么恒非零要么恒零，所以「某个」和「每个」等价。

3. Abel 公式的推导（二阶情况）

对 \(y'' + p(x)y' + q(x)y = 0\)，设 \(y_1, y_2\) 是两个解。

\(W = y_1 y_2' - y_1' y_2\)。对 \(W\) 求导：

\[W' = y_1 y_2'' - y_1'' y_2\]

因为 \(y_i'' = -py_i' - qy_i\)，代入得

\[W' = y_1(-py_2' - qy_2) - (-py_1' - qy_1)y_2 = -p(y_1y_2' - y_1'y_2) = -pW\]

这是关于 \(W\) 的一阶线性 ODE，解为

\[W(x) = W(x_0) \exp\left(-\int_{x_0}^{x} p(s)\,ds\right)\]

4. 降阶法

已知一个解 \(y_1 \neq 0\)，可用 Abel 公式求第二个线性无关解。设 \(y_2 = y_1 v(x)\)，则

\[W = y_1(y_1 v)' - y_1'(y_1 v) = y_1^2 v'\]

由 Abel 公式 \(v' = W/(y_1^2) = Ce^{-\int p}/(y_1^2)\)，积分得

\[y_2 = y_1 \int \frac{e^{-\int p(s)\,ds}}{y_1^2(x)}\,dx\]
🌐 实际应用

降阶求解：在实际中，高阶方程往往可以通过物理直觉猜出一个特解。Abel 公式/降阶法允许从这个已知解出发，系统地构造完整的基本解组——无需再猜。

Green 函数：常数变易法中需要 Wronskian 的逆。\(W \neq 0\)（由 Abel 公式保证）是构造 Green 函数和公式解的前提。

Sturm-Liouville 理论：在特征值问题 \(-(py')' + qy = \lambda wy\) 中，Wronskian 的性质决定了特征函数的零点分布和正交性。

✏️ 例题分析与易错点

例题：用降阶法求第二个解

已知 \(y_1 = x\) 是 \(x^2 y'' - 2xy' + 2y = 0\) 的解，求 \(y_2\)。

第 1 步：化为标准形 \(y'' - \frac{2}{x}y' + \frac{2}{x^2}y = 0\)，得 \(p(x) = -2/x\)。

第 2 步：由降阶公式

\[y_2 = y_1 \int \frac{e^{-\int(-2/x)\,dx}}{y_1^2}\,dx = x \int \frac{e^{2\ln x}}{x^2}\,dx = x \int \frac{x^2}{x^2}\,dx = x \int dx = x^2\]

第 3 步：验证 \(y_2 = x^2\)：\(x^2(2) - 2x(2x) + 2(x^2) = 2x^2 - 4x^2 + 2x^2 = 0\)。✓

结论：通解 \(y = c_1 x + c_2 x^2\)。

例题 2：利用 Abel 公式估计 Wronskian

已知 \(y'' + \sin(x)\,y' + e^x y = 0\) 的两个解满足 \(W(0) = 3\)。求 \(W(\pi)\)。

第 1 步：标准形中 \(p(x) = \sin x\)。

第 2 步：Abel 公式 \(W(x) = W(0)\exp\!\left(-\int_0^x \sin s\,ds\right) = 3\exp(\cos x - 1)\)。

第 3 步：\(W(\pi) = 3\exp(\cos\pi - 1) = 3e^{-2}\)。

注意：无需显式解，Abel 公式仅从系数 \(p(x)\) 即可确定 Wronskian 的演化。

常见易错点

  • ❌ 对一般函数误用 Wronskian 判据。\(W \neq 0 \Rightarrow\) 线性无关对任何函数成立，但 \(W = 0 \Rightarrow\) 线性相关仅对 ODE 的解成立。反例：\(y_1 = x^2\)，\(y_2 = x|x|\)，\(W(0) = 0\) 但线性无关（它们不是同一个 ODE 的解）。
  • ❌ 降阶法中忘记先化为标准形（首项系数为 1）再提取 \(p(x)\)。
  • ❌ 积分时丢失积分常数——降阶法中只需一个特解，可以取积分常数为 0。
⚡ 快速检测
若两个解的 Wronskian \(W(x_0) = 0\) 在某点 \(x_0\)，能否断言它们线性相关？

03常系数齐次方程与特征方程

🎯 为什么研究这个问题

当系数为常数时（物理中最常见的情况），高阶线性 ODE 有一种优美的代数求解方法：

  • 指数猜测：令 \(y = e^{\lambda x}\) 代入方程，微分方程变为关于 \(\lambda\) 的代数方程（特征方程）。这将无穷维的微分方程问题转化为有限维的多项式求根问题。
  • 根的类型决定解的形态：实根 → 指数增长/衰减，复根 → 振荡，重根 → 多项式调制的指数。物理上分别对应过阻尼、欠阻尼、临界阻尼。
  • 完全可解：只要能求出特征多项式的所有根，就能写出通解的显式表达式——不需要积分。
🔍 抽象化方法

微分算子 \(D = d/dx\) 在指数函数上的作用特别简单：

  • \(D(e^{\lambda x}) = \lambda e^{\lambda x}\)——指数函数是 \(D\) 的特征函数，\(\lambda\) 是特征值。
  • 常系数方程 \(p(D)y = 0\) 等价于求多项式算子 \(p(D)\) 的核，而 \(p(D)(e^{\lambda x}) = p(\lambda)e^{\lambda x}\)。
  • 因此 \(e^{\lambda x} \in \ker p(D) \Leftrightarrow p(\lambda) = 0\)——特征根完全决定了指数型解。

对重根的处理涉及 Jordan 标准形的思想：\((D - \lambda)^k\) 的核由 \(\{e^{\lambda x}, xe^{\lambda x}, \ldots, x^{k-1}e^{\lambda x}\}\) 张成。

📐 公式推导 / 模型建立

1. 特征方程

对常系数方程 \(a_ny^{(n)} + a_{n-1}y^{(n-1)} + \cdots + a_0y = 0\)，令 \(y = e^{\lambda x}\)：

\[a_n\lambda^n + a_{n-1}\lambda^{n-1} + \cdots + a_0 = 0\]

这是 \(n\) 次代数方程，在 \(\mathbb{C}\) 上恰好有 \(n\) 个根（计重数）。

2. 不同类型根对应的基本解

情形 1：不等实根 \(\lambda_1 \neq \lambda_2\)。

通解（二阶）：\(y = c_1 e^{\lambda_1 x} + c_2 e^{\lambda_2 x}\)。

情形 2：重根 \(\lambda\) 为 \(k\) 重根。

\(e^{\lambda x}\) 只提供 1 个解，还需 \(k-1\) 个。基本解为

\[e^{\lambda x},\; xe^{\lambda x},\; x^2 e^{\lambda x},\; \ldots,\; x^{k-1}e^{\lambda x}\]

为什么 \(x e^{\lambda x}\) 是解？ 考虑 \((D - \lambda)^2\)。令 \(y = e^{\lambda x} v(x)\)，则 \((D - \lambda)[e^{\lambda x}v] = e^{\lambda x}v'\)，再作用一次 \((D - \lambda)[e^{\lambda x}v'] = e^{\lambda x}v''\)。所以 \((D-\lambda)^2 y = 0 \Leftrightarrow v'' = 0 \Leftrightarrow v = c_1 + c_2 x\)。

3. 共轭复根与振荡解

情形 3：共轭复根 \(\lambda = \alpha \pm i\beta\)（\(\beta \neq 0\)）。

由 Euler 公式 \(e^{i\theta} = \cos\theta + i\sin\theta\)：

\[e^{(\alpha + i\beta)x} = e^{\alpha x}(\cos\beta x + i\sin\beta x)\]

取实部和虚部得到两个实解。通解贡献为

\[e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\]

若 \(\alpha \pm i\beta\) 是 \(k\) 重复根，则还需乘上 \(1, x, \ldots, x^{k-1}\)。

4. 汇总：\(n\) 阶方程的基本解组构造

将特征多项式在 \(\mathbb{C}\) 上因式分解，每个根按以上规则贡献基本解，共 \(n\) 个线性无关解。

🌐 实际应用

阻尼振动的三种模态：对 \(mx'' + cx' + kx = 0\)，特征方程 \(m\lambda^2 + c\lambda + k = 0\) 的判别式 \(\Delta = c^2 - 4mk\) 决定三种物理行为：

  • \(\Delta > 0\)（两个负实根）：过阻尼——系统缓慢回到平衡位置，无振荡。
  • \(\Delta = 0\)（重负实根）：临界阻尼——最快回到平衡位置而不振荡。门关闭器的理想设计。
  • \(\Delta < 0\)（复根 \(\alpha \pm i\beta\)）：欠阻尼——衰减振荡，\(\alpha < 0\) 控制衰减速率，\(\beta\) 为振荡频率。

电路工程：RLC 电路中的过渡过程完全类比力学振动，电气工程师通过调整 R、L、C 来控制电路的阻尼特性。

✏️ 例题分析与易错点

例题：三阶方程的重根

求 \(y''' - 3y'' + 3y' - y = 0\) 的通解。

第 1 步：写出特征方程 \(\lambda^3 - 3\lambda^2 + 3\lambda - 1 = 0\)。

第 2 步：识别因式分解 \((\lambda - 1)^3 = 0\)。三重根 \(\lambda = 1\)。

第 3 步：基本解组 \(\{e^x,\; xe^x,\; x^2 e^x\}\)。

通解：\(y = (c_1 + c_2 x + c_3 x^2)e^x\)。

例题 2：复根方程

求 \(y'' + 2y' + 5y = 0\) 的通解。

第 1 步：特征方程 \(\lambda^2 + 2\lambda + 5 = 0\)，\(\Delta = 4 - 20 = -16\)。

第 2 步：\(\lambda = (-2 \pm 4i)/2 = -1 \pm 2i\)。\(\alpha = -1\)，\(\beta = 2\)。

通解：\(y = e^{-x}(c_1\cos 2x + c_2\sin 2x)\)。

物理含义：\(e^{-x}\) 为衰减包络，三角函数描述频率 2 的振荡——典型的欠阻尼响应。

常见易错点

  • ❌ 重根时忘记乘 \(x\) 的幂次。三重根 \(\lambda = 1\) 给出 \(e^x, xe^x, x^2e^x\)，而不是三个相同的 \(e^x\)。
  • ❌ 复根时写出复指数而非实三角函数。最终答案应为实函数形式 \(e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\)。
  • ❌ 混淆 \(\alpha\) 和 \(\beta\) 的物理意义。\(\alpha\) 控制包络线的指数变化，\(\beta\) 是振荡频率（角频率）。\(\alpha < 0\) 对应衰减，\(\alpha > 0\) 对应不稳定增长。

04非齐次方程的求解方法

🎯 为什么研究这个问题

实际物理系统几乎总有外力驱动，对应非齐次方程 \(L[y] = g(x)\)：

  • 受迫振动：\(mx'' + cx' + kx = F_0\cos\omega t\)——外力 \(F(t)\) 驱动弹簧系统。
  • 电路的交流驱动：RLC 电路接入正弦电压源。
  • 梁的载荷：\(EIy^{(4)} = w(x)\)——分布载荷 \(w(x)\) 引起梁的挠曲。

两种主要求解方法各有优势：待定系数法快速高效但仅适用于特殊右端函数，常数变易法完全通用但计算量较大。

🔍 抽象化方法

非齐次方程的求解有两条路径，各有适用范围和代数内核：

  • 待定系数法的代数本质：由于 \(D(e^{\alpha x}) = \alpha e^{\alpha x}\)，指数-多项式-三角函数构成的函数类对微分算子 \(p(D)\) 封闭。因此可以在有限维子空间中「猜」特解的形式，将问题化为代数方程组。
  • 常数变易法的泛函分析视角：将齐次通解中的常数 \(c_i\) 替换为待定函数 \(u_i(x)\)，通过额外约束条件使系统变为可解的代数系统。本质上是利用 Wronskian 矩阵的可逆性。

两种方法互为补充：待定系数法在常系数+特殊右端时效率最高，常数变易法则对任意连续右端 \(g(x)\) 普遍适用。

📐 公式推导 / 模型建立

1. 常数变易法（Variation of Parameters）

设齐次方程的基本解组为 \(y_1, \ldots, y_n\)。令

\[y_p = u_1(x)y_1 + u_2(x)y_2 + \cdots + u_n(x)y_n\]

施加 \(n-1\) 个约束条件（消除高阶导数中的 \(u_i''\) 等项）：

\[\sum_{i=1}^{n} u_i' y_i = 0, \quad \sum_{i=1}^{n} u_i' y_i' = 0, \quad \ldots, \quad \sum_{i=1}^{n} u_i' y_i^{(n-2)} = 0\]

代入原方程后，第 \(n\) 个方程为 \(\sum u_i' y_i^{(n-1)} = g(x)/a_n\)。合并为线性系统

\[\begin{pmatrix} y_1 & \cdots & y_n \\ y_1' & \cdots & y_n' \\ \vdots & & \vdots \\ y_1^{(n-1)} & \cdots & y_n^{(n-1)} \end{pmatrix} \begin{pmatrix} u_1' \\ u_2' \\ \vdots \\ u_n' \end{pmatrix} = \begin{pmatrix} 0 \\ 0 \\ \vdots \\ g/a_n \end{pmatrix}\]

系数矩阵即 Wronskian 矩阵，\(W \neq 0\)（基本解组线性无关），由 Cramer 法则唯一解出各 \(u_i'\)，再积分得 \(u_i\)。

2. 待定系数法（Undetermined Coefficients）

适用于常系数方程且右端 \(g(x)\) 形如 \(P_m(x)e^{\alpha x}\)（\(P_m\) 为 \(m\) 次多项式）或 \(e^{\alpha x}[P(x)\cos\beta x + Q(x)\sin\beta x]\) 的情况。

规则：设特解形式为

\[y_p = x^s \cdot R_m(x) \cdot e^{\alpha x}\]

其中 \(R_m(x)\) 是待定的 \(m\) 次多项式，\(s\) 是 \(\alpha\) 作为特征根的重数（若 \(\alpha\) 不是特征根则 \(s = 0\)）。

\(x^s\) 因子的必要性：若 \(\alpha\) 是特征根（\(s \geq 1\)），则 \(e^{\alpha x}\) 已是齐次解，代入 \(L[e^{\alpha x}] = 0\)。需要乘以 \(x^s\) 使得 \(L[x^s e^{\alpha x}]\) 的首项非零。

3. 共振项的处理

对 \(g(x) = e^{\alpha x}[P\cos\beta x + Q\sin\beta x]\)，将 \(\alpha + i\beta\) 视为「复频率」。若 \(\alpha + i\beta\) 是特征根的 \(s\) 重根，则特解中需乘 \(x^s\)。

4. 叠加原理与右端分解

若右端 \(g(x) = g_1(x) + g_2(x) + \cdots\)，由线性性可分别对每个 \(g_i\) 求特解 \(y_{p,i}\)，再相加得 \(y_p = \sum y_{p,i}\)。例如 \(g = 3e^{2x} + 5\cos x\) 可分解为两个独立子问题，各自用待定系数法求解后求和。这是线性叠加原理在非齐次方程中的直接应用。

🌐 实际应用

共振现象：受迫振动 \(mx'' + kx = F_0\cos\omega t\)。当驱动频率 \(\omega\) 等于固有频率 \(\omega_0 = \sqrt{k/m}\) 时，\(i\omega_0\) 是特征根（\(s = 1\)），特解含 \(x\sin\omega_0 t\) 因子——振幅线性增长，这就是共振。实际工程中通过加阻尼（\(c > 0\)）或避开共振频率来防止结构损坏。

电气工程中的频率响应：交流电路中，待定系数法的复数形式即为相量法（phasor method），是电路分析的标准工具。特解的振幅和相位给出系统的频率响应函数。

Green 函数法：常数变易法的核心公式可以写成卷积形式 \(y_p(x) = \int G(x,s)g(s)\,ds\)，其中 \(G\) 是 Green 函数。这为更一般的边值问题和 PDE 提供了统一框架。

✏️ 例题分析与易错点

例题：待定系数法

求 \(y'' - y = e^x\) 的通解。

第 1 步：齐次解。特征方程 \(\lambda^2 - 1 = 0\)，\(\lambda = \pm 1\)。\(y_h = c_1 e^x + c_2 e^{-x}\)。

第 2 步：确定特解形式。\(g(x) = e^x\)，\(\alpha = 1\) 是单重特征根（\(s = 1\)）。设 \(y_p = Axe^x\)。

第 3 步：代入方程。\(y_p' = A(1 + x)e^x\)，\(y_p'' = A(2 + x)e^x\)。

\[y_p'' - y_p = A(2 + x)e^x - Axe^x = 2Ae^x = e^x\]

\(A = 1/2\)。

通解：\(y = c_1 e^x + c_2 e^{-x} + \frac{1}{2}xe^x\)。

例题 2：常数变易法

求 \(y'' + y = \sec x\) 的特解。

第 1 步：齐次解 \(y_1 = \cos x\)，\(y_2 = \sin x\)，\(W = 1\)。

第 2 步：\(u_1' = -y_2 g / W = -\sin x \sec x = -\tan x\)，\(u_2' = y_1 g / W = \cos x \sec x = 1\)。

第 3 步：\(u_1 = \ln|\cos x|\)，\(u_2 = x\)。

特解：\(y_p = \cos x \ln|\cos x| + x\sin x\)。此例中 \(\sec x\) 不是指数-多项式类，待定系数法不适用，必须用常数变易法。

常见易错点

  • ❌ 使用待定系数法时忘记检查 \(e^{\alpha x}\) 是否已是齐次解。若 \(\alpha\) 是 \(s\) 重特征根，必须乘 \(x^s\)——否则代入后右端恒为零，无法确定系数。
  • ❌ 对含三角函数的右端 \(g = \cos\beta x\)，只设 \(y_p = A\cos\beta x\) 而遗漏 \(B\sin\beta x\) 项。微分会混合 \(\cos\) 和 \(\sin\)，两者必须同时出现。
  • ❌ 常数变易法中，忘记先将方程化为标准形（首项系数为 1）再代入公式。\(g(x)/a_n\) 中的 \(a_n\) 不可省略。
