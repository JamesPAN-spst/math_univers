一阶常微分方程

01可分离变量方程与齐次方程

🎯 为什么研究这个问题

自然界最直接的建模方式往往导致一阶 ODE：牛顿冷却定律 \(dT/dt = -k(T - T_{\text{env}})\)、放射性衰变 \(dN/dt = -\lambda N\)、 种群增长 \(dP/dt = rP\)。这些方程有一个共性——变量可以分离 到等号两侧分别积分。可分离变量方程是最简单也最基本的一类 ODE， 齐次方程通过换元 \(v = y/x\) 化归为可分离型，体现了「化归」思想。

🔍 抽象化方法

核心思想是「分离与化归」：将方程中不同变量的有关项分到等号两侧， 使积分成为可能。齐次方程 \(y' = f(y/x)\) 通过引入比值变量 \(v = y/x\) 降低耦合度，这是 ODE 求解中「换元降阶」策略的原型。

📐 公式推导 / 模型建立

可分离变量方程：\(y' = f(x)g(y)\)。当 \(g(y) \neq 0\) 时分离变量： \[\frac{dy}{g(y)} = f(x)\,dx\] 两边积分得 \(\int \frac{dy}{g(y)} = \int f(x)\,dx + C\)。 注意：\(g(y_0) = 0\) 的常数解 \(y \equiv y_0\) 需要单独验证。 齐次方程：\(y' = f(y/x)\)（\(f\) 是 \(y/x\) 的函数）。 令 \(v = y/x\)，则 \(y = vx\)，\(y' = v + xv'\)，代入得 \[v + xv' = f(v) \quad\Longrightarrow\quad \frac{dv}{f(v) - v} = \frac{dx}{x}\] 这是关于 \(v\) 和 \(x\) 的可分离变量方程。积分后回代 \(v = y/x\)。

🌐 实际应用

- 放射性衰变 \(N' = -\lambda N\)（可分离）→ \(N(t) = N_0 e^{-\lambda t}\) - 化学反应动力学：二级反应 \(dc/dt = -kc^2\) → \(c(t) = c_0/(1+kc_0 t)\)

✏️ 例题分析与易错点

例题：求解 \((x^2 + y^2)\,dx - 2xy\,dy = 0\)。 解答：改写为 \(y' = \frac{x^2+y^2}{2xy}\)。 令 \(v = y/x\)，则 \(y' = \frac{1+v^2}{2v}\)。 \(v + xv' = \frac{1+v^2}{2v}\)，\(xv' = \frac{1-v^2}{2v}\)， \(\frac{2v\,dv}{1-v^2} = \frac{dx}{x}\)，积分得 \(-\ln|1-v^2| = \ln|x| + C_1\)，即 \(|x(1-v^2)| = C\)。 回代 \(v = y/x\)：\(|x - y^2/x| = C\)，即 \(x^2 - y^2 = Cx\)。 易错点：忘记检验 \(g(y) = 0\) 可能给出的奇异解。 如 \(y' = y^2\) 中 \(y \equiv 0\) 是解但不在通解 \(y = -1/(x+C)\) 中。

02一阶线性方程与 Bernoulli 方程

🎯 为什么研究这个问题

一阶线性方程 \(y' + p(x)y = q(x)\) 在工程和物理中无处不在： RC 电路的电压响应、药物在血液中的浓度变化、一阶系统的强迫振动。 它有完整的求解公式，是 ODE 理论的基石。Bernoulli 方程和 Riccati 方程则是重要的非线性推广，但能通过巧妙换元化归为线性方程。

🔍 抽象化方法

线性方程的核心是「叠加原理」：齐次解空间是一维线性空间， 通解 = 齐次通解 + 特解。积分因子法的本质是乘以一个函数使左端 变为全导数 \(d(\mu y)/dx\)，这是微分运算的逆操作。

📐 公式推导 / 模型建立

一阶线性方程 \(y' + p(x)y = q(x)\)。 积分因子 \(\mu(x) = e^{\int p(x)\,dx}\)，两边乘 \(\mu\)： \[\frac{d}{dx}[\mu(x) y] = \mu(x) q(x)\] 积分得通解： \[y(x) = e^{-\int p\,dx}\left[\int e^{\int p\,dx}\,q(x)\,dx + C\right]\] Bernoulli 方程 \(y' + p(x)y = q(x)y^n\)（\(n \neq 0,1\)）。 令 \(v = y^{1-n}\)，则 \(v' = (1-n)y^{-n}y'\)。 原方程两边同除 \(y^n\) 得 \(y^{-n}y' + p(x)y^{1-n} = q(x)\)， 即 \(\frac{1}{1-n}v' + p(x)v = q(x)\)，化为关于 \(v\) 的一阶线性方程。 Riccati 方程 \(y' = a(x)y^2 + b(x)y + c(x)\)： 已知特解 \(y_1\) 时，令 \(y = y_1 + 1/v\) 可化为一阶线性方程。

🌐 实际应用

- RC 电路：\(RC\,dV_C/dt + V_C = V_{\text{in}}(t)\)， 是标准一阶线性方程，描述电容充放电过程 - Logistic 方程 \(P' = rP(1-P/K)\) 是 Bernoulli 方程（\(n=2\)）

✏️ 例题分析与易错点

例题：求解 \(y' + \frac{2}{x}y = x^2\)，\(y(1)=0\)。 解答：\(p(x)=2/x\)，积分因子 \(\mu = e^{\int 2/x\,dx} = x^2\)。 乘以 \(\mu\)：\((x^2 y)' = x^4\)，积分得 \(x^2 y = x^5/5 + C\)。 \(y = x^3/5 + C/x^2\)。代入 \(y(1)=0\)：\(0 = 1/5 + C\)，\(C = -1/5\)。 故 \(y = \frac{x^3 - x^{-2}}{5} = \frac{x^5 - 1}{5x^2}\)。 易错点：积分因子中忘记常数可以省略（取 \(C=0\) 即可）， 或对 Bernoulli 方程在 \(n>1\) 时忘记 \(y=0\) 也是解。

03恰当方程与积分因子

🎯 为什么研究这个问题

当方程可以写成某个二元函数的全微分 \(dF = 0\) 时，解就是等值线 \(F(x,y) = C\)。但大多数方程不是天然恰当的——此时需要寻找 积分因子使它变得恰当。恰当方程的判别法和积分因子的求法， 构成了一阶 ODE 求解的另一条重要路径。

🔍 抽象化方法

恰当方程的本质是外微分：\(M\,dx + N\,dy = dF\) 当且仅当 \(dF\) 是 恰当 1-形式。恰当性条件 \(\partial M/\partial y = \partial N/\partial x\) 就是 Poincaré 引理在 \(\mathbb{R}^2\) 上的具体表现。

📐 公式推导 / 模型建立

恰当方程：\(M(x,y)\,dx + N(x,y)\,dy = 0\) 称为恰当的，如果存在 \(F(x,y)\) 使得 \(F_x = M\)，\(F_y = N\)。 判别条件（单连通域上）： \[\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\] 求解方法：从 \(F_x = M\) 积分得 \(F = \int M\,dx + \varphi(y)\)， 再由 \(F_y = N\) 确定 \(\varphi'(y)\)。 积分因子：若不恰当，寻找 \(\mu(x,y)\) 使 \(\mu M\,dx + \mu N\,dy = 0\) 恰当。特殊情况： - 若 \(\frac{M_y - N_x}{N}\) 仅依赖 \(x\)，则 \(\mu = e^{\int \frac{M_y-N_x}{N}dx}\) - 若 \(\frac{N_x - M_y}{M}\) 仅依赖 \(y\)，则 \(\mu = e^{\int \frac{N_x-M_y}{M}dy}\)

🌐 实际应用

- 力学中保守力场 \(\mathbf{F} = -\nabla V\) 的等势线即为恰当方程的解 - 热力学中状态方程 \(dU = T\,dS - P\,dV\) 的恰当性条件给出 Maxwell 关系

✏️ 例题分析与易错点

例题：判断 \((2xy + 3)\,dx + (x^2 + 4y)\,dy = 0\) 是否恰当并求解。 解答：\(M = 2xy+3\)，\(N = x^2+4y\)。 \(M_y = 2x\)，\(N_x = 2x\)。\(M_y = N_x\)，恰当。 \(F = \int (2xy+3)\,dx = x^2y + 3x + \varphi(y)\)。 \(F_y = x^2 + \varphi'(y) = x^2 + 4y\)，故 \(\varphi'(y) = 4y\)， \(\varphi(y) = 2y^2\)。通解：\(x^2 y + 3x + 2y^2 = C\)。 易错点：对 \(M\) 关于 \(x\) 积分时，「常数」\(\varphi(y)\) 不能漏掉， 否则 \(F_y\) 将缺少关键项，导致矛盾或错误。

04包络、奇解与方程的几何

🎯 为什么研究这个问题

ODE 的通解是一族曲线（由任意常数 \(C\) 参数化），这族曲线可能 拥有「包络」——一条与每条曲线相切的特殊曲线。包络满足方程 但不包含在通解族中，称为奇解。理解奇解对于完整描述方程的 解集至关重要，也揭示了 ODE 解结构的微妙几何性质。

📐 公式推导 / 模型建立

通解族的包络：设通解为 \(F(x, y, C) = 0\)。包络由以下方程组确定： \[F(x, y, C) = 0 \quad\text{且}\quad \frac{\partial F}{\partial C}(x, y, C) = 0\] 消去 \(C\) 得到包络曲线。 奇解：满足 ODE 但不能由通解 \(y = y(x; C)\) 对任何 \(C\) 取值得到。 奇解上每一点都与通解族中某条曲线相切。 Clairaut 方程 \(y = xy' + f(y')\) 的通解为 \(y = Cx + f(C)\)（直线族）， 奇解由消去 \(p\)：\(x + f'(p) = 0\)，\(y = px + f(p)\) 得到参数形式。 此奇解就是直线族的包络。 判别曲线法：对 \(F(x,y,y') = 0\)，\(p\)-判别曲线由消去 \(y'\) 得到： \(F = 0,\; F_{y'} = 0\)。该曲线包含奇解，但也可能包含非解（需验证）。

🌐 实际应用

- 光学中的焦散线(caustic)是光线族的包络 - 几何光学中波前的形成与包络密切相关

✏️ 例题分析与易错点

例题：求 Clairaut 方程 \(y = xy' + (y')^2\) 的通解和奇解。 解答：通解：\(y = Cx + C^2\)（直线族）。 求包络：\(F(x,y,C) = Cx + C^2 - y = 0\)，\(F_C = x + 2C = 0\)。 由 \(C = -x/2\) 代入：\(y = (-x/2)x + x^2/4 = -x^2/4\)。 奇解：\(y = -x^2/4\)（抛物线，与每条直线相切）。 验证：\(y' = -x/2\)，\(xy' + (y')^2 = -x^2/2 + x^2/4 = -x^2/4 = y\)。✓ 易错点：\(p\)-判别曲线不一定全是奇解，需要代回原方程验证。
