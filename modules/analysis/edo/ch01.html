一阶常微分方程

01可分离变量方程与齐次方程

🎯 为什么研究这个问题

自然界最直接的建模方式往往导致一阶 ODE。考虑以下几个例子：

  • 牛顿冷却定律：\(\frac{dT}{dt} = -k(T - T_{\text{env}})\) — 物体温度变化速率与温差成正比；
  • 放射性衰变：\(\frac{dN}{dt} = -\lambda N\) — 衰变速率与现有原子数成正比；
  • 种群增长：\(\frac{dP}{dt} = rP\) — 指数增长模型（Malthus 模型）。

这些方程有一个共性——右端可以写成 \(f(t) \cdot g(y)\) 的乘积形式，变量可以"分离"到等号两侧分别积分。可分离变量方程是最简单也最基本的一类 ODE，而齐次方程通过换元 \(v = y/x\) 可化归为可分离型——这体现了 ODE 求解中最核心的「化归」思想：把不会解的方程变成会解的。

🔍 抽象化方法

核心思想是「分离与化归」：

  • 分离：将方程中只含 \(x\) 的项和只含 \(y\) 的项分到等号两侧，使两边独立可积；
  • 化归：当方程不直接可分离时，通过换元（如 \(v = y/x\)）将其转化为可分离型。

齐次方程 \(y' = f(y/x)\) 的关键观察是：右端只依赖比值 \(y/x\)，而非 \(x,y\) 的独立值。引入 \(v = y/x\) 作为新变量后，方程中 \(x\) 和 \(v\) 的耦合被解开，化为可分离型。这个「换元降阶」策略是整个 ODE 求解方法论的原型——后续的 Bernoulli 换元、降阶法等都遵循相同理念。

📐 公式推导 / 模型建立

1. 可分离变量方程

标准形式：\(y' = f(x)g(y)\)，右端是 \(x\) 的函数与 \(y\) 的函数之积。

求解步骤：当 \(g(y) \neq 0\) 时，将 \(y\) 的因子移到左边：

\[\frac{dy}{g(y)} = f(x)\,dx\]

两边分别积分：

\[\int \frac{dy}{g(y)} = \int f(x)\,dx + C\]

这一步的合理性来自：\(y'dx = dy\)（微分记号的严格化由积分变换定理保证）。

2. 奇异解的检查

在分离变量时我们除以了 \(g(y)\)，因此 \(g(y_0) = 0\) 对应的常数解 \(y \equiv y_0\) 必须单独代入验证。例如 \(y' = y^2\) 中 \(y \equiv 0\) 是解，但不在通解 \(y = -1/(x + C)\) 中——它是奇异解。一般地，所有使 \(g(y_0) = 0\) 的常数函数都必须逐一检验是否满足原方程。

3. 齐次方程的判别

形式：\(y' = f\!\left(\frac{y}{x}\right)\)，即右端是 \(y/x\) 的函数。

判别方法：把 \(x, y\) 同时替换为 \(tx, ty\)，如果方程不变，则为齐次。等价地，右端函数 \(h(x,y)\) 满足 \(h(tx,ty) = h(x,y)\)（零次齐次）。

4. 齐次方程的换元求解

令 \(v = y/x\)，则 \(y = vx\)，由乘积法则：

\[y' = v + xv'\]

代入方程 \(y' = f(v)\) 得：

\[v + xv' = f(v) \quad\Longrightarrow\quad xv' = f(v) - v\]

分离变量：

\[\frac{dv}{f(v) - v} = \frac{dx}{x}\]

这是关于 \(v\) 和 \(x\) 的可分离变量方程。积分后回代 \(v = y/x\) 即得原方程的通解。

注意：当 \(f(v_0) = v_0\) 时，\(y = v_0 x\) 是一条过原点的直线解，分离变量时被除掉，需单独检查。

🌐 实际应用

放射性衰变：\(N' = -\lambda N\)（可分离），分离积分得 \(N(t) = N_0 e^{-\lambda t}\)。半衰期 \(t_{1/2} = \ln 2/\lambda\) 直接由此推出。碳-14 定年法正是基于这一模型。

化学反应动力学：二级反应 \(dc/dt = -kc^2\)，分离变量得 \(c(t) = c_0/(1 + kc_0 t)\)。与一级反应的指数衰减不同，二级反应浓度以代数速率衰减——衰减更慢。

自由落体（含阻力）：\(mv' = mg - bv^2\)，可分离为 \(\frac{dv}{g - (b/m)v^2} = dt\)。积分给出终端速度 \(v_\infty = \sqrt{mg/b}\) 和趋近的时间尺度。

✏️ 例题分析与易错点

例题：齐次方程求解

求解 \((x^2 + y^2)\,dx - 2xy\,dy = 0\)。

第 1 步 — 改写为标准形式：\(y' = \frac{x^2+y^2}{2xy}\)。

第 2 步 — 判别齐次：分子分母都是 \(x,y\) 的二次齐次式，比值为零次齐次 → 齐次方程。

第 3 步 — 换元：令 \(v = y/x\)，则 \(y' = \frac{1+v^2}{2v}\)，代入得：

\[v + xv' = \frac{1+v^2}{2v} \quad\Rightarrow\quad xv' = \frac{1+v^2 - 2v^2}{2v} = \frac{1-v^2}{2v}\]

第 4 步 — 分离积分：

\[\frac{2v\,dv}{1-v^2} = \frac{dx}{x}\]

积分得 \(-\ln|1-v^2| = \ln|x| + C_1\)，即 \(|x(1-v^2)| = C\)。

第 5 步 — 回代：\(v = y/x\)，得 \(x^2 - y^2 = Cx\)。

例题 2：可分离变量方程与奇异解

求 \(y' = y(1-y)\) 的所有解。

第 1 步 — 分离变量：当 \(y \neq 0, 1\) 时，\(\displaystyle\frac{dy}{y(1-y)} = dx\)。

第 2 步 — 部分分式积分：\(\displaystyle\frac{1}{y(1-y)} = \frac{1}{y} + \frac{1}{1-y}\)，积分得 \(\ln\!\left|\frac{y}{1-y}\right| = x + C_1\)。

第 3 步 — 整理：\(\displaystyle y = \frac{1}{1 + Ce^{-x}}\)（Logistic 曲线）。

第 4 步 — 检查奇异解：\(y \equiv 0\) 与 \(y \equiv 1\) 满足原方程但不在通解族中，是两条奇异解。

常见错误

  • ❌ 忘记检验 \(g(y) = 0\) 的奇异解。如 \(y' = y^2\) 中 \(y \equiv 0\) 是解但不在通解 \(y = -1/(x+C)\) 中。
  • ❌ 齐次方程换元后忘记回代 \(v = y/x\)，答案中残留 \(v\)。
  • ❌ 分离变量时对 \(\int \frac{2v}{1-v^2}\,dv\) 积分符号搞错：正确结果是 \(-\ln|1-v^2|\)（注意负号）。
  • ✅ 检验技巧：求完通解后取一组特定的 \(C\) 值代回原方程验算。

02一阶线性方程与 Bernoulli 方程

🎯 为什么研究这个问题

一阶线性方程 \(y' + p(x)y = q(x)\) 是 ODE 中最重要的方程类型之一，原因有三：

  • 普遍性：RC 电路的电压响应、药物在血液中的浓度变化、一阶控制系统的强迫响应都是这种形式；
  • 可解性：它有完整的通解公式（积分因子法），不像多数非线性方程只能近似求解；
  • 基石作用：高阶线性方程理论（ch03）的叠加原理、常数变易法等都是在这里首先建立的。

Bernoulli 方程 \(y' + p(x)y = q(x)y^n\) 和 Riccati 方程是重要的非线性推广——但通过巧妙换元可以化归为线性方程。这展示了一个深刻的方法论：非线性世界中的某些问题可以通过变量替换回到线性的舒适区。

🔍 抽象化方法

线性方程的核心是叠加原理：

  • 齐次方程 \(y' + py = 0\) 的解空间是一维线性空间：\(y_h = Ce^{-\int p\,dx}\)，任何常数倍仍是解；
  • 非齐次方程的通解 = 齐次通解 + 一个特解：\(y = y_h + y_p\)；
  • 任意两个特解之差是齐次解（线性性的直接推论）。

积分因子法的本质：乘以一个函数 \(\mu(x)\) 使左端变为全导数 \(\frac{d}{dx}[\mu \cdot y]\)。这不是凭空的技巧——它恰好是求解线性算子 \(L[y] = y' + py\) 的「逆操作」：把微分方程变回积分问题。

📐 公式推导 / 模型建立

1. 一阶线性方程的积分因子法

标准形式：\(y' + p(x)y = q(x)\)。

构造积分因子：寻找 \(\mu(x)\) 使得 \(\mu(y' + py) = (\mu y)'\)。展开右边：\((\mu y)' = \mu' y + \mu y'\)，比较得 \(\mu' = \mu p\)，即 \(\frac{\mu'}{\mu} = p\)。积分：

\[\mu(x) = e^{\int p(x)\,dx}\]

（积分常数取 0 即可，因为 \(\mu\) 只需要满足比例关系。）

2. 通解公式的推导

两边乘 \(\mu\)，左端变为全导数：

\[\frac{d}{dx}[\mu(x) y] = \mu(x) q(x)\]

积分得：

\[y(x) = e^{-\int p\,dx}\left[\int e^{\int p\,dx}\,q(x)\,dx + C\right]\]

这个公式虽然看起来复杂，但结构清晰：\(e^{-\int p\,dx}\) 是齐次解（衰减/增长因子），括号内的积分给出特解。

3. Bernoulli 方程

形式：\(y' + p(x)y = q(x)y^n\)（\(n \neq 0, 1\)）。

换元：令 \(v = y^{1-n}\)，则 \(v' = (1-n)y^{-n}y'\)。

原方程两边同除 \(y^n\)：\(y^{-n}y' + p(x)y^{1-n} = q(x)\)。

注意到 \(y^{-n}y' = \frac{1}{1-n}v'\)，代入得：

\[\frac{1}{1-n}v' + p(x)v = q(x) \quad\Longrightarrow\quad v' + (1-n)p(x)v = (1-n)q(x)\]

这是关于 \(v\) 的一阶线性方程，用积分因子法求解即可。

注意：当 \(n > 1\) 时，\(y = 0\) 也是方程的解（代入验证 \(0 = 0\)），但在换元时被除掉了——需要单独检查。

4. Riccati 方程

形式：\(y' = a(x)y^2 + b(x)y + c(x)\)。一般情况下不可积。

但若已知一个特解 \(y_1(x)\)，令 \(y = y_1 + 1/v\)，代入后二次项消去，化为关于 \(v\) 的一阶线性方程。

这说明 Riccati 方程的「困难」本质在于找到第一个特解——之后的工作都是线性的。

🌐 实际应用

RC 电路：电容电压满足 \(RC\frac{dV_C}{dt} + V_C = V_{\text{in}}(t)\)，这是标准一阶线性方程。积分因子 \(\mu = e^{t/RC}\)，时间常数 \(\tau = RC\) 决定了充放电速度。阶跃输入下的解 \(V_C(t) = V_0(1 - e^{-t/RC})\) 是工程中最常见的瞬态响应。

Logistic 方程：\(P' = rP(1 - P/K)\)，整理为 \(P' - rP = -(r/K)P^2\)，这是 Bernoulli 方程（\(n=2\)）。令 \(v = 1/P\) 化为线性方程，得经典解 \(P(t) = \frac{K}{1 + (K/P_0 - 1)e^{-rt}}\)——S 形增长曲线，广泛用于种群生态学和流行病学。

药物代谢：血药浓度模型 \(c' + kc = D\delta(t)\)，其中 \(k\) 是消除速率，\(D\) 是给药剂量。一阶线性方程给出指数衰减的消除曲线，药物半衰期 \(t_{1/2} = \ln 2/k\)。

✏️ 例题分析与易错点

例题：一阶线性方程的初值问题

求解 \(y' + \frac{2}{x}y = x^2\)，\(y(1) = 0\)。

第 1 步 — 计算积分因子：\(p(x) = 2/x\)，\(\mu = e^{\int 2/x\,dx} = e^{2\ln|x|} = x^2\)。

第 2 步 — 乘以 \(\mu\) 并积分：

\[(x^2 y)' = x^4 \quad\Rightarrow\quad x^2 y = \frac{x^5}{5} + C\]

第 3 步 — 求通解：\(y = \frac{x^3}{5} + \frac{C}{x^2}\)。

第 4 步 — 代入初值：\(y(1) = 0\)：\(0 = \frac{1}{5} + C\)，\(C = -\frac{1}{5}\)。

故 \(y = \frac{x^5 - 1}{5x^2}\)。

例题 2：Bernoulli 方程

求解 \(y' + y = y^3\)。

第 1 步 — 识别参数：\(p = 1\)，\(q = 1\)，\(n = 3\)；令 \(v = y^{1-3} = y^{-2}\)。

第 2 步 — 换元：\(v' = -2y^{-3}y'\)，整理为 \(v' - 2v = -2\)。

第 3 步 — 线性方程求解：积分因子 \(\mu = e^{-2x}\)，\((e^{-2x}v)' = -2e^{-2x}\)，积分得 \(v = 1 + Ce^{2x}\)。

第 4 步 — 回代：\(y^{-2} = 1 + Ce^{2x}\)，故 \(y = \pm(1 + Ce^{2x})^{-1/2}\)。另检查 \(y \equiv 0\) 也满足原方程。

常见错误

  • ❌ 积分因子中保留积分常数：\(e^{2\ln|x| + C}\)。积分因子只需要一个特解，取 \(C = 0\) 即可。
  • ❌ 对 Bernoulli 方程在 \(n > 1\) 时忘记 \(y = 0\) 也是解——换元 \(v = y^{1-n}\) 在 \(y = 0\) 处无意义。
  • ❌ 使用通解公式但忘记 \(q(x)\) 不是常数时需要保留在积分号内。
  • ✅ 验证技巧：将求出的 \(y\) 代回原方程的左端 \(y' + py\)，检查是否等于右端 \(q(x)\)。
⚡ 快速检测
Bernoulli 方程 \(y' + y = y^3\) 通过哪个换元化为线性方程？

03恰当方程与积分因子

🎯 为什么研究这个问题

当方程可以写成某个二元函数的全微分 \(dF = 0\) 时，解就是等值线 \(F(x,y) = C\)——方程的「解」变成了一个隐式曲线族，求解问题化为寻找势函数。

但大多数一阶方程并非天然恰当。此时我们需要积分因子——一个乘上去就能让方程变恰当的函数。这与多元微积分中「保守场」的概念直接相关：

  • 恰当方程 ↔ 保守力场 \(\mathbf{F} = -\nabla V\)
  • 积分因子 ↔ 使非保守场变成保守场的「标量乘子」
  • 恰当性条件 ↔ 旋度为零（二维情况下混合偏导相等）
🔍 抽象化方法

恰当方程的本质是外微分。一阶微分形式 \(\omega = M\,dx + N\,dy\) 恰当当且仅当它是某个函数 \(F\) 的全微分：\(\omega = dF\)。

在单连通域上，Poincaré 引理告诉我们：闭形式 = 恰当形式。"闭"的条件就是 \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\)——这是因为 \(d\omega = \left(\frac{\partial N}{\partial x} - \frac{\partial M}{\partial y}\right)dx \wedge dy = 0\)。

如果域不单连通（如有洞），闭形式不一定恰当——这是代数拓扑（de Rham 上同调）的起点。

📐 公式推导 / 模型建立

1. 恰当方程的判别

方程 \(M(x,y)\,dx + N(x,y)\,dy = 0\) 恰当，意味着存在 \(F(x,y)\) 使得 \(F_x = M\)，\(F_y = N\)。

由 Schwarz 定理（混合偏导交换），恰当的必要条件：

\[\frac{\partial M}{\partial y} = F_{xy} = F_{yx} = \frac{\partial N}{\partial x}\]

在单连通域上，这也是充分条件。

2. 恰当方程的求解

第 1 步：从 \(F_x = M\) 对 \(x\) 积分：

\[F(x,y) = \int M(x,y)\,dx + \varphi(y)\]

其中 \(\varphi(y)\) 是"积分常数"（对 \(x\) 是常数，但可以依赖 \(y\)）。

第 2 步：由 \(F_y = N\) 确定 \(\varphi'(y)\)：

\[\frac{\partial}{\partial y}\int M\,dx + \varphi'(y) = N \quad\Rightarrow\quad \varphi'(y) = N - \frac{\partial}{\partial y}\int M\,dx\]

右端必须仅依赖 \(y\)（否则方程不恰当），积分得 \(\varphi(y)\)。

第 3 步：通解为 \(F(x,y) = C\)。

3. 积分因子的一般理论

若方程不恰当，寻找 \(\mu(x,y)\) 使得 \(\mu M\,dx + \mu N\,dy = 0\) 恰当。一般来说求 \(\mu\) 需解偏微分方程 \(N\mu_x - M\mu_y = (M_y - N_x)\mu\)，这本身比原问题更难。幸运的是，许多实际方程允许特殊形式的积分因子。

4. 仅依赖一个变量的积分因子

有两种重要的可显式求解的情况：

  • 若 \(\frac{M_y - N_x}{N}\) 仅依赖 \(x\)，则 \(\mu(x) = \exp\!\left(\int \frac{M_y - N_x}{N}\,dx\right)\)
  • 若 \(\frac{N_x - M_y}{M}\) 仅依赖 \(y\)，则 \(\mu(y) = \exp\!\left(\int \frac{N_x - M_y}{M}\,dy\right)\)

这两个公式的推导方式相同：假设 \(\mu\) 只依赖一个变量，代入恰当性条件 \((\mu M)_y = (\mu N)_x\)，展开后解出 \(\mu'/\mu\)。

🌐 实际应用

保守力场：在力学中，若力 \(\mathbf{F} = (M, N)\) 是保守的（\(M_y = N_x\)），则存在势能 \(V\) 使 \(\mathbf{F} = -\nabla V\)。沿任何闭合路径做功为零。恰当方程 \(M\,dx + N\,dy = 0\) 的解 \(F = C\) 就是等势线。

热力学 Maxwell 关系：态函数 \(U\) 的全微分 \(dU = T\,dS - P\,dV\) 是恰当的（因为 \(U\) 是状态函数），恰当性条件给出 Maxwell 关系 \(\frac{\partial T}{\partial V}\Big|_S = -\frac{\partial P}{\partial S}\Big|_V\)——一个非常有用的热力学恒等式。

流体力学：二维不可压流 \(\nabla \cdot \mathbf{v} = 0\) 意味着 \(v_x\,dy - v_y\,dx = 0\) 恰当，存在流函数 \(\psi\) 使 \(v_x = \psi_y\)，\(v_y = -\psi_x\)。

✏️ 例题分析与易错点

例题：恰当方程求解

判断 \((2xy + 3)\,dx + (x^2 + 4y)\,dy = 0\) 是否恰当并求解。

第 1 步 — 验证恰当性：\(M = 2xy+3\)，\(N = x^2+4y\)。

\(M_y = 2x = N_x\) ✓ 恰当。

第 2 步 — 求势函数：\(F = \int (2xy+3)\,dx = x^2y + 3x + \varphi(y)\)。

\(F_y = x^2 + \varphi'(y) = N = x^2 + 4y\)，故 \(\varphi'(y) = 4y\)，\(\varphi(y) = 2y^2\)。

通解：\(x^2 y + 3x + 2y^2 = C\)。

例题 2：利用积分因子求解

求解 \(y\,dx + 2x\,dy = 0\)。

第 1 步 — 检验恰当性：\(M = y\)，\(N = 2x\)；\(M_y = 1 \neq N_x = 2\)，不恰当。

第 2 步 — 寻找积分因子：\(\displaystyle\frac{M_y - N_x}{N} = \frac{1 - 2}{2x} = -\frac{1}{2x}\) 仅依赖 \(x\)。故 \(\mu = e^{\int -1/(2x)\,dx} = x^{-1/2}\)。

第 3 步 — 验证恰当性：乘以 \(\mu\) 后 \(\tilde{M} = x^{-1/2}y\)，\(\tilde{N} = 2x^{1/2}\)；\(\tilde{M}_y = x^{-1/2} = \tilde{N}_x\)。✓

第 4 步 — 求势函数：\(F = \int x^{-1/2}y\,dx = 2x^{1/2}y + \varphi(y)\)；\(F_y = 2x^{1/2} + \varphi' = 2x^{1/2}\)，知 \(\varphi = 0\)。通解 \(2\sqrt{x}\,y = C\)。

常见错误

  • ❌ 对 \(M\) 关于 \(x\) 积分时漏掉"常数" \(\varphi(y)\)——这会导致 \(F_y\) 丢失项，使 \(F_y = N\) 出现矛盾。
  • ❌ 混淆哪个变量求偏导哪个变量积分：\(F_x = M\) 意味着对 \(x\) 积分，\(y\) 当作常数。
  • ❌ 不恰当时强行用恰当方程的方法求解，忘记先找积分因子。
  • ✅ 快速验证：求出 \(F\) 后，分别计算 \(F_x\) 和 \(F_y\)，检查是否等于 \(M\) 和 \(N\)。

04包络、奇解与方程的几何

🎯 为什么研究这个问题

ODE 的通解是一族曲线（由任意常数 \(C\) 参数化）。这族曲线有时会"聚集"在某条特殊曲线附近——每条曲线都与它相切。这条特殊曲线叫做包络。

包络满足原方程但不属于通解族（对任何 \(C\) 取值都得不到它），称为奇解。奇解的存在揭示了 ODE 解结构的微妙之处：

  • 在奇解经过的点上，解的唯一性定理（Lipschitz 条件）失效；
  • 不考虑奇解就无法完整描述方程的所有解；
  • 几何上，包络常对应物理中的极限状态（如焦散线、极限环）。
🔍 抽象化方法

从微分几何的视角看，通解族 \(F(x,y,C) = 0\) 定义了 \((x,y,C)\) 空间中的一个曲面。投影到 \((x,y)\) 平面时：

  • 一般点对应一条（或有限条）通过的曲线——解唯一。
  • 投影的折叠处（\(\partial F/\partial C = 0\) 的位置）对应包络——此处相邻参数值的曲线几乎重合，发生"切触"。

因此求包络的方法是自然的：找到曲面上 \(C\) 方向的"转折点"，即联立 \(F = 0\) 和 \(F_C = 0\)，消去 \(C\)。

📐 公式推导 / 模型建立

1. 包络的求法

设通解为 \(F(x, y, C) = 0\)。包络由以下方程组确定：

\[F(x, y, C) = 0 \quad\text{且}\quad \frac{\partial F}{\partial C}(x, y, C) = 0\]

从第二个方程解出 \(C = C(x,y)\)，代入第一个方程消去 \(C\)，得到包络曲线。

直觉：\(F_C = 0\) 意味着参数 \(C\) 的微小变化不改变曲线经过的点——即该点 is "两条无限接近的曲线的交点"，这正是包络的定义。

2. 奇解与唯一性

奇解是满足 ODE 但不能由通解对任何 \(C\) 取值得到的解。在奇解上的每一点，它与通解族中的某条曲线相切。

为什么唯一性失效？在奇解与通解的切点处，两个不同的解（奇解和通解中的一条曲线）通过同一个点且方向相同——这意味着 \(f(x,y)\) 在该处不满足 Lipschitz 条件。

3. Clairaut 方程

形式：\(y = xy' + g(y')\)。令 \(y' = p\)，则 \(y = xp + g(p)\)。

对 \(x\) 求导：\(p = p + xp' + g'(p)p'\)，即 \((x + g'(p))p' = 0\)。

两种情况：

  • \(p' = 0\)，即 \(p = C\)（常数）→ 通解 \(y = Cx + g(C)\)——这是一族直线。
  • \(x + g'(p) = 0\)，即 \(x = -g'(p)\)，连同 \(y = xp + g(p)\) 给出参数形式的奇解（以 \(p\) 为参数）——这就是直线族的包络。

4. p-判别曲线法

对一般方程 \(F(x, y, y') = 0\)，将 \(y' = p\) 视为参数，联立：

\[F(x, y, p) = 0 \quad\text{且}\quad F_p(x, y, p) = 0\]

消去 \(p\) 得到 \(p\)-判别曲线。它包含奇解，但也可能包含退化因子（如节点轨迹）——需要将结果代回原方程验证。

🌐 实际应用

光学焦散线：光线经凹面镜反射后形成的焦散线（caustic），就是光线族的包络。近轴光线的焦散是一个点（焦点），但大角度光线的焦散是一条明亮的曲线——咖啡杯底部的光斑就是日常可见的例子。

波前与惠更斯原理：波前在传播过程中的形状由次波球的包络确定。Huygens 原理本质上就是在求包络——每个次波源发出球面波，所有球面波的公切面构成新的波前。

几何光学：子午面上光线族 \(y = x\tan\alpha - \frac{x^2}{4f\cos^2\alpha}\) 的包络给出抛物面焦散的形状。

✏️ 例题分析与易错点

例题：Clairaut 方程的通解与奇解

求解 \(y = xy' + (y')^2\)。

第 1 步 — 通解：令 \(y' = C\)（常数），得直线族 \(y = Cx + C^2\)。

第 2 步 — 求包络：\(F(x, y, C) = Cx + C^2 - y = 0\)，\(F_C = x + 2C = 0\)。

由 \(C = -x/2\) 代入：\(y = (-x/2)x + x^2/4 = -x^2/4\)。

第 3 步 — 验证奇解：\(y = -x^2/4\)，\(y' = -x/2\)。代入原方程：

\[xy' + (y')^2 = x(-x/2) + x^2/4 = -x^2/2 + x^2/4 = -x^2/4 = y \quad\checkmark\]

奇解 \(y = -x^2/4\) 是一条抛物线，与每条直线 \(y = Cx + C^2\) 相切。

例题 2：用消参法求包络

求曲线族 \((x - C)^2 + y^2 = 1\)（圆心在 \(x\) 轴上的单位圆族）的包络。

第 1 步 — 对参数求导：\(F_C = -2(x - C) = 0\)，得 \(C = x\)。

第 2 步 — 代回消参：\((x - x)^2 + y^2 = 1\)，即 \(y^2 = 1\)。

第 3 步 — 几何解读：包络为两条水平线 \(y = 1\) 和 \(y = -1\)，每条线与所有单位圆相切。

常见错误

  • ❌ 将 \(p\)-判别曲线直接当作奇解而不代回验证——它可能包含非解部分。
  • ❌ 混淆通解（含参数 \(C\) 的一族曲线）和特解（取定 \(C\) 后的一条曲线）。奇解不是特解——它不能通过取任何 \(C\) 值得到。
  • ❌ Clairaut 方程中对 \(x\) 求导后忘记因式分解 \((x + g'(p))p' = 0\)，只取了 \(p' = 0\) 而漏掉了另一支。
  • ✅ 几何验证：画出通解的直线族和奇解曲线，检查它们确实相切。
