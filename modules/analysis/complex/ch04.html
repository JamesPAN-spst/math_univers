留数理论

01留数定义与计算

🎯 为什么研究这个问题

Cauchy 定理说全纯函数的闭路积分为零。但如果路径围上了奇点呢？ 留数（residue）正是度量奇点对积分贡献的量——Laurent 展开中 \(a_{-1}\) 系数。留数定理将围道积分归结为有限个留数的求和， 是计算复积分和实积分的万能工具。

📐 公式推导 / 模型建立

留数定义：\(f\) 在 \(z_0\) 有孤立奇点， \(\text{Res}(f, z_0) = a_{-1} = \frac{1}{2\pi i}\oint f(z) dz\) （绕 \(z_0\) 的小圆）。 留数定理：\(f\) 在区域 \(D\) 内除孤立奇点 \(z_1, \ldots, z_n\) 外全纯。则 \[\oint_{\partial D} f(z)\, dz = 2\pi i \sum_{k=1}^n \text{Res}(f, z_k)\] 计算公式： - 一阶极点：\(\text{Res}(f, z_0) = \lim_{z \to z_0} (z - z_0) f(z)\) - \(m\) 阶极点：\(\text{Res}(f, z_0) = \frac{1}{(m-1)!} \lim_{z \to z_0} \frac{d^{m-1}}{dz^{m-1}} [(z-z_0)^m f(z)]\) - \(f = p/q\) 型（\(q\) 有简单零点）：\(\text{Res}(f, z_0) = p(z_0)/q'(z_0)\)

✏️ 例题分析与易错点

例题：\(\text{Res}(e^z/z^3, 0)\)。 解答：\(e^z/z^3 = (1 + z + z^2/2 + \cdots)/z^3 = z^{-3} + z^{-2} + z^{-1}/2 + \cdots\)。 \(\text{Res} = a_{-1} = 1/2\)。 例题：\(\text{Res}(\frac{z}{z^2+1}, i)\)。 解答：\(p/q\) 型，\(q'(z) = 2z\)。\(\text{Res} = i/(2i) = 1/2\)。 易错点：高阶极点的留数公式需要 \((m-1)\) 次求导+除以 \((m-1)!\)。 常忘阶乘。

⚡ 快速检测
\(\text{Res}(\frac{1}{z(z-1)^2}, 0) = ?\)

02留数定理计算实积分

🎯 为什么研究这个问题

留数定理最精彩的应用是计算困难的实积分。通过选择适当的围道 （半圆、矩形、扇形等），将实轴上的积分转化为复平面上的围道 积分，再用留数定理求值。这种方法能处理许多初等方法无法攻克 的积分。

📐 公式推导 / 模型建立

类型 I：三角积分 \(\int_0^{2\pi} R(\cos\theta, \sin\theta) d\theta\) 令 \(z = e^{i\theta}\), \(\cos\theta = (z + z^{-1})/2\), \(\sin\theta = (z - z^{-1})/(2i)\), \(d\theta = dz/(iz)\)。转化为 \(\oint_{|z|=1}\) 的积分。 类型 II：有理函数 \(\int_{-\infty}^\infty f(x) dx\) 用上半平面半圆围道。若 \(|f(z)| = O(|z|^{-2})\)， 半圆弧上积分 \(\to 0\)（\(R \to \infty\)）， \[\int_{-\infty}^\infty f(x) dx = 2\pi i \sum \text{Res}(f, z_k)\] （\(z_k\) 为上半平面极点）。 类型 III：含 \(e^{ix}\) 的积分（Jordan 引理）： \(\int_{-\infty}^\infty f(x) e^{ix} dx\)，用上半平面围道， Jordan 引理保证半圆弧贡献 \(\to 0\)。

🌐 实际应用

量子场论中 Feynman 积分、概率论中特征函数的反演公式 都依赖于留数计算。

✏️ 例题分析与易错点

例题：\(\int_{-\infty}^\infty \frac{dx}{1+x^2}\)。 解答：\(f(z) = 1/(1+z^2) = 1/((z+i)(z-i))\)。 上半平面极点：\(z = i\)（一阶）。 \(\text{Res}(f, i) = 1/(2i)\)。 \(\int = 2\pi i \cdot 1/(2i) = \pi\)。✓（用 \(\arctan\) 验证。） 例题：\(\int_0^\infty \frac{\cos x}{1 + x^2} dx\)。略写为 \(\frac{1}{2}\text{Re}\int_{-\infty}^\infty \frac{e^{ix}}{1+x^2} dx = \frac{1}{2}\text{Re}(2\pi i \cdot \frac{e^{-1}}{2i}) = \frac{\pi}{2e}\)。 易错点：半圆弧贡献不总为零！需检查 \(|f|\) 的衰减速度 或用 Jordan 引理（仅当被积函数含 \(e^{iaz}\), \(a > 0\) 时适用）。

⚡ 快速检测
\(\int_{-\infty}^\infty \frac{x^2}{(1+x^2)^2} dx = ?\)

03特殊围道与高级技巧

🎯 为什么研究这个问题

不是所有实积分都能用标准半圆围道解决。含有 \(\log x\)、\(x^a\) （分数幂）或在正半轴上积分的问题需要"钥匙孔"围道或矩形围道。 掌握围道选择是留数方法的核心艺术。

📐 公式推导 / 模型建立

含对数的积分：\(\int_0^\infty \frac{f(x)}{x^a} dx\)（\(0 < a < 1\)）。 用钥匙孔围道（keyhole contour）：沿正实轴上方和下方绕一圈， \(x^{-a}\) 在分支切割上下值相差因子 \(e^{-2\pi i a}\)： \[(1 - e^{-2\pi i a}) \int_0^\infty \frac{f(x)}{x^a} dx = 2\pi i \sum \text{Res}\] 含 \(\log\) 的积分：\(\int_0^\infty \frac{\log x}{1+x^2} dx\)。 用钥匙孔围道，\(\log\) 在切线上下差 \(2\pi i\)： \[\oint \frac{\log z}{1+z^2} dz = \int_0^\infty \frac{\log x}{1+x^2} dx + \int_0^\infty \frac{\log x + 2\pi i}{1+x^2} dx\] 矩形围道：用于 \(e^{ax}/(e^x + 1)\) 型积分（利用 \(e^{2\pi i}\) 的周期性）。

🌐 实际应用

Gamma 函数的积分表示和 Mellin 变换的反演公式 都依赖特殊围道技巧。

✏️ 例题分析与易错点

例题：\(\int_0^\infty \frac{dx}{1+x^3}\)。 解答：用扇形围道（\(0\) 到 \(R\)、弧、\(Re^{2\pi i/3}\) 到 \(0\)）。 \(f(z) = 1/(1+z^3)\)，极点 \(z = e^{i\pi/3}\) 在扇形内。 边 \(0 \to Re^{2\pi i/3}\)：\(z = te^{2\pi i/3}\)，积分 \(= e^{2\pi i/3} I\)。 \((1 - e^{2\pi i/3}) I = 2\pi i \text{Res}(f, e^{i\pi/3})\)。 \(\text{Res} = 1/(3e^{2\pi i/3})\)。计算得 \(I = 2\pi/(3\sqrt{3})\)。 易错点：分支切割的选择影响结果。不同的 \(\arg\) 约定 给出不同的 \(\log z\) 分支，必须全程一致。

04留数在无穷远点与求和公式

🎯 为什么研究这个问题

标准留数定理处理有限远奇点。在无穷远点也可以定义留数—— 这对处理"所有奇点之和"问题很有用。此外，留数还能用来 计算无穷级数的和，将离散求和问题转化为连续积分问题。

📐 公式推导 / 模型建立

无穷远处的留数：\(\text{Res}(f, \infty) = -\frac{1}{2\pi i} \oint_{|z|=R} f(z) dz\)（\(R\) 足够大，顺时针）。 等价：\(\text{Res}(f, \infty) = -\text{Res}\left(\frac{1}{z^2}f(1/z), 0\right)\)。 留数总和定理：所有有限远留数之和 + 无穷远处留数 = 0。 级数求和：对 \(\cot(\pi z)\) 或 \(\csc^2(\pi z)\) 利用留数： \(\pi \cot(\pi z)\) 在 \(z = n\)（\(n \in \mathbb{Z}\)）有一阶极点， 留数 = 1。选围道使 \(\cot\) 有界，得 \[\sum_{n=-\infty}^{\infty} f(n) = -\pi \sum_k \text{Res}(f(z)\cot(\pi z), z_k)\] （\(z_k\) 为 \(f\) 的极点）。 经典结果：\(\sum_{n=1}^\infty 1/n^2 = \pi^2/6\)。 取 \(f(z) = 1/z^2\)，用上述公式。

🌐 实际应用

数论中 Riemann zeta 函数在偶数点的值 \(\zeta(2k) = (-1)^{k+1}(2\pi)^{2k}B_{2k}/(2(2k)!)\) 可用留数方法证明。

✏️ 例题分析与易错点

例题：用留数法证 \(\sum_{n=1}^\infty \frac{1}{n^2} = \frac{\pi^2}{6}\)。 解答：取 \(f(z) = 1/z^2\)，考虑 \(\oint_{\gamma_N} \frac{\pi\cot(\pi z)}{z^2} dz\) （\(\gamma_N\) 为围住 \(|n| \leq N\) 的正方形）。 \(\cot(\pi z)\) 在 \(\gamma_N\) 上有界，积分 \(\to 0\)（\(N \to \infty\)）。 极点：\(z = 0\)（三阶，来自 \(\cot/z^2\) 的组合， 留数 \(= -\pi^2/3\)）和 \(z = n \neq 0\)（一阶，留数 \(= 1/n^2\)）。 \(0 = 2\sum_{n=1}^\infty 1/n^2 + (-\pi^2/3)\)，故 \(\sum = \pi^2/6\)。 易错点：无穷远处留数的符号约定——绕向是顺时针，多一个负号。
