解析延拓与整函数

01解析延拓的概念与方法

🎯 为什么研究这个问题

全纯函数有惊人的刚性——在一个小区域上的值决定了它在更大区域 上的值。解析延拓正是利用这种刚性，将函数从小区域"延伸"到 大区域。最著名的例子是 Riemann zeta 函数：原始定义 \(\zeta(s) = \sum n^{-s}\) 仅在 \(\text{Re}(s) > 1\) 收敛，但可以 解析延拓到除 \(s = 1\) 外的全复平面。

🔍 抽象化方法

解析延拓的本质是唯一性定理的逆应用——如果两个区域的重叠部分 上函数一致，则功能上它们是"同一个函数"。

📐 公式推导 / 模型建立

定义：\(f_1: D_1 \to \mathbb{C}\) 全纯，\(f_2: D_2 \to \mathbb{C}\) 全纯， \(D_1 \cap D_2 \neq \emptyset\)，\(f_1 = f_2\) 在 \(D_1 \cap D_2\) 上。 则 \(f_2\) 是 \(f_1\) 到 \(D_2\) 的解析延拓。 唯一性：延拓（若存在）唯一。 幂级数方法：在 \(z_0\) 处 Taylor 展开 → 收敛圆盘扩大到边界奇点 → 在圆盘内新的点展开 → 沿路径链式延拓。 Schwarz 反射原理：\(f\) 在上半平面全纯，在实轴上连续且取实值。 则 \(f\) 可延拓到下半平面：\(f(\bar{z}) = \overline{f(z)}\)。 自然边界：延拓不总是可能的。 \(f(z) = \sum z^{2^n}\) 在 \(|z| = 1\) 上的每一点都是奇点—— 单位圆是自然边界，不能延拓到外面。

🌐 实际应用

Riemann zeta 函数 \(\zeta(s) = \sum n^{-s}\)（\(\text{Re}\, s > 1\)） 解析延拓到 \(\mathbb{C} \setminus \{1\}\)，满足函数方程 \(\zeta(s) = 2^s\pi^{s-1}\sin(\pi s/2)\Gamma(1-s)\zeta(1-s)\)。

✏️ 例题分析与易错点

例题：\(f(z) = \sum_{n=0}^\infty z^n = 1/(1-z)\)（\(|z| < 1\)）。 \(g(z) = 1/(1-z)\) 定义在 \(\mathbb{C} \setminus \{1\}\) 上。 \(g\) 是 \(f\) 的解析延拓。 原级数在 \(|z| = 1\) 发散，但函数本身在 \(z \neq 1\) 处全纯。 易错点：解析延拓可能导致多值函数。例如 \(\log z\) 沿绕 原点的路径延拓后值变化 \(2\pi i\)——需引入 Riemann 面。

02整函数与增长阶

🎯 为什么研究这个问题

在全复平面上全纯的函数称为整函数。多项式是最简单的整函数， 但 \(e^z\)、\(\sin z\) 等超越整函数增长快得多。整函数的增长速度 由"阶"（order）量化，Hadamard 因式分解定理表明整函数完全 由其零点和增长阶决定——这是解析数论的关键工具。

🔍 抽象化方法

整函数 = 全局幂级数。其增长阶和类型刻画了函数在无穷远处 的行为，Hadamard 理论将零点分布与增长速度联系起来。

📐 公式推导 / 模型建立

整函数的阶：\(\rho = \limsup_{r \to \infty} \frac{\log\log M(r)}{\log r}\) （\(M(r) = \max_{|z|=r} |f(z)|\)）。 - \(e^z\)：\(M(r) = e^r\)，\(\rho = 1\) - 多项式：\(\rho = 0\) - \(e^{z^2}\)：\(\rho = 2\) Weierstrass 因式分解定理：给定零点序列 \(\{a_n\}\)（\(a_n \to \infty\)）， 存在整函数以 \(\{a_n\}\) 为零点集： \[f(z) = z^m e^{g(z)} \prod_{n=1}^{\infty} E_p\left(\frac{z}{a_n}\right)\] \(E_p(u) = (1-u)\exp(u + u^2/2 + \cdots + u^p/p)\)（初等因子）。 Hadamard 因式分解定理：阶 \(\rho\) 有限的整函数： \[f(z) = z^m e^{Q(z)} \prod E_p(z/a_n)\] 其中 \(Q\) 是次数 \(\leq \rho\) 的多项式，\(p \leq \rho\)。

✏️ 例题分析与易错点

例题：\(\sin(\pi z)\) 的 Weierstrass 分解。 解答：零点 \(a_n = n\)（\(n \in \mathbb{Z}\)）。 \(\sin(\pi z) = \pi z \prod_{n=1}^{\infty}(1 - z^2/n^2)\)。 这给出 \(\zeta(2) = \pi^2/6\)（Euler 的证明）。 易错点：初等因子 \(E_p\) 的 \(p\) 不能太小——需要保证 无穷乘积收敛。\(p\) 由零点的增长速度决定。

03Gamma 函数与 Zeta 函数初步

🎯 为什么研究这个问题

Gamma 函数 \(\Gamma(s) = \int_0^\infty t^{s-1}e^{-t} dt\) 将阶乘 从整数推广到复数（\(\Gamma(n+1) = n!\)）。它是整个分析学、数论 和物理学中最重要的特殊函数。Zeta 函数 \(\zeta(s) = \sum n^{-s}\) 编码了素数分布的信息（Euler 积）。

🔍 抽象化方法

\(\Gamma\) 和 \(\zeta\) 是将离散对象（阶乘、素数计数） "连续化"的工具——它们通过解析延拓获得全局意义。

📐 公式推导 / 模型建立

Gamma 函数：\(\Gamma(s) = \int_0^\infty t^{s-1}e^{-t} dt\) （\(\text{Re}\, s > 0\)）。递推：\(\Gamma(s+1) = s\Gamma(s)\)。 极点：\(s = 0, -1, -2, \ldots\)（一阶极点）。 \(\Gamma\) 无零点。 反射公式：\(\Gamma(s)\Gamma(1-s) = \frac{\pi}{\sin(\pi s)}\)。 Stirling 近似：\(\Gamma(s) \sim \sqrt{2\pi} s^{s-1/2} e^{-s}\)（\(|s| \to \infty\)）。 Riemann Zeta 函数：\(\zeta(s) = \sum_{n=1}^\infty n^{-s}\)（\(\text{Re}\, s > 1\)）。 Euler 积：\(\zeta(s) = \prod_p (1 - p^{-s})^{-1}\)。 函数方程：\(\xi(s) = \xi(1-s)\)， 其中 \(\xi(s) = \pi^{-s/2}\Gamma(s/2)\zeta(s)\)。 解析延拓到 \(s = 1\) 处一阶极点，elsewhere 全纯。

🌐 实际应用

素数定理 \(\pi(x) \sim x/\ln x\) 的证明依赖于 \(\zeta(s)\) 在 \(\text{Re}\, s = 1\) 上无零点。Riemann 猜想断言所有非平凡零点 在 \(\text{Re}\, s = 1/2\) 上。

✏️ 例题分析与易错点

例题：\(\Gamma(1/2) = ?\) 解答：\(\Gamma(1/2) = \int_0^\infty t^{-1/2} e^{-t} dt\)。 令 \(t = u^2\)：\(= 2\int_0^\infty e^{-u^2} du = \sqrt{\pi}\)（Gauss 积分）。 或用反射公式：\(\Gamma(1/2)^2 = \pi/\sin(\pi/2) = \pi\)， \(\Gamma(1/2) = \sqrt{\pi}\)。 易错点：\(\Gamma(0)\) 不存在（\(s = 0\) 是极点）。 \(\Gamma(n) = (n-1)!\)，不是 \(n!\)。

04Picard 定理与值分布

🎯 为什么研究这个问题

Liouville 定理说有界整函数是常数。如果放宽到"缺少一个值"呢？ 小 Picard 定理给出更强的结论：非常数整函数最多缺少一个值。 大 Picard 定理将此推广到本性奇点附近。这些深层结果揭示了 全纯函数取值的"必然丰富性"。

📐 公式推导 / 模型建立

小 Picard 定理：非常数整函数 \(f\) 的像至少包含 \(\mathbb{C}\) 中除至多一个点外的所有值。 即：若 \(f\) 不取两个不同的值 \(a, b\)，则 \(f\) 是常数。 例外的一个值可以发生：\(e^z\) 不取 0，但取所有其他复数值。 大 Picard 定理（陈述）：在本性奇点的任意邻域内， 全纯函数取到除至多一个值外的所有复数值——而且取无穷多次。 与 Casorati-Weierstrass 的比较： C-W: 本性奇点附近像稠密（弱结论）。 大 Picard: 像覆盖几乎全部（强得多）。 证明思路（小 Picard）：若 \(f\) 不取 \(0\) 和 \(1\)， 则 \(f\) 可提升到 \(\mathbb{H}\)（万有覆叠）， 得有界全纯函数，由 Liouville 为常数。

✏️ 例题分析与易错点

例题：\(e^z\) 的像是 \(\mathbb{C} \setminus \{0\}\)。 确认只缺 \(0\) 这一个值。对任意 \(w \neq 0\)， \(e^z = w\) 有解 \(z = \log w\)。 例题：\(e^{1/z}\) 在 \(z = 0\)（本性奇点）的任意小邻域内 取到除 \(0\) 外的所有值。 易错点：小 Picard 说"至多缺一个值"是最优的——\(e^z\) 确实缺一个值。但不能缺两个。
