共形映射

01共形映射的定义与基本性质

🎯 为什么研究这个问题

全纯函数不仅是函数——它还是几何变换。导数非零的全纯映射 保持角度不变（共形），这意味着无穷小的"形状"被保持。 共形映射理论将复分析与几何紧密联系，是流体力学、电磁学 和地图投影的数学基础。

🔍 抽象化方法

共形性 = 局部保角 + 保向。\(f'(z_0) \neq 0\) 保证在 \(z_0\) 处 的微观放大因子 \(|f'(z_0)|\) 和旋转角 \(\arg f'(z_0)\) 一致。

📐 公式推导 / 模型建立

共形映射：\(f: D \to \Omega\) 全纯且 \(f'(z) \neq 0\) 在 \(D\) 上。 则 \(f\) 在每一点保持两条曲线的夹角（大小和方向）。 几何意义：\(f(z) \approx f(z_0) + f'(z_0)(z - z_0)\)（局部线性近似）。 乘以 \(f'(z_0) = |f'(z_0)| e^{i\theta}\)： - 伸缩因子 \(|f'(z_0)|\) - 旋转角 \(\theta = \arg f'(z_0)\) 两条曲线的夹角不变。 单叶（单射）全纯函数自动是共形映射（\(f'\) 非零由单射推出）。 反函数：\(f\) 单叶全纯 ⇒ \(f^{-1}\) 全纯（开映射定理的复分析版）。

🌐 实际应用

机翼设计：Joukowski 变换 \(w = z + 1/z\) 将圆映为翼型， 共形性保证流速场正确变换。

✏️ 例题分析与易错点

例题：\(f(z) = z^2\) 在 \(z_0 = 1 + i\) 处的伸缩因子和旋转角。 解答：\(f'(z) = 2z\), \(f'(1+i) = 2(1+i) = 2\sqrt{2} e^{i\pi/4}\)。 伸缩因子 \(= 2\sqrt{2}\)，旋转角 \(= \pi/4 = 45°\)。 易错点：\(z^2\) 在 \(z = 0\) 处 \(f'(0) = 0\)，不共形！ 角度加倍（\(\theta \to 2\theta\)），不保角。

02Möbius 变换

🎯 为什么研究这个问题

Möbius 变换（分式线性变换）\(w = (az + b)/(cz + d)\) 是最基本 的共形映射。它保持"广义圆"（圆与直线）不变，且恰好构成 Riemann 球面的自同构群。任何两个圆盘之间的共形映射都可以 用 Möbius 变换实现，它是更复杂共形映射的基本构件。

🔍 抽象化方法

Möbius 变换 ↔ \(\text{PSL}(2, \mathbb{C})\) 矩阵群。 变换的复合 = 矩阵乘法，求逆 = 矩阵求逆。

📐 公式推导 / 模型建立

定义：\(T(z) = \frac{az + b}{cz + d}\)（\(ad - bc \neq 0\)）。 性质： (1) 将广义圆（圆和直线）映为广义圆 (2) 保持交比：\((z_1, z_2; z_3, z_4) = \frac{(z_1 - z_3)(z_2 - z_4)}{(z_1 - z_4)(z_2 - z_3)}\) (3) 三点唯一确定：指定三对像即可确定 \(T\) (4) \(\text{Aut}(\hat{\mathbb{C}}) = \text{PSL}(2, \mathbb{C})\) 单位圆盘的自同构群：\(T(z) = e^{i\theta} \frac{z - a}{1 - \bar{a}z}\) （\(|a| < 1\)，\(\theta \in \mathbb{R}\)）。 上半平面 ↔ 单位圆盘：\(w = \frac{z - i}{z + i}\)（Cayley 变换）。 将 \(\text{Im}\, z > 0\) 映为 \(|w| < 1\)。

🌐 实际应用

Smith 圆图（微波工程）本质上是 Möbius 变换： 阻抗 ↔ 反射系数的映射 \(\Gamma = (Z - Z_0)/(Z + Z_0)\)。

✏️ 例题分析与易错点

例题：求将 \(0, 1, \infty\) 映为 \(1, i, -1\) 的 Möbius 变换。 解答：\((z, 0; 1, \infty) = z\)，\((w, 1; i, -1) = \frac{(w-i)(1+1)}{(w+1)(1-i)}\)。 令两者相等：\(z = \frac{2(w - i)}{(w + 1)(1 - i)}\)。 解出 \(w = \frac{(1 - i)z + 2i}{(1 - i)z + 2}\)（化简后）。 易错点：\(ad - bc = 0\) 时退化为常数映射，不是 Möbius 变换。

03Riemann 映射定理

🎯 为什么研究这个问题

在复平面中，不同形状的单连通区域看起来很不一样——矩形、 椭圆、多边形等。Riemann 映射定理给出了惊人的结论： 除了 \(\mathbb{C}\) 本身，任何单连通区域都可以共形映射到 单位圆盘！这意味着从共形几何的角度，所有"有洞"以外的 区域都是"一样的"。

🔍 抽象化方法

Riemann 映射定理是复分析中最深刻的存在性定理之一。 证明（Koebe 方法）使用正规族理论——从候选映射族中 用 Montel 定理提取收敛子列。

📐 公式推导 / 模型建立

定理 (Riemann)：设 \(D \subsetneq \mathbb{C}\) 为单连通区域， \(D \neq \mathbb{C}\)。则存在双全纯映射（共形同构）\(f: D \to \mathbb{D}\)。 若固定 \(z_0 \in D\) 和 \(f(z_0) = 0\), \(f'(z_0) > 0\)，则 \(f\) 唯一。 证明框架（不展开细节）： Step 1. 取 \(\mathcal{F} = \{f: D \to \mathbb{D} \text{ 单射全纯},\; f(z_0) = 0\}\)。 Step 2. \(\mathcal{F}\) 非空（利用 \(D \neq \mathbb{C}\)，取平方根分支后适当 Möbius）。 Step 3. 取 \(|f'(z_0)|\) 的上确界——由 Montel 定理（正规族），存在子列 一致收敛于 \(f_0 \in \mathcal{F}\)。 Step 4. 证 \(f_0\) 满射（否则可构造一个 \(|f'(z_0)|\) 更大的映射，矛盾）。 不包括 \(D = \mathbb{C}\)：\(\mathbb{C}\) 不能共形映射到 \(\mathbb{D}\)（Liouville）。

🌐 实际应用

数值共形映射（Schwarz-Christoffel 公式）在计算流体力学和 电场计算中用于将复杂区域映射为简单区域。

✏️ 例题分析与易错点

例题：上半平面 \(\mathbb{H} = \{z : \text{Im}\, z > 0\}\) 共形映射到 \(\mathbb{D}\)。 解答：Cayley 变换 \(f(z) = (z - i)/(z + i)\)。 验证：\(f(i) = 0\), \(f(\mathbb{R}) = \{|w| = 1\}\)（实轴映为单位圆），\(f(\mathbb{H}) = \mathbb{D}\)。 易错点：Riemann 映射定理仅对严格子区域成立。 \(\mathbb{C} \to \mathbb{D}\) 的双全纯映射不存在。 且定理仅保证存在性——显式求出映射通常很困难。

04Schwarz-Christoffel 公式与应用

🎯 为什么研究这个问题

Riemann 映射定理保证了映射的存在性，但如何显式构造？ 对多边形区域，Schwarz-Christoffel 公式给出从上半平面到 多边形的共形映射的导数公式。这是为数不多的能写出 "几乎显式"共形映射的情形，在工程计算中极为重要。

📐 公式推导 / 模型建立

Schwarz-Christoffel 公式：设多边形 \(P\) 的顶点为 \(w_1, \ldots, w_n\)， 内角为 \(\alpha_1\pi, \ldots, \alpha_n\pi\)。从上半平面 \(\mathbb{H}\) 到 \(P\) 的共形映射 \(f\) 满足 \[f'(z) = C \prod_{k=1}^n (z - x_k)^{\alpha_k - 1}\] 其中 \(x_k \in \mathbb{R}\) 是顶点的原像，\(C\) 是常数。 原理：\(\arg f'(z)\) 在实轴上经过 \(x_k\) 时跳变 \((\alpha_k - 1)\pi\)， 恰好使像的方向旋转产生内角 \(\alpha_k \pi\)。 特例： - 无穷条带 \(\{0 < \text{Im}\, w < \pi\}\)：\(f(z) = \log z\) - 等腰直角三角形：椭圆积分 - 矩形：椭圆积分（涉及 Schwarz 函数的反函数）

🌐 实际应用

翼型周围流场：用 S-C 公式将多边形翼型映为半平面， 在半平面上解 Laplace 方程后映射回去。

✏️ 例题分析与易错点

例题：从 \(\mathbb{H}\) 到无穷条带 \(\{0 < \text{Im}\, w < h\}\) 的共形映射。 解答：\(f(z) = \frac{h}{\pi}\log z\)。 验证：\(\text{Im}(\frac{h}{\pi}\log z) = \frac{h}{\pi}\arg z\)。 \(z \in \mathbb{H}\) 时 \(\arg z \in (0, \pi)\)， 故 \(\text{Im}\, f \in (0, h)\)。✓ 易错点：S-C 公式中 \(x_k\) 的具体值（"accessory parameters"） 通常无法显式确定，需数值求解。只有形状对称时才有解析解。
