Taylor 与 Laurent 展开

01Taylor 展开与解析性

🎯 为什么研究这个问题

前面提到全纯函数可以幂级数展开，现在用 Cauchy 积分公式给出 严格证明。Taylor 定理不仅确立了"全纯 = 解析"，还给出了 系数的积分表示和收敛半径的精确计算，是复分析的基础工程。

🔍 抽象化方法

Taylor 展开是全纯函数的"坐标化"——在每个圆盘上， 函数由系数序列唯一确定。

📐 公式推导 / 模型建立

定理：\(f\) 在 \(|z - z_0| < R\) 上全纯。则 \[f(z) = \sum_{n=0}^{\infty} a_n (z - z_0)^n, \quad a_n = \frac{f^{(n)}(z_0)}{n!} = \frac{1}{2\pi i} \oint \frac{f(\zeta)}{(\zeta - z_0)^{n+1}} d\zeta\] 级数在每个紧子集上绝对一致收敛。 证明：由 Cauchy 公式 \(f(z) = \frac{1}{2\pi i}\oint \frac{f(\zeta)}{\zeta - z} d\zeta\)。 \(\frac{1}{\zeta - z} = \frac{1}{(\zeta - z_0)(1 - (z-z_0)/(\zeta - z_0))} = \sum_{n=0}^\infty \frac{(z - z_0)^n}{(\zeta - z_0)^{n+1}}\)。 交换求和与积分（一致收敛），得 Taylor 展开。 收敛半径 = \(z_0\) 到最近奇点的距离。

✏️ 例题分析与易错点

例题：\(f(z) = \frac{1}{2-z}\) 在 \(z_0 = 0\) 展开。 解答：\(\frac{1}{2-z} = \frac{1}{2} \cdot \frac{1}{1 - z/2} = \sum \frac{z^n}{2^{n+1}}\)，\(R = 2\)（奇点 \(z=2\)）。✓ 在 \(z_0 = 1\) 展开：\(\frac{1}{2-%z} = \frac{1}{1-(z-1)} = \sum (z-1)^n\)，\(R = 1\)（到 \(z=2\) 距离 1）。✓ 易错点：展开中心不同，收敛半径不同。 不要默认 \(R\) 不变。

⚡ 快速检测

02Laurent 展开与奇点分类

🎯 为什么研究这个问题

Taylor 展开只对没有奇点的区域有效。奇点附近怎么办？Laurent 展开将幂级数推广为"双向级数"（包含负幂项 \((z-z_0)^{-n}\)）， 能够精确描述函数在奇点附近的行为。负幂部分（主部）完全 决定了奇点的类型——可去奇点、极点或本性奇点。

🔍 抽象化方法

Laurent 展开在环形区域 \(r < |z - z_0| < R\) 成立， 将函数分解为"正则部分"（解析）和"主部"（含奇点信息）。

📐 公式推导 / 模型建立

Laurent 定理：\(f\) 在环形区域 \(r < |z - z_0| < R\) 上全纯。则 \[f(z) = \sum_{n=-\infty}^{\infty} a_n (z - z_0)^n\] \(a_n = \frac{1}{2\pi i} \oint \frac{f(\zeta)}{(\zeta - z_0)^{n+1}} d\zeta\)。 奇点分类（\(z_0\) 为孤立奇点）： - 可去奇点：主部 = 0（\(a_n = 0\)，\(n < 0\)）。\(\lim_{z \to z_0} f\) 存在有限。 - \(m\) 阶极点：\(a_{-m} \neq 0\)，\(a_n = 0\)（\(n < -m\)）。 等价于 \(f(z) = g(z)/(z - z_0)^m\)（\(g\) 全纯, \(g(z_0) \neq 0\)）。 - 本性奇点：主部有无穷多非零项。\(\lim_{z \to z_0} f\) 不存在。 Riemann 可去奇点定理：\(f\) 在 \(z_0\) 附近有界 ⇒ 可去奇点。 Casorati-Weierstrass 定理：本性奇点附近 \(f\) 的像在 \(\mathbb{C}\) 中稠密（大 Picard：甚至取到除至多一个值外的所有值）。

✏️ 例题分析与易错点

例题：\(f(z) = \frac{e^z - 1}{z^2}\) 在 \(z = 0\) 的奇点类型。 解答：\(e^z - 1 = z + z^2/2 + z^3/6 + \cdots\)。 \(f(z) = 1/z + 1/2 + z/6 + \cdots\)。 主部 \(= 1/z\)（一项），故 \(z = 0\) 是一阶极点。 例题：\(e^{1/z}\) 在 \(z = 0\) 的奇点类型。 解答：\(e^{1/z} = 1 + 1/z + 1/(2z^2) + \cdots\)——无穷多负幂项。 本性奇点。 易错点：\(\sin z / z\) 在 \(z = 0\) 看似有奇点，实际是可去的 （\(\lim = 1\)）。不要看到 \(1/z\) 就说是极点。

03零点与极点的代数计数

🎯 为什么研究这个问题

全纯函数的零点有确定的"阶"（重数），极点也有阶。零点和极点 的阶数在围道积分中体现为"绕数"。辐角原理将零点和极点的 计数转化为积分，这是复分析独有的计数工具。

📐 公式推导 / 模型建立

零点的阶：\(f(z_0) = 0\)，\(f(z) = (z - z_0)^m g(z)\)（\(g(z_0) \neq 0\)）， 则 \(z_0\) 是 \(m\) 阶零点。 极点的阶：\(z_0\) 是 \(m\) 阶极点 ⇔ \((z - z_0)^m f(z)\) 在 \(z_0\) 全纯非零。 辐角原理：\(f\) 在区域 \(D\) 内亚纯，\(\gamma = \partial D\)。 \[\frac{1}{2\pi i} \oint_\gamma \frac{f'(z)}{f(z)}\, dz = N - P\] \(N\) = \(D\) 内零点数（计重数），\(P\) = \(D\) 内极点数（计阶数）。 证明：\(f'/f\) 在零点 \(z_0\)（\(m\) 阶）处： \(f'/f = m/(z - z_0) + \) holomorphic，留数 \(= m\)。 在极点处类似获得留数 \(= -\) 阶数。 Rouché 定理：若 \(|f - g| < |g|\) 在 \(\gamma\) 上， 则 \(f\) 和 \(g\) 在 \(\gamma\) 内有相同数目的零点。

🌐 实际应用

控制论中 Nyquist 稳定性判据基于辐角原理—— 系统稳定 ⇔ 闭环传递函数在右半平面无零点。

✏️ 例题分析与易错点

例题：\(f(z) = z^5 + 3z + 1\) 在 \(|z| < 1\) 内有几个零点？ 解答（Rouché）：取 \(g(z) = 3z\)，\(f - g = z^5 + 1\)。 在 \(|z| = 1\)：\(|f - g| = |z^5 + 1| \leq 2\)，\(|g| = 3\)。 \(|f - g| < |g|\)。\(g\) 有 1 个零点（\(z = 0\)），故 \(f\) 也有 1 个零点。 易错点：Rouché 定理中 \(|f - g| < |g|\) 必须在曲线上 而非曲线内部成立。

04半纯函数与级数展开技巧

🎯 为什么研究这个问题

Laurent 展开是局部的——仅在一个环形区域成立。实际计算中， 我们经常需要在不同区域做不同的展开，或对有多个奇点的函数 做部分分式分解。掌握这些展开技巧是计算留数、反 Laplace 变换 和渐近分析的基础。

📐 公式推导 / 模型建立

亚纯函数：仅有极点作为奇点的函数。在 \(\hat{\mathbb{C}}\) 上 亚纯函数 = 有理函数。 部分分式：\(f(z) = \sum_k \sum_{j=1}^{m_k} \frac{c_{k,j}}{(z - z_k)^j} + g(z)\) （\(g\) 整函数）。 不同区域的 Laurent 展开：\(f(z) = 1/((z-1)(z-2))\)。 - \(|z| < 1\)：\(\frac{-1}{z-1} + \frac{1}{z-2} = \sum z^n - \frac{1}{2}\sum (z/2)^n = \sum (1 - 2^{-n-1})z^n\) - \(1 < |z| < 2\)：\(\frac{1}{z} \cdot \frac{1}{1-1/z} - \frac{1}{2}\sum (z/2)^n = \sum_{n=0}^\infty z^{-n-1} - \sum_{n=0}^\infty z^n/2^{n+1}\) - \(|z| > 2\)：展开两项为 \(1/z\) 的幂级数 展开技巧：根据 \(|z|\) 与奇点距离的大小关系选择 \(1/(1-w) = \sum w^n\)（\(|w| < 1\)）或 \(-1/(w-1) = -\sum 1/w^n\)（\(|w| > 1\)）。

🌐 实际应用

信号处理中 Z 变换的收敛域和部分分式分解直接对应 不同区域的 Laurent 展开。

✏️ 例题分析与易错点

例题：\(f(z) = \frac{1}{z(z-1)}\) 在 \(0 < |z| < 1\) 的 Laurent 展开。 解答：部分分式 \(= -1/z + 1/(z-1) = -1/z - 1/(1-z) = -1/z - \sum_{n=0}^\infty z^n = -z^{-1} - 1 - z - z^2 - \cdots\)。 易错点：不同环形区域给出不同的 Laurent 展开！ 同一函数的展开不唯一——取决于环形区域。
