01复可微与 Cauchy-Riemann 方程

🎯 为什么研究这个问题

实函数的可微性要求极限只从左右两个方向趋近。但复数平面是 二维的——极限可以从任意方向趋近。这一更强的条件产生了惊人的 后果：复可微一次 ⇒ 无穷次可微 + 解析，这在实分析中完全不成立。 Cauchy-Riemann 方程将复可微条件翻译为实二维语言。

🔍 抽象化方法

复可微性统一了实部和虚部的耦合关系。CR 方程 \(u_x = v_y\), \(u_y = -v_x\) 是调和方程的特殊形式， 将复分析与 PDE（调和函数论）连接起来。

📐 公式推导 / 模型建立

复导数：\(f'(z_0) = \lim_{h \to 0} \frac{f(z_0 + h) - f(z_0)}{h}\) （\(h \in \mathbb{C}\) 从任意方向趋于 0）。 Cauchy-Riemann 方程：设 \(f = u + iv\)，\(z = x + iy\)。 \(f\) 在 \(z_0\) 复可微 \(\Rightarrow\) \[\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}\] 证明：令 \(h = t\)（实方向）得 \(f' = u_x + iv_x\)； 令 \(h = it\) 得 \(f' = v_y - iu_y\)。两者相等即 CR 方程。 反之，CR 方程 + 偏导数连续 ⇒ 复可微。 推论：\(u\) 和 \(v\) 都是调和函数： \(\Delta u = u_{xx} + u_{yy} = 0\)，\(\Delta v = 0\)。 （由 CR 方程：\(u_{xx} = v_{yx} = -u_{yy}\)。）

🌐 实际应用

流体力学中，不可压缩无旋流的速度势 \(\phi\) 和流函数 \(\psi\) 恰好满足 CR 方程——复势 \(f = \phi + i\psi\) 是全纯的。

✏️ 例题分析与易错点

例题：验证 \(f(z) = z^2 = (x^2 - y^2) + i(2xy)\) 满足 CR 方程。 解答：\(u = x^2 - y^2\), \(v = 2xy\)。 \(u_x = 2x = v_y\)，\(u_y = -2y = -v_x\)。✓ \(f'(z) = u_x + iv_x = 2x + i(2y) = 2(x + iy) = 2z\)。✓ 易错点：\(f(z) = \bar{z} = x - iy\) 不满足 CR（\(u_x = 1 \neq v_y = -1\)）， 故 \(\bar{z}\) 不全纯！共轭运算破坏全纯性。

02全纯函数的基本性质与初等函数

🎯 为什么研究这个问题

复指数函数、三角函数、对数函数在复数域上的行为远比实数域丰富。 \(e^z\) 变成周期函数（虚方向周期 \(2\pi i\)）、\(\ln z\) 变成多值函数、 \(z^a\) 的定义需要选择"分支"。理解这些初等全纯函数是复分析的基础。

📐 公式推导 / 模型建立

复指数：\(e^z = e^x(\cos y + i\sin y)\)（Euler 公式）。 性质：\(e^{z+w} = e^z e^w\)，\(|e^z| = e^x\)，周期 \(2\pi i\)。 复对数：\(\log z = \ln|z| + i\arg z\)（多值！）。 主值：\(\text{Log}\, z = \ln|z| + i\text{Arg}\, z\)（\(\text{Arg} \in (-\pi, \pi]\)）。 \(\text{Log}\) 在负实轴上不连续——分支切割。 复幂：\(z^a = e^{a \log z}\)（一般多值）。 例：\(i^i = e^{i \log i} = e^{i(i\pi/2 + 2k\pi i)} = e^{-\pi/2 - 2k\pi}\)（实数！）。 Liouville 定理：有界整函数为常数。 证明（预览，用 Cauchy 估计）：\(|f'(z)| \leq M/R \to 0\)（\(R \to \infty\)）， 故 \(f' = 0\)。

🌐 实际应用

电气工程中，交流电路分析用 \(e^{i\omega t}\) 表示信号， 阻抗 \(Z = R + iX\) 是全纯函数理论的直接应用。

✏️ 例题分析与易错点

例题：求 \(\log(-1)\) 的所有值。 解答：\(\log(-1) = \ln|-1| + i\arg(-1) = 0 + i(\pi + 2k\pi) = i(2k+1)\pi\)（\(k \in \mathbb{Z}\)）。 主值 \(\text{Log}(-1) = i\pi\)。 易错点：把 \(\log(z_1 z_2) = \log z_1 + \log z_2\) 当作恒等式。 多值函数下等号变为集合相等，取主值后不一定成立。

03解析函数与收敛幂级数

🎯 为什么研究这个问题

实分析中，无穷次可微不代表等于 Taylor 级数（如 \(e^{-1/x^2}\)）。 但复分析中，全纯一次 ⇒ 可以展开为收敛幂级数！这一"全纯 = 解析" 的等价性是复分析最核心的奇迹之一——复可微性远比实可微性强。

🔍 抽象化方法

全纯函数在收敛圆盘内完全由系数决定——幂级数提供了"无穷维坐标"。 收敛半径由最近奇点的距离决定。

📐 公式推导 / 模型建立

幂级数：\(f(z) = \sum_{n=0}^\infty a_n (z - z_0)^n\)。 收敛半径：\(R = 1/\limsup |a_n|^{1/n}\)（Hadamard 公式）。 级数在 \(|z - z_0| < R\) 内绝对一致收敛（紧子集上）， 在 \(|z - z_0| > R\) 发散。 逐项求导：\(f'(z) = \sum n a_n (z - z_0)^{n-1}\)， 收敛半径不变。全纯函数无穷次可微。 核心定理（后证）：全纯 ⇔ 解析（局部可展为幂级数）。 每个全纯函数在其定义域内每点可展开 Taylor 级数： \[f(z) = \sum_{n=0}^{\infty} \frac{f^{(n)}(z_0)}{n!}(z - z_0)^n\] 收敛半径 = \(z_0\) 到最近奇点的距离。

✏️ 例题分析与易错点

例题：\(f(z) = 1/(1-z)\)。求以 \(z_0 = 0\) 为中心的 Taylor 级数和收敛半径。 解答：\(f(z) = \sum_{n=0}^\infty z^n\)，\(R = 1\)。 最近奇点 \(z = 1\)，\(|z_0 - 1| = 1 = R\)。✓ 以 \(z_0 = i\) 为中心呢？\(f(z) = \frac{1}{1-i-(z-i)} = \frac{1}{1-i} \cdot \frac{1}{1 - (z-i)/(1-i)} = \sum \frac{(z-i)^n}{(1-i)^{n+1}}\)。 \(R = |1-i| = \sqrt{2}\)（到奇点 \(z=1\) 的距离）。✓ 易错点：实轴上无奇点不意味实 Taylor 级数收敛半径无穷。 \(1/(1+x^2)\) 实轴处处光滑，但 Taylor 级数在 \(|x| > 1\) 发散—— 因为复奇点 \(z = \pm i\) 距原点为 1。

04全纯函数的刚性与唯一性

🎯 为什么研究这个问题

全纯函数有惊人的"刚性"：如果两个全纯函数在一个有聚点的集合上 相等，则它们在整个连通区域上相等（唯一性定理/零点孤立性）。 这意味着全纯函数完全由"很少的信息"决定——这在实分析中 不可能（光滑函数可以在任意开集外为 0）。

🔍 抽象化方法

零点孤立性是"全纯 = 解析"的直接推论——幂级数不能在非 孤立零点处非零。全纯函数的值在区域内"全局锁定"。

📐 公式推导 / 模型建立

零点孤立性定理：设 \(f\) 在区域 \(D\) 上全纯且 \(f \not\equiv 0\)。 则 \(f\) 的零点集 \(\{z : f(z) = 0\}\) 没有聚点在 \(D\) 内。 证明：设 \(f(z_0) = 0\)。Taylor 展开 \(f(z) = (z - z_0)^m g(z)\)（\(g(z_0) \neq 0\), \(m \geq 1\)）。 \(g\) 连续故在 \(z_0\) 附近 \(g \neq 0\)，故 \(z_0\) 是孤立零点。 唯一性定理：设 \(f, g\) 在连通区域 \(D\) 上全纯，且 \(f = g\) 在 具有 \(D\) 内聚点的子集 \(S\) 上。则 \(f \equiv g\) 在 \(D\) 上。 最大模原理：非常数全纯函数 \(|f|\) 在区域内部不取最大值。 \[\max_{z \in \overline{D}} |f(z)| = \max_{z \in \partial D} |f(z)|\]

🌐 实际应用

解析延拓的理论基础：全纯函数从小区域"唯一"延拓到大区域。

✏️ 例题分析与易错点

例题：若 \(f\) 在单位圆盘上全纯，且 \(f(1/n) = 0\) 对所有 \(n \geq 1\)， 证明 \(f \equiv 0\)。 解答：\(\{1/n\}\) 的聚点 \(0 \in D\)（单位圆盘）。 由唯一性定理 \(f \equiv 0\)。 易错点：零点集可以是无穷的（如 \(\sin z\) 在 \(z = n\pi\) 处为零）， 但不能有聚点在定义域内。\(\sin z\) 的零点 \(n\pi \to \infty\)， 聚点不在任何有界区域内。
