Fourier 变换

01Fourier 变换的定义与基本性质

🎯 为什么研究这个问题

Fourier 级数处理周期函数，但物理世界的信号通常不是周期的。 Fourier 变换将级数推广到非周期函数：把离散频率 \(n\) 推广为 连续频率 \(\xi\)，求和 \(\sum\) 推广为积分 \(\int\)。\(L^1\) 和 \(L^2\) 上的 Fourier 变换是信号处理和 PDE 的基本工具。

🔍 抽象化方法

Fourier 变换是"时域 ↔ 频域"的同构。在 \(L^2\) 上它是 等距同构（Plancherel），在 \(L^1\) 上连续但非满射。

📐 公式推导 / 模型建立

\(L^1\) 定义：\(\hat{f}(\xi) = \int_{\mathbb{R}} f(x) e^{-2\pi i x \xi}\, dx\)。 基本性质： - 线性 - 平移：\(\widehat{f(\cdot - a)}(\xi) = e^{-2\pi i a\xi} \hat{f}(\xi)\) - 调制：\(\widehat{e^{2\pi i bx} f}(\xi) = \hat{f}(\xi - b)\) - 伸缩：\(\widehat{f(\lambda \cdot)}(\xi) = \frac{1}{|\lambda|}\hat{f}(\xi/\lambda)\) - 求导：\(\widehat{f'}(\xi) = 2\pi i\xi \hat{f}(\xi)\) - 卷积：\(\widehat{f * g} = \hat{f} \cdot \hat{g}\) Riemann-Lebesgue：\(f \in L^1 \Rightarrow \hat{f} \in C_0\) （连续且趋于 0）。但 \(\hat{f}\) 不一定在 \(L^1\) 中。 Gauss 函数：\(f(x) = e^{-\pi x^2}\)。 \(\hat{f}(\xi) = e^{-\pi\xi^2}\)（Gauss 函数是 Fourier 变换的不动点！）。

🌐 实际应用

量子力学中，位置表象 \(\psi(x)\) 和动量表象 \(\hat{\psi}(p)\) 通过 Fourier 变换关联；不确定性原理 \(\Delta x \cdot \Delta p \geq \hbar/2\) 是 Fourier 变换的数学性质。

✏️ 例题分析与易错点

例题：\(f(x) = \chi_{[-1/2, 1/2]}(x)\)（特征函数）的 Fourier 变换。 解答：\(\hat{f}(\xi) = \int_{-1/2}^{1/2} e^{-2\pi ix\xi} dx = \frac{\sin(\pi\xi)}{\pi\xi} = \text{sinc}(\xi)\)。 易错点：\(\hat{f} = \text{sinc} \notin L^1(\mathbb{R})\)！ 虽然 \(f \in L^1\)，但 \(\hat{f}\) 不一定在 \(L^1\) 中。 这是反变换公式不能直接对所有 \(L^1\) 函数使用的原因。

02反转公式与 Plancherel 定理

🎯 为什么研究这个问题

如果知道 \(\hat{f}\)，能否恢复 \(f\)？反转公式 \(f(x) = \int \hat{f}(\xi) e^{2\pi ix\xi} d\xi\) 给出肯定回答—— 但需要 \(\hat{f} \in L^1\)。更根本的是 Plancherel 定理： Fourier 变换在 \(L^2\) 上是等距同构，保持 \(L^2\) 范数不变。

🔍 抽象化方法

Plancherel 定理说 Fourier 变换是 \(L^2\) 上的"旋转" （酉变换），保持所有几何结构（范数、内积、正交性）。

📐 公式推导 / 模型建立

反转公式：若 \(f, \hat{f} \in L^1\)，则 \[f(x) = \int_{\mathbb{R}} \hat{f}(\xi) e^{2\pi ix\xi}\, d\xi\] a.e. 成立。 Plancherel 定理：Fourier 变换延拓为 \(L^2(\mathbb{R})\) 上的等距同构（酉算子）： \[\|\hat{f}\|_2 = \|f\|_2, \quad \forall f \in L^2\] （即 \(\int |\hat{f}|^2 = \int |f|^2\)——能量守恒。） Parseval 恒等式：\(\int f\bar{g}\, dx = \int \hat{f}\overline{\hat{g}}\, d\xi\) （内积守恒）。 构造：\(L^1 \cap L^2\) 上定义 Fourier 变换，证明等距性， 再由 \(L^1 \cap L^2\) 在 \(L^2\) 中稠密，唯一延拓到 \(L^2\)。

✏️ 例题分析与易错点

例题：\(f(x) = e^{-|x|}\)。\(\hat{f}(\xi) = \frac{2}{1 + (2\pi\xi)^2}\) （计算：分 \(x > 0\) 和 \(x < 0\) 积分）。 验证 Plancherel：\(\|f\|_2^2 = \int e^{-2|x|} dx = 1\)。 \(\|\hat{f}\|_2^2 = \int \frac{4}{(1 + 4\pi^2\xi^2)^2} d\xi = 1\)（部分分式积分）。✓ 易错点：\(L^2\) Fourier 变换不能用积分定义（\(L^2\) 函数不一定 在 \(L^1\) 中），必须通过密度论证或截断定义。

03Schwartz 空间与缓增分布

🎯 为什么研究这个问题

\(L^1\) 和 \(L^2\) 上的 Fourier 变换各有局限。Schwartz 空间 \(\mathcal{S}(\mathbb{R})\)——快速递降函数空间——是 Fourier 变换 的"完美领地"：\(\mathcal{F}: \mathcal{S} \to \mathcal{S}\) 是同构。 其对偶空间——缓增分布 \(\mathcal{S}'\)——允许对 \(\delta\) 函数等 广义函数做 Fourier 变换，是 PDE 和物理学的标准框架。

🔍 抽象化方法

\(\mathcal{S}\) 是 \(L^1\) 和 \(L^2\) 的公共"核心"， \(\mathcal{S}'\) 是它们的公共"包络"。Fourier 变换在 \(\mathcal{S} \hookrightarrow L^2 \hookrightarrow \mathcal{S}'\) 的每个层次都有良好定义。

📐 公式推导 / 模型建立

Schwartz 空间：\(\mathcal{S}(\mathbb{R}^n) = \{f \in C^\infty : \sup |x^\alpha D^\beta f| < \infty, \forall \alpha, \beta\}\)。 即：\(f\) 和所有导数比任何多项式都衰减得快。 \(\mathcal{F}: \mathcal{S} \to \mathcal{S}\) 是拓扑同构 （\(\hat{f}\) 的光滑性来自 \(f\) 的衰减，\(\hat{f}\) 的衰减来自 \(f\) 的光滑性）。 缓增分布：\(\mathcal{S}' = \mathcal{S}(\mathbb{R}^n)^*\)（连续线性泛函）。 \(L^p \subseteq \mathcal{S}'\)（\(1 \leq p \leq \infty\)）。 \(\delta\) 分布：\(\langle \delta, \varphi \rangle = \varphi(0)\)。 \(\hat{\delta} = 1\)（\(\mathcal{F}\delta(\xi) = \int \delta(x) e^{-2\pi ix\xi} dx = 1\)）。 \(\hat{1} = \delta\)（对偶性）。 分布的 Fourier 变换：\(\langle \hat{T}, \varphi \rangle = \langle T, \hat{\varphi} \rangle\)。

🌐 实际应用

量子场论中，传播子 \(G(x-y) = \int \frac{e^{ip(x-y)}}{p^2 + m^2} d^4p\) 作为缓增分布的 Fourier 变换有严格定义。

✏️ 例题分析与易错点

例题：\(\hat{\delta}' = ?\)（\(\delta'\) 的 Fourier 变换）。 解答：\(\langle \hat{\delta}', \varphi \rangle = \langle \delta', \hat{\varphi} \rangle = -\hat{\varphi}'(0) = -\int(-2\pi ix)\varphi(x) dx = \langle 2\pi ix, \varphi\rangle\)。 故 \(\hat{\delta}' = 2\pi i x\)（作为分布）。 易错点：\(\hat{1}\) 不是普通函数——它是 \(\delta\) 分布。 不能写 \(\hat{1}(\xi) = 0\)（\(\xi \neq 0\)）然后忽略 \(\xi = 0\)。

04不确定性原理

🎯 为什么研究这个问题

一个函数不能同时在时域和频域上都"很集中"——这就是 Heisenberg 不确定性原理的数学形式。在信号处理中，这意味着时间分辨率和 频率分辨率不能同时任意精确。不确定性原理是 Fourier 分析 最深层的概念之一。

🔍 抽象化方法

不确定性原理体现了 Fourier 变换的"互补性"—— 函数的"紧致性"和其 Fourier 变换的"紧致性"互相对立。

📐 公式推导 / 模型建立

Heisenberg 不确定性原理：设 \(f \in L^2(\mathbb{R})\), \(\|f\|_2 = 1\)。 定义位置方差 \((\Delta x)^2 = \int x^2 |f(x)|^2 dx\)， 频率方差 \((\Delta \xi)^2 = \int \xi^2 |\hat{f}(\xi)|^2 d\xi\)。则 \[\Delta x \cdot \Delta \xi \geq \frac{1}{4\pi}\] 等号当且仅当 \(f\) 是 Gauss 函数 \(f(x) = Ce^{-ax^2}\)（\(a > 0\)）。 证明：\(\int |xf|^2 \cdot \int |f'|^2 \geq |\int xf \cdot \overline{f'}|^2\) （Cauchy-Schwarz）。 分部积分：\(\int xf\overline{f'} dx = -\frac{1}{2}\int |f|^2 dx = -1/2\)。 用 \(\widehat{f'}(\xi) = 2\pi i\xi \hat{f}(\xi)\) 将 \(\int |f'|^2\) 换为 \((2\pi)^2 \int \xi^2 |\hat{f}|^2\)。 得 \((\Delta x)^2 (2\pi\Delta\xi)^2 \geq 1/4\)，即 \(\Delta x \Delta\xi \geq 1/(4\pi)\)。

🌐 实际应用

量子力学：\(\Delta x \cdot \Delta p \geq \hbar/2\)（\(p = \hbar\xi\)）。 雷达/通信：脉冲越短，频带越宽。

✏️ 例题分析与易错点

例题：验证 Gauss 函数 \(f(x) = (2a/\pi)^{1/4} e^{-ax^2}\) 达到等号。 解答：\(\|f\|_2 = 1\)。\(\Delta x = 1/(2\sqrt{a})\)。 \(\hat{f}(\xi) = (2/(\pi a))^{1/4} e^{-\pi^2\xi^2/a}\)... 简化后 \(\Delta \xi = \sqrt{a}/(2\pi)\)。 \(\Delta x \cdot \Delta \xi = 1/(4\pi)\)。等号成立。✓ 易错点：不确定性原理的精确常数取决于 Fourier 变换的约定 （\(2\pi\) 放在哪里）。不同教材常数不同。
