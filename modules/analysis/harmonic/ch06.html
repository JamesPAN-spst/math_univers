01多分辨分析与小波基

🎯 为什么研究这个问题

Fourier 分析在频域精确但丢失了时间信息——不知道频率成分 "何时出现"。小波分析通过伸缩和平移的母小波族 \(\psi_{j,k}(x) = 2^{j/2}\psi(2^j x - k)\) 同时捕获时间和频率 信息。多分辨分析（MRA）提供了系统构造小波基的框架。

🔍 抽象化方法

MRA 将 \(L^2\) 分解为嵌套子空间链 \(\cdots V_{-1} \subseteq V_0 \subseteq V_1 \cdots\)， 每层的"新信息" \(W_j = V_{j+1} \ominus V_j\) 由小波基张成。 这是 Littlewood-Paley 频率分层的时域对应。

📐 公式推导 / 模型建立

MRA 公理：\(\{V_j\}_{j \in \mathbb{Z}}\) 满足 (1) \(V_j \subseteq V_{j+1}\) (2) \(f \in V_j \Leftrightarrow f(2\cdot) \in V_{j+1}\)（伸缩） (3) \(\bigcap V_j = \{0\}\), \(\overline{\bigcup V_j} = L^2\) (4) 存在 \(\phi\)（尺度函数）使 \(\{\phi(\cdot - k)\}_{k \in \mathbb{Z}}\) 是 \(V_0\) 的 ONB 小波空间：\(W_j = V_{j+1} \ominus V_j\)（正交补）。 \(L^2 = \bigoplus_{j \in \mathbb{Z}} W_j\)。 存在 \(\psi\)（母小波）使 \(\{2^{j/2}\psi(2^j x - k)\}_{j,k}\) 是 \(L^2\) 的 ONB。 尺度方程：\(\phi(x) = \sqrt{2}\sum_k h_k \phi(2x - k)\)（\(h_k\) 为滤波器系数）。 小波方程：\(\psi(x) = \sqrt{2}\sum_k g_k \phi(2x - k)\)， \(g_k = (-1)^k h_{1-k}\)（正交镜像滤波器）。

🌐 实际应用

JPEG 2000 图像压缩使用 CDF 9/7 小波；去噪通过 小波阈值化处理。

✏️ 例题分析与易错点

例题：Haar 小波。\(\phi = \chi_{[0,1)}\)（尺度函数）。 \(h_0 = h_1 = 1/\sqrt{2}\)。 \(\psi = \chi_{[0,1/2)} - \chi_{[1/2,1)}\)。 验证：\(\psi_{j,k}(x) = 2^{j/2}\psi(2^j x - k)\) 构成 \(L^2(\mathbb{R})\) 的 ONB。 易错点：Haar 小波不连续——对光滑函数逼近效率低。 Daubechies 小波是连续/光滑的替代品，但支集更长。

02Daubechies 小波族

🎯 为什么研究这个问题

Haar 小波太粗糙（不连续），Fourier 基太全局（无局部化）。 Daubechies 构造了一族紧支集正交小波 \(\psi_N\)——随着 \(N\) 增大， 小波越光滑，消失矩越多。这一突破使小波分析从理论走向 实际应用。

📐 公式推导 / 模型建立

消失矩：\(\int x^k \psi(x) dx = 0\)（\(k = 0, 1, \ldots, N-1\)）。 消失矩越多，小波对多项式"视而不见"，更善于捕捉细节。 Daubechies 条件：滤波器 \(h_0, \ldots, h_{2N-1}\) 满足 (1) 正交性：\(\sum_k h_k h_{k+2l} = \delta_{l0}\) (2) 归一化：\(\sum h_k = \sqrt{2}\) (3) \(N\) 阶消失矩条件 最小支集长度 \(= 2N - 1\)。 Daubechies-\(N\) 小波（\(N = 1\)：Haar；\(N = 2\)：db2；\(\ldots\)）。 \(N\) 增大 ⇒ 更光滑（连续性阶约 \(\sim 0.2N\)），但支集更长。 正则性：\(\psi_N \in C^{0.2N}\)（近似），\(N \to \infty\) 时光滑性趋于无穷。

🌐 实际应用

指纹压缩（FBI 标准使用 WSQ 小波），心电图分析（db4 小波 与 QRS 波群形状匹配）。

✏️ 例题分析与易错点

例题：db2（\(N = 2\)）滤波器系数。 解答：\(h_0 = (1+\sqrt{3})/(4\sqrt{2})\), \(h_1 = (3+\sqrt{3})/(4\sqrt{2})\), \(h_2 = (3-\sqrt{3})/(4\sqrt{2})\), \(h_3 = (1-\sqrt{3})/(4\sqrt{2})\)。 支集 \(= [0, 3]\)（\(2N - 1 = 3\)）。一阶消失矩：\(\int \psi = 0\), \(\int x\psi = 0\)。 易错点：Daubechies 小波没有解析表达式（除 Haar）， 只能通过滤波器系数迭代计算。

03离散小波变换与快速算法

🎯 为什么研究这个问题

理论上小波变换是连续的积分变换，但实际应用中我们需要离散版本。 Mallat 的快速小波变换（FWT）利用 MRA 的嵌套结构，通过 滤波器组的级联实现 \(O(N)\) 复杂度的分解与重构——比 FFT 的 \(O(N\log N)\) 还快！

🔍 抽象化方法

FWT 的原理：将信号分解为"逼近系数"（低频）和"细节系数" （高频），逐层递归——每层将采样率减半。

📐 公式推导 / 模型建立

分解：设信号 \(f \in V_J\)（最精细层）。 \(V_J = V_{J-1} \oplus W_{J-1}\)。 逼近系数：\(a_{J-1}[k] = \langle f, \phi_{J-1,k} \rangle = \sum_n h_{n-2k} a_J[n]\) 细节系数：\(d_{J-1}[k] = \langle f, \psi_{J-1,k} \rangle = \sum_n g_{n-2k} a_J[n]\) 即：低通滤波 \(h\) + 下采样 → 逼近系数 高通滤波 \(g\) + 下采样 → 细节系数 递归到 \(J-1, J-2, \ldots, 0\)。 重构：上采样 + 滤波器 \(\tilde{h}, \tilde{g}\)： \(a_j[n] = \sum_k a_{j-1}[k] \tilde{h}_{n-2k} + \sum_k d_{j-1}[k] \tilde{g}_{n-2k}\) 复杂度：\(N + N/2 + N/4 + \cdots = O(N)\)（每层数据量减半）。

🌐 实际应用

MP3 音频压缩的子带编码本质上是小波变换的变体。 地震信号分析中用小波检测不同时刻的频率变化。

✏️ 例题分析与易错点

例题：Haar FWT 一步分解 \(a = [4, 6, 5, 1]\)。 解答：\(h = [1/\sqrt{2}, 1/\sqrt{2}]\), \(g = [1/\sqrt{2}, -1/\sqrt{2}]\)。 \(a'[0] = (4+6)/\sqrt{2} = 5\sqrt{2}\), \(a'[1] = (5+1)/\sqrt{2} = 3\sqrt{2}\)。 \(d'[0] = (4-6)/\sqrt{2} = -\sqrt{2}\), \(d'[1] = (5-1)/\sqrt{2} = 2\sqrt{2}\)。 （或用非归一化版本：逼近 \(= [5, 3]\)，细节 \(= [-1, 2]\)。） 易错点：FWT 输入长度必须是 \(2^J\)（或用边界延拓处理）。 下采样使得信号长度逐层减半。

04时频分析与短时 Fourier 变换

🎯 为什么研究这个问题

Fourier 变换完全丢失时间信息，小波变换在低频处时间分辨率差。 短时 Fourier 变换（STFT）通过窗函数 \(g\) 在每个时刻取一段 信号做 Fourier 变换，同时保留时间和频率信息。Gabor 框架理论 为 STFT 的离散化提供了数学基础。

🔍 抽象化方法

STFT 将函数映到时频平面（phase space）上的函数。 不确定性原理限制了时频分辨率的乘积——Gabor 原子 \(g_{m,n}(x) = e^{2\pi imx}g(x-n)\) 不能太密也不能太稀。

📐 公式推导 / 模型建立

短时 Fourier 变换 (STFT)： \[Vg f(x, \xi) = \int f(t) \overline{g(t-x)} e^{-2\pi it\xi}\, dt\] \(= \langle f, M_\xi T_x g \rangle\)（\(T_x\) 平移, \(M_\xi\) 调制）。 反转（若 \(g, \gamma\) 满足条件）： \(f = \int\int Vg f(x,\xi) M_\xi T_x \gamma\, dx\, d\xi\) Gabor 框架：\(\{e^{2\pi imx}g(x-n)\}_{m,n \in \mathbb{Z}}\) 构成 \(L^2\) 的框架 \(\Leftrightarrow\) 存在 \(A, B > 0\) 使 \[A\|f\|^2 \leq \sum_{m,n} |Vg f(n, m)|^2 \leq B\|f\|^2\] Balian-Low 定理：若 Gabor 系统 \(\{g_{m,n}\}\)（整数格点） 构成 \(L^2\) 的正交基，则 \(g\) 不能同时在时域和频域快速衰减 （\(\int x^2|g|^2 \cdot \int \xi^2|\hat{g}|^2 = \infty\)）。

🌐 实际应用

语音识别中的"频谱图"就是 \(|Vg f(x,\xi)|^2\) 的可视化。 超声波检测中用 Gabor 变换定位缺陷的时间位置和频率特征。

✏️ 例题分析与易错点

例题：以 \(g(t) = e^{-\pi t^2}\)（Gauss 窗）为窗的 STFT 在 \(f(t) = e^{2\pi i \xi_0 t}\)（纯频率）上的结果。 解答：\(Vg f(x, \xi) = \int e^{2\pi i\xi_0 t} e^{-\pi(t-x)^2} e^{-2\pi it\xi} dt\) \(= e^{2\pi ix(\xi_0 - \xi)} e^{-\pi(\xi - \xi_0)^2}\)。 在时频平面上集中在 \(\xi = \xi_0\) 线上（频率精确定位）， 时间方向无局部化（纯频率信号在所有时间都存在）。 易错点：Balian-Low 定理说 Gabor 正交基不能有好的 时频集中性。但 Gabor 框架（非正交）可以有好的集中性。
