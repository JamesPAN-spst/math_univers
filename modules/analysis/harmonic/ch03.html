分布理论

01分布的定义与基本运算

🎯 为什么研究这个问题

经典函数不够用——\(\delta\) "函数"、Heaviside 阶跃函数的导数、 PDE 的基本解都不是传统意义下的函数。Schwartz 的分布理论 将"函数"推广为"作用在测试函数上的连续线性泛函"，使得 每个分布都可以求导、做 Fourier 变换——这为 PDE 理论奠定了 严格的数学基础。

🔍 抽象化方法

分布将"函数"从"取值规则"转变为"对测试函数的作用"。 这种对偶视角允许用积分和极限统一处理所有广义函数。

📐 公式推导 / 模型建立

测试函数空间：\(\mathcal{D}(\mathbb{R}^n) = C_c^\infty(\mathbb{R}^n)\) （光滑紧支集函数），配适当拓扑。 分布：\(T \in \mathcal{D}'\)，即 \(T: \mathcal{D} \to \mathbb{R}\) 线性且连续（\(\varphi_n \to 0\) in \(\mathcal{D} \Rightarrow T(\varphi_n) \to 0\)）。 正则分布：\(f \in L^1_{\text{loc}} \mapsto T_f\)， \(\langle T_f, \varphi \rangle = \int f\varphi\, dx\)。 \(\delta\) 分布：\(\langle \delta, \varphi \rangle = \varphi(0)\)。 不是正则分布（没有 \(L^1_{\text{loc}}\) 函数 \(f\) 使 \(\int f\varphi = \varphi(0)\)）。 运算： - 加法、标量乘法（线性空间） - 与光滑函数相乘：\(\langle aT, \varphi \rangle = \langle T, a\varphi \rangle\) - 求导：\(\langle T', \varphi \rangle = -\langle T, \varphi' \rangle\) - 卷积（与紧支集分布或测试函数）

🌐 实际应用

电磁学中 \(\nabla \cdot \mathbf{E} = \rho/\varepsilon_0\)， 点电荷 \(\rho = q\delta(\mathbf{r})\) 只在分布意义下有意义。

✏️ 例题分析与易错点

例题：\(H(x) = \begin{cases} 1 & x > 0 \\ 0 & x < 0 \end{cases}\)（Heaviside）的分布导数。 解答：\(\langle H', \varphi \rangle = -\langle H, \varphi' \rangle = -\int_0^\infty \varphi'(x) dx = \varphi(0) = \langle \delta, \varphi \rangle\)。 故 \(H' = \delta\)。✓ 易错点：分布的乘积一般无法定义（\(\delta \cdot \delta\) 无意义）。 只有分布与光滑函数的乘积有定义。

⚡ 快速检测
\(\langle x\delta, \varphi \rangle = ?\)

02分布的微分与基本解

🎯 为什么研究这个问题

分布理论最强大的特点是：每个分布都可以任意次求导。不再需要 函数可微！这彻底解决了 PDE 中"解不够光滑"的困境。基本解 （Green 函数）是 \(LG = \delta\) 的解，它通过卷积给出非齐次 方程 \(Lu = f\) 的解 \(u = G * f\)。

🔍 抽象化方法

分布求导使得"导数"成为纯代数运算——不再需要极限过程。 基本解将 PDE 的求解归结为卷积运算。

📐 公式推导 / 模型建立

分布导数：\(\langle D^\alpha T, \varphi \rangle = (-1)^{|\alpha|} \langle T, D^\alpha \varphi \rangle\)。 每个分布都有各阶分布导数。 基本解：若线性微分算子 \(L\) 满足 \(LG = \delta\)， 则 \(G\) 是 \(L\) 的基本解。\(Lu = f\) 的解为 \(u = G * f\) （\(L(G*f) = (LG)*f = \delta * f = f\)）。 例： - Laplacian \(\Delta\)：\(G(x) = \frac{1}{2\pi}\log|x|\)（\(n = 2\)）， \(G(x) = \frac{-1}{4\pi|x|}\)（\(n = 3\)）。 - 热方程 \(\partial_t - \Delta\)：\(G(x,t) = \frac{1}{(4\pi t)^{n/2}} e^{-|x|^2/(4t)}\)。 Malgrange-Ehrenpreis 定理（陈述）：每个常系数线性偏微分 算子都有基本解。

🌐 实际应用

位势论：电荷分布 \(\rho\) 产生的电位 \(\phi = G * \rho\) （\(G\) 是 Laplacian 的基本解）。

✏️ 例题分析与易错点

例题：\(-u'' = f\) 在 \(\mathbb{R}\) 上。基本解是什么？ 解答：\(-G'' = \delta\)。\(G(x) = -|x|/2\)。 验证：\(G'(x) = -\text{sgn}(x)/2\)，\(G'' = -\delta\)。✓ 解：\(u = G * f = -\frac{1}{2}\int |x - y| f(y) dy\)。 易错点：基本解不唯一（差一个齐次方程的解）。 物理中通常取衰减的基本解。

⚡ 快速检测
\(\mathbb{R}^3\) 中 Laplacian 的基本解是？

03缓增分布的 Fourier 变换

🎯 为什么研究这个问题

普通 Fourier 变换要求函数可积或平方可积，但分布不一定满足 这些条件。缓增分布 \(\mathcal{S}'\) 上的 Fourier 变换通过 与 Schwartz 空间的对偶定义，使得 \(\delta\)、多项式、 \(e^{ix}\) 等"函数"都有 Fourier 变换。

🔍 抽象化方法

\(\mathcal{F}: \mathcal{S}' \to \mathcal{S}'\) 是同构， 推广了 \(L^2\) 上的 Plancherel 定理。

📐 公式推导 / 模型建立

定义：对 \(T \in \mathcal{S}'\)， \[\langle \hat{T}, \varphi \rangle = \langle T, \hat{\varphi} \rangle, \quad \forall \varphi \in \mathcal{S}\] 性质： - \(\hat{T}\) 仍为缓增分布 - 与 \(L^1, L^2\) 上的定义相容 - \(\widehat{D^\alpha T} = (2\pi i\xi)^\alpha \hat{T}\) - \(\widehat{x^\alpha T} = (-1)^{|\alpha|}(2\pi i)^{-|\alpha|} D^\alpha \hat{T}\) 经典公式： - \(\hat{\delta} = 1\)，\(\hat{1} = \delta\) - \(\widehat{e^{2\pi iax}} = \delta(\xi - a)\) - \(\widehat{|x|^{-\alpha}}\)：\(= C_\alpha |\xi|^{\alpha - n}\)（适当正则化后） - Poisson 求和公式：\(\sum_{n \in \mathbb{Z}} f(n) = \sum_{n \in \mathbb{Z}} \hat{f}(n)\)

🌐 实际应用

X 射线衍射中，晶体的周期结构 \(\sum \delta(x - na)\) 的 Fourier 变换 \(= (1/a)\sum \delta(\xi - n/a)\)（倒格子），这解释了衍射斑点的分布。

✏️ 例题分析与易错点

例题：\(\widehat{\text{sgn}}(\xi) = ?\)（\(\text{sgn}(x) = \pm 1\)）。 解答：\(\text{sgn}'(x) = 2\delta(x)\)。 \(2\pi i\xi \cdot \widehat{\text{sgn}} = 2\)，故 \(\widehat{\text{sgn}} = \frac{1}{\pi i\xi}\) （主值意义下）\(= \text{p.v.}\frac{1}{\pi i\xi}\)。 易错点：\(\hat{1} = \delta\) 不能从积分 \(\int e^{-2\pi ix\xi} dx\) 直接得到 （积分不收敛）。必须用分布意义定义。

⚡ 快速检测
Poisson 求和公式 \(\sum f(n) = \sum \hat{f}(n)\) 等价于？

04Sobolev 嵌入定理初步

🎯 为什么研究这个问题

分布导数允许定义"弱导数"，从而定义 Sobolev 空间 \(W^{k,p} = \{f \in L^p : D^\alpha f \in L^p, |\alpha| \leq k\}\)。 Sobolev 嵌入定理建立了"弱可微性"与"经典正则性"的桥梁： 足够多的弱导数在 \(L^p\) 中 ⇒ 函数实际上连续。这是 PDE 正则性理论的基石。

🔍 抽象化方法

Sobolev 空间用 Fourier 变换刻画：\(f \in H^s \Leftrightarrow (1 + |\xi|^2)^{s/2}\hat{f} \in L^2\)。更高的 Sobolev 正则性 = Fourier 变换衰减更快。

📐 公式推导 / 模型建立

Sobolev 空间：\(W^{k,p}(\mathbb{R}^n) = \{f \in L^p : D^\alpha f \in L^p, |\alpha| \leq k\}\)， 范数 \(\|f\|_{W^{k,p}} = \sum_{|\alpha| \leq k} \|D^\alpha f\|_p\)。 \(H^s\) 空间：\(H^s(\mathbb{R}^n) = W^{s,2}\)（\(s\) 可以是分数）， \(\|f\|_{H^s}^2 = \int (1 + |\xi|^2)^s |\hat{f}(\xi)|^2 d\xi\)。 Sobolev 嵌入定理（陈述）： - \(kp > n\)（\(k = \) 导数阶, \(p = \) 可积指标, \(n = \) 维数）： \(W^{k,p}(\mathbb{R}^n) \hookrightarrow C(\mathbb{R}^n)\) （甚至 Hölder 连续）。 - \(kp = n\)：临界情形（Trudinger 不等式）。 - \(kp < n\)：\(W^{k,p} \hookrightarrow L^q\)（\(q = np/(n-kp)\)）。 一维特例：\(W^{1,1}(\mathbb{R}) \hookrightarrow C_0(\mathbb{R})\) （\(n = 1, k = 1, p = 1, kp = 1 = n\)）。

🌐 实际应用

椭圆 PDE 正则性：弱解 \(\in H^1\) ⇒ 如果方程系数足够光滑， 则 \(u \in H^{k+2}\) ⇒ \(u \in C^k\)（Sobolev 嵌入）。

✏️ 例题分析与易错点

例题：\(n = 2\), \(f \in W^{1,3}(\mathbb{R}^2)\)。\(f\) 连续吗？ 解答：\(kp = 1 \times 3 = 3 > 2 = n\)。由嵌入定理 \(f\) 连续。✓ 例题：\(n = 3\), \(g \in H^1(\mathbb{R}^3)\)。\(g\) 连续吗？ 解答：\(kp = 1 \times 2 = 2 < 3 = n\)。不保证连续。 但 \(g \in L^6\)（\(q = 3 \times 2/(3 - 2) = 6\)）。 易错点：Sobolev 嵌入不给出 \(W^{k,p} \subseteq L^\infty\) （当 \(kp < n\) 时），只给出 \(L^q\) 嵌入。

⚡ 快速检测
\(W^{2,2}(\mathbb{R}^3)\) 嵌入 \(C(\mathbb{R}^3)\) 吗？
