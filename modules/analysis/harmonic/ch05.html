奇异积分

01Hilbert 变换与共轭函数

🎯 为什么研究这个问题

Fourier 级数的部分和算子在 \(L^p\)（\(1 < p < \infty\)）上有界吗？ 这个问题归结为 Hilbert 变换 \(Hf(x) = \text{p.v.}\frac{1}{\pi}\int \frac{f(t)}{x-t} dt\) 的 \(L^p\) 有界性。Hilbert 变换是原型的"奇异积分算子"， 它连接了调和分析与复分析（对应于全纯函数的实部和虚部）。

🔍 抽象化方法

Hilbert 变换是 Fourier 乘子 \(-i\text{sgn}(\xi)\)——在频域上 它只改变相位（正频率乘 \(-i\)，负频率乘 \(+i\)），不改变模。 这使 \(H\) 成为 \(L^2\) 上的酉算子。

📐 公式推导 / 模型建立

定义：\(Hf(x) = \text{p.v.}\frac{1}{\pi}\int_{-\infty}^{\infty} \frac{f(t)}{x-t}\, dt = \lim_{\varepsilon \to 0} \frac{1}{\pi}\int_{|x-t|>\varepsilon} \frac{f(t)}{x-t}\, dt\) Fourier 乘子：\(\widehat{Hf}(\xi) = -i\,\text{sgn}(\xi)\,\hat{f}(\xi)\)。 故 \(\|Hf\|_2 = \|f\|_2\)（\(L^2\) 等距）。 M. Riesz 定理：\(H: L^p \to L^p\) 有界（\(1 < p < \infty\)）： \[\|Hf\|_p \leq C_p \|f\|_p\] 但 \(H\) 在 \(L^1\) 和 \(L^\infty\) 上不有界（弱 \((1,1)\) 型）。 与复分析的关系：\(f + iHf\) 是上半平面的全纯函数的边界值。 \(u + iv\) 全纯，\(u\) 调和 ⇒ \(v = Hu\)（调和共轭）。

🌐 实际应用

信号处理中的"解析信号"：\(z(t) = f(t) + iHf(t)\)， 瞬时频率 \(= \frac{d}{dt}\arg z(t)\)。

✏️ 例题分析与易错点

例题：\(H(\chi_{[-1,1]})(x) = ?\) 解答：\(Hf(x) = \frac{1}{\pi}\text{p.v.}\int_{-1}^1 \frac{dt}{x-t} = \frac{1}{\pi}\ln\left|\frac{x+1}{x-1}\right|\)（\(x \notin \{-1,1\}\)）。 易错点：\(Hf\) 对 \(f \in L^1\) 可能不在 \(L^1\) 中（只是弱 \(L^1\)）。 需要 \(1 < p < \infty\) 才有 \(L^p\) 有界性。

02Calderón-Zygmund 奇异积分

🎯 为什么研究这个问题

Hilbert 变换是一维的。多维 PDE 需要多维奇异积分——如 Riesz 变换 \(R_j f = c_n \text{p.v.}\int \frac{x_j - y_j}{|x-y|^{n+1}} f(y) dy\)。 Calderón-Zygmund 理论建立了奇异积分算子 \(L^p\) 有界性的 一般框架，是现代调和分析的核心。

🔍 抽象化方法

CZ 理论的精神：奇异积分的核 \(K(x)\) 在原点有不可积奇异性 （\(|K(x)| \sim |x|^{-n}\)），但通过取主值和光滑性条件 （Hörmander 条件），奇异性被"抵消"。

📐 公式推导 / 模型建立

Calderón-Zygmund 核：\(K: \mathbb{R}^n \setminus \{0\} \to \mathbb{C}\) 满足： (1) \(|K(x)| \leq C|x|^{-n}\)（尺寸条件） (2) \(|K(x-y) - K(x)| \leq C|y|/|x|^{n+1}\)（\(|x| > 2|y|\)）（光滑性/Hörmander） CZ 定理：设 \(T: L^2 \to L^2\) 有界，且 \(Tf(x) = \int K(x-y)f(y)dy\) （\(x \notin \text{supp}\, f\)），\(K\) 为 CZ 核。则： - \(T: L^p \to L^p\) 有界（\(1 < p < \infty\)） - \(T: L^1 \to L^{1,\infty}\) 弱 \((1,1)\) 有界 Riesz 变换：\(R_j = \partial_j(-\Delta)^{-1/2}\)， \(\widehat{R_j f}(\xi) = -i\xi_j/|\xi| \hat{f}(\xi)\)。 \(R_j\) 是 CZ 算子。 应用：\(\|\partial_i\partial_j u\|_p \leq C\|\Delta u\|_p\) （二阶导数被 Laplacian 控制——Calderón-Zygmund 不等式）。

🌐 实际应用

椭圆 PDE 正则性：\(-\Delta u = f \in L^p \Rightarrow D^2 u \in L^p\) （Calderón-Zygmund 估计），是 Schauder 理论的 \(L^p\) 版本。

✏️ 例题分析与易错点

例题：\(\mathbb{R}^2\) 中 Riesz 变换 \(R_1 f = \text{p.v.}\frac{1}{2\pi}\int \frac{x_1-y_1}{|x-y|^3} f(y) dy\)。 验证核满足 \(|K(x)| \leq C|x|^{-2}\)。 解答：\(|K(x)| = \frac{|x_1|}{2\pi|x|^3} \leq \frac{1}{2\pi|x|^2}\)。✓（\(n = 2\)） 易错点：CZ 核在原点不可积（\(|x|^{-n}\) 在 \(\mathbb{R}^n\) 中 不局部可积），必须用主值定义积分。

03极大函数与 Calderón-Zygmund 分解

🎯 为什么研究这个问题

Hardy-Littlewood 极大函数 \(Mf(x) = \sup_r \frac{1}{|B_r|}\int_{B_r(x)} |f|\) 是调和分析最基本的工具——它控制了 Fourier 级数、奇异积分和 微分算子的"最坏情形"行为。Calderón-Zygmund 分解将函数 分解为"好部分"（小且有界）和"坏部分"（大但稀疏）， 是证明弱 \((1,1)\) 估计的核心手法。

🔍 抽象化方法

极大函数体现了"平均值控制逐点值"的思想。 CZ 分解体现了"大处稀疏、小处有界"的二分原则。

📐 公式推导 / 模型建立

Hardy-Littlewood 极大函数： \[Mf(x) = \sup_{r > 0} \frac{1}{|B(x,r)|}\int_{B(x,r)} |f(y)|\, dy\] 弱 \((1,1)\) 估计：\(|\{Mf > \lambda\}| \leq \frac{C_n}{\lambda}\|f\|_1\)。 证明用 Vitali 覆盖引理。 \(L^p\) 有界性（\(1 < p \leq \infty\)）： \(\|Mf\|_p \leq C_p \|f\|_p\)（插值 + Marcinkiewicz）。 Calderón-Zygmund 分解：给定 \(f \in L^1\), \(\lambda > 0\)， 分解 \(f = g + b\)（\(g\) = "好", \(b\) = "坏"）： - \(|g| \leq C\lambda\) a.e. - \(b = \sum b_k\), \(b_k\) 支在互不相交的方体 \(Q_k\) 上 - \(\int b_k = 0\), \(\sum |Q_k| \leq C\|f\|_1/\lambda\) Lebesgue 微分定理（极大函数的推论）： a.e. \(x\): \(\lim_{r \to 0} \frac{1}{|B_r|}\int_{B_r(x)} f = f(x)\)。

✏️ 例题分析与易错点

例题：\(f = \chi_{[0,1]}\) 在 \(\mathbb{R}\) 上。\(Mf(x)\) 在 \(x > 1\) 的行为？ 解答：\(x > 1\)：\(Mf(x) = \sup_r \frac{1}{2r}\int_{x-r}^{x+r} \chi_{[0,1]}\)。 最优 \(r\)：取 \(r = x\)，\(Mf(x) \geq \frac{1}{2x}\int_0^1 1\, dt = \frac{1}{2x}\)。 实际 \(Mf(x) = \frac{1}{2x}\)（\(x > 1\)）。衰减如 \(1/x\)，不在 \(L^1\) 中。 易错点：\(f \in L^1\) 不意味 \(Mf \in L^1\)（\(Mf\) 只在弱 \(L^1\) 中）。 对 \(1 < p \leq \infty\)，\(Mf \in L^p\) 当 \(f \in L^p\)。

04Littlewood-Paley 理论

🎯 为什么研究这个问题

分析函数的频率结构不仅要知道总频谱，还要知道不同频率段 的贡献。Littlewood-Paley 理论将函数按频率"分层" （\(f = \sum_j \Delta_j f\)），每层只包含频率约 \(2^j\) 的部分。 关键定理是平方函数估计：\(\|f\|_p \sim \|(\sum |\Delta_j f|^2)^{1/2}\|_p\)， 它将 \(L^p\) 范数的估计分解为逐层估计。

🔍 抽象化方法

LP 分解是"多尺度分析"的原型——函数在不同频率尺度上的 行为近似独立。这一思想贯穿小波分析和偏微分方程。

📐 公式推导 / 模型建立

频率分解：选择光滑函数 \(\psi\) 使 \(\sum_{j \in \mathbb{Z}} |\hat{\psi}(2^{-j}\xi)|^2 = 1\)（\(\xi \neq 0\)）。 定义 \(\Delta_j f = (\hat{\psi}(2^{-j}\cdot)\hat{f})^\vee\)。 \(f = \sum_j \Delta_j f\)（频率分解）。 Littlewood-Paley 定理：对 \(1 < p < \infty\)， \[\|f\|_p \sim \left\|\left(\sum_{j} |\Delta_j f|^2\right)^{1/2}\right\|_p\] 即 \(L^p\) 范数等价于平方函数的 \(L^p\) 范数。 应用：证明 Fourier 乘子定理（Mihlin 定理）， 估计伪微分算子的 \(L^p\) 有界性。 Bernstein 不等式：\(\text{supp}\,\hat{f} \subseteq \{|\xi| \sim 2^j\}\) ⇒ \(\|D^\alpha f\|_p \lesssim 2^{j|\alpha|} \|f\|_p\)。

🌐 实际应用

非线性 PDE 中，Littlewood-Paley 分解用于估计非线性项 的频率交互（paradifferential calculus）。

✏️ 例题分析与易错点

例题：\(f \in L^2(\mathbb{R})\)。验证 \(\|f\|_2^2 = \sum_j \|\Delta_j f\|_2^2\)。 解答：Plancherel + \(\sum |\hat{\psi}(2^{-j}\xi)|^2 = 1\)： \(\sum \|\Delta_j f\|_2^2 = \sum \int |\hat{\psi}(2^{-j}\xi)|^2 |\hat{f}|^2 d\xi = \int |\hat{f}|^2 = \|f\|_2^2\)。✓ （\(p = 2\) 时 LP 定理是 Parseval 的推论。） 易错点：LP 定理 \(p \neq 2\) 时不是直接推论——需要 向量值 CZ 理论或 Stein 的插值论证。
