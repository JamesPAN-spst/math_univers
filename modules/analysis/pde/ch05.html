Fourier 方法

01Fourier 级数在 PDE 中的应用框架

🎯 为什么研究这个问题

Fourier 方法是求解线性 PDE（热方程、波方程、Laplace 方程） 最系统的工具。核心思想是将解展开为正交函数系（三角函数、 Bessel 函数等）的级数，使 PDE 解耦为独立的 ODE。Fourier 在 1807 年提出用三角级数表示任意函数的大胆猜想，开启了整个 现代分析的新纪元。

🔍 抽象化方法

Fourier 方法的本质是在 Hilbert 空间 \(L^2\) 中将函数投影到 算子（如 \(-d^2/dx^2\)）的特征函数基上。PDE 在特征基下对角化， 每个分量独立演化。这是「谱方法」思想的原型。

📐 公式推导 / 模型建立

一般框架（以热方程为例）： 1. 分离变量得到 Sturm-Liouville 问题 \(X'' + \lambda X = 0\)，边界条件 → 特征值 \(\lambda_n\)，特征函数 \(X_n\) 2. 特征函数的正交性：\(\int_0^L X_m X_n\,dx = 0\)（\(m \neq n\)） 3. 初值展开：\(\phi(x) = \sum b_n X_n(x)\)， \(b_n = \frac{\langle\phi, X_n\rangle}{\|X_n\|^2}\) 4. 时间部分独立求解得 \(T_n(t)\) 5. 叠加：\(u(x,t) = \sum b_n T_n(t) X_n(x)\) Dirichlet 条件：\(X_n = \sin(n\pi x/L)\) Neumann 条件：\(X_n = \cos(n\pi x/L)\)（含 \(n=0\) 常数项） 周期条件：\(X_n = e^{in\pi x/L}\)（或 \(\cos, \sin\) 组合） Fourier 系数的计算：利用正交性提取系数 \[b_n = \frac{2}{L}\int_0^L \phi(x)\sin\frac{n\pi x}{L}\,dx\]

🌐 实际应用

- 谱方法是现代科学计算的基础（天气预报中球谐函数展开） - 量子力学中的定态 Schrödinger 方程本质上是 Fourier 方法

02Sturm-Liouville 理论

🎯 为什么研究这个问题

Fourier 方法中反复出现的特征值问题 \(X'' + \lambda X = 0\) 是 一般 Sturm-Liouville 问题的特例。Sturm-Liouville 理论保证 特征值实数、特征函数正交、形成完备基——这为 Fourier 方法 提供了严格的数学基础，也推广到圆柱坐标(Bessel)和球坐标(Legendre)。

🔍 抽象化方法

Sturm-Liouville 问题是无穷维的「特征值问题」：自伴算子在 \(L^2\) 上的谱分解。特征函数基的完备性类比于有限维的特征向量分解， Parseval 等式类比于 \(\|x\|^2 = \sum|x_i|^2\)。

📐 公式推导 / 模型建立

一般 Sturm-Liouville 问题： \[-(p(x)y')' + q(x)y = \lambda w(x)y, \quad x \in [a,b]\] 配以边界条件（Dirichlet/Neumann/Robin/周期）。 定理（正则 SL 问题）： 1. 特征值实数且可列 \(\lambda_1 < \lambda_2 < \cdots \to \infty\) 2. 对应不同特征值的特征函数正交：\(\int_a^b w(x)y_m y_n\,dx = 0\)（\(m \neq n\)） 3. 特征函数 \(\{y_n\}\) 在 \(L^2_w([a,b])\) 中构成完备正交基 4. 第 \(n\) 个特征函数在 \((a,b)\) 内恰有 \(n-1\) 个零点 证明要点（实特征值）：Sturm-Liouville 算子 \(L[y] = -(py')'+qy\) 关于权函数 \(w\) 的内积是自伴的 \(\langle Ly, z\rangle_w = \langle y, Lz\rangle_w\) （通过分部积分验证），自伴算子的特征值必为实数。

🌐 实际应用

- 量子力学中 \(-\hbar^2/(2m)\psi'' + V\psi = E\psi\) 就是 Sturm-Liouville 问题 - Bessel 方程、Legendre 方程是特殊的 SL 问题

✏️ 例题分析与易错点

例题：验证 \(y'' + \lambda y = 0\)，\(y(0)=y(\pi)=0\) 的特征值和零点规律。 解答：\(\lambda_n = n^2\)，\(y_n = \sin nx\)（\(n=1,2,\ldots\)）。 \(y_1 = \sin x\) 在 \((0,\pi)\) 内 0 个零点。 \(y_2 = \sin 2x\) 在 \((0,\pi)\) 内 1 个零点（\(x=\pi/2\)）。 \(y_n\) 在 \((0,\pi)\) 内 \(n-1\) 个零点。✓ 符合 Sturm-Liouville 理论预测。 易错点：完备性（\(y_n\) 构成基）不等于逐点收敛。Fourier 级数在 不连续点处收敛到左右极限的平均值，不是函数值本身。

03Fourier 变换方法

🎯 为什么研究这个问题

当域是全空间 \(\mathbb{R}^n\) 时，分离变量不产生离散特征值， 取而代之的是连续谱——这正是 Fourier 变换发挥作用的场景。 Fourier 变换将微分算子 \(\partial/\partial x\) 变为乘以 \(i\xi\)， 将 PDE 变为关于频率变量的代数方程（或 ODE），极大简化了求解。

🔍 抽象化方法

Fourier 变换是 \(L^2(\mathbb{R}^n)\) 上的酉变换（Plancherel 定理）， 将空间域问题转化为频率域问题。微分 \(\leftrightarrow\) 乘法、 卷积 \(\leftrightarrow\) 乘积的对偶关系是 PDE 求解的核心工具。

📐 公式推导 / 模型建立

Fourier 变换求解热方程（全空间）：\(u_t = \kappa u_{xx}\)。 \(\hat{u}_t = -4\pi^2\kappa\xi^2 \hat{u}\)（\(\partial_x^2 \to -4\pi^2\xi^2\)）。 这是关于 \(t\) 的 ODE：\(\hat{u}(\xi,t) = \hat{\phi}(\xi)e^{-4\pi^2\kappa\xi^2 t}\)。 反变换 \(u = \mathcal{F}^{-1}[\hat{\phi} \cdot \hat{\Phi}] = \phi * \Phi\)，得到卷积公式。 Fourier 变换求解波方程：\(u_{tt} = c^2 u_{xx}\)。 \(\hat{u}_{tt} = -4\pi^2 c^2\xi^2 \hat{u}\)，即 \(\hat{u}_{tt} + \omega^2(\xi)\hat{u} = 0\)，\(\omega = 2\pi c|\xi|\)。 每个频率分量是简谐振动！ \[\hat{u}(\xi,t) = \hat{\phi}(\xi)\cos(2\pi c|\xi|t) + \frac{\hat{\psi}(\xi)}{2\pi c|\xi|}\sin(2\pi c|\xi|t)\] Fourier 变换求解 Poisson 方程：\(\Delta u = f\)。 \(-4\pi^2|\xi|^2\hat{u} = \hat{f}\)，\(\hat{u} = -\hat{f}/(4\pi^2|\xi|^2)\)。 反变换得 \(u = \Phi * f\)（\(\Phi\) 是基本解）。

🌐 实际应用

- 信号处理中的频域滤波直接对应 PDE 的 Fourier 方法 - 光学中 Fraunhofer 衍射是波方程 Fourier 变换解的远场近似

✏️ 例题分析与易错点

例题：用 Fourier 变换求解 \(u_t + au_x = 0\)（\(\mathbb{R}\) 上）。 解答：\(\hat{u}_t + 2\pi ia\xi\hat{u} = 0\)（ODE）， \(\hat{u}(\xi,t) = \hat{\phi}(\xi)e^{-2\pi ia\xi t}\)。 反变换利用时移性：\(u(x,t) = \phi(x-at)\)（右行波）。与特征线法一致。 易错点：Fourier 变换的不同约定（\(2\pi\) 放在指数还是系数中） 会导致公式差异。需始终使用同一约定。

04非齐次问题与 Duhamel 原理

🎯 为什么研究这个问题

真实世界的热源、外力驱动导致非齐次 PDE \(u_t - \kappa\Delta u = f(x,t)\)。 Duhamel 原理将非齐次问题分解为一族齐次初值问题的「叠加」， 类比于 ODE 中的常数变易法。这是将齐次解理论推广到非齐次情况 的系统方法。

🔍 抽象化方法

Duhamel 原理的核心是线性叠加 + 时间延迟：将源项 \(f(x,s)\) 在每个 时间片 \(ds\) 上看作一个瞬时脉冲，每个脉冲从 \(t = s\) 开始演化 \(t - s\) 秒后叠加。

📐 公式推导 / 模型建立

Duhamel 原理（热方程版）： 问题：\(u_t - \kappa\Delta u = f(x,t)\)，\(u(x,0) = 0\)。 设 \(w(x,t;s)\) 是齐次问题 \(w_t - \kappa\Delta w = 0\)（\(t > s\)）， \(w(x,s;s) = f(x,s)\) 的解。则 \[u(x,t) = \int_0^t w(x,t;s)\,ds\] 证明：\(u_t = w(x,t;t) + \int_0^t w_t(x,t;s)\,ds = f(x,t) + \int_0^t \kappa\Delta w\,ds\) \(= f(x,t) + \kappa\Delta u\)。✓ 全空间具体公式： \[u(x,t) = \int_0^t \int_{\mathbb{R}^n} \Phi(x-y, t-s) f(y,s)\,dy\,ds\] 其中 \(\Phi\) 是热核。这是「Green 函数法」的时空版本。 波方程的类似形式：\(u_{tt} = c^2\Delta u + f\)， \(u(x,t) = \int_0^t w(x,t;s)\,ds\)，\(w\) 满足齐次波方程且 \(w(x,s;s)=0\)，\(w_t(x,s;s)=f(x,s)\)。

🌐 实际应用

- 加热棒问题：外部热源 \(f(x,t)\) 下的温度场 - 受迫振动：外力驱动下的弦/膜振动

✏️ 例题分析与易错点

例题：口述 Duhamel 原理如何将 \(u_t - u_{xx} = e^{-t}\sin x\)（\(u(x,0)=0\)，\(x \in \mathbb{R}\)） 化归为齐次问题。 解答：对每个 \(s \in [0,t]\)，求解 \(w_t = w_{xx}\)（\(t > s\)），\(w(x,s) = e^{-s}\sin x\)。 解：\(w(x,t;s) = e^{-s}\sin x \cdot e^{-(t-s)}\)（因 \(\sin x\) 是 \(-\partial_{xx}\) 的特征函数）。 \(u = \int_0^t e^{-s}\sin x \cdot e^{-(t-s)}\,ds = \sin x \cdot e^{-t}\int_0^t 1\,ds = t e^{-t}\sin x\)。 验证：\(u_t = (1-t)e^{-t}\sin x\)，\(u_{xx} = -te^{-t}\sin x\)， \(u_t - u_{xx} = (1-t+t)e^{-t}\sin x = e^{-t}\sin x\)。✓ 易错点：Duhamel 积分的上限是 \(t\)（不是 \(\infty\)）， 且被积函数中 \(w\) 的时间参数是 \(t-s\)（不是 \(t\)）。
