热方程

01热方程的物理推导与基本性质

🎯 为什么研究这个问题

热方程 \(u_t = \kappa \Delta u\) 是人类最早研究的 PDE 之一（Fourier, 1822）。 它不仅描述热传导，还出现在扩散过程、金融中的 Black-Scholes 模型、 概率论中 Brown 运动的密度函数等众多领域。热方程具有的平滑化效应 和无穷传播速度是其最显著的数学特征。

🔍 抽象化方法

热方程是典型的抛物型 PDE，其解算子是一个「平滑化算子」： 即使初值不连续，正时间后的解也是无穷可微的。这与双曲型方程 保持奇异性的行为形成鲜明对比。

📐 公式推导 / 模型建立

物理推导：Fourier 热传导定律 \(\mathbf{q} = -\kappa\nabla u\)（热流正比于温度梯度） 和能量守恒 \(\partial u/\partial t = -\text{div}\,\mathbf{q}\) 结合得 \[\frac{\partial u}{\partial t} = \kappa \Delta u = \kappa\left(\frac{\partial^2 u}{\partial x_1^2} + \cdots + \frac{\partial^2 u}{\partial x_n^2}\right)\] 基本性质： 1. 线性：解的叠加原理成立 2. 平滑化：\(u(\cdot, t) \in C^\infty\) 对所有 \(t > 0\)，即使初值仅 \(L^1\) 3. 无穷传播速度：初值仅在一点非零，正时间后 \(u > 0\) 处处成立 4. 不可逆性：热方程反向 \(u_t = -\kappa\Delta u\) 不适定（Hadamard） 基本解/热核（\(n\) 维）： \[\Phi(x,t) = \frac{1}{(4\pi\kappa t)^{n/2}} \exp\left(-\frac{|x|^2}{4\kappa t}\right), \quad t > 0\]

🌐 实际应用

- 金融数学：Black-Scholes 方程通过变量替换可化为热方程 - AI 中的扩散模型(DDPM)的正向过程对应加噪 ≈ 热方程演化

✏️ 例题分析与易错点

例题：验证热核 \(\Phi(x,t) = (4\pi\kappa t)^{-1/2}e^{-x^2/(4\kappa t)}\) 满足一维热方程。 解答：\(\Phi_t = \Phi\left(-\frac{1}{2t} + \frac{x^2}{4\kappa t^2}\right)\)， \(\Phi_{xx} = \Phi\left(-\frac{1}{2\kappa t} + \frac{x^2}{4\kappa^2 t^2}\right)\)， \(\kappa\Phi_{xx} = \Phi\left(-\frac{1}{2t} + \frac{x^2}{4\kappa t^2}\right) = \Phi_t\)。✓ 易错点：热核在 \(t = 0\) 时不是普通函数而是 \(\delta\)-分布。 \(\Phi(\cdot, t) \to \delta(x)\) 当 \(t \to 0^+\)。

02分离变量法与 Fourier 级数解

🎯 为什么研究这个问题

分离变量法是求解矩形/圆域上 PDE 最经典的方法。基本思想是 猜测解可以写成各变量乘积的形式 \(u(x,t) = X(x)T(t)\)，将 PDE 分解为若干 ODE。每个 ODE 独立求解后，通过 Fourier 级数叠加 满足初值条件的整体解。这就是 Fourier（1807）的革命性方法。

🔍 抽象化方法

分离变量法的数学本质是将 PDE 算子的解展开为特征函数序列。 Fourier 级数是 Sturm-Liouville 理论（\(L^2\) 上的谱理论）在特殊情况 （三角函数系）下的实现。这开启了从有限维到无穷维的跨越。

📐 公式推导 / 模型建立

问题：\(u_t = \kappa u_{xx}\)，\(0 < x < L\)，\(u(0,t)=u(L,t)=0\)，\(u(x,0) = \phi(x)\)。 步骤 1：设 \(u = X(x)T(t)\)，代入得 \(XT' = \kappa X''T\)，\(T'/(\kappa T) = X''/X = -\lambda\)（常数）。 步骤 2：空间 ODE \(X'' + \lambda X = 0\)，\(X(0) = X(L) = 0\)。 特征值 \(\lambda_n = (n\pi/L)^2\)，\(n = 1,2,\ldots\) 特征函数 \(X_n(x) = \sin(n\pi x/L)\)。 步骤 3：时间 ODE \(T_n' = -\kappa\lambda_n T_n\)，\(T_n(t) = e^{-\kappa(n\pi/L)^2 t}\)。 步骤 4：叠加 \(u(x,t) = \sum_{n=1}^\infty b_n \sin\frac{n\pi x}{L} e^{-\kappa(n\pi/L)^2 t}\)， \(b_n = \frac{2}{L}\int_0^L \phi(x)\sin\frac{n\pi x}{L}\,dx\)。

🌐 实际应用

- 金属棒两端保持零温度时的温度衰减 - 量子力学中无穷势阱的能级问题用相同的特征值方法

✏️ 例题分析与易错点

例题：求解 \(u_t = u_{xx}\)，\(u(0,t)=u(\pi,t)=0\)，\(u(x,0) = \sin 2x\)。 解答：初值恰好是 \(n=2\) 的特征函数：\(b_2 = 1\)，\(b_n = 0\)（\(n \neq 2\)）。 \(u(x,t) = \sin 2x \cdot e^{-4t}\)。温度按指数衰减，高频模衰减更快。 易错点：Fourier 系数的计算中，正交性 \(\int_0^L \sin(m\pi x/L)\sin(n\pi x/L)\,dx = 0\) （\(m \neq n\)）是关键。忘记归一化因子 \(2/L\) 会导致系数错误。

03热核与全空间初值问题

🎯 为什么研究这个问题

在无穷域 \(\mathbb{R}^n\) 上，没有边界条件的束缚，初值问题 \(u(x,0) = \phi(x)\) 的解可以用热核的卷积优美地表达。热核是 热方程的「脉冲响应」——它描述了一点热源的温度分布如何随时间 扩散。热核的高斯结构与概率论中正态分布的密切联系并非巧合。

🔍 抽象化方法

热核 \(\Phi(x,t)\) 是热方程的 Green 函数（基本解）。 解 \(u = \Phi * \phi\) 的卷积结构是线性 PDE 的普遍特征， 它将初值问题转化为积分运算。

📐 公式推导 / 模型建立

全空间初值问题：\(u_t = \kappa\Delta u\)，\(u(x,0) = \phi(x)\)，\(x \in \mathbb{R}^n\)。 解的表达式（卷积）： \[u(x,t) = (\Phi(\cdot,t) * \phi)(x) = \int_{\mathbb{R}^n} \frac{1}{(4\pi\kappa t)^{n/2}} e^{-|x-y|^2/(4\kappa t)} \phi(y)\,dy\] 性质： 1. 对 \(t > 0\)，\(u \in C^\infty\)（即使 \(\phi\) 不连续） 2. \(u(\cdot, t) \to \phi\) 当 \(t \to 0^+\)（在适当意义下） 3. \(\int_{\mathbb{R}^n} u(x,t)\,dx = \int_{\mathbb{R}^n} \phi(x)\,dx\)（质量守恒） Fourier 变换视角：\(\hat{u}(\xi, t) = \hat{\phi}(\xi)e^{-4\pi^2\kappa|\xi|^2 t}\)， 每个频率分量 \(\xi\) 以速率 \(4\pi^2\kappa|\xi|^2\) 指数衰减。

🌐 实际应用

- Brown 运动的转移密度恰好是热核：\(p(x,t|y,0) = \Phi(x-y,t)\) - 图像处理中高斯模糊 = 热方程在图像上的短时演化

✏️ 例题分析与易错点

例题：求 \(u_t = u_{xx}\)，\(u(x,0) = e^{-|x|}\) 在 \(\mathbb{R}\) 上的解。 解答：\(u(x,t) = \int_{-\infty}^\infty \frac{1}{\sqrt{4\pi t}}e^{-(x-y)^2/(4t)} e^{-|y|}\,dy\)。 利用 Fourier 变换：\(\hat{\phi}(\xi) = \frac{2}{1+4\pi^2\xi^2}\)， \(\hat{u}(\xi,t) = \frac{2e^{-4\pi^2\xi^2 t}}{1+4\pi^2\xi^2}\)。反变换需数值计算。 易错点：热核公式要求 \(\phi\) 增长不太快（至多 \(e^{a|x|^2}\)）。 若 \(\phi(x) = e^{x^2}\)，卷积积分发散，解不存在。

04极大值原理与唯一性

🎯 为什么研究这个问题

极大值原理是 PDE 理论中最强大的定性工具之一。对于热方程， 它断言：温度的最大值只能出现在初始时刻或边界上，不会在 内部自发产生——这符合热力学第二定律的直觉。极大值原理可以 直接推出解的唯一性和对数据的连续依赖。

🔍 抽象化方法

极大值原理将 PDE 解的整体控制化归为边界/初值的控制。 它不需要解的显式公式，是纯定性的工具。这种思想在椭圆型和 抛物型方程中都成立，但双曲型方程没有类似的原理。

📐 公式推导 / 模型建立

弱极大值原理：设 \(u_t \leq \kappa\Delta u\) 在 \(Q_T = \Omega \times (0,T]\) 内， 则 \(\max_{\overline{Q_T}} u = \max_{\Gamma_T} u\)， 其中 \(\Gamma_T = (\Omega \times \{0\}) \cup (\partial\Omega \times [0,T])\)（抛物边界）。 证明思路（\(\kappa = 1\)）：反证。设 \(u\) 在内部点 \((x^*, t^*)\) 取最大值。 在该点 \(u_t \geq 0\)（因右边无法增大），\(\Delta u \leq 0\)（极值的 Hessian 负半定）。 但 \(u_t = \Delta u\) 要求两者相等，矛盾可通过微扰 \(v = u - \epsilon t\) 锐化。 唯一性推论：设 \(u_1, u_2\) 均解热方程且同初边值。 \(w = u_1 - u_2\) 满足 \(w_t = \kappa\Delta w\)，\(w|_{\Gamma_T} = 0\)。 对 \(w\) 和 \(-w\) 用极大值原理：\(\max w \leq 0\) 且 \(\min w \geq 0\)，即 \(w \equiv 0\)。 连续依赖：\(\|u_1 - u_2\|_\infty \leq \|\phi_1 - \phi_2\|_\infty + \|g_1 - g_2\|_\infty\)。

🌐 实际应用

- 扩散模型中将极大值原理推广到更复杂的抛物型 PDE 来保证正性
