Green 函数

01Green 函数的定义与物理意义

🎯 为什么研究这个问题

Green 函数 \(G(x,y)\) 描述在 \(y\) 点施加单位点源时系统在 \(x\) 处的 响应。一旦求出 Green 函数，任意源 \(f\) 的解可通过卷积 \(u(x) = \int G(x,y)f(y)\,dy\) 一次性写出。Green 函数是线性 PDE 最强大的求解框架：解的存在性、正则性、渐近行为都可从 \(G\) 读出。

🔍 抽象化方法

Green 函数是微分算子 \(L\) 的逆算子的积分核：\(L^{-1}f(x) = \int G(x,y)f(y)\,dy\)。 在分布论中，\(G\) 是方程 \(LG = \delta_y\) 的解。将 PDE 问题转化为 积分方程，是泛函分析「算子理论」的核心思想。

📐 公式推导 / 模型建立

定义：设 \(\Omega\) 是有界域，\(G(x,y)\) 满足 \[-\Delta_x G(x,y) = \delta(x-y), \quad x \in \Omega\] \(G(x,y) = 0\)（\(x \in \partial\Omega\)，Dirichlet 情况）。 分解：\(G(x,y) = \Phi(x-y) + H(x,y)\)，其中 \(\Phi\) 是基本解（自由空间 Green 函数），\(H\) 是调和修正项 \(\Delta_x H = 0\)（\(x \in \Omega\)），\(H(x,y) = -\Phi(x-y)\)（\(x \in \partial\Omega\)）。 用 Green 函数求解 Poisson 方程：\(-\Delta u = f\) in \(\Omega\)，\(u|_{\partial\Omega} = g\)。 Green 第二恒等式得： \[u(y) = \int_\Omega G(x,y)f(x)\,dx + \int_{\partial\Omega} g(x)\frac{\partial G}{\partial \nu_x}(x,y)\,dS_x\] Green 函数的对称性（重要！）：\(G(x,y) = G(y,x)\)。 证明：令 \(u = G(\cdot,y)\)，\(v = G(\cdot,z)\)，Green 第二恒等式 → \(\int_\Omega(u\Delta v - v\Delta u) = 0\)，而 \(\Delta u = -\delta_y\)，\(\Delta v = -\delta_z\)， 得 \(u(z) - v(y) = 0\)，即 \(G(z,y) = G(y,z)\)。

🌐 实际应用

- 电磁学中的「像电荷法」就是 Green 函数的物理实现 - 结构力学中的影响函数就是 Green 函数

✏️ 例题分析与易错点

例题：求 \(\mathbb{R}^3\) 中 Laplace 方程的基本解（自由空间 Green 函数）。 解答：对 \(-\Delta\Phi = \delta\) 在以 \(0\) 为心的球面 \(B_r\) 上积分： \(\int_{B_r}-\Delta\Phi = 1\)，即 \(-\Phi'(r)\cdot 4\pi r^2 = 1\)， \(\Phi'(r) = -1/(4\pi r^2)\)，\(\Phi(r) = 1/(4\pi r)\)。 三维基本解：\(\Phi(x) = \frac{1}{4\pi|x|}\)。 易错点：二维基本解是 \(\Phi = -\frac{1}{2\pi}\ln|x|\)（注意负号）， 归一化常数随维数不同。别把二维和三维的公式搞混。

02镜像法（反射法）

🎯 为什么研究这个问题

对特殊几何区域（半空间、球、楔形域），可以用「虚拟像源」 的方法巧妙地构造 Green 函数，不需要求解 Laplace 方程。 镜像法是物理直觉（电像法）与数学严格性的完美结合，也是 数值方法 BEM 的理论基础之一。

🔍 抽象化方法

镜像法利用 Laplace 方程的对称性：像源放在域外， 使 \(G|_{\partial\Omega} = 0\) 自动满足。本质上是找到 使修正项 \(H\) 具有解析表达式的几何配置。

📐 公式推导 / 模型建立

半空间 \(\{x_3 > 0\}\) 的 Green 函数： 对源点 \(y = (y_1, y_2, y_3)\)（\(y_3 > 0\)），取像点 \(y^* = (y_1, y_2, -y_3)\)。 \[G(x,y) = \Phi(x-y) - \Phi(x-y^*) = \frac{1}{4\pi|x-y|} - \frac{1}{4\pi|x-y^*|}\] 当 \(x_3 = 0\) 时 \(|x-y| = |x-y^*|\)，故 \(G = 0\) ✓。 \(-\Delta_x G = \delta(x-y)\)（\(x_3 > 0\)）因为 \(y^*\) 不在域内。 球域的 Green 函数（Kelvin 反演）：设域为 \(B_R(0)\)，源点 \(y\)。 像点 \(y^* = \frac{R^2}{|y|^2}y\)（关于球的反演点），像强度修正为 \(R/|y|\)： \[G(x,y) = \Phi(x-y) - \frac{R}{|y|}\Phi(x-y^*)\] 验证 \(x \in \partial B_R\) 时 \(|x-y^*|/|x-y| = R/|y|\) → \(G = 0\)。 Poisson 积分公式（球的 Dirichlet 问题的显式解）： \[u(y) = \frac{R^2 - |y|^2}{4\pi R}\int_{|x|=R}\frac{g(x)}{|x-y|^3}\,dS_x\]

🌐 实际应用

- 静电学中的像电荷法（接地导体平面/球面附近的点电荷） - 计算流体力学中壁面附近的涡像

✏️ 例题分析与易错点

例题：求上半平面 \(\{y > 0\}\)（二维）中 Laplace 方程的 Green 函数。 解答：源点 \((a,b)\)（\(b > 0\)），像点 \((a,-b)\)。 \(G(x,y;a,b) = -\frac{1}{2\pi}\ln\sqrt{(x-a)^2+(y-b)^2} + \frac{1}{2\pi}\ln\sqrt{(x-a)^2+(y+b)^2}\) \(= \frac{1}{4\pi}\ln\frac{(x-a)^2+(y+b)^2}{(x-a)^2+(y-b)^2}\)。 验证 \(y = 0\) 时分子 = 分母，\(G = 0\)。✓ 易错点：像点必须在域外，否则 \(\delta\) 项会出现在域内， Green 函数就不满足原方程。

03Green 函数的性质与 Poisson 核

🎯 为什么研究这个问题

Green 函数不仅是求解工具，它本身蕴含丰富的信息： 法向导数 \(\partial G/\partial\nu\) 就是 Poisson 核，给出了 Dirichlet 问题 解的边界积分表示。Green 函数的性质（正性、对称性、奇异性） 直接反映了 PDE 解的定性行为。

🔍 抽象化方法

Green 函数将 PDE 问题完全转化为边界积分问题（BIE）， 降低了一维。这是边界元方法（BEM）的理论基础，也是 位势论的核心工具。

📐 公式推导 / 模型建立

Green 表示公式：\(u \in C^2(\bar\Omega)\)，\(-\Delta u = f\)，则 \[u(y) = \int_\Omega G(x,y)f(x)\,dx + \int_{\partial\Omega} g(x)\frac{\partial G}{\partial\nu_x}(x,y)\,dS_x\] Poisson 核：\(P(x,y) = \frac{\partial G}{\partial\nu_x}(x,y)\)（\(x \in \partial\Omega\)）。 性质：\(P(x,y) > 0\)，\(\int_{\partial\Omega}P(x,y)\,dS_x = 1\)。 直观：Poisson 核是边界上的概率权重。 球的 Poisson 核（三维）： \[P(x,y) = \frac{R^2 - |y|^2}{4\pi R|x-y|^3}, \quad |x| = R\] 当 \(y \to x_0 \in \partial B_R\) 时，\(P\) 趋于 \(\delta_{x_0}\) → \(u(y) \to g(x_0)\)（连续性）。 Green 函数的正性：\(G(x,y) > 0\)（\(x,y \in \Omega\)，\(x \neq y\)）。 证明用极大值原理：\(G - \Phi \geq 0\) 在边界上（\(G = 0\)，\(\Phi < 0\) 当 \(|x| = R\) 足够大）， \(\Delta(G - \Phi) = 0\)，极大值原理 → \(G \geq \Phi > 0\)。

🌐 实际应用

- 概率论中 Green 函数与 Brown 运动首次离开时间分布的关系 - 热方程中 Green 函数给出温度场的时空传播规律

✏️ 例题分析与易错点

例题：利用 Poisson 积分公式求解单位圆盘上的 Dirichlet 问题。 \(\Delta u = 0\)（\(r < 1\)），\(u(1,\theta) = \cos\theta\)。 解答：二维 Poisson 核 \(P(r,\theta-\phi) = \frac{1-r^2}{2\pi(1-2r\cos(\theta-\phi)+r^2)}\)。 \(u(r,\theta) = \frac{1}{2\pi}\int_0^{2\pi}\frac{(1-r^2)\cos\phi}{1-2r\cos(\theta-\phi)+r^2}\,d\phi = r\cos\theta\)。 (因为调和函数 \(r\cos\theta\) 在边界上恰好等于 \(\cos\theta\)) 易错点：Poisson 核的归一化 \(\int P = 1\) 在 2D 是对角度积分 \(\frac{1}{2\pi}\int_0^{2\pi}\)， 不要漏掉 \(1/(2\pi)\) 因子。

04Green 函数与谱展开

🎯 为什么研究这个问题

Green 函数还有一种完全不同的表达方式：用 Laplace 算子的 特征函数展开。这建立了 Green 函数与 Sturm-Liouville 理论、 Fourier 方法之间的深刻联系，也为估计 Green 函数的渐近 行为提供了谱论工具。

🔍 抽象化方法

Green 函数的谱展开 \(G = \sum \phi_n(x)\phi_n(y)/\lambda_n\) 是 算子 \((-\Delta)^{-1}\) 在特征基下的矩阵表示（无穷维对角矩阵）。 特征值与 Green 函数的奇异性之间存在精确的渐近关系（Weyl 定律）。

📐 公式推导 / 模型建立

谱展开：设 \(\{(\lambda_n, \phi_n)\}\) 是 \(-\Delta\) 在 \(\Omega\)（Dirichlet BC）的特征系统， \(-\Delta\phi_n = \lambda_n\phi_n\)，\(\{\phi_n\}\) 正交归一。则 \[G(x,y) = \sum_{n=1}^\infty \frac{\phi_n(x)\phi_n(y)}{\lambda_n}\] 推导：\(-\Delta_x G = \delta(x-y) = \sum_n \phi_n(x)\phi_n(y)\)（完备性）， 令 \(G = \sum a_n(y)\phi_n(x)\)，代入得 \(\lambda_n a_n(y) = \phi_n(y)\)， 故 \(a_n = \phi_n(y)/\lambda_n\)。 例：矩形 \([0,a]\times[0,b]\) 的 Green 函数： \(\phi_{mn} = \frac{2}{\sqrt{ab}}\sin\frac{m\pi x}{a}\sin\frac{n\pi y}{b}\)， \(\lambda_{mn} = \pi^2(m^2/a^2 + n^2/b^2)\)。 \[G = \frac{4}{ab}\sum_{m,n=1}^\infty \frac{\sin\frac{m\pi x_1}{a}\sin\frac{n\pi y_1}{b}\sin\frac{m\pi x_2}{a}\sin\frac{n\pi y_2}{b}}{\pi^2(m^2/a^2+n^2/b^2)}\]

🌐 实际应用

- 量子力学中的「传播子」\(K(x,y;t) = \sum_n \phi_n(x)\phi_n(y)e^{-\lambda_n t}\) 是含时 Green 函数 - Weyl 渐近公式 \(N(\lambda) \sim C_d\text{Vol}(\Omega)\lambda^{d/2}\) 连接特征值分布与几何

✏️ 例题分析与易错点

例题：写出区间 \([0,\pi]\)（Dirichlet BC）上 \(-u'' = f\) 的 Green 函数谱展开。 解答：\(\phi_n = \sqrt{2/\pi}\sin nx\)，\(\lambda_n = n^2\)。 \(G(x,y) = \frac{2}{\pi}\sum_{n=1}^\infty \frac{\sin nx\sin ny}{n^2}\)。 验证：\(-G_{xx} = \frac{2}{\pi}\sum \sin nx\sin ny = \delta(x-y)\)。 也可直接构造：\(G(x,y) = \begin{cases}x(π-y)/\pi,& x \leq y\\ y(\pi-x)/\pi,& x > y\end{cases}\)。 两种表达等价！ 易错点：谱展开收敛可能很慢（特别在 \(x=y\) 附近，\(G\) 有对数奇异性）， 数值计算时直接用解析公式效率更高。
