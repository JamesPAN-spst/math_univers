波动方程

01波动方程与 d'Alembert 公式

🎯 为什么研究这个问题

波动方程 \(u_{tt} = c^2 u_{xx}\) 描述弦的振动、声波、电磁波等传播现象。 1747 年 d'Alembert 给出一维通解：解是两个反向行波的叠加。这个 公式揭示了波传播最本质的特征——信息以有限速度 \(c\) 传播， 与热方程的无穷传播速度形成鲜明对比。

🔍 抽象化方法

波动方程可以因式分解为 \((\partial_t - c\partial_x)(\partial_t + c\partial_x)u = 0\)， 揭示了两族特征线 \(x \pm ct = \text{const}\) 的几何结构。 d'Alembert 公式本质是沿特征线积分初值数据。

📐 公式推导 / 模型建立

一维波动方程：\(u_{tt} = c^2 u_{xx}\)。令 \(\xi = x+ct\)，\(\eta = x-ct\) （特征坐标），方程变为 \(u_{\xi\eta} = 0\)，通解 \(u = F(\xi) + G(\eta) = F(x+ct) + G(x-ct)\)。 d'Alembert 公式（初值问题 \(u(x,0)=\phi(x)\)，\(u_t(x,0)=\psi(x)\)）： \[u(x,t) = \frac{\phi(x+ct) + \phi(x-ct)}{2} + \frac{1}{2c}\int_{x-ct}^{x+ct}\psi(s)\,ds\] 验证：\(u(x,0) = \phi(x)\)（\(t=0\) 时积分为零）。✓ \(u_t(x,0) = \frac{c\phi'(x+ct)-c\phi'(x-ct)+\psi(x+ct)+\psi(x-ct)}{2}\big|_{t=0} = \psi(x)\)。✓

🌐 实际应用

- 弦的振动：吉他弦被拨动后的波形演化 - 地震波 P 波和 S 波以不同速度传播

✏️ 例题分析与易错点

例题：用 d'Alembert 公式求解 \(u_{tt} = 4u_{xx}\)，\(u(x,0)=\cos x\)，\(u_t(x,0)=0\)。 解答：\(c=2\)，\(\psi = 0\)，\(u(x,t) = \frac{\cos(x+2t)+\cos(x-2t)}{2}\)。 由和差化积：\(u = \cos x \cos 2t\)（驻波）。 易错点：d'Alembert 公式只适用于全空间（\(x \in \mathbb{R}\)）情况。 有界域需要用分离变量或反射法处理边界条件。

⚡ 快速检测
d'Alembert 公式 \(u = \frac{1}{2}[\phi(x+ct)+\phi(x-ct)]\)（\(\psi=0\)）

02依赖域、影响域与能量守恒

🎯 为什么研究这个问题

波动方程的一个关键特征是信息传播速度有限：点 \((x_0, t_0)\) 处的 解仅由初值在区间 \([x_0 - ct_0, x_0 + ct_0]\) 上的值决定 （依赖域），初值在 \(x_0\) 处的扰动仅影响特征锥 \(|x - x_0| \leq ct\) 内的点（影响域）。此外，波动方程保持总能量不变——没有耗散。

🔍 抽象化方法

依赖域和影响域的概念是「因果性」的数学表述：物理中信息传播不能 超光速，对应于双曲型 PDE 的有限传播速度。能量守恒则反映了波动 方程的时间可逆性（与热方程的不可逆性对比）。

📐 公式推导 / 模型建立

依赖域：d'Alembert 公式中，\(u(x_0,t_0)\) 仅涉及 \(\phi\) 在 \([x_0-ct_0, x_0+ct_0]\) 上的值和 \(\psi\) 在同一区间的积分。 此区间就是 \((x_0,t_0)\) 的依赖域。 影响域：初值在 \(x_0\) 处的扰动传播到的区域：\(\{(x,t) : |x-x_0| \leq ct\}\)。 能量守恒：定义能量 \[E(t) = \frac{1}{2}\int_{-\infty}^{\infty}\left[u_t^2 + c^2 u_x^2\right]\,dx\] 求导（利用波动方程 \(u_{tt} = c^2 u_{xx}\)）： \(\frac{dE}{dt} = \int(u_t u_{tt} + c^2 u_x u_{xt})\,dx = \int u_t(c^2 u_{xx}) + c^2 u_x u_{xt}\,dx\) \(= c^2[u_t u_x]_{-\infty}^{\infty} = 0\)（当 \(u\) 衰减足够快时）。 故 \(E(t) = E(0) = \frac{1}{2}\int[\psi^2 + c^2(\phi')^2]\,dx\)。

🌐 实际应用

- 地震学中确定某次地震影响的地理范围需要计算影响域 - 声学中能量守恒解释了远距离声波的振幅衰减（几何扩散，而非耗散）

03高维波动方程与 Huygens 原理

🎯 为什么研究这个问题

在三维空间中，波动方程的解有一个惊人的性质——Huygens 原理： 初始脉冲产生一个膨胀的球面波前，波前通过后信号完全消失 （「清晰的尾巴」）。但在二维中这不成立——扔石子入水后涟漪 会持续很长时间（残波/尾波）。这种维度依赖的行为是三维 物理学（声学、电磁学）的数学基础。

🔍 抽象化方法

Huygens 原理的维度依赖性反映了波方程解表示公式在奇数维和 偶数维中的本质差异：奇数维的 Kirchhoff 公式仅涉及球面积分 （无体积积分），而偶数维的 Hadamard 降维法引入了体积积分项。

📐 公式推导 / 模型建立

三维 Kirchhoff 公式：\(u_{tt} = c^2\Delta u\)，\(u(x,0)=\phi\)，\(u_t(x,0)=\psi\)。 \[u(x,t) = \frac{\partial}{\partial t}\left[\frac{1}{4\pi c^2 t}\int_{|y-x|=ct}\phi(y)\,dS\right] + \frac{1}{4\pi c^2 t}\int_{|y-x|=ct}\psi(y)\,dS\] 注意：积分仅在球面 \(|y-x| = ct\) 上！不涉及球体内部。 Huygens 原理（三维）：点源初始脉冲 \(\phi = \delta(x)\)，\(\psi = 0\) 产生的波 在时刻 \(t\) 集中在球面 \(|x| = ct\) 上。波前通过后 \(u = 0\)。 物理意义：声音听到后就消失（不会拖尾）。 二维（Hadamard 降维法）：把二维问题看成不依赖 \(z\) 的三维问题。 球面积分变为圆盘积分——有体积积分项： \[u(x,t) = \frac{1}{2\pi c}\frac{\partial}{\partial t}\int_{|y-x| \leq ct}\frac{\phi(y)}{\sqrt{c^2t^2-|y-x|^2}}\,dy + \cdots\] Huygens 原理不成立：球内积分意味着过去所有时刻的影响都残留。

🌐 实际应用

- 三维声学中 Huygens 原理保证语音清晰可辨（无拖尾） - 二维水波的涟漪持续存在解释了池塘投石后的长时间波纹

✏️ 例题分析与易错点

例题：说明为什么二维中没有 Huygens 原理。 解答：向平静水面投一块石子（二维波动方程的点源问题）。 波前向外扩展，但水面在波前通过后仍在振荡——这是因为二维 Kirchhoff 公式包含对圆盘（而非仅圆周）上的积分， 圆盘内部的贡献产生「尾波」。 对比三维：手拍一下产生的声波过后就安静了。 易错点：Huygens 原理仅在奇数空间维度(\(n=1,3,5,\ldots\))成立， 但一维虽是奇数维，严格来说不算 Huygens（因一维中波只沿两个方向传播）。 精确的 Huygens 原理指 \(n \geq 3\) 的奇数维。

04弦振动分析与应用

🎯 为什么研究这个问题

有界弦的振动（如吉他弦、钢琴弦）是波动方程在有界域上的经典应用。 分离变量法给出驻波叠加形式的解，每个驻波模对应一个固有频率。 基频和泛音的组合决定了乐器的音色——这是物理学与音乐的美妙联结。

📐 公式推导 / 模型建立

有界弦问题：\(u_{tt} = c^2 u_{xx}\)，\(0 < x < L\)， \(u(0,t) = u(L,t) = 0\)（固定端），\(u(x,0) = \phi(x)\)，\(u_t(x,0) = \psi(x)\)。 分离变量 \(u = X(x)T(t)\)： \(X'' + \lambda X = 0\)，\(X(0)=X(L)=0\) → \(\lambda_n = (n\pi/L)^2\)，\(X_n = \sin\frac{n\pi x}{L}\)。 \(T'' + c^2\lambda_n T = 0\) → \(T_n = A_n\cos\omega_n t + B_n\sin\omega_n t\)， \(\omega_n = cn\pi/L\)（第 \(n\) 阶固有频率）。 \[u(x,t) = \sum_{n=1}^\infty \left(A_n\cos\omega_n t + B_n\sin\omega_n t\right)\sin\frac{n\pi x}{L}\] \(A_n = \frac{2}{L}\int_0^L\phi(x)\sin\frac{n\pi x}{L}\,dx\)， \(B_n = \frac{2}{n\pi c}\int_0^L\psi(x)\sin\frac{n\pi x}{L}\,dx\)。 基频 \(\omega_1 = c\pi/L\)，泛音 \(\omega_n = n\omega_1\)（整数倍 → 和声关系）。

🌐 实际应用

- 弦乐器的音高由 \(\omega_1 = \pi\sqrt{T/\rho}/L\) 决定（\(T\)=张力，\(\rho\)=线密度） - 桥梁工程中的共振频率分析

✏️ 例题分析与易错点

例题：长 \(\pi\) 的弦，\(c = 1\)，初始形状 \(\phi(x) = \sin x + \frac{1}{2}\sin 3x\)， 初速 \(\psi = 0\)。求 \(u(x,t)\)。 解答：\(L = \pi\)，已是 Fourier 正弦级数。\(A_1 = 1\)，\(A_3 = 1/2\)，其余 \(A_n = 0\)。 \(B_n = 0\)（因 \(\psi = 0\)）。 \(u = \sin x \cos t + \frac{1}{2}\sin 3x \cos 3t\)。 两个驻波的叠加，频率比 \(1:3\)（基频和第三泛音）。 易错点：\(\omega_n = cn\pi/L\)，不是 \(n\pi/L\)。\(c\) 的遗漏会导致频率错误。
