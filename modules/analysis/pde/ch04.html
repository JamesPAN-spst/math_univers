Laplace 与 Poisson 方程

01调和函数与均值性质

🎯 为什么研究这个问题

Laplace 方程 \(\Delta u = 0\) 描述稳态温度分布、静电势、引力势、 不可压缩流体的速度势等。其解——调和函数——具有极端的正则性 （解析函数！）和深刻的几何性质。均值性质是其最直观的刻画： 调和函数在每一点的值等于周围任意球面上的平均值。

🔍 抽象化方法

均值性质揭示了椭圆型方程的本质：无「信息源」时值完全由周围 决定。这与随机游走（Brown 运动）停时问题的联系并非偶然—— 调和函数是鞅（在概率意义下）。

📐 公式推导 / 模型建立

Laplace 方程：\(\Delta u = \sum_{i=1}^n \frac{\partial^2 u}{\partial x_i^2} = 0\)。 均值性质：若 \(u\) 在 \(\Omega\) 上调和，则对任意 \(B_r(x) \subset \Omega\)： \[u(x) = \frac{1}{|\partial B_r(x)|}\int_{\partial B_r(x)} u\,dS = \frac{1}{|B_r(x)|}\int_{B_r(x)} u\,dy\] （球面均值 = 体积均值 = 中心点值） 证明思路（球面均值）：定义 \(\phi(r) = \frac{1}{|\partial B_r|}\int_{\partial B_r} u\,dS\)。 求导：\(\phi'(r) = \frac{1}{|\partial B_r|}\int_{\partial B_r} \frac{\partial u}{\partial n}\,dS = \frac{1}{|\partial B_r|}\int_{B_r}\Delta u\,dy = 0\)。 故 \(\phi(r)\) 是常数，\(\phi(r) = \lim_{r \to 0}\phi(r) = u(x)\)。 逆定理：满足均值性质的连续函数是调和的（Koebe 定理）。

🌐 实际应用

- 静电学中导体空腔内无电荷则电势为调和函数 - 图像处理中 Laplace 方程用于图像修复（inpainting）

✏️ 例题分析与易错点

例题：若 \(u\) 在单位圆 \(|x| \leq 1\) 上调和，\(u|_{|x|=1} = \cos\theta\)。 不用求解，求 \(u(0)\)。 解答：均值性质：\(u(0) = \frac{1}{2\pi}\int_0^{2\pi}\cos\theta\,d\theta = 0\)。 易错点：均值性质需要球面完全在 \(\Omega\) 内。如果球面跨越边界， 公式不成立。

02极大值原理与唯一性

🎯 为什么研究这个问题

调和函数不能在内部取到最大值（除非是常数函数）——这看似简单的 命题（极大值原理）是椭圆型 PDE 理论最有力的工具。它立即给出 Dirichlet 问题的唯一性和对边值的连续依赖，而且完全不需要 解的显式公式。

📐 公式推导 / 模型建立

强极大值原理：若 \(u\) 在连通开域 \(\Omega\) 上调和，则 \(u\) 不能在 \(\Omega\) 内部取到最大值（除非 \(u\) 是常数）。 证明（均值性质法）：设 \(u(x_0) = \max_\Omega u\)。 \(u(x_0) = \frac{1}{|B_r|}\int_{B_r(x_0)} u\,dy \leq u(x_0)\)。 等号成立要求 \(u \equiv u(x_0)\) 在 \(B_r\) 上。 连通性 ⇒ \(u\) 在整个 \(\Omega\) 上为常数。 弱极大值原理：\(\max_{\overline{\Omega}} u = \max_{\partial\Omega} u\)。 Dirichlet 唯一性：设 \(u_1, u_2\) 解 \(\Delta u = 0\)， \(u_1|_{\partial\Omega} = u_2|_{\partial\Omega}\)。 \(w = u_1 - u_2\) 调和且 \(w|_{\partial\Omega} = 0\)。 极大值原理 ⇒ \(\max w \leq 0\)，对 \(-w\) 类似 ⇒ \(w \equiv 0\)。 比较原理：\(\Delta u = 0 \leq \Delta v = f \geq 0\) 在 \(\Omega\) 中，\(u|_{\partial\Omega} \leq v|_{\partial\Omega}\) ⇒ \(u \leq v\) 在 \(\Omega\) 中。

🌐 实际应用

- 椭圆型方程数值解的误差估计：误差满足类似的极大值原理

✏️ 例题分析与易错点

例题：\(\Delta u = 0\) 在单位圆内，\(u|_{|x|=1} \leq 5\)。证明 \(u(x) \leq 5\) 处处成立。 解答：极大值原理：\(\max_{\overline{B_1}} u = \max_{|x|=1} u \leq 5\)。✓ 易错点：极大值原理的前提是 \(u\) 调和（非次调和）。 若 \(\Delta u > 0\)（严格次调和），极大值原理的结论可能更强。

03基本解与 Green 函数

🎯 为什么研究这个问题

Green 函数是 PDE 理论中最强大的概念之一：如果知道了 Green 函数， 就能通过积分公式直接写出任意边值/源项问题的解。Green 函数是 Laplace 算子的「逆算子核」——它描述了一个点源电荷在某区域中 产生的电势（考虑了边界效应）。

🔍 抽象化方法

Green 函数 = 基本解 + 修正项（用于满足边界条件）。 在物理上，基本解是自由空间的点源响应，修正项是边界反射的「像」。 这是镜像法(method of images)的数学框架。

📐 公式推导 / 模型建立

基本解 \(\Phi(x)\)：\(\Delta\Phi = \delta(x)\)。 \[\Phi(x) = \begin{cases} -\frac{1}{2\pi}\ln|x| & n=2 \\ \frac{1}{n(n-2)\omega_n}|x|^{2-n} & n \geq 3 \end{cases}\] 其中 \(\omega_n\) 为 \(n\) 维单位球体积。 三维：\(\Phi(x) = -\frac{1}{4\pi|x|}\)（库仑势/引力势）。 Green 函数：\(G(x,y)\) 对固定 \(y \in \Omega\) 满足 \(\Delta_x G = \delta(x-y)\) 在 \(\Omega\) 中，\(G|_{x \in \partial\Omega} = 0\)。 分解：\(G(x,y) = \Phi(x-y) + H(x,y)\)，其中 \(H\) 调和补偿边界条件。 Green 表示公式：\(\Delta u = f\) 在 \(\Omega\)，\(u|_{\partial\Omega} = g\) ⇒ \[u(x) = \int_\Omega G(x,y)f(y)\,dy - \int_{\partial\Omega}\frac{\partial G}{\partial n_y}(x,y)g(y)\,dS_y\] 半空间的 Green 函数（镜像法）：\(\Omega = \{x_n > 0\}\)。 \(G(x,y) = \Phi(x-y) - \Phi(x-y^*)\)，\(y^* = (y_1,\ldots,-y_n)\)（镜像点）。

🌐 实际应用

- 静电学中的镜像电荷法即是 Green 函数的物理实现 - 声学中 Green 函数用于计算腔体内的声场

✏️ 例题分析与易错点

例题：用镜像法写出上半平面 \(\{y > 0\}\) 中 \(\Delta u = 0\)，\(u(x,0) = g(x)\) 的解。 解答：Green 函数 \(G(x,y;\xi,\eta) = -\frac{1}{2\pi}[\ln r - \ln r^*]\)， \(r = \sqrt{(x-\xi)^2+(y-\eta)^2}\)，\(r^* = \sqrt{(x-\xi)^2+(y+\eta)^2}\)。 Poisson 积分公式：\(u(x,y) = \frac{y}{\pi}\int_{-\infty}^\infty \frac{g(\xi)}{(x-\xi)^2+y^2}\,d\xi\)。 验证：\(y \to 0^+\) 时被积函数趋于 \(\pi\delta(x-\xi)g(\xi)\)。✓ 易错点：镜像点是关于边界的反射，不是关于源点。 上半平面中 \((x,y)\) 的镜像是 \((x,-y)\)，不是 \((-x,y)\)。

04Poisson 方程与 Dirichlet/Neumann 问题

🎯 为什么研究这个问题

Poisson 方程 \(\Delta u = f\) 是带源项的 Laplace 方程，描述了 有电荷/质量分布时的电势/引力势。Dirichlet 和 Neumann 两种 边值问题分别对应物理上固定电位和固定电场法向分量的情况。 理解这两种问题的可解性条件和物理意义是学习椭圆型 PDE 的关键。

📐 公式推导 / 模型建立

Poisson 方程：\(\Delta u = f\) 在 \(\Omega\) 中。 Dirichlet 问题：\(u|_{\partial\Omega} = g\)。 存在唯一性：对光滑域，\(f \in L^2, g \in H^{1/2}(\partial\Omega)\) 时存在唯一弱解。 Neumann 问题：\(\frac{\partial u}{\partial n}\Big|_{\partial\Omega} = h\)。 可解性条件（相容条件）： \[\int_\Omega f\,dx = \int_{\partial\Omega} h\,dS\] 证明：\(\int_\Omega \Delta u\,dx = \int_{\partial\Omega}\frac{\partial u}{\partial n}\,dS\)（散度定理）， 左 \(= \int f\)，右 \(= \int h\)。 若条件满足，解存在但仅确定到常数（\(u + C\) 也是解）。 唯一性需加归一化条件如 \(\int_\Omega u\,dx = 0\)。 物理解释： - Dirichlet = 边界温度固定（恒温器） - Neumann = 边界热流固定（绝热/加热） - 相容条件 = 稳态时进出系统的热流必须平衡

🌐 实际应用

- 静电场：Dirichlet 对应固定电位导体，Neumann 对应固定面电荷密度 - 流体力学：不可压缩流体的压力场满足 Poisson 方程

✏️ 例题分析与易错点

例题：在单位圆盘上求解 \(\Delta u = 0\)，\(\frac{\partial u}{\partial n}\Big|_{r=1} = \cos\theta\)。 验证相容条件。 解答：\(\int_0^{2\pi}\cos\theta\,d\theta = 0\)。✓（相容条件满足。） 设 \(u = r\cos\theta + C\)（试解）。\(\Delta u = 0\)。✓ \(\frac{\partial u}{\partial r}\Big|_{r=1} = \cos\theta\)。✓ 取 \(C=0\) 为归一化选择。 易错点：Neumann 问题忘记检查相容条件就直接求解， 可能得到矛盾。如 \(\Delta u = 0\)，\(\partial u/\partial n = 1\)（常数）在单位圆上， \(\int 1\,dS = 2\pi \neq 0 = \int \Delta u\)，无解。
