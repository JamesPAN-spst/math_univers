01二阶线性 PDE 的分类

🎯 为什么研究这个问题

不同物理现象——热传导、波传播、静电场——对应的偏微分方程有着 根本不同的数学性质。1830 年代，人们意识到二阶线性 PDE 可以像 二次曲线一样按「椭圆/抛物/双曲」分类，每类有独特的解的行为、 边值条件要求和求解策略。这个分类是学习 PDE 的第一步。

🔍 抽象化方法

分类的核心思想是对主符号（最高阶项系数矩阵）做特征值分析。 这与二次型的分类（正定/半正定/不定）完全类比，是线性代数 在 PDE 理论中的应用。

📐 公式推导 / 模型建立

一般二阶线性 PDE (二元)： \[Au_{xx} + 2Bu_{xy} + Cu_{yy} + \text{低阶项} = 0\] 主符号矩阵 \(\begin{pmatrix} A & B \\ B & C \end{pmatrix}\)，判别式 \(\Delta = B^2 - AC\)。 分类： - 椭圆型 (\(\Delta < 0\))：二特征值同号。典型：Laplace 方程 \(\Delta u = 0\) - 抛物型 (\(\Delta = 0\))：一个特征值为零。典型：热方程 \(u_t = u_{xx}\) - 双曲型 (\(\Delta > 0\))：二特征值异号。典型：波动方程 \(u_{tt} = c^2 u_{xx}\) 物理对应： 椭圆型 → 稳态/平衡问题（边值问题 BVP） 抛物型 → 扩散/耗散过程（初边值问题 IBVP） 双曲型 → 波传播（初值问题 IVP） 变系数情况：分类可能随点 \((x,y)\) 变化。 如 Tricomi 方程 \(u_{xx} + yu_{yy} = 0\)：\(y>0\) 椭圆，\(y=0\) 抛物，\(y<0\) 双曲。

🌐 实际应用

- 航空动力学中跨声速流的 Tricomi 方程 - 选择正确的数值方法（分类决定了离散格式和稳定性条件）

✏️ 例题分析与易错点

例题：分类 \(u_{xx} + 4u_{xy} + 3u_{yy} = 0\)。 解答：\(A=1\)，\(B=2\)，\(C=3\)。\(\Delta = B^2 - AC = 4 - 3 = 1 > 0\)。 双曲型。特征方向由 \(A\,dy^2 - 2B\,dx\,dy + C\,dx^2 = 0\) 给出。 \(dy/dx = (B \pm \sqrt{\Delta})/A = (2 \pm 1)/1 = 1\) 或 \(3\)。 易错点：混淆判别式的符号约定。有的教材定义 \(\Delta = AC - B^2\) （取负号），此时椭圆型对应 \(\Delta > 0\)。需注意教材约定。

02一阶线性 PDE 与特征线法

🎯 为什么研究这个问题

一阶线性 PDE 是最简单的偏微分方程，其求解有一个优美的几何方法 ——特征线法。沿特征线，PDE 化为 ODE，可以用常微分方程的工具求解。 特征线法不仅给出解析解，还揭示了信息如何在空间中传播， 是理解高阶 PDE（尤其是双曲型方程）的基础。

🔍 抽象化方法

特征线法将 PDE 的求解转化为沿特征曲线的 ODE 求解。这体现了 「降维」思想：偏微分（多变量）→ 常微分（单变量），是 PDE 理论 中最基本的策略之一。

📐 公式推导 / 模型建立

一阶线性 PDE：\(a(x,y)u_x + b(x,y)u_y = c(x,y,u)\)。 特征线方程（Cauchy 版）： \[\frac{dx}{a} = \frac{dy}{b} = \frac{du}{c}\] 沿特征线 \((x(s), y(s), u(s))\) 满足 ODE 系统： \[\frac{dx}{ds} = a,\quad \frac{dy}{ds} = b,\quad \frac{du}{ds} = c\] 传输方程 \(u_t + cu_x = 0\)（\(a=c, b=1\)）： 特征线 \(dx/dt = c\)，即 \(x - ct = \text{const}\)。 沿特征线 \(du/dt = 0\)，故 \(u = f(x-ct)\)（右行波）。 初值 \(u(x,0) = \phi(x)\) ⇒ \(u(x,t) = \phi(x-ct)\)。

🌐 实际应用

- 传输方程描述了以速度 \(c\) 传播的信号（如声波的简化模型） - 交通流中车辆密度的传播

✏️ 例题分析与易错点

例题：用特征线法求解 \(u_t + 2u_x = 0\)，\(u(x,0) = e^{-x^2}\)。 解答：特征线 \(dx/dt = 2\)，\(x = 2t + x_0\)。沿特征线 \(u = \text{const}\)。 \(u(x,t) = u(x_0, 0) = e^{-x_0^2} = e^{-(x-2t)^2}\)。 解是初始高斯脉冲以速度 2 向右平移。 易错点：对拟线性方程（\(c\) 依赖 \(u\)），特征线不再平行， 可能交叉形成激波。此时经典解不存在，需引入弱解。

03拟线性方程与激波形成

🎯 为什么研究这个问题

当特征线的速度依赖于解本身（拟线性方程），不同点上的信息 传播速度不同，特征线可能「交叉」——这就是激波(shock)。 激波出现在超声速飞行、爆炸波、交通堵塞等现象中。 激波的形成标志着经典解的崩溃，需要引入弱解的概念。

🔍 抽象化方法

拟线性方程的特征线方程本身变为非线性 ODE，特征线的曲率和 交叉行为编码了解的非线性结构。激波的本质是特征线族的包络， 这与 ODE 中「奇解」的概念有深层联系。

📐 公式推导 / 模型建立

拟线性方程：\(u_t + c(u)u_x = 0\)（\(c\) 依赖 \(u\)）。 特征线：\(dx/dt = c(u)\)，沿特征线 \(u = \text{const}\)。 但不同特征线上 \(u\) 不同 → \(c(u)\) 不同 → 斜率不同 → 可能相交。 Burgers 方程（无粘性）：\(u_t + uu_x = 0\)。 特征线 \(x = u_0 t + x_0\)（直线，斜率 \(1/u_0\)）。 初值 \(u(x,0) = \phi(x)\)：隐式解 \(u = \phi(x - ut)\)。 激波形成时间：\(t_s = -1/\min_x \phi'(x)\)（当 \(\phi' < 0\) 时）。 \(t > t_s\) 后特征线交叉，经典解不存在。 Rankine-Hugoniot 条件：激波速度 \[s = \frac{f(u_R) - f(u_L)}{u_R - u_L}\] 其中 \(f(u) = u^2/2\)（Burgers 情况），\(u_L, u_R\) 为激波左右值。

🌐 实际应用

- 交通流模型：交通密度满足守恒律 \(\rho_t + f(\rho)_x = 0\)，堵车 = 激波 - 气体动力学中的 Euler 方程和激波管问题

✏️ 例题分析与易错点

例题：\(u_t + uu_x = 0\)，\(u(x,0) = \begin{cases} 1, & x<0 \\ 0, & x>0 \end{cases}\)。 解答：\(x < 0\) 的特征线：\(x = t + x_0\)（斜率 1）。 \(x > 0\) 的特征线：\(x = x_0\)（斜率 \(\infty\)/竖直）。 特征线在 \(t=0\) 处立即交叉（初值不连续）。 Rankine-Hugoniot：\(s = (0+1)/2 = 1/2\)。 激波解：\(u = \begin{cases} 1, & x < t/2 \\ 0, & x > t/2 \end{cases}\)。 易错点：不是所有间断都是激波，还需满足熵条件（Lax 条件： \(c(u_L) > s > c(u_R)\)）来排除非物理的稀疏激波。

04边值条件与适定性

🎯 为什么研究这个问题

一个 PDE 本身不足以确定唯一解——还需要恰当的边值/初值条件。 不同类型的 PDE 需要不同类型的条件：椭圆型需要边值条件， 抛物型需要初值+边值，双曲型需要初值。给错条件（过多或过少、 错误类型）会导致解不存在或不唯一。Hadamard 的「适定性」概念 将这些要求统一为三个条件：存在、唯一、连续依赖。

📐 公式推导 / 模型建立

Hadamard 适定性(Well-posedness)：PDE 问题是适定的，如果： 1. 存在性：解存在 2. 唯一性：解唯一 3. 连续依赖：解连续依赖于数据（小扰动→ 小偏差） 边值条件类型： - Dirichlet 条件：\(u|_{\partial\Omega} = g\)（规定函数值） - Neumann 条件：\(\partial u/\partial n|_{\partial\Omega} = h\)（规定法向导数） - Robin/混合条件：\(\alpha u + \beta \partial u/\partial n = g\) 各类型 PDE 的典型问题： \[\text{椭圆型 (Laplace): BVP} \quad \Delta u = 0 \text{ in } \Omega,\; u|_{\partial\Omega} = g\] \[\text{抛物型 (热方程): IBVP} \quad u_t = \Delta u,\; u(x,0) = \phi(x),\; u|_{\partial\Omega} = g\] \[\text{双曲型 (波方程): IVP} \quad u_{tt} = c^2\Delta u,\; u(x,0) = \phi,\; u_t(x,0) = \psi\] 不适定的例子（Hadamard 例）：Laplace 方程配 Cauchy 数据 \(u(x,0) = 0\)，\(u_y(x,0) = n^{-1}\sin nx\)。解 \(u = n^{-2}\sinh(ny)\sin(nx)\) 随 \(n \to \infty\) 对任意 \(y > 0\) 发散，不连续依赖初值→ 不适定。

🌐 实际应用

- 工程有限元分析中，正确的边界条件设置是得到有意义结果的前提 - 逆问题（从测量反推参数）通常是不适定的，需正则化

✏️ 例题分析与易错点

例题：为什么 Laplace 方程 \(\Delta u = 0\) 不能配初值条件 \(u(x,0)\) 和 \(u_y(x,0)\)？ 解答：这是 Hadamard 不适定例子。直觉：椭圆型方程的解在整个域上 同时决定（无信息传播方向），不存在「初始时刻」的概念。 Cauchy-Kowalewski 定理虽保证解析解存在，但解不连续依赖于数据。 易错点：「解存在」不等于「问题适定」。Hadamard 例中解存在且唯一， 但不连续依赖数据，仍不适定。
