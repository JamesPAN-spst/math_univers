概率空间

01概率空间的公理化构造

🎯 为什么研究这个问题

掷骰子、抽彩票、天气预报——日常生活充满不确定性。如何将含糊的 「可能性」变成严格的数学对象？1933 年 Kolmogorov 用测度论给出了 答案：概率空间 \((\Omega, \mathcal{F}, P)\) 的三条公理统一了整个 概率论的基础，使所有关于随机现象的推理都有坚实的逻辑根基。

🔍 抽象化方法

核心抽象：将「不确定性」编码为三元组 \((\Omega, \mathcal{F}, P)\)。 \(\Omega\) 是所有可能结果的集合，\(\mathcal{F}\)（\(\sigma\)-代数）编码 我们能够「提问」的全部事件，\(P\) 对每个事件给出 \([0,1]\) 中的答案。 这直接推广了 Lebesgue 测度论的框架。

📐 公式推导 / 模型建立

Kolmogorov 公理：设 \(\Omega \neq \emptyset\)，\(\mathcal{F}\) 为 \(\Omega\) 上的 \(\sigma\)-代数，\(P: \mathcal{F} \to [0,1]\) 满足： (P1) 非负性：\(P(A) \geq 0\)，\(\forall A \in \mathcal{F}\) (P2) 规范性：\(P(\Omega) = 1\) (P3) 可列可加性：若 \(A_1, A_2, \ldots\) 两两不相交， \[P\!\left(\bigcup_{n=1}^{\infty} A_n\right) = \sum_{n=1}^{\infty} P(A_n)\] 基本推论： - 补事件：\(P(A^c) = 1 - P(A)\)（令 \(A_1=A, A_2=A^c\)，由 P2+P3） - 单调性：\(A \subseteq B \Rightarrow P(A) \leq P(B)\) - 容斥：\(P(A \cup B) = P(A) + P(B) - P(A \cap B)\) 概率的连续性：若 \(A_n \uparrow A\)，则 \(P(A_n) \to P(A)\)。 证明：令 \(B_1 = A_1\)，\(B_n = A_n \setminus A_{n-1}\)（不交化），则 \[P(A) = P\!\left(\bigcup_n B_n\right) = \sum_n P(B_n) = \lim_N \sum_{n=1}^{N} P(B_n) = \lim_N P(A_N)\]

🌐 实际应用

保险精算中，概率空间是计算保费和理赔风险的基础框架； 量子力学中，Born 规则本质上定义了一个概率测度。

✏️ 例题分析与易错点

例题：\(\Omega = \{1,2,3,4,5,6\}\)，\(\mathcal{F} = 2^\Omega\)， \(P\) 均匀。求「点数为奇数或大于 4」的概率。 解答：\(A_1 = \{1,3,5\}\)，\(A_2 = \{5,6\}\)。 \(P(A_1 \cup A_2) = 3/6 + 2/6 - 1/6 = 4/6 = 2/3\)。 易错点：忘记减去 \(P(A_1 \cap A_2) = P(\{5\}) = 1/6\)， 直接相加得 \(5/6\)，重复计算了元素 5。

⚡ 快速检测
Kolmogorov 公理中的可列可加性能否弱化为有限可加性？

02条件概率与 Bayes 公式

🎯 为什么研究这个问题

当我们获得部分信息时，需要更新对事件的概率判断。医学检测阳性后， 患病的真实概率是多少？这类「已知证据后更新信念」的问题催生了条件 概率和 Bayes 公式——从垃圾邮件过滤到法医鉴定都依赖于此工具。

📐 公式推导 / 模型建立

条件概率定义：设 \(P(B) > 0\)， \[P(A|B) = \frac{P(A \cap B)}{P(B)}\] 全概率公式：若 \(B_1, \ldots, B_n\) 是 \(\Omega\) 的划分（两两不交， 并为 \(\Omega\)），且 \(P(B_i) > 0\)，则 \[P(A) = \sum_{i=1}^{n} P(A|B_i)\,P(B_i)\] 证明：\(A = \bigcup_i (A \cap B_i)\)，由不交性直接展开。 Bayes 公式：将「原因→结果」反转为「结果→原因」： \[P(B_j|A) = \frac{P(A|B_j)\,P(B_j)}{\sum_{i=1}^{n} P(A|B_i)\,P(B_i)}\] 先验 \(P(B_j)\) 经似然 \(P(A|B_j)\) 更新为后验 \(P(B_j|A)\)。

🌐 实际应用

医学检测：患病率 1%，灵敏度 99%，假阳性率 5%。阳性后真实患病 概率仅约 \(0.99 \times 0.01 / (0.99 \times 0.01 + 0.05 \times 0.99) \approx 16.7\%\)。

✏️ 例题分析与易错点

例题：工厂线 A 占产量 60%，次品率 2%；线 B 占 40%，次品率 5%。 随机取到次品，来自线 A 的概率？ 解答：\(P(\text{次}) = 0.02 \times 0.6 + 0.05 \times 0.4 = 0.032\)。 \(P(A|\text{次}) = 0.012/0.032 = 0.375\)。 易错点：直觉认为「A 产量多故次品多来自 A」，实际 \(P(B|\text{次}) = 0.625\)， B 的高次品率主导了结果。

03事件的独立性

🎯 为什么研究这个问题

两个事件「没有关系」到底是什么意思？一枚硬币的正反不影响另一枚， 但如何严格化这种「互不影响」？独立性的数学定义精炼了此直觉。然而 「两两独立」并不等于「相互独立」——Bernstein 的经典反例警示我们 必须对独立性的层次保持警惕。

📐 公式推导 / 模型建立

定义：事件 \(A, B\) 独立（记 \(A \perp\!\!\!\perp B\)）iff \[P(A \cap B) = P(A) \cdot P(B)\] 等价地（若 \(P(B) > 0\)）：\(P(A|B) = P(A)\)。 相互独立 vs 两两独立：\(A_1, \ldots, A_n\) 相互独立要求对 每个子集 \(S \subseteq \{1,\ldots,n\}\), \[P\!\left(\bigcap_{i \in S} A_i\right) = \prod_{i \in S} P(A_i)\] 两两独立仅要求 \(|S| = 2\) 的情形。 Bernstein 反例：\(\Omega = \{1,2,3,4\}\) 等概率， \(A = \{1,2\}\)，\(B = \{1,3\}\)，\(C = \{1,4\}\)。 \(P(A \cap B) = P(A \cap C) = P(B \cap C) = 1/4\) ✓ 两两独立 但 \(P(A \cap B \cap C) = 1/4 \neq 1/8 = P(A)P(B)P(C)\) ✗

🌐 实际应用

密码学中密钥各位需要相互独立（非仅两两独立），否则攻击者可 利用高阶相关性来破解加密方案。

✏️ 例题分析与易错点

例题：掷两枚骰子，\(A\) = 第一枚偶数，\(B\) = 第二枚偶数， \(C\) = 两枚之和为偶数。验证是否相互独立。 解答：\(P(A)=P(B)=P(C)=1/2\)。\(P(A \cap B) = 1/4\) ✓ \(A \cap C\)：第一偶+和偶 → 第二也偶，故 \(A \cap C = A \cap B\)， \(P(A \cap C) = 1/4\) ✓。类似 \(P(B \cap C) = 1/4\) ✓。 但 \(P(A \cap B \cap C) = P(A \cap B) = 1/4 \neq 1/8\)。非相互独立。 易错点：只检查了两两条件就断言相互独立，遗漏三重交集的验证。

04\(\sigma\)-代数与信息的数学表达

🎯 为什么研究这个问题

为什么概率论要用「神秘」的 \(\sigma\)-代数，而非简单取所有子集？ 在不可数 \(\Omega\)（如实数轴）上，并非所有子集都能合理赋予概率 （Vitali 集的教训）。\(\sigma\)-代数精确描述了「可获取的信息」—— 这个视角在条件期望和随机过程中至关重要。
