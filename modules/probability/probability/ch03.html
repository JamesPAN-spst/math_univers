01期望的定义与线性性

🎯 为什么研究这个问题

随机变量的「平均值」是概率论最基本的总结性指标。赌场的期望收益、 保险的预期赔付、投资的平均回报都依赖于期望。期望的线性性是 其最强大的性质——无需独立性就能把复杂问题分解为简单部分之和。

🔍 抽象化方法

期望本质是 Lebesgue 积分 \(E[X] = \int_\Omega X\,dP\)。这将 离散情形（求和）和连续情形（积分）统一在同一个框架下。

📐 公式推导 / 模型建立

离散情形：\(E[X] = \sum_i x_i P(X = x_i)\)。 连续情形：\(E[X] = \int_{-\infty}^{\infty} x f_X(x)\,dx\)。 一般定义（Lebesgue 积分）： \[E[X] = \int_\Omega X(\omega)\,dP(\omega)\] 存在性要求：\(E[|X|] < \infty\)（\(X \in L^1\)）。 线性性：对任意随机变量 \(X, Y\)（无需独立）， \[E[aX + bY] = aE[X] + bE[Y]\] 证明直接由 Lebesgue 积分的线性性获得。 LOTUS（无意识统计学家定理）： \[E[g(X)] = \int g(x) f_X(x)\,dx\] 无需先求 \(g(X)\) 的分布即可计算期望。

🌐 实际应用

保险精算中的纯保费 = 期望损失；赌场中「庄家优势」意味着 玩家期望收益为负。

✏️ 例题分析与易错点

例题：随机取 \(n\) 个人的帽子随机分配，期望有多少人取回自己的帽子？ 解答：令 \(X_i = 1\) 若第 \(i\) 人取回自己帽子。\(E[X_i] = 1/n\)。 由线性性：\(E[\sum_i X_i] = \sum_i E[X_i] = n \cdot 1/n = 1\)。 无论 \(n\) 多大，期望始终等于 1。 易错点：认为线性性需要 \(X_i\) 独立。实际上 \(X_i\) 高度相关， 但线性性对任意随机变量成立，无需独立。

02方差与协方差

🎯 为什么研究这个问题

期望只告诉我们「中心在哪」，但没有告诉我们「波动有多大」。方差 衡量了随机变量偏离均值的程度。而协方差和相关系数刻画了 两个变量「同涨同跌」的程度——这在金融投资组合理论中至关重要。

📐 公式推导 / 模型建立

方差： \[\text{Var}(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2\] 第二个等号证明：展开平方用线性性。 \(\text{Var}(X) \geq 0\)，等号 iff \(X\) 几乎必然为常数。 协方差： \[\text{Cov}(X,Y) = E[(X-\mu_X)(Y-\mu_Y)] = E[XY] - E[X]E[Y]\] 相关系数：\(\rho(X,Y) = \text{Cov}(X,Y) / (\sigma_X \sigma_Y)\)， \(|\rho| \leq 1\)（由 Cauchy-Schwarz 不等式）。 方差的加法： \[\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) + 2\text{Cov}(X,Y)\] 若 \(X \perp\!\!\!\perp Y\)，则 \(\text{Cov}(X,Y) = 0\)，方差可加。

🌐 实际应用

Markowitz 投资组合理论中，资产组合的风险（方差）取决于各资产之间 的协方差矩阵，分散化降低风险的数学基础就在于此。

✏️ 例题分析与易错点

例题：\(X \sim \text{Poisson}(\lambda)\)，求 \(\text{Var}(X)\)。 解答：\(E[X] = \lambda\)。 \(E[X(X-1)] = \sum_{k=0}^{\infty} k(k-1) e^{-\lambda}\lambda^k/k! = \lambda^2\)。 \(E[X^2] = E[X(X-1)] + E[X] = \lambda^2 + \lambda\)。 \(\text{Var}(X) = \lambda^2 + \lambda - \lambda^2 = \lambda\)。 易错点：Poisson 分布的均值和方差都等于 \(\lambda\)，但这是 Poisson 的特殊性质，不要误以为所有分布均值=方差。

03矩母函数与特征函数

🎯 为什么研究这个问题

分布函数和密度函数完整描述了随机变量，但并不总是易于操作。矩母 函数和特征函数提供了「变换域」的视角——就像 Laplace/Fourier 变换一样，它们将卷积变为乘法，极大简化了独立和之分布的计算。

🔍 抽象化方法

概率分布的 Fourier 变换视角：\(\varphi_X(t) = E[e^{itX}]\) 总存在， 且唯一确定分布，提供了处理概率问题的强力调和分析工具。

📐 公式推导 / 模型建立

矩母函数(MGF)： \[M_X(t) = E[e^{tX}] = \sum_{k=0}^{\infty} \frac{E[X^k]}{k!} t^k\] 优点：\(E[X^k] = M_X^{(k)}(0)\)（各阶矩的生成函数）。 缺点：不一定存在（如 Cauchy 分布 \(M(t) = \infty\)，\(t \neq 0\)）。 特征函数(CF)： \[\varphi_X(t) = E[e^{itX}] = \int e^{itx} dF_X(x)\] 始终存在（因 \(|e^{itx}| = 1\)）且唯一确定分布。 关键性质：\(X \perp\!\!\!\perp Y\) 时 \(\varphi_{X+Y}(t) = \varphi_X(t) \cdot \varphi_Y(t)\) （卷积→乘法）。这是证明 CLT 的核心工具。

🌐 实际应用

特征函数法是证明中心极限定理的标准方法；矩母函数在 概率母函数(PGF)形式下用于分支过程的细维量分析。

✏️ 例题分析与易错点

例题：用 MGF 证明 \(X \sim N(0,1)\) 的 \(E[X^2] = 1\)，\(E[X^4] = 3\)。 解答：\(M(t) = e^{t^2/2}\)。\(M'(t) = te^{t^2/2}\)，\(M'(0)=0=E[X]\)。 \(M''(t) = (1+t^2)e^{t^2/2}\)，\(M''(0)=1=E[X^2]\)。 继续求导得 \(M^{(4)}(0) = 3 = E[X^4]\)。 易错点：混淆 MGF 和 CF。MGF 为 \(E[e^{tX}]\)，CF 为 \(E[e^{itX}]\)。 对正态分布：\(M(t)=e^{t^2/2}\) 但 \(\varphi(t)=e^{-t^2/2}\)（指数符号不同）。

04重要不等式

🎯 为什么研究这个问题

在概率论中，精确计算往往困难，但不等式提供了强大的「粗略估计」 工具。Markov 和 Chebyshev 不等式仅用均值/方差就能给出尾部概率 的界；Jensen 不等式揭示了凸性对期望的影响。这些工具是大数定律 和集中不等式的基础。

📐 公式推导 / 模型建立

Markov 不等式：若 \(X \geq 0\)，\(a > 0\)， \[P(X \geq a) \leq \frac{E[X]}{a}\] 证明：\(E[X] = E[X \cdot \mathbf{1}_{X \geq a}] + E[X \cdot \mathbf{1}_{X < a}] \geq a \cdot P(X \geq a)\)。 Chebyshev 不等式：对 \(X\) 任意，\(\varepsilon > 0\)， \[P(|X - \mu| \geq \varepsilon) \leq \frac{\text{Var}(X)}{\varepsilon^2}\] 证明：对 \((X - \mu)^2\) 用 Markov， \(P(|X-\mu|^2 \geq \varepsilon^2) \leq E[(X-\mu)^2]/\varepsilon^2\)。 Jensen 不等式：若 \(g\) 为凸函数，\(E[|X|] < \infty\)，则 \[g(E[X]) \leq E[g(X)]\]

🌐 实际应用

Chebyshev 不等式直接用于弱大数律律的证明； Jensen 不等式证明了 \(E[\ln X] \leq \ln E[X]\)（AM-GM 的概率版）。

✏️ 例题分析与易错点

例题：\(X\) 的均值为 100，方差 25。求 \(P(|X - 100| \geq 20)\) 的上界。 解答：Chebyshev：\(P(|X-100| \geq 20) \leq 25/400 = 1/16 = 0.0625\)。 易错点：Markov 不等式要求 \(X \geq 0\)，若直接对可取负值的 \(X\) 用 Markov 会得到错误结果。应用于 \(|X - \mu|^2\) 则合法。
