随机变量与分布

01随机变量与分布函数

🎯 为什么研究这个问题

概率空间描述了事件的概率，但实际问题通常关心的是数值结果：掷骰子 的点数、股价的涨幅、等待时间的长短。随机变量 \(X: \Omega \to \mathbb{R}\) 是从「抽象结果」到「数值」的桥梁，分布函数 \(F_X\) 则完整刻画了 \(X\) 取各种值的概率规律。

🔍 抽象化方法

随机变量的本质是可测函数：\(X^{-1}(B) \in \mathcal{F}\)， \(\forall B \in \mathcal{B}(\mathbb{R})\)。可测性保证了「\(X \leq x\)」 这类事件具有良定义的概率。

📐 公式推导 / 模型建立

定义：\(X: (\Omega, \mathcal{F}) \to (\mathbb{R}, \mathcal{B}(\mathbb{R}))\) 为随机变量 iff \(\{\omega: X(\omega) \leq x\} \in \mathcal{F}\)， \(\forall x \in \mathbb{R}\)。 分布函数(CDF)： \[F_X(x) = P(X \leq x) = P(\{\omega: X(\omega) \leq x\})\] CDF 的三大性质： (1) 单调递增：\(x_1 < x_2 \Rightarrow F(x_1) \leq F(x_2)\) (2) 右连续：\(\lim_{x \downarrow x_0} F(x) = F(x_0)\) (3) 极限：\(\lim_{x \to -\infty} F(x) = 0\)， \(\lim_{x \to +\infty} F(x) = 1\) 反之，满足 (1)-(3) 的函数唯一确定一个概率分布。 跳跃点对应概率质量：\(P(X = x_0) = F(x_0) - F(x_0^-)\)。

🌐 实际应用

可靠性工程中，设备寿命 \(T\) 的分布函数 \(F_T(t) = P(T \leq t)\) 直接给出失效概率，指导维护策略和保修定价。

✏️ 例题分析与易错点

例题：\(X\) 的分布函数为 \(F(x) = 0\)（\(x<0\)），\(F(x) = x^2\)（\(0 \leq x \leq 1\)），\(F(x)=1\)（\(x>1\)）。 求 \(P(1/4 < X \leq 3/4)\)。 解答：\(P(1/4 < X \leq 3/4) = F(3/4) - F(1/4) = 9/16 - 1/16 = 1/2\)。 易错点：写成 \(F(3/4) - F(1/4)\) 还是 \(F(3/4^-) - F(1/4)\)？ 当 CDF 连续时两者相同。但若有跳跃点，\((a,b]\) 对应 \(F(b)-F(a)\)， \([a,b]\) 对应 \(F(b)-F(a^-)\)。

⚡ 快速检测
分布函数 \(F_X\) 为什么要求右连续而非左连续？

02离散型与连续型随机变量

🎯 为什么研究这个问题

随机变量分为两大类：离散型（取可数多个值）和连续型（取连续范围的 值）。二项分布、Poisson 分布是离散的经典；正态分布、指数分布是 连续的代表。掌握常见分布族是应用概率论的基本功。

📐 公式推导 / 模型建立

离散型随机变量：\(X\) 取至多可数个值 \(\{x_1, x_2, \ldots\}\)， 由概率质量函数(PMF) \(p(x_i) = P(X = x_i)\) 完全刻画， 满足 \(\sum_i p(x_i) = 1\)。 连续型随机变量：存在密度函数 \(f_X(x) \geq 0\) 使得 \[F_X(x) = \int_{-\infty}^{x} f_X(t)\,dt\] 此时 \(P(X = x_0) = 0\)，\(P(a < X \leq b) = \int_a^b f_X(t)\,dt\)。 常见离散分布： - Bernoulli(\(p\))：\(P(X=1)=p\)，\(P(X=0)=1-p\) - 二项 \(B(n,p)\)：\(P(X=k)=\binom{n}{k}p^k(1-p)^{n-k}\) - Poisson(\(\lambda\))：\(P(X=k)=e^{-\lambda}\lambda^k/k!\) 常见连续分布： - 均匀 \(U(a,b)\)：\(f(x)=1/(b-a)\)，\(x \in [a,b]\) - 指数 \(\text{Exp}(\lambda)\)：\(f(x)=\lambda e^{-\lambda x}\)，\(x \geq 0\) - 正态 \(N(\mu,\sigma^2)\)： \[f(x) = \frac{1}{\sigma\sqrt{2\pi}}\exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)\]

🌐 实际应用

Poisson 分布模拟呼叫中心来电数、放射性衰变计数；指数分布 建模电子元件寿命（无记忆性）；正态分布在中心极限定理下成为 大样本统计的基石。

✏️ 例题分析与易错点

例题：某服务器每小时平均收到 5 个请求（Poisson 分布）。 求一小时内恰好收到 3 个请求的概率。 解答：\(\lambda = 5\)，\(P(X=3) = e^{-5} \cdot 5^3 / 3! = 125e^{-5}/6 \approx 0.1404\)。 易错点：混淆 Poisson 的参数 \(\lambda\)（均值=方差）与指数分布的 参数 \(\lambda\)（速率）。两者关系：Poisson 的事件间隔服从 \(\text{Exp}(\lambda)\)。

⚡ 快速检测
连续型随机变量 \(X\) 满足 \(P(X = 5) = ?\)

03随机变量的函数

🎯 为什么研究这个问题

已知 \(X\) 的分布，\(g(X)\) 的分布是什么？例如 \(X \sim N(0,1)\)， \(Y = X^2\) 服从什么分布？再如收入 \(X\) 已知，税后收入 \(g(X)\) 的 分布如何？变量变换是连接已知分布与实际需求的桥梁。

📐 公式推导 / 模型建立

离散情形：若 \(Y = g(X)\)，\(X\) 取值 \(\{x_i\}\)，则 \(P(Y = y) = \sum_{i: g(x_i) = y} P(X = x_i)\)。 连续情形（\(g\) 严格单调可微）：设 \(Y = g(X)\)，\(h = g^{-1}\)， \[f_Y(y) = f_X(h(y)) \cdot |h'(y)|\] 推导（以 \(g\) 严格递增为例）： \(F_Y(y) = P(g(X) \leq y) = P(X \leq h(y)) = F_X(h(y))\)。 两边对 \(y\) 求导：\(f_Y(y) = f_X(h(y)) \cdot h'(y)\)。 若 \(g\) 递减则取绝对值。 非单调情形：分段处理。如 \(Y = X^2\)： \[F_Y(y) = P(X^2 \leq y) = P(-\sqrt{y} \leq X \leq \sqrt{y}) = F_X(\sqrt{y}) - F_X(-\sqrt{y})\] 求导得 \(f_Y(y) = \frac{1}{2\sqrt{y}}[f_X(\sqrt{y}) + f_X(-\sqrt{y})]\)，\(y > 0\)。

🌐 实际应用

若 \(X \sim N(0,1)\)，则 \(X^2 \sim \chi^2(1)\)（自由度 1 的卡方分布）， 这是假设检验中卡方检验的基础。

✏️ 例题分析与易错点

例题：\(X \sim \text{Exp}(1)\)（\(f_X(x) = e^{-x}\)，\(x \geq 0\)）。 求 \(Y = \sqrt{X}\) 的密度。 解答：\(g(x) = \sqrt{x}\) 在 \(x > 0\) 上递增，\(h(y) = y^2\)，\(h'(y) = 2y\)。 \(f_Y(y) = f_X(y^2) \cdot 2y = e^{-y^2} \cdot 2y\)，\(y > 0\)。 （这是 Rayleigh 分布！） 易错点：忘记乘以 \(|h'(y)|\) 因子（Jacobian），导致密度不归一。

⚡ 快速检测
\(X \sim U(0,1)\)，\(Y = -\ln X\)。\(Y\) 服从什么分布？

04联合分布与边缘分布

🎯 为什么研究这个问题

单个随机变量不足以描述多维随机现象：身高与体重、温度与湿度、 资产收益率之间的联合波动。联合分布刻画了多个随机变量之间的 全部概率关系，而边缘分布是从联合分布「投影」到单个变量的视角。

📐 公式推导 / 模型建立

联合分布函数：\(F_{X,Y}(x,y) = P(X \leq x, Y \leq y)\)。 联合密度（连续情形）：存在 \(f_{X,Y}(x,y) \geq 0\) 使得 \[F_{X,Y}(x,y) = \int_{-\infty}^{x}\int_{-\infty}^{y} f_{X,Y}(s,t)\,dt\,ds\] 边缘密度：通过「积掉」另一个变量获得： \[f_X(x) = \int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dy\] 注意：边缘分布不能唯一确定联合分布！相同的边缘分布可对应 不同的联合分布（不同的相关结构）。 独立性等价条件：\(X \perp\!\!\!\perp Y\) iff \(f_{X,Y}(x,y) = f_X(x) \cdot f_Y(y)\)，\(\forall x,y\)。

🌐 实际应用

金融中 Copula 理论就是在固定边缘分布的前提下，研究联合分布 如何刻画资产之间的尾部相关性。

✏️ 例题分析与易错点

例题：\((X,Y)\) 在三角形 \(\{(x,y): 0 \leq x \leq 1, 0 \leq y \leq x\}\) 上均匀分布。求边缘密度 \(f_X(x)\)。 解答：三角形面积 \(= 1/2\)，故 \(f_{X,Y}(x,y) = 2\)。 \(f_X(x) = \int_0^x 2\,dy = 2x\)，\(0 \leq x \leq 1\)。 易错点：均匀分布的密度不是 1 而是 \(1/\text{面积}\)。 此处面积 \(= 1/2\)，故密度 \(= 2\)，不是 1。

⚡ 快速检测
已知 \(f_X\) 和 \(f_Y\)，能否唯一确定 \(f_{X,Y}\)？
