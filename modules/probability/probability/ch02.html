随机变量与分布

01随机变量与分布函数

🎯 为什么研究这个问题

概率空间给出了事件的概率，但科学家和工程师更关心一个具体数字：温度是多少？股价涨了多少？排队要等几分钟？如何把「抽象结果」映射为可计算的数值？

  • 质量控制：工厂逐件检测产品尺寸，每件产品对应一个数值测量——这就是随机变量。
  • 金融定价：股票日收益率将市场状态映射为百分比变化，分布函数刻画了涨跌的全部概率规律。
  • 医学检测：血液指标将健康状态量化为连续数值，医生依据分布函数设定正常参考区间。

随机变量 \(X: \Omega \to \mathbb{R}\) 正是从「抽象结果」到「数值」的桥梁，而分布函数 \(F_X(x) = P(X \leq x)\) 用一个单调函数完整编码了 \(X\) 的全部统计信息。

🔍 抽象化方法

随机变量的本质是可测函数 \(X: (\Omega, \mathcal{F}) \to (\mathbb{R}, \mathcal{B}(\mathbb{R}))\)。可测性条件 \(\{X \leq x\} \in \mathcal{F}\) 保证了一切形如「\(X\) 不超过 \(x\)」的事件都可被赋予概率。

分布函数 \(F_X(x) = P(X \leq x)\) 是概率论的核心抽象：它像一张「概率地图」，将实数轴上的概率分配压缩为一个右连续非递减函数。不同支撑——有限、可数、连续——对应 CDF 的不同形态：阶梯形、分段光滑、处处连续。

这种抽象的力量在于统一性：无论 \(\Omega\) 多么复杂，给出 \(F_X\) 就唯一确定了 \(X\) 在 \(\mathbb{R}\) 上的概率测度 \(\mu_X(B) = P(X \in B)\)。从集合论到函数语言的转换，是测度论与应用概率之间的关键桥梁。

📐 公式推导 / 模型建立

随机变量的严格定义

\(X: (\Omega, \mathcal{F}) \to (\mathbb{R}, \mathcal{B}(\mathbb{R}))\) 为随机变量 iff \(\{\omega: X(\omega) \leq x\} \in \mathcal{F}\)，\(\forall x \in \mathbb{R}\)。直觉：\(X\) 把不可观测的 \(\omega\) 映射为可观测的数字，可测性保证概率可算。

分布函数 (CDF)

\[F_X(x) = P(X \leq x) = P(\{\omega: X(\omega) \leq x\})\] CDF 将概率信息编码为一个函数：在 \(x\) 处的函数值就是 \(X\) 落在 \((-\infty, x]\) 的概率。

CDF 的三大性质

(1) 非递减：\(x_1 < x_2 \Rightarrow F(x_1) \leq F(x_2)\)——概率在累积。
(2) 右连续：\(\lim_{x \downarrow x_0} F(x) = F(x_0)\)——由概率的从上连续性。
(3) 边界：\(\lim_{x \to -\infty} F(x) = 0\)，\(\lim_{x \to +\infty} F(x) = 1\)——概率的规范性。

反之，满足 (1)-(3) 的函数唯一确定一个概率分布。

跳跃点与概率质量

\(P(X = x_0) = F(x_0) - F(x_0^-)\)。跳跃即离散质量：连续分布在每点跳跃为零，纯离散分布则由一系列跳跃组成。

🌐 实际应用

可靠性工程：设备寿命 \(T\) 的 CDF \(F_T(t) = P(T \leq t)\) 直接给出失效概率，指导预防性维修和保修定价策略。

金融风控：VaR（在险价值）定义为收益分布的分位数 \(F_X^{-1}(\alpha)\)，是银行监管资本充足率的核心指标。

天气预报：温度预测值的 CDF 告诉决策者超过某阈值（如霜冻线 0°C）的概率，指导农业和交通应急。

✏️ 例题分析与易错点

例 1：连续 CDF 的区间概率

\(F(x) = 0\)（\(x<0\)），\(F(x) = x^2\)（\(0 \leq x \leq 1\)），\(F(x)=1\)（\(x>1\)）。求 \(P(1/4 < X \leq 3/4)\)。

解：\(P(1/4 < X \leq 3/4) = F(3/4) - F(1/4) = 9/16 - 1/16 = 1/2\)。

例 2：混合 CDF 的跳跃点

\(F(x) = 0.4 \cdot \mathbf{1}_{x \geq 1} + 0.6(1 - e^{-(x-1)})\mathbf{1}_{x \geq 1}\)。求 \(P(X = 1)\) 和 \(P(X > 2)\)。

解：\(F(1) = 0.4\)，\(F(1^-) = 0\)，故 \(P(X=1) = 0.4\)。\(P(X > 2) = 1 - F(2) = 0.6e^{-1} \approx 0.221\)。

常见错误：

  • 区间类型弄混：\((a,b]\) 用 \(F(b)-F(a)\)，\([a,b]\) 用 \(F(b)-F(a^-)\)；CDF 连续时两者无区别。
  • 忘记 CDF 是右连续的，在跳跃点遗漏点质量 \(P(X=x_0) = F(x_0) - F(x_0^-)\)。
  • 将密度值 \(f(x_0)\) 当作概率：连续分布中 \(P(X = x_0) = 0\)，\(f(x_0)\) 是密度不是概率。
⚡ 快速检测
分布函数 \(F_X\) 为什么要求右连续而非左连续？

02离散型与连续型随机变量

🎯 为什么研究这个问题

分布函数告诉我们「概率在实数轴上如何分配」，但更细致的问题是：概率质量究竟是集中在孤立点上，还是弥散在连续区间中？

  • 网络安全：单位时间攻击次数是离散计数（Poisson），需要离散模型支撑入侵检测阈值。
  • 工业制造：零部件尺寸误差是连续量（正态），质量管控依赖连续分布的分位数。
  • 保险精算：理赔次数（离散）与单笔理赔额（连续）分别建模，两者卷积才得总赔付分布。

离散与连续的分类决定了计算手段（求和 vs. 积分）和建模工具（PMF vs. 密度），是概率论应用的分水岭。

🔍 抽象化方法

从测度论视角，离散与连续的分类源于概率测度 \(\mu_X\) 关于计数测度或 Lebesgue 测度的绝对连续性。离散型的 PMF \(p(x_i)\) 是 Radon-Nikodym 导数关于计数测度的取值。

连续型的密度 \(f_X(x)\) 则是 \(d\mu_X/d\lambda\)（关于 Lebesgue 测度的 RN 导数）。密度不是概率——它可以大于 1——而是概率在无穷小区间 \(dx\) 上的「权重」：\(P(x < X \leq x+dx) \approx f(x)\,dx\)。

常见分布族（二项、Poisson、正态、指数等）并非孤立存在，而是通过参数化组成指数族，这一统一框架在统计推断中至关重要。

📐 公式推导 / 模型建立

离散型随机变量与 PMF

\(X\) 取至多可数个值 \(\{x_1, x_2, \ldots\}\)，由概率质量函数 \(p(x_i) = P(X = x_i)\) 完全刻画，满足 \(\sum_i p(x_i) = 1\)。直觉：概率质量「堆积」在离散点上。

连续型随机变量与密度

存在密度函数 \(f_X(x) \geq 0\) 使得 \[F_X(x) = \int_{-\infty}^{x} f_X(t)\,dt\] 此时 \(P(X = x_0) = 0\)，\(P(a < X \leq b) = \int_a^b f_X(t)\,dt\)。直觉：概率像流体「弥散」在区间上。

常见离散分布族

Bernoulli(\(p\))：\(P(X=1)=p\)，\(P(X=0)=1-p\)。
二项 \(B(n,p)\)：\(P(X=k)=\binom{n}{k}p^k(1-p)^{n-k}\)——\(n\) 次独立试验中成功次数。
Poisson(\(\lambda\))：\(P(X=k)=e^{-\lambda}\lambda^k/k!\)——稀有事件计数的极限模型。

常见连续分布族

均匀 \(U(a,b)\)：\(f(x)=1/(b-a)\)，\(x \in [a,b]\)。
指数 \(\text{Exp}(\lambda)\)：\(f(x)=\lambda e^{-\lambda x}\)，\(x \geq 0\)——唯一具有无记忆性的连续分布。
正态 \(N(\mu,\sigma^2)\)：\[f(x) = \frac{1}{\sigma\sqrt{2\pi}}\exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)\] 对称钟形，中心极限定理的归宿。

🌐 实际应用

电信工程：Poisson 分布模拟呼叫中心每小时来电数、放射性衰变计数，指导资源调度。

可靠性分析：指数分布建模电子元件寿命（无记忆性），其危险率恒为常数简化了维修策略设计。

统计学基石：正态分布在中心极限定理下成为大样本统计的核心，置信区间和假设检验都建立在此基础上。

✏️ 例题分析与易错点

例 1：Poisson 分布的计算

某服务器每小时平均收到 5 个请求（Poisson 分布），求一小时内恰好 3 个的概率。

解：\(\lambda = 5\)，\(P(X=3) = e^{-5} \cdot 5^3 / 3! = 125e^{-5}/6 \approx 0.1404\)。

例 2：连续型概率计算

\(X \sim \text{Exp}(2)\)，求 \(P(X > 1)\)。

解：\(P(X>1) = 1 - F(1) = e^{-2} \approx 0.135\)。

常见错误：

  • 混淆 Poisson 的参数 \(\lambda\)（均值=方差）与指数分布的参数 \(\lambda\)（速率）。两者关系：Poisson 的事件间隔服从 \(\text{Exp}(\lambda)\)。
  • 对连续型分布计算 \(P(X = a)\)，忘记单点概率恒为 0。
  • 二项分布中漏掉组合数 \(\binom{n}{k}\)，只写了 \(p^k(1-p)^{n-k}\)。
⚡ 快速检测
连续型随机变量 \(X\) 满足 \(P(X = 5) = ?\)

03随机变量的函数

🎯 为什么研究这个问题

已知 \(X\) 的分布，对 \(X\) 做一个函数变换后，\(g(X)\) 服从什么分布？这是从「已知模型」推导「感兴趣量」的核心问题。

  • 统计推断：\(X \sim N(0,1)\) 时 \(X^2\) 服从什么分布？答案（\(\chi^2(1)\)）是卡方检验的理论根基。
  • 财务分析：收入 \(X\) 已知，税后收入 \(g(X) = X - T(X)\) 的分布决定了居民可支配收入的不确定性。
  • 信号处理：通过非线性器件后信号 \(Y = g(X)\) 的噪声分布如何变化？直接影响接收机设计。

变量变换是连接已知分布与实际需求的桥梁，其核心工具是雅可比行列式（一维即 \(|h'(y)|\)）。

🔍 抽象化方法

给定 \(X\) 的分布 \(\mu_X\) 和 Borel 函数 \(g\)，\(Y = g(X)\) 的分布是 \(\mu_X\) 的前推测度 \(g_* \mu_X(B) = \mu_X(g^{-1}(B))\)。这是测度论中「换元」的本质。

当 \(g\) 可逆且光滑时，密度变换公式中的 Jacobian \(|h'(y)|\) 起到了「体积伸缩因子」的作用——正如多元微积分中的换元公式。CDF 法（先求 \(F_Y\) 再求导）是最通用的推导路径。

非单调映射（如 \(Y = X^2\)）需要分段处理逆像，本质是对逆像的多个分支分别计算 Jacobian 再求和。

📐 公式推导 / 模型建立

离散情形

若 \(Y = g(X)\)，\(X\) 取值 \(\{x_i\}\)，则 \(P(Y = y) = \sum_{i: g(x_i) = y} P(X = x_i)\)。直觉：把映射到同一个 \(y\) 值的所有 \(x_i\) 的概率加起来。

连续单调情形——密度变换公式

设 \(Y = g(X)\)，\(g\) 严格单调可微，\(h = g^{-1}\)：\[f_Y(y) = f_X(h(y)) \cdot |h'(y)|\]

推导过程（CDF 法）

以 \(g\) 严格递增为例：\(F_Y(y) = P(g(X) \leq y) = P(X \leq h(y)) = F_X(h(y))\)。两边对 \(y\) 求导：\(f_Y(y) = f_X(h(y)) \cdot h'(y)\)。递减时取绝对值统一为 \(|h'(y)|\)。

非单调情形——\(Y = X^2\) 典型推导

\[F_Y(y) = P(X^2 \leq y) = P(-\sqrt{y} \leq X \leq \sqrt{y}) = F_X(\sqrt{y}) - F_X(-\sqrt{y})\] 求导得 \(f_Y(y) = \frac{1}{2\sqrt{y}}[f_X(\sqrt{y}) + f_X(-\sqrt{y})]\)，\(y > 0\)。核心思路：两个逆像分支各贡献一项。

🌐 实际应用

假设检验：\(X \sim N(0,1)\) 时 \(X^2 \sim \chi^2(1)\)，这是卡方检验的基础。

蒙特卡洛抽样：概率积分变换 \(F(X) \sim U(0,1)\) 是从任意分布生成随机数的通用方法。

Box-Muller 变换：从两个独立 \(U(0,1)\) 生成标准正态随机数，是 CDF 法与极坐标变换的经典组合。

✏️ 例题分析与易错点

例 1：单调变换

\(X \sim \text{Exp}(1)\)，求 \(Y = \sqrt{X}\) 的密度。

解：\(h(y) = y^2\)，\(h'(y) = 2y\)。\(f_Y(y) = f_X(y^2) \cdot 2y = 2ye^{-y^2}\)（\(y > 0\)）——即 Rayleigh 分布。

例 2：非单调变换

\(X \sim N(0,1)\)，求 \(Y = X^2\) 的密度。

解：\(F_Y(y) = P(-\sqrt{y} \leq X \leq \sqrt{y}) = 2\Phi(\sqrt{y}) - 1\)。求导：\(f_Y(y) = \frac{1}{\sqrt{2\pi y}} e^{-y/2}\)（\(y>0\)），即 \(\chi^2(1)\)。

常见错误：

  • 忘记乘以 Jacobian \(|h'(y)|\)，导致密度不归一。
  • 非单调函数用单调公式直接套，遗漏了第二个逆像分支。
  • 混淆 \(f_Y(y)\) 和 \(f_X(y)\)：变换后的密度是关于 \(y\) 的函数，不是将 \(y\) 代入原密度。
⚡ 快速检测
\(X \sim U(0,1)\)，\(Y = -\ln X\)。\(Y\) 服从什么分布？

04联合分布与边缘分布

🎯 为什么研究这个问题

现实世界的不确定性几乎总是多维的——单个随机变量远远不够描述变量之间的协同关系。

  • 金融投资：资产A和资产B的收益率联合波动决定了投资组合的风险，单看边缘无法捕捉尾部相依。
  • 医学研究：血压与血糖的联合分布揭示代谢综合征的风险模式，边缘正常不代表联合正常。
  • 气象预测：温度与湿度的联合分布决定了体感温度和极端天气预警的阈值设计。

联合分布刻画了多个变量之间的全部概率关系，边缘分布是其「投影」——联合信息远多于边缘之和。

🔍 抽象化方法

联合分布 \(\mu_{X,Y}\) 定义在乘积空间 \(\mathbb{R}^2\) 的 Borel \(\sigma\)-代数上，蕴含了 \(X\) 和 \(Y\) 之间全部的概率关联。边缘分布是联合分布在坐标轴上的「投影」。

关键洞察：联合分布包含的信息严格多于两个边缘分布——相同的 \(f_X\) 和 \(f_Y\) 可以对应无穷多种联合分布（Copula 理论）。这正是为什么不能从边缘「逆向重建」联合。

独立性是最特殊的联合结构：\(f_{X,Y} = f_X \cdot f_Y\) 将乘积空间退化为边缘的张量积，此时联合分布完全由边缘决定。

📐 公式推导 / 模型建立

联合分布函数

\(F_{X,Y}(x,y) = P(X \leq x, Y \leq y)\)。联合密度（连续情形）满足 \[F_{X,Y}(x,y) = \int_{-\infty}^{x}\int_{-\infty}^{y} f_{X,Y}(s,t)\,dt\,ds\] 直觉：\(f_{X,Y}(x,y)\) 是概率在 \((x,y)\) 处的「面密度」。

边缘密度——投影操作

\[f_X(x) = \int_{-\infty}^{\infty} f_{X,Y}(x,y)\,dy\] 几何直觉：沿 \(y\) 轴「压扁」联合密度曲面，投影到 \(x\) 轴上。

联合 ≠ 边缘之和

相同的边缘分布可对应无穷多种联合分布（Copula 理论）。这意味着从边缘分布无法反推联合分布——相关结构是额外信息。

独立性的等价刻画

\(X \perp\!\!\!\perp Y\) iff \(f_{X,Y}(x,y) = f_X(x) \cdot f_Y(y)\)，\(\forall x,y\)。独立是最特殊的联合结构：联合密度分解为边缘的乘积。

🌐 实际应用

金融风控：Copula 理论在固定边缘分布的前提下，研究联合分布如何刻画资产之间的尾部相关性。

军事代码分析：联合分布用于建模多个信号源的干扰结构，边缘分布提取单个源的统计特征。

流行病学：多个风险因子（吸烟、饮食、基因）的联合分布揭示了疾病风险的交互作用，边缘分布用于单因素筛查。

✏️ 例题分析与易错点

例 1：三角形上的均匀分布

\((X,Y)\) 在 \(\{0 \leq x \leq 1, 0 \leq y \leq x\}\) 上均匀。求 \(f_X(x)\)。

解：三角形面积 \(= 1/2\)，故 \(f_{X,Y} = 2\)。\(f_X(x) = \int_0^x 2\,dy = 2x\)（\(0 \leq x \leq 1\)）。

例 2：检验独立性

\(f_{X,Y}(x,y) = 6x\)（\(0

解：\(f_X(x) = 6x(1-x)\)，\(f_Y(y) = 3y^2\)。\(f_X(x)f_Y(y) = 18x(1-x)y^2 \neq 6x\)，故不独立。

常见错误：

  • 均匀分布的密度不是 1 而是 \(1/\text{面积}\)，忘记除以区域面积。
  • 積分上下限弄错：边缘密度的积分范围由联合密度的支撑决定。
  • 从边缘分布逆推联合分布：边缘不能唯一确定联合（除非独立）。
⚡ 快速检测
