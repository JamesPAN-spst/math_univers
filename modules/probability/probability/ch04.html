01独立随机变量的严格定义

🎯 为什么研究这个问题

Ch.01 中我们定义了事件的独立性，但随机变量的独立性需要更 精密的 \(\sigma\)-代数语言。为什么「\(X, Y\) 独立」不能简单定义为 「\(\text{Cov}=0\)」？因为不相关只消除了线性关联，而独立性要求 完全没有任何关联——包括非线性的。

🔍 抽象化方法

\(X \perp\!\!\!\perp Y\) iff \(\sigma(X)\) 与 \(\sigma(Y)\) 独立（任取两个 \(\sigma\)-代数中的 事件都独立）。这是最一般的定义，涵盖了任意可测函数的 独立性。

📐 公式推导 / 模型建立

严格定义：\(X\) 和 \(Y\) 独立 iff 对所有 \(A, B \in \mathcal{B}(\mathbb{R})\)， \[P(X \in A,\; Y \in B) = P(X \in A) \cdot P(Y \in B)\] 等价于：\(F_{X,Y}(x,y) = F_X(x) \cdot F_Y(y)\)，\(\forall x,y\)。 重要关系： - \(X \perp\!\!\!\perp Y \Rightarrow \text{Cov}(X,Y) = 0\)（独立 → 不相关） - 反之不成立！反例：\(X \sim N(0,1)\)，\(Y = X^2\)， \(\text{Cov}(X,Y) = E[X^3] = 0\)，但 \(Y\) 完全由 \(X\) 决定。 独立时乘积的期望：\(E[XY] = E[X] \cdot E[Y]\)。 更一般地：\(E[g(X)h(Y)] = E[g(X)] \cdot E[h(Y)]\)。

🌐 实际应用

统计学中，i.i.d. 样本的假设（独立同分布）是几乎所有经典推断 的基础。打破独立假设会导致模型失效（如时间序列数据）。

✏️ 例题分析与易错点

例题：\(X \sim N(0,1)\)，\(Y = X^2\)。证明不相关但不独立。 解答：\(\text{Cov}(X,Y) = E[XY] - E[X]E[Y] = E[X^3] - 0 \cdot 1 = 0\) （正态分布奇数阶矩为 0）。 但 \(P(Y \leq 1 | X = 2) = 0\)（因 \(Y = 4\)），而 \(P(Y \leq 1) > 0\)。 故 \(X, Y\) 不独立。 易错点：从不相关推出独立。仅当联合正态分布时， 不相关 \(\Rightarrow\) 独立。

02独立随机变量之和与卷积

🎯 为什么研究这个问题

若 \(X, Y\) 独立，\(Z = X + Y\) 服从什么分布？这是概率论最基本 的问题之一。答案是卷积公式——它将两个密度「折叠」成新密度。 卷积计算在变换域（特征函数/MGF）下将简化为简单乘法。

📐 公式推导 / 模型建立

设 \(X \perp\!\!\!\perp Y\)，\(Z = X + Y\)，则 \(Z\) 的密度为卷积： \[(f_X * f_Y)(z) = \int_{-\infty}^{\infty} f_X(z-y) f_Y(y)\,dy\] 推导：\(F_Z(z) = P(X+Y \leq z) = \iint_{x+y \leq z} f_X(x)f_Y(y)\,dx\,dy\) （独立性使联合密度 = 边缘之积）。换元 \(x = z-y\) 得卷积。 在变换域中： \[\varphi_{X+Y}(t) = \varphi_X(t) \cdot \varphi_Y(t)\] 应用：\(X \sim N(\mu_1, \sigma_1^2)\)，\(Y \sim N(\mu_2, \sigma_2^2)\) 独立， \(X + Y \sim N(\mu_1+\mu_2, \sigma_1^2+\sigma_2^2)\)（特征函数相乘即得）。

🌐 实际应用

信号处理中，信号+噪声的分布是卷积；保险中，多个独立理赔的 总额分布也通过卷积计算。

✏️ 例题分析与易错点

例题：\(X \sim \text{Exp}(1)\)，\(Y \sim \text{Exp}(1)\) 独立。 求 \(Z = X + Y\) 的密度。 解答：\((f*f)(z) = \int_0^z e^{-(z-y)} e^{-y}\,dy = e^{-z} \cdot z\)，\(z > 0\)。 即 \(Z \sim \text{Gamma}(2, 1)\)。 易错点：卷积积分的上下限。当 \(f_X, f_Y\) 仅在正实数上有定义时， 积分从 \(0\) 到 \(z\)（不是 \(-\infty\) 到 \(+\infty\)）。

03Borel-Cantelli 引理

🎯 为什么研究这个问题

一个事件序列 \(A_1, A_2, \ldots\) 中，「无穷多个」会发生的条件是什么？ Borel-Cantelli 引理给出了干净的答案：概率之和收敛则只发生有限 次；若独立且发散则发生无穷次。这是概率论中最常用的 0-1 型结果。

🔍 抽象化方法

「无穷多个 \(A_n\) 发生」的事件即上极限： \(\limsup_n A_n = \bigcap_{n=1}^{\infty} \bigcup_{k=n}^{\infty} A_k\)。

📐 公式推导 / 模型建立

BC1（第一部分）：若 \(\sum_n P(A_n) < \infty\)，则 \[P(\limsup_n A_n) = 0\] 证明：\(P(\bigcup_{k=n}^{\infty} A_k) \leq \sum_{k=n}^{\infty} P(A_k) \to 0\) （级数尾部趋于 0）。取交集得结果。 BC2（第二部分）：若 \(A_n\) 相互独立且 \(\sum_n P(A_n) = \infty\)，则 \[P(\limsup_n A_n) = 1\] 证明要点：\(P(\bigcap_{k=n}^{m} A_k^c) = \prod_{k=n}^{m}(1-P(A_k)) \leq \exp(-\sum_{k=n}^{m} P(A_k)) \to 0\)（因级数发散）。

🌐 实际应用

在随机算法中，BC 用于证明“算法最终必然成功”或 “异常事件只发生有限次”。

✏️ 例题分析与易错点

例题：\(A_n\) 独立，\(P(A_n) = 1/n\)。证明无穷多个 \(A_n\) 几乎必然发生。 解答：\(\sum P(A_n) = \sum 1/n = \infty\)，且 \(A_n\) 独立， 由 BC2：\(P(\limsup A_n) = 1\)。 易错点：BC2 的独立性条件不可省略。若 \(A_n = A\)（同一事件）， \(P(A) = 1/2\)，\(\sum P(A_n) = \infty\)，但 \(P(\limsup A_n) = P(A) = 1/2 \neq 1\)。

04Kolmogorov 0-1 律与尾 \(\sigma\)-代数

🎯 为什么研究这个问题

独立随机变量序列的「尾部事件」——只与 \(X_{n+1}, X_{n+2}, \ldots\) 有关而与前有限个无关的事件——只能有概率 0 或 1。这个 深刻的 0-1 律表明尾部事件是「确定性」的——要么必然发生， 要么必然不发生。

🔍 抽象化方法

尾 \(\sigma\)-代数 \(\mathcal{T} = \bigcap_n \sigma(X_n, X_{n+1}, \ldots)\)。 0-1 律将钢下移到了熵的层面：尾部事件无法携带普通信息。

📐 公式推导 / 模型建立

尾 \(\sigma\)-代数：\(\mathcal{T}_n = \sigma(X_n, X_{n+1}, \ldots)\)， \(\mathcal{T} = \bigcap_{n=1}^{\infty} \mathcal{T}_n\)。 尾部事件的例子：\(\{\sum_n X_n \text{ 收敛}\}\)， \(\{\limsup_n S_n/n = c\}\)，\(\{\omega: \bar{X}_n \to \mu\}\)。 Kolmogorov 0-1 律：若 \(X_1, X_2, \ldots\) 独立，则对所有 \(A \in \mathcal{T}\)， \[P(A) = 0 \quad \text{or} \quad P(A) = 1\] 证明要点：\(\mathcal{T}\) 与 \(\sigma(X_1, \ldots, X_n)\) 独立（对所有 \(n\)）， 因此 \(\mathcal{T}\) 与 \(\sigma(X_1, X_2, \ldots)\) 独立，但 \(\mathcal{T} \subseteq \sigma(X_1, X_2, \ldots)\)，所以 \(\mathcal{T}\) 与自己独立， 即 \(P(A) = P(A \cap A) = P(A)^2\)，故 \(P(A) \in \{0,1\}\)。

🌐 实际应用

0-1 律保证了：对 i.i.d. 序列，\(\bar{X}_n \to \mu\) 的概率只能是 0 或 1，这和强大数定律关系密切。

✏️ 例题分析与易错点

例题：\(X_n\) i.i.d.，\(A = \{\omega: \sum_n X_n \text{ 收敛}\}\)。 证明 \(P(A) \in \{0, 1\}\)。 解答：\(A \in \mathcal{T}\)（级数是否收敛不依赖于前有限项）。 由 Kolmogorov 0-1 律，\(P(A) = 0\) 或 \(P(A) = 1\)。 易错点：误以为 0-1 律对任意事件成立。它仅适用于尾事件 （属于 \(\mathcal{T}\)），且需要独立性。
