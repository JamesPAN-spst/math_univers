条件期望

01条件期望的定义

🎯 为什么研究这个问题

在获得部分信息 \(\mathcal{G}\) 后，对 \(X\) 的「最佳预测」是什么？ 条件期望 \(E[X|\mathcal{G}]\) 给出了答案：它是 \(\mathcal{G}\)-可测的 随机变量，在 \(L^2\) 意义下最佳逼近 \(X\)。这个概念是鞅论、 随机过程和贝叶斯统计的基石。

🔍 抽象化方法

\(E[X|\mathcal{G}]\) 是 \(X\) 在 \(L^2(\Omega, \mathcal{G}, P)\) 上的正交投影。 它保留了 \(\mathcal{G}\) 能「看见」的信息，丢弃了其余部分。

📐 公式推导 / 模型建立

定义：\(E[X|\mathcal{G}]\) 是满足以下两条的随机变量： (1) \(E[X|\mathcal{G}]\) 是 \(\mathcal{G}\)-可测的 (2) 对所有 \(A \in \mathcal{G}\)， \[\int_A E[X|\mathcal{G}]\,dP = \int_A X\,dP\] 存在唯一性：由 Radon-Nikodym 定理保证（a.s. 唯一）。 \(L^2\) 投影解释：若 \(X \in L^2\)，则 \[E[X|\mathcal{G}] = \arg\min_{Y \in L^2(\mathcal{G})} E[(X - Y)^2]\] 即 \(E[X|\mathcal{G}]\) 是 \(X\) 在闭子空间 \(L^2(\mathcal{G})\) 上的 正交投影。

🌐 实际应用

金融中，\(E[X|\mathcal{F}_t]\) 是基于 \(t\) 时刻信息对未来收益的 最佳预测；机器学习中的敏慏性分析也涉及条件期望。

✏️ 例题分析与易错点

例题：\(X \sim U(0,1)\)，\(\mathcal{G} = \sigma(\mathbf{1}_{X > 1/2})\) （只知 \(X\) 是否大于 \(1/2\)）。求 \(E[X|\mathcal{G}]\)。 解答：\(\mathcal{G} = \{\emptyset, [0,1/2], (1/2,1], \Omega\}\)。 在 \(\{X \leq 1/2\}\) 上：\(E[X|\mathcal{G}] = E[X|X \leq 1/2] = 1/4\)。 在 \(\{X > 1/2\}\) 上：\(E[X|\mathcal{G}] = E[X|X > 1/2] = 3/4\)。 易错点：\(E[X|\mathcal{G}]\) 是随机变量（依赖于 \(\omega\)），不是常数！

02条件期望的核心性质

🎯 为什么研究这个问题

条件期望的强大之处在于它的一系列代数性质：线性性、塔性质、 可测抽出、独立时的简化。这些性质使得复杂的条件期望计算 可以逐步化简，是鞅论和随机过程的计算工具箱。

🔍 抽象化方法

条件期望的四条核心性质（线性、塔、独立简化、可测抽出）不是孤立的公式，而是 \(L^2\) 正交投影的自然推论。塔性质说的是「先投到小子空间再投到更小子空间 = 直接投到更小子空间」。

可测抽出 \(E[XY|\mathcal{G}] = YE[X|\mathcal{G}]\) 对应线性代数中的「已知系数提出来」——\(\mathcal{G}\)-可测的 \(Y\) 在投影中扮演常数角色。

独立性使投影退化为标量：若 \(X \perp\!\!\!\perp \mathcal{G}\)，则 \(E[X|\mathcal{G}] = E[X]\)，因为 \(\mathcal{G}\) 不包含关于 \(X\) 的信息。这些代数规则构成了鞅计算和随机过程推导的工具箱。

📐 公式推导 / 模型建立

核心性质（设 \(\mathcal{G}_2 \subseteq \mathcal{G}_1 \subseteq \mathcal{F}\)）： (i) 线性性：\(E[aX + bY | \mathcal{G}] = aE[X|\mathcal{G}] + bE[Y|\mathcal{G}]\) (ii) 塔性质(Tower property)： \[E[E[X|\mathcal{G}_1] | \mathcal{G}_2] = E[X|\mathcal{G}_2]\] 特别地：\(E[E[X|\mathcal{G}]] = E[X]\)（全期望公式）。 (iii) 独立时：若 \(X \perp\!\!\!\perp \mathcal{G}\)，则 \(E[X|\mathcal{G}] = E[X]\)。 (iv) 可测抽出：若 \(Y\) 是 \(\mathcal{G}\)-可测的，则 \[E[XY|\mathcal{G}] = Y \cdot E[X|\mathcal{G}]\]

🌐 实际应用

塔性质是动态规划中 Bellman 方程的数学基础； 可测抽出在金融定价中广泛使用。

✏️ 例题分析与易错点

例题：设 \(X, Y\) 独立，\(E[X] = 3\)。求 \(E[X + Y^2 | Y]\)。 解答：由线性性：\(E[X + Y^2 | Y] = E[X|Y] + E[Y^2|Y]\)。 \(X \perp\!\!\!\perp Y\)，故 \(E[X|Y] = E[X] = 3\)。 \(Y^2\) 是 \(\sigma(Y)\)-可测的，故 \(E[Y^2|Y] = Y^2\)。 结果：\(E[X + Y^2 | Y] = 3 + Y^2\)。 易错点：将 \(E[Y^2|Y]\) 误算为 \(E[Y^2] = E[Y]^2 + \text{Var}(Y)\)。 \(Y^2\) 已是 \(\sigma(Y)\)-可测的，可立即抽出。

03条件方差公式

🎯 为什么研究这个问题

总方差可以分解为两部分：「组间方差」和「组内方差的期望」。这个 分解在方差分析(ANOVA)、Rao-Blackwell 定理和混合模型中都 起关键作用。

🔍 抽象化方法

条件方差公式 \(\text{Var}(X) = E[\text{Var}(X|Y)] + \text{Var}(E[X|Y])\) 是 \(L^2\) 分解的方差版本，类比勾股定理：总变异 = 组内变异 + 组间变异。

\(\text{Var}(E[X|Y])\) 衡量了条件均值随 \(Y\) 变化的幅度——即 \(Y\) 能「解释」多少 \(X\) 的变异。\(E[\text{Var}(X|Y)]\) 是给定 \(Y\) 后仍无法消除的「残余不确定性」。

此分解在统计中即 ANOVA 的理论内核，在机器学习中对应偏差-方差分解的条件版本，在精算中用于层次模型的风险分析。

📐 公式推导 / 模型建立

条件方差公式(Eve’s law / Law of Total Variance)： \[\text{Var}(X) = E[\text{Var}(X|\mathcal{G})] + \text{Var}(E[X|\mathcal{G}])\] 证明：记 \(\mu_G = E[X|\mathcal{G}]\)。 \(\text{Var}(X) = E[X^2] - (E[X])^2\)。 \(E[\text{Var}(X|\mathcal{G})] = E[E[X^2|\mathcal{G}] - \mu_G^2] = E[X^2] - E[\mu_G^2]\)。 \(\text{Var}(\mu_G) = E[\mu_G^2] - (E[\mu_G])^2 = E[\mu_G^2] - (E[X])^2\)。 两式相加即得。

🌐 实际应用

ANOVA 中 SST = SSB + SSW 恰好是条件方差公式的离散版本； Rao-Blackwell 定理用它证明充分统计量降低方差。

✏️ 例题分析与易错点

例题：\(N \sim \text{Poisson}(\lambda)\)，给定 \(N = n\) 时 \(X | N \sim \text{Binomial}(n, p)\)。求 \(\text{Var}(X)\)。 解答：\(E[X|N] = Np\)，\(\text{Var}(X|N) = Np(1-p)\)。 \(E[\text{Var}(X|N)] = E[Np(1-p)] = \lambda p(1-p)\)。 \(\text{Var}(E[X|N]) = \text{Var}(Np) = p^2 \lambda\)。 \(\text{Var}(X) = \lambda p(1-p) + p^2 \lambda = \lambda p\)。 易错点：将 \(\text{Var}(X|N)\) 当作常数而非随机变量。 \(\text{Var}(X|N) = Np(1-p)\) 是依赖于 \(N\) 的。

⚡ 快速检测
条件方差公式 \(\text{Var}(X) = E[\text{Var}(X|Y)] + \text{Var}(E[X|Y])\)

04正则条件分布与鞅的引入

🎯 为什么研究这个问题

条件期望自然引向条件分布，而正则条件分布将条件概率从事件 提升到分布的层次。更重要的是，条件期望开启了鞅论的大门： \(E[X_{n+1}|\mathcal{F}_n] = X_n\) 就是鞅的定义，它描述了 「公平赌博」的数学结构。

🔍 抽象化方法

条件期望→滤→适应过程→鞅，这条概念链构成了 现代概率论与随机过程的核心主线。

📐 公式推导 / 模型建立

正则条件分布：给定 \(\mathcal{G}\)，如何定义 \(P(X \in B | \mathcal{G})\)？形式上，它是一个随机核 \(K(\omega, B)\)，满足： (a) 对固定 \(\omega\)，\(B \mapsto K(\omega, B)\) 是概率测度 (b) 对固定 \(B\)，\(\omega \mapsto K(\omega, B) = E[\mathbf{1}_{X \in B}|\mathcal{G}]\) 鞅的预告：设 \(\{\mathcal{F}_n\}\) 为滤，过程 \(\{X_n\}\) 适应于 \(\{\mathcal{F}_n\}\)。 若对所有 \(n\)： \[E[X_{n+1}|\mathcal{F}_n] = X_n \quad \text{a.s.}\] 则 \(\{X_n\}\) 为鞅。鞅的直觉：「知道过去后，对未来的 最佳预测就是当前值」——公平赌博。

🌐 实际应用

股价的“有效市场假说”可用鞅建模；赌博中赌徒财富过程是 （上）鞅，解释了为什么没有必胜策略。

✏️ 例题分析与易错点

例题：\(X_n\) 为简单随机游走 \(X_n = \sum_{i=1}^{n} Y_i\)， \(Y_i = \pm 1\) 等概率。证明 \(\{X_n\}\) 是鞅。 解答：\(E[X_{n+1}|\mathcal{F}_n] = E[X_n + Y_{n+1}|\mathcal{F}_n] = X_n + E[Y_{n+1}] = X_n + 0 = X_n\)。 （用了 \(Y_{n+1} \perp\!\!\!\perp \mathcal{F}_n\) 和可测抽出） 易错点：验证鞅时忘记检查可积性 \(E[|X_n|] < \infty\)。
