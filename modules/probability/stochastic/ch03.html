01滤与鞅的定义

🎯 为什么研究这个问题

鞅是概率论中最优美的概念之一，它描述了「公平赌博」的精确 数学结构。从股价模型到随机分析，鞅无处不在。它的定义 \(E[X_{n+1}|\mathcal{F}_n] = X_n\) 意味着：基于当前世界状态的最佳 预测就是当前值本身。

🔍 抽象化方法

滤(filtration) \(\{\mathcal{F}_n\}\) 描述信息的累积；适应过程要求 \(X_n\) 只能依赖于到 \(n\) 时刻的信息；鞅则添加了 「无漂移」的约束。

📐 公式推导 / 模型建立

定义：滤 \(\{\mathcal{F}_n\}_{n \geq 0}\) 是 \(\sigma\)-代数的递增序列： \(\mathcal{F}_0 \subseteq \mathcal{F}_1 \subseteq \cdots \subseteq \mathcal{F}\)。 过程 \(\{X_n\}\) 适应于 \(\{\mathcal{F}_n\}\) iff \(X_n\) 是 \(\mathcal{F}_n\)-可测 （即 \(X_n\) 只依赖于时刻 \(n\) 及之前的信息）。 鞅：若 \(E[|X_n|] < \infty\) 且 \[E[X_{n+1}|\mathcal{F}_n] = X_n \quad \text{a.s.}, \forall n\] 上鞅：\(E[X_{n+1}|\mathcal{F}_n] \leq X_n\)（期望趋降） 下鞅：\(E[X_{n+1}|\mathcal{F}_n] \geq X_n\)（期望趋升） 性质：\(\{X_n\}\) 鞅 iff \(\{X_n\}\) 同时是上鞅和下鞅。 \(\{X_n\}\) 鞅 \(\Rightarrow\) \(E[X_n] = E[X_0]\)（常数）。

🌐 实际应用

金融中，折现股价在有效市场假说下是鞅；赌博中， 公平赌徒的财富过程是鞅。

✏️ 例题分析与易错点

例题：证明 \(M_n = e^{\theta S_n - n \psi(\theta)}\) 是鞅，其中 \(S_n = \sum X_i\)， \(X_i\) i.i.d.，\(\psi(\theta) = \ln E[e^{\theta X_1}]\)。 解答：\(E[M_{n+1}|\mathcal{F}_n] = M_n \cdot E[e^{\theta X_{n+1} - \psi(\theta)}] = M_n \cdot e^{-\psi(\theta)} E[e^{\theta X}] = M_n \cdot 1 = M_n\)。 易错点：忘记验证 \(E[|M_n|] < \infty\)（可积性条件）。

02停时与 Doob 可选停时定理

🎯 为什么研究这个问题

在赌博中，「赢了就走」是一种停止策略。停时是基于当前信息 做出的「何时停止」的决策。Doob 可选停时定理告诉我们： 对于鞅（公平游戏），任何停止策略都无法创造获利机会。

🔍 抽象化方法

停时 \(\tau\) 是「不偏瞀未来」的随机时刻：\(\{\tau \leq n\} \in \mathcal{F}_n\)。

📐 公式推导 / 模型建立

停时定义：\(\tau: \Omega \to \{0,1,2,\ldots\} \cup \{\infty\}\) 为停时 iff \(\{\tau \leq n\} \in \mathcal{F}_n\)，\(\forall n\)。 Doob 可选停时定理：若 \(\{X_n\}\) 是鞅，\(\tau\) 是 a.s. 有界停时 （\(\tau \leq N\) a.s.），则 \[E[X_\tau] = E[X_0]\] 更一般的版本：若 \(\{X_n\}\) 鞅且 \(\tau\) a.s. 有限，则 \(E[X_\tau] = E[X_0]\) 在以下任一条件下成立： (a) \(\tau\) 有界 (b) \(E[\tau] < \infty\) 且 \(|X_{n+1} - X_n|\) 有界 (c) \(E[|X_\tau|] < \infty\) 且 \(E[X_n \mathbf{1}_{\tau > n}] \to 0\)

🌐 实际应用

证明赌博中无必胜策略；金融中的期权定价。

✏️ 例题分析与易错点

例题：随机游走 \(S_n = \sum_{i=1}^n Y_i\)（\(Y_i = \pm 1\) 等概率）。 \(\tau = \min\{n: S_n = a \text{ or } S_n = -b\}\)，\(S_0 = 0\)。求 \(P(S_\tau = a)\)。 解答：\(S_n\) 是鞅。\(E[\tau] < \infty\)（可证），且 \(|S_{n+1}-S_n|=1\)。 \(E[S_\tau] = E[S_0] = 0\)。\(E[S_\tau] = a \cdot P(S_\tau = a) + (-b)(1-P(S_\tau = a))\)。 解得 \(P(S_\tau = a) = b/(a+b)\)。 易错点：直接用 OST 而不验证其适用条件（如 \(E[\tau] < \infty\)）。

03Doob 鞅收敛定理与极大不等式

🎯 为什么研究这个问题

鞅会收敛吗？Doob 的收敛定理给出了肯定答案：\(L^1\)-有界上鞅 a.s. 收敛。而 Doob 极大不等式控制了鞅的波动大小，是 鞅论中最重要的工具性结果。

🔍 抽象化方法

收敛定理的核心思想是「穿越」(upcrossing)论证： 有界上鞅穿越任意区间 \([a,b]\) 的次数有限，即不会无限振荡。

📐 公式推导 / 模型建立

Doob 鞅收敛定理：若 \(\{X_n\}\) 为上鞅且 \(\sup_n E[X_n^+] < \infty\)， 则 \(X_n \to X_\infty\) a.s.，且 \(E[|X_\infty|] < \infty\)。 证明要点：设 \(U_n([a,b])\) = \(X_0, \ldots, X_n\) 穿越 \([a,b]\) 的次数。 Doob 穿越引理：\((b-a) E[U_n([a,b])] \leq E[(X_n - a)^-]\)。 有界性保证 \(E[U_\infty] < \infty\)，故 \(U_\infty < \infty\) a.s.， 即 \(X_n\) 不无限振荡，必收敛。 Doob 极大不等式：若 \(\{X_n\}\) 为下鞅，则 \[P\!\left(\max_{0 \leq k \leq n} X_k \geq \lambda\right) \leq \frac{E[X_n^+]}{\lambda}\] \(L^p\) 版本 (\(p > 1\))： \[E\!\left[\max_{0 \leq k \leq n} |X_k|^p\right] \leq \left(\frac{p}{p-1}\right)^p E[|X_n|^p]\]

🌐 实际应用

收敛定理保证了许多随机算法的收敛性；极大不等式用于 控制随机过程的尾部概率。

✏️ 例题分析与易错点

例题：\(X_n\) 为非负上鞅，\(E[X_0] = 10\)。求 \(P(\max_n X_n \geq 50)\) 的上界。 解答：\(X_n\) 非负上鞅，则 \(-X_n\) 是非正下鞅。 由鞅性质 \(E[X_n] \leq E[X_0] = 10\)。 由 Doob 极大不等式：\(P(\max_n X_n \geq 50) \leq 10/50 = 1/5\)。 易错点：对上鞅与下鞅的极大不等式方向混淆。

04Doob 分解与鞅变换

🎯 为什么研究这个问题

任何适应可积过程都可分解为「鞅部分」+「可预测部分」。 Doob 分解是随机版的「信号+漂移」分离。鞅变换则是 离散版随机积分，为 Itô 积分铺路。

🔍 抽象化方法

Doob 分解将随机世界分为「惊喜」（鞅）和「趋势」（可预测）， 是随机分析的基本分解工具。

📐 公式推导 / 模型建立

Doob 分解：任何适应可积过程 \(\{X_n\}\) 可唯一分解为 \[X_n = X_0 + M_n + A_n\] 其中 \(M_n\) 是鞅（\(M_0 = 0\)），\(A_n\) 是可预测过程（\(A_0 = 0\)， \(A_n\) 是 \(\mathcal{F}_{n-1}\)-可测的）。 构造：\(A_n = \sum_{k=1}^{n}(E[X_k|\mathcal{F}_{k-1}] - X_{k-1})\)， \(M_n = X_n - X_0 - A_n\)。 若 \(\{X_n\}\) 是上鞅，则 \(A_n\) 递减；若是鞅，则 \(A_n = 0\)。 鞅变换（离散随机积分）：若 \(\{M_n\}\) 是鞅，\(\{H_n\}\) 可预测，则 \[(H \cdot M)_n = \sum_{k=1}^{n} H_k (M_k - M_{k-1})\] 也是鞅（「确定性赌注策略不能将公平游戏变为有利」）。

🌐 实际应用

鞅变换是离散版 Itô 积分，在金融中对应自融资交易策略。

✏️ 例题分析与易错点

例题：\(X_n\) 为随机游走（鞅）。\(H_n = 1\)（常数策略）。 \((H \cdot X)_n = \sum_{k=1}^n (X_k - X_{k-1}) = X_n - X_0\)。 这仍是鞅（整体平移），确认确定性策略无法获利。 易错点：\(H_n\) 必须是可预测的（\(\mathcal{F}_{n-1}\)-可测）， 不能依赖于未来信息。
