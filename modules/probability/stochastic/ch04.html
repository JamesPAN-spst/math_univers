01Brown 运动的定义与存在性

🎯 为什么研究这个问题

Brown 运动（标准 Wiener 过程）是整个随机分析的基石。它最初 用于描述花粉在水中的不规则运动(1827)，后被 Einstein(1905) 和 Wiener(1923) 严格化。它是随机积分、随机微分方程和金融 数学的起点。

🔍 抽象化方法

Brown 运动是随机游走的连续极限（Donsker 定理）， 也是 Gauss 过程中最核心的例子。

📐 公式推导 / 模型建立

定义：\(\{B_t\}_{t \geq 0}\) 为标准 Brown 运动 iff (B1) \(B_0 = 0\) (B2) 独立增量：\(B_t - B_s \perp\!\!\!\perp B_u - B_v\)（无重叠区间） (B3) 正态增量：\(B_t - B_s \sim N(0, t-s)\)（\(t > s\)） (B4) 连续路径：\(t \mapsto B_t(\omega)\) a.s. 连续 存在性（概念）：可通过 Kolmogorov 延拓定理或 Lévy-Ciesielski 构造（用 Schauder 基展开为级数）证明存在性。 协方差结构：\(E[B_s B_t] = \min(s, t)\)。

🌐 实际应用

金融学中，股价模型的核心随机驱动力；物理中，扩散过程的 基本模型；机器学习中，score-based 扩散模型的基础。

✏️ 例题分析与易错点

例题：计算 \(E[B_3 \cdot B_5]\)。 解答：\(E[B_3 B_5] = \min(3,5) = 3\)。 验证：\(B_3 B_5 = B_3(B_3 + (B_5-B_3))\)，\(E[B_3^2] = 3\)， \(E[B_3(B_5-B_3)] = 0\)（独立增量）。 易错点：认为 \(E[B_s B_t] = st\)。实际是 \(\min(s,t)\)。

02路径性质与二次变差

🎯 为什么研究这个问题

Brown 运动的路径极其“粗糙”：连续但处处不可微。这不仅是 数学奇异，更是随机微积分存在的根本原因：因为 \(B_t\) 有 无界变差，经典 Riemann-Stieltjes 积分失效，必须引入 Itô 积分。二次变差 \([B]_t = t\) 是这一切的关键。

🔍 抽象化方法

二次变差是随机微积分中 \((dB)^2 = dt\) 的严格版本。

📐 公式推导 / 模型建立

a.s. 处处不可微：\(\lim_{h \to 0} (B_{t+h} - B_t)/h\) a.s. 不存在。 直觉：\((B_{t+h}-B_t)/h \sim N(0, 1/h) \to \infty\)。 无界变差：在 \([0,T]\) 上，一次变差 \(\sum |B_{t_{i+1}} - B_{t_i}| = \infty\) a.s. 二次变差： \[[B]_T = \lim_{\|\Pi\| \to 0} \sum_i (B_{t_{i+1}} - B_{t_i})^2 = T \quad \text{(in \(L^2\) and a.s.)}\] 证明要点：\(E[(B_{t_{i+1}}-B_{t_i})^2] = \Delta t_i\)， \(\text{Var}[(B_{t_{i+1}}-B_{t_i})^2] = 2(\Delta t_i)^2\)。 总方差 = \(2\sum (\Delta t_i)^2 \leq 2 \max \Delta t_i \cdot T \to 0\)。

🌐 实际应用

二次变差 \([B]_t = t\) 是 Itô 公式中额外 \(\frac{1}{2}f'' dt\) 项的来源。

✏️ 例题分析与易错点

例题：证明 \(B_t^2 - t\) 是鞅。 解答：\(E[B_{t+s}^2 - (t+s)|\mathcal{F}_t] = E[(B_t + (B_{t+s}-B_t))^2|\mathcal{F}_t] - t - s = B_t^2 + 2B_t \cdot 0 + s - t - s = B_t^2 - t\) ✓ 易错点：\(B_t^2\) 不是鞅（\(E[B_t^2] = t\) 递增），但 \(B_t^2 - t\) 才是鞅（扣除了二次变差）。

03Brown 运动的鞅性与反射原理

🎯 为什么研究这个问题

Brown 运动是连续时间鞅的典范。反射原理则利用路径对称性将 “首达时间”问题转化为简单的尾部概率计算，是计算 Brown 运动首达时分布的优雅方法。

📐 公式推导 / 模型建立

\(B_t\) 是鞅：\(E[B_{t+s}|\mathcal{F}_t] = B_t + E[B_{t+s}-B_t] = B_t\)。 反射原理：设 \(\tau_a = \inf\{t: B_t = a\}\)（\(a > 0\)）。 定义反射过程 \(\tilde{B}_t = B_t \mathbf{1}_{t < \tau_a} + (2a - B_t)\mathbf{1}_{t \geq \tau_a}\)。 由对称性，\(\tilde{B}\) 也是 Brown 运动。 因此： \[P(\max_{0 \leq s \leq t} B_s \geq a) = 2 P(B_t \geq a) = 2(1 - \Phi(a/\sqrt{t}))\] 首达时分布：\(f_{\tau_a}(t) = \frac{a}{\sqrt{2\pi t^3}} e^{-a^2/(2t)}\)，\(t > 0\)。

🌐 实际应用

金融中的障碍期权（股价首次触及障碍的时间分布）。

✏️ 例题分析与易错点

例题：\(B_t\) 标准 Brown 运动。求 \(P(\max_{0 \leq s \leq 1} B_s \geq 2)\)。 解答：\(P(\max B_s \geq 2) = 2P(B_1 \geq 2) = 2(1 - \Phi(2)) \approx 2 \times 0.0228 = 0.0456\)。 易错点：误以为 \(P(\max B_s \geq a) = P(B_t \geq a)\)（少了系数 2）。

04标度不变性与 Brown 桥

🎯 为什么研究这个问题

Brown 运动具有美妙的自相似性：放大或缩小时间尺度后，统计 特征不变。而 Brown 桥是「确定起点和终点」的 Brown 运动， 在统计检验（如 KS 检验）中扮演关键角色。

🔍 抽象化方法

标度不变性是 Brown 运动作为分形对象的体现： Hurst 指数 \(H = 1/2\)。

📐 公式推导 / 模型建立

标度不变性：对任意 \(c > 0\)， \[\{c^{-1/2} B_{ct}\}_{t \geq 0} \overset{d}{=} \{B_t\}_{t \geq 0}\] 证明：只需验证两边的有限维分布相同。\(c^{-1/2}B_{ct}\) 是 Gauss 过程，均值 0，协方差 \(c^{-1}\min(cs,ct)=\min(s,t)\) ✓ Brown 桥：\(B_t^0 = B_t - tB_1\)（\(0 \leq t \leq 1\)）。 性质：\(B_0^0 = B_1^0 = 0\)，\(B_t^0\) 是 Gauss 过程， \[E[B_s^0 B_t^0] = \min(s,t) - st \qquad (0 \leq s,t \leq 1)\]

🌐 实际应用

KS 检验统计量的极限分布由 Brown 桥的最大值决定； 隐含波动率模型中用到 Brown 桥。

✏️ 例题分析与易错点

例题：求 Brown 桥在 \(t = 1/2\) 的方差。 解答：\(\text{Var}(B_{1/2}^0) = \min(1/2, 1/2) - (1/2)^2 = 1/2 - 1/4 = 1/4\)。 易错点：认为 Brown 桥的方差与 Brown 运动相同。 实际 \(\text{Var}(B_t^0) = t(1-t) \leq t = \text{Var}(B_t)\)。
