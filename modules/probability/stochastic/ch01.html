01Markov 性质与转移矩阵

🎯 为什么研究这个问题

天气预报、股价模型、网页排名算法——这些矋看迫不相关的问题都有一个 共同结构：未来只依赖于当前状态，不依赖于历史。这就是 Markov 性质（无记忆性）。Markov 链是最基本的随机过程模型，它将复杂的 随机动态归结为矩阵运算。

🔍 抽象化方法

无记忆性将随机过程的全部信息压缩到一个转移矩阵 \(P\)。 \(n\) 步转移只需计算 \(P^n\)——矩阵幂替代了复杂的路径计算。

📐 公式推导 / 模型建立

定义：取值于可数状态空间 \(S\) 的序列 \(\{X_n\}\) 是 Markov 链 iff \[P(X_{n+1} = j \;|\; X_n = i, X_{n-1}, \ldots, X_0) = P(X_{n+1} = j \;|\; X_n = i)\] 转移概率：\(p_{ij} = P(X_{n+1} = j \;|\; X_n = i)\)（时齐情形）。 转移矩阵 \(P = (p_{ij})\) 满足：\(p_{ij} \geq 0\)， \(\sum_j p_{ij} = 1\)（行随机矩阵）。 \(n\) 步转移概率： \[p_{ij}^{(n)} = P(X_{m+n} = j \;|\; X_m = i) = (P^n)_{ij}\] Chapman-Kolmogorov 方程：\(P^{m+n} = P^m \cdot P^n\)。

🌐 实际应用

Google PageRank 算法将网页视为 Markov 链的状态， 平稳分布对应网页重要性排名。

✏️ 例题分析与易错点

例题：天气模型，\(S = \{\text{晴}, \text{雨}\}\)， \(P = \begin{pmatrix} 0.7 & 0.3 \\ 0.4 & 0.6 \end{pmatrix}\)。 今天晴，后天晴的概率？ 解答：\(P^2 = P \cdot P\)。\((P^2)_{11} = 0.7 \times 0.7 + 0.3 \times 0.4 = 0.61\)。 易错点：混淆行和列的含义。约定 \(p_{ij}\) 是从 \(i\)（行）到 \(j\)（列）。

02状态分类与不可约性

🎯 为什么研究这个问题

并非所有状态都一样：有些状态会被反复访问（常返），有些将被 永久离开（暂态）。另外，「不可约」意味着任意两状态可达， 「非周期」意味着不会被困在固定周期中。这些分类决定了 Markov 链的长期行为。

📐 公式推导 / 模型建立

可达性：\(i \to j\) iff \(\exists n \geq 1\) 使 \(p_{ij}^{(n)} > 0\)。 互通(communicate)：\(i \leftrightarrow j\) iff \(i \to j\) 且 \(j \to i\)。 不可约(irreducible)：所有状态互通。 常返(recurrent)：从 \(i\) 出发，回到 \(i\) 的概率 = 1： \[f_i = P(\exists n \geq 1: X_n = i \;|\; X_0 = i) = 1\] 暂态(transient)：\(f_i < 1\)。 等价判定：\(i\) 常返 iff \(\sum_{n=1}^{\infty} p_{ii}^{(n)} = \infty\)。 正常返 vs 零常返：设 \(\mu_i = E[T_i \;|\; X_0 = i]\)（平均回返时间）。 \(\mu_i < \infty\)：正常返；\(\mu_i = \infty\)：零常返。 周期：\(d(i) = \gcd\{n \geq 1: p_{ii}^{(n)} > 0\}\)。\(d(i)=1\) 即非周期。

🌐 实际应用

网络的连通性分析可用不可约性判断；随机游走在 \(\mathbb{Z}\) 上 常返但在 \(\mathbb{Z}^3\) 上暂态（Pólya 定理）。

✏️ 例题分析与易错点

例题：\(\mathbb{Z}\) 上简单随机游走（\(p = q = 1/2\)）是常返还是暂态？ 解答：\(p_{00}^{(2n)} = \binom{2n}{n}(1/2)^{2n} \sim 1/\sqrt{\pi n}\)。 \(\sum p_{00}^{(2n)} \sim \sum 1/\sqrt{n} = \infty\)，故常返。 但 \(\mu_0 = \infty\)，是零常返。 易错点：常返不等于「很快回来」。\(\mathbb{Z}\) 上的随机游走 确定会回来，但平均回返时间无穷大。

03平稳分布与收敛

🎯 为什么研究这个问题

Markov 链运行很久后会达到「平衡」吗？平稳分布 \(\pi\) 描述了 这种平衡状态，它满足 \(\pi P = \pi\)。什么条件下平稳分布存在、 唯一，且链确实收敛到它？这是 Markov 链理论的核心问题。

🔍 抽象化方法

平稳分布是转移矩阵的「左特征向量」（特征值 = 1）。这将 Markov 链的长期行为与线性代数的谱理论联系起来。

📐 公式推导 / 模型建立

定义：分布 \(\pi = (\pi_j)\) 为平稳分布 iff \[\pi_j = \sum_i \pi_i p_{ij}, \quad \forall j \qquad (\text{即 } \pi P = \pi)\] 且 \(\sum_j \pi_j = 1\)，\(\pi_j \geq 0\)。 存在与唯一性定理：若 Markov 链不可约且正常返，则 平稳分布唯一存在，且 \(\pi_i = 1/\mu_i\)。 收敛定理：若不可约、正常返、非周期，则 \[\lim_{n \to \infty} p_{ij}^{(n)} = \pi_j, \quad \forall i, j\] 收敛速度用混合时间(mixing time)度量： \(t_{\text{mix}}(\varepsilon) = \min\{n: \max_i \|P^n(i,\cdot) - \pi\|_{TV} \leq \varepsilon\}\)。

🌐 实际应用

MCMC 方法利用收敛定理：构造以目标分布为平稳分布的 Markov 链，运行足够久后即得到目标分布的样本。

✏️ 例题分析与易错点

例题：天气模型 \(P = \begin{pmatrix} 0.7 & 0.3 \\ 0.4 & 0.6 \end{pmatrix}\)。 求平稳分布。 解答：\(\pi P = \pi\)：\(\pi_1 = 0.7\pi_1 + 0.4\pi_2\)， \(\pi_2 = 0.3\pi_1 + 0.6\pi_2\)。加上 \(\pi_1 + \pi_2 = 1\)。 \(0.3\pi_1 = 0.4\pi_2 \Rightarrow \pi_1/\pi_2 = 4/3\)。 \(\pi = (4/7, 3/7)\)。 易错点：解 \(\pi P = \pi\) 时忘记加归一化条件 \(\sum \pi_j = 1\)。

04可逆链与细致平衡

🎯 为什么研究这个问题

如何构造以指定 \(\pi\) 为平稳分布的 Markov 链？细致平衡条件 提供了简单的充分条件。Metropolis-Hastings 算法和 Gibbs 采样 正是基于此原理设计的。

🔍 抽象化方法

可逆性是时间反演对称性：链的前进和后退看起来一样。

📐 公式推导 / 模型建立

细致平衡(detailed balance)：若存在分布 \(\pi\) 使得 \[\pi_i p_{ij} = \pi_j p_{ji}, \quad \forall i, j\] 则 \(\pi\) 是平稳分布。 证明：\(\sum_i \pi_i p_{ij} = \sum_i \pi_j p_{ji} = \pi_j \sum_i p_{ji} = \pi_j\)。 Metropolis-Hastings：给定目标 \(\pi\) 和提议矩阵 \(Q\)， 令接受率 \[\alpha(i,j) = \min\!\left(1, \frac{\pi_j q_{ji}}{\pi_i q_{ij}}\right)\] 则转移 \(p_{ij} = q_{ij} \alpha(i,j)\)（\(i \neq j\)）满足细致平衡。

🌐 实际应用

MCMC 中的 Metropolis-Hastings 和 Gibbs 采样是贝叶斯统计和 机器学习中从复杂后验分布采样的标准工具。

✏️ 例题分析与易错点

例题：\(S = \{1,2,3\}\)，\(\pi = (0.2, 0.5, 0.3)\)。提议矩阵为均匀 \(q_{ij} = 1/2\)（\(i \neq j\)）。求 \(\alpha(1,2)\)。 解答：\(\alpha(1,2) = \min(1, \pi_2 q_{21} / (\pi_1 q_{12})) = \min(1, 0.5/(0.2)) = \min(1, 2.5) = 1\)。 易错点：忘记 M-H 接受率中 \(q_{ji}/q_{ij}\) 的比值（当提议矩阵 不对称时）。

⚡ 快速检测
细致平衡 \(\pi_i p_{ij} = \pi_j p_{ji}\) 是 \(\pi\) 为平稳分布的
