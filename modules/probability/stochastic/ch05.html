Poisson 过程

01Poisson 过程的定义与等价表述

🎯 为什么研究这个问题

如果 Brown 运动描述的是「连续波动」，Poisson 过程描述的就是 「随机事件的计数」：电话来电数、放射性衰变、网站访问量。 Poisson 过程是仅次于 Brown 运动的第二重要随机过程。

🔍 抽象化方法

Poisson 过程 = 「连续时间上的随机计数」，与 Brown 运动互补： 一个是连续路径+连续状态，另一个是跳跃路径+离散状态。

📐 公式推导 / 模型建立

三种等价定义： (1) 计数定义：\(\{N(t)\}\) 为 Poisson(\(\lambda\)) 过程 iff \(N(0)=0\)，独立增量，\(N(t)-N(s) \sim \text{Poisson}(\lambda(t-s))\)。 (2) 微分定义：\(P(N(t+h)-N(t)=1) = \lambda h + o(h)\)， \(P(N(t+h)-N(t) \geq 2) = o(h)\)。 (3) 构造定义：间隔时间 \(T_i \overset{iid}{\sim} \text{Exp}(\lambda)\)， 到达时间 \(S_n = \sum_{i=1}^{n} T_i\)，\(N(t) = \max\{n: S_n \leq t\}\)。 等价性的关键：\(S_n \sim \text{Gamma}(n, \lambda)\)， \[P(N(t) \geq n) = P(S_n \leq t)\]

🌐 实际应用

电信流量建模、保险理赔计数、放射性衰变。

✏️ 例题分析与易错点

例题：某网站平均每分钟 2 次访问。求 3 分钟内恰好 5 次访问的概率。 解答：\(N(3) \sim \text{Poisson}(6)\)。 \(P(N(3) = 5) = e^{-6} \cdot 6^5 / 5! = 7776 e^{-6}/120 \approx 0.1606\)。 易错点：混淆参数 \(\lambda\) (速率) 和 \(\lambda t\) (期间期望值)。

⚡ 快速检测
Poisson 过程中，第 \(n\) 个事件的到达时间 \(S_n\) 服从

02叠加、分解与非齐次 Poisson 过程

🎯 为什么研究这个问题

当多个独立 Poisson 源合并时，结果仍是 Poisson。当事件被 随机分类时，各类独立。这两个简洁的性质极大简化了 复杂系统的建模。非齐次版本则允许事件速率随时间变化。

📐 公式推导 / 模型建立

叠加：若 \(N_1 \sim \text{PP}(\lambda_1)\)，\(N_2 \sim \text{PP}(\lambda_2)\) 独立， 则 \(N_1 + N_2 \sim \text{PP}(\lambda_1 + \lambda_2)\)。 分解：\(N \sim \text{PP}(\lambda)\)，每个事件独立地以概率 \(p\) 归为 I 类， \(1-p\) 归为 II 类。则 \(N_I \sim \text{PP}(\lambda p)\)， \(N_{II} \sim \text{PP}(\lambda(1-p))\)，且 \(N_I \perp\!\!\!\perp N_{II}\)。 非齐次 Poisson 过程：强度函数 \(\lambda(t)\) 随时间变化。 \(N(t) - N(s) \sim \text{Poisson}\!\left(\int_s^t \lambda(u)\,du\right)\)， 仍保持独立增量。

🌐 实际应用

电话流量：不同时段来电率不同用非齐次建模； 保险：不同类型理赔对应叠加随机点过程的分解。

✏️ 例题分析与易错点

例题：某路口左转 (\(\lambda_1=2\)/min) 和直行 (\(\lambda_2=3\)/min) 的车辆 分别为 Poisson。求 1 分钟内总车流超过 7 辆的概率。 解答：总车流 \(\sim \text{Poisson}(5)\)。 \(P(N > 7) = 1 - \sum_{k=0}^{7} e^{-5} 5^k/k! \approx 0.1334\)。 易错点：叠加时参数相加（不相乘）。

⚡ 快速检测
\(N \sim \text{PP}(10)\) 的事件以 \(p=0.3\) 分为 A 类。\(N_A\) 服从

03复合 Poisson 过程

🎯 为什么研究这个问题

当每次事件不仅被计数，还带有随机大小（如保险理赔额、地震震级）时， 复合 Poisson 过程将计数和随机大小结合起来。

📐 公式推导 / 模型建立

定义：\(S(t) = \sum_{i=1}^{N(t)} Y_i\)，其中 \(N \sim \text{PP}(\lambda)\)， \(Y_i\) i.i.d.（与 \(N\) 独立）。 矩计算：\(E[S(t)] = \lambda t \cdot E[Y]\)， \[\text{Var}(S(t)) = \lambda t \cdot E[Y^2]\] （不是 \(\lambda t \cdot \text{Var}(Y)\)！） 证明：用条件方差公式： \(\text{Var}(S) = E[\text{Var}(S|N)] + \text{Var}(E[S|N]) = E[N \cdot \text{Var}(Y)] + \text{Var}(N \cdot E[Y]) = \lambda t \text{Var}(Y) + (E[Y])^2 \lambda t = \lambda t E[Y^2]\)。 \(S(t)\) 的 MGF：\(M_{S(t)}(\theta) = \exp(\lambda t (M_Y(\theta) - 1))\)。

🌐 实际应用

保险中的集体风险模型：\(S(t)\) = 时间 \(t\) 内的累计理赔总额。

✏️ 例题分析与易错点

例题：保险公司理赔到达率 \(\lambda = 5\)/天，每次理赔额 \(Y \sim \text{Exp}(1/1000)\) （均值 1000 元）。求一天累计理赔额的均值和方差。 解答：\(E[S(1)] = 5 \times 1000 = 5000\) 元。 \(E[Y^2] = \text{Var}(Y) + (E[Y])^2 = 10^6 + 10^6 = 2 \times 10^6\)。 \(\text{Var}(S(1)) = 5 \times 2 \times 10^6 = 10^7\)。 易错点：误用 \(\text{Var}(S) = \lambda t \cdot \text{Var}(Y)\)。正确是 \(\lambda t \cdot E[Y^2]\)（包含了均值平方项）。

⚡ 快速检测
复合 Poisson \(S(t) = \sum_{i=1}^{N(t)} Y_i\) 的方差包含

04Poisson 过程的应用

🎯 为什么研究这个问题

Poisson 过程在保险精算中有典型应用：Cramér-Lundberg 模型描述 保险公司的盈余过程，“破产概率”是精算学的核心问题。

📐 公式推导 / 模型建立

Cramér-Lundberg 模型：盈余过程 \[R(t) = u + ct - S(t)\] \(u\) = 初始资本，\(c\) = 保费率，\(S(t) = \sum_{i=1}^{N(t)} Y_i\)。 破产概率：\(\psi(u) = P(\exists t > 0: R(t) < 0)\)。 安全负荷条件：\(c > \lambda E[Y]\)（保费超过期望理赔）。 指数理赔下：若 \(Y \sim \text{Exp}(\alpha)\)，则 \[\psi(u) = \frac{\lambda}{\alpha c} e^{-(\alpha - \lambda/c) u}\]

🌐 实际应用

再保险定价、网络流量建模、地震风险评估。

✏️ 例题分析与易错点

例题：\(u = 10\)，\(c = 3\)，\(\lambda = 2\)，\(Y \sim \text{Exp}(1)\)。求破产概率。 解答：检查 \(c = 3 > \lambda E[Y] = 2\) ✓。 \(\alpha = 1\)，\(\psi(10) = (2/3) e^{-(1-2/3) \cdot 10} = (2/3) e^{-10/3} \approx 0.024\)。 易错点：忘记检查安全负荷条件。若 \(c \leq \lambda E[Y]\)， 破产概率 = 1（必然破产）。

⚡ 快速检测
Cramér-Lundberg 模型中，增加初始资本 \(u\) 对破产概率的影响
