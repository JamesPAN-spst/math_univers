重复博弈

01有限次重复博弈

🎯 为什么研究这个问题

很多现实互动不是一次性的：企业反复竞争、商业伙伴长期合作、 国际关系的反复博弈。直觉上，反复互动应该促进合作， 但有限次重复的结果令人失望：逆向归纳将合作完全瓦解。

🔍 抽象化方法

有限重复博弈是动态博弈的一种，SPE 量化了「末日效应」。

📐 公式推导 / 模型建立

有限次重复囚徒困境：重复 \(T\) 次。 逆向归纳：第 \(T\) 轮是单次囚徒困境，占优策略为背叛。 第 \(T-1\) 轮：第 \(T\) 轮不受影响（占优背叛），故第 \(T-1\) 轮等价于单次博弈， 仍然背叛。以此类推… 定理：有限次重复囚徒困境的唯一 SPE 是每轮都背叛。 \[\sigma_i^*(t) = D \quad \forall t = 1, \ldots, T \quad \forall i\] 末日效应：固定的终点导致合作无法维持， 因为末轮无惩罚可能，递归地毛到每一轮。

🌐 实际应用

- “最后一天”问题：离职前员工可能消极工作

✏️ 例题分析与易错点

例题：两人囚徒困境重复 100 次。找 SPE。 解答：第 100 轮占优背叛，第 99 轮无法用第 100 轮威胁， 仍占优背叛…递归到第 1 轮。SPE = 每轮背叛。 易错点：实验中人类常在有限重复博弈中合作， 这与 SPE 预测矛盾，说明人类行为与完全理性有偏差。

02无限次重复博弈与民间定理

🎯 为什么研究这个问题

无确定终点时末日效应消失，合作变得可能。民间定理(Folk Theorem) 表明，在无限重复博弈中，只要玩家足够耐心（贴现因子足够大）， 几乎任何合理的支付都可以通过 NE 实现，包括合作。

🔍 抽象化方法

无限重复 + 贴现因子 \(\delta < 1\) → 未来的威胁变得可信。 民间定理刻画了均衡支付的可行域。

📐 公式推导 / 模型建立

无限重复博弈：每轮玩同一个阶段博弈 \(G\)， 总支付为贴现和： \[U_i = (1-\delta) \sum_{t=0}^{\infty} \delta^t u_i^{(t)}\] 針锋触发策略(Grim Trigger)： 始于合作，一旦对方背叛则永久背叛。 合作的 NE 条件（囚徒困境中“背叛”不划算）： \[\frac{-1}{1-\delta} \geq 0 + \frac{\delta(-2)}{1-\delta} \implies \delta \geq \frac{1}{2}\] 民间定理：对于任何个体理性支付向量 \((v_1, \ldots, v_n)\)， 若 \(v_i > \bar{v}_i\)（各人收益超过安全值）， 则当 \(\delta\) 足够大时存在 SPE 实现该支付。

🌐 实际应用

- 贸易伙伴之间的长期合作与信任建立 - 密码学中反复互动协议的安全性分析

✏️ 例题分析与易错点

例题：无限重复囚徒困境中用針锋触发实现合作的最低 \(\delta\)。 囚徒困境支付：CC=(-1,-1), CD=(-3,0), DC=(0,-3), DD=(-2,-2)。 解答：背叛的临时收益为 \(0-(-1)=1\)，未来惩罚为每轮 \(-2-(-1)=-1\)。 不背叛条件：\(1 \leq \frac{\delta}{1-\delta} \cdot 1\)，\(\delta \geq 1/2\)。 易错点：民间定理说的是「存在」均衡支持合作，不说合作是唯一 NE。

03进化博弈与进化稳定策略

🎯 为什么研究这个问题

传统博弈论假设玩家完全理性，但生物/社会中的策略选择 往往通过模仿学习而非深思熟虑。进化博弈论用自然选择的 逻辑替代理性假设：收益高的策略被更多“采纳”。

🔍 抽象化方法

从「理性选择」到「种群动力学」：策略的优劣由其在种群中的 性能决定，而非个体的理性推理。

📐 公式推导 / 模型建立

进化稳定策略(ESS)：\(s^*\) 是 ESS iff 对所有变异策略 \(s \neq s^*\)： \[u(s^*, s^*) > u(s, s^*) \quad \text{(严格占优)}\] 或 \[u(s^*, s^*) = u(s, s^*) \text{ 且 } u(s^*, s) > u(s, s) \quad \text{(二级条件)}\] 直觉：\(s^*\) 的种群中没有变异可以入侵。 ESS 与 NE 的关系：每个 ESS 都是对称 NE， 但反之不然。ESS 比 NE 更强。 \[\text{ESS} \subset \text{symmetric NE}\]

🌐 实际应用

- 动物行为学中的鹰鸽博弈（Hawk-Dove） - 社会规范的演化解释

✏️ 例题分析与易错点

例题：鹰鸽博弈矩阵：\(\begin{pmatrix}(-1,-1)&(2,0)\\(0,2)&(1,1)\end{pmatrix}\)。 找 ESS。 解答：纯策略检查： 鹰(H): \(u(H,H)=-1 < u(D,H)=0\)，不满足严格占优→非 ESS。 鸿(D): \(u(D,D)=1 > u(H,D)=0? \;\) \(2>1\)... \(u(H,D)=2>u(D,D)=1\)， 不满足第一条件→非 ESS。 无纯策略 ESS，混合 ESS 为混合 NE（可证 \(p^*=1/2\) 是 ESS）。 易错点：ESS 比 NE 更强，不是每个 NE 都是 ESS。

04复制动力学方程

🎯 为什么研究这个问题

ESS 描述「均衡状态」，但没说怎么到达。复制动力学(Replicator Dynamics)用微分方程描述策略分布的演化过程： 成功的策略其复制量增加，与生物演化完美对应。

🔍 抽象化方法

策略比例为动态系统的状态，收益驱动进化。 与微分方程建立桥梁。

📐 公式推导 / 模型建立

复制动力学：设 \(x_k\) 为策略 \(k\) 的种群比例， \[\dot{x}_k = x_k [f_k(x) - \bar{f}(x)]\] 其中 \(f_k(x)\) 是策略 \(k\) 的适应度（期望支付），\(\bar{f}(x) = \sum x_k f_k(x)\) 是种群平均。 性质： - \(\sum \dot{x}_k = 0\)，策略分布始终在单纯形上 - NE 是复制动力学的不动点 - ESS 是复制动力学的渐近稳定点 \[\text{ESS} \Rightarrow \text{渐近稳定} \Rightarrow \text{Lyapunov 稳定}\]

🌐 实际应用

- 传染病学中的行为动力学（接种/不接种的演化） - 多智能体系统中策略学习的收敛性分析

✏️ 例题分析与易错点

例题：2×2 对称博弈 \(A = \begin{pmatrix}3&0\\5&1\end{pmatrix}\)。 写出复制动力学方程并分析稳定性。 解答：设 \(x\) 为选策略 1 的比例。 \(f_1 = 3x + 0(1-x) = 3x\), \(f_2 = 5x + 1(1-x) = 4x+1\)。 \(\bar{f} = xf_1 + (1-x)f_2\)。 \(\dot{x} = x(f_1 - \bar{f}) = x(1-x)(f_1 - f_2) = x(1-x)(3x - 4x - 1) = x(1-x)(-x-1)\)。 因为 \(-x-1 < 0\) 恒成立，故 \(\dot{x} < 0\)（当 \(x \in (0,1)\)）。 所以 \(x \to 0\)，策略 2 是唯一稳定均衡。 易错点：\(f_1 - f_2\) 的符号决定演化方向，不是 \(f_1\) 本身。
