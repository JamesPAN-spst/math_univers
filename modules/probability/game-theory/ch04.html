01扩展型博弈与博弈树

🎯 为什么研究这个问题

现实中许多博弈是顺序行动的：国际谈判中先报价再还价， 棋类博弈中交替行棋。策略型表示丢失了行动顺序信息。 扩展型表示（博弈树）保留了时间结构，允许分析动态决策。

🔍 抽象化方法

博弈树 = 有向树，节点为决策点/自然节点/终端节点。 从策略型到扩展型是信息的增加。

📐 公式推导 / 模型建立

扩展型博弈的组成： - 博弈树 \(T\)：有根有向树 - 决策节点分配函数 \(P: \text{nodes} \to N\) - 信息集 \(H\)：将同一玩家的节点分组，同一信息集内无法区分 - 支付向量 \(u: \text{terminals} \to \mathbb{R}^n\) 行为策略：\(s_i\) 为每个信息集指定一个行动。 完美信息博弈：每个信息集都是单元素集， 玩家知道博弈的完整历史。 \[|h| = 1 \quad \forall h \in H\]

🌐 实际应用

- 商业谈判中的顺序报价建模 - 棋类博弈的博弈树表示与 AI 搜索

✏️ 例题分析与易错点

例题：两人分蛋糕博弈：玩家1提议分配 \((x, 1-x)\)，玩家2接受或拒绝。 拒绝则双方得 0。画出博弈树。 解答：根节点为玩家1选 \(x\)， 下层玩家2选择 Accept/Reject。 终端支付：Accept→\((x,1-x)\)，Reject→\((0,0)\)。 易错点：不要混淆「完美信息」与「完全信息」（后者指知道支付矩阵）。

02逆向归纳与子博弈完美均衡

🎯 为什么研究这个问题

动态博弈可能有「不可信的威胁」：玩家声称会做某事但真到那时 不会执行。子博弈完美均衡(SPE)通过逆向归纳消除这类威胁。 这是动态博弈分析的核心工具。

🔍 抽象化方法

子博弈完美 = 在每个子博弈中都是 NE 的策略组合。 NE 的加强版，排除了不可信威胁。

📐 公式推导 / 模型建立

子博弈：从博弈树的某个节点开始的子树，本身构成一个博弈。 子博弈完美均衡(SPE)：\(\sigma^*\) 是 SPE iff \(\sigma^*\) 在每个子博弈 \(G'\) 中归纳出的策略都是 \(G'\) 的 NE。 逆向归纳(Backward Induction)： (1) 从博弈树的叶子开始 (2) 对每个倒数第一个决策节点，选择使该玩家支付最大的行动 (3) 将该节点替换为对应的支付向量 (4) 重复直到根节点 Zermelo 定理：每个有限完美信息博弈都有纯策略 SPE。 \[\text{Finite + Perfect info} \Rightarrow \exists \text{ pure SPE}\]

🌐 实际应用

- 国际谈判中识别不可信的威胁 - 算法设计中的动态规划 ↔ 逆向归纳

✏️ 例题分析与易错点

例题：分蛋糕博弈的 SPE。玩家1提议 \(x \in [0,1]\)，玩家2接受/拒绝。 解答：逆向归纳：玩家2对任何 \(1-x > 0\) 都应接受。 玩家1知道这一点，因此提议 \(x \to 1\)。 SPE: 玩家1几乎拿走全部，玩家2接受。 易错点：「玩家2威胁拒绝不公平的提议」不是 SPE， 因为拒绝得 0 不如接受任何正数。

03不完美信息博弈与贝叶斯 NE

🎯 为什么研究这个问题

很多现实博弈中玩家不知道对手的类型或私有信息： 拍卖中不知道他人估值，谈判中不知道对方的底线。 Harsanyi 以「自然」的随机抽取将不完全信息转化为不完美信息。

🔍 抽象化方法

不完全信息博弈 → Harsanyi 转换 → 不完美信息博弈 + 贝叶斯更新。

📐 公式推导 / 模型建立

Harsanyi 转换：「自然」先抽取每个玩家的类型 \(\theta_i \sim F_i\)， 每人知道自己的类型但不知道他人的。 贝叶斯 NE：策略 \(\sigma_i(\theta_i)\) 是贝叶斯 NE iff \[\forall i,\; \forall \theta_i,\; \forall s_i':\quad E_{\theta_{-i}|\theta_i}[u_i(\sigma_i(\theta_i), \sigma_{-i}(\theta_{-i}), \theta)] \geq E_{\theta_{-i}|\theta_i}[u_i(s_i', \sigma_{-i}(\theta_{-i}), \theta)]\] 每个类型在给定信念下最优。

🌐 实际应用

- 拍卖设计中竞价者的私有估值建模

✏️ 例题分析与易错点

例题：二价拍卖中每人估值 \(v_i \sim U[0,1]\)。证明真实报价是贝叶斯 NE。 解答：二价拍卖中支付 = 第二高价，与自己报价无关（只影响胜负）。 报高不增加支付但可能赢得负收益的物品， 报低可能失去正收益的机会。因此诚实报价 = 占优策略。 易错点：不是所有拍卖都有诚实报价均衡，这是二价拍卖的特殊性质。

04合作博弈与 Shapley 值

🎯 为什么研究这个问题

非合作博弈假设玩家独立行动，但现实中常有联盟、合作、 分配的情境（如联合投资、共同项目、政治联盟）。 Shapley 值提供了「公平」分配的公理化方案。

🔍 抽象化方法

合作博弈 = \((N, v)\)，其中 \(v: 2^N \to \mathbb{R}\)，\(v(\emptyset)=0\)。 \(v(S)\) = 联盟 \(S\) 能获得的总价值。

📐 公式推导 / 模型建立

Shapley 值：玩家 \(i\) 的分配为 \[\phi_i(v) = \sum_{S \subseteq N \setminus \{i\}} \frac{|S|! (n-|S|-1)!}{n!} [v(S \cup \{i\}) - v(S)]\] 直觉：\(v(S \cup \{i\}) - v(S)\) = \(i\) 对联盟 \(S\) 的边际贡献， 对所有可能的加入顺序取平均。 Shapley 公理： (1) 效率：\(\sum_i \phi_i = v(N)\) (2) 对称：可互换玩家得相同分配 (3) 虚拟玩家：不贡献者得 0 (4) 可加性：\(\phi_i(v+w) = \phi_i(v) + \phi_i(w)\) 这四条公理唯一确定了 Shapley 值。

🌐 实际应用

- ML 中的 SHAP 值：用 Shapley 值解释特征贡献 - 投票权力指数（联合国安理会的投票权分析）

✏️ 例题分析与易错点

例题：3 人博弈：\(v(\{1\})=0, v(\{2\})=0, v(\{3\})=0\), \(v(\{1,2\})=6, v(\{1,3\})=6, v(\{2,3\})=0, v(\{1,2,3\})=12\)。 计算 Shapley 值。 解答：\(\phi_1\)： 加入 \(\emptyset\): \(v(\{1\})-v(\emptyset) = 0\)，权重 \(2!/6 = 1/3\) 加入 \(\{2\}\): \(6-0=6\)，权重 \(1/6\) 加入 \(\{3\}\): \(6-0=6\)，权重 \(1/6\) 加入 \(\{2,3\}\): \(12-0=12\)，权重 \(1/3\) \(\phi_1 = 0 \cdot 1/3 + 6/6 + 6/6 + 12/3 = 0 + 1 + 1 + 4 = 6\)。 同理 \(\phi_2 = \phi_3 = 3\)。验证：\(6+3+3=12 = v(N)\)✓ 易错点：权重不是均匀的，而是 \(\frac{|S|!(n-|S|-1)!}{n!}\)。
