01社会选择与显示原理

🎯 为什么研究这个问题

博弈论通常分析「给定规则下的策略」，机制设计反过来： 如何设计规则使参与者的自私行为导向理想结果？ 这是“逆向博弈论”，应用于拍卖、投票、公共品等场景。 Arrow 不可能定理和 Gibbard-Satterthwaite 定理揭示了基本限制。

🔍 抽象化方法

机制 = 从参与者报告到结果的映射。 显示原理：只需考虑诚实报告的机制。

📐 公式推导 / 模型建立

社会选择函数：\(f: \Theta^n \to X\)，\(\Theta\) 为类型空间，\(X\) 为结果空间。 机制：\(\mathcal{M} = (M_1, \ldots, M_n, g)\)，\(M_i\) 为消息空间， \(g: M_1 \times \cdots \times M_n \to X\) 为结果函数。 激励相容：机制 \(\mathcal{M}\) 实现 \(f\) iff 存在均衡策略使得结果 = \(f(\theta)\) 对所有类型 \(\theta\)。 显示原理(Revelation Principle)：若 \(f\) 可由机制 \(\mathcal{M}\) 实现， 则它可由直接机制（\(M_i = \Theta_i\)，即报告类型）实现。 \[\forall i,\; \forall \theta_i,\; \forall \theta_i':\quad E[u_i(f(\theta_i, \theta_{-i}), \theta_i)] \geq E[u_i(f(\theta_i', \theta_{-i}), \theta_i)]\]

🌐 实际应用

- 政府采购机制设计 - 分布式系统中的激励相容协议

✏️ 例题分析与易错点

例题：解释为什么显示原理简化了机制设计问题。 解答：原先需要在所有可能的机制中搜索（无穷多）， 显示原理说只需考虑直接机制（从 \(\Theta^n\) 到 \(X\)）， 搜索空间大大缩小。 易错点：显示原理不说诚实报告总是最优的， 它说的是如果有任何机制能实现 \(f\)，那么直接机制也能。

02VCG 机制与激励相容

🎯 为什么研究这个问题

如何设计一个机制使得诚实报告是占优策略？Vickrey-Clarke-Groves(VCG) 机制是拟线性框架下唯一实现诚实报告的机制，广泛应用于 拍卖、公共品供给、资源分配等场景。

🔍 抽象化方法

VCG 的核心思想：让每个参与者「内化」他对他人的外部性。

📐 公式推导 / 模型建立

VCG 机制： (1) 结果规则：最大化社会总价值 \[x^*(\hat{\theta}) = \arg\max_x \sum_i v_i(x, \hat{\theta}_i)\] (2) 支付规则（Clarke Pivot）： \[t_i = \sum_{j \neq i} v_j(x^*(\hat{\theta}), \hat{\theta}_j) - \sum_{j \neq i} v_j(x^*_{-i}(\hat{\theta}_{-i}), \hat{\theta}_j)\] 其中 \(x^*_{-i}\) 是没有 \(i\) 时的最优结果。 占优激励相容的证明思路： \(i\) 的净收益 = \(v_i(x^*) + t_i = v_i(x^*) + [\sum_{j\neq i} v_j(x^*)] - C\) \(= \sum_j v_j(x^*) - C\)，正是社会总价值减去与 \(i\) 报告无关的常数。 因此最大化自己收益 = 最大化社会总价值，诚实报告占优。

🌐 实际应用

- Google 广告拍卖（GSP 的 VCG 变体） - 频谱竞购交易中的组合拍卖

✏️ 例题分析与易错点

例题：3 人竞标 1 个物品，估值 \(v_1=10, v_2=7, v_3=5\)。VCG 价格？ 解答：\(x^*\): 分配给玩家1（价值最高）。 \(t_1\): \(\sum_{j\neq 1} v_j(x^*) - \sum_{j\neq 1} v_j(x^*_{-1})\) \(= (0+0) - (7+0) = -7\)。玩家1 支付 7（第二高估值）。 这正是二价拍卖！VCG 一般化了 Vickrey 拍卖。 易错点：VCG 价格不是自己的报价，而是自己参与对他人造成的外部性。

03Vickrey 拍卖与收益等价定理

🎯 为什么研究这个问题

拍卖是机制设计最自然的应用场景。英式/荷式/密封一价/二价拍卖 各有优劣。惊人的收益等价定理表明：在标准假设下，这些拍卖 格式给卖家带来相同的期望收入。

🔍 抽象化方法

拍卖设计 = 机制设计的特例，\(X\) = {分配 + 支付}。 收益等价定理将不同拍卖格式统一起来。

📐 公式推导 / 模型建立

四种标准拍卖： - 英式（English）：公开加价，最后竞价者胜 - 荷式（Dutch）：从高价开始降价，第一个叫停的胜 - 密封一价：最高报价者以自己报价支付 - Vickrey（密封二价）：最高报价者以第二高报价支付 收益等价定理：若各竞价者独立私有估值，风险中性，对称，则 \[E[\text{Revenue}_{\text{English}}] = E[\text{Revenue}_{\text{Dutch}}] = E[\text{Revenue}_{\text{First-price}}] = E[\text{Revenue}_{\text{Vickrey}}]\] Vickrey 拍卖中诚实报价是占优策略，这与 KP2 的分析一致。

🌐 实际应用

- 电信频谱拍卖的设计 - eBay 的代理竞价机制 ≈ Vickrey 拍卖

✏️ 例题分析与易错点

例题：2 人拍卖，\(v_i \sim U[0,1]\)。求一价拍卖的均衡竞价策略。 解答：对称均衡中 \(b(v) = \beta v\)。 竞价者最大化 \((v-b) \cdot P(\text{win}) = (v-b) \cdot F(b/\beta)\)。 \(U[0,1]\) 时 \(F(x)=x\)，最大化 \((v-b)(b/\beta)\)。 FOC: \(-b/\beta + (v-b)/\beta = 0\)，\(b = v/2\)。 期望收入 \(= E[v_{(2)}] = 1/3\)（与 Vickrey 等价！）。 易错点：一价拍卖中竞价 \(\neq\) 估值，而 Vickrey 中 \(b = v\)。

04机制设计的现代应用

🎯 为什么研究这个问题

机制设计不仅是理论，它深刻影响了现代经济：从联邦通信委员会 的频谱拍卖（创造数百亿美元收入）到平台经济的匹配机制， Myerson-Satterthwaite 不可能定理揭示了双边交易的基本限制。

📐 公式推导 / 模型建立

Myerson 最优拍卖：卖家最大化期望收入的拍卖为 \[\max \sum_i E[\phi_i(v_i) x_i(v)]\] 其中 \(\phi_i(v) = v - \frac{1-F(v)}{f(v)}\) 为虚拟估值(virtual valuation)。 分配给虚拟估值最高且非负的竞价者。 Myerson-Satterthwaite 不可能定理： 双边交易（卖家估值 \(v_s\)，买家估值 \(v_b\)，均私有）中， 不存在同时满足激励相容、个体理性、预算平衡和事后有效率的机制。 即「充分高效的双边谈判」不可能。

🌐 实际应用

- 平台经济中的匹配机制（肾脏交换、学校录取） - Roth 的市场设计工作（也获得了诺贝尔奖）

✏️ 例题分析与易错点

例题：解释 Myerson 最优拍卖中为什么要设保留价。 解答：当 \(\phi_i(v_i) < 0\) 时，分配会降低期望收入， 应不分配（设保留价）。对 \(U[0,1]\)，\(\phi(v) = 2v-1\)， 保留价 = \(1/2\)。这解释了现实中拍卖为什么设底价。 易错点：最优拍卖可能不分配物品（当所有人虚拟估值都负时）， 这在社会效率上是有损失的。
