Girsanov 定理

01测度变换与 Radon-Nikodym 导数

🎯 为什么研究这个问题

金融数学的核心思想：换一个概率测度，使得折现股价变成鞅。 这个「测度变换」由 Girsanov 定理实现，它将“有漂移”变为 “无漂移”。

🔍 抽象化方法

绝对连续的测度变换 \(Q \ll P\) 通过 R-N 导数 \(dQ/dP\) 实现。

📐 公式推导 / 模型建立

设 \(P, Q\) 是 \(\Omega\) 上的概率测度，\(Q \ll P\)。 Radon-Nikodym 定理：存在 \(Z = dQ/dP \geq 0\)，\(E_P[Z] = 1\)，使 \[E_Q[X] = E_P[ZX]\] 在滤子 \(\mathcal{F}_t\) 上，\(Z_t = E_P[Z | \mathcal{F}_t]\) 是鞅。 变换后的期望：\(E_Q[X_t | \mathcal{F}_s] = E_P[Z_t X_t / Z_s | \mathcal{F}_s]\)。

🌐 实际应用

风险中性定价：转换到等价鞅测度 \(Q\) 下的期望值。

✏️ 例题分析与易错点

例题：\(P\) 下 \(X \sim N(0,1)\)。\(Z = e^{\theta X - \theta^2/2}\)。 求 \(X\) 在 \(Q\) 下的分布。 解答：\(E_Q[e^{iuX}] = E_P[Ze^{iuX}] = e^{iu\theta - u^2/2}\)。 故 \(X \sim N(\theta, 1)\) under \(Q\)。测度变换改了均值！ 易错点：\(Z\) 必须满足 \(E_P[Z]=1\)，否则 \(Q\) 不是概率测度。

⚡ 快速检测
R-N 导数 \(Z = dQ/dP\) 必须满足

02Girsanov 定理

🎯 为什么研究这个问题

如何改变 BM 的漂移？Girsanov 定理明确给出：用指数鞅作为 R-N 导数，可以把 \(P\) 下的有漂移过程变为 \(Q\) 下的 BM。

🔍 抽象化方法

Girsanov = “消除漂移的工具”。从物理测度转向风险中性测度。

📐 公式推导 / 模型建立

Girsanov 定理：设 \(\theta_t\) 适应，满足 Novikov 条件。定义 \[Z_T = \mathcal{E}_T(\theta) = \exp\!\left(-\int_0^T \theta_t dB_t - \frac{1}{2}\int_0^T \theta_t^2 dt\right)\] 令 \(dQ = Z_T dP\)。则 \[\tilde{B}_t := B_t + \int_0^t \theta_s \, ds\] 在 \(Q\) 下是 Brown 运动。 即：\(P\) 下的 \(B_t + \int \theta ds\) → \(Q\) 下的 BM。 等价地：\(P\) 下 \(B_t\) 有漂移 \(-\theta\)，\(Q\) 下 \(\tilde{B}\) 无漂移。

🌐 实际应用

Black-Scholes 定价的核心步骤：转换到风险中性测度。

✏️ 例题分析与易错点

例题：\(P\) 下 \(X_t = \mu t + \sigma B_t\)。找 \(Q\) 使 \(X_t\) 在 \(Q\) 下是 BM。 解答：\(X_t = \sigma(\frac{\mu}{\sigma} t + B_t) = \sigma(B_t + \frac{\mu}{\sigma}t)\)。 取 \(\theta_t = \mu/\sigma\)。\(\tilde{B}_t = B_t + \frac{\mu}{\sigma}t\)。 则 \(X_t = \sigma \tilde{B}_t\)，\(\tilde{B}\) 在 \(Q\) 下是 BM。 易错点：\(\theta\) 的符号取决于约定。严格按定理陈述写。

03等价鞅测度 (EMM)

🎯 为什么研究这个问题

在无套利市场中，存在一个测度 \(Q\) 使得折现资产价格都是鞅。 这个 \(Q\) 就是等价鞅测度 (EMM)。期权定价 = \(Q\) 下的折现期望。

🔍 抽象化方法

无套利 \(\Leftrightarrow\) EMM 存在（基本定理的第一部分）。 完备市场 \(\Leftrightarrow\) EMM 唯一。

📐 公式推导 / 模型建立

股票模型：\(dS_t = \mu S_t dt + \sigma S_t dB_t\)。无风险利率 \(r\)。 折现股价：\(\tilde{S}_t = e^{-rt} S_t\)。 EMM 要求 \(\tilde{S}\) 是 \(Q\)-鞅。用 Girsanov 取 \[\theta = \frac{\mu - r}{\sigma} \quad (\text{市场风险价格})\] 则 \(\tilde{B}_t = B_t + \theta t\) 在 \(Q\) 下是 BM，且 \[d\tilde{S}_t = \sigma \tilde{S}_t \, d\tilde{B}_t\] 折现股价是 \(Q\)-鞅 ✓

🌐 实际应用

所有衍生品定价的基础：价格 = \(E_Q[e^{-rT} \text{payoff}]\)。

✏️ 例题分析与易错点

例题：\(\mu=0.1, r=0.05, \sigma=0.2\)。求市场风险价格。 解答：\(\theta = (0.1-0.05)/0.2 = 0.25\)。 在 \(Q\) 下，\(S_t = S_0 \exp((r-\sigma^2/2)t + \sigma\tilde{B}_t)\)。 易错点：在 \(Q\) 下，\(S\) 的漂移变为 \(r\)（不是 \(\mu\)）。 “任何资产在 \(Q\) 下漂移都是 \(r\)”。

04Novikov 条件与应用

🎯 为什么研究这个问题

Girsanov 定理需要 \(Z_T\) 是真鞅（而不仅是局部鞅）。Novikov 条件 是最常用的充分条件，保证了测度变换的合法性。

📐 公式推导 / 模型建立

Novikov 条件： \[E_P\!\left[\exp\!\left(\frac{1}{2}\int_0^T \theta_t^2 \, dt\right)\right] < \infty\] \(\Rightarrow \mathcal{E}_T(\theta)\) 是真鞅，\(E_P[\mathcal{E}_T] = 1\)。 简化：若 \(\theta_t\) 有界，\(|\theta_t| \leq M\)，则 \(E[\exp(M^2 T/2)] < \infty\) 自动满足。 警告：若 Novikov 不满足，\(\mathcal{E}(\theta)\) 可能是严格局部鞅， \(E_P[\mathcal{E}_T] < 1\)，测度变换失败。

🌐 实际应用

金融模型中验证 EMM 存在性时检查 Novikov。

✏️ 例题分析与易错点

例题：\(\theta_t = \mu/\sigma\)（常数）。验证 Novikov。 解答：\(E[e^{\mu^2 T/(2\sigma^2)}] = e^{\mu^2 T/(2\sigma^2)} < \infty\) ✓ 因为积分内是确定性常数。 易错点：当 \(\theta_t\) 随机且可能很大时，Novikov 可能不满足。 如 \(\theta_t = B_t^2\) 就会报废。
