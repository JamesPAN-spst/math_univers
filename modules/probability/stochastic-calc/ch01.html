01Brown 运动的路径性质

🎯 为什么研究这个问题

要建立随机积分，首先必须理解積分引元——Brown 运动的路径 性质。\(B_t\) 的路径处处连续但处处不可微，其变差无界，经典 Riemann-Stieltjes 积分完全失效。

🔍 抽象化方法

“处处不可微”迎尖峰”与“二次变差有界”是随机微积分区别于 经典微积分的根本原因。

📐 公式推导 / 模型建立

Brown 运动 \(B_t\) 的关键性质： - 连续路径：\(t \mapsto B_t(\omega)\) a.s. 连续 - 独立增量：\(B_t - B_s \perp B_s - B_r\) (\(r < s < t\)) - 正态增量：\(B_t - B_s \sim N(0, t-s)\) 二次变差：对分割 \(\Pi: 0=t_0 < t_1 < \cdots < t_n = T\)， \[\sum_{i=0}^{n-1}(B_{t_{i+1}} - B_{t_i})^2 \xrightarrow{L^2} T\] 即 \([B, B]_T = T\)。而一次变差 \(\sum|B_{t_{i+1}}-B_{t_i}| \to \infty\) a.s. 因此 \(B\) 不是有界变差函数，R-S 积分不适用。

🌐 实际应用

金融市场中股价的对数收益率常用 Brown 运动建模。

✏️ 例题分析与易错点

例题：计算 \(E\!\left[\sum_{i=0}^{n-1}(B_{t_{i+1}}-B_{t_i})^2\right]\)， 其中 \(t_i = iT/n\)。 解答：\(E[(B_{t_{i+1}}-B_{t_i})^2] = T/n\)。 \(\sum = n \cdot T/n = T\)。而方差 \(= \sum \text{Var}((\Delta B_i)^2) = \sum 2(T/n)^2 = 2T^2/n \to 0\)。故 \(L^2\) 收敛到 \(T\)。 易错点：二次变差 \(= T\) 是确定性的，而不是随机的。 这与一次变差（\(\to \infty\)）形成鲜明对比。

02Riemann-Stieltjes 积分的失效与新理论的必要性

🎯 为什么研究这个问题

我们想计算 \(\int_0^T f(B_t) dB_t\)。但在经典 Riemann-Stieltjes 意义下， 积分元 \(dB_t\) 的变差无界，积分不存在。必须另辟蹊径。

🔍 抽象化方法

“选点方式”决定结果：Itô 选左端点，Stratonovich 选中点， 结果不同——这在经典积分中不会发生。

📐 公式推导 / 模型建立

R-S 积分要求积分元有界变差。但 \(B_t\) 的一次变差 \(= \infty\)，故 \(\int f \, dB_t\) 在 R-S 意义下不存在。 选点依赖性示例：考虑 \(\int_0^T B_t \, dB_t\)。 Itô（左端点）：\(\sum B_{t_i}(B_{t_{i+1}}-B_{t_i}) \to \frac{1}{2}B_T^2 - \frac{1}{2}T\) Stratonovich（中点）：\(\sum \frac{B_{t_i}+B_{t_{i+1}}}{2}(B_{t_{i+1}}-B_{t_i}) \to \frac{1}{2}B_T^2\) 差异 \(= T/2\)，这正是二次变差的一半。 \[\int_0^T B_t \, dB_t = \frac{1}{2}B_T^2 - \frac{1}{2}T \quad \text{(Itô)}\]

🌐 实际应用

Itô 积分用于金融建模（保持鞅性质），Stratonovich 用于物理建模。

✏️ 例题分析与易错点

例题：用 Itô 和想计算 \(\int_0^1 B_t \, dB_t\)。 解答：\(\sum B_{t_i} \Delta B_i = \frac{1}{2}\sum[(B_{t_i}+\Delta B_i)^2 - B_{t_i}^2 - (\Delta B_i)^2]\) \(= \frac{1}{2}[B_1^2 - \sum(\Delta B_i)^2] \to \frac{1}{2}B_1^2 - \frac{1}{2}\)。 易错点：不要把经典积分的 \(\int x \, dx = x^2/2\) 直接套用。 Itô 积分多了 \(-T/2\) 修正项。

03随机积分在金融中的动机

🎯 为什么研究这个问题

积分 \(\int_0^T H_t \, dS_t\) 的金融意义：\(H_t\) 是持仓量，\(dS_t\) 是价格 变动，积分 = 累计收益。这个「随机积分 = 交易收益」的解释 是金融数学的基石。

🔍 抽象化方法

核心抽象是将离散交易的累计损益 \(\sum_{i} H_{t_i}(S_{t_{i+1}}-S_{t_i})\) 推广为连续时间的随机积分 \(\int_0^T H_t\,dS_t\)。这要求被积过程 \(H_t\) 满足两个约束：适应性（不依赖未来信息）和 \(L^2\) 可积性（方差有限）。

自融资组合的财富变化完全由持仓对价格变动的响应决定：\(dV_t = H_t\,dS_t\)，没有外部现金注入。这将组合管理问题转化为随机积分的性质问题。

关键层次链：离散再平衡 → 连续随机积分 → 鞅刻画 → 无套利定价。每一步都依赖 Itô 理论给出严格的极限意义和可积条件。

📐 公式推导 / 模型建立

策略 \(H_t\) 适应于信息流 \(\mathcal{F}_t\)（只能基于历史信息做决策）。 财富过程： \[V_T = V_0 + \int_0^T H_t \, dS_t\] 自融资条件：无额外资金注入， \(V_t = H_t S_t + (V_0 + \int_0^t H_u \, dS_u - H_t S_t)\)。 无套利条件要求折现财富 \(\tilde{V}_t = V_t/B_t\) 是鞅： \[E^Q[\tilde{V}_T | \mathcal{F}_t] = \tilde{V}_t\]

🌐 实际应用

期权定价、对冲策略、风险管理。

✏️ 例题分析与易错点

例题：持有 \(H_t = 1\) 股股票，\(S_t = S_0 e^{\sigma B_t - \sigma^2 t/2}\)。 求累计收益。 解答：收益 = \(\int_0^T dS_t = S_T - S_0\)。 简单地，就是买入持有的利润 = 最终价格 − 初始价格。 易错点：\(H_t\) 必须是适应过程（不抱未来信息）。否则 “策略”不可实施。

04随机积分的粗略构造思路

🎯 为什么研究这个问题

既然 R-S 积分失效，如何定义随机积分？关键思路是：先对简单过程 定义，再用 \(L^2\) 近似推广。类似于建立 Lebesgue 积分的思路。

🔍 抽象化方法

简单过程 → \(L^2\) 极限，与 Lebesgue 积分的 “简单函数 → 极限” 路径完全类似。

📐 公式推导 / 模型建立

第 1 步：对简单过程 \(H_t = \sum_i c_i \mathbf{1}_{(t_i, t_{i+1}]}\)， \[\int_0^T H_t \, dB_t := \sum_i c_i (B_{t_{i+1}} - B_{t_i})\] Itô 等距（isometry）： \[E\!\left[\left(\int_0^T H_t \, dB_t\right)^2\right] = E\!\left[\int_0^T H_t^2 \, dt\right]\] 第 2 步：对一般适应过程 \(H \in \mathcal{L}^2\)，用简单过程 \(H^{(n)} \to H\) 在 \(L^2\) 中近似，定义 \(\int H \, dB := \lim_{n} \int H^{(n)} \, dB\)。 由等距保证极限存在。

🌐 实际应用

这个构造思路是整个随机分析的基础架构。

✏️ 例题分析与易错点

例题：验证 Itô 等距对简单过程 \(H = c \cdot \mathbf{1}_{(0,T]}\)。 解答：\(\int_0^T c \, dB_t = c B_T\)。\(E[c^2 B_T^2] = c^2 T\)。 \(E[\int_0^T c^2 \, dt] = c^2 T\) ✓ 易错点：等距仅对适应过程成立。若 \(H\) “眩未来”（非适应）， Itô 等距失效。
