01Black-Scholes 公式

🎯 为什么研究这个问题

Black-Scholes 公式是金融数学的最著名结果，给出了欧式期权的 无套利价格。它将前几章的所有工具综合运用：GBM + Itô 公式 + Girsanov + 鞅表示。

🔍 抽象化方法

期权价格 = 风险中性测度下的折现期望 \(e^{-rT}E_Q[\text{payoff}]\)。

📐 公式推导 / 模型建立

欧式看涨期权：payoff = \((S_T - K)^+\)。 在 \(Q\) 下 \(S_T = S_0 e^{(r-\sigma^2/2)T + \sigma\tilde{B}_T}\)。 \[C = e^{-rT} E_Q[(S_T - K)^+] = S_0 \Phi(d_1) - Ke^{-rT}\Phi(d_2)\] 其中 \[d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}, \quad d_2 = d_1 - \sigma\sqrt{T}\] 推导要点：\(S_T > K \Leftrightarrow \tilde{B}_T > -d_2 \sigma \sqrt{T}\)。 分拆 \(E_Q[S_T \mathbf{1}_{S_T>K}]\) 使用测度变换得到 \(\Phi(d_1)\)。

🌐 实际应用

期权交易、风险管理、波动率曲面。

✏️ 例题分析与易错点

例题：\(S_0=100, K=105, r=0.05, \sigma=0.2, T=1\)。求看涨期权价格。 解答：\(d_1 = (\ln(100/105)+(0.05+0.02) \times 1)/(0.2) = (-0.0488+0.07)/0.2 = 0.106\)。 \(d_2 = 0.106 - 0.2 = -0.094\)。 \(C = 100\Phi(0.106) - 105e^{-0.05}\Phi(-0.094) \approx 100 \times 0.542 - 99.88 \times 0.463 \approx 8.0\)。 易错点：\(d_1\) 和 \(d_2\) 中的 \(r + \sigma^2/2\) 不能写成 \(r - \sigma^2/2\)。

02Feynman-Kac 公式

🎯 为什么研究这个问题

Feynman-Kac 定理将 PDE 和 SDE 连接起来：某些 PDE 的解可以表示为 随机过程的期望，反之亦然。这是随机分析与 PDE 的桥梁。

🔍 抽象化方法

PDE 解 = 随机期望。这个“SDE-PDE 对偶”是现代数学的深刻联系。

📐 公式推导 / 模型建立

Feynman-Kac：设 \(u(t,x)\) 解 PDE \[\frac{\partial u}{\partial t} + b(x)\frac{\partial u}{\partial x} + \frac{1}{2}\sigma^2(x)\frac{\partial^2 u}{\partial x^2} - r u = 0\] \(u(T,x) = f(x)\)。则 \[u(t,x) = E\!\left[e^{-r(T-t)} f(X_T) \;\big|\; X_t = x\right]\] 其中 \(dX = b(X) dt + \sigma(X) dB\)。 证明要点：对 \(e^{-rt}u(t,X_t)\) 用 Itô 公式， PDE 条件保证 \(dt\) 项为零，故它是鞅。 \(e^{-rt}u(t,x) = E[e^{-rT}f(X_T) | X_t=x]\)。

🌐 实际应用

期权定价（BS PDE 的概率解释）；扩散模型的分数生成。

✏️ 例题分析与易错点

例题：热方程 \(u_t = \frac{1}{2}u_{xx}\)，\(u(0,x) = f(x)\)。 用 Feynman-Kac 写出解。 解答：\(b=0, \sigma=1, r=0\)。\(u(0,x) = E[f(x + B_T)]\)。 即 \(u(0,x) = \int f(x+y) \frac{e^{-y^2/(2T)}}{\sqrt{2\pi T}} dy\)。 这就是热核卷积。 易错点：FK 公式中的 \(b, \sigma\) 必须与 PDE 的系数匹配。

03Score-based 扩散模型初步

🎯 为什么研究这个问题

近年最热门的 AI 应用：扩散生成模型（DDPM/Score SDE）将数据加噪 再去噪，用 SDE 的反转实现生成。这是随机微积分在 AI 中的 最前沿应用。

🔍 抽象化方法

前向过程 = 加噪 SDE，反向过程 = 去噪 SDE。 网络学习的是 score \(\nabla_x \log p_t(x)\)。

📐 公式推导 / 模型建立

前向 SDE：\(dX_t = f(X_t, t) dt + g(t) dB_t\)。 将数据 \(X_0 \sim p_{\text{data}}\) 驱动至噪声 \(X_T \sim N(0, I)\)。 Anderson 反转 SDE： \[dX_t = [f(X_t, t) - g(t)^2 \nabla_x \log p_t(X_t)] dt + g(t) d\bar{B}_t\] 其中 \(\bar{B}_t\) 是反向 BM，\(\nabla_x \log p_t\) 是 score 函数。 训练：用神经网络拟合 score： \[\min_{s_\theta} E_{t, X_0, X_t}\left[\|s_\theta(X_t, t) - \nabla \log p_{t|0}(X_t|X_0)\|^2\right]\]

🌐 实际应用

图像生成（Stable Diffusion）、视频/音频生成、蛋白质结构预测。

✏️ 例题分析与易错点

例题：最简单情形：OU 前向过程 \(dX = -X dt + \sqrt{2} dB\)。 \(p_\infty(x) = N(0,1)\)。Score: \(\nabla \log p_\infty(x) = -x\)。 反向 SDE: \(dX = (-X + 2 \cdot (-X))(-dt) + \sqrt{2} d\bar{B}\) \(= (X + 2X) dt + \sqrt{2} d\bar{B} = 3X dt + \sqrt{2} d\bar{B}\)（简化）。 易错点：反向 SDE 的时间方向相反，符号处理要细心。

04随机最优控制与粗糙路径展望

🎯 为什么研究这个问题

随机微积分的两个现代前沿方向：随机最优控制（HJB 方程）和 粗糙路径理论（处理比 BM 更不规则的路径）。仅简要介绍思想。

🔍 抽象化方法

随机控制 = “在不确定性下做最优决策”。 粗糙路径 = “超越 BM 的”随机积分理论。

📐 公式推导 / 模型建立

随机控制：最小化 \(E[\int_0^T L(X_t, u_t) dt + g(X_T)]\)， 其中 \(dX = b(X, u) dt + \sigma dB\)，\(u_t\) 是控制策略。 HJB 方程： \[-V_t + \sup_u \left[-L(x,u) - b(x,u)V_x - \frac{1}{2}\sigma^2 V_{xx}\right] = 0\] \(V(T,x) = g(x)\)。\(V\) 是值函数。 粗糙路径理论（展望）：对 Hölder 约束的路径 \(\gamma: [0,T] \to \mathbb{R}^d\)，定义 \(p\)-变差，用代数结构建立 随机积分，超越经典 Itô 框架。

🌐 实际应用

HJB 用于金融组合优化（Merton 问题）、机器人控制。 粗糙路径用于非鞅模型、机器学习中的签名方法。

✏️ 例题分析与易错点

例题（展望）：Merton 问题简化：财富 \(dW = (rW + (\mu-r)\pi W) dt + \sigma \pi W dB\)。\(\pi\) 是股票比例。最大化 \(E[\ln W_T]\)。 最优 \(\pi^* = (\mu - r)/\sigma^2\)（Merton 线）。 易错点：这些是高级话题，此处仅作“展望”，不做深入发展。
