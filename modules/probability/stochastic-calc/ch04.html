01SDE 的含义与存在唯一性

🎯 为什么研究这个问题

物理和金融中的随机演化用 SDE 描述：\(dX_t = b(X_t) dt + \sigma(X_t) dB_t\)。 这是 ODE 加上了“噪声”的扩展。解的存在和唯一性是讨论一切 性质的前提。

🔍 抽象化方法

SDE = 随机积分方程。形式上写 \(dX = b \, dt + \sigma \, dB\)， 严格含义是 \(X_t = X_0 + \int_0^t b(X_s) ds + \int_0^t \sigma(X_s) dB_s\)。

📐 公式推导 / 模型建立

SDE 的严格含义： \[X_t = X_0 + \int_0^t b(X_s) \, ds + \int_0^t \sigma(X_s) \, dB_s\] 存在唯一性定理（Itô）：若 \(b, \sigma\) 满足 1. Lipschitz 条件：\(|b(x)-b(y)| + |\sigma(x)-\sigma(y)| \leq K|x-y|\) 2. 线性增长：\(|b(x)|^2 + |\sigma(x)|^2 \leq C(1+|x|^2)\) 则 SDE 有唯一强解。 证明要点：用 Picard 迭代 \(X_t^{(n+1)} = X_0 + \int b(X_s^{(n)}) ds + \int \sigma(X_s^{(n)}) dB_s\)。 由 Lipschitz + Itô 等距 + Gronwall 不等式证明收敛。

🌐 实际应用

股价动态、利率模型、扩散过程、物理系统。

✏️ 例题分析与易错点

例题：验证 \(X_t = X_0 e^{\sigma B_t + (\mu-\sigma^2/2)t}\) 解 \(dX = \mu X dt + \sigma X dB\)。 解答：用 Itô 公式对 \(f(t,B_t) = X_0 e^{\sigma B_t + (\mu-\sigma^2/2)t}\)： \(df = (\mu-\sigma^2/2)f dt + \sigma f dB + \frac{1}{2}\sigma^2 f dt = \mu f dt + \sigma f dB\) ✓ 易错点：\(dX = \mu X dt + \sigma X dB\) 的解不是 \(X_0 e^{\mu t + \sigma B_t}\)。 必须有 \(-\sigma^2/2\) 修正。

02线性 SDE 与 Ornstein-Uhlenbeck 过程

🎯 为什么研究这个问题

线性 SDE 有显式解，是最重要的可解案例。OU 过程是“带均值 回复”的扩散，广泛用于利率模型和物理建模。

📐 公式推导 / 模型建立

线性 SDE：\(dX_t = (a(t) + b(t)X_t) dt + (c(t) + d(t)X_t) dB_t\)。 可用积分因子法求解。 OU 过程：\(dX_t = -\theta X_t \, dt + \sigma \, dB_t\)，\(\theta > 0\)。 积分因子 \(e^{\theta t}\)：\(d(e^{\theta t}X_t) = \sigma e^{\theta t} dB_t\)。 \[X_t = X_0 e^{-\theta t} + \sigma \int_0^t e^{-\theta(t-s)} dB_s\] 性质：\(E[X_t] = X_0 e^{-\theta t} \to 0\)， \(\text{Var}(X_t) = \frac{\sigma^2}{2\theta}(1-e^{-2\theta t}) \to \frac{\sigma^2}{2\theta}\)。 平稳分布：\(X_\infty \sim N(0, \sigma^2/(2\theta))\)。

🌐 实际应用

Vasicek 利率模型；物理中的朗之万方程；机器学习的 SGD 分析。

✏️ 例题分析与易错点

例题：\(dX = -2X dt + 3 dB\)，\(X_0 = 1\)。求 \(E[X_1]\) 和 \(\text{Var}(X_1)\)。 解答：\(\theta=2, \sigma=3\)。\(E[X_1] = e^{-2} \approx 0.135\)。 \(\text{Var}(X_1) = \frac{9}{4}(1-e^{-4}) \approx 2.209\)。 易错点：OU 是正态过程（\(X_t\) 可负），不混淆与 GBM（始终为正）。

03SDE 解的 Markov 性质

🎯 为什么研究这个问题

SDE 的解通常是 Markov 过程，即未来只依赖当前状态。这连接了 SDE（随机分析）和 PDE（偏微分方程）的世界。

🔍 抽象化方法

扩散过程 + Markov 性 → Kolmogorov 方程（PDE）。

📐 公式推导 / 模型建立

设 \(X_t\) 解 \(dX = b(X) dt + \sigma(X) dB\)。 记转移函数 \(u(t,x) = E[f(X_T) | X_t = x]\)。 后向 Kolmogorov 方程： \[\frac{\partial u}{\partial t} + b(x) \frac{\partial u}{\partial x} + \frac{1}{2}\sigma^2(x) \frac{\partial^2 u}{\partial x^2} = 0\] 边界条件：\(u(T, x) = f(x)\)。 前向 Kolmogorov 方程（Fokker-Planck）： \[\frac{\partial p}{\partial t} = -\frac{\partial}{\partial x}(b \cdot p) + \frac{1}{2}\frac{\partial^2}{\partial x^2}(\sigma^2 p)\] 其中 \(p(t,x)\) 是 \(X_t\) 的密度。

🌐 实际应用

期权定价 = 解 PDE；Score-based 扩散模型使用 Fokker-Planck。

✏️ 例题分析与易错点

例题：\(dX = \sigma dB\)（无漂移 BM）。写出 Kolmogorov 方程。 解答：\(b=0\)，\(\sigma=\sigma_0\)。后向：\(u_t + \frac{1}{2}\sigma_0^2 u_{xx} = 0\)。 这就是热方程（时间反向）。解：正态分布的卷积。 易错点：后向 Kolmogorov 是“终值问题”（\(u(T,x)=f(x)\)）， 不是“初值问题”。时间方向是反的。

04数值方法初步

🎯 为什么研究这个问题

大多数 SDE 没有解析解，需要数值模拟。Euler-Maruyama 是最基本的 离散化方法，类似于 ODE 的 Euler 方法。

📐 公式推导 / 模型建立

Euler-Maruyama 方法：\(\Delta t = T/N\)，\(Z_n \sim N(0,1)\) i.i.d.。 \[X_{n+1} = X_n + b(X_n) \Delta t + \sigma(X_n) \sqrt{\Delta t} \, Z_n\] 强收敛阶：\(E[\sup|X_t - X_t^{(N)}|] = O(\sqrt{\Delta t}) = O(N^{-1/2})\)。 弱收敛阶：\(|E[f(X_T)] - E[f(X_T^{(N)})]| = O(\Delta t) = O(N^{-1})\)。 Milstein 方法（强收敛 \(O(\Delta t)\)）： \[X_{n+1} = X_n + b\Delta t + \sigma \sqrt{\Delta t} Z_n + \frac{1}{2}\sigma\sigma'[(\sqrt{\Delta t} Z_n)^2 - \Delta t]\]

🌐 实际应用

Monte Carlo 期权定价、随机模拟、扩散模型的采样。

✏️ 例题分析与易错点

例题：用 EM 方法模拟 GBM：\(dS = 0.1 S dt + 0.2 S dB\)，\(S_0=100\)。 \(\Delta t = 0.01\)。一步：\(S_1 = 100 + 0.1 \times 100 \times 0.01 + 0.2 \times 100 \times 0.1 \times Z_1 = 100.1 + 2Z_1\)。 易错点：\(\sqrt{\Delta t}\) 不能写成 \(\Delta t\)。波动项乘的是 \(\sqrt{\Delta t} \cdot Z\)，不是 \(\Delta t \cdot Z\)。
