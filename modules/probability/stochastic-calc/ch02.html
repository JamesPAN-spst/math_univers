Itô 积分

01Itô 积分的严格定义

🎯 为什么研究这个问题

第 1 章给出了构造思路，现在精确定义 Itô 积分并探讨它的性质。 \(\int_0^T H_t \, dB_t\) 的“左端点”选择使得积分是鞅，这对金融建模 至关重要。

🔍 抽象化方法

Itô 积分 = 某种“随机的 Lebesgue 积分”，定义域是 \(\mathcal{L}^2\) 适应过程。

📐 公式推导 / 模型建立

定义：对 \(H \in \mathcal{L}^2 := \{H \text{ 适应}: E[\int_0^T H_t^2 dt] < \infty\}\)， \[I(H) = \int_0^T H_t \, dB_t\] 由简单过程近似 + Itô 等距定义。 核心性质： 1. 线性：\(I(aH + bG) = aI(H) + bI(G)\) 2. 等距：\(E[I(H)^2] = E[\int_0^T H_t^2 dt]\) 3. 鞅性：\(M_t = \int_0^t H_s \, dB_s\) 是鞅 4. 均值为零：\(E[I(H)] = 0\) 证明鞅性（对简单过程）： \[E[M_t | \mathcal{F}_s] = M_s + E\!\left[\sum_{t_i \geq s} c_i(B_{t_{i+1}\wedge t}-B_{t_i\wedge t})\bigg|\mathcal{F}_s\right] = M_s\] 因为 \(c_i\) 是 \(\mathcal{F}_{t_i}\)-可测的，且 \(B\) 有独立增量。

🌐 实际应用

Itô 积分的鞅性是无套利定价的数学基础。

✏️ 例题分析与易错点

例题：计算 \(E[\int_0^1 t \, dB_t]\) 和 \(E[(\int_0^1 t \, dB_t)^2]\)。 解答：\(E[\int_0^1 t \, dB_t] = 0\)（鞅性）。 \(E[(\int_0^1 t \, dB_t)^2] = \int_0^1 t^2 \, dt = 1/3\)（等距）。 易错点：均值为零不意味着积分等于零，方差一般不为零。

02Itô 积分的计算技巧

🎯 为什么研究这个问题

实际计算 Itô 积分时，不能直接用求原函数的方法（因为链式法则 已失效）。需要新的计算工具，主要是 Itô 公式（下一章）。 本节先练习基本计算。

📐 公式推导 / 模型建立

常用计算： \[\int_0^T dB_t = B_T\] \[\int_0^T B_t \, dB_t = \frac{1}{2}B_T^2 - \frac{1}{2}T\] \[\int_0^T B_t^2 \, dB_t = \frac{1}{3}B_T^3 - \int_0^T B_t \, dt\] （最后一个由 Itô 公式得到） Itô 等距的应用：\(\text{Var}(\int_0^T f(t) dB_t) = \int_0^T f(t)^2 dt\)（确定 \(f\)）。

🌐 实际应用

期权定价中常需计算此类积分。

✏️ 例题分析与易错点

例题：求 \(\text{Var}(\int_0^T e^t \, dB_t)\)。 解答：\(e^t\) 是确定函数（适应）。 \(\text{Var} = E[(\int e^t dB_t)^2] = \int_0^T e^{2t} dt = \frac{1}{2}(e^{2T}-1)\)。 易错点：请区分 \(\int f(t) \, dB_t\)（随机积分）和 \(\int f(t) \, dt\)（普通积分）。前者是随机变量，后者是确定数。

03Itô vs Stratonovich

🎯 为什么研究这个问题

两种随机积分都广泛使用。Itô 在金融中占主导（保持鞅性）， Stratonovich 在物理中常用（保持经典链式法则）。理解两者关系是 必要的。

🔍 抽象化方法

Itô = 左端点（非预知），Stratonovich = 中点（对称）。

📐 公式推导 / 模型建立

Stratonovich 积分定义： \[\int_0^T H_t \circ dB_t := \lim \sum H_{(t_i+t_{i+1})/2}(B_{t_{i+1}}-B_{t_i})\] 转换关系： \[\int_0^T H_t \circ dB_t = \int_0^T H_t \, dB_t + \frac{1}{2}[H, B]_T\] 当 \(H_t = f(B_t)\) 时，\([H,B]_T = \int_0^T f'(B_t) dt\)。故 \[\int f(B_t) \circ dB_t = \int f(B_t) \, dB_t + \frac{1}{2}\int f'(B_t) \, dt\] Stratonovich 保持经典链式法则：\(\int B_t \circ dB_t = \frac{1}{2}B_T^2\)。

🌐 实际应用

物理建模（如朗之万方程的物理解释）适合 Stratonovich。

✏️ 例题分析与易错点

例题：计算 \(\int_0^T B_t^2 \circ dB_t\)。 解答：用转换关系，\(f(x)=x^2\)，\(f'(x)=2x\)。 \(= \int B_t^2 dB_t + \frac{1}{2}\int 2B_t dt = \frac{1}{3}B_T^3 - \int B_t dt + \int B_t dt = \frac{1}{3}B_T^3\)。 即 Stratonovich 意义下，\(\int x^2 dx = x^3/3\) —— 经典公式。 易错点：不要在两种积分之间混淆。理论计算时先确定用哪种。

04随机积分的高级性质

🎯 为什么研究这个问题

Itô 积分不仅有等距和鞅性，还有 Burkholder-Davis-Gundy 不等式、 指数鞅等重要性质，这些是后续定理的关键工具。

🔍 抽象化方法

指数鞅 = “随机的指数函数”，在测度变换和金融模型中无处不在。

📐 公式推导 / 模型建立

BDG 不等式：对连续鞅 \(M\)， \[c_p E\!\left[[M]_T^{p/2}\right] \leq E\!\left[\sup_{t \leq T} |M_t|^p\right] \leq C_p E\!\left[[M]_T^{p/2}\right]\] 指数鞅：对 \(H \in \mathcal{L}^2\)， \[\mathcal{E}_t(H) = \exp\!\left(\int_0^t H_s \, dB_s - \frac{1}{2}\int_0^t H_s^2 \, ds\right)\] \(\mathcal{E}(H)\) 是鞅（当满足 Novikov 条件）。 Novikov 条件：\(E[\exp(\frac{1}{2}\int_0^T H_t^2 dt)] < \infty\) \(\Rightarrow \mathcal{E}(H)\) 是真鞅（而不仅是局部鞅）。

🌐 实际应用

指数鞅是 Girsanov 定理中测度变换的 Radon-Nikodym 导数。

✏️ 例题分析与易错点

例题：证明 \(\mathcal{E}_t := \exp(B_t - t/2)\) 是鞅。 解答：\(H_t = 1\)。Novikov: \(E[e^{T/2}] < \infty\) ✓。 直接验证：\(E[\mathcal{E}_t | \mathcal{F}_s] = \mathcal{E}_s \cdot E[e^{B_t-B_s-(t-s)/2}|\mathcal{F}_s]\) \(= \mathcal{E}_s \cdot e^{(t-s)/2-(t-s)/2} = \mathcal{E}_s\) ✓ 易错点：\(e^{B_t}\) 不是鞅（缺少 \(-t/2\) 修正）。必须有修正项。
