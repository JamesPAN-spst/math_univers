Itô 公式

01一维 Itô 公式

🎯 为什么研究这个问题

经典微积分的链式法则在随机微积分中失效。Itô 公式是它的替代品， 多了一个“二阶修正项”。这是随机微积分中最重要的公式， 没有之一。

🔍 抽象化方法

\(dB_t^2 = dt\) 而不是 0（经典中 \(dx^2 = 0\)）。这一修正产生了 整个随机微积分的差异。

📐 公式推导 / 模型建立

设 \(X_t\) 满足 \(dX_t = \mu_t \, dt + \sigma_t \, dB_t\)。 对 \(f \in C^2\)，Itô 公式： \[df(X_t) = f'(X_t) \, dX_t + \frac{1}{2} f''(X_t) \sigma_t^2 \, dt\] 展开： \[f(X_T) = f(X_0) + \int_0^T f'(X_t)\mu_t \, dt + \int_0^T f'(X_t)\sigma_t \, dB_t + \frac{1}{2}\int_0^T f''(X_t)\sigma_t^2 \, dt\] 证明要点：Taylor 展开到二阶，然后用 \(dB_t dB_t = dt\)， \(dt \cdot dB_t = 0\)，\(dt \cdot dt = 0\)。

🌐 实际应用

Black-Scholes 公式的推导、SDE 的求解、期权对冲。

✏️ 例题分析与易错点

例题：\(X_t = B_t\)，\(f(x) = x^2\)。用 Itô 公式求 \(d(B_t^2)\)。 解答：\(dX = dB_t\)（\(\mu=0, \sigma=1\)）。\(f'=2x, f''=2\)。 \(d(B_t^2) = 2B_t dB_t + \frac{1}{2} \cdot 2 \cdot 1 dt = 2B_t dB_t + dt\)。 积分：\(B_T^2 = 2\int_0^T B_t dB_t + T\)。 即 \(\int_0^T B_t dB_t = \frac{1}{2}(B_T^2 - T)\) ✓ 易错点：忘记二阶项 \(\frac{1}{2}f'' \sigma^2 dt\)。这是最常见错误。

02多维 Itô 公式

🎯 为什么研究这个问题

实际问题往往涉及多个随机过程（如多加股票、多维扩散）。 多维 Itô 公式是高维随机分析的基本工具。

📐 公式推导 / 模型建立

设 \(\mathbf{X}_t = (X_t^1, \ldots, X_t^d)\)，\(f: \mathbb{R}^d \to \mathbb{R}\)，\(f \in C^2\)。 多维 Itô 公式： \[df = \sum_i \frac{\partial f}{\partial x_i} dX_t^i + \frac{1}{2}\sum_{i,j} \frac{\partial^2 f}{\partial x_i \partial x_j} d[X^i, X^j]_t\] 二次变差规则： \(dB_t^i \cdot dB_t^j = \rho_{ij} dt\)，\(dB_t \cdot dt = 0\)，\(dt \cdot dt = 0\)。 乘积公式（特例）： \[d(X_t Y_t) = X_t dY_t + Y_t dX_t + d[X,Y]_t\]

🌐 实际应用

多资产组合的动态、多维扩散模型。

✏️ 例题分析与易错点

例题：\(X_t = e^{B_t^1}\)，\(Y_t = B_t^2\)，\(B^1 \perp B^2\)。求 \(d(X_tY_t)\)。 解答：\(dX_t = X_t dB_t^1 + \frac{1}{2}X_t dt\)（由 Itô 公式）。\(dY_t = dB_t^2\)。 \(d[X,Y] = 0\)（因为 \(dB^1 \cdot dB^2 = 0\)）。 \(d(XY) = X dY + Y dX = e^{B^1} dB^2 + B^2 e^{B^1}(dB^1 + dt/2)\)。 易错点：\(d[X,Y]\) 只在 \(X, Y\) 的波动部分相关时非零。

⚡ 快速检测
\(d(X_tY_t)\) 相比经典乘法则 \(XdY + YdX\) 多了

03几何 Brown 运动 (GBM)

🎯 为什么研究这个问题

股价不能为负，所以要用乘性模型。GBM 是金融建模的默认模型， 也是 Itô 公式的最重要应用。

🔍 抽象化方法

GBM = 「指数转换后的 BM」，对数收益率是正态的。

📐 公式推导 / 模型建立

SDE：\(dS_t = \mu S_t \, dt + \sigma S_t \, dB_t\)。 求解：令 \(f(x) = \ln x\)，\(f' = 1/x\)，\(f'' = -1/x^2\)。 Itô 公式： \[d(\ln S_t) = \frac{1}{S_t} dS_t + \frac{1}{2} \cdot \left(-\frac{1}{S_t^2}\right) \sigma^2 S_t^2 \, dt = (\mu - \frac{\sigma^2}{2}) dt + \sigma \, dB_t\] 积分：\(\ln S_T - \ln S_0 = (\mu - \sigma^2/2)T + \sigma B_T\)。 \[S_T = S_0 \exp\!\left((\mu - \frac{\sigma^2}{2})T + \sigma B_T\right)\] 即 \(\ln(S_T/S_0) \sim N((\mu - \sigma^2/2)T, \sigma^2 T)\)。

🌐 实际应用

Black-Scholes 模型、期权定价、Monte Carlo 模拟。

✏️ 例题分析与易错点

例题：\(S_0 = 100\)，\(\mu = 0.1\)，\(\sigma = 0.2\)，\(T = 1\)。 求 \(E[S_1]\) 和 \(P(S_1 > 120)\)。 解答：\(E[S_1] = S_0 e^{\mu T} = 100 e^{0.1} \approx 110.5\)。 \(\ln(S_1/100) \sim N(0.08, 0.04)\)。\(P(S_1 > 120) = P(Z > (\ln 1.2 - 0.08)/0.2)\) \(= P(Z > (0.182 - 0.08)/0.2) = P(Z > 0.51) \approx 0.305\)。 易错点：\(E[S_T] = S_0 e^{\mu T}\)（不是 \(S_0 e^{(\mu-\sigma^2/2)T}\)）。 期望和中位数不同！

04Itô 公式与 Lévy 刻画

🎯 为什么研究这个问题

Itô 过程的一般形式 \(dX = \mu dt + \sigma dB\) 的性质可以从 Lévy 刻画的角度理解：任何连续鞅可分解为 BM 变换 + 漂移。

🔍 抽象化方法

Lévy 刻画：连续局部鞅 \(\Leftrightarrow\) 时间变换的 BM。

📐 公式推导 / 模型建立

Itô 过程（一般形式）： \[X_t = X_0 + \int_0^t \mu_s \, ds + \int_0^t \sigma_s \, dB_s\] 其中 \(\mu, \sigma\) 适应。\(X\) 的二次变差 \([X]_t = \int_0^t \sigma_s^2 \, ds\)。 Lévy 刻画：若 \(M_t\) 是连续局部鞅，\([M]_t = \langle M\rangle_t\) a.s. 连续递增，则存在 BM \(W\) 使得 \[M_t = W_{\langle M \rangle_t}\] 这说明任何连续鞅只是“变速播放”的 BM。

🌐 实际应用

证明随机过程的分布性质（如证明 GBM 是对数正态）。

✏️ 例题分析与易错点

例题：\(M_t = \int_0^t 2 \, dB_s = 2B_t\)。用 Lévy 刻画描述。 解答：\(\langle M \rangle_t = 4t\)。故 \(M_t = W_{4t}\)，其中 \(W\) 是 BM。 验证：\(M_t \sim N(0, 4t)\)，\(W_{4t} \sim N(0, 4t)\) ✓ 易错点：Lévy 刻画仅对连续局部鞅成立。带跳跃的鞅不适用。

⚡ 快速检测
Lévy 刻画说明连续局部鞅本质上是
