贝叶斯推断

01先验-后验框架

🎯 为什么研究这个问题

贝叶斯统计认为 \(\theta\) 本身是随机的。先验 \(\pi(\theta)\) 表达了 观测数据前的信念。数据 \(x\) 通过似然更新信念得到后验 \(\pi(\theta|x)\)。这个 “先验 + 数据 → 后验” 的流程是贝叶斯推断的核心。

🔍 抽象化方法

贝叶斯公式 = 条件概率的自然推广： \(\pi(\theta|x) \propto\) 先验 \(\times\) 似然。

📐 公式推导 / 模型建立

贝叶斯定理： \[\pi(\theta|x) = \frac{L(\theta|x)\pi(\theta)}{\int L(\theta|x)\pi(\theta)d\theta} \propto L(\theta|x)\pi(\theta)\] 后验 \(\propto\) 似然 \(\times\) 先验。分母只是归一化常数。 当数据量大时，后验趋近于 MLE： \[\pi(\theta|x) \approx N\!\left(\hat{\theta}_{MLE}, \frac{1}{nI(\hat{\theta})}\right)\] 说明先验影响在大样本下消失。

🌐 实际应用

垃圾邮件过滤器（朴素贝叶斯）；推荐系统；医学诊断。

✏️ 例题分析与易错点

例题：\(X|\lambda \sim \text{Poisson}(\lambda)\)，先验 \(\lambda \sim \text{Gamma}(2, 1)\)。 观察 \(x = 5\)。求后验。 解答：\(\pi(\lambda|5) \propto \frac{e^{-\lambda}\lambda^5}{5!} \cdot \lambda e^{-\lambda} = \frac{\lambda^6 e^{-2\lambda}}{5!}\)。 识别为 \(\text{Gamma}(7, 2)\)，即 \(\lambda|x \sim \text{Gamma}(2+5, 1+1)\)。 后验均值 \(= 7/2 = 3.5\)。 易错点：先验的超参数更新规则要根据共轭关系推导， 不能模板化套用。

02共轭先验族

🎯 为什么研究这个问题

后验的计算通常涉及复杂积分。共轭先验使得后验与先验属于 同一分布族，只更新超参数，计算简洁。对指数族，共轭先验总存在。

🔍 抽象化方法

共轭性：先验族 \(\mathcal{F}\)对似然共轭 iff 后验 \(\in \mathcal{F}\)。

📐 公式推导 / 模型建立

定义：先验族 \(\{\pi(\theta|\eta): \eta\}\) 对似然 \(L(\theta|x)\) 共轭 iff \(\forall \eta\)，\(\pi(\theta|x) \in \{\pi(\theta|\eta')\}\)。 常见共轭对： \[\text{Bin}(n,p) \leftrightarrow \text{Beta}(\alpha,\beta) \to \text{Beta}(\alpha+x, \beta+n-x)\] \[\text{Poisson}(\lambda) \leftrightarrow \text{Gamma}(\alpha,\beta) \to \text{Gamma}(\alpha+\sum x_i, \beta+n)\] \[N(\mu, \sigma^2) \leftrightarrow N(\mu_0, \tau^2) \to N\!\left(\frac{\mu_0/\tau^2 + n\bar{x}/\sigma^2}{1/\tau^2+n/\sigma^2}, \frac{1}{1/\tau^2+n/\sigma^2}\right)\]

🌐 实际应用

在线学习（Online Learning）中随新数据到来依次更新超参数。

✏️ 例题分析与易错点

例题：\(X_i|p \sim \text{Bernoulli}(p)\)，\(n=10\)，\(\sum x_i = 3\)。 先验 \(p \sim \text{Beta}(2, 2)\)。求后验。 解答：后验 \(p \sim \text{Beta}(2+3, 2+7) = \text{Beta}(5, 9)\)。 先验均值 = 0.5，后验均值 = 5/14 ≈ 0.357。 MLE = 0.3，后验均值介于先验和 MLE 之间 ✓ 易错点：共轭先验的超参数更新规则必须记住，不是所有族都一样。

03无信息先验与 Jeffreys 先验

🎯 为什么研究这个问题

如果对 \(\theta\) 真的没有先验知识，应该用什么先验？扁平先验 \(\pi(\theta) \propto 1\) 简单但不对参数变换不变（如 \(\theta\) 与 \(\phi = \ln\theta\) 得到不同结果）。Jeffreys 先验解决了这个问题。

🔍 抽象化方法

Jeffreys 先验用 Fisher 信息提供的自然度量来定义 \(\theta\) 空间上的 “均匀分布”。

📐 公式推导 / 模型建立

Jeffreys 先验： \[\pi_J(\theta) \propto \sqrt{I(\theta)}\] 其中 \(I(\theta)\) 是 Fisher 信息。 参数变换不变性：若 \(\phi = g(\theta)\)，则 \(\pi_J(\phi) \propto \sqrt{I_\phi(\phi)}\)。即用 \(\theta\) 计算还是用 \(\phi\) 计算，得到的后验一致。 例：Bernoulli(\(p\))。\(I(p) = 1/(p(1-p))\)。 \(\pi_J(p) \propto p^{-1/2}(1-p)^{-1/2} = \text{Beta}(1/2, 1/2)\)。

🌐 实际应用

在“无先验知识”的客观贝叶斯分析中采用，如司法统计。

✏️ 例题分析与易错点

例题：\(X_i \sim N(\mu, \sigma^2)\)，\(\sigma\) 已知。Jeffreys 先验 for \(\mu\)？ 解答：\(I(\mu) = 1/\sigma^2\) = 常数。\(\pi_J(\mu) \propto 1\)。 此时 Jeffreys = 扁平先验。 易错点：Jeffreys 先验常是不正常的（improper, \(\int \pi = \infty\)）。 必须确保后验是正常的所谓「后验正常性」。

04贝叶斯因子与模型比较

🎯 为什么研究这个问题

模型选择是统计的核心问题：用线性还是二次回归？贝叶斯因子 提供了模型比较的自然方法，内置了对模型复杂度的惩罚（Occam 剃刀）。

🔍 抽象化方法

贝叶斯因子 = “边缘似然”的比，综合了参数空间上的平均拟合度。

📐 公式推导 / 模型建立

边缘似然： \[m(x|M_k) = \int L(\theta|x, M_k) \pi(\theta|M_k) d\theta\] 贝叶斯因子： \[BF_{12} = \frac{m(x|M_1)}{m(x|M_2)}\] 后验模型概率： \[\frac{P(M_1|x)}{P(M_2|x)} = BF_{12} \cdot \frac{P(M_1)}{P(M_2)}\] 后验比 = 贝叶斯因子 \(\times\) 先验比。 Occam 效应：\(m(x) = \int L(\theta)\pi(\theta)d\theta\)。复杂模型的 \(\pi(\theta)\) 分散在大参数空间上，自动被惩罚。

🌐 实际应用

变量选择、模型平均、贝叶斯网络中的结构学习。

✏️ 例题分析与易错点

例题：\(M_1: X \sim N(0,1)\) vs \(M_2: X \sim N(0, 10)\)。观察 \(x = 0.5\)。 \(BF_{12} = ?\) 解答：\(m(x|M_1) = \phi(0.5) = 0.3521\)， \(m(x|M_2) = \phi(0.5/\sqrt{10})/\sqrt{10} = 0.3970/3.162 = 0.1256\)。 \(BF_{12} = 0.3521/0.1256 \approx 2.8\)。微弱支持 \(M_1\)。 易错点：贝叶斯因子对先验选择很敏感（尤其是不正常先验）。 这就是所谓的「Jeffreys-Lindley 悖论」。
