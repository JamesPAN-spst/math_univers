01置信区间的基本概念

🎯 为什么研究这个问题

点估计给出一个值，但不告诉我们不确定性有多大。置信区间 \([L, U]\) 同时提供了估计值和不确定度量：「We are 95% confident that \(\theta\) lies in \([L, U]\)」。

🔍 抽象化方法

置信区间是对偶于假设检验的集合估计方法：\(\theta\) 的 \(1-\alpha\) 置信区间恰好是所有不被水平 \(\alpha\) 检验拒绝的参数值集合。从覆盖概率的角度看，置信区间是一个随机集合，其频率性质（重复抽样覆盖率 \(\geq 1-\alpha\)）与贝叶斯可信区间（后验概率 \(\geq 1-\alpha\)）在概念上根本不同，但在大样本下往往数值近似。

📐 公式推导 / 模型建立

\(1-\alpha\) 置信区间：随机区间 \([L(X), U(X)]\) 满足 \[P_\theta(L(X) \leq \theta \leq U(X)) \geq 1-\alpha, \quad \forall \theta\] 重要区分：\(\theta\) 是固定的，\([L, U]\) 是随机的。 不是「\(\theta\) 以 95% 的概率落在区间中」， 而是「此构造方法在重复抽样中有 95% 的时间能覆盖真值」。 不同 \(\alpha\) 分配： 等尾 \(\alpha/2 + \alpha/2\) 或单侧 \(\alpha + 0\)。

🌐 实际应用

民调报告中“误差范围 \(\pm 3\%\)”就是置信区间半宽。

✏️ 例题分析与易错点

例题：\(X_i \sim N(\mu, 4)\) i.i.d.，\(n = 25\)，\(\bar{x} = 12.3\)。 求 95% 置信区间。 解答：\(\sigma\) 已知 = 2。\(\bar{X} \pm z_{0.025} \cdot \sigma/\sqrt{n} = 12.3 \pm 1.96 \times 2/5 = 12.3 \pm 0.784\)。 CI: \([11.516, 13.084]\)。 易错点：“\(\mu\) 以 95% 的概率在此区间中”是错误的频率解释。 数据收集后，\([11.516, 13.084]\) 是确定的，\(\mu\) 要么在内要么不在。

02枢轴量方法

🎯 为什么研究这个问题

枢轴量是构造置信区间的系统方法：找一个包含 \(\theta\) 和 \(X\) 但 分布不依赖 \(\theta\) 的量 \(Q(X, \theta)\)，然后反解得到区间。

🔍 抽象化方法

枢轴量 = “数据与参数的已知函数关系”，类似于英语中的 “pivot”（支点）。

📐 公式推导 / 模型建立

枢轴量 \(Q(X, \theta)\) 满足：\(Q\) 的分布不依赖于任何未知参数。 步骤： 1. 找枢轴量 \(Q(X, \theta)\) 2. 找 \(a, b\) 使 \(P(a \leq Q \leq b) = 1-\alpha\) 3. 反解 \(a \leq Q(X,\theta) \leq b\) 得 \(L \leq \theta \leq U\) 典型例子：\(\sigma\) 已知时， \[Q = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)\] \(\sigma\) 未知时， \[Q = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t_{n-1}\]

🌐 实际应用

\(t\)-置信区间是最常用的小样本区间估计方法。

✏️ 例题分析与易错点

例题：\(X_i \sim N(\mu, \sigma^2)\) i.i.d.，\(\sigma\) 未知，\(n = 16\)， \(\bar{x} = 5.2\)，\(s = 1.5\)。求均值的 95% CI。 解答：枢轴量 \(T = (\bar{X}-\mu)/(S/\sqrt{n}) \sim t_{15}\)。 \(t_{0.025,15} = 2.131\)。CI: \(5.2 \pm 2.131 \times 1.5/4 = 5.2 \pm 0.799\)。 即 \([4.401, 5.999]\)。 易错点：当 \(\sigma\) 未知时必须用 \(t\) 分布而不是 \(Z\) 分布。 \(n\) 很大时两者近似，但小样本时差异显著。

03Bootstrap 置信区间

🎯 为什么研究这个问题

当没有解析的枢轴量时怎么办？Bootstrap 用重抽样模拟估计量的 抽样分布，从而构造置信区间。它对分布的假设最少，是「计算机 时代」的产物。

🔍 抽象化方法

核心思想：用经验分布 \(\hat{F}_n\) 代替真实分布 \(F\)，通过重抽样 估计统计量的波动性。

📐 公式推导 / 模型建立

算法步骤： 1. 从数据 \((x_1,\ldots,x_n)\) 有放回抽取 \(n\) 个得 \(x^*\) 2. 计算 \(\hat{\theta}^* = T(x^*)\) 3. 重复 \(B\) 次，得 \(\hat{\theta}^*_1, \ldots, \hat{\theta}^*_B\) 百分位数法： \[CI = [\hat{\theta}^*_{(\alpha/2)}, \hat{\theta}^*_{(1-\alpha/2)}]\] 即取 Bootstrap 分布的 \(\alpha/2\) 和 \(1-\alpha/2\) 分位数。 BCa 方法（偏差校正加速）更精确， 考虑了偏差和偏斜。

🌐 实际应用

在没有解析公式的复杂统计量（如中位数、相关系数、 机器学习模型精度）的不确定性量化。

✏️ 例题分析与易错点

例题：数据 \((1.2, 3.5, 2.1, 4.7, 3.3)\)。用 Bootstrap 估计中位数的 CI。 解答：重抽样 \(B=1000\) 次，每次有放回抽 5 个，计算中位数。 得 1000 个 Bootstrap 中位数，取 2.5% 和 97.5% 分位数作为 CI。 例如得到 CI = \([2.1, 4.7]\)。 易错点：Bootstrap 要求原始样本能合理代表总体。若原始 样本太小或有强偏，Bootstrap 也会失效。

04贝叶斯可信区间

🎯 为什么研究这个问题

频率派置信区间的解释“绕口”。贝叶斯可信区间直接回答 “给定数据后，\(\theta\) 以 95% 的概率在哪个范围”——这是大多数 人直觉想要的答案。

🔍 抽象化方法

贝叶斯可信区间是后验分布 \(\pi(\theta|x)\) 的分位数（或 HPD 区域）。

📐 公式推导 / 模型建立

后验分布：\(\pi(\theta|x) \propto L(\theta|x)\pi(\theta)\)。 等尾可信区间：\([\theta_{\alpha/2}, \theta_{1-\alpha/2}]\)， 其中 \(\theta_q\) 是 \(\pi(\theta|x)\) 的 \(q\) 分位数。 HPD (Highest Posterior Density) 区间： \[C = \{\theta: \pi(\theta|x) \geq c\}\] 其中 \(c\) 使 \(P(\theta \in C|x) = 1-\alpha\)。 HPD 是包含后验概率 \(1-\alpha\) 的最短区间。

🌐 实际应用

贝叶斯 A/B 测试中把“方案 B 优于方案 A 的概率”直接给出。

✏️ 例题分析与易错点

例题：\(X|p \sim \text{Bin}(10, p)\)，先验 \(p \sim \text{Beta}(1,1)\)。 观察 \(x=7\)。求 95% 可信区间。 解答：后验 \(p|x \sim \text{Beta}(8, 4)\)。 Beta(8,4) 的 2.5% 分位数 ≈ 0.395，97.5% 分位数 ≈ 0.894。 95% 可信区间：\([0.395, 0.894]\)。 易错点：可信区间和置信区间概念不同。可信区间是贝叶斯概念， 依赖先验选择；置信区间是频率派概念。
