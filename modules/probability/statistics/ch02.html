01估计量的优良性

🎯 为什么研究这个问题

有了统计模型后，如何从数据中估计 \(\theta\)？估计量有好坏之分： 无偏性表示「平均而言不偏离」，一致性表示「样本越多越准」， 有效性表示「方差最小」。MSE 将偏差和方差统一衡量。

🔍 抽象化方法

将估计量视为样本空间到参数空间的映射，其优良性由分布层面的泛函性质（期望、方差）刻画。无偏性是一阶矩条件，有效性是二阶矩最优，一致性则是序列收敛。MSE 作为损失函数将偏差与方差统一到同一尺度下，揭示了精度与复杂度之间的根本张力——偏差-方差权衡。

📐 公式推导 / 模型建立

无偏性：\(E_\theta[\hat{\theta}] = \theta\)，\(\forall \theta \in \Theta\)。 一致性：\(\hat{\theta}_n \xrightarrow{P} \theta\)。 有效性：在无偏估计量中方差最小。 均方误差(MSE)： \[\text{MSE}(\hat{\theta}) = E[(\hat{\theta}-\theta)^2] = \text{Var}(\hat{\theta}) + (\text{Bias})^2\] 其中 \(\text{Bias} = E[\hat{\theta}] - \theta\)。 偏差-方差权衡：有时稍许偏差但方差更小的估计量 总 MSE 更低（如 James-Stein 估计量）。

🌐 实际应用

模型评价中，MSE = 偏差² + 方差 就是「偏差-方差权衡」 的数学表达，在机器学习中指导模型复杂度的选择。

✏️ 例题分析与易错点

例题：\(X_i \sim N(\mu, \sigma^2)\) i.i.d.。 样本方差 \(S^2 = \frac{1}{n-1}\sum(X_i-\bar{X})^2\)。证明 \(S^2\) 无偏。 解答：\(E[\sum(X_i-\bar{X})^2] = E[\sum X_i^2 - n\bar{X}^2] = n(\sigma^2+\mu^2) - n(\sigma^2/n + \mu^2) = (n-1)\sigma^2\)。 故 \(E[S^2] = \sigma^2\) ✓ 易错点：用 \(\frac{1}{n}\sum(X_i-\bar{X})^2\) 会得到有偏估计（偏差 \(-\sigma^2/n\)）。

⚡ 快速检测
\(\text{MSE}(\hat{\theta}) = 0\) 意味着

02极大似然估计(MLE)

🎯 为什么研究这个问题

MLE 是最广泛使用的参数估计方法：找使数据出现概率最大的 \(\theta\)。 它具有渐近有效性、渐近正态性等优良性质，且对参数变换 不变（不变性）。

🔍 抽象化方法

MLE = Fisher 信息几何下的「最佳方向」。它在大样本下 达到 Cramér-Rao 下界，是渐近最精确的估计量。

📐 公式推导 / 模型建立

MLE 定义： \[\hat{\theta}_{MLE} = \arg\max_\theta L(\theta|x) = \arg\max_\theta \ell(\theta)\] 当内部解时，解似然方程：\(\ell'(\theta) = 0\)。 渐近性质（在正则条件下）： \[\hat{\theta}_{MLE} \xrightarrow{d} N\!\left(\theta_0, \frac{1}{nI(\theta_0)}\right)\] 其中 \(I(\theta)\) 是 Fisher 信息。 不变性：若 \(\hat{\theta}\) 是 \(\theta\) 的 MLE，则 \(g(\hat{\theta})\) 是 \(g(\theta)\) 的 MLE。

🌐 实际应用

Logistic 回归、混合模型、神经网络的训练都是 MLE 或其变体。

✏️ 例题分析与易错点

例题：\(X_i \sim \text{Exp}(\lambda)\) i.i.d.。求 \(\hat{\lambda}_{MLE}\)。 解答：\(\ell(\lambda) = n\ln\lambda - \lambda\sum x_i\)。 \(\ell'(\lambda) = n/\lambda - \sum x_i = 0\)。\(\hat{\lambda} = n/\sum x_i = 1/\bar{x}\)。 易错点：MLE 不一定无偏。如 \(\sigma^2\) 的 MLE 是 \(\frac{1}{n}\sum(X_i-\bar{X})^2\)（偏差 \(-\sigma^2/n\)）。

03矩估计与 MAP 估计

🎯 为什么研究这个问题

当似然方程难解时，矩估计提供了简单的替代方案：用样本矩 估计总体矩。而从贝叶斯视角，MAP 估计在 MLE 基础上融入了 先验信息，对应于正则化。

🔍 抽象化方法

矩估计的核心思想是大数律——样本矩依概率收敛到总体矩，从而通过矩方程反解参数。MAP 估计则将估计问题嵌入贝叶斯框架：后验 \(\propto\) 似然 \(\times\) 先验，取众数即得 MAP。从优化角度看，MLE 是无约束极值，MAP 是带正则项的极值，矩估计是矩匹配方程的解——三种方法分别对应不同的信息利用策略。

📐 公式推导 / 模型建立

矩估计：设 \(E[g_k(X)] = h_k(\theta)\)（\(k=1,\ldots,d\)）。 解 \(\frac{1}{n}\sum g_k(X_i) = h_k(\hat{\theta})\) 得 \(\hat{\theta}\)。 MAP 估计：给定先验 \(\pi(\theta)\)， \[\hat{\theta}_{MAP} = \arg\max_\theta [\ell(\theta) + \ln\pi(\theta)]\] MAP = MLE + 正则化项。 关系：当先验均匀（\(\pi \equiv\) 常数）时，MAP = MLE。 实践中，L2 正则 \(\Leftrightarrow\) Gauss 先验，L1 正则 \(\Leftrightarrow\) Laplace 先验。

🌐 实际应用

Ridge 回归 = MAP + Gauss 先验；LASSO = MAP + Laplace 先验。

✏️ 例题分析与易错点

例题：\(X_i \sim N(\mu, 1)\) i.i.d.，先验 \(\mu \sim N(0, \tau^2)\)。 求 \(\hat{\mu}_{MAP}\)。 解答：\(\hat{\mu}_{MAP} = \arg\max[-\frac{n}{2}(\bar{x}-\mu)^2 - \frac{\mu^2}{2\tau^2}]\)。 求导 = 0：\(n(\bar{x}-\mu) - \mu/\tau^2 = 0\)。 \(\hat{\mu}_{MAP} = \frac{n\tau^2}{1 + n\tau^2} \bar{x}\)。当 \(\tau \to \infty\) 时趋近 \(\bar{x}\) (MLE)。 易错点：混淆 MAP 和后验均值。MAP 是后验的众数， 后验均值是贝叶斯最优估计（最小化 MSE）。

04Fisher 信息与 Cramér-Rao 下界

🎯 为什么研究这个问题

估计量的方差能够低到什么程度？Cramér-Rao 下界给出了 一个其不可逾越的下限。Fisher 信息量化了“数据关于 \(\theta\) 包含多少信息”。

🔍 抽象化方法

Fisher 信息是统计流形上的度量（联系信息几何模块）。

📐 公式推导 / 模型建立

Fisher 信息： \[I(\theta) = E\!\left[\left(\frac{\partial}{\partial\theta}\ln f(X|\theta)\right)^2\right] = -E\!\left[\frac{\partial^2}{\partial\theta^2}\ln f(X|\theta)\right]\] i.i.d. 样本：\(I_n(\theta) = nI(\theta)\)。 Cramér-Rao 下界：对任意无偏估计量 \(\hat{\theta}\)， \[\text{Var}(\hat{\theta}) \geq \frac{1}{nI(\theta)}\] 证明要点：用 Cauchy-Schwarz， \(1 = \text{Cov}(\hat{\theta}, \ell')^2 \leq \text{Var}(\hat{\theta}) \cdot \text{Var}(\ell')\)， 而 \(\text{Var}(\ell') = nI(\theta)\)。

🌐 实际应用

要达到某精度的估计至少需多少样本？\(n \geq 1/(\varepsilon^2 I(\theta))\)。

✏️ 例题分析与易错点

例题：\(X_i \sim \text{Bernoulli}(p)\)。求 \(I(p)\) 和 CR 下界。 解答：\(\ln f = x\ln p + (1-x)\ln(1-p)\)。 \(I(p) = E[(X/p - (1-X)/(1-p))^2] = 1/(p(1-p))\)。 CR 下界：\(\text{Var}(\hat{p}) \geq p(1-p)/n\)。 而 \(\hat{p} = \bar{X}\) 的方差 = \(p(1-p)/n\)，达到了下界！ 易错点：CR 下界仅对无偏估计量成立。有偏时需修正。
