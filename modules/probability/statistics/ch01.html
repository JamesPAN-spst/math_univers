01统计模型与参数化

🎯 为什么研究这个问题

统计推断的第一步是确定数据的数学模型。统计模型 \(\{P_\theta : \theta \in \Theta\}\) 是一族甙选分布，随机样本的真实分布被假定属于这个族。 统计学的任务就是从数据中推断未知参数 \(\theta\)。

🔍 抽象化方法

\(\theta\) 编码了我们不知道的东西；\(\Theta\) 编码了我们认为可能的 范围。统计量是 \(T(X)\)：仅依赖于数据的函数（不含 \(\theta\)）。

📐 公式推导 / 模型建立

统计模型：\(\{P_\theta : \theta \in \Theta\}\)，\(\Theta \subseteq \mathbb{R}^d\)。 观测数据 \(X = (X_1, \ldots, X_n) \sim P_\theta\)。 似然函数：对连续模型， \[L(\theta|x) = \prod_{i=1}^{n} f(x_i|\theta)\] 对数似然：\(\ell(\theta) = \sum_{i=1}^{n} \ln f(x_i|\theta)\)。 统计量 \(T(X)\) 是数据的可测函数，不依赖于 \(\theta\)。 例：\(\bar{X}\)，\(S^2\)，\(\max X_i\)。

🌐 实际应用

临床试验中设定统计模型（如生存分析用 Cox 模型）； 机器学习中的模型选择（类别的参数表达能力）。

✏️ 例题分析与易错点

例题：\(X_1, \ldots, X_n \overset{iid}{\sim} N(\mu, \sigma^2)\)。 写出似然函数。 解答：\(L(\mu,\sigma^2) = (2\pi\sigma^2)^{-n/2} \exp\!\left(-\frac{\sum(x_i-\mu)^2}{2\sigma^2}\right)\)。 \(\ell = -\frac{n}{2}\ln(2\pi\sigma^2) - \frac{\sum(x_i-\mu)^2}{2\sigma^2}\)。 易错点：似然函数是 \(\theta\) 的函数（数据固定），而密度函数是 \(x\) 的函数（\(\theta\) 固定）。两者表达式相同，视角不同。

02充分统计量与因子化定理

🎯 为什么研究这个问题

数据 \(X\) 可能很大，但关于 \(\theta\) 的信息可能可以用较小的统计量 \(T(X)\) 完整捕获。充分统计量的意义是：知道 \(T\) 后，数据 中不再含有关于 \(\theta\) 的额外信息。

🔍 抽象化方法

充分性将数据压缩为「无损摘要」。\(T\) 充分 iff \(P(X|T,\theta) = P(X|T)\) （给定 \(T\) 后，\(X\) 的条件分布不依赖 \(\theta\)）。

📐 公式推导 / 模型建立

充分统计量定义：\(T(X)\) 对 \(\theta\) 充分 iff \(P_\theta(X \in A \;|\; T(X) = t)\) 不依赖于 \(\theta\)。 Fisher-Neyman 因子化定理：\(T\) 充分 iff 存在函数 \(g, h\) 使得 \[f(x|\theta) = g(T(x), \theta) \cdot h(x)\] 即似然分解为 “依赖 \(\theta\)+\(T\) 的部分” × “只依赖 \(x\) 的部分”。 证明要点：条件密度 \(f(x|T=t,\theta) = f(x|\theta)/f_T(t|\theta)\)。 若 \(f(x|\theta) = g(T(x),\theta) h(x)\)，则 \(f_T(t|\theta) = g(t,\theta) c(t)\)， 故 \(f(x|T=t,\theta) = h(x)/c(t)\) 与 \(\theta\) 无关。

🌐 实际应用

数据压缩：充分统计量允许用低维摘要代替原始数据， 在大数据场景下节省存储和计算。

✏️ 例题分析与易错点

例题：\(X_i \sim \text{Poisson}(\lambda)\) i.i.d.。证明 \(T = \sum X_i\) 充分。 解答：\(f(\mathbf{x}|\lambda) = \prod e^{-\lambda}\lambda^{x_i}/x_i! = e^{-n\lambda} \lambda^{\sum x_i} / \prod x_i!\)。 令 \(g(t,\lambda) = e^{-n\lambda}\lambda^t\)，\(h(x) = 1/\prod x_i!\)。由因子化 ✓ 易错点：不是任何统计量都是充分的。\(T = X_1\) 不是 \(\text{Poisson}(\lambda)\) 模型中的充分统计量。

03最小充分统计量

🎯 为什么研究这个问题

\(X\) 本身总是充分的（平凡的充分性）。我们要找的是「最小」 的充分统计量——它用最简洁的方式捕获全部 \(\theta\) 信息。

🔍 抽象化方法

最小充分统计量是充分统计量的函数（即每个充分统计量都能 “算出”它）。

📐 公式推导 / 模型建立

定义：充分统计量 \(T\) 是最小充分的 iff 对每个充分统计量 \(T'\)，\(T\) 是 \(T'\) 的函数。 判定方法：\(T\) 最小充分 iff \[\frac{f(x|\theta)}{f(y|\theta)} \text{ 不依赖 } \theta \;\Leftrightarrow\; T(x) = T(y)\] 即似然比​的值仅通过 \(T\) 取值来编码。

🌐 实际应用

最小充分统计量结合 Rao-Blackwell 定理可以改进估计量的 效率。

✏️ 例题分析与易错点

例题：\(X_i \sim U(0, \theta)\) i.i.d.。找最小充分统计量。 解答：\(f(\mathbf{x}|\theta) = \theta^{-n} \mathbf{1}_{\max x_i \leq \theta} \mathbf{1}_{\min x_i \geq 0}\)。 \(f(\mathbf{x})/f(\mathbf{y}) = \mathbf{1}_{\max x_i \leq \theta}/\mathbf{1}_{\max y_i \leq \theta}\)。 此比不依赖 \(\theta\) iff \(\max x_i = \max y_i\)。故 \(T = X_{(n)}\) 最小充分。 易错点：误以为 \(\bar{X}\) 是充分的。对 \(U(0,\theta)\)，\(\bar{X}\) 不充分。

04指数族

🎯 为什么研究这个问题

统计学中最重要的分布族——指数族——涵盖了正态、指数、Poisson、 二项、Gamma 等几乎所有常见分布。它的统一形式使得充分统计量、 MLE 等都有简洁形式。

🔍 抽象化方法

指数族是满足 \(\ln f = \eta \cdot T - A(\eta) + \ln h\) 的分布族。 它的自然参数 \(\eta\) 与充分统计量 \(T\) 成对偶。

📐 公式推导 / 模型建立

指数族形式： \[f(x|\theta) = h(x) \exp(\eta(\theta) \cdot T(x) - A(\theta))\] \(\eta(\theta)\)：自然参数；\(T(x)\)：充分统计量；\(A(\theta)\)：对数配分函数。 性质：\(A(\theta)\) 是矩生成函数： \[E[T] = A'(\eta), \quad \text{Var}(T) = A''(\eta)\] 常见例子： - \(N(\mu, \sigma^2)\)：\(\eta = (\mu/\sigma^2, -1/(2\sigma^2))\)，\(T = (x, x^2)\) - \(\text{Poisson}(\lambda)\)：\(\eta = \ln\lambda\)，\(T = x\)，\(A = e^\eta\) - \(\text{Exp}(\lambda)\)：\(\eta = -\lambda\)，\(T = x\)，\(A = -\ln(-\eta)\)

🌐 实际应用

广义线性模型(GLM)的基础就是指数族；机器学习中的 softmax 回归也是指数族形式。

✏️ 例题分析与易错点

例题：证明 \(\text{Bernoulli}(p)\) 属于指数族。 解答：\(f(x|p) = p^x(1-p)^{1-x} = (1-p)\exp(x \ln\frac{p}{1-p})\)。 形式：\(h=1\)，\(\eta = \ln(p/(1-p))\)（对数几率），\(T=x\)，\(A = \ln(1+e^\eta)\)。 易错点：指数族的参数化不唯一。\(\eta\) 是自然参数， 与常用参数 \(\theta\) 的关系可能是非线性的。
