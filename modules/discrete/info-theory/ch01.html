熵与信息量

01自信息与 Shannon 熵

🎯 为什么研究这个问题

「信息」是日常概念，但 Shannon 在 1948 年将其精确定量化—— 不确定性越大，信息量越多。熵度量了随机变量的平均不确定性， 它不仅是信息论的基石，也是数据压缩、通信理论、机器学习的 核心概念。理解熵是打开信息论所有结果的钥匙。

🔍 抽象化方法

核心抽象：用对数度量惊讶度。小概率事件携带更多信息， 正如「狗咬人」不是新闻而「人咬狗」才是。

📐 公式推导 / 模型建立

自信息：事件 \(x\) 的信息量 \(I(x) = -\log_2 P(x)\)（单位：bit）。 性质：\(I(x) \geq 0\)；\(P(x)=1 \Rightarrow I(x)=0\)； 独立事件：\(I(xy)=I(x)+I(y)\)。 Shannon 熵： \[H(X) = -\sum_{x \in \mathcal{X}} p(x) \log_2 p(x) = \mathbb{E}[I(X)]\] 性质： (1) \(H(X) \geq 0\)，\(H(X)=0\) iff \(X\) 确定 (2) \(H(X) \leq \log_2|\mathcal{X}|\)，等号当且仅当均匀分布 (3) 二元熵：\(H(p) = -p\log p - (1-p)\log(1-p)\)，\(H(1/2)=1\) bit 最大熵原理：在给定约束下，选择熵最大的分布。 无约束：均匀分布；给定均值+方差：正态分布。

🌐 实际应用

- 数据压缩：熵 = 平均最短编码长度的下界 - 机器学习：信息增益 = 决策树分裂的标准

✏️ 例题分析与易错点

例题：弹琴每节钢琴课有 4 个可能等级 A/B/C/D，概率分别为 \(1/2, 1/4, 1/8, 1/8\)。求熵。 解答：\(H = -\frac{1}{2}\log\frac{1}{2} - \frac{1}{4}\log\frac{1}{4} - 2 \times \frac{1}{8}\log\frac{1}{8}\) \(= \frac{1}{2} + \frac{2}{4} + \frac{6}{8} = \frac{1}{2}+\frac{1}{2}+\frac{3}{4} = \frac{7}{4} = 1.75\) bit。 易错点：\(0 \log 0\) 约定为 0（由极限 \(\lim_{p\to 0} p\log p = 0\)）， 不是未定义。

⚡ 快速检测
抛一枚均匀硬币的熵为多少 bit？

02联合熵、条件熵与链式法则

🎯 为什么研究这个问题

当两个随机变量一起考虑时，它们的不确定性如何叠加？联合熵、 条件熵和链式法则提供了答案。条件熵度量「知道 \(Y\) 后 \(X\) 的剩余不确定性」，链式法则将联合熵分解为边际熵与条件熵 之和——这些工具是分析多变量信息结构的基础。

📐 公式推导 / 模型建立

联合熵： \[H(X,Y) = -\sum_{x,y} p(x,y) \log p(x,y)\] 条件熵： \[H(X|Y) = -\sum_{x,y} p(x,y) \log p(x|y) = \sum_y p(y) H(X|Y=y)\] 链式法则： \[H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)\] 推广：\(H(X_1,\ldots,X_n) = \sum_{i=1}^n H(X_i | X_1,\ldots,X_{i-1})\)。 条件降低熵：\(H(X|Y) \leq H(X)\)，等号当 \(X, Y\) 独立。 「知道更多不会增加不确定性」。 独立时：\(H(X,Y) = H(X)+H(Y)\)。

🌐 实际应用

- 条件熵用于度量特征选择中的信息增益 - 链式法则是数据压缩中预测编码的理论基础

✏️ 例题分析与易错点

例题：\((X,Y)\) 的联合分布为：\(P(0,0)=1/4, P(0,1)=1/4, P(1,0)=1/4, P(1,1)=1/4\)。求 \(H(X), H(Y|X), H(X,Y)\)。 解答：\(X, Y\) 均等分布，\(H(X)=1\)。 \(X,Y\) 独立，\(H(Y|X) = H(Y) = 1\)。 \(H(X,Y) = H(X)+H(Y|X) = 2\) bit。 验证：\(H(X,Y) = -4 \times \frac{1}{4}\log\frac{1}{4} = 2\)。✓ 易错点：\(H(X|Y)\) 是对 \(Y\) 的平均，不是对某个特定 \(y\) 值的熵。

⚡ 快速检测
若 \(X\) 完全决定 \(Y\)（即 \(Y=f(X)\)），则 \(H(Y|X) = ?\)

03KL 散度与 Gibbs 不等式

🎯 为什么研究这个问题

两个分布有多「不同」？KL 散度提供了一种自然的度量，它度量用 分布 \(Q\) 近似 \(P\) 时的额外编码代价。尽管 KL 散度不是真正的 距离（不对称），它在机器学习、统计推断、信息几何中无处不在。 Gibbs 不等式 (\(D_{KL} \geq 0\)) 是信息论最基本的不等式。

🔍 抽象化方法

KL 散度度量「用错误模型编码的额外代价」——它是信息论与 统计学的桥梁。

📐 公式推导 / 模型建立

KL 散度： \[D_{KL}(P \| Q) = \sum_x p(x) \log \frac{p(x)}{q(x)}\] Gibbs 不等式：\(D_{KL}(P \| Q) \geq 0\)，等号当 \(P = Q\)。 证明：由 \(\ln x \leq x - 1\)（等号 \(x=1\)）， \[-D_{KL} = \sum p \ln\frac{q}{p} \leq \sum p\left(\frac{q}{p}-1\right) = \sum q - \sum p = 0\] 性质：\(D_{KL}(P\|Q) \neq D_{KL}(Q\|P)\)（非对称）， 不满足三角不等式，因此不是距离。 与交叉熵的关系： \[D_{KL}(P\|Q) = H(P, Q) - H(P)\] 其中交叉熵 \(H(P,Q) = -\sum p(x) \log q(x)\)。

🌐 实际应用

- 机器学习：最小化交叉熵损失 = 最小化 \(D_{KL}(P_{\text{data}} \| P_\theta)\) - 变分推断：ELBO 中 \(D_{KL}(q(z|x) \| p(z))\) 作为正则项

✏️ 例题分析与易错点

例题：\(P = (1/2, 1/2)\)，\(Q = (1/4, 3/4)\)。求 \(D_{KL}(P\|Q)\)。 解答：\(D_{KL} = \frac{1}{2}\log\frac{1/2}{1/4} + \frac{1}{2}\log\frac{1/2}{3/4} = \frac{1}{2}\log 2 + \frac{1}{2}\log\frac{2}{3} = \frac{1}{2} + \frac{1}{2}(-0.585) = 0.2075\) bit。 易错点：\(D_{KL}(P\|Q) \neq D_{KL}(Q\|P)\)。确认方向： \(D_{KL}(P\|Q)\) 中 \(P\) 在对数内，\(Q\) 在对数外。

04互信息与数据处理不等式

🎯 为什么研究这个问题

\(X\) 和 \(Y\) 之间共享了多少信息？互信息 \(I(X;Y)\) 度量了这一量。 它是对称的、非负的，且等于零当且仅当 \(X, Y\) 独立。 数据处理不等式则说明信息只会在处理过程中丢失，不会被创造。

🔍 抽象化方法

互信息统一了熵、条件熵、联合熵之间的关系， 类似于集合论中的交集 \(|A \cap B| = |A|+|B|-|A\cup B|\)。

📐 公式推导 / 模型建立

互信息： \[I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X)+H(Y)-H(X,Y)\] 等价定义：\(I(X;Y) = D_{KL}(P_{XY} \| P_X P_Y)\)， 度量联合分布与独立分布的偏离程度。 性质：\(I(X;Y) \geq 0\)（由 Gibbs 不等式）， \(I(X;Y) = 0\) iff \(X \perp Y\)。 \(I(X;Y) = I(Y;X)\)（对称）。 数据处理不等式：若 \(X \to Y \to Z\) 构成 Markov 链，则 \[I(X; Z) \leq I(X; Y)\] 「处理数据不会增加信息」。证明：链式法则 + \(I(X; Z | Y) = 0\)（Markov 性）。

🌐 实际应用

- 特征选择：选择 \(I(X_i; Y)\) 最大的特征 - VAE: ELBO = \(\mathbb{E}[\log p(x|z)] - D_{KL}(q(z|x) \| p(z))\)

✏️ 例题分析与易错点

例题：\(X \sim \text{Bernoulli}(1/2)\)，\(Y = X \oplus N\)，\(N \sim \text{Bernoulli}(p)\) 独立于 \(X\)。 求 \(I(X; Y)\)。 解答：\(H(Y) = 1\)（\(Y\) 也是均匀的）。 \(H(Y|X) = H(N) = H(p)\)。 \(I(X;Y) = 1 - H(p)\)。\(p=0\)：\(I=1\)（无噪声），\(p=1/2\)：\(I=0\)（完全噪声）。 易错点：混淆 \(I(X;Y)\) 与 \(H(X|Y)\)。\(I\) 是「知道 \(Y\) 后减少的 不确定性」，\(H(X|Y)\) 是「剩余不确定性」。
