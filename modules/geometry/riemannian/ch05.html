01Rauch 比较定理

🎯 为什么研究这个问题

如果流形的截面曲率有上界或下界，测地线的行为如何与常曲率空间 比较？Rauch 比较定理说：曲率越大，相邻测地线聚焦越快—— Jacobi 场的长度被常曲率空间中的 Jacobi 场所控制。这是 所有比较几何定理的出发点。

🔍 抽象化方法

比较几何的核心思想：用简单模型空间（常曲率）作为参照， 通过曲率界控制几何量（距离、体积、角度等）。

📐 公式推导 / 模型建立

Rauch 比较定理：\(\gamma: [0, L] \to M\) 测地线，\(\tilde{\gamma}: [0, L] \to \tilde{M}\) 对应方向的测地线（\(\tilde{M}\) 是常曲率 \(\kappa\) 空间）。 \(J, \tilde{J}\) 分别是 Jacobi 场，\(J(0) = \tilde{J}(0) = 0\)， \(|J'(0)| = |\tilde{J}'(0)|\)。 若 \(K_M \leq \kappa\)（\(M\) 的截面曲率 \(\leq\) 模型曲率）， 则（在无共轭点范围内）： \[|J(t)| \geq |\tilde{J}(t)|, \quad \forall t \in [0, L]\] （\(M\) 中测地线比模型中发散得更快。） 若 \(K_M \geq \kappa\)：不等号反向，\(|J(t)| \leq |\tilde{J}(t)|\) （\(M\) 中测地线聚焦更快）。 证明要点：比较 \(f(t) = |J(t)|^2\) 和 \(\tilde{f}(t) = |\tilde{J}(t)|^2\) 的 微分不等式（Sturm 比较定理）。

🌐 实际应用

- Rauch 是 Toponogov 和 Bishop-Gromov 比较定理的基础

✏️ 例题分析与易错点

例题：\((M^2, g)\)，\(K \leq 0\)。从 \(p\) 出发的两条测地线初始夹角 \(\alpha\)。 距离 \(p\) 为 \(t\) 处两条测地线的距离 \(d(t)\) 与 \(\mathbb{R}^2\) 相比如何？ 解答：\(\mathbb{R}^2\) 中 \(\tilde{d}(t) \approx t\alpha\)（小 \(t\) 时）。 \(K \leq 0 = \kappa\)，Rauch 比较：\(|J(t)| \geq |\tilde{J}(t)|\)。 所以 \(d(t) \geq t\alpha\)——负曲率空间中测地线发散更快。 易错点：这是"小范围"行为。全局上负曲率空间的测地线可能再次靠近 （如果流形紧致且基本群非平凡）。

⚡ 快速检测
\(K \geq 1\)（截面曲率下界为 1），Jacobi 场与 \(S^n(1)\) 比较？

02Toponogov 比较定理

🎯 为什么研究这个问题

Rauch 比较作用在"无穷小"层面（Jacobi 场），Toponogov 比较定理 则是"全局"版本——它比较的是测地三角形的边长和角度。曲率 下界 \(K \geq \kappa\) 意味着三角形"更胖"（角度更大），曲率上界 意味着三角形"更瘦"。这是整体几何学的强大工具。

🔍 抽象化方法

Toponogov = Rauch 的全局化。它通过比较三角形将曲率信息 转化为距离估计——这是 Alexandrov 几何的起点。

📐 公式推导 / 模型建立

Toponogov 比较定理（三角形版本）： \((M, g)\) 完备，\(K \geq \kappa\)。\(\Delta pqr\) 为 \(M\) 中的测地三角形。 \(\tilde{\Delta}\tilde{p}\tilde{q}\tilde{r}\) 为常曲率 \(\kappa\) 空间中 对应边长相同的三角形。 \[\angle_p \leq \tilde{\angle}_{\tilde{p}}, \quad \angle_q \leq \tilde{\angle}_{\tilde{q}}, \quad \angle_r \leq \tilde{\angle}_{\tilde{r}}\] （\(K \geq \kappa\) 时角度比模型空间小或相等。） 铰链版本：给定两边 \(a, b\) 和夹角 \(\alpha\)。 \(K \geq \kappa\)：\(c \leq \tilde{c}\)（第三边更短或相等）。 \(K \leq \kappa\)：\(c \geq \tilde{c}\)。 推论：\(K \geq 0\) 时，与 \(\mathbb{R}^n\)（\(\kappa = 0\)）比较： 三角形地不等式取"胖"方向。

🌐 实际应用

- Alexandrov 空间（度量空间中的曲率界）用 Toponogov 的不等式作为定义

✏️ 例题分析与易错点

例题：\(S^2(1)\)（\(K = 1\)）上等边测地三角形，边长 \(a = \pi/3\)。 求每个角 \(\alpha\)。 解答：球面余弦律：\(\cos a = \cos^2 a + \sin^2 a \cos \alpha\)， \(\cos(\pi/3) = \cos^2(\pi/3) + \sin^2(\pi/3)\cos\alpha\)， \(1/2 = 1/4 + (3/4)\cos\alpha\)，\(\cos\alpha = 1/3\)， \(\alpha = \arccos(1/3) \approx 70.5°\)。 \(\mathbb{R}^2\) 中等边三角形每角 \(60°\)。\(70.5° > 60°\)（更胖），符合 \(K > 0\)。 易错点：球面余弦律不同于欧式余弦律——别直接用欧式公式。

⚡ 快速检测
\(K \geq \kappa > 0\) 流形中测地三角形内角和与常曲率 \(\kappa\) 空间比较？

03Bishop-Gromov 体积比较

🎯 为什么研究这个问题

Ricci 曲率下界如何影响测地球的体积？Bishop-Gromov 定理说： 如果 \(\mathrm{Ric} \geq (n-1)\kappa\)，则测地球 \(B(p,r)\) 的体积 不超过常曲率 \(\kappa\) 空间中同半径球的体积——且体积比 \(\mathrm{Vol}(B(p,r))/V_\kappa(r)\) 关于 \(r\) 单调递减。这是 Gromov 预紧性定理和几何分析中大量结果的基础。

🔍 抽象化方法

Ricci 曲率 \(\geq (n-1)\kappa\) 控制体积增长率——比较几何从 "线性"（Jacobi 场/距离）推广到"体积"层面。

📐 公式推导 / 模型建立

Bishop 不等式：\(\mathrm{Ric} \geq (n-1)\kappa\)， \[\mathrm{Vol}(B(p,r)) \leq V_\kappa^n(r)\] \(V_\kappa^n(r)\) 是 \(n\) 维常曲率 \(\kappa\) 空间中半径 \(r\) 球的体积。 Gromov 体积比较（单调性）： \[\frac{\mathrm{Vol}(B(p, r))}{V_\kappa^n(r)} \text{ 关于 } r \text{ 单调递减}\] 且 \(\lim_{r \to 0} = 1\)。 推论： - 加倍不等式：\(\mathrm{Vol}(B(p, 2r)) \leq C \cdot \mathrm{Vol}(B(p, r))\)（\(C\) 依赖于 \(n, \kappa\)） - 有限基本群（\(\kappa > 0\)）：\(\mathrm{Vol}(M)\) 有限 \(\Rightarrow\) \(M\) 紧致 \(\kappa = 0\) 特例：\(\mathrm{Ric} \geq 0\)， \(\mathrm{Vol}(B(p,r)) \leq \omega_n r^n\)（\(\omega_n\) = \(\mathbb{R}^n\) 中单位球体积）。

🌐 实际应用

- Gromov 预紧性定理（Ricci 下界 + 直径上界 \(\Rightarrow\) 度量空间的紧致序列有收敛子列）

✏️ 例题分析与易错点

例题：\(S^3(1)\)（\(\mathrm{Ric} = 2\)，即 \(\kappa = 1\)）。 \(V_1^3(r)\) = \(S^3(1)\) 中半径 \(r\) 球的体积。 求 \(\mathrm{Vol}(S^3(1))\) 并验证 Bishop-Gromov。 解答：\(\mathrm{Vol}(S^3(1)) = 2\pi^2\)。 \(V_1^3(\pi) = \mathrm{Vol}(S^3(1)) = 2\pi^2\)（半径 \(\pi\) 覆盖整个球面）。 \(\mathrm{Vol}(B(p, r)) / V_1^3(r) = 1\)（常数，因为 \(S^3\) 本身是模型空间）。 单调递减退化为常数——等号情形。 易错点：Bishop-Gromov 中 \(V_\kappa^n(r)\) 是模型空间的体积，不是欧式体积。

⚡ 快速检测
\(\mathrm{Ric} \geq 0\) 蕴含 \(\mathrm{Vol}(B(p,r))\) 至多以何种速率增长？

04Synge 定理与正曲率拓扑

🎯 为什么研究这个问题

正截面曲率对流形拓扑的约束极为强烈。Synge 定理说：偶数维 紧致正曲率流形是单连通的！这是纯粹的曲率-拓扑联系： 通过第二变分论证闭测地线可以"缩短"来得到矛盾。

🔍 抽象化方法

Synge 的关键观察：正曲率使"平行移动绕一圈"后产生旋转， 这个旋转可以用来"缩短"闭测地线。

📐 公式推导 / 模型建立

Synge 定理：\(M^n\) 紧致定向，\(K > 0\)。 (1) \(n\) 偶数 \(\Rightarrow\) \(M\) 单连通 (2) \(n\) 奇数 \(\Rightarrow\) \(M\) 可定向 证明思路（\(n\) 偶数）：若 \(\pi_1(M) \neq 0\)，取最短非可缩 闭测地线 \(\gamma\)。沿 \(\gamma\) 平行移动一个正交向量场 \(E(t)\)。 \(n\) 偶数 \(\Rightarrow\) \(\gamma\) 正交补空间是奇数维 \(\Rightarrow\) 平行移动后 \(E\) 被保定向的正交变换作用——奇数维正交变换 有不动方向 \(\Rightarrow\) 存在 \(E(0) = E(L)\)（周期性）。 用 \(E\) 做第二变分：\(E''(0) = -\int K(\gamma', E)|\gamma'|^2 dt < 0\) （\(K > 0\)，\(E\) 平行 \(\Rightarrow\) 没有 \(D^2E\) 项）。 \(\gamma\) 不是能量最小——矛盾。 推论：\(S^{2n}\) 上 \(K > 0\) \(\Rightarrow\) 单连通（\(S^{2n}\) 本身当然单连通）。 偶数维紧致正曲率流形 \(\cong S^n\) 或 \(\cong \mathbb{C}P^{n/2}\)（\(1/4\)-pinching 条件下）。

🌐 实际应用

- Synge 排除了 \(K > 0\) 的偶维紧致流形有非平凡基本群的可能性

✏️ 例题分析与易错点

例题：\(\mathbb{C}P^2\) 是 4 维紧致正曲率流形。\(\pi_1(\mathbb{C}P^2) = ?\) 解答：Synge 定理：\(\dim = 4\)（偶数），\(K > 0\)（Fubini-Study 度量， \(K \in [1/4, 1]\)）。 \(\Rightarrow\) \(\mathbb{C}P^2\) 单连通，\(\pi_1 = 0\)。✓ 易错点：\(\mathbb{R}P^3\)（3 维，\(K > 0\)）的 \(\pi_1 = \mathbb{Z}/2\)—— 奇数维不保证单连通。Synge 仅给出"可定向"。

⚡ 快速检测
\(M^4\) 紧致，\(K > 0\)。以下哪个一定成立？
