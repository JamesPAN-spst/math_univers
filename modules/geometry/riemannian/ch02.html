联络与测地线

01Levi-Civita 联络回顾

🎯 为什么研究这个问题
  • 联络是计算的起点：Levi-Civita 联络由无挠 + 度量兼容唯一确定，是黎曼几何中一切计算（测地线、曲率、Jacobi 场）的基础。
  • 平行搬运的规范：联络规定了如何在流形上"平行搬运"向量——不同联络给出不同的搬运方式，Levi-Civita 联络是度量内在决定的最自然选择。
  • 法坐标的威力：在法坐标中 \(\Gamma^k_{ij}(p) = 0\)，极大简化了局部计算。但 \(\partial\Gamma \neq 0\)——这正是曲率的来源。
🔍 抽象化方法
  • 度量决定联络：Levi-Civita 联络是黎曼度量"内在确定"的微分结构。度量不仅量化距离，还唯一规定了平行搬运的方式。
  • Koszul 公式：由无挠和度量兼容两个条件，代数地推导出联络关于度量的显式表达，证明了存在唯一性。
📐 公式推导 / 模型建立

1. Koszul 公式

由无挠性 \(\nabla_X Y - \nabla_Y X = [X,Y]\) 和度量兼容性 \(Xg(Y,Z) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)\) 推出：

\[2g(\nabla_X Y, Z) = Xg(Y,Z) + Yg(X,Z) - Zg(X,Y) + g([X,Y],Z) - g([X,Z],Y) - g([Y,Z],X)\]

2. Christoffel 符号

取 \(X = \partial_i, Y = \partial_j, Z = \partial_l\)（坐标向量场，\([\partial_i, \partial_j] = 0\)），得：

\[\Gamma^k_{ij} = \frac{1}{2}g^{kl}\left(\frac{\partial g_{il}}{\partial x^j} + \frac{\partial g_{jl}}{\partial x^i} - \frac{\partial g_{ij}}{\partial x^l}\right)\]

3. 协变导数

沿曲线 \(\gamma\) 的向量场 \(V\) 的协变导数：\(\frac{DV^k}{dt} = \frac{dV^k}{dt} + \Gamma^k_{ij}\dot{\gamma}^i V^j\)。平行搬运条件：\(DV/dt = 0\)。

4. 法坐标

在 \(p\) 处选标准正交基 \(\{e_i\}\)，通过 \(\exp_p\) 定义法坐标。此坐标中 \(g_{ij}(0) = \delta_{ij}\)，\(\Gamma^k_{ij}(0) = 0\)，但 \(\partial_l\Gamma^k_{ij}(0)\) 与曲率张量直接相关。

🌐 实际应用

广义相对论中的自由落体：在弯曲时空中，自由粒子沿 Levi-Civita 联络的测地线运动。Christoffel 符号 \(\Gamma^\mu_{\alpha\beta}\) 扮演"引力场"的角色——Einstein 等效原理说在法坐标（局部惯性系）中 \(\Gamma = 0\)，即自由落体中感受不到引力。

流形上的机器学习：黎曼 SGD（随机梯度下降）使用 Levi-Civita 联络做平行搬运来更新参数——在双曲空间或 Stiefel 流形上的优化中，梯度需要沿测地线搬运回切空间。

张量方程的化简：法坐标在一点处消去所有 \(\Gamma\)，使复杂的张量恒等式验证变得清晰——例如 Bianchi 恒等式在法坐标中几乎直接可见。

✏️ 例题分析与易错点

例题：双曲平面的 Christoffel 符号

题目：双曲平面 \(\mathbb{H}^2\)：\(ds^2 = (dx^2 + dy^2)/y^2\)（上半平面模型）。求 \(\Gamma^y_{xx}\)。

第一步 — 度量分量：\(g_{xx} = g_{yy} = 1/y^2\)，\(g_{xy} = 0\)，\(g^{yy} = y^2\)。

第二步 — 代入公式：\(\Gamma^y_{xx} = \frac{1}{2}g^{yy}(\partial_x g_{xy} + \partial_x g_{xy} - \partial_y g_{xx}) = \frac{1}{2}y^2(0 + 0 - (-2/y^3)) = \frac{1}{2}y^2 \cdot \frac{2}{y^3} = \frac{1}{y}\)。

结论：\(\Gamma^y_{xx} = 1/y\)。双曲平面的 Christoffel 符号反映了度量随 \(y\) 的变化——接近 \(y = 0\) 边界时"距离膨胀"。

常见错误

  • ❌ 计算 \(\partial_y g_{xx}\) 时误用 \(\partial_y(1/y^2) = -2y\)。
    ✅ 正确结果是 \(\partial_y(y^{-2}) = -2y^{-3} = -2/y^3\)。幂函数求导：\(\partial_y(y^n) = ny^{n-1}\)。
  • ❌ 将 Christoffel 符号当作张量：认为在所有坐标中 \(\Gamma = 0\) 等价于空间平坦。
    ✅ \(\Gamma\) 不是张量，可以通过坐标变换在一点处消去（法坐标），但曲率（\(\partial\Gamma\) 的组合）不能消去。
  • ❌ 混淆度量兼容性和无挠性：认为只要 \(\nabla g = 0\) 就能唯一确定联络。
    ✅ 度量兼容的联络有无穷多个（加上不同的挠率张量）。唯一性需要无挠性 + 度量兼容性两个条件。

02测地线方程与指数映射

🎯 为什么研究这个问题
  • 最短路径推广：测地线是黎曼流形上"直线"的推广——欧式空间中的直线、球面上的大圆都是测地线的特例。
  • 二阶 ODE 的几何意义：测地线方程 \(\nabla_{\gamma'}\gamma' = 0\) 是"协变加速度为零"，物理对应——自由粒子在弯曲空间中沿测地线运动。
  • 指数映射搭建桥梁：\(\exp_p\) 将切空间的线性结构"映射"到流形的弯曲结构，是法坐标、割迹、单射半径等概念的基础。
🔍 抽象化方法
  • 变分视角：测地线 = 能量泛函或长度泛函的临界点。变分法将几何问题转化为 ODE。
  • 指数映射的微分：\(d(\exp_p)_0 = \mathrm{id}_{T_pM}\)——在原点处指数映射是恒等映射，因此局部是微分同胚。
📐 公式推导 / 模型建立

1. 测地线方程

\(\gamma: I \to M\) 是测地线 \(\iff\) 协变加速度为零：

\[\nabla_{\gamma'}\gamma' = 0 \iff \ddot{\gamma}^k + \Gamma^k_{ij}\dot{\gamma}^i\dot{\gamma}^j = 0\]

这是关于 \(\gamma^k(t)\) 的二阶 ODE 系统。初值 \(\gamma(0) = p\)，\(\gamma'(0) = v\) 唯一确定解（存在唯一性定理）。

2. 指数映射

定义 \(\exp_p: T_pM \supset U \to M\)，\(\exp_p(v) = \gamma_v(1)\)。性质：\(d(\exp_p)_0 = \mathrm{id}\)（局部微分同胚）。缩放性：\(\exp_p(tv) = \gamma_v(t)\)。

3. 测地线的基本性质

(1) 保速率：\(|\gamma'(t)|_g\) 沿测地线恒定（度量兼容性的直接推论）。(2) 局部最短：足够短的测地段是两端点间的最短路径。(3) 重参数化：等速参数化的测地线满足测地线方程。

4. Gauss 引理

\(\exp_p\) 将径向方向映为径向测地线，且径向方向始终正交于测地球面 \(\partial B_r(p)\)。等价地：\(g(\partial/\partial r, V) = 0\)（\(V\) 切于测地球面）。

🌐 实际应用

广义相对论中的自由粒子轨迹：卫星轨道、光线弯曲都是弯曲时空中的测地线。水星近日点进动 \(\Delta\varphi \approx 43''/\text{century}\) 是 Schwarzschild 测地线偏离牛顿轨道的直接结果。

流形上的插值与平均：计算机图形学中，在 SO(3)（旋转群）上两个姿态之间的测地线给出 SLERP（球面线性插值）。黎曼 Fréchet 均值 \(\bar{x} = \arg\min_x \sum d(x, x_i)^2\) 用指数映射迭代求解。

惯性导航系统：惯性导航通过积分加速度来推算位置。在地球曲面上，"无加速运动"对应测地线——大圆航线。指数映射将局部加速度转化为全局位置更新。

✏️ 例题分析与易错点

例题：球面上的测地线

题目：\(S^2\) 上从 \((\pi/2, 0)\)（赤道上点）出发，初速 \(v = \partial_\theta\) 的测地线是什么？

第一步 — 建立方程：测地线方程 \(\ddot{\theta} + \Gamma^\theta_{\varphi\varphi}\dot{\varphi}^2 = 0\)，\(\ddot{\varphi} + 2\Gamma^\varphi_{\theta\varphi}\dot{\theta}\dot{\varphi} = 0\)。

第二步 — 代入初值：\(\dot{\varphi}(0) = 0\)，故 \(\ddot{\theta}(0) = 0\)，\(\ddot{\varphi}(0) = 0\)。由唯一性，\(\varphi(t) \equiv 0\)。

第三步 — 求解：\(\theta(t) = \pi/2 + t\)，这是经线 \(\varphi = 0\) 上的大圆弧。

结论：球面上所有测地线都是大圆。但只有长度 \(\leq \pi\) 的测地段是最短路径——超过半个大圆后，走另一半更短。

常见错误

  • ❌ 认为测地线一定是最短路径。
    ✅ 测地线是长度泛函的临界点，不一定是极小值。球面上超过半圆的大圆弧是测地线但非最短。
  • ❌ 忘记测地线保速率：对测地线做非等速重参数化后仍称其为测地线。
    ✅ 测地线方程隐含等速条件。非等速曲线满足 \(\nabla_{\gamma'}\gamma' = f\gamma'\)（预测地线），不是测地线。
  • ❌ 混淆指数映射的定义域：认为 \(\exp_p\) 总定义在整个 \(T_pM\) 上。
    ✅ 一般只在 \(0\) 的某邻域 \(U \subseteq T_pM\) 上有定义。\(\exp_p\) 定义在全 \(T_pM\) 上 \(\iff\) 流形测地完备（Hopf-Rinow）。

03测地线的变分与能量泛函

🎯 为什么研究这个问题
  • 变分法的几何体现：测地线是长度泛函 \(L\) 或能量泛函 \(E\) 的临界点——变分法将"最短路径"问题转化为微分方程。
  • 能量更方便：\(E(\gamma) = \frac{1}{2}\int |\gamma'|^2\, dt\) 是二次泛函（\(L^2\) 范数），比 \(L = \int |\gamma'|\, dt\)（\(L^1\) 范数）更容易求变分。
  • 第二变分与稳定性：\(E\) 的第二变分涉及 Jacobi 场和曲率张量——这是研究测地线是否"真正最短"的数学框架。
🔍 抽象化方法
  • Euler-Lagrange 方程：测地线方程是能量泛函 \(E\) 的 Euler-Lagrange 方程，对应 Lagrangian \(\mathcal{L} = \frac{1}{2}g_{ij}\dot{\gamma}^i\dot{\gamma}^j\)。
  • 二次型结构：\(E\) 的第二变分是关于变分场 \(V\) 的二次型，其核空间恰好是 Jacobi 场——曲率何时使第二变分不定，决定了测地线的稳定性。
📐 公式推导 / 模型建立

1. 能量泛函与长度泛函

\(E(\gamma) = \frac{1}{2}\int_a^b g(\gamma', \gamma')\, dt\)，\(L(\gamma) = \int_a^b |\gamma'|_g\, dt\)。由 Cauchy-Schwarz：\(L^2 \leq 2(b-a)E\)，等号当且仅当 \(|\gamma'|\) 为常数。

2. 第一变分

\(\gamma_s(t)\) 为固定端点的变分，\(V(t) = \partial_s\gamma_s|_{s=0}\)：

\[\frac{d}{ds}\Big|_{s=0} E(\gamma_s) = -\int_a^b g(\nabla_{\gamma'}\gamma', V)\, dt + g(\gamma', V)\Big|_a^b\]

固定端点 \(V(a) = V(b) = 0\) 时，\(E'(0) = 0\) 对所有 \(V\) 成立 \(\iff\) \(\nabla_{\gamma'}\gamma' = 0\)（测地线方程）。

3. 第二变分

若 \(\gamma\) 是测地线，\(V\) 是正则 Jacobi 场方向的变分场：

\[E''(0) = \int_a^b \left[g\!\left(\frac{DV}{dt}, \frac{DV}{dt}\right) - g(R(\gamma', V)\gamma', V)\right] dt\]

此二次型的符号由曲率决定：正曲率使第二项为正（趋于不稳定），负曲率使之为负（更稳定）。

4. E 与 L 的关系

最小化 \(E\) 的曲线自动等速，因此也最小化 \(L\)。实践中用 \(E\) 代替 \(L\) 做变分更方便：不需处理 \(|\gamma'|\) 在分母中带来的奇异性。

🌐 实际应用

计算机图形学中的测地线插值：在曲面网格上两点之间求测地线（最短路径）用于纹理映射、形变传播。离散测地线算法（如 MMP、Chen-Han）在三角网格上精确展开测地线。

有限元方法中的变分原理：能量泛函的变分结构直接对应有限元弱形式——热传导方程 \(\partial_t u = \Delta_g u\) 的弱解就是能量泛函的梯度流，可在黎曼流形上的三角网格上数值求解。

物理中的最小作用量：经典力学中 Hamilton 原理说粒子运动轨迹使作用量 \(S = \int L\, dt\) 取极值。在弯曲时空中，这归结为测地线方程——自由落体的 Lagrangian 恰好是 \(\frac{1}{2}g_{\mu\nu}\dot{x}^\mu\dot{x}^\nu\)。

✏️ 例题分析与易错点

例题：欧式空间中的第一变分

题目：在 \(\mathbb{R}^n\) 中，\(\gamma_0(t) = (1-t)p + tq\)（直线），变分 \(\gamma_s(t) = \gamma_0(t) + sV(t)\)，\(V(0) = V(1) = 0\)。验证 \(E'(0) = 0\)。

第一步 — 计算加速度：\(\gamma_0'(t) = q - p\) 为常向量，\(\nabla_{\gamma_0'}\gamma_0' = 0\)（\(\mathbb{R}^n\) 平坦，常速直线是测地线）。

第二步 — 代入公式：\(E'(0) = -\int_0^1 g(0, V)\, dt = 0\)。✔

结论：\(\mathbb{R}^n\) 中直线段是能量泛函的临界点。由于 \(\mathbb{R}^n\) 曲率为零，第二变分 \(E''(0) = \int |V'|^2\, dt > 0\)——直线段是严格极小值。

常见错误

  • ❌ 对 \(L\) 直接做变分时忘记 \(|\gamma'|\) 在分母中：\(\delta L\) 含 \(1/|\gamma'|\) 项，非等速曲线的公式更复杂。
    ✅ 实践中总是用 \(E\) 做变分。\(E\) 的临界点自动等速，同时也是 \(L\) 的临界点。
  • ❌ 认为能量泛函的临界点一定是长度极小值。
    ✅ 临界点 = 测地线，但不一定最短。球面上绕远路的大圆弧是测地线（临界点）但非最短路径。第二变分符号决定极小/鞍点。
  • ❌ 混淆 \(E\) 和 \(L\) 的极值关系：以为最小化 \(L\) 等价于最小化 \(E\)。
    ✅ 最小化 \(E\) 的曲线自动等速且最小化 \(L\)。但最小化 \(L\) 的曲线不一定等速——需重参数化。这是 \(E\) 更方便的原因。

04测地线完备性与 Hopf-Rinow 定理

🎯 为什么研究这个问题
  • 根本存在性问题：测地线何时可以无限延伸？任意两点之间是否存在最短路径？Hopf-Rinow 定理回答了这些黎曼几何中最基本的存在性问题。
  • 三种完备性等价：测地完备性（所有测地线可延伸到 \(\mathbb{R}\)）\(\iff\) 度量完备性（Cauchy 列收敛）\(\iff\) 闭有界集紧致——连接微分几何、分析和拓扑三个层面。
  • 不完备 = 奇点：广义相对论中测地不完备性暗示"奇点"的存在——Penrose-Hawking 奇点定理正是利用了此概念。
🔍 抽象化方法
  • 完备性的统一框架：Hopf-Rinow 定理将看似不同的概念（ODE 延伸、Cauchy 完备、有界集紧致）统一为等价条件。
  • 紧致 vs. 完备：紧致流形一定完备（有界闭子集 = 自身），但完备不蕴含紧致（如 \(\mathbb{R}^n\)）。
📐 公式推导 / 模型建立

1. Hopf-Rinow 定理

\((M, g)\) 连通黎曼流形。以下条件等价：

(1) \((M, d_g)\) 作为度量空间完备（Cauchy 列收敛）
(2) \(M\) 的闭有界子集紧致
(3) \(\exists\, p \in M\)，\(\exp_p\) 定义在整个 \(T_pM\) 上
(4) \(\forall\, p \in M\)，\(\exp_p\) 定义在整个 \(T_pM\) 上（测地完备）

2. 最短测地线的存在

若以上条件成立，则 \(\forall\, p, q \in M\)，存在最短测地线 \(\gamma\) 连接 \(p\) 和 \(q\)，满足 \(L(\gamma) = d_g(p,q)\)。

3. 证明思路

(3)\(\Rightarrow\)最短测地线存在：对 \(q \in M\)，设 \(r = d(p,q)\)。在测地球面 \(\partial B_\varepsilon(p)\) 上取使 \(d(\cdot, q)\) 最小的点 \(m\)，沿径向测地线延伸到 \(r\)，利用三角不等式证明终点即 \(q\)。(4)\(\Rightarrow\)(1)：测地完备 \(\Rightarrow\) Cauchy 列 \(\{x_n\}\) 有界，取最短测地线构造极限点。

4. 反例：不完备流形

\(M = (0,1)\)（开区间，标准度量）：测地线 \(\gamma(t) = a + vt\) 在有限时间离开 \(M\)，不完备。\(M = \mathbb{R}^2 \setminus \{0\}\)：虽然不完备（Cauchy 列可趋向原点），但任意两点间仍存在连接曲线（正定性保证 infimum 有限）。

🌐 实际应用

广义相对论中的奇点定理：Penrose 奇点定理证明：在合理的能量条件下，黑洞内部时空必然测地不完备——存在有限仿射参数内终止的测地线（"奇点"）。Hawking 定理对宇宙大爆炸做了类似推导。

完备流形上的分析：完备黎曼流形上 Laplacian 本质自伴（Strichartz 定理），热核存在且唯一，Sobolev 不等式成立——大量分析工具依赖完备性假设。

度量几何中的 Gromov-Hausdorff 收敛：完备黎曼流形的序列在 Gromov-Hausdorff 距离下的收敛理论（预紧性定理）是现代黎曼几何的核心工具，完备性保证极限空间的良好性质。

✏️ 例题分析与易错点

例题：开区间的不完备性

题目：\(M = (0, 1)\)（标准度量）是否测地完备？

第一步 — 测地线分析：测地线 \(\gamma(t) = a + vt\)。取 \(a = 1/2\)，\(v = 1\)：\(\gamma(t) = 1/2 + t\)，在 \(t = 1/2\) 时 \(\gamma = 1 \notin (0,1)\)。

第二步 — Cauchy 列：\(\{1 - 1/n\}\) 是 Cauchy 列但不收敛于 \((0,1)\) 中的点。✔ 度量不完备。

结论：\((0,1)\) 测地不完备且度量不完备，验证了 Hopf-Rinow 定理的等价性。闭区间 \([0,1]\) 有边界不是流形，\(\mathbb{R}\) 是完备的。

常见错误

  • ❌ 混淆完备性与紧致性：认为完备流形一定紧致。
    ✅ \(\mathbb{R}^n\) 完备但非紧致。紧致 \(\Rightarrow\) 完备（有界闭子集 = 自身，紧致），但反向不成立。
  • ❌ 认为 Hopf-Rinow 保证最短测地线唯一：两点间存在最短测地线但不一定唯一。
    ✅ \(S^n\) 上对径点有无穷条最短测地线（所有经过两极的大圆弧）。
  • ❌ 将 Hopf-Rinow 推广到伪黎曼流形（Lorentz 流形）。
    ✅ Hopf-Rinow 定理仅对黎曼流形（正定度量）成立。Lorentz 流形中测地完备不等价于度量完备，需要不同的理论框架。
