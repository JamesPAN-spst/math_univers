联络与测地线

01Levi-Civita 联络回顾

🎯 为什么研究这个问题

在微分几何模块中已初步介绍了 Levi-Civita 联络——无挠 + 度量兼容 唯一确定的联络。在黎曼几何中，它是一切计算的起点：测地线、曲率、 Jacobi 场都基于它。这里从黎曼几何的视角重新审视 Christoffel 符号 和协变导数的计算技巧。

🔍 抽象化方法

Levi-Civita 联络是黎曼度量"内在确定"的微分结构—— 度量不仅量化距离，还规定了"平行搬运"的方式。

📐 公式推导 / 模型建立

Koszul 公式（决定 \(\nabla\)）： \[2g(\nabla_X Y, Z) = Xg(Y,Z) + Yg(X,Z) - Zg(X,Y) + g([X,Y],Z) - g([X,Z],Y) - g([Y,Z],X)\] Christoffel 符号： \[\Gamma^k_{ij} = \frac{1}{2}g^{kl}\left(\frac{\partial g_{il}}{\partial x^j} + \frac{\partial g_{jl}}{\partial x^i} - \frac{\partial g_{ij}}{\partial x^l}\right)\] 协变导数：沿 \(\gamma\) 的向量场 \(V\)， \[\frac{DV^k}{dt} = \frac{dV^k}{dt} + \Gamma^k_{ij}\dot{\gamma}^i V^j\] 法坐标 (Normal Coordinates)：在 \(p\) 处选标准正交基 \(\{e_i\}\)， \(\exp_p: T_pM \to M\)。法坐标中 \(g_{ij}(0) = \delta_{ij}\)， \(\Gamma^k_{ij}(0) = 0\)（但 \(\partial \Gamma \neq 0\)——这就是曲率）。

🌐 实际应用

- 法坐标简化了张量方程的局部验证（在一点处 \(\Gamma = 0\)）

✏️ 例题分析与易错点

例题：双曲平面 \(\mathbb{H}^2\)：\(ds^2 = (dx^2 + dy^2)/y^2\)（上半平面模型）。 求 \(\Gamma^y_{xx}\)。 解答：\(g_{xx} = g_{yy} = 1/y^2\)，\(g_{xy} = 0\)。 \(\Gamma^y_{xx} = \frac{1}{2}g^{yy}\left(\partial_x g_{xy} + \partial_x g_{xy} - \partial_y g_{xx}\right)\) \(= \frac{1}{2} y^2 \left(0 + 0 - (-2/y^3)\right) = \frac{1}{2} y^2 \cdot \frac{2}{y^3} = \frac{1}{y}\)。 易错点：\(\partial_y g_{xx} = \partial_y(1/y^2) = -2/y^3\)，不是 \(-2y\)。

02测地线方程与指数映射

🎯 为什么研究这个问题

测地线是黎曼流形上的"最短路径"（更准确地：临界路径）。它推广 了欧式空间中的直线和球面上的大圆。测地线方程是二阶 ODE， 指数映射 \(\exp_p\) 将切空间射入流形——法坐标正是由指数映射定义的。

🔍 抽象化方法

测地线 = "加速度为零"的曲线（\(\nabla_{\gamma'}\gamma' = 0\)）。 指数映射是切空间到流形的"最自然"的映射。

📐 公式推导 / 模型建立

测地线方程：\(\gamma: I \to M\) 是测地线 \(\iff\) \[\nabla_{\gamma'}\gamma' = 0 \iff \ddot{\gamma}^k + \Gamma^k_{ij}\dot{\gamma}^i\dot{\gamma}^j = 0\] 二阶 ODE 系统。初始条件 \(\gamma(0) = p\)，\(\gamma'(0) = v \in T_pM\) 唯一确定测地线。 指数映射：\(\exp_p: T_pM \supset U \to M\)， \(\exp_p(v) = \gamma_v(1)\)（\(\gamma_v\) 是初始速度 \(v\) 的测地线）。 \(d(\exp_p)_0 = \mathrm{id}_{T_pM}\)（局部微分同胚）。 测地线的性质： (1) 保速率：\(|\gamma'(t)|\) 恒定（度量兼容） (2) 等参数：\(\gamma\) 可以用弧长参数化 (3) 局部最短：足够短的测地段是两端点间的最短路径 Gauss 引理：\(\exp_p\) 将射线映为径向测地线，且径向方向 正交于测地球面。

🌐 实际应用

- 卫星轨道（弯曲时空中的测地线）是广义相对论中自由粒子的轨迹

✏️ 例题分析与易错点

例题：\(S^2\)（单位球面）上从 \((\pi/2, 0)\)（赤道上点）出发， 初速 \(v = \partial/\partial\theta\) 的测地线是什么？ 解答：测地线方程（\(\ddot{\theta} + \Gamma^\theta_{\varphi\varphi}\dot{\varphi}^2 = 0\) 等）。 初始 \(\dot{\varphi} = 0\)，故 \(\ddot{\theta} = 0\)，\(\theta(t) = \pi/2 + t\)， \(\varphi(t) = 0\)。这是通过该点的经线（大圆）。 易错点：球面上所有测地线都是大圆——但不是所有曲线在球面上 最短（只有长度 \(\leq \pi R\) 的测地段是最短的）。

03测地线的变分与能量泛函

🎯 为什么研究这个问题

为什么测地线是"最短路径"？这通过变分法证明：测地线是长度泛函 \(L(\gamma)\) 或能量泛函 \(E(\gamma)\) 的临界点。能量泛函在技术上更 方便（它是 \(L^2\) 而非 \(L^1\)，更容易求变分），其临界点方程正是 测地线方程。

🔍 抽象化方法

测地线 = 能量泛函的 "Euler-Lagrange 方程"的解。 \(E\) 的第二变分涉及 Jacobi 场和曲率——这是曲率如何影响测地线 稳定性的数学框架。

📐 公式推导 / 模型建立

能量泛函：\(E(\gamma) = \frac{1}{2}\int_a^b g(\gamma', \gamma')\,dt\)。 长度泛函：\(L(\gamma) = \int_a^b |\gamma'|\,dt\)。 \(E\) 的临界点（在固定端点的变分下）满足： \[\nabla_{\gamma'}\gamma' = 0 \quad (\text{测地线方程})\] 第一变分：\(\gamma_s(t)\) 是 \(\gamma = \gamma_0\) 的变分， \(V(t) = \partial_s \gamma_s|_{s=0}\)（变分场）。 \[\frac{d}{ds}\Big|_{s=0} E(\gamma_s) = -\int_a^b g(\nabla_{\gamma'}\gamma', V)\,dt + g(\gamma', V)\Big|_a^b\] 固定端点 \(V(a) = V(b) = 0\)，\(E'(0) = 0 \iff \nabla_{\gamma'}\gamma' = 0\)。 \(E\) vs \(L\)：由 Cauchy-Schwarz，\(L(\gamma)^2 \leq 2(b-a)E(\gamma)\)， 等号在等速参数化时成立。\(E\) 的极值点自动等速，因此也是 \(L\) 的极值点。

🌐 实际应用

- 计算机图形学中的测地线插值（流形上的"直线"插值）

✏️ 例题分析与易错点

例题：在 \(\mathbb{R}^n\) 中，选 \(\gamma_0(t) = (1-t)p + tq\)（直线）， 变分 \(\gamma_s(t) = \gamma_0(t) + sV(t)\)，\(V(0) = V(1) = 0\)。 验证 \(E'(0) = 0\)。 解答：\(\gamma_0'(t) = q - p\)，\(\nabla_{\gamma_0'}\gamma_0' = 0\) （因 \(\mathbb{R}^n\) 平坦，常速直线是测地线）。 \(E'(0) = -\int_0^1 g(0, V)\,dt = 0\)。✓ 易错点：\(L\) 的变分在非等速参数化时更复杂（含 \(|\gamma'|\) 在分母）， 所以通常用 \(E\) 而非 \(L\)。

04测地线完备性与 Hopf-Rinow 定理

🎯 为什么研究这个问题

测地线何时可以无限延伸？黎曼流形何时"完备"？Hopf-Rinow 定理 给出了几个等价条件——测地完备性等价于度量完备性（Cauchy 列 收敛），也等价于闭有界集紧致。对完备流形，任意两点之间存在 最短测地线——这是黎曼几何中最基本的存在性定理之一。

🔍 抽象化方法

Hopf-Rinow 定理连接了微分几何（测地线延伸）、度量空间理论 （完备性）和拓扑（紧致性）三个不同层面的完备性概念。

📐 公式推导 / 模型建立

Hopf-Rinow 定理：\((M, g)\) 连通黎曼流形。以下等价： (1) \((M, d_g)\) 作为度量空间完备（Cauchy 列收敛） (2) \(M\) 的闭有界子集是紧致的 (3) \(\exists\, p \in M\)，\(\exp_p\) 定义在整个 \(T_pM\) 上 (4) \(\forall\, p \in M\)，\(\exp_p\) 定义在整个 \(T_pM\) 上 （测地完备：所有测地线可延伸到 \(\mathbb{R}\)） 若以上成立，则 \(\forall\, p, q \in M\)，存在最短测地线 \(\gamma: [0,1] \to M\)，\(\gamma(0) = p\)，\(\gamma(1) = q\)，\(L(\gamma) = d(p,q)\)。 注意：(1)-(4) 不蕴含 \(M\) 紧致（如 \(\mathbb{R}^n\) 完备但非紧致）。 但 (2) 说"有界 \(\Rightarrow\) 紧致"——对非紧致流形是因为有界集 本身不是整个 \(M\)。

🌐 实际应用

- 广义相对论中测地不完备性暗示"奇点"的存在（Penrose 奇点定理）

✏️ 例题分析与易错点

例题：\(M = (0, 1)\)（开区间，标准度量）是否测地完备？ 解答：否。测地线 \(\gamma(t) = a + vt\) 可能在有限时间离开 \((0,1)\)。 例如 \(\gamma(0) = 1/2\)，\(\gamma'(0) = 1\)：\(\gamma(t) = 1/2 + t\)， \(t = 1/2\) 时 \(\gamma = 1 \notin (0,1)\)。 也是度量不完备的：\(\{1-1/n\}\) 是 Cauchy 列但不收敛于 \((0,1)\) 中的点。 易错点：\([0,1]\) 不是流形（有边界），\((0,1)\) 是流形但不完备。 \(\mathbb{R}\) 是完备的。
