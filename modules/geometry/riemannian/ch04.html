测地线与完备性

01Jacobi 场与共轭点

🎯 为什么研究这个问题

测地线之间如何"分散"或"聚焦"？Jacobi 场描述了"无穷近"测地线 的行为——它是测地线变分的线性化。共轭点是 Jacobi 场首次变为零 的位置，标志着测地线失去局部最短性的临界点。正曲率使测地线聚焦 （球面上经线在对径点重合），负曲率使它们发散。

🔍 抽象化方法

Jacobi 方程 = 测地偏差方程。曲率通过 Jacobi 方程控制 测地线族的行为——这是"曲率影响几何"的最直接体现。

📐 公式推导 / 模型建立

Jacobi 方程：\(\gamma\) 是测地线，\(J\) 是沿 \(\gamma\) 的向量场。 \(J\) 是 Jacobi 场 \(\iff\) \[\frac{D^2 J}{dt^2} + R(\gamma', J)\gamma' = 0\] 这是二阶线性 ODE，给定 \(J(0), J'(0)\) 后解唯一。 Jacobi 场空间维数 \(= 2n\)。 共轭点：\(\gamma(t_0)\) 是 \(\gamma(0)\) 的共轭点 \(\iff\) \(\exists\) 非零 Jacobi 场 \(J\)，\(J(0) = 0\)，\(J(t_0) = 0\)。 几何意义：共轭点 \(\Leftrightarrow\) \(\exp_p\) 在 \(t_0 v\) 处奇异 （微分不满秩）。从该点起测地线不再是最短的。 常曲率空间：\(K > 0\)（球面）：\(J(t) = \sin(\sqrt{K}t) E(t)\) （\(E\) 平行，\(J\) 在 \(t = \pi/\sqrt{K}\) 首次为零——对径点是共轭点）。 \(K = 0\)（\(\mathbb{R}^n\)）：\(J(t) = tE(t)\)，无共轭点。 \(K < 0\)（\(\mathbb{H}^n\)）：\(J(t) = \sinh(\sqrt{|K|}t) E(t)\)，无共轭点。

🌐 实际应用

- 引力透镜效应：质量弯曲时空（正 Ricci 曲率），使光线聚焦

✏️ 例题分析与易错点

例题：\(S^2(1)\) 上从北极出发的测地线（经线）的共轭点。 解答：\(K = 1\)。正交 Jacobi 场 \(J(t) = \sin(t) E(t)\)。 \(J(0) = 0\)，\(J(\pi) = 0\)。共轭点在 \(t = \pi\)（南极）。 所有经线在南极交汇——这就是共轭点的几何含义。 长度 \(> \pi\) 的测地段不再最短。 易错点：共轭点不一定是"对径点"——它取决于曲率在测地线沿途的积分效果。

02割点与单射半径

🎯 为什么研究这个问题

共轭点是测地线失去最短性的一种原因（\(\exp_p\) 奇异），但另一种是 "两条不同的最短测地线相遇"——割点。割迹是从 \(p\) 出发各方向 割点的集合，它是 \(M \setminus \{p\}\) 中"最远能确保测地线唯一最短" 的边界。单射半径 \(\mathrm{inj}(p)\) 是到割迹的距离。

🔍 抽象化方法

\(\exp_p\) 只在单射半径内是微分同胚。割迹 = \(\exp_p\) 停止单射的位置。

📐 公式推导 / 模型建立

割点 (cut point)：沿测地线 \(\gamma(t) = \exp_p(tv)\) 前进， \(\gamma(t_c)\) 是割点 \(\iff\) \(t_c = \sup\{t : \gamma|_{[0,t]} \text{ 是到 } \gamma(t) \text{ 的唯一最短测地线}\}\)。 割点的出现原因（二者之一）： (1) \(\gamma(t_c)\) 是共轭点 (2) 存在另一条长度相同的最短测地线从 \(p\) 到 \(\gamma(t_c)\) 割迹 (cut locus)：\(\mathrm{Cut}(p) = \{\text{所有方向的割点}\} \subseteq M\)。 单射半径：\(\mathrm{inj}(p) = d(p, \mathrm{Cut}(p)) = \inf_v t_c(v)\)。 \(\mathrm{inj}(M) = \inf_p \mathrm{inj}(p)\)。 \(\exp_p\) 是 \(B(0, \mathrm{inj}(p)) \subseteq T_pM\) 到 \(B(p, \mathrm{inj}(p)) \subseteq M\) 的微分同胚。

✏️ 例题分析与易错点

例题：\(S^n(1)\) 上北极 \(N\) 的割点和单射半径。 解答：从 \(N\) 出发的测地线（大圆弧）全部在南极 \(S\) 相遇（\(t = \pi\)）。 \(S\) 既是共轭点也是（所有方向的）割点。 \(\mathrm{Cut}(N) = \{S\}\)（单点），\(\mathrm{inj}(N) = \pi\)。 易错点：平坦环面 \(T^2 = \mathbb{R}^2/\mathbb{Z}^2\)，原点的割迹不是单点 而是一个正方形边缘——有无穷多个割点。

\(\mathrm{inj}(\mathbb{R}^n) = ?\)

03Cartan-Hadamard 定理

🎯 为什么研究这个问题

非正截面曲率 (\(K \leq 0\)) 对流形的拓扑有极强限制。Cartan-Hadamard 定理说：完备单连通 \(K \leq 0\) 的流形微分同胚于 \(\mathbb{R}^n\)—— 特别地是可缩的！这意味着非正曲率流形的拓扑复杂性完全来自基本群。

🔍 抽象化方法

\(K \leq 0 + \) 完备 + 单连通 \(\Rightarrow\) 全局微分同胚于 \(\mathbb{R}^n\)。 指数映射 \(\exp_p\) 在 \(K \leq 0\) 时无共轭点，成为覆叠映射， 单连通则是微分同胚。

📐 公式推导 / 模型建立

Cartan-Hadamard 定理：\((M, g)\) 完备，\(K \leq 0\)。 则 \(\forall p \in M\)，\(\exp_p: T_pM \to M\) 是覆叠映射。 若 \(M\) 还单连通，则 \(\exp_p\) 是微分同胚：\(M \cong \mathbb{R}^n\)。 证明要点： (1) \(K \leq 0\) \(\Rightarrow\) 无共轭点（Jacobi 场不为零） (2) 无共轭点 \(\Rightarrow\) \(\exp_p\) 是局部微分同胚 (3) 完备 \(\Rightarrow\) \(\exp_p\) 定义在整个 \(T_pM\) 上 (4) 局部同胚 + 完备 + 覆叠空间理论 \(\Rightarrow\) 覆叠映射 (5) 单连通 \(\Rightarrow\) 覆叠映射是微分同胚 推论： - \(\mathbb{H}^n\) 微分同胚于 \(\mathbb{R}^n\)（但度量不同） - \(K \leq 0\) 流形的 \(\pi_k = 0\)（\(k \geq 2\)），所有拓扑信息在 \(\pi_1\) 中

🌐 实际应用

- 几何群论中非正曲率群（CAT(0) 群）的性质来源

✏️ 例题分析与易错点

例题：双曲平面 \(\mathbb{H}^2\)（\(K = -1\)）。验证 Cartan-Hadamard。 解答：\(\mathbb{H}^2\) 完备（Hopf-Rinow），\(K = -1 \leq 0\)，单连通。 Cartan-Hadamard：\(\exp_p: \mathbb{R}^2 \to \mathbb{H}^2\) 微分同胚。 这就是双曲平面的"圆盘模型"——以 \(p\) 为中心用极坐标参数化整个 \(\mathbb{H}^2\)。 易错点：环面 \(T^2\) 有 \(K = 0\) 且完备，但不单连通。 \(\exp_p\) 是覆叠（不是微分同胚）。\(T^2 \not\cong \mathbb{R}^2\)。

04Bonnet-Myers 定理与直径估计

🎯 为什么研究这个问题

正 Ricci 曲率对流形拓扑有强烈约束——Bonnet-Myers 定理说： 如果 \(\mathrm{Ric} \geq (n-1)K > 0\)，则流形直径 \(\leq \pi/\sqrt{K}\)， 因此流形紧致且基本群有限。这与 Cartan-Hadamard 形成鲜明对比 （负曲率 \(\to\) 非紧致，正曲率 \(\to\) 紧致）。

🔍 抽象化方法

Ricci \(> 0\) 迫使测地线聚焦（第二变分分析），限制了直径和拓扑。

📐 公式推导 / 模型建立

Bonnet-Myers 定理：\((M^n, g)\) 完备，\(\mathrm{Ric} \geq (n-1)K > 0\)。 则： (1) \(\mathrm{diam}(M) \leq \pi/\sqrt{K}\) (2) \(M\) 紧致 (3) \(|\pi_1(M)| < \infty\)（基本群有限） 证明思路（直径估计）： 取最短测地线 \(\gamma\) 从 \(p\) 到 \(q\)，长度 \(L = d(p,q)\)。 对能量泛函做第二变分：选平行正交场 \(E_i\)，令 \(V_i(t) = \sin(\pi t/L) E_i\)。 \[E''(0) = -\int_0^L \left[\frac{\pi^2}{L^2} - K(E_i, \gamma')\right] \sin^2\frac{\pi t}{L}\, dt\] 对 \(n-1\) 个 \(V_i\) 求和： \(\sum E_i'' \leq 0\) 要求 \((n-1)\pi^2/L^2 \geq (n-1)K\)，即 \(L \leq \pi/\sqrt{K}\)。 若 \(L > \pi/\sqrt{K}\)，\(\gamma\) 不是最短——矛盾。

🌐 实际应用

- \(S^n(1)\) 的 \(\mathrm{Ric} = (n-1)\)，直径 \(= \pi = \pi/\sqrt{1}\)（等号成立）

✏️ 例题分析与易错点

例题：\((M^3, g)\)，\(\mathrm{Ric} \geq 2\)（即 \(K = 1\)）。估计直径和 \(|\pi_1|\)。 解答：Bonnet-Myers：\(\mathrm{diam}(M) \leq \pi\)。 \(M\) 紧致。\(\pi_1(M)\) 有限。 万有覆叠 \(\tilde{M}\) 也紧致，直径 \(\leq \pi\)。 已知 \(S^3\) 的 \(\mathrm{Ric} = 2\)，\(\mathrm{diam} = \pi\)（等号）。 易错点：Bonnet-Myers 给出的是上界。\(\mathrm{diam}\) 可以 \(< \pi/\sqrt{K}\)。 等号条件很特殊（仅常曲率 \(K\) 球面达到）。
