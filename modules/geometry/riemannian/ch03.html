曲率

01Riemann 曲率张量

🎯 为什么研究这个问题

曲率是黎曼几何的核心概念——它量化空间的"弯曲程度"。Riemann 曲率张量 \(R\) 是一个 \((1,3)\) 类型张量，完全编码了空间的内在曲率信息。 它衡量的是平行移动的路径依赖性：沿一个封闭小回路平行移动向量， 回来后向量旋转了——旋转量正比于曲率。

🔍 抽象化方法

\(R(X,Y)Z\) 测量 \(\nabla_X\) 和 \(\nabla_Y\) 的"不可交换程度"—— 是联络的"二阶障碍"。

📐 公式推导 / 模型建立

定义： \[R(X,Y)Z = \nabla_X\nabla_Y Z - \nabla_Y\nabla_X Z - \nabla_{[X,Y]}Z\] 坐标中：\(R^l{}_{kij} = \partial_i\Gamma^l_{jk} - \partial_j\Gamma^l_{ik} + \Gamma^l_{im}\Gamma^m_{jk} - \Gamma^l_{jm}\Gamma^m_{ik}\)。 对称性（Levi-Civita 联络）： (1) \(R_{ijkl} = -R_{jikl} = -R_{ijlk}\) (2) \(R_{ijkl} = R_{klij}\)（对称对） (3) 第一 Bianchi：\(R_{ijkl} + R_{iklj} + R_{iljk} = 0\) (4) 第二 Bianchi：\(\nabla_m R_{ijkl} + \nabla_i R_{jmkl} + \nabla_j R_{mikl} = 0\) 独立分量数：\(n^2(n^2-1)/12\)（\(n=2\) 时 \(1\) 个，\(n=3\) 时 \(6\) 个，\(n=4\) 时 \(20\) 个）。 \[n = 2: \quad R_{1212} \text{ 完全确定 } R\]

🌐 实际应用

- Einstein 场方程中曲率张量描述引力场

✏️ 例题分析与易错点

例题：\(S^2(R)\)（半径 \(R\)）的曲率张量。 解答：常曲率空间 \(K = 1/R^2\)。 \(R_{ijkl} = K(g_{ik}g_{jl} - g_{il}g_{jk})\)。 \(R_{\theta\varphi\theta\varphi} = K \cdot (g_{\theta\theta}g_{\varphi\varphi} - g_{\theta\varphi}^2) = \frac{1}{R^2} \cdot R^4\sin^2\theta = R^2\sin^2\theta\)。 易错点：指标升降很重要。\(R^\theta{}_{\varphi\theta\varphi} = g^{\theta\theta}R_{\theta\varphi\theta\varphi} = \frac{1}{R^2} \cdot R^2\sin^2\theta = \sin^2\theta\)。

⚡ 快速检测
\(n\) 维黎曼流形的 Riemann 张量有多少个独立分量？

02截面曲率与 Ricci 曲率

🎯 为什么研究这个问题

Riemann 曲率张量有很多分量，不易直观理解。截面曲率 \(K(\Pi)\) 取一个 2 维平面 \(\Pi \subseteq T_pM\) 测量"该方向的弯曲"——它 推广了 Gauss 曲率。Ricci 曲率是截面曲率的迹（平均），标量曲率 再取迹。Einstein 方程直接涉及 Ricci 曲率。

🔍 抽象化方法

Riemann \(\to\) 截面曲率（2 维信息）\(\to\) Ricci（迹）\(\to\) 标量曲率（双迹）。 信息逐步减少，但在低维中截面曲率完全决定 Riemann。

📐 公式推导 / 模型建立

截面曲率：\(\Pi = \mathrm{span}(u,v) \subseteq T_pM\)（2 维平面）， \[K(\Pi) = \frac{R(u,v,v,u)}{g(u,u)g(v,v) - g(u,v)^2}\] \(K\) 与 \(u,v\) 的选取无关（只依赖平面 \(\Pi\)）。 \(n = 2\) 时 \(K = \) Gauss 曲率。 常截面曲率 \(K\) 的空间：\(S^n(1/\sqrt{K})\), \(\mathbb{R}^n\), \(\mathbb{H}^n\) （\(K > 0, = 0, < 0\)）。 Ricci 曲率：\(\mathrm{Ric}(u,v) = \sum_{i=1}^n R(e_i, u, v, e_i)\)（\(\{e_i\}\) 正交基）。 即 \(\mathrm{Ric}_{ij} = R^k{}_{ikj}\)（\(R\) 的迹）。 \(\mathrm{Ric}(v,v)/|v|^2\) = 经过 \(v\) 方向的截面曲率的平均。 标量曲率：\(S = g^{ij}\mathrm{Ric}_{ij} = \sum_i \mathrm{Ric}(e_i, e_i)\)。 \[S = \sum_{i \neq j} K(\Pi_{ij})\]

🌐 实际应用

- Ricci 曲率 \(\geq (n-1)K > 0\) \(\Rightarrow\) 流形直径有上界（Bonnet-Myers） - Ricci 流 \(\partial_t g = -2\mathrm{Ric}\) 是 Perelman 证明几何化猜想的工具

✏️ 例题分析与易错点

例题：\(S^n(R)\) 的 Ricci 曲率和标量曲率。 解答：截面曲率 \(K = 1/R^2\)（常数，所有平面）。 \(\mathrm{Ric}(v,v) = (n-1)K|v|^2 = (n-1)/R^2 \cdot |v|^2\)。 \(\mathrm{Ric}_{ij} = (n-1)/R^2 \cdot g_{ij}\)（Einstein 流形）。 \(S = n(n-1)/R^2\)。 易错点：\(\mathrm{Ric}\) 是 \((n-1)\) 个截面曲率之和（不是平均值）， 标量曲率是 \(\binom{n}{2}\) 个截面曲率之和乘以 2（或其他约定——注意系数约定）。

03Gauss 曲率与 Gauss-Bonnet 定理

🎯 为什么研究这个问题

Gauss-Bonnet 定理是微分几何中最深刻的定理之一：它将曲面的 Gauss 曲率（局部几何/分析量）积分后等于 Euler 特征（全局拓扑 不变量）。"几何完全决定拓扑"——这是局部-整体原理的巅峰体现。 高维推广是 Chern-Gauss-Bonnet 定理。

🔍 抽象化方法

Gauss-Bonnet = 曲率（分析/几何）与 Euler 特征（拓扑）的桥梁。 这是指标定理的原型。

📐 公式推导 / 模型建立

Gauss 曲率：2 维黎曼曲面 \((M^2, g)\)，\(K = R_{1212} / \det(g_{ij})\)。 \(K\) 是 2 维 Riemann 张量的唯一独立分量。 Gauss-Bonnet 定理（闭曲面）：\(M\) 闭定向 2 维黎曼流形， \[\int_M K\, dA = 2\pi \chi(M)\] 带边界版本：\(M\) 紧致带边界，\(\partial M\) 分段光滑， \[\int_M K\, dA + \int_{\partial M} \kappa_g\, ds + \sum_i \alpha_i = 2\pi \chi(M)\] \(\kappa_g\) = 边界测地曲率，\(\alpha_i\) = 角点的外角。 推论： - \(\chi(S^2) = 2\)：\(\int K\, dA = 4\pi\)（\(K = 1/R^2\) 对球面） - \(\chi(T^2) = 0\)：\(\int K\, dA = 0\)（环面的总曲率为零）

🌐 实际应用

- 大地测量学：三角形内角和偏差 \(= \int_\Delta K\, dA\) - Chern-Gauss-Bonnet 在高维推广为指标定理（Atiyah-Singer）的特例

✏️ 例题分析与易错点

例题：\(S^2(R)\) 的 Gauss-Bonnet 验证。 解答：\(K = 1/R^2\)（常数），\(\mathrm{Area}(S^2(R)) = 4\pi R^2\)。 \(\int K\, dA = (1/R^2)(4\pi R^2) = 4\pi\)。 \(2\pi\chi(S^2) = 2\pi \cdot 2 = 4\pi\)。✓ 易错点：\(\chi(T^2) = 0\)，所以环面上正曲率和负曲率区域的贡献完全抵消。 环面不可能有处处正的 Gauss 曲率（否则 \(\int K > 0 \neq 0\)）。

04常曲率空间与空间形式

🎯 为什么研究这个问题

截面曲率恒为常数 \(K\) 的黎曼流形——空间形式——是几何中最对称 的空间。\(K > 0\) 是球面，\(K = 0\) 是欧氏空间，\(K < 0\) 是双曲空间。 它们的分类是一个经典问题：完备单连通的常曲率空间只有这三种 （唯一性定理），一般常曲率空间是它们被离散群作用的商。

🔍 抽象化方法

常曲率 = 最大对称。Schur 定理：如果截面曲率在每点各方向相同 且 \(\dim \geq 3\)，则 \(K\) 全局为常数。

📐 公式推导 / 模型建立

模型空间： - \(K > 0\)：\(S^n(1/\sqrt{K})\)（半径 \(1/\sqrt{K}\) 的球面） - \(K = 0\)：\(\mathbb{R}^n\) - \(K < 0\)：\(\mathbb{H}^n\)（双曲空间） 唯一性定理：完备单连通 \(n\) 维常曲率 \(K\) 黎曼流形与模型空间等距（\(n \geq 2\)）。 空间形式：\(M = \tilde{M} / \Gamma\)，\(\tilde{M}\) 为模型空间， \(\Gamma\) 是 \(\mathrm{Isom}(\tilde{M})\) 的离散自由作用子群。 - \(K > 0\)：球面空间形式（如 \(\mathbb{R}P^n = S^n / \mathbb{Z}_2\)，透镜空间） - \(K = 0\)：Bieberbach 群的商（如 \(T^n = \mathbb{R}^n / \mathbb{Z}^n\)） - \(K < 0\)：双曲流形 Schur 定理：\(\dim M \geq 3\)，如果截面曲率在每点 \(K_p(\Pi)\) 对所有 \(\Pi\) 相同（\(= K(p)\)），则 \(K(p) = \) 常数。 （证明用第二 Bianchi 恒等式。）

🌐 实际应用

- 宇宙学中三种标准空间模型（正曲率/平坦/双曲）对应宇宙的三种几何

✏️ 例题分析与易错点

例题：\(M = \mathbb{R}^2 / \mathbb{Z}^2\)（平坦环面），截面曲率 \(K = ?\) 解答：\(\tilde{M} = \mathbb{R}^2\) 是 \(K = 0\) 的模型空间。 \(\Gamma = \mathbb{Z}^2\) 通过平移作用，等距变换不改变曲率。 \(K = 0\)（平坦环面）。 易错点：环面可以嵌入 \(\mathbb{R}^3\)（显式参数化），但嵌入度量 的 Gauss 曲率有正有负区域——那不是内在的平坦度量。
