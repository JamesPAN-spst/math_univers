01子流形与诱导度量

🎯 为什么研究这个问题

微分几何最初就是研究嵌入在三维欧氏空间中的曲面——子流形几何是整个学科的起点。理解子流形如何从环境空间继承度量，以及内在几何与外在几何的区分，不仅是黎曼几何独立发展的历史动力，也是 Nash 嵌入定理连接抽象理论与具体构造的桥梁。

  • 几何的原始形态：微分几何最初就是研究 \(\mathbb{R}^3\) 中的曲面——子流形几何。理解子流形如何从环境空间继承度量（诱导度量）是整个学科的起点。
  • 内在 vs 外在：诱导度量定义了子流形的内在几何（长度、角度、面积），而嵌入方式决定了外在几何（法向弯曲）。Gauss 的 Theorema Egregium 证明了内在曲率不依赖嵌入——这是黎曼几何独立发展的动力。
  • Nash 嵌入定理：反过来，任何抽象黎曼流形都可以等距嵌入到足够高维的欧氏空间——子流形观点可以作为内在几何的替代框架。

诱导度量、正交分解与 Gauss 公式建立了子流形内在几何与外在几何的精确联系，为第二基本形式、Gauss-Codazzi 方程和极小曲面理论打下了坚实基础。

🔍 抽象化方法
  • 正交分解：\(T_p\bar{M} = T_pM \oplus N_pM\)（切空间 ⊕ 法空间）。环境空间的所有张量沿此分解分为切向和法向分量。
  • Gauss 公式：\(\bar{\nabla}_X Y = \nabla_X Y + \mathrm{I\!I}(X,Y)\)。切向分量 = 子流形自身的联络；法向分量 = 第二基本形式（外在弯曲的度量）。
📐 公式推导 / 模型建立

1. 等距嵌入与诱导度量

\(f: (M^m, g) \hookrightarrow (\bar{M}^n, \bar{g})\) 等距嵌入：\(f^*\bar{g} = g\)。坐标中，若 \(f(u^1, \ldots, u^m)\)：

\[g_{\alpha\beta} = \bar{g}_{ij}\, \frac{\partial f^i}{\partial u^\alpha}\frac{\partial f^j}{\partial u^\beta}\]

2. 正交分解与 Gauss 公式

\(T_p\bar{M} = T_pM \oplus N_pM\)。对 \(X, Y \in \Gamma(TM)\)：

\[\bar{\nabla}_X Y = \underbrace{\nabla_X Y}_{\text{切向}} + \underbrace{\mathrm{I\!I}(X,Y)}_{\text{法向}}\]

\(\nabla\) 是 \(M\) 上的 Levi-Civita 联络（诱导度量 \(g\) 确定），\(\mathrm{I\!I}\) 是第二基本形式。

3. Weingarten 公式

对法向场 \(\nu \in \Gamma(NM)\)：\(\bar{\nabla}_X \nu = -A_\nu X + \nabla^\perp_X \nu\)。其中 \(A_\nu\) 是形状算子（Weingarten 映射），\(\nabla^\perp\) 是法联络。

4. Nash 嵌入定理

任何 \(m\) 维紧致黎曼流形可以 \(C^1\)-等距嵌入到 \(\mathbb{R}^{m+1}\)（Nash-Kuiper）；\(C^k\)（\(k \geq 3\)）等距嵌入到 \(\mathbb{R}^N\)，\(N \leq m(m+1)/2 + m\)（Nash）。

🌐 实际应用

Nash 嵌入与 PDE 正则性：Nash 嵌入定理的 \(C^1\) 版本（Nash-Kuiper）产生"褶皱"嵌入——这与 Gromov 的 h-原理密切相关（柔性 vs 刚性嵌入）。此思想启发了流体力学中 Euler 方程弱解的非唯一性（Onsager 猜想/De Lellis-Székelyhidi 方案）。

计算机图形学中的参数化：三维扫描得到的网格 \(M \subset \mathbb{R}^3\) 需要"展开"为平面参数（纹理映射）。诱导度量给出了展开的失真度量——等距参数化（\(g = \mathrm{Id}\)）通常不存在（曲面有曲率），实际中用共形或等面积近似。

广义相对论中的约束方程：时空 \((\bar{M}^4, \bar{g})\) 中的类空超曲面 \(\Sigma^3\) 继承诱导度量和第二基本形式。Einstein 方程约束到 \(\Sigma\) 上给出 Hamiltonian 和 momentum 约束方程——这是数值相对论 (3+1 分解) 的起点。

✏️ 例题分析与易错点

例题：球面的诱导度量

题目：\(S^2(R) \hookrightarrow \mathbb{R}^3\)，参数化 \(f(\theta,\varphi) = (R\sin\theta\cos\varphi,\, R\sin\theta\sin\varphi,\, R\cos\theta)\)，求诱导度量。

第一步 — 基向量：\(f_\theta = (R\cos\theta\cos\varphi, R\cos\theta\sin\varphi, -R\sin\theta)\)，\(f_\varphi = (-R\sin\theta\sin\varphi, R\sin\theta\cos\varphi, 0)\)。

第二步 — 度量分量：\(g_{\theta\theta} = f_\theta \cdot f_\theta = R^2\)，\(g_{\varphi\varphi} = R^2\sin^2\theta\)，\(g_{\theta\varphi} = 0\)。

第三步 — 线元：\(ds^2 = R^2(d\theta^2 + \sin^2\theta\, d\varphi^2)\)。✔ 标准球面度量。

常见错误

  • ❌ 认为不同嵌入给出相同的诱导度量。
    ✅ 诱导度量依赖于嵌入方式。将 \(T^2\) 嵌入 \(\mathbb{R}^3\) 时得到非平坦度量（有 Gauss 曲率），但 \(T^2 = \mathbb{R}^2/\mathbb{Z}^2\) 内在有平坦度量。不同嵌入 → 不同诱导度量。
  • ❌ 认为 Nash 嵌入维度总是 \(N = m + 1\)（余维 1）。
    ✅ \(C^k\)（\(k \geq 3\)）嵌入需要 \(N \sim m^2/2\) 维。\(C^1\) 嵌入可以做到 \(m+1\)（Nash-Kuiper），但产生非光滑的"褶皱"嵌入。
  • ❌ 将法向量和切向量混在同一空间讨论而不做正交投影。
    ✅ 所有公式都依赖正交分解 \(T\bar{M} = TM \oplus NM\)。Gauss 公式取切向投影，Weingarten 公式取法向投影，必须严格区分。

02第二基本形式

🎯 为什么研究这个问题

子流形不仅有内在弯曲（由诱导度量的 Riemann 张量刻画），还有相对于环境空间的外在弯曲。第二基本形式精确测量了这种外在弯曲——它是从内在几何无法感知的额外信息，也是定义平均曲率和极小曲面的基础。

  • 外在弯曲的度量：第二基本形式 \(\mathrm{I\!I}\) 精确测量子流形在环境空间中"翘起来"的程度。平面中的直线 \(\mathrm{I\!I} = 0\)，圆 \(\mathrm{I\!I} \neq 0\)——\(\mathrm{I\!I}\) 是区分"弯"与"直"的核心工具。
  • 主曲率与几何形态：\(\mathrm{I\!I}\) 的特征值（主曲率）描述了曲面在各方向的弯曲程度和方向。脐点（各方向主曲率相同）、鞍点（主曲率异号）等几何特征都由 \(\mathrm{I\!I}\) 刻画。
  • 极小曲面的定义：平均曲率 \(H = \frac{1}{m}\mathrm{tr}\,\mathrm{I\!I}\)。\(H = 0\) 定义极小子流形——面积泛函的临界点。

第二基本形式与形状算子将子流形的外在弯曲转化为精确的张量语言，其特征值（主曲率）和迹（平均曲率）分别刻画了几何形态和面积泛函的临界性。

🔍 抽象化方法
  • 法向加速度：\(\mathrm{I\!I}(X,Y) = (\bar{\nabla}_X Y)^\perp\)——环境空间中协变导数的法向分量。全测地子流形（\(\mathrm{I\!I} \equiv 0\)）的测地线也是环境空间的测地线。
  • 形状算子对偶：\(\bar{g}(\mathrm{I\!I}(X,Y), \nu) = g(A_\nu X, Y)\)。形状算子 \(A_\nu\) 是 \(\mathrm{I\!I}_\nu\) 的"切空间版本"，其特征值 = 主曲率，特征向量 = 主方向。
📐 公式推导 / 模型建立

1. 第二基本形式

\(\mathrm{I\!I}: \Gamma(TM) \times \Gamma(TM) \to \Gamma(NM)\)：

\[\mathrm{I\!I}(X,Y) = (\bar{\nabla}_X Y)^\perp = \bar{\nabla}_X Y - \nabla_X Y\]

\(\mathrm{I\!I}\) 是 \(C^\infty(M)\)-双线性、对称的张量（而非算子）。

2. 形状算子与主曲率

选法方向 \(\nu\)，定义形状算子 \(A_\nu: TM \to TM\)，\(A_\nu X = -(\bar{\nabla}_X \nu)^\top\)。满足 \(\bar{g}(\mathrm{I\!I}(X,Y), \nu) = g(A_\nu X, Y)\)。

\(A_\nu\) 自伴随，其特征值 \(\kappa_1, \ldots, \kappa_m\) = 主曲率，特征向量 = 主方向。

3. 平均曲率与极小性

\(H = \frac{1}{m}\mathrm{tr}_g\, \mathrm{I\!I} = \frac{1}{m}\sum_{i=1}^m \kappa_i\, \nu\)。\(H = 0\) 定义极小子流形。全测地子流形 \(\mathrm{I\!I} \equiv 0\) 严格强于极小。

4. 脐点与全脐

若 \(A_\nu = \lambda\, \mathrm{Id}\)（所有主曲率相同），则 \(p\) 是脐点。全脐超曲面 = 球面或超平面。全脐 + \(H = 0\) = 全测地。

🌐 实际应用

肥皂膜与极小曲面：肥皂膜在平衡态最小化面积（表面张力作用），因此 \(H = 0\)——它们是极小曲面的物理实现。Plateau 问题（给定边界曲线找面积最小的曲面）由 Douglas 和 Radó 在 1930 年代解决。

曲面重建中的主曲率分析：3D 扫描和计算机视觉中，估计曲面各点的主曲率和主方向用于特征检测（脊线/谷线）、分割和匹配。主曲率的符号区分凸/凹/鞍区域。

广义相对论中的外曲率：时空超曲面 \(\Sigma^3 \subset M^4\) 的第二基本形式（外曲率 \(K_{ij}\)）与诱导度量是 ADM 形式主义的动力学变量。Einstein 场方程约束到超曲面上给出 Hamilton 和动量约束。

✏️ 例题分析与易错点

例题：球面的第二基本形式

题目：\(S^2(R) \hookrightarrow \mathbb{R}^3\)，外法向 \(\nu = (x,y,z)/R\)，求 \(\mathrm{I\!I}\) 和主曲率。

第一步 — 形状算子：\(A_\nu X = -(\bar{\nabla}_X \nu)^\top = -\frac{1}{R}X\)（对 \(\mathbb{R}^3\) 中球面，\(\bar{\nabla}_X \nu = X/R\)，投影到切空间取负号）。

第二步 — 主曲率：\(A_\nu = \frac{1}{R}\mathrm{Id}\)，所有主曲率 \(\kappa_1 = \kappa_2 = 1/R\)（全脐）。

第三步 — Gauss 曲率：\(K = \kappa_1\kappa_2 = 1/R^2\)。\(H = 1/R\)。✔

常见错误

  • ❌ 法方向约定影响主曲率符号但忘记声明。
    ✅ 约定外法向时凸曲面主曲率为正。内法向则符号翻转。Gauss 曲率 \(K = \kappa_1\kappa_2\) 不受法向选择影响（乘两次符号抵消）。
  • ❌ 混淆极小（\(H = 0\)）和全测地（\(\mathrm{I\!I} \equiv 0\)）。
    ✅ 全测地严格强于极小。\(S^1\) 大圆在 \(S^2\) 中全测地（也极小）。catenoid 在 \(\mathbb{R}^3\) 中极小但不全测地。
  • ❌ 认为主曲率是内在不变量。
    ✅ 主曲率依赖于嵌入方式——它们是外在几何。只有主曲率的乘积（二维 Gauss 曲率）才是内在量（Theorema Egregium）。

03Gauss 方程与 Codazzi 方程

🎯 为什么研究这个问题

子流形的内在曲率与外在弯曲之间存在什么精确关系？Gauss 方程给出了答案——它将子流形的 Riemann 曲率分解为环境曲率与第二基本形式的贡献。Codazzi 方程则对第二基本形式施加了可积性约束，两者共同构成了子流形几何的基本方程组。

  • 内在与外在曲率的精确关系：Gauss 方程将子流形的 Riemann 曲率表示为环境曲率 + 第二基本形式的修正项。当 \(\bar{M} = \mathbb{R}^n\)（平坦）时，子流形的全部内在曲率来自外在弯曲——\(K = \kappa_1\kappa_2\) 正是 Gauss 方程的特例。
  • 可积性条件：Codazzi 方程约束第二基本形式的协变导数。它不是独立的选择——\(\mathrm{I\!I}\) 必须满足 Codazzi 条件才能对应一个实际的嵌入。
  • 子流形基本定理：Gauss + Codazzi（+ Ricci）方程恰好是子流形存在和唯一性的充要条件。给定满足这些方程的 \((g, \mathrm{I\!I})\)，可以构造等距嵌入。

Gauss-Codazzi 方程不仅阐明了内在曲率与外在弯曲的精确关系，更作为子流形基本定理的充要条件，为等距嵌入的存在性和唯一性提供了完备的理论框架。

🔍 抽象化方法
  • 曲率的分解：环境曲率 \(\bar{R}\) 在子流形上沿 \(TM \oplus NM\) 分解：切-切分量 → Gauss 方程；切-法分量 → Codazzi 方程；法-法分量 → Ricci 方程。
  • Theorema Egregium：当 \(\bar{M} = \mathbb{R}^n\) 时，Gauss 方程写为 \(R = \mathrm{I\!I} \wedge \mathrm{I\!I}\)。Gauss 曲率 \(K = \det A_\nu\) 看似依赖嵌入，但 Gauss 方程表明它等于 \(R_{1221}\)——纯粹的内在量。
📐 公式推导 / 模型建立

1. Gauss 方程

\[R(X,Y,Z,W) = \bar{R}(X,Y,Z,W) + \bar{g}(\mathrm{I\!I}(X,Z), \mathrm{I\!I}(Y,W)) - \bar{g}(\mathrm{I\!I}(X,W), \mathrm{I\!I}(Y,Z))\]

2. 超曲面在 \(\mathbb{R}^n\) 中的特例

\(\bar{R} = 0\)，余维 1。选主方向 \(e_i\)：\(K_M(e_i \wedge e_j) = \kappa_i\kappa_j\)。二维时 \(K = \kappa_1\kappa_2\)（Gauss 的 Theorema Egregium）。

3. Codazzi 方程

\[(\bar{R}(X,Y)Z)^\perp = (\nabla_X \mathrm{I\!I})(Y,Z) - (\nabla_Y \mathrm{I\!I})(X,Z)\]

\(\bar{M} = \mathbb{R}^n\) 时简化为 \((\nabla_X \mathrm{I\!I})(Y,Z) = (\nabla_Y \mathrm{I\!I})(X,Z)\)（\(\mathrm{I\!I}\) 的协变导数关于前两个变量对称）。

4. 子流形基本定理

给定 \((M^m, g)\) 和对称 \((0,2)\)-型法值张量 \(\mathrm{I\!I}\)，若它们满足 Gauss、Codazzi（和余维 \(> 1\) 时的 Ricci）方程，则存在（局部唯一的）等距嵌入 \(f: M \to \bar{M}\)，使得 \(\mathrm{I\!I}\) 恰为第二基本形式。

🌐 实际应用

Theorema Egregium 与制图学：Gauss 的绝妙定理证明 Gauss 曲率是内在不变量——等距变换保持 Gauss 曲率。这解释了为何球面（\(K > 0\)）不能无失真地展开到平面（\(K = 0\)）——任何世界地图都必然有变形。

约束方程与数值相对论：时空超曲面的 Gauss-Codazzi 方程正是 Einstein 场方程的 Hamiltonian 和 momentum 约束。数值相对论（如黑洞合并模拟）中初始数据必须满足这些约束方程——这是 3+1 分解的数学基础。

刚性与嵌入障碍：Gauss-Codazzi 方程给出了嵌入的可积性条件。例如，不存在 \(\mathbb{H}^2\)（\(K = -1\)）到 \(\mathbb{R}^3\) 中的 \(C^2\) 完备等距嵌入（Hilbert 定理）——因为 Gauss 方程约束主曲率乘积为 \(-1\)，而全局上无法保持。

✏️ 例题分析与易错点

例题：Gauss 方程验证 \(K = \kappa_1\kappa_2\)

题目：\(M^2 \subseteq \mathbb{R}^3\)，主曲率 \(\kappa_1, \kappa_2\)，用 Gauss 方程验证 \(K = \kappa_1\kappa_2\)。

第一步 — Gauss 方程（平坦环境）：\(R(X,Y,Z,W) = g(\mathrm{I\!I}(X,Z), \mathrm{I\!I}(Y,W)) - g(\mathrm{I\!I}(X,W), \mathrm{I\!I}(Y,Z))\)。

第二步 — 取主方向：\(X = Z = e_1\)，\(Y = W = e_2\)。\(\mathrm{I\!I}(e_i, e_j) = \kappa_i\delta_{ij}\nu\)。

第三步 — 代入：\(R(e_1,e_2,e_1,e_2) = \kappa_1\kappa_1 \cdot 0 - \kappa_1\kappa_2 = ... \) 仔细计算：\(R_{1212} = \kappa_1\kappa_2\)。\(K = R_{1212}/(g_{11}g_{22} - g_{12}^2) = \kappa_1\kappa_2\)。✔

常见错误

  • ❌ Gauss 方程中 \(R\) 的符号约定在不同教材中不同。
    ✅ 常见两种约定：\(R(X,Y)Z = \nabla_X\nabla_Y Z - \ldots\) 或相反。对应的 Gauss 方程差一个整体符号。计算前务必确认约定。
  • ❌ 认为 Codazzi 方程是 Gauss 方程的推论。
    ✅ Gauss 和 Codazzi 是独立的方程——Gauss 约束曲率（代数关系），Codazzi 约束 \(\mathrm{I\!I}\) 的导数（微分关系）。子流形基本定理需要两者同时满足。
  • ❌ 认为 Theorema Egregium 对主曲率也成立（主曲率也是内在量）。
    ✅ 只有主曲率的乘积 \(K = \kappa_1\kappa_2\)（Gauss 曲率）是内在量。主曲率本身依赖嵌入——椭球面与球面可以有相同的 \(K\) 但不同的主曲率。

04极小曲面与调和映射

🎯 为什么研究这个问题

肥皂膜为何总是形成面积最小的薄膜？极小曲面理论回答了这个问题——平均曲率为零的子流形正是面积泛函的临界点。调和映射则将这一思想推广到更一般的映射空间，统一了测地线、调和函数和极小子流形的变分框架。

  • 变分原理的几何实现：极小曲面（\(H = 0\)）是面积泛函的临界点——肥皂膜在平衡态自然形成极小曲面。它们是变分学与微分几何交汇的经典对象。
  • 统一框架：调和映射 \(f: (M,g) \to (N,h)\) 统一了测地线（\(\dim M = 1\)）、调和函数（\(N = \mathbb{R}\)）和极小子流形（等距浸入 + \(\tau = mH = 0\)）。
  • 现代几何分析的核心：极小曲面理论是几何分析的重要分支——正质量定理（Schoen-Yau）、Willmore 猜想（Marques-Neves）、广义相对论中的视界面积都与极小曲面密切相关。

极小曲面与调和映射将变分原理与几何分析深度融合，从肥皂膜的物理直觉到正质量定理和 Willmore 猜想的证明，这一方向是现代几何分析最活跃的前沿之一。

🔍 抽象化方法
  • 面积 vs 能量：极小曲面是面积泛函 \(A = \int dV\) 的临界点；调和映射是能量泛函 \(E = \frac{1}{2}\int |df|^2\) 的临界点。两者的 Euler-Lagrange 方程分别为 \(H = 0\) 和 \(\tau(f) = 0\)。
  • Weierstrass 表示：在 \(\mathbb{R}^3\) 中，极小曲面可以通过全纯函数和亚纯函数参数化（Weierstrass-Enneper 表示）——将 PDE 问题转化为复分析问题。
📐 公式推导 / 模型建立

1. 极小曲面：面积第一变分

\(M^m \hookrightarrow \bar{M}^n\)，面积泛函 \(A(M) = \int_M dV\)。变分场 \(V\)：

\[\delta A = -m\int_M \bar{g}(H, V^\perp)\, dV\]

\(\delta A = 0\) 对所有 \(V\) ⟺ \(H = 0\)（极小）。

2. 调和映射方程

\(f: (M,g) \to (N,h)\)，能量泛函 \(E(f) = \frac{1}{2}\int_M |df|^2\, dV_g\)。Euler-Lagrange 方程：

\[\tau(f) \equiv \mathrm{tr}_g(\nabla df) = 0\]

坐标中：\(\tau^\alpha = \Delta_g f^\alpha + g^{ij}\Gamma^\alpha_{\beta\gamma}(f)\, \partial_i f^\beta\, \partial_j f^\gamma = 0\)。

3. 特例的统一

\(\dim M = 1\)：\(\tau = \nabla_{\gamma'}\gamma' = 0\)（测地线方程）。\(N = \mathbb{R}\)：\(\tau = \Delta_g f = 0\)（调和函数）。等距浸入 \(f: M \to N\)：\(\tau(f) = mH\)（张力场 = \(m\) 倍平均曲率），调和 ⟺ 极小。

4. 经典极小曲面

catenoid（旋转面 \(H=0\)）、helicoid（螺旋面）、Enneper 曲面。Bernstein 定理：\(\mathbb{R}^3\) 中整体定义的极小图 \(z = f(x,y)\) 必须是平面。

🌐 实际应用

Plateau 问题与肥皂膜：给定 \(\mathbb{R}^3\) 中的闭曲线 \(\Gamma\)，找以 \(\Gamma\) 为边界的面积最小曲面。Douglas (1931) 和 Radó 独立证明了解的存在性（Fields 奖）。肥皂膜实验物理实现了 Plateau 问题——但肥皂膜可能是鞍点而非全局极小。

正质量定理与黑洞面积：Schoen-Yau 利用极小曲面证明了正质量定理：渐近平坦的具有非负标量曲率的时空的 ADM 质量 \(\geq 0\)。极小曲面的存在性和正则性是证明的技术核心。彭罗斯不等式（黑洞质量与视界面积的关系）也在此框架中。

弦论中的世界面：弦在时空中运动扫出世界面（2 维曲面）。经典弦的运动方程恰是调和映射方程 \(\tau(f) = 0\)——世界面是时空中的极小曲面。Polyakov 作用量 \(S_P = \int |df|^2\) 的变分直接给出调和映射方程。

✏️ 例题分析与易错点

例题：catenoid 的极小性验证

题目：\(\mathbb{R}^3\) 中 catenoid \(r(u,v) = (\cosh u\cos v,\, \cosh u\sin v,\, u)\)，验证 \(H = 0\)。

第一步 — 第一基本形式：\(E = G = \cosh^2 u\)，\(F = 0\)。

第二步 — 第二基本形式：法向量 \(\nu = (-\cos v, -\sin v, \sinh u)/\cosh u\)。\(e = -\mathrm{sech}\, u\)，\(f = 0\)，\(g = \mathrm{sech}\, u\)。

第三步 — 平均曲率：\(H = \frac{eG - 2fF + gE}{2(EG - F^2)} = \frac{-\mathrm{sech}\, u \cdot \cosh^2 u + \mathrm{sech}\, u \cdot \cosh^2 u}{2\cosh^4 u} = 0\)。✔ 极小曲面。

常见错误

  • ❌ 认为极小曲面一定是面积最小的（全局极小）。
    ✅ 极小 = 面积的临界点（可能是鞍点）。catenoid 是极小曲面但不稳定——某些变分可以减小面积。稳定极小曲面需要第二变分 \(\geq 0\)。
  • ❌ 混淆能量泛函 \(E(f)\) 和面积泛函 \(A(f)\)。
    ✅ 共形映射下二者的临界点一致（2维），但一般不同。调和映射最小化能量，极小子流形最小化面积。等距浸入时 \(\tau = mH\)，两者等价。
  • ❌ 认为 Bernstein 定理在高维也成立（整体极小图 = 超平面）。
    ✅ Bernstein 定理在 \(\mathbb{R}^n\) 中仅对 \(n \leq 8\) 成立（Simons, Bombieri-De Giorgi-Giusti）。\(n \geq 9\) 存在非平面的完备极小图。
