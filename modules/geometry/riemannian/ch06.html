01子流形与诱导度量

🎯 为什么研究这个问题

将一个黎曼流形嵌入另一个（如曲面嵌入 \(\mathbb{R}^3\)）时，子流形 继承了环境空间的度量——诱导度量。子流形几何研究"弯曲空间中的 弯曲子集"：内在几何（Gauss 曲率）和外在几何（法向弯曲/第二 基本形式）如何交互。

🔍 抽象化方法

子流形 = 嵌入的流形。诱导度量 = 限制环境度量到切空间。 内在 vs 外在几何的区别是子流形理论的核心。

📐 公式推导 / 模型建立

等距嵌入：\(f: (M, g) \hookrightarrow (\bar{M}, \bar{g})\)，\(f^*\bar{g} = g\)。 \(f\) 是嵌入且保持度量。 诱导度量：\(M \subseteq \bar{M}\) 子流形，\(g = \bar{g}|_{TM \times TM}\)。 坐标中：若 \(f: U \to \bar{M}\)，\(f(u^1, \ldots, u^m)\)， \[g_{\alpha\beta} = \bar{g}_{ij} \frac{\partial f^i}{\partial u^\alpha}\frac{\partial f^j}{\partial u^\beta}\] 正交分解：\(T_p\bar{M} = T_pM \oplus N_pM\)（切空间 \(\oplus\) 法空间）， \(\bar{g}\)-正交。\(N_pM\) = 法丛在 \(p\) 处的纤维。 Gauss 公式：\(\bar{\nabla}_X Y = \nabla_X Y + \mathrm{I\!I}(X,Y)\) （环境协变导数 = 切向分量 + 法向分量）。

🌐 实际应用

- Nash 嵌入定理：任何黎曼流形都可以等距嵌入到足够高维的 \(\mathbb{R}^N\)

✏️ 例题分析与易错点

例题：\(S^2(R) \hookrightarrow \mathbb{R}^3\)。 参数化 \(f(\theta,\varphi) = (R\sin\theta\cos\varphi, R\sin\theta\sin\varphi, R\cos\theta)\)。 验证诱导度量。 解答：\(g_{\theta\theta} = R^2\)，\(g_{\varphi\varphi} = R^2\sin^2\theta\)，\(g_{\theta\varphi} = 0\)。 \(ds^2 = R^2(d\theta^2 + \sin^2\theta\,d\varphi^2)\)。✓ 与球面度量一致。 易错点：不同嵌入可能给出不同的诱导度量。Nash 嵌入定理的维数 \(N\) 可能很大（\(n\) 维流形需要 \(N \sim n^2\) 维空间）。

02第二基本形式

🎯 为什么研究这个问题

子流形的"弯曲方式"有两层含义：内在弯曲（Gauss 曲率）和外在弯曲 （它在环境空间中如何"翘起来"）。第二基本形式 \(\mathrm{I\!I}\) 精确 测量外在弯曲——它是 Gauss 公式中的法向分量。平面中的直线 \(\mathrm{I\!I} = 0\)，圆的 \(\mathrm{I\!I} \neq 0\)。

🔍 抽象化方法

第二基本形式 = \(\bar{\nabla}_X Y\) 的法向分量 = 子流形在环境空间中 的"加速度"。全测地子流形 (\(\mathrm{I\!I} = 0\)) 的测地线也是 环境空间的测地线。

📐 公式推导 / 模型建立

第二基本形式：\(\mathrm{I\!I}: \Gamma(TM) \times \Gamma(TM) \to \Gamma(NM)\)， \[\mathrm{I\!I}(X,Y) = (\bar{\nabla}_X Y)^\perp = \bar{\nabla}_X Y - \nabla_X Y\] \(\mathrm{I\!I}\) 是 \(C^\infty(M)\)-双线性对称的。 法曲率：选法方向 \(\nu\)，\(\mathrm{I\!I}_\nu(X,Y) = \bar{g}(\mathrm{I\!I}(X,Y), \nu)\)， 是 \(TM\) 上的对称双线性形式。 其特征值 \(\kappa_1, \ldots, \kappa_m\) = 主曲率（\(m = \dim M\)）。 平均曲率：\(H = \frac{1}{m}\mathrm{tr}_g \mathrm{I\!I} = \frac{1}{m}\sum \kappa_i \cdot \nu\)。 \(H = 0\) \(\iff\) 极小子流形（面积临界点）。 全测地子流形：\(\mathrm{I\!I} \equiv 0\)。\(M\) 的测地线 = \(\bar{M}\) 的测地线。 例：\(S^1\) 大圆是 \(S^2\) 中的全测地子流形。

🌐 实际应用

- 极小曲面（\(H = 0\)）在肥皂膜实验和材料科学中自然出现

✏️ 例题分析与易错点

例题：\(S^2(R) \hookrightarrow \mathbb{R}^3\)。法方向 \(\nu = (x,y,z)/R\)（外法向）。 求 \(\mathrm{I\!I}\) 和主曲率。 解答：\(\mathrm{I\!I}(X,Y) = -\frac{1}{R}g(X,Y)\nu\)（球面等距于 \(\kappa = 1/R\)）。 所有主曲率 \(= 1/R\)（脐点：所有方向曲率相等）。 \(H = 1/R\)，Gauss 曲率 \(K = \kappa_1 \kappa_2 = 1/R^2\)。 易错点：法方向约定（内/外法向）影响主曲率的符号。 约定外法向时，凸曲面主曲率为正。

03Gauss 方程与 Codazzi 方程

🎯 为什么研究这个问题

子流形的内在曲率（Gauss 曲率）和外在曲率（第二基本形式）如何 联系？Gauss 方程用 \(\mathrm{I\!I}\) 将子流形的 Riemann 曲率表示为 环境空间曲率 + 修正项。Codazzi 方程则约束 \(\mathrm{I\!I}\) 的 协变导数。两者是"子流形基本定理"的核心方程。

🔍 抽象化方法

Gauss 方程 = 内在曲率与外在曲率的关系式。 Codazzi 方程 = 第二基本形式的"可积性条件"。

📐 公式推导 / 模型建立

Gauss 方程： \[R(X,Y,Z,W) = \bar{R}(X,Y,Z,W) + \bar{g}(\mathrm{I\!I}(X,Z), \mathrm{I\!I}(Y,W)) - \bar{g}(\mathrm{I\!I}(X,W), \mathrm{I\!I}(Y,Z))\] 余维 1 特例（超曲面 \(M^{n-1} \subseteq \bar{M}^n\)）： \(\bar{M} = \mathbb{R}^n\)（\(\bar{R} = 0\)）时： \[K_M(\Pi) = \kappa_i \kappa_j\] （\(\kappa_i, \kappa_j\) = \(\Pi\) 方向的主曲率）。 Codazzi 方程（Codazzi-Mainardi）： \[(\bar{R}(X,Y)Z)^\perp = (\bar{\nabla}_X \mathrm{I\!I})(Y,Z) - (\bar{\nabla}_Y \mathrm{I\!I})(X,Z)\] \(\bar{M} = \mathbb{R}^n\) 时：\((\nabla_X \mathrm{I\!I})(Y,Z) = (\nabla_Y \mathrm{I\!I})(X,Z)\)。 子流形基本定理（陈述）：给定满足 Gauss、Codazzi（和 Ricci）方程的 度量 \(g\) 和对称张量 \(\mathrm{I\!I}\)，可以（局部地）将 \(M\) 等距嵌入 \(\bar{M}\)。

🌐 实际应用

- Gauss 方程推导经典的 Gauss Theorema Egregium（Gauss 曲率是内在量）

✏️ 例题分析与易错点

例题：\(M^2 \subseteq \mathbb{R}^3\)，主曲率 \(\kappa_1, \kappa_2\)。 验证 Gauss 方程给出 \(K = \kappa_1 \kappa_2\)。 解答：\(\bar{R} = 0\)（\(\mathbb{R}^3\) 平坦）。取主方向 \(e_1, e_2\)。 \(R(e_1, e_2, e_2, e_1) = 0 + \mathrm{I\!I}(e_1, e_2) \cdot \mathrm{I\!I}(e_2, e_1) - \mathrm{I\!I}(e_1, e_1) \cdot \mathrm{I\!I}(e_2, e_2)\)。 但 \(\mathrm{I\!I}(e_i, e_j) = \kappa_i \delta_{ij} \nu\)， 所以 \(R_{1221} = 0 - \kappa_1 \kappa_2 = -\kappa_1\kappa_2\)。 \(K = R_{1221}/(g_{11}g_{22}-g_{12}^2) = \kappa_1\kappa_2\)（符号约定调整后）。 易错点：Gauss 方程中的符号约定（\(R\) 的定义）在不同教材中不同， 可能差一个整体符号。

04极小曲面与调和映射

🎯 为什么研究这个问题

极小曲面（\(H = 0\)）是面积泛函的临界点——肥皂膜形成的曲面就是 极小曲面。它们是微分几何和变分学的经典研究对象。调和映射是 极小曲面的推广：\(f: (M,g) \to (N,h)\) 使能量泛函 \(E(f)\) 临界。 测地线、极小曲面、调和函数都是调和映射的特例。

🔍 抽象化方法

极小曲面 = 面积临界 = 平均曲率为零。 调和映射 = 能量临界 = 张力场为零。两者都是变分问题的解。

📐 公式推导 / 模型建立

极小曲面：\(M^m \hookrightarrow \bar{M}^n\)，\(H = 0\)。 等价地，\(M\) 是面积泛函 \(A(M) = \int_M dV\) 在保边界变分下的临界点。 第一变分公式（面积）： \(\delta A = -m\int_M g(H, V^\perp)\, dV\) （\(V\) = 变分场，\(V^\perp\) = 法向分量）。 \(\delta A = 0\) \(\forall V \iff H = 0\)。 调和映射：\(f: (M, g) \to (N, h)\)，\(E(f) = \frac{1}{2}\int_M |df|^2\, dV_g\)。 Euler-Lagrange 方程：张力场 \(\tau(f) = \mathrm{tr}_g(\nabla df) = 0\)。 坐标中：\(\tau^\alpha(f) = \Delta_g f^\alpha + g^{ij}\Gamma^\alpha_{\beta\gamma}(f) \frac{\partial f^\beta}{\partial x^i}\frac{\partial f^\gamma}{\partial x^j} = 0\)。 特例： - \(\dim M = 1\)：调和映射 = 测地线 - \(N = \mathbb{R}\)：调和映射 = 调和函数 \(\Delta f = 0\) - 等距浸入 \(f: M \to N\)，\(\tau(f) = mH\)：调和 \(\iff\) 极小

🌐 实际应用

- Plateau 问题：给定 \(\mathbb{R}^3\) 中的闭曲线，找面积最小的曲面 - 弦论中世界面的运动方程是调和映射方程

✏️ 例题分析与易错点

例题：\(\mathbb{R}^3\) 中的 catenoid \(r(u,v) = (\cosh u \cos v, \cosh u \sin v, u)\) 是极小曲面吗？ 解答：\(E = \cosh^2 u\)，\(F = 0\)，\(G = \cosh^2 u\)。 \(e = -\mathrm{sech}\,u\)（法曲率分量），\(f = 0\)，\(g = \mathrm{sech}\,u\)。 \(H = \frac{eG - 2fF + gE}{2(EG-F^2)} = \frac{-\mathrm{sech}\,u \cdot \cosh^2 u + \mathrm{sech}\,u \cdot \cosh^2 u}{2\cosh^4 u} = 0\)。 是极小曲面。 易错点：catenoid 是极小曲面但不是面积最小的（它只是临界点/鞍点）。
