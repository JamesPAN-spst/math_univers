黎曼度量

01黎曼度量的定义

🎯 为什么研究这个问题

光滑流形上可以定义切空间、微分形式，但无法测量长度和角度—— 因为流形自身没有"距离"的概念。黎曼度量 \(g\) 为每点切空间装配 了内积，由此可以定义曲线长度、测地线、面积、体积等所有度量几何 概念。这是从拓扑到几何的关键跨越。

🔍 抽象化方法

黎曼度量 = 切丛上的光滑正定对称 \((0,2)\)-张量场。 它将"光滑流形"提升为"黎曼流形"——有了度量才有几何。

📐 公式推导 / 模型建立

定义：黎曼度量 \(g\) 是 \(M\) 上的光滑 \((0,2)\)-张量场， 对每个 \(p \in M\)，\(g_p: T_pM \times T_pM \to \mathbb{R}\) 是 正定对称双线性形式（内积）。 坐标表示：\(g = g_{ij}(x)\, dx^i \otimes dx^j\)（Einstein 求和）。 矩阵 \((g_{ij}(x))\) 对称正定。 线元：\(ds^2 = g_{ij}\, dx^i\, dx^j\)。 存在性：任何（第二可数 Hausdorff）光滑流形都有黎曼度量 （用单位分解将局部欧式度量拼合）。 逆度量：\(g^{ij}\) 是 \((g_{ij})^{-1}\) 的分量。 升降指标：\(v^i = g^{ij} v_j\)，\(v_i = g_{ij} v^j\)。 \[g(u, v) = g_{ij} u^i v^j, \quad |v|_g = \sqrt{g(v,v)}\]

🌐 实际应用

- 广义相对论中时空度量 \(g_{\mu\nu}\) 是 Lorentz 签名的（不正定） - 机器学习中的信息几何：Fisher 信息矩阵是参数空间上的黎曼度量

✏️ 例题分析与易错点

例题：\(S^2\) 上球面度量。极坐标 \((\theta, \varphi)\)。 \(ds^2 = d\theta^2 + \sin^2\theta\, d\varphi^2\)。 求 \(\theta = \pi/4, \varphi = 0\) 处 \(v = \partial/\partial\varphi\) 的长度。 解答：\(|v|^2 = g_{\varphi\varphi} = \sin^2(\pi/4) = 1/2\)。 \(|v| = 1/\sqrt{2}\)。 易错点：在极坐标中 \(|\partial/\partial\varphi|\) 依赖于 \(\theta\) （赤道最长，极点为零），这反映了球面的几何。

02曲线长度、距离与等距

🎯 为什么研究这个问题

有了黎曼度量，就可以测量曲线长度 \(L(\gamma) = \int |\gamma'(t)|\,dt\)， 从而定义黎曼距离 \(d(p,q) = \inf_\gamma L(\gamma)\)。这个距离使 黎曼流形成为度量空间，其拓扑与原始流形拓扑一致。等距映射保持 距离——是黎曼几何的"同构"。

🔍 抽象化方法

黎曼度量 \(\to\) 曲线长度 \(\to\) 距离函数 \(\to\) 度量空间。 等距 = 保长度 = 保度量张量。

📐 公式推导 / 模型建立

曲线长度：\(\gamma: [a,b] \to M\)， \[L(\gamma) = \int_a^b \sqrt{g_{\gamma(t)}(\gamma'(t), \gamma'(t))}\, dt = \int_a^b |\gamma'(t)|_g\, dt\] 黎曼距离： \[d_g(p,q) = \inf\{L(\gamma) : \gamma \text{ 是从 } p \text{ 到 } q \text{ 的分段光滑曲线}\}\] \((M, d_g)\) 是度量空间，\(d_g\) 的诱导拓扑 = \(M\) 的流形拓扑。 等距 (Isometry)：\(f: (M,g) \to (N,h)\)，\(f^*h = g\)，即 \(h(df(u), df(v)) = g(u,v)\) \(\forall u, v\)。 等距 \(\Rightarrow\) 保距（\(d_h(f(p), f(q)) = d_g(p,q)\)）。 局部等距：\(f^*h = g\)，但 \(f\) 不一定全局双射 （如 \(\mathbb{R} \to S^1\) 的展开映射）。

🌐 实际应用

- GPS 定位利用地球表面的黎曼距离（大圆距离）计算

✏️ 例题分析与易错点

例题：\(S^2(R)\)（半径 \(R\)）上从北极到赤道的经线长度。 解答：\(\gamma(\theta) = (\theta, 0)\)，\(\theta \in [0, \pi/2]\)。 \(|\gamma'| = \sqrt{g_{\theta\theta}} = R\)（度量 \(ds^2 = R^2(d\theta^2 + \sin^2\theta\,d\varphi^2)\)）。 \(L = \int_0^{\pi/2} R\,d\theta = \pi R/2\)（球半周的四分之一）。 易错点：黎曼距离 \(d(N, \text{赤道上点})\) 等于 \(\pi R / 2\)（沿经线走最短）， 但欧式距离 \(= R\)。不要混淆内在距离与外在距离。

03体积形式与黎曼测度

🎯 为什么研究这个问题

黎曼度量不仅给出长度，还给出面积和体积——通过黎曼体积形式 \(dV_g = \sqrt{\det g}\, dx^1 \wedge \cdots \wedge dx^n\)。这是 在流形上做积分的正确方式，也是定义 Laplace 算子和热方程的基础。

🔍 抽象化方法

体积形式 = 度量决定的自然 \(n\)-形式（定向流形上）。 黎曼测度 = 将体积形式变为 Borel 测度。

📐 公式推导 / 模型建立

体积形式：\((M^n, g)\) 定向黎曼流形。 \[dV_g = \sqrt{\det(g_{ij})}\, dx^1 \wedge \cdots \wedge dx^n\] 在正交标架 \(\{e_i\}\) 下：\(dV_g = e^1 \wedge \cdots \wedge e^n\) （\(e^i\) 对偶余标架）。 体积：\(\mathrm{Vol}(M) = \int_M dV_g\)。 坐标变换不变性：\(\sqrt{\det g}\) 在坐标变换下的变换律精确 补偿了 \(dx\) 的变换，确保 \(dV_g\) 内在。 Hodge 星算子：\(*: \Omega^k(M) \to \Omega^{n-k}(M)\)， \(\alpha \wedge *\beta = g(\alpha, \beta)\, dV_g\)。 \(**\omega = (-1)^{k(n-k)} \omega\)（定向黎曼流形上）。 Laplacian：\(\Delta f = \mathrm{div}(\nabla f) = *d*df\) （= Laplace-Beltrami 算子）。

🌐 实际应用

- 热方程 \(\partial_t u = \Delta_g u\) 和波方程在黎曼流形上的推广 - 谱几何："能否听出鼓的形状？"（\(\Delta_g\) 的特征值 \(\leftrightarrow\) 几何）

✏️ 例题分析与易错点

例题：\(S^2(R)\)（半径 \(R\)）的面积。 解答：\(dV = R^2 \sin\theta\, d\theta \wedge d\varphi\)。 \(\mathrm{Area} = \int_0^{2\pi}\int_0^{\pi} R^2 \sin\theta\, d\theta\, d\varphi = R^2 \cdot 2\pi \cdot 2 = 4\pi R^2\)。 易错点：\(\sqrt{\det g} = R^2 \sin\theta\)（不是 \(R \sin\theta\) 或 \(\sin\theta\)）。 忘记 \(R^2\) 因子是常见错误。

04等距群与 Killing 向量场

🎯 为什么研究这个问题

黎曼流形的对称性由等距群 \(\mathrm{Isom}(M,g)\) 刻画——它是保持 度量的微分同胚的集合。等距群的无穷小版本是 Killing 向量场： 沿 Killing 场的流是等距变换。球面 \(S^n\) 的等距群是 \(O(n+1)\)， \(\mathbb{R}^n\) 的是欧式运动群——对称性越多几何结构越"规则"。

🔍 抽象化方法

Killing 方程 \(\mathcal{L}_X g = 0\) 将"无穷小对称"线性化。 Killing 场构成 Lie 代数，对应等距群的 Lie 代数。

📐 公式推导 / 模型建立

等距群：\(\mathrm{Isom}(M, g) = \{\varphi: M \to M \text{ 微分同胚} : \varphi^*g = g\}\)。 它是 Lie 群（Myers-Steenrod 定理）。 Killing 向量场：\(X \in \Gamma(TM)\)，\(\mathcal{L}_X g = 0\) （\(g\) 沿 \(X\) 的 Lie 导数为零）。 Killing 方程（坐标中）： \[\nabla_i X_j + \nabla_j X_i = 0\] Killing 场的空间是 Lie 代数 \(\mathfrak{isom}(M,g)\)，\(\dim \leq n(n+1)/2\)。 等号在常曲率空间（\(\mathbb{R}^n\), \(S^n\), \(\mathbb{H}^n\)）上成立。 例：\(\mathbb{R}^n\) 的 Killing 场 = 平移 + 旋转，\(\dim = n + \binom{n}{2} = n(n+1)/2\)。 \(S^2\) 的 Killing 场：\(\dim = 3\)（\(\mathfrak{so}(3)\)）。

🌐 实际应用

- 广义相对论中 Killing 向量场对应守恒量（Noether 定理） - 时间平移 Killing 场 → 能量守恒

✏️ 例题分析与易错点

例题：\(\mathbb{R}^2\) 上 \(X = -y\partial_x + x\partial_y\) 是 Killing 场吗？ 解答：\(g = dx^2 + dy^2\)（标准度量）。 \(\nabla_i X_j + \nabla_j X_i\)？平坦空间 \(\nabla = \partial\)。 \(X_1 = -y\), \(X_2 = x\)（降指标后）。 \(\partial_1 X_1 + \partial_1 X_1 = 0+0 = 0\)。 \(\partial_1 X_2 + \partial_2 X_1 = 1 + (-1) = 0\)。 \(\partial_2 X_2 + \partial_2 X_2 = 0+0 = 0\)。 所有分量为零，\(X\) 是 Killing 场。（它生成旋转。） 易错点：Killing 方程中用的是协变导数 \(\nabla_i\)（不是偏导数）， 但在平坦空间标准坐标中两者相同。
