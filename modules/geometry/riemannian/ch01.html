黎曼度量

01黎曼度量的定义

🎯 为什么研究这个问题

光滑流形赋予我们微分结构与切空间，却无法回答最基本的几何问题：两点间距离多远？两条曲线的夹角多大？一个区域的面积多少？黎曼度量正是弥补这一空缺的核心工具——它为每个切空间装配内积，使流形从"柔软的拓扑对象"升级为"刚性的几何对象"。

  • 从拓扑到几何的跨越：光滑流形具备切空间与微分结构，但无法测量长度、角度或面积。黎曼度量为每个切空间装配内积，是从拓扑流形进入度量几何的第一步。
  • 统一几何范式：球面几何、双曲几何、欧式几何都是黎曼度量的特例——一个框架统一处理所有弯曲空间。
  • 物理与应用驱动：广义相对论需要在弯曲时空上定义距离和曲率，信息几何需要在参数空间上度量统计距离——都依赖黎曼度量。

从拓扑到几何的跨越、从欧式到弯曲空间的统一、从纯数学到物理与信息科学的驱动，黎曼度量的引入标志着现代微分几何的真正起点。

🔍 抽象化方法
  • 张量场视角：黎曼度量 \(g\) 是切丛上的光滑截面 \(g \in \Gamma(S^2 T^*M)\)，每点 \(p\) 处 \(g_p\) 是正定对称双线性形式（内积）。
  • 从光滑流形到黎曼流形：加上度量后流形拥有了长度、角度、面积、体积、测地线、曲率等全部几何工具。
  • 范畴论：黎曼流形范畴的态射是等距映射（保度量的微分同胚），同构是等距同构。
📐 公式推导 / 模型建立

1. 黎曼度量的定义

\(g\) 是 \(M\) 上光滑 \((0,2)\)-张量场：对每个 \(p \in M\)，\(g_p: T_pM \times T_pM \to \mathbb{R}\) 满足对称性 \(g_p(u,v) = g_p(v,u)\) 和正定性 \(g_p(v,v) > 0\)（\(v \neq 0\)）。坐标表示：

\[g = g_{ij}(x)\, dx^i \otimes dx^j, \quad (g_{ij}(x)) \text{ 对称正定}\]

2. 线元与内积

线元 \(ds^2 = g_{ij}\, dx^i\, dx^j\) 给出无穷小距离的平方。切向量的长度和夹角：

\[|v|_g = \sqrt{g(v,v)}, \quad \cos\theta = \frac{g(u,v)}{|u|_g\, |v|_g}\]

3. 存在性（单位分解方法）

任何第二可数 Hausdorff 光滑流形都承认黎曼度量：在每个坐标邻域取标准欧式度量 \(\delta_{ij}\)，用单位分解 \(\{\rho_\alpha\}\) 拼合 \(g = \sum_\alpha \rho_\alpha g_\alpha\)。正定性在凸组合下保持。

4. 逆度量与升降指标

逆度量 \(g^{ij}\) 满足 \(g^{ik}g_{kj} = \delta^i_j\)。升降指标：\(v^i = g^{ij}v_j\)，\(\omega_i = g_{ij}\omega^j\)，建立 \(TM\) 与 \(T^*M\) 的同构（音乐同构 \(\sharp, \flat\)）。

🌐 实际应用

广义相对论中的时空度量：Einstein 的广义相对论将引力描述为时空弯曲。时空度量 \(g_{\mu\nu}\) 采用 Lorentz 签名 \((-,+,+,+)\)（伪黎曼度量），Schwarzschild 度量描述球对称黑洞外部时空，其中 \(g_{tt} = -(1 - 2GM/rc^2)\) 的零点对应事件视界。

信息几何与统计学习：参数空间 \(\Theta\) 上的 Fisher 信息矩阵 \(I_{ij}(\theta) = \mathbb{E}[\partial_i \log p \cdot \partial_j \log p]\) 构成天然的黎曼度量。正态分布族的 Fisher 度量给出双曲平面 \(\mathbb{H}^2\)，KL 散度在此度量下近似为测地距离的平方。

计算机图形学中的网格处理：三角网格上的离散黎曼度量用于纹理映射和参数化——将三维曲面展平到二维时，保角映射最小化角度畸变，保面积映射最小化面积畸变，两者都依赖度量张量的特征值分析。

✏️ 例题分析与易错点

例题：球面度量下切向量长度

题目：\(S^2\) 上球面度量 \(ds^2 = d\theta^2 + \sin^2\theta\, d\varphi^2\)。求 \(\theta = \pi/4, \varphi = 0\) 处切向量 \(v = \partial/\partial\varphi\) 的长度。

第一步 — 写出度量矩阵：\(g = \begin{pmatrix} 1 & 0 \\ 0 & \sin^2\theta \end{pmatrix}\)，在 \(\theta = \pi/4\) 处 \(g = \begin{pmatrix} 1 & 0 \\ 0 & 1/2 \end{pmatrix}\)。

第二步 — 计算长度：\(|v|^2 = g(\partial_\varphi, \partial_\varphi) = g_{\varphi\varphi} = \sin^2(\pi/4) = 1/2\)，故 \(|v| = 1/\sqrt{2}\)。

结论：在极坐标中 \(|\partial/\partial\varphi|\) 依赖于纬度 \(\theta\)——赤道处最大（\(= 1\)），极点处为零（坐标奇异），这反映了球面的内在几何。

常见错误

  • ❌ 用欧式内积计算球面上切向量长度（得 \(|\partial_\varphi| = 1\)）。
    ✅ 必须用球面度量 \(g_{\varphi\varphi} = \sin^2\theta\)，结果依赖 \(\theta\)。在 \(\theta = \pi/4\) 处 \(|\partial_\varphi| = 1/\sqrt{2}\)。
  • ❌ 混淆 \(\det g\) 和 \(\sqrt{\det g}\)：在面积公式中用了 \(\det g = \sin^2\theta\) 而非 \(\sqrt{\det g} = \sin\theta\)。
    ✅ 体积元为 \(dV = \sqrt{\det g}\, d\theta\, d\varphi = \sin\theta\, d\theta\, d\varphi\)。
  • ❌ 忘记黎曼度量的正定性要求，将 Lorentz 度量 \((-,+,+,+)\) 当作黎曼度量。
    ✅ 黎曼度量要求正定（\(g(v,v) > 0\)，\(v \neq 0\)）。Lorentz 签名属于伪黎曼几何范畴。

02曲线长度、距离与等距

🎯 为什么研究这个问题

有了度量张量之后，流形上自然产生的第一个量就是曲线长度——沿路径逐点用内积测量速度并积分。长度概念进而诱导出距离函数，使黎曼流形具备度量空间结构，而等距映射则成为度量几何中最自然的等价关系。

  • 度量几何的第一步：有了黎曼度量，自然要定义曲线长度 \(L(\gamma) = \int |\gamma'(t)|_g\, dt\) 和点间距离 \(d(p,q) = \inf_\gamma L(\gamma)\)。这使黎曼流形成为度量空间。
  • 拓扑与度量一致：黎曼距离诱导的度量拓扑恰好等于流形的原始拓扑——几何结构和拓扑结构是兼容的。
  • 等距——黎曼几何的同构：等距映射保持所有度量信息（距离、角度、面积），是黎曼几何中最自然的等价关系。

从度量到长度、再到距离和等距，这条线索构成了黎曼几何中从局部内积到全局度量空间的完整链条，也为后续的测地线理论和比较几何奠定了基础。

🔍 抽象化方法
  • 层级递进：黎曼度量 \(\to\) 曲线长度泛函 \(\to\) 黎曼距离函数 \(\to\) 度量空间结构。每一步都由前一步自然导出。
  • 映射保持结构：等距 = 保度量张量（\(f^*h = g\)）\(\Rightarrow\) 保距（\(d_h(f(p),f(q)) = d_g(p,q)\)），局部等距只要求 \(f^*h = g\) 但不要求全局双射。
📐 公式推导 / 模型建立

1. 曲线长度

\(\gamma: [a,b] \to M\) 是分段光滑曲线，其长度为：

\[L(\gamma) = \int_a^b \sqrt{g_{\gamma(t)}(\gamma'(t), \gamma'(t))}\, dt = \int_a^b |\gamma'(t)|_g\, dt\]

长度不依赖于参数化（重参数化不变性）。

2. 黎曼距离

定义距离函数：

\[d_g(p,q) = \inf\{L(\gamma) : \gamma \text{ 是从 } p \text{ 到 } q \text{ 的分段光滑曲线}\}\]

\((M, d_g)\) 是度量空间，且 \(d_g\) 的诱导拓扑与 \(M\) 的原始流形拓扑一致。

3. 等距与局部等距

\(f: (M,g) \to (N,h)\) 是等距 \(\iff\) \(f^*h = g\)（即 \(h(df(u), df(v)) = g(u,v)\)，\(\forall u, v\)）。等距 \(\Rightarrow\) 保距。局部等距只要求 \(f^*h = g\) 但 \(f\) 不一定全局双射（如 \(\mathbb{R} \to S^1\) 的卷绕映射）。

4. 能量泛函

曲线的能量泛函 \(E(\gamma) = \frac{1}{2}\int_a^b |\gamma'(t)|_g^2\, dt\)。Cauchy-Schwarz 给出 \(L(\gamma)^2 \leq 2(b-a)\, E(\gamma)\)，等号在常速参数化时成立。最小化能量等价于最小化长度（等速曲线中）。

🌐 实际应用

GPS 与大地测量：GPS 定位利用地球表面的黎曼距离（大圆距离/Vincenty 公式）。两点间的测地线距离不同于直线欧式距离——在跨洋航线中，沿大圆弧飞行比沿纬线飞行更短。

最优传输与 Wasserstein 距离：概率测度空间上的 Wasserstein 距离 \(W_2(\mu,\nu)\) 可理解为无穷维黎曼流形（概率空间）上的测地距离（Otto 计算），这是最优传输理论与黎曼几何的深层联系。

自动驾驶路径规划：在具有不均匀代价函数的地形上，将通行代价编码为黎曼度量，最优路径就是测地线——Eikonal 方程 \(|\nabla u|_g = 1\) 的解给出距离函数。

✏️ 例题分析与易错点

例题：球面上经线长度

题目：求 \(S^2(R)\)（半径 \(R\)）上从北极到赤道的经线长度。

第一步 — 参数化曲线：\(\gamma(\theta) = (\theta, 0)\)，\(\theta \in [0, \pi/2]\)。度量 \(ds^2 = R^2(d\theta^2 + \sin^2\theta\, d\varphi^2)\)。

第二步 — 计算速度：\(\gamma'(\theta) = \partial_\theta\)，\(|\gamma'|^2 = g_{\theta\theta} = R^2\)，故 \(|\gamma'| = R\)。

第三步 — 积分：\(L = \int_0^{\pi/2} R\, d\theta = \pi R / 2\)。

结论：北极到赤道的黎曼距离为 \(\pi R/2\)，而欧式距离为 \(R\)。二者的差异体现了内在距离（沿曲面走）与外在距离（穿过环境空间）的根本区别。

常见错误

  • ❌ 混淆黎曼距离与欧式距离：认为球面上两点间距离等于三维空间中的直线段长。
    ✅ 黎曼距离是沿流形内部路径的最短长度，必须在曲面上走。\(d_g = \pi R/2 \neq R = d_{\text{Euclid}}\)。
  • ❌ 忘记长度的参数化无关性：用不同参数化得到不同长度。
    ✅ \(L(\gamma)\) 不依赖于参数化。但能量泛函 \(E(\gamma)\) 依赖参数化——仅在常速参数化下 \(E\) 与 \(L^2\) 成比例。
  • ❌ 将局部等距当作全局等距：认为 \(f^*h = g\) 蕴含 \(f\) 是微分同胚。
    ✅ 局部等距不一定是双射。如 \(\mathbb{R} \to S^1\)（\(t \mapsto e^{it}\)）满足 \(f^*g_{S^1} = dt^2 = g_{\mathbb{R}}\)，但不是全局同构。

03体积形式与黎曼测度

🎯 为什么研究这个问题

度量不仅赋予长度和角度，还自然地赋予面积和体积——黎曼体积形式将流形上的积分建立在坐标无关的坚实基础上，使得热方程、谱理论和几何测度论等分析工具能够在弯曲空间中运作。

  • 从长度到体积：黎曼度量不仅给出长度，还给出面积和体积——通过黎曼体积形式 \(dV_g\)，这是在流形上做积分的正确方式。
  • 分析工具的基础：Laplace-Beltrami 算子 \(\Delta_g\)、热方程、Sobolev 空间等分析工具都建立在黎曼测度之上。
  • 谱几何："听出鼓的形状"：\(\Delta_g\) 的特征值谱编码了流形的几何信息（Weyl 渐近公式），连接分析与几何。

体积形式、Hodge 星算子与 Laplace-Beltrami 算子共同构成了黎曼流形上分析工具的基本框架，将微分几何与偏微分方程、谱理论深度联结。

🔍 抽象化方法
  • 体积形式：度量决定的自然 \(n\)-形式（定向流形上），坐标无关地定义积分。
  • Hodge 对偶：Hodge 星算子 \(*\) 利用度量和定向将 \(k\)-形式映射到 \((n-k)\)-形式，是定义 Laplacian 和上同调对偶的关键。
📐 公式推导 / 模型建立

1. 黎曼体积形式

\((M^n, g)\) 定向黎曼流形，体积形式为：

\[dV_g = \sqrt{\det(g_{ij})}\, dx^1 \wedge \cdots \wedge dx^n\]

在正交标架 \(\{e_i\}\) 下简化为 \(dV_g = e^1 \wedge \cdots \wedge e^n\)。体积 \(\mathrm{Vol}(M) = \int_M dV_g\)。

2. 坐标变换不变性

\(\sqrt{\det g}\) 在坐标变换下的变换律精确补偿了 \(dx\) 的变换（Jacobian 因子），确保 \(dV_g\) 是坐标无关的内在量。

3. Hodge 星算子

\(*: \Omega^k(M) \to \Omega^{n-k}(M)\)，定义为 \(\alpha \wedge *\beta = g(\alpha, \beta)\, dV_g\)。性质：\(**\omega = (-1)^{k(n-k)}\omega\)（定向黎曼流形上）。

4. Laplace-Beltrami 算子

\(\Delta_g f = \mathrm{div}(\nabla f) = *d*df = \frac{1}{\sqrt{\det g}}\partial_i(\sqrt{\det g}\, g^{ij}\partial_j f)\)。在 \(\mathbb{R}^n\) 上退化为 \(\sum_i \partial_i^2 f\)。

🌐 实际应用

流形上的热方程与扩散：热方程 \(\partial_t u = \Delta_g u\) 描述流形上的热传导。热核 \(K(t,x,y)\) 的短时渐近展开 \(K(t,x,y) \sim (4\pi t)^{-n/2} e^{-d(x,y)^2/4t} \sum a_k t^k\) 中系数 \(a_k\) 编码了曲率信息。

谱几何与 Kac 问题："能否听出鼓的形状？"——\(\Delta_g\) 的特征值谱 \(0 = \lambda_0 < \lambda_1 \leq \lambda_2 \leq \cdots\) 决定了 Weyl 渐近：\(N(\lambda) \sim C_n\, \mathrm{Vol}(M)\, \lambda^{n/2}\)。谱可以"听到"维数和体积，但不能完全确定形状（存在等谱不等距的流形对）。

几何测度论与极小曲面：黎曼测度是研究流形上子集面积（Hausdorff 测度）和极小曲面（面积泛函临界点）的基础，在肥皂泡、黑洞视界等物理问题中有直接应用。

✏️ 例题分析与易错点

例题：球面面积

题目：计算 \(S^2(R)\)（半径 \(R\)）的面积。

第一步 — 写出体积元：球面度量 \(g = R^2 d\theta^2 + R^2\sin^2\theta\, d\varphi^2\)，\(\det g = R^4\sin^2\theta\)，\(dV = R^2\sin\theta\, d\theta\, d\varphi\)。

第二步 — 积分：\(\mathrm{Area} = \int_0^{2\pi}\int_0^\pi R^2\sin\theta\, d\theta\, d\varphi = R^2 \cdot 2\pi \cdot 2 = 4\pi R^2\)。

结论：\(S^2(R)\) 的面积为 \(4\pi R^2\)，与初等几何结果一致，验证了黎曼体积形式公式的正确性。

常见错误

  • ❌ 计算 \(\sqrt{\det g}\) 时忘记取根号：用 \(R^4\sin^2\theta\) 代替 \(R^2\sin\theta\)。
    ✅ 体积元是 \(\sqrt{\det g}\, d\theta\, d\varphi = R^2\sin\theta\, d\theta\, d\varphi\)，必须先取行列式再开方。
  • ❌ 对非定向流形直接定义体积形式。
    ✅ 体积形式需要定向。非定向流形（如 Klein 瓶）上可定义黎曼密度（\(|dV_g|\)）代替体积形式来做积分。
  • ❌ 混淆 Hodge 星算子的作用方向：以为 \(*\) 将 \(k\)-形式映到 \(k\)-形式。
    ✅ \(*: \Omega^k \to \Omega^{n-k}\)。例如 \(n = 3\) 时 \(*dx = dy \wedge dz\)（1-形式变为 2-形式）。

04等距群与 Killing 向量场

🎯 为什么研究这个问题

对称性是理解几何空间最有力的视角。黎曼流形的对称性由等距群精确刻画——它是一个 Lie 群，其 Lie 代数恰好是 Killing 向量场空间。从 Noether 定理到宇宙学模型，对称性分析贯穿了黎曼几何与理论物理的深层联系。

  • 对称性刻画几何：黎曼流形的对称性由等距群 \(\mathrm{Isom}(M,g)\) 刻画——球面 \(S^n\) 有最大对称性 \(O(n+1)\)，一般流形的等距群较小甚至平凡。
  • 无穷小对称：Killing 向量场是等距群的无穷小版本（Lie 代数），满足 \(\mathcal{L}_X g = 0\)——沿 Killing 场的流保持度量。
  • 守恒量与物理：Noether 定理将每个 Killing 场对应一个守恒量，时间平移 → 能量守恒，空间旋转 → 角动量守恒。

等距群与 Killing 向量场将流形的对称性从直觉提升为精确的代数结构，也为物理中的守恒律和高对称空间的分类提供了数学基础。

🔍 抽象化方法
  • Lie 群与 Lie 代数：等距群 \(\mathrm{Isom}(M,g)\) 是 Lie 群（Myers-Steenrod 定理），其 Lie 代数是 Killing 向量场空间 \(\mathfrak{isom}(M,g)\)。
  • 维数上界：Killing 场空间 \(\dim \leq n(n+1)/2\)，等号仅在常曲率空间上成立（最大对称空间）。
📐 公式推导 / 模型建立

1. 等距群

\(\mathrm{Isom}(M,g) = \{\varphi: M \to M \text{ 微分同胚} : \varphi^*g = g\}\) 是 Lie 群（Myers-Steenrod 定理）。\(\dim \mathrm{Isom} \leq n(n+1)/2\)。

2. Killing 方程

\(X\) 是 Killing 向量场 \(\iff\) \(\mathcal{L}_X g = 0\)。坐标形式：

\[\nabla_i X_j + \nabla_j X_i = 0\]

这是关于 \(X\) 的线性一阶 PDE 系统。初值 \((X(p), \nabla X(p))\) 唯一决定 Killing 场（若存在），故 \(\dim \mathfrak{isom} \leq n + \binom{n}{2} = n(n+1)/2\)。

3. 经典例子

\(\mathbb{R}^n\)：Killing 场 = 平移 + 旋转，\(\dim = n(n+1)/2\)。\(S^n\)：\(\mathrm{Isom} = O(n+1)\)，\(\dim = n(n+1)/2\)。一般黎曼流形的等距群可以很小（甚至平凡）。

4. Killing 场的性质

Killing 场在 Lie 括号下封闭：\([X, Y]\) 也是 Killing 场。沿测地线 \(\gamma\)，量 \(g(X, \gamma')\) 是常数（守恒量）。如果 \(K\) 为截面曲率且处处为常数，则 \(M\) 具有最大对称性。

🌐 实际应用

广义相对论中的守恒量：Killing 向量场通过 Noether 定理对应守恒量。Schwarzschild 时空有时间平移 Killing 场 \(\partial_t\)（对应能量守恒）和三个旋转 Killing 场（对应角动量守恒）。Kerr 黑洞只有 \(\partial_t\) 和 \(\partial_\varphi\) 两个 Killing 场。

齐性空间与对称空间：等距群传递作用的空间（齐性空间 \(G/H\)）在粒子物理和宇宙学中广泛出现。FLRW 宇宙模型假设空间截面是最大对称的（\(S^3\)、\(\mathbb{R}^3\) 或 \(\mathbb{H}^3\)），对应正/零/负曲率宇宙。

几何流中的对称性简化：Ricci 流 \(\partial_t g = -2\mathrm{Ric}\) 保持等距群——如果初始度量有对称性，演化后的度量也有。这大大简化了高对称空间上 Ricci 流的分析。

✏️ 例题分析与易错点

例题：验证旋转生成元是 Killing 场

题目：\(\mathbb{R}^2\) 上标准度量 \(g = dx^2 + dy^2\)。验证 \(X = -y\,\partial_x + x\,\partial_y\) 是 Killing 场。

第一步 — 降指标：平坦空间中 \(X_1 = g_{1j}X^j = -y\)，\(X_2 = g_{2j}X^j = x\)。

第二步 — 验证 Killing 方程：\(\nabla = \partial\)（平坦空间）。\(\partial_1 X_1 + \partial_1 X_1 = 0\)，\(\partial_1 X_2 + \partial_2 X_1 = 1 + (-1) = 0\)，\(\partial_2 X_2 + \partial_2 X_2 = 0\)。全部分量为零。✔

结论：\(X\) 是 Killing 场，它生成绕原点的旋转。其流为 \(\varphi_t(x,y) = (x\cos t - y\sin t,\, x\sin t + y\cos t)\)。

常见错误

  • ❌ 在弯曲空间中用偏导数 \(\partial_i\) 代替协变导数 \(\nabla_i\) 验证 Killing 方程。
    ✅ Killing 方程 \(\nabla_i X_j + \nabla_j X_i = 0\) 中必须用协变导数。平坦空间标准坐标中 \(\Gamma = 0\)，两者相同，但弯曲空间中有 \(\Gamma^k_{ij}\) 修正项。
  • ❌ 认为 Killing 场的存在性不受曲率限制：任何流形都有 \(n(n+1)/2\) 个 Killing 场。
    ✅ 这是上界。一般黎曼流形可能没有任何非平凡 Killing 场（\(\dim \mathfrak{isom} = 0\)）。
  • ❌ 混淆等距群与共形群：认为保角映射也是等距。
    ✅ 共形变换 \(\varphi^*g = e^{2\sigma}g\) 保角但不保距。等距是共形变换的特例（\(\sigma = 0\)）。
