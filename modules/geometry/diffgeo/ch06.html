联络与曲率初步

01仿射联络与 Christoffel 符号

🎯 为什么研究这个问题

在 \(\mathbb{R}^n\) 中，我们可以直接比较不同点的切向量（平行平移）。 但在弯曲流形上，不同点的切空间没有自然的对应关系。仿射联络 \(\nabla\) 填补了这个空白——它定义了如何"沿曲线运输"切向量。 Christoffel 符号是联络在坐标卡中的具体表显，是计算的基本工具。

🔍 抽象化方法

联络 = 在向量丛上定义"导数"的额外结构。它不来自流形本身， 而是额外选取的数据（直到有度量时才有自然的选取——Levi-Civita）。

📐 公式推导 / 模型建立

仿射联络：\(\nabla: \Gamma(TM) \times \Gamma(TM) \to \Gamma(TM)\)， \((X, Y) \mapsto \nabla_X Y\)，满足： (1) \(\nabla_{fX+gY} Z = f\nabla_X Z + g\nabla_Y Z\)（\(C^\infty\)-线性于 \(X\)） (2) \(\nabla_X(Y+Z) = \nabla_X Y + \nabla_X Z\) (3) \(\nabla_X(fY) = (Xf)Y + f\nabla_X Y\)（Leibniz 法则于 \(Y\)） Christoffel 符号：坐标中 \(\nabla_{\partial_i} \partial_j = \Gamma^k_{ij} \partial_k\)。 对一般 \(Y = Y^j \partial_j\)： \[(\nabla_X Y)^k = X^i \frac{\partial Y^k}{\partial x^i} + \Gamma^k_{ij} X^i Y^j\] \(\Gamma^k_{ij}\) 共 \(n^3\) 个分量（\(n = \dim M\)）。 \(\Gamma\) 不是张量（坐标变换时变换律含非齐次项）。

🌐 实际应用

- 广义相对论中 Christoffel 符号编码引力场的强度

✏️ 例题分析与易错点

例题：\(\mathbb{R}^2\) 极坐标 \((r, \theta)\)。欧式度量 \(ds^2 = dr^2 + r^2 d\theta^2\)。求 \(\Gamma^r_{\theta\theta}\)。 解答：\(g_{rr} = 1\)，\(g_{\theta\theta} = r^2\)，其余 \(= 0\)。 \(\Gamma^r_{\theta\theta} = -\frac{1}{2} g^{rr} \frac{\partial g_{\theta\theta}}{\partial r} = -\frac{1}{2} \cdot 1 \cdot 2r = -r\)。 物理意义：圆周方向运动产生向心加速度项。 易错点：极坐标中 \(\Gamma\) 不全为零——即使空间是平坦的， 弯曲的坐标系也会产生非零 Christoffel 符号。

02协变导数与平行移动

🎯 为什么研究这个问题

普通导数 \(dY/dt\) 沿曲线对向量场求导在弯曲流形上没有意义（不同 点的切空间不能直接相减）。协变导数 \(\nabla_{\gamma'} Y\) 利用联络 定义了沿曲线的"正确导数"。平行移动则是"导数为零"的运输—— 保持向量"平行"地沿曲线滑动。平行移动的路径依赖性就是曲率。

🔍 抽象化方法

协变导数 = 联络沿曲线的限制。平行移动 = 联络定义的"搬运方式"。

📐 公式推导 / 模型建立

协变导数沿曲线：设 \(\gamma: I \to M\)，\(Y\) 是沿 \(\gamma\) 的 向量场（\(Y(t) \in T_{\gamma(t)}M\)）。 \[\frac{DY}{dt} = \nabla_{\gamma'(t)} Y = \left(\frac{dY^k}{dt} + \Gamma^k_{ij} \dot{\gamma}^i Y^j\right) \partial_k\] 平行移动：\(Y\) 沿 \(\gamma\) 平行 \(\iff \frac{DY}{dt} = 0\)，即 \[\frac{dY^k}{dt} + \Gamma^k_{ij} \dot{\gamma}^i Y^j = 0\] 这是一阶线性 ODE，给定初始 \(Y(0) \in T_{\gamma(0)}M\) 后 解唯一存在。 平行移动的路径依赖：从 \(p\) 沿不同路径移动到 \(q\)， 平行移动的结果一般不同——差异由曲率测量。 在平坦空间中，平行移动与路径无关。

🌐 实际应用

- 广义相对论中自由粒子沿测地线运动 = 四速度平行移动 - 陀螺仪的进动效应与弯曲空间中的平行移动有关

✏️ 例题分析与易错点

例题：\(S^2\) 上从北极出发，沿经线到赤道，再沿赤道走 \(90°\)， 再沿经线回北极。初始向南的切向量经过平行移动后旋转了多少度？ 解答：这是经典的"球面三角形平行移动"。 三角形三个角都是 \(90°\)，内角和 \(= 270°\)，角亏 \(= 90°\)。 向量旋转角度 = 面积 / \(R^2 = (\pi/2)/1 = \pi/2 = 90°\)。 （Gauss-Bonnet 的体现：曲率导致平行移动的路径依赖。） 易错点：平坦空间（如 \(\mathbb{R}^2\)）中平行移动与路径无关， 但弯曲空间中一定有路径依赖——这正是"曲率"的直觉来源。

03挠率与曲率张量

🎯 为什么研究这个问题

联络有两个基本的"障碍量"：挠率测量"平行四边形是否闭合"， 曲率测量"平行移动的路径依赖性"。两者都是张量，是联络的 内在不变量。无挠条件在黎曼几何中自然满足（Levi-Civita 联络）， 但在某些物理理论（Einstein-Cartan 理论）中挠率非零。

🔍 抽象化方法

挠率和曲率分别是联络"一阶"和"二阶"的特征： 挠率涉及 \(\nabla_X Y - \nabla_Y X\)，曲率涉及 \(\nabla_X \nabla_Y - \nabla_Y \nabla_X\)。

📐 公式推导 / 模型建立

挠率张量： \[T(X, Y) = \nabla_X Y - \nabla_Y X - [X, Y]\] 坐标中 \(T^k_{ij} = \Gamma^k_{ij} - \Gamma^k_{ji}\)。 \(T = 0 \iff \Gamma^k_{ij} = \Gamma^k_{ji}\)（对称联络）。 Riemann 曲率张量： \[R(X,Y)Z = \nabla_X \nabla_Y Z - \nabla_Y \nabla_X Z - \nabla_{[X,Y]} Z\] 坐标中： \[R^l{}_{kij} = \partial_i \Gamma^l_{jk} - \partial_j \Gamma^l_{ik} + \Gamma^l_{im} \Gamma^m_{jk} - \Gamma^l_{jm} \Gamma^m_{ik}\] \(R = 0 \iff\) 平行移动与路径无关 \(\iff\) 局部可以找到"平行坐标 系"使 \(\Gamma \equiv 0\)。 Bianchi 恒等式（第一）：无挠时 \(R(X,Y)Z + R(Y,Z)X + R(Z,X)Y = 0\)。

🌐 实际应用

- Einstein 场方程 \(R_{\mu\nu} - \frac{1}{2}Rg_{\mu\nu} = 8\pi T_{\mu\nu}\) 中曲率编码引力场

✏️ 例题分析与易错点

例题：\(\mathbb{R}^n\)（标准联络，\(\Gamma = 0\)）的曲率张量是什么？ 解答：\(R^l_{kij} = \partial_i(0) - \partial_j(0) + 0 - 0 = 0\)。 \(\mathbb{R}^n\) 是平坦的，曲率为零。 易错点：极坐标中 \(\Gamma \neq 0\)，但 \(R = 0\)（\(\mathbb{R}^2\) 仍是平坦的）。 \(\Gamma = 0\) 是坐标依赖的，但 \(R = 0\) 是内在性质。

04Levi-Civita 联络

🎯 为什么研究这个问题

仿射联络有很多种选择，哪个是"最自然"的？在黎曼流形上， Levi-Civita 联络由两个条件唯一确定：无挠 + 度量兼容。这是 黎曼几何的基本定理，保证了测地线、曲率等概念都有唯一的 标准定义。

🔍 抽象化方法

Levi-Civita 联络是度量决定的唯一联络——几何内在地确定了 如何"平行搬运"。

📐 公式推导 / 模型建立

条件：\((M, g)\) 黎曼流形，\(\nabla\) 满足： (1) 无挠：\(T = 0\)，即 \(\nabla_X Y - \nabla_Y X = [X, Y]\) (2) 度量兼容：\(\nabla g = 0\)，即 \(Xg(Y,Z) = g(\nabla_X Y, Z) + g(Y, \nabla_X Z)\) 存在唯一性（Levi-Civita 基本定理）： 由 Koszul 公式唯一确定 \(\nabla\)： \[2g(\nabla_X Y, Z) = Xg(Y,Z) + Yg(X,Z) - Zg(X,Y)\] \[+ g([X,Y],Z) - g([X,Z],Y) - g([Y,Z],X)\] Christoffel 符号： \[\Gamma^k_{ij} = \frac{1}{2} g^{kl}\left(\frac{\partial g_{jl}}{\partial x^i} + \frac{\partial g_{il}}{\partial x^j} - \frac{\partial g_{ij}}{\partial x^l}\right)\]

🌐 实际应用

- 黎曼几何中所有标准概念（测地线、曲率、Jacobi 场）都基于 Levi-Civita 联络 - 广义相对论中时空的 Levi-Civita 联络决定引力场

✏️ 例题分析与易错点

例题：\(S^2\) 上球面度量 \(ds^2 = d\theta^2 + \sin^2\theta\,d\varphi^2\)。 求 \(\Gamma^\theta_{\varphi\varphi}\)。 解答：\(g_{\theta\theta} = 1\)，\(g_{\varphi\varphi} = \sin^2\theta\)， \(g^{\theta\theta} = 1\)。 \(\Gamma^\theta_{\varphi\varphi} = -\frac{1}{2}g^{\theta\theta}\frac{\partial g_{\varphi\varphi}}{\partial\theta} = -\frac{1}{2} \cdot 2\sin\theta\cos\theta = -\sin\theta\cos\theta\)。 易错点：忘记 \(g^{kl}\) 是逆度量（不是 \(g_{kl}\)）。对对角度量 \(g^{kk} = 1/g_{kk}\)。
