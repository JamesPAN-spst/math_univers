01余切空间与微分形式

🎯 为什么研究这个问题

向量场"有方向"，但物理中许多对象（如功、通量）是"吃掉方向给出 数值"的——它们不是向量而是余向量。余切空间 \(T_p^*M\) 是切空间的 对偶，\(k\)-形式则是反对称多重线性映射。微分形式是积分和 Stokes 定理的自然语言——在流形上，"被积对象"不是函数而是形式。

🔍 抽象化方法

微分形式 = 余切丛的反对称截面。\(k\)-形式"吃"\(k\) 个切向量 输出标量，是流形上积分的正确推广。

📐 公式推导 / 模型建立

余切空间：\(T_p^*M = (T_pM)^* = \mathrm{Hom}(T_pM, \mathbb{R})\)。 坐标基 \(\{dx^1|_p, \ldots, dx^n|_p\}\) 对偶于 \(\{\partial/\partial x^i|_p\}\)： \(dx^i(\partial/\partial x^j) = \delta^i_j\)。 \(k\)-形式：\(\omega \in \Omega^k(M)\) 是光滑截面 \(\omega_p: \underbrace{T_pM \times \cdots \times T_pM}_{k} \to \mathbb{R}\)， 反对称多重线性。 局部表示： \[\omega = \sum_{i_1 < \cdots < i_k} \omega_{i_1 \cdots i_k}(x)\, dx^{i_1} \wedge \cdots \wedge dx^{i_k}\] 特例： - \(0\)-形式 = 光滑函数 \(f \in C^\infty(M)\) - \(1\)-形式 \(\omega = \sum \omega_i dx^i\)（"余向量场"） - \(n\)-形式 = 体积形式（\(n = \dim M\)）

🌐 实际应用

- 热力学中 \(\delta W = P\,dV\)，\(\delta Q = T\,dS\) 是 1-形式 - 电磁场张量 \(F = E_i dx^i \wedge dt + B_{ij} dx^i \wedge dx^j\) 是 2-形式

✏️ 例题分析与易错点

例题：\(\mathbb{R}^3\) 上 \(\omega = x\,dy \wedge dz + y\,dz \wedge dx + z\,dx \wedge dy\)。 求 \(\omega_{(1,0,0)}(\partial/\partial y, \partial/\partial z)\)。 解答：在 \((1,0,0)\) 处，\(\omega = 1 \cdot dy \wedge dz + 0 + 0 = dy \wedge dz\)。 \(dy \wedge dz(\partial/\partial y, \partial/\partial z) = dy(\partial/\partial y) \cdot dz(\partial/\partial z) - dy(\partial/\partial z) \cdot dz(\partial/\partial y) = 1 \cdot 1 - 0 \cdot 0 = 1\)。 易错点：反对称性——交换两个输入变号： \(\omega(Y, X) = -\omega(X, Y)\)。

02外积与外微分

🎯 为什么研究这个问题

如何将两个微分形式"乘"在一起？如何对微分形式"求导"？外积 \(\wedge\) 是反对称的乘法运算，外微分 \(d\) 是将 \(k\)-形式映为 \((k+1)\)-形式 的"万能导数"。\(d^2 = 0\) 这个简洁的恒等式是 de Rham 上同调的 基础，也是 Maxwell 方程内禀形式的核心。

🔍 抽象化方法

外代数 \(\Omega^*(M) = \bigoplus_k \Omega^k(M)\) 配合 \(\wedge\) 和 \(d\) 构成微分分次代数。\(d^2 = 0\) 产生链复形结构。

📐 公式推导 / 模型建立

外积：\(\wedge: \Omega^k(M) \times \Omega^l(M) \to \Omega^{k+l}(M)\)， 反对称性 \(\alpha \wedge \beta = (-1)^{kl} \beta \wedge \alpha\)。 基元素：\(dx^{i_1} \wedge \cdots \wedge dx^{i_k}\)。 外微分：\(d: \Omega^k(M) \to \Omega^{k+1}(M)\)，唯一满足： (1) \(df\) 是 \(f\) 的全微分（\(k=0\) 时） (2) \(d(\alpha \wedge \beta) = d\alpha \wedge \beta + (-1)^k \alpha \wedge d\beta\) (3) \(d^2 = 0\) 坐标中：\(\omega = \sum \omega_I dx^I\)， \[d\omega = \sum_I \sum_j \frac{\partial \omega_I}{\partial x^j} dx^j \wedge dx^I\] \(d^2 = 0\) 的证明：对函数 \(f\)， \(d(df) = d\left(\sum \frac{\partial f}{\partial x^i} dx^i\right) = \sum_{j,i} \frac{\partial^2 f}{\partial x^j \partial x^i} dx^j \wedge dx^i = 0\) （\(\frac{\partial^2 f}{\partial x^j \partial x^i}\) 关于 \(i,j\) 对称， 但 \(dx^j \wedge dx^i\) 反对称）。一般情况由 Leibniz 法则归纳。

🌐 实际应用

- Maxwell 方程可写为 \(dF = 0\)，\(d{*F} = J\)（\(F\) 为电磁 2-形式）

✏️ 例题分析与易错点

例题：\(\omega = x\,dy - y\,dx\)（\(\mathbb{R}^2\) 上 1-形式）。求 \(d\omega\)。 解答：\(d\omega = dx \wedge dy - dy \wedge dx = dx \wedge dy + dx \wedge dy = 2\,dx \wedge dy\)。 （因为 \(d(x\,dy) = dx \wedge dy\)，\(d(-y\,dx) = -dy \wedge dx = dx \wedge dy\)。） 易错点：\(d(-y\,dx) = -dy \wedge dx\)，然后利用反对称性 \(-dy \wedge dx = +dx \wedge dy\)。符号处理是常见出错点。

03拉回与闭/恰当形式

🎯 为什么研究这个问题

如何将目标空间上的微分形式"拉回"到源空间？拉回 \(f^*\) 将 \(N\) 上的形式变为 \(M\) 上的形式，且与 \(d\) 交换。闭形式（\(d\omega = 0\)） 与恰当形式（\(\omega = d\eta\)）的关系是拓扑信息的载体： 闭但不恰当的形式检测出空间中的"洞"。

🔍 抽象化方法

拉回 \(f^*\) 是微分形式层面的函子操作，保持外积和外微分。 \(\ker d / \mathrm{im}\, d\) 就是 de Rham 上同调——纯拓扑不变量。

📐 公式推导 / 模型建立

拉回：\(f: M \to N\) 光滑，\(\omega \in \Omega^k(N)\)。 \((f^*\omega)_p(v_1, \ldots, v_k) = \omega_{f(p)}(df_p(v_1), \ldots, df_p(v_k))\)。 性质： - \(f^*(\alpha \wedge \beta) = f^*\alpha \wedge f^*\beta\) - \(f^* \circ d = d \circ f^*\)（与外微分交换） - \((g \circ f)^* = f^* \circ g^*\) 闭形式：\(\omega\) 闭 \(\iff d\omega = 0\)。 恰当形式：\(\omega\) 恰当 \(\iff \exists \eta,\; \omega = d\eta\)。 \(d^2 = 0 \Rightarrow\) 恰当 \(\Rightarrow\) 闭（\(d(d\eta) = 0\)）。 反之不一定成立（取决于拓扑）。 \[\text{恰当} \subseteq \text{闭}, \quad H^k_{dR}(M) = \frac{\ker(d: \Omega^k \to \Omega^{k+1})} {\mathrm{im}(d: \Omega^{k-1} \to \Omega^k)}\]

🌐 实际应用

- \(\omega = \frac{-y\,dx + x\,dy}{x^2+y^2}\) 在 \(\mathbb{R}^2 \setminus \{0\}\) 上闭但不恰当——检测出原点处的"洞"

✏️ 例题分析与易错点

例题：\(\omega = \frac{-y\,dx + x\,dy}{x^2 + y^2}\) 在 \(\mathbb{R}^2 \setminus \{0\}\) 上。 验证 \(d\omega = 0\) 但 \(\omega\) 不恰当。 解答：\(d\omega\)：令 \(P = -y/(x^2+y^2)\)，\(Q = x/(x^2+y^2)\)。 \(\partial Q/\partial x = (y^2 - x^2)/(x^2+y^2)^2\)， \(\partial P/\partial y = -(x^2-y^2)/(x^2+y^2)^2 = (y^2-x^2)/(x^2+y^2)^2\)。 \(\partial Q/\partial x = \partial P/\partial y\)，故 \(d\omega = 0\)。 不恰当：\(\oint_{S^1} \omega = 2\pi \neq 0\)（沿单位圆积分）。 若 \(\omega = df\)，则 \(\oint df = 0\)，矛盾。 易错点：\(d\omega = 0\) 只是恰当的必要条件。空间有"洞"时可能不充分。

04de Rham 上同调与 Poincaré 引理

🎯 为什么研究这个问题

de Rham 上同调群 \(H^k_{dR}(M)\) 用微分形式（分析工具）计算出 拓扑不变量——这是分析与拓扑之间最深刻的桥梁之一。\(H^k_{dR}\) 测量的是 \(k\) 维的"闭而不恰当"的形式空间，反映了流形中 \(k\) 维 "洞"的数量。Poincaré 引理保证在"无洞"的空间上 \(H^k = 0\)。

🔍 抽象化方法

de Rham 上同调是链复形 \(0 \to \Omega^0 \xrightarrow{d} \Omega^1 \xrightarrow{d} \cdots\) 的上同调。de Rham 定理建立了它与奇异上同调的同构。

📐 公式推导 / 模型建立

de Rham 上同调： \[H^k_{dR}(M) = \frac{\{\omega \in \Omega^k(M) : d\omega = 0\}} {\{d\eta : \eta \in \Omega^{k-1}(M)\}}\] Poincaré 引理：若 \(M\) 可缩（如星形域），则 \(H^k_{dR}(M) = 0\)（\(k \geq 1\)），\(H^0_{dR}(M) = \mathbb{R}\)。 证明思路：构造同伦算子 \(K: \Omega^k \to \Omega^{k-1}\)， 使 \(Kd + dK = \mathrm{id}\)（\(k \geq 1\)）。 若 \(d\omega = 0\)，则 \(\omega = dK\omega\)，即 \(\omega\) 恰当。 经典计算： - \(H^0_{dR}(M) \cong \mathbb{R}^c\)（\(c\) = 连通分量数） - \(H^1_{dR}(S^1) = \mathbb{R}\)（\(d\theta\) 闭不恰当） - \(H^k_{dR}(S^n) = \begin{cases}\mathbb{R} & k=0,n \\ 0 & \text{otherwise}\end{cases}\) de Rham 定理（陈述）：\(H^k_{dR}(M) \cong H^k(M; \mathbb{R})\) （奇异上同调），这是分析与拓扑的统一。

🌐 实际应用

- 物理中的规范势：电磁势 \(A\) 在无洞区域上全局存在（\(F = dA\)）， 但在有洞区域可能只是局部存在（Aharonov-Bohm 效应）

✏️ 例题分析与易错点

例题：计算 \(H^1_{dR}(\mathbb{R}^2 \setminus \{0\})\)。 解答：\(\omega = \frac{-y\,dx + x\,dy}{x^2+y^2}\) 闭不恰当（\(\oint_{S^1} \omega = 2\pi\)）。 任何闭 1-形式 \(\eta\) 可写为 \(\eta = c\omega + df\)（\(c \in \mathbb{R}\)，\(f\) 光滑）。 故 \(H^1_{dR}(\mathbb{R}^2 \setminus \{0\}) = \mathbb{R}\)， 由 \([\omega]\) 生成。 易错点：\(\mathbb{R}^2 \setminus \{0\}\) 与 \(S^1\) 同伦等价， 所以 de Rham 上同调相同（同伦不变性）。
