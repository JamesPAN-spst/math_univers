01流形的定向与带边流形

🎯 为什么研究这个问题

要在流形上积分，首先需要确定"正方向"——这就是定向。在曲面上， 定向决定了法向量的方向；在高维流形上，定向决定了顶阶形式的符号。 带边流形（如半空间 \(\{x^n \geq 0\}\)）是 Stokes 定理的自然舞台： \(\partial M\) 是 \(M\) 的边界，继承一个诱导定向。

🔍 抽象化方法

定向 = 顶阶形式的全局一致符号选取。不可定向流形（莫比乌斯带、 Klein 瓶）上顶阶形式改变符号，积分需修正（或用密度替代）。

📐 公式推导 / 模型建立

定向：\(n\) 维流形 \(M\) 可定向 \(\iff\) 存在处处不为零的 \(n\)-形式 \(\omega \in \Omega^n(M)\)（体积形式）。 等价地：存在图册使所有转移函数的 Jacobian 行列式为正。 带边流形：\(M\) 的每点邻域同胚于 \(\mathbb{R}^n\) 或 \(\mathbb{H}^n = \{x \in \mathbb{R}^n : x^n \geq 0\}\)。 边界 \(\partial M = \{p : p \text{ 对应 } x^n = 0\}\)， \(\partial M\) 是 \((n-1)\) 维无边流形。 诱导定向：\(M\) 定向时，\(\partial M\) 继承"外法向量优先"的定向规则。 \[(\text{外法向}, e_1, \ldots, e_{n-1}) \text{ 正定向于 } M \iff (e_1, \ldots, e_{n-1}) \text{ 正定向于 } \partial M\]

🌐 实际应用

- 莫比乌斯带不可定向——上面的蚂蚁走一圈后左右翻转

✏️ 例题分析与易错点

例题：\(D^2 = \{(x,y) : x^2+y^2 \leq 1\}\) 是带边流形。 \(\partial D^2 = S^1\)。确定诱导定向。 解答：\(D^2\) 取标准定向（\(dx \wedge dy\)）。 外法向量为 \(\vec{n} = (x, y)/|(x,y)|\)（向外）。 \((\vec{n}, \vec{e})\) 正定向 \(\iff\) \(\vec{e}\) 逆时针。 所以 \(\partial D^2 = S^1\) 的诱导定向是逆时针方向。 易错点：Stokes 定理要求 \(\partial M\) 使用诱导定向。 方向搞反会导致积分差一个负号。

02微分形式的积分

🎯 为什么研究这个问题

在流形上，积分的"被积对象"不是函数而是顶阶微分形式。这是因为 坐标变换时 Jacobian 行列式恰好给出微分形式在拉回下的变换因子， 保证积分的坐标无关性。体积形式提供了流形上的"测度"。

🔍 抽象化方法

积分的坐标无关性来自微分形式的变换律： \(\int_{\varphi(U)} \omega = \int_U \varphi^*\omega\)（拉回公式）。

📐 公式推导 / 模型建立

\(n\)-形式在 \(n\) 维定向流形上的积分： 设 \((U, \varphi)\) 是坐标卡，\(\omega = f(x)\, dx^1 \wedge \cdots \wedge dx^n\)。 \[\int_U \omega = \int_{\varphi(U)} f(x)\, dx^1 \cdots dx^n\] （右边是 \(\mathbb{R}^n\) 中的 Lebesgue 积分）。 坐标变换律：若 \(\psi = \varphi \circ \phi^{-1}\) 是转移函数， \[\int f \circ \psi \cdot |\det J_\psi|\, dy = \int f\, dx\] 定向保证 \(\det J_\psi > 0\)，故可去绝对值。 配分函数：对一般 \(\omega \in \Omega^n(M)\)，取从属于坐标覆盖的 单位分解 \(\{\rho_\alpha\}\)，\(\int_M \omega = \sum_\alpha \int_{U_\alpha} \rho_\alpha \omega\)。 体积形式：\(dV_g = \sqrt{\det(g_{ij})}\, dx^1 \wedge \cdots \wedge dx^n\) （黎曼流形上）。

🌐 实际应用

- 曲面面积 \(A = \int_S dA\)，曲线长度 \(L = \int_\gamma ds\) 都是形式积分的特例

✏️ 例题分析与易错点

例题：计算 \(\int_{S^1} d\theta\)（\(S^1\) 逆时针定向，\(\theta\) 为角度坐标）。 解答：\(d\theta = \frac{-y\,dx + x\,dy}{x^2+y^2}|_{S^1}\)。 参数化：\(x = \cos t\)，\(y = \sin t\)，\(t \in [0, 2\pi]\)。 \(d\theta|_{S^1} = dt\)。\(\int_0^{2\pi} dt = 2\pi\)。 易错点：\(\theta\) 不是 \(S^1\) 上的全局光滑函数（在某点不连续）， 所以 \(d\theta\) 闭但不恰当。

03广义 Stokes 定理

🎯 为什么研究这个问题

Green 定理、Gauss 散度定理、经典 Stokes 旋度定理看似不同， 实则都是同一个定理的特例——广义 Stokes 定理：\(\int_M d\omega = \int_{\partial M} \omega\)。 这是微分几何中最优美的公式之一，将体上的"微分"与边界上的 "积分"联系起来。

🔍 抽象化方法

Stokes 定理是 \(d\) 与 \(\int\) 的对偶：外微分是边界算子的 "伴随"。这在同调/上同调理论中有深远意义。

📐 公式推导 / 模型建立

广义 Stokes 定理：设 \(M\) 是 \(n\) 维紧致定向带边流形， \(\omega \in \Omega^{n-1}(M)\)。则 \[\int_M d\omega = \int_{\partial M} \omega\] 证明思路（\(\mathbb{R}^n\) 中紧支撑情况）： 设 \(\omega = f\, dx^1 \wedge \cdots \wedge \widehat{dx^i} \wedge \cdots \wedge dx^n\)。 \(d\omega = (-1)^{i-1} \frac{\partial f}{\partial x^i} dx^1 \wedge \cdots \wedge dx^n\)。 对第 \(i\) 个变量用 Fubini 定理： \(\int \frac{\partial f}{\partial x^i} dx^i = f|_{\text{边界}}\)。 逐坐标方向验证，用单位分解拼合。 经典特例： - Green（\(n=2\)，\(M \subseteq \mathbb{R}^2\)）：\(\oint_{\partial M} P\,dx + Q\,dy = \iint_M (Q_x - P_y)\,dx\,dy\) - 散度定理（\(n=3\)，\(\omega = F_1 dy \wedge dz + \cdots\)）：\(\oint_{\partial M} \vec{F} \cdot d\vec{S} = \iiint_M \nabla \cdot \vec{F}\,dV\) - 经典 Stokes（\(n=2\)，\(M\) 是 \(\mathbb{R}^3\) 中的曲面）：\(\oint_{\partial M} \vec{F} \cdot d\vec{r} = \iint_M (\nabla \times \vec{F}) \cdot d\vec{S}\)

🌐 实际应用

- 电磁学中 Gauss 定律：\(\oint \vec{E} \cdot d\vec{S} = Q/\varepsilon_0\) - 流体力学中 Kelvin 环量定理

✏️ 例题分析与易错点

例题：用 Stokes 定理计算 \(\oint_C y\,dx + x\,dy\)， \(C\) 是 \(x^2 + y^2 = 1\) 逆时针。 解答：\(\omega = y\,dx + x\,dy\)。\(d\omega = dy \wedge dx + dx \wedge dy = 0\)。 所以 \(\oint_C \omega = \int_{D^2} d\omega = 0\)。 易错点：若 \(\omega = -y\,dx + x\,dy\)，则 \(d\omega = 2\,dx \wedge dy\)， \(\oint = 2 \cdot \text{面积} = 2\pi\)。注意正负号。

04散度定理与 Maxwell 方程

🎯 为什么研究这个问题

Maxwell 方程组是电磁学的基石，传统上用向量微积分（散度、旋度） 表述。用微分形式语言重写后，四个 Maxwell 方程简化为两个： \(dF = 0\) 和 \(d{*F} = J\)。这不仅更优美，而且能推广到弯曲时空 ——这是广义相对论中电磁场理论的基础。

🔍 抽象化方法

微分形式语言消除了坐标依赖，揭示了物理定律的几何本质。 散度定理的流形推广是 Stokes 定理的直接应用。

📐 公式推导 / 模型建立

散度定理（流形版）：\((M^n, g)\) 紧致黎曼流形带边。 向量场 \(X\) 的散度 \(\mathrm{div}(X) = *d(*X^\flat)\)（或用联络 \(\mathrm{div}(X) = \nabla_i X^i\)），则 \[\int_M \mathrm{div}(X)\, dV_g = \int_{\partial M} g(X, \nu)\, dA\] 其中 \(\nu\) 是外法向量。这是 Stokes 定理对 \((n-1)\)-形式的特例。 Maxwell 方程的形式语言： 电磁场 2-形式 \(F = E_i dx^i \wedge dt + B_{ij} dx^i \wedge dx^j\)。 \[dF = 0 \quad (\text{Faraday + 无磁单极})\] \[d{*F} = J \quad (\text{Ampère + Gauss})\] \(J\) 是电流 3-形式。\(d^2 = 0\) 自动给出电荷守恒 \(dJ = 0\)。

🌐 实际应用

- 广义相对论中的弯曲时空用同样的形式语言描述电磁场 - Yang-Mills 方程推广了 \(dF = 0\)，\(d_A {*F} = J\) 到非阿贝尔规范群

✏️ 例题分析与易错点

例题：在 \(\mathbb{R}^3\) 中验证散度定理： \(\vec{F} = (x,y,z)\)，\(M = \{x^2+y^2+z^2 \leq 1\}\)。 解答：\(\nabla \cdot \vec{F} = 3\)。 \(\iiint_M 3\,dV = 3 \cdot \frac{4\pi}{3} = 4\pi\)。 \(\oiint_{\partial M} \vec{F} \cdot d\vec{S} = \oiint_{S^2} (x,y,z) \cdot (x,y,z)\,dA = \oiint 1\,dA = 4\pi\)。一致。 易错点：\(\vec{F} \cdot \vec{n}\) 在球面上恰好等于 \(1\)（因为 \((x,y,z)\) 在 \(S^2\) 上是单位外法向），简化了计算。
