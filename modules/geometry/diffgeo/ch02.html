切丛与向量场

01切丛与向量丛

🎯 为什么研究这个问题

将每一点的切空间"黏合"在一起，得到切丛 \(TM = \bigsqcup_p T_pM\)。 切丛本身是一个 \(2n\) 维流形，是所有切向量"共存"的空间。更一般 地，向量丛将向量空间参数化地放在流形上每一点，是纤维丛理论的 起点，也是规范场论的数学语言。

🔍 抽象化方法

向量丛 = 流形 + 每点附着一个向量空间，整体构成光滑流形。 截面 = 为每一点选择一个向量（如向量场就是切丛的截面）。

📐 公式推导 / 模型建立

切丛：\(TM = \bigsqcup_{p \in M} T_pM = \{(p, v) : p \in M, v \in T_pM\}\)。 自然投影 \(\pi: TM \to M\)，\(\pi(p, v) = p\)。 坐标卡 \((U, x^i)\) on \(M\) 诱导 \((TU, x^i, v^i)\) on \(TM\)： \(v = \sum v^i \partial/\partial x^i|_p\)。 \(\dim TM = 2 \dim M\)。 向量丛：\((E, \pi, M)\)，\(\pi: E \to M\)，每个纤维 \(E_p = \pi^{-1}(p)\) 同构于 \(\mathbb{R}^k\)（\(k\) = 秩），且局部平凡化 \(\Phi: \pi^{-1}(U) \to U \times \mathbb{R}^k\)。 截面 (Section)：\(s: M \to E\)，\(\pi \circ s = \mathrm{id}_M\)。 即为每个 \(p\) 选取 \(s(p) \in E_p\)。光滑截面记为 \(\Gamma(E)\)。 \[\text{向量场} = \Gamma(TM), \quad \text{余向量场} = \Gamma(T^*M)\]

🌐 实际应用

- 纤维丛是规范场论（Yang-Mills 理论）的数学框架 - 切丛的截面（向量场）描述物理中的速度场、力场

✏️ 例题分析与易错点

例题：\(TS^1\) 与 \(S^1 \times \mathbb{R}\) 微分同胚（\(S^1\) 的切丛平凡）。 解答：\(S^1\) 是 1 维流形，\(T_\theta S^1 \cong \mathbb{R}\)。 定义 \(\Phi: TS^1 \to S^1 \times \mathbb{R}\)，\(\Phi(\theta, v\partial/\partial\theta) = (\theta, v)\)。 \(\Phi\) 是光滑双射且 \(\Phi^{-1}\) 光滑，所以 \(TS^1 \cong S^1 \times \mathbb{R}\)。 直觉：\(S^1\) 上存在处处不为零的切向量场（逆时针方向），所以切丛平凡。 易错点：\(TS^2\) 不平凡！（"毛球定理"：\(S^2\) 上不存在处处不为零 的连续切向量场。）

02向量场的局部表示与微分算子

🎯 为什么研究这个问题

向量场不仅是几何对象（每点附着一个方向），还可以视为微分算子—— 对函数进行方向导数运算。这个"几何-代数"双重观点是微分几何的 精髓：向量场既能画成箭头（几何），也能作用于函数（代数）。

🔍 抽象化方法

\(X \in \Gamma(TM)\) 可视为 \(C^\infty(M) \to C^\infty(M)\) 的 \(\mathbb{R}\)-线性 映射，满足 Leibniz 法则。这将向量场"代数化"。

📐 公式推导 / 模型建立

局部表示：坐标卡 \((U, x^i)\) 下， \(X|_U = \sum_{i=1}^n X^i(x) \frac{\partial}{\partial x^i}\)， \(X^i \in C^\infty(U)\)。 作为微分算子：\(X(f)(p) = \sum X^i(p) \frac{\partial f}{\partial x^i}(p)\)， 其中 \(f \in C^\infty(M)\)。 性质： (1) \(\mathbb{R}\)-线性：\(X(af + bg) = aX(f) + bX(g)\) (2) Leibniz 法则：\(X(fg) = X(f)g + fX(g)\) (3) \(C^\infty(M)\)-模结构：\((fX)(g) = f \cdot X(g)\) 反过来，满足 (1)(2) 的 \(C^\infty(M) \to C^\infty(M)\) 的映射 恰好是向量场（推导的整体版）。

✏️ 例题分析与易错点

例题：在 \(\mathbb{R}^2\) 上，\(X = x \partial/\partial x + y \partial/\partial y\)。 求 \(X(x^2 + y^2)\)。 解答：\(X(x^2+y^2) = x \cdot 2x + y \cdot 2y = 2(x^2+y^2)\)。 几何意义：\(X\) 是径向向量场，\(x^2+y^2\) 沿径向每单位增加 \(2r^2\)。 易错点：\(X(f)\) 是函数（不是数），结果是 \(C^\infty(M)\) 中的元素。

⚡ 快速检测

03流与积分曲线

🎯 为什么研究这个问题

给定向量场 \(X\)，"沿着箭头走"会走出什么样的轨迹？积分曲线就是 向量场的"轨迹线"，它在每一点的切向量恰好是向量场在该点的值。 流 \(\phi_t\) 则是将积分曲线组织成一族变换——它描述了向量场驱动 的"连续对称运动"，是经典力学（Hamilton 流）和动力系统的核心。

🔍 抽象化方法

向量场 ↔ 一阶常微分方程 ↔ 单参数变换群（流）。 这是微分几何中微分方程与几何的核心联系。

📐 公式推导 / 模型建立

积分曲线：\(\gamma: I \to M\) 是 \(X\) 的积分曲线 \(\iff\) \(\gamma'(t) = X_{\gamma(t)},\; \forall t \in I\)。 坐标中：\(\dot{x}^i(t) = X^i(x^1(t), \ldots, x^n(t))\)——一阶 ODE 组。 由 Picard-Lindelöf 定理，初始条件 \(\gamma(0) = p\) 下 局部存在唯一的积分曲线。 流 (Flow)：\(\phi: (-\varepsilon, \varepsilon) \times M \to M\)， \(\phi_t(p) = \gamma_p(t)\)。 \[\phi_0 = \mathrm{id}_M, \quad \phi_{t+s} = \phi_t \circ \phi_s\] （单参数变换群性质，在定义域内成立。） 紧流形上 \(X\) 的积分曲线可延拓到全 \(\mathbb{R}\)（完备向量场）， 此时 \(\{\phi_t\}_{t \in \mathbb{R}}\) 构成单参数微分同胚群。

🌐 实际应用

- Hamilton 力学中相空间上的 Hamilton 向量场生成时间演化流 - 流体力学中速度场的积分曲线 = 流线

✏️ 例题分析与易错点

例题：\(X = -y\partial/\partial x + x\partial/\partial y\)（\(\mathbb{R}^2\) 上旋转场）。 求流 \(\phi_t\)。 解答：ODE：\(\dot{x} = -y\)，\(\dot{y} = x\)。 解：\(x(t) = x_0 \cos t - y_0 \sin t\)，\(y(t) = x_0 \sin t + y_0 \cos t\)。 即 \(\phi_t\) 是绕原点逆时针旋转角度 \(t\) 的变换。 易错点：流 \(\phi_t\) 不一定全局定义（若向量场不完备，解可能爆破）。

⚡ 快速检测

04Lie 括号与相关向量场

🎯 为什么研究这个问题

两个向量场 \(X, Y\) 的流 \(\phi_t, \psi_s\) 是否可交换？Lie 括号 \([X,Y]\) 精确测量了这种不可交换性：\([X,Y] = 0\) 当且仅当流可交 换。Lie 括号赋予向量场空间一个 Lie 代数结构，是 Lie 群与 Lie 代数理论的几何起源。

🔍 抽象化方法

\([X,Y]\) 作为导子的交换子：\([X,Y](f) = X(Y(f)) - Y(X(f))\)。 这将微分算子的"非交换性"几何化。

📐 公式推导 / 模型建立

定义：\([X,Y] = XY - YX\) 作为微分算子的交换子。 即 \([X,Y](f) = X(Y(f)) - Y(X(f))\)。 关键事实：\([X,Y]\) 仍是向量场（一阶微分算子），虽然 \(XY\) 是二阶算子。 坐标中：若 \(X = X^i \partial_i\)，\(Y = Y^j \partial_j\)， \[[X,Y]^k = X^i \frac{\partial Y^k}{\partial x^i} - Y^i \frac{\partial X^k}{\partial x^i}\] 性质： (1) 反对称：\([X,Y] = -[Y,X]\) (2) Jacobi 恒等式：\([[X,Y],Z] + [[Y,Z],X] + [[Z,X],Y] = 0\) (3) \([fX, gY] = fg[X,Y] + f(Xg)Y - g(Yf)X\) 流的交换性：\([X,Y] = 0 \iff \phi_t \circ \psi_s = \psi_s \circ \phi_t\)。 \(f\)-相关：若 \(f: M \to N\) 光滑，\(X \in \Gamma(TM)\)，\(Y \in \Gamma(TN)\)， \(X, Y\) 是 \(f\)-相关的 \(\iff df \circ X = Y \circ f\)。 此时 \([X_1, X_2]\) 与 \([Y_1, Y_2]\) 也 \(f\)-相关。

🌐 实际应用

- Lie 括号 \([X,Y] = 0\) 的充要条件"流可交换"在经典力学中意味着 两个守恒量独立（Liouville 可积性）

✏️ 例题分析与易错点

例题：\(X = \partial/\partial x\)，\(Y = x \partial/\partial y\)。 求 \([X,Y]\)。 解答：\([X,Y](f) = X(Y(f)) - Y(X(f))\)。 \(Y(f) = x \partial f/\partial y\)，\(X(Y(f)) = \partial/\partial x(x \partial f/\partial y) = \partial f/\partial y + x \partial^2 f/\partial x \partial y\)。 \(X(f) = \partial f/\partial x\)，\(Y(X(f)) = x \partial^2 f/\partial x \partial y\)。 \([X,Y](f) = \partial f/\partial y\)。即 \([X,Y] = \partial/\partial y\)。 易错点：计算 Lie 括号时必须使用 Leibniz 法则，不能直接对系数求导。
