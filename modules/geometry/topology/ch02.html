连续映射与同胚

01连续映射的定义与等价刻画

🎯 为什么研究这个问题

拓扑学的核心问题是理解"什么性质在连续变形下不变"。为此必须先 精确定义连续映射。与分析中 \(\varepsilon\)-\(\delta\) 定义不同，拓扑中 的连续性用开集的原像刻画，摆脱了对度量的依赖。这个定义的优雅 之处在于：它统一了分析中各种连续性概念，并且可以推广到任何拓扑空间。

🔍 抽象化方法

连续映射的本质是"保持拓扑结构"：它将目标空间的开集拉回为源空 间的开集。这是函子思想的雏形——连续映射是拓扑空间范畴的态射。

📐 公式推导 / 模型建立

定义：\(f: (X, \tau_X) \to (Y, \tau_Y)\) 连续，当且仅当 \[\forall V \in \tau_Y,\; f^{-1}(V) \in \tau_X\] 四种等价刻画（设 \(f: X \to Y\)）： (1) \(V\) 开 \(\Rightarrow f^{-1}(V)\) 开（开集原像定义） (2) \(F\) 闭 \(\Rightarrow f^{-1}(F)\) 闭（闭集原像） (3) \(\forall x \in X\)，\(f(x)\) 的每个邻域 \(V\)，\(f^{-1}(V)\) 是 \(x\) 的邻域（邻域定义） (4) \(\forall A \subseteq X\)，\(f(\overline{A}) \subseteq \overline{f(A)}\)（闭包条件） 证明 (1)\(\Rightarrow\)(2)：\(F\) 闭 \(\Rightarrow Y \setminus F\) 开 \(\Rightarrow f^{-1}(Y \setminus F) = X \setminus f^{-1}(F)\) 开 \(\Rightarrow f^{-1}(F)\) 闭。 复合保连续：若 \(f: X \to Y\), \(g: Y \to Z\) 连续，则 \(g \circ f\) 连续：\((g \circ f)^{-1}(W) = f^{-1}(g^{-1}(W))\)。

🌐 实际应用

- 在泛函分析中，弱拓扑恰好是使所有线性泛函连续的最弱拓扑 - 连续映射是拓扑空间范畴 \(\mathbf{Top}\) 的态射

✏️ 例题分析与易错点

例题：设 \(f: \mathbb{R} \to \mathbb{R}\)，\(f(x) = x^2\)。 用开集原像定义证明 \(f\) 在标准拓扑下连续。 解答：设 \((a,b)\) 为 \(\mathbb{R}\) 的开区间。 若 \(b \leq 0\)，则 \(f^{-1}((a,b)) = \emptyset\)，开。 若 \(a < 0 < b\)，\(f^{-1}((a,b)) = (-\sqrt{b}, \sqrt{b})\)，开。 若 \(0 \leq a < b\)，\(f^{-1}((a,b)) = (-\sqrt{b},-\sqrt{a}) \cup (\sqrt{a},\sqrt{b})\)，开。 任何开集是开区间的并，原像是开集的并，仍开。 易错点：只验证了开区间的原像，忘记说明"一般开集是开区间的并" 这一步——这正是使用基的好处。

⚡ 快速检测

02同胚与拓扑不变量

🎯 为什么研究这个问题

什么时候两个拓扑空间"本质上相同"？答案是同胚——一个连续的双射 且逆映射也连续。同胚是拓扑学的"同构"概念。被同胚保持的性质称 为拓扑不变量，是分类空间的关键工具。找到足够多的拓扑不变量来 区分空间、或证明两空间同胚，是拓扑学的核心任务。

🔍 抽象化方法

同胚 = 拓扑范畴中的同构。拓扑不变量 = 在同构下不变的属性。 这与群论中"群同构保持阶"的思想完全类似。

📐 公式推导 / 模型建立

定义：\(f: X \to Y\) 是同胚（homeomorphism），若 \(f\) 是双射且 \(f, f^{-1}\) 均连续。记 \(X \cong Y\)。 注意：连续双射未必是同胚！需要 \(f^{-1}\) 也连续。 反例：\(f: [0, 2\pi) \to S^1\)，\(f(t) = (\cos t, \sin t)\)。 \(f\) 是连续双射，但 \(f^{-1}\) 在 \((1,0)\) 处不连续。 拓扑不变量（同胚保持的性质）： - 紧致性、连通性、道路连通性 - 第一/第二可数、可分性 - \(T_0, T_1, T_2\) 等分离性质 - 基本群 \(\pi_1(X, x_0)\)（同胚诱导群同构） - Euler 特征数 \(\chi\) \[X \cong Y \Rightarrow \text{所有拓扑不变量相同}\] 逆命题不成立（相同不变量不保证同胚）。

🌐 实际应用

- 拓扑数据分析(TDA)利用拓扑不变量（Betti 数）识别数据形状 - 纽结理论中用不变量区分不同纽结

✏️ 例题分析与易错点

例题：证明 \(\mathbb{R}\) 与 \((0,1)\) 同胚。 解答：取 \(f: (0,1) \to \mathbb{R}\)，\(f(x) = \tan(\pi x - \pi/2)\)。 \(f\) 是连续严格单调函数，像为 \(\mathbb{R}\)， \(f^{-1}(y) = \frac{1}{\pi}\arctan(y) + \frac{1}{2}\) 也连续。 故 \((0,1) \cong \mathbb{R}\)。 易错点：混淆"同胚 \(\cong\)"与"等距"。\((0,1)\) 和 \(\mathbb{R}\) 同胚但 不等距（\((0,1)\) 有界而 \(\mathbb{R}\) 无界——但有界性不是拓扑不变量！）。

03开映射、闭映射与粘合引理

🎯 为什么研究这个问题

连续映射保证"开集的原像是开集"，但正向的问题——"开集的像是否 是开集"——则需要额外条件。开映射和闭映射刻画了这种正向保结构 性。粘合引理则说明：如果一个映射在足够好的覆盖的每片上连续， 那么整体连续。这是构造连续映射的核心工具。

📐 公式推导 / 模型建立

开映射：\(f: X \to Y\)，若 \(\forall U \in \tau_X\)，\(f(U) \in \tau_Y\)。 闭映射：\(f: X \to Y\)，若 \(\forall F\) 闭，\(f(F)\) 闭。 开映射 \(\neq\) 连续（如投影 \(\pi_1: \mathbb{R}^2 \to \mathbb{R}\) 是开映射且连续， 但许多开映射不连续）。 连续 \(\neq\) 开映射（\(f(x) = x^2\) 连续但非开映射：\(f((-1,1)) = [0,1)\)）。 同胚的等价刻画：\(f\) 是同胚 \(\iff\) \(f\) 是连续双射且开映射 \(\iff\) \(f\) 是连续双射且闭映射。 粘合引理（开覆盖版）：设 \(\{U_\alpha\}\) 是 \(X\) 的开覆盖， \(f: X \to Y\)。若 \(f|_{U_\alpha}: U_\alpha \to Y\) 对每个 \(\alpha\) 连续， 则 \(f\) 连续。 证明：设 \(V \subseteq Y\) 开，则 \(f^{-1}(V) \cap U_\alpha = (f|_{U_\alpha})^{-1}(V)\) 在 \(U_\alpha\) 中开， 故在 \(X\) 中开。\(f^{-1}(V) = \bigcup_\alpha (f^{-1}(V) \cap U_\alpha)\) 是开集的并，故开。 粘合引理（有限闭覆盖版）：类似结论对有限闭覆盖成立。

🌐 实际应用

- 粘合引理是定义分片函数（如样条）连续性的理论保证 - 商映射理论依赖开/闭映射的判定

✏️ 例题分析与易错点

例题：定义 \(f: \mathbb{R} \to \mathbb{R}\)， \(f(x) = x\)（\(x \leq 0\)），\(f(x) = x+1\)（\(x > 0\)）。 \(f\) 是否连续？是否是开映射？ 解答：\(f\) 不连续（在 \(x=0\) 处有跳跃，\(f^{-1}((0.5, 1.5))\) 不是开集）。 \(f\) 在 \((-\infty, 0]\) 和 \((0, +\infty)\) 上分别连续，但 \(\{(-\infty, 0], (0,+\infty)\}\) 不是开覆盖（\((-\infty, 0]\) 不开）， 故粘合引理不适用。 易错点：试图对非开覆盖使用开覆盖版粘合引理；或对无限闭覆盖 使用有限闭覆盖版。

04初始拓扑、终止拓扑与嵌入

🎯 为什么研究这个问题

给定一族映射，如何选择"恰好合适"的拓扑使它们连续？初始拓扑 （最弱使映射连续的拓扑）和终止拓扑（最强使映射连续的拓扑）回 答了这个问题。子空间拓扑、积拓扑是初始拓扑的特例；商拓扑是终 止拓扑的特例。嵌入则是将子空间"等同地"放入更大空间的方式。

🔍 抽象化方法

初始/终止拓扑体现了"万有性质"(universal property)思想： 通过要求特定的映射连续来唯一确定拓扑。这是范畴论中极限/余极 限概念的具体体现。

📐 公式推导 / 模型建立

初始拓扑：给定 \(f_\alpha: X \to (Y_\alpha, \tau_\alpha)\)， \(X\) 上使所有 \(f_\alpha\) 连续的最弱拓扑 \(\tau_i\) 的子基为 \[\mathcal{S} = \{f_\alpha^{-1}(V) : V \in \tau_\alpha,\; \alpha \in A\}\] 万有性质：\(g: Z \to X\) 连续 \(\iff \forall \alpha,\; f_\alpha \circ g\) 连续。 终止拓扑：给定 \(f_\alpha: (Y_\alpha, \tau_\alpha) \to X\)， \(X\) 上使所有 \(f_\alpha\) 连续的最强拓扑为 \[\tau_f = \{U \subseteq X : \forall \alpha,\; f_\alpha^{-1}(U) \in \tau_\alpha\}\] 万有性质：\(g: X \to Z\) 连续 \(\iff \forall \alpha,\; g \circ f_\alpha\) 连续。 嵌入 (Embedding)：\(f: X \to Y\) 是嵌入，若 \(f: X \to f(X)\) 是同胚 （其中 \(f(X)\) 带子空间拓扑）。即 \(f\) 是单射、连续、且 \(f^{-1}: f(X) \to X\) 连续。

🌐 实际应用

- 弱拓扑是泛函分析中对偶空间上的初始拓扑 - Whitney 嵌入定理：任何 \(n\) 维流形可嵌入 \(\mathbb{R}^{2n+1}\)

✏️ 例题分析与易错点

例题：设 \(X = \mathbb{R}\)，\(f: X \to \mathbb{R}\)，\(f(x) = x^2\)， \(g: X \to \mathbb{R}\)，\(g(x) = x^3\)。求 \(X\) 上关于 \(\{f, g\}\) 的初始拓扑中，\(\{0\}\) 是否为开集？ 解答：初始拓扑的子基为 \(\{f^{-1}(V), g^{-1}(V) : V \text{ 开}\}\)。 \(f^{-1}(\{0\}) = \{0\}\) 但 \(\{0\}\) 在 \(\mathbb{R}\) 中不开， 我们需要 \(f^{-1}(V)\) 形式。\(f^{-1}((-\varepsilon, \varepsilon)) = (-\sqrt{\varepsilon}, \sqrt{\varepsilon})\)。 \(g^{-1}((-\varepsilon, \varepsilon)) = (-\varepsilon^{1/3}, \varepsilon^{1/3})\)。 \(\{0\} = \bigcap_n (-1/n, 1/n)\) 不是有限交，故 \(\{0\}\) 不是开集。 易错点：混淆"子基元素"与"拓扑中的开集"——子基元素的有限交构成 基，但无限交不一定在拓扑中。
