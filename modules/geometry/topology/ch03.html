分离公理

01T₀、T₁ 与 T₂ (Hausdorff) 分离公理

🎯 为什么研究这个问题

在一般拓扑空间中，"不同的点可能无法用开集区分"——这对分析直觉 来说非常奇怪。分离公理是对空间施加的递进式限制，使得空间越来越 接近我们熟悉的度量空间。其中最重要的是 Hausdorff 条件(T₂)，它 保证极限的唯一性——没有它，连序列收敛到唯一极限这种基本事实都 不成立。

🔍 抽象化方法

分离公理的层次：T₀ → T₁ → T₂ → T₃ → T₄，每一级要求更 强的"用开集分离点/集合"的能力。越高的分离性，空间越"好"。

📐 公式推导 / 模型建立

\(T_0\) (Kolmogorov)：\(\forall x \neq y\)，\(\exists U\) 开， \(x \in U, y \notin U\) 或 \(y \in U, x \notin U\)（至少一个方向）。 \(T_1\)：\(\forall x \neq y\)，\(\exists U\) 开，\(x \in U, y \notin U\) （两个方向都行）。等价条件：单点集 \(\{x\}\) 都是闭集。 \(T_2\) (Hausdorff)：\(\forall x \neq y\)，\(\exists U, V\) 开， \(x \in U, y \in V, U \cap V = \emptyset\)。 极限唯一性（\(T_2\) 的关键后果）：若 \(x_n \to a\) 且 \(x_n \to b\)， 取不相交开邻域 \(U \ni a\)，\(V \ni b\)。当 \(n\) 充分大时 \(x_n \in U\) 且 \(x_n \in V\)，矛盾。故 \(a = b\)。 蕴含关系： \[T_2 \Rightarrow T_1 \Rightarrow T_0\] 反之均不成立。

🌐 实际应用

- 代数几何中的 Zariski 拓扑是 \(T_1\)（闭点）但通常不是 \(T_2\) - Hausdorff 性在流形论中是基本假设之一

✏️ 例题分析与易错点

例题：证明余有限拓扑在无穷集 \(X\) 上是 \(T_1\) 但不是 \(T_2\)。 解答：\(T_1\)：\(\{x\}\) 的补 \(X \setminus \{x\}\) 的补是 \(\{x\}\)（有限）， 所以 \(X \setminus \{x\}\) 是开集，即 \(\{x\}\) 闭。 非 \(T_2\)：设 \(U, V\) 非空开集，则 \(X \setminus U\) 和 \(X \setminus V\) 均有限。\(X \setminus (U \cap V) = (X \setminus U) \cup (X \setminus V)\) 有限。由于 \(X\) 无穷，\(U \cap V \neq \emptyset\)， 故不存在不相交的非空开集。 易错点：在有限集上余有限拓扑 = 离散拓扑，此时满足 \(T_2\)！ "\(T_1\) 但非 \(T_2\)"只在无穷集上成立。

02正则空间(T₃)与正规空间(T₄)

🎯 为什么研究这个问题

\(T_2\) 可以用开集分离两个点，但能否分离一个点和一个闭集？能否 分离两个不相交的闭集？这些更强的要求分别定义了正则性(\(T_3\))和 正规性(\(T_4\))。正规空间尤为重要，因为它支撑了 Urysohn 引理和 Tietze 扩张定理等强大工具。

🔍 抽象化方法

分离的对象从"点-点"升级为"点-集"和"集-集"。度量空间自动满足 所有分离公理，正是因为距离函数提供了天然的分离机制。

📐 公式推导 / 模型建立

正则空间 (\(T_3\))：\(T_1\) 且对任意 \(x \in X\) 和闭集 \(F \not\ni x\)，存在不相交开集 \(U \ni x\)，\(V \supseteq F\)。 等价条件：\(\forall x \in U\)（\(U\) 开），\(\exists V\) 开， \(x \in V \subseteq \overline{V} \subseteq U\)。 正规空间 (\(T_4\))：\(T_1\) 且对任意不相交闭集 \(F_1, F_2\)， 存在不相交开集 \(U_1 \supseteq F_1\)，\(U_2 \supseteq F_2\)。 \[T_4 \Rightarrow T_3 \Rightarrow T_2 \Rightarrow T_1 \Rightarrow T_0\] 关键事实： - 度量空间 \(\Rightarrow T_4\)（利用距离函数 \(d(x, F) = \inf_{y \in F} d(x,y)\)） - 紧 Hausdorff 空间 \(\Rightarrow T_4\) - 正则 + Lindelöf \(\Rightarrow T_4\)

✏️ 例题分析与易错点

例题：证明度量空间 \((X, d)\) 是正规 (\(T_4\)) 的。 解答：设 \(F_1, F_2\) 为不相交闭集。定义 \(U_1 = \{x : d(x, F_1) < d(x, F_2)\}\)， \(U_2 = \{x : d(x, F_2) < d(x, F_1)\}\)。 \(U_1, U_2\) 开（\(d(\cdot, F)\) 连续），\(F_1 \subseteq U_1\) （\(x \in F_1 \Rightarrow d(x, F_1)=0 < d(x, F_2)\)， 因为 \(F_2\) 闭且 \(x \notin F_2\)）。类似 \(F_2 \subseteq U_2\)。 \(U_1 \cap U_2 = \emptyset\)。 易错点：必须用 \(F_1 \cap F_2 = \emptyset\) 且 \(F_2\) 闭来保证 \(x \in F_1 \Rightarrow d(x, F_2) > 0\)。

03Urysohn 引理与 Tietze 扩张定理

🎯 为什么研究这个问题

正规空间能用开集分离闭集，但能否做到更精细——用连续函数分离？ Urysohn 引理给出了肯定答案：正规空间中两个不相交闭集之间总 存在连续函数"平滑过渡"。这是拓扑空间理论中最深刻的结果之一， 也是 Tietze 扩张定理（将闭子集上的连续函数延拓到整个空间）的基础。

🔍 抽象化方法

Urysohn 引理将"几何分离"（开集）提升为"分析分离"（连续函数）， 在拓扑与实分析之间架起桥梁。

📐 公式推导 / 模型建立

Urysohn 引理：设 \(X\) 是 \(T_4\) 空间，\(F_0, F_1\) 为不相交闭集。 则存在连续函数 \(f: X \to [0,1]\) 使得 \(f|_{F_0} = 0\)，\(f|_{F_1} = 1\)。 证明思路：对每个二进有理数 \(r = p/2^n \in [0,1]\)， 构造开集 \(U_r\) 使 \(F_0 \subseteq U_r\)（\(r > 0\)），且 \(r < s \Rightarrow \overline{U_r} \subseteq U_s\)。 利用正规性递归构造：\(U_1 = X \setminus F_1\)， \(F_0 \subseteq U_0\)，\(\overline{U_0} \subseteq U_1\)。 继续二分：\(\overline{U_0} \subseteq U_{1/2} \subseteq \overline{U_{1/2}} \subseteq U_1\)。 定义 \(f(x) = \inf\{r : x \in U_r\}\)，则 \(f\) 连续且 \[f|_{F_0} = 0, \quad f|_{F_1} = 1\] Tietze 扩张定理（陈述）：\(X\) 为 \(T_4\) 空间，\(A \subseteq X\) 闭， \(f: A \to [a,b]\) 连续。则存在连续延拓 \(F: X \to [a,b]\)，\(F|_A = f\)。

🌐 实际应用

- Tietze 定理在偏微分方程中保证边界条件可延拓为全局函数 - Urysohn 度量化定理：正则 + \(C_2\) \(\Rightarrow\) 可度量化

✏️ 例题分析与易错点

例题：利用 Urysohn 引理，在 \(\mathbb{R}\) 上构造一个连续函数 \(f\) 满足 \(f(x) = 0\)（\(x \leq 0\)），\(f(x) = 1\)（\(x \geq 1\)）， \(0 \leq f \leq 1\)。 解答：\(\mathbb{R}\) 是度量空间故正规。\(F_0 = (-\infty, 0]\)， \(F_1 = [1, +\infty)\) 不相交闭集。Urysohn 引理保证 \(f\) 存在。 具体构造：\(f(x) = 0\)（\(x \leq 0\)），\(f(x) = x\)（\(0 \leq x \leq 1\)）， \(f(x) = 1\)（\(x \geq 1\)）。 易错点：Urysohn 引理只保证存在性，不给出唯一函数。满足条件 的函数有无穷多个。

04分离公理的层次关系与度量空间

🎯 为什么研究这个问题

分离公理从 \(T_0\) 到 \(T_4\) 形成了严格的层次。理解这个层次的关 键在于知道"哪些蕴含是严格的"以及"度量空间在哪一层"。度量空间 自动满足所有分离公理，是"最好"的空间类型之一。Urysohn 度量化 定理反过来告诉我们：满足特定可数和分离条件的空间可以度量化。

📐 公式推导 / 模型建立

完整蕴含链： \[\text{度量} \Rightarrow T_4 \Rightarrow T_3 \Rightarrow T_2 \Rightarrow T_1 \Rightarrow T_0\] 每个 \(\Rightarrow\) 都不可逆（存在反例）。 度量空间满足所有分离公理：对 \(x \neq y\)，取 \(\varepsilon = d(x,y)/2 > 0\)，则 \(B(x,\varepsilon) \cap B(y,\varepsilon) = \emptyset\)（\(T_2\)）。 对闭集的分离利用连续函数 \(d(\cdot, F)\)。 Urysohn 度量化定理（陈述）：\(X\) 正则 (\(T_3\)) + \(C_2\) \(\Rightarrow\) \(X\) 可度量化。即存在度量 \(d\) 使 \(\tau_d = \tau\)。 反例总结： - \(T_0\) 但非 \(T_1\)：Sierpiński 空间 \(\{0,1\}\)，\(\tau = \{\emptyset, \{0\}, \{0,1\}\}\) - \(T_1\) 但非 \(T_2\)：无穷集上余有限拓扑 - \(T_2\) 但非 \(T_3\)：需要更复杂的构造 - \(T_3\) 但非 \(T_4\)：Sorgenfrey 平面 \(\mathbb{R}_\ell \times \mathbb{R}_\ell\)

🌐 实际应用

- 度量化定理是泛函分析中判断拓扑向量空间是否可度量化的工具 - 在计算机科学中，度量空间是最近邻搜索算法的基础

✏️ 例题分析与易错点

例题：Sierpiński 空间 \(S = \{0,1\}\)，\(\tau = \{\emptyset, \{0\}, \{0,1\}\}\)。 判断它满足哪些分离公理。 解答：\(T_0\)：\(0 \neq 1\)，\(\{0\} \ni 0\) 但 \(1 \notin \{0\}\)。✓ 非 \(T_1\)：不存在开集 \(U\) 使 \(1 \in U, 0 \notin U\) （唯一含 \(1\) 的开集是 \(\{0,1\}\)）。 \(\{1\}\) 的补是 \(\{0\}\)，\(\{0\}\) 是开集， 所以 \(\{1\}\) 是闭的。但 \(\{0\}\) 的补是 \(\{1\}\)， \(\{1\}\) 不是开集，所以 \(\{0\}\) 不闭。故不是 \(T_1\)。 易错点：混淆 \(T_0\) 和 \(T_1\)——\(T_0\) 只要求"单方向"可区分。
