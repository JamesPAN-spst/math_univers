完备度量空间

01Cauchy 序列与完备空间

🎯 为什么研究这个问题

在 \(\mathbb{Q}\) 中，序列 \(1, 1.4, 1.41, 1.414, \ldots\) 的项越来越 接近，但极限 \(\sqrt{2}\) 不在 \(\mathbb{Q}\) 中。完备性正是弥补这个 "缺口"的性质：每个 Cauchy 列（项之间距离趋于零的列）都收敛。 完备性是分析学的基石——不动点定理、Baire 范畴定理、函数空间的 完备化都依赖于它。

🔍 抽象化方法

完备性将"内在的收敛趋势"（Cauchy 条件）与"实际收敛"等同起来。 这是度量空间理论中特有的概念（一般拓扑空间无法定义 Cauchy 列）。

📐 公式推导 / 模型建立

Cauchy 序列：\((x_n)\) 在度量空间 \((X, d)\) 中是 Cauchy 列 \(\iff\) \[\forall \varepsilon > 0,\; \exists N,\; \forall m, n \geq N,\; d(x_m, x_n) < \varepsilon\] 完备空间：\((X, d)\) 完备 \(\iff\) 每个 Cauchy 列在 \(X\) 中收敛。 基本事实： - 收敛序列 \(\Rightarrow\) Cauchy 列（反之需完备性）。 - \(\mathbb{R}\) 完备（由确界原理证明）。 - \(\mathbb{R}^n\) 完备（分量逐个收敛）。 - 完备空间的闭子集完备；完备子空间在更大空间中是闭的。 - \(\mathbb{Q}\) 不完备：\(\sqrt{2}\) 的十进制逼近是 Cauchy 列但不收敛。

🌐 实际应用

- Banach 空间（完备赋范空间）是泛函分析的基础框架 - 完备性保证迭代算法（如 Newton 法）的收敛

✏️ 例题分析与易错点

例题：证明 \((C[0,1], d_\infty)\)（连续函数空间，\(d_\infty(f,g) = \sup|f-g|\)）完备。 解答：设 \((f_n)\) 为 Cauchy 列。对每个 \(x\)，\((f_n(x))\) 是 \(\mathbb{R}\) 中 Cauchy 列 \(\Rightarrow f_n(x) \to f(x)\)。 Cauchy 条件保证一致收敛：\(\forall \varepsilon,\; \exists N,\; \sup|f_m - f_n| < \varepsilon\)，令 \(m \to \infty\) 得 \(\sup|f - f_n| \leq \varepsilon\)。 一致极限保连续：\(f \in C[0,1]\)。故 \(f_n \to f\) 在 \(d_\infty\) 下。 易错点：逐点收敛不保证极限连续（需一致收敛）。 \(d_\infty\) 的完备性关键在于一致收敛。

02Banach 不动点定理

🎯 为什么研究这个问题

方程 \(f(x) = x\)（不动点方程）在数学中无处不在：迭代算法、微分 方程的解、博弈的 Nash 均衡。Banach 不动点定理（压缩映射原理） 给出了最简洁的不动点存在唯一性条件：在完备度量空间中，压缩映射 有且只有一个不动点，且迭代总收敛到它。

🔍 抽象化方法

压缩映射原理的核心是"迭代缩小距离"：\(d(f(x), f(y)) \leq q \cdot d(x,y)\) （\(q < 1\)），几何级数式收敛保证极限存在（完备性）和唯一性。

📐 公式推导 / 模型建立

定理 (Banach)：设 \((X, d)\) 完备，\(f: X \to X\) 满足 \(d(f(x), f(y)) \leq q \cdot d(x, y)\)（\(0 \leq q < 1\)）。 则 \(f\) 有唯一不动点 \(x^*\)，且 \(\forall x_0 \in X\)， \(x_n = f(x_{n-1}) \to x^*\)。 证明框架： 取任意 \(x_0\)，令 \(x_n = f^n(x_0)\)。 \(d(x_{n+1}, x_n) \leq q^n d(x_1, x_0)\)。 \[d(x_m, x_n) \leq \sum_{k=n}^{m-1} q^k d(x_1, x_0) \leq \frac{q^n}{1-q} d(x_1, x_0) \to 0\] 故 \((x_n)\) 是 Cauchy 列，完备性 \(\Rightarrow x_n \to x^*\)。 连续性：\(f(x^*) = f(\lim x_n) = \lim f(x_n) = \lim x_{n+1} = x^*\)。 唯一性：若 \(f(y^*) = y^*\)，\(d(x^*, y^*) = d(f(x^*), f(y^*)) \leq q \cdot d(x^*, y^*)\)， \((1-q)d(x^*, y^*) \leq 0\)，故 \(x^* = y^*\)。

🌐 实际应用

- Picard-Lindelöf 定理：\(y' = f(t,y)\) 在 Lipschitz 条件下 有唯一局部解（通过将 ODE 转化为积分方程，在 \(C\) 空间中应用压缩映射） - 隐函数定理的一种证明方法

✏️ 例题分析与易错点

例题：\(f: [0,1] \to [0,1]\)，\(f(x) = (x + 1/x)/3\)... 不对， 修改：\(f: [0,1] \to [0,1]\)，\(f(x) = x/2 + 1/4\)。 证明 \(f\) 有唯一不动点并求之。 解答：\(|f(x) - f(y)| = |x-y|/2\)，\(q = 1/2 < 1\)。\([0,1]\) 完备。 Banach 定理保证唯一不动点。解 \(x = x/2 + 1/4\)：\(x/2 = 1/4\)，\(x^* = 1/2\)。 从 \(x_0 = 0\) 迭代：\(x_1 = 1/4, x_2 = 3/8, x_3 = 7/16 \to 1/2\)。 易错点：压缩常数 \(q\) 必须严格小于 \(1\)。 \(d(f(x),f(y)) \leq d(x,y)\)（\(q = 1\)，非扩张映射）不保证不动点存在。

03Baire 范畴定理

🎯 为什么研究这个问题

"大多数连续函数处处不可微"——这种违反直觉的语句如何严格化？Baire 范畴定理提供了工具：在完备度量空间中，可数个"稠密开集"的交仍然 稠密——即"稠密"的集合"足够多"。反过来，可数个"无处稠密"集合 的并不能铺满整个空间。这是证明存在性的非构造性方法的典范。

🔍 抽象化方法

Baire 范畴引入"第一纲集"（meagre，可数个无处稠密集之并）与 "第二纲集"（非第一纲），为"大"/"小"提供了拓扑意义上的度量。

📐 公式推导 / 模型建立

Baire 范畴定理：设 \((X, d)\) 为完备度量空间（或局部紧 Hausdorff 空间）。 若 \(\{G_n\}_{n=1}^{\infty}\) 是可数个稠密开集，则 \[\bigcap_{n=1}^{\infty} G_n \text{ 在 } X \text{ 中稠密}\] 证明框架：设 \(B_0\) 为任意非空开球。 \(G_1\) 稠密 \(\Rightarrow G_1 \cap B_0 \neq \emptyset\)，取闭球 \(\overline{B_1} \subseteq G_1 \cap B_0\)，\(\mathrm{diam}(B_1) < 1\)。 递归：\(\overline{B_{n+1}} \subseteq G_{n+1} \cap B_n\)， \(\mathrm{diam}(B_n) < 1/n\)。 \(\{\overline{B_n}\}\) 是嵌套闭集列，直径趋于 \(0\)。 完备性（Cantor 交集定理）\(\Rightarrow \bigcap \overline{B_n} = \{x^*\}\)。 \(x^* \in \bigcap G_n \cap B_0\)。故 \(\bigcap G_n\) 与任何开球相交，即稠密。 推论：完备度量空间不是第一纲集（不能写成可数个无处稠密集之并）。

🌐 实际应用

- 证明 \(C[0,1]\) 中"大多数"函数处处不可微（\(C^1\) 函数是第一纲集） - 一致有界原理（Banach-Steinhaus）的证明基于 Baire 定理

✏️ 例题分析与易错点

例题：利用 Baire 定理证明 \(\mathbb{R}\) 不可数。 解答：假设 \(\mathbb{R} = \{r_1, r_2, \ldots\}\) 可数。 \(\{r_n\}\) 是闭集，\(\mathbb{R} \setminus \{r_n\}\) 是稠密开集。 Baire 定理：\(\bigcap_n (\mathbb{R} \setminus \{r_n\}) = \mathbb{R} \setminus \{r_1, r_2, \ldots\} = \emptyset\) 应该稠密（非空），但 \(= \emptyset\)，矛盾。 易错点：Baire 定理要求完备性。\(\mathbb{Q}\) 可以写成可数个 单点集之并（第一纲集），不矛盾（\(\mathbb{Q}\) 不完备）。

04完备化构造

🎯 为什么研究这个问题

不完备的度量空间总可以"补全"为完备空间——正如 \(\mathbb{Q}\) 补全 为 \(\mathbb{R}\)。完备化不仅是理论上的存在性结果，更是实数构造 （Cauchy 列方法）的普遍化。这个构造在泛函分析中广泛使用： 赋范空间的完备化给出 Banach 空间。

🔍 抽象化方法

完备化 = 等距嵌入到完备空间中使像稠密。唯一性由万有性质保证。 这类似于群的完备化（profinte completion）或域的完备化（\(p\)-adic 数）。

📐 公式推导 / 模型建立

构造：设 \((X, d)\) 为度量空间。 定义 Cauchy 列的等价关系：\((x_n) \sim (y_n) \iff d(x_n, y_n) \to 0\)。 令 \(\hat{X} = \{\text{Cauchy 列}\} / {\sim}\)。 度量：\(\hat{d}([(x_n)], [(y_n)]) = \lim_{n \to \infty} d(x_n, y_n)\)。 （极限存在：\(|d(x_m, y_m) - d(x_n, y_n)| \leq d(x_m, x_n) + d(y_m, y_n) \to 0\)。） 性质： (1) \((\hat{X}, \hat{d})\) 完备。 (2) \(\iota: X \to \hat{X}\)，\(\iota(x) = [(x, x, x, \ldots)]\) 是等距嵌入。 (3) \(\iota(X)\) 在 \(\hat{X}\) 中稠密。 万有性质：若 \(f: X \to Y\) 一致连续，\(Y\) 完备， 则 \(f\) 唯一延拓为 \(\hat{f}: \hat{X} \to Y\) 一致连续。 \[\hat{\mathbb{Q}} = \mathbb{R}\]

🌐 实际应用

- \(\mathbb{R}\) 的 Cauchy 构造：\(\hat{\mathbb{Q}} = \mathbb{R}\) - \(L^p\) 空间是可积函数空间在 \(\|\cdot\|_p\) 下的完备化

✏️ 例题分析与易错点

例题：描述 \(((0,1), |\cdot|)\) 的完备化。 解答：\((0,1)\) 与 \(\mathbb{R}\) 同胚但不等距。等距地看，\((0,1)\) 中的 Cauchy 列要么收敛到 \((0,1)\) 中的点，要么"收敛"到 \(0\) 或 \(1\)。 完备化 \(= [0,1]\)：添加两个端点。\(\iota: (0,1) \hookrightarrow [0,1]\) 是等距嵌入，像在 \([0,1]\) 中稠密。 易错点：完备化依赖于度量，不仅依赖于拓扑。 \((0,1)\) 在标准度量下完备化为 \([0,1]\)，但存在与 \(\mathbb{R}\) 等距的度量使完备化为 \(\mathbb{R}\)。
