Chapitre 7

完备度量空间

01Cauchy 序列与完备空间

🎯 为什么研究这个问题

在分析学与拓扑学的交汇处，一个核心问题反复出现：当一列元素彼此越来越接近时，它们是否最终"落在"空间之内？有理数 \(\mathbb{Q}\) 中序列 \(1, 1.4, 1.41, 1.414, \ldots\) 的每一项都是有理数，项与项之间的距离趋于零，但极限 \(\sqrt{2}\) 却"漏出"了 \(\mathbb{Q}\) 之外。完备性正是对这种缺口的修补——保证内在趋近一定导向实际收敛。

  • 数值分析：Newton 法、Picard 迭代等算法在每一步生成逼近序列；若底层空间不完备，迭代可能逼近一个"不存在的解"。
  • 泛函分析：函数空间 \(L^p\) 和 \(C(K)\) 的完备性是 Banach 空间理论的起点，一致有界原理、开映射定理都以完备性为前提。
  • 数论：\(\mathbb{Q}\) 在不同绝对值下的完备化产生实数 \(\mathbb{R}\) 和 \(p\)-adic 数 \(\mathbb{Q}_p\)，它们是现代数论的两大完备化窗口。

这些例子汇聚到同一个主题：完备性将 Cauchy 条件——一种纯内在的、不依赖极限点的收敛判据——提升为真正的收敛定理，使度量空间成为可靠的分析与计算平台。

🔍 抽象化方法

将收敛性从"极限存在"转化为"项间距离可控"是 Cauchy 条件的核心抽象。在一般拓扑空间中只能讨论极限（需要先有极限点），而度量空间中 Cauchy 条件 \(d(x_m, x_n) < \varepsilon\) 自洽地衡量"趋近趋势"，无须引用待确认的极限。

  • 内在 vs 外在收敛：收敛是外在的（需要极限点 \(L\)），Cauchy 是内在的（仅需空间内信息 \(d(x_m, x_n)\)）。完备性等价于两者重合。
  • 闭性 ↔ 完备性：完备空间的闭子集完备，完备子集在更大空间中是闭的。因此在度量空间中"子空间完备 \(\iff\) 子空间闭"。
  • 绝对收敛判别：\(\sum d(x_{n+1}, x_n) < \infty\) ⟹ \((x_n)\) 是 Cauchy 列，在完备空间中收敛——类似级数的绝对收敛判别。

闭性桥梁与绝对收敛判别为后续的 Banach 不动点定理和 Baire 范畴定理提供了统一的推理基础。

📐 公式推导 / 模型建立

1. Cauchy 序列

\((x_n)\) 在度量空间 \((X, d)\) 中是 Cauchy 列：

\[\forall \varepsilon > 0,\; \exists N \in \mathbb{N},\; \forall m, n \geq N:\; d(x_m, x_n) < \varepsilon\]

2. 完备空间的定义

\((X, d)\) 完备 \(\iff\) 每个 Cauchy 列在 \(X\) 中收敛。收敛序列一定是 Cauchy 列（三角不等式），反方向恰恰需要完备性。

3. 基本事实

  • \(\mathbb{R}\) 完备（由确界原理/单调有界定理证明）。\(\mathbb{R}^n\) 完备（分量逐个收敛）。
  • 完备空间的闭子集完备；完备子集在更大空间中是闭的。
  • \(\mathbb{Q}\) 不完备：\(\sqrt{2}\) 的十进制逼近是 Cauchy 列但极限不在 \(\mathbb{Q}\) 中。
  • \((0,1)\) 不完备：\(x_n = 1/n\) 是 Cauchy 列但 \(0 \notin (0,1)\)。

4. Cantor 交集定理

在完备度量空间中，嵌套闭集列 \(F_1 \supseteq F_2 \supseteq \cdots\)，若 \(\mathrm{diam}(F_n) \to 0\)，则 \(\bigcap F_n\) 恰好有一个点。这是 Baire 定理证明的关键工具。

🌐 实际应用

泛函分析：完备赋范空间（Banach 空间）如 \(L^p\)、\(\ell^p\)、\(C(K)\) 是现代分析的基础框架。Hilbert 空间 \(L^2\) 是量子力学态空间的数学模型，其完备性保证了谱分解的收敛性。

数值计算：Newton 法、Picard 迭代、梯度下降等算法生成的逼近序列在完备空间中满足 Cauchy 条件即可断言收敛；若空间不完备，算法可能逼近一个不存在于空间中的点。

数论与代数几何：\(\mathbb{Q}\) 关于 \(p\)-adic 绝对值的完备化得到 \(\mathbb{Q}_p\)，它是局部域理论和 Hasse 原理的基石，在二次型分类与椭圆曲线有理点判定中发挥关键作用。

✏️ 例题分析与易错点

例题 1：\((C[0,1], d_\infty)\) 的完备性

题目：证明连续函数空间 \((C[0,1], d_\infty)\)（\(d_\infty(f,g) = \sup_{x \in [0,1]} |f(x) - g(x)|\)）是完备的。

第一步 — 逐点极限：设 \((f_n)\) 为 Cauchy 列。对每个 \(x \in [0,1]\)，\((f_n(x))\) 是 \(\mathbb{R}\) 中的 Cauchy 列（\(|f_m(x) - f_n(x)| \leq d_\infty(f_m, f_n)\)），由 \(\mathbb{R}\) 完备性得 \(f_n(x) \to f(x)\)。

第二步 — 一致收敛：Cauchy 条件给出 \(\forall \varepsilon, \exists N, \sup|f_m - f_n| < \varepsilon\)。令 \(m \to \infty\)，得 \(\sup|f - f_n| \leq \varepsilon\)。故收敛是一致的。

第三步 — 极限连续：一致极限保连续，所以 \(f \in C[0,1]\)，即 \(f_n \xrightarrow{d_\infty} f \in C[0,1]\)。✔

例题 2：\((0,1)\) 不完备

题目：证明 \(((0,1), |\cdot|)\) 在标准度量下不完备。

第一步 — 构造 Cauchy 列：取 \(x_n = 1/n\)。对 \(m > n\)，\(|x_m - x_n| = |1/m - 1/n| \leq 1/n \to 0\)，故 \((x_n)\) 是 Cauchy 列。

第二步 — 极限不在空间内：\(\lim_{n \to \infty} x_n = 0\)，但 \(0 \notin (0,1)\)。存在 Cauchy 列不收敛于 \((0,1)\) 中的点，故 \((0,1)\) 不完备。✔

常见错误

  • ❌ 混淆逐点收敛与一致收敛，认为逐点极限连续就完成了证明。
    ✅ 逐点极限不保证连续性（如 \(f_n(x) = x^n\) 逐点收敛到不连续函数）。必须证明一致收敛。
  • ❌ 认为"去掉端点不影响完备性"，断言 \((C(0,1), d_\infty)\) 也完备。
    ✅ \((0,1)\) 上的连续函数可以无界，\(d_\infty\) 可能不是有限度量。需要指定有界连续函数 \(C_b(0,1)\) 才完备。
  • ❌ 认为 Cauchy 列一定有界。
    ✅ 在度量空间中 Cauchy 列确实有界（正确！），但在不完备空间中不一定收敛。有界 ≠ 收敛。

02Banach 不动点定理

🎯 为什么研究这个问题

许多问题最终归结为寻找一个"自洽的状态"——方程 \(f(x) = x\) 的解。当一个系统反复将自身输出作为下一步输入时，它是否会稳定下来？Banach 不动点定理给出了迄今最简洁的回答：只要映射足够"压缩"且空间完备，不动点就存在、唯一，而且可以通过迭代精确逼近。

  • 微分方程：Picard-Lindelöf 定理将 ODE \(y' = f(t,y)\) 转化为积分算子的不动点问题，由 Banach 定理在小区间上保证解的存在唯一。
  • 数值分析：Newton 法、不动点迭代、梯度下降的收敛性分析都以 Banach 定理为理论骨架——压缩常数 \(q\) 直接控制收敛速率。
  • 经济学与博弈论：动态规划中的 Bellman 方程 \(V = TV\) 在折扣因子 \(\gamma < 1\) 下定义压缩算子，Banach 定理保证最优值函数的唯一存在和值迭代的收敛。

Banach 定理的力量在于其构造性：不同于 Brouwer 不动点定理（只断言存在），它给出显式迭代序列 \(x_{n+1} = f(x_n)\) 并附带先验误差估计，可直接用于算法设计。

🔍 抽象化方法

Banach 不动点定理的抽象核心是"几何衰减 + 完备性 = 收敛"。映射每次将点对之间的距离至少缩小固定比例 \(q < 1\)，迭代序列的相邻距离以几何级数衰减，因此构成 Cauchy 列；完备性则保证极限存在。

  • 几何级数估计：\(d(x_{n+1}, x_n) \leq q^n d(x_1, x_0)\)——衰减速率由 \(q\) 决定，\(q\) 越小收敛越快。
  • 唯一性——"压缩杀死分叉"：若有两个不动点 \(x^*, y^*\)，则 \(d(x^*, y^*) = d(f(x^*), f(y^*)) \leq q \cdot d(x^*, y^*)\)，\(q < 1\) 迫使 \(d = 0\)。
  • 误差控制：先验估计 \(d(x_n, x^*) \leq \frac{q^n}{1 - q} d(x_1, x_0)\)；后验估计 \(d(x_n, x^*) \leq \frac{q}{1 - q} d(x_n, x_{n-1})\)，后者更实用。

这三个视角——收敛、唯一、误差——共同构成了 Banach 定理在理论与计算中双重有效的原因。

📐 公式推导 / 模型建立

1. 定理陈述

设 \((X, d)\) 完备，\(f: X \to X\) 满足 \(d(f(x), f(y)) \leq q \cdot d(x, y)\)（\(0 \leq q < 1\)）。则 \(f\) 有唯一不动点 \(x^*\)，且 \(\forall x_0 \in X\)，\(x_n = f^n(x_0) \to x^*\)。

2. Cauchy 列论证

取任意 \(x_0\)，令 \(x_n = f(x_{n-1})\)。由压缩性：

\[d(x_{n+1}, x_n) \leq q \cdot d(x_n, x_{n-1}) \leq \cdots \leq q^n \cdot d(x_1, x_0)\]

对 \(m > n\)，三角不等式给出：

\[d(x_m, x_n) \leq \sum_{k=n}^{m-1} d(x_{k+1}, x_k) \leq \frac{q^n}{1 - q} d(x_1, x_0) \to 0\]

故 \((x_n)\) 是 Cauchy 列，完备性 \(\Rightarrow x_n \to x^*\)。

3. 不动点验证与唯一性

连续性：\(f(x^*) = f(\lim x_n) = \lim f(x_n) = \lim x_{n+1} = x^*\)。唯一性：若 \(f(y^*) = y^*\)，则 \(d(x^*, y^*) = d(f(x^*), f(y^*)) \leq q \cdot d(x^*, y^*)\)，因 \(q < 1\) 得 \(d(x^*, y^*) = 0\)。

4. 误差估计

先验估计将迭代次数与精度关联：

\[d(x_n, x^*) \leq \frac{q^n}{1 - q}\, d(x_1, x_0)\]

这来自 \(d(x_n, x^*) \leq \sum_{k=n}^{\infty} d(x_{k+1}, x_k) \leq q^n d(x_1,x_0) \cdot \frac{1}{1-q}\)。后验估计更实用：\(d(x_n, x^*) \leq \frac{q}{1-q}\,d(x_n, x_{n-1})\)，无需回溯到初始步。两个估计都表明收敛是几何速率的。

🌐 实际应用

微分方程：Picard-Lindelöf 定理将 ODE \(y' = f(t, y)\) 转化为积分方程 \(y(t) = y_0 + \int_0^t f(s, y(s))\,ds\)，Lipschitz 条件保证 Picard 算子在足够小的区间上是压缩映射，Banach 定理给出局部解的存在唯一。

隐函数定理：一种经典证法将 \(F(x, y) = 0\) 转化为不动点问题 \(y = g(x, y)\)，通过 Banach 定理在 \(F_y\) 可逆的条件下得到局部解 \(y = \varphi(x)\)。

动态规划：Bellman 方程 \(V = TV\) 中的 Bellman 算子在折扣因子 \(\gamma < 1\) 下是 \(\sup\)-范数空间上的压缩映射，Banach 定理保证最优值函数唯一存在，且值迭代算法收敛。

✏️ 例题分析与易错点

例题 1：简单压缩映射

题目：\(f: [0,1] \to [0,1]\)，\(f(x) = x/2 + 1/4\)。证明唯一不动点并求之。

第一步 — 验证压缩性：\(|f(x) - f(y)| = |x - y|/2\)，压缩常数 \(q = 1/2 < 1\)。✔

第二步 — 完备性：\([0,1]\) 是 \(\mathbb{R}\) 的闭子集，故完备。✔

第三步 — 不动点：解 \(x = x/2 + 1/4\) 得 \(x^* = 1/2\)。从 \(x_0 = 0\) 迭代：\(x_1 = 1/4,\; x_2 = 3/8,\; x_3 = 7/16 \to 1/2\)。✔

例题 2：\(\cos x\) 在 \([0,1]\) 上的不动点

题目：证明方程 \(x = \cos x\) 在 \([0,1]\) 上有唯一解，并估计收敛速率。

第一步 — 映射到自身：\(\cos([0,1]) = [\cos 1, 1] \approx [0.54, 1] \subseteq [0,1]\)。✔

第二步 — 压缩常数：中值定理给出 \(|\cos x - \cos y| = |\sin c|\,|x - y|\)（\(c \in [0,1]\)），\(\sup |\sin c| = \sin 1 \approx 0.841 < 1\)，故 \(q = \sin 1\)。✔

第三步 — 结论：Banach 定理保证唯一不动点 \(x^* \approx 0.739\)。先验估计：10 次迭代后误差 \(\leq \frac{(\sin 1)^{10}}{1 - \sin 1}\,|x_1 - x_0| \approx 0.01\)。✔

常见错误

  • ❌ 将 \(q = 1\)（非扩张映射）的情况也套用 Banach 定理。
    ✅ 压缩常数必须严格小于 1。例：\(f(x) = x + 1\) 在 \(\mathbb{R}\) 上满足 \(d(f(x), f(y)) = d(x, y)\) 但没有不动点。
  • ❌ 忘记检验空间的完备性。
    ✅ \(f: (0,1) \to (0,1),\; f(x) = x/2\) 是压缩映射，但 \((0,1)\) 上不动点 \(0 \notin (0,1)\)，空间不完备导致定理失效。
  • ❌ 认为"映射到自身 + 连续"就够了。
    ✅ Brouwer 定理（紧凸集上连续自映射有不动点）不保证唯一性，且不给出收敛序列。Banach 定理需要全局压缩条件。

03Baire 范畴定理

🎯 为什么研究这个问题

"大多数连续函数处处不可微"——这个违反直觉的命题该如何严格理解？Baire 范畴定理为"大多数"提供了精确的拓扑语义：在完备度量空间中，可数个稠密开集的交仍然稠密。这意味着"例外集"（第一纲集）虽然可以很大（甚至满测），但在拓扑意义上始终是"薄"的。

  • 泛函分析：一致有界原理、开映射定理、闭图像定理——泛函分析三大基石全部以 Baire 定理为证明核心。
  • 实分析：Baire 定理证明"典型"连续函数处处不可微，"典型" \(L^1\) 函数的 Fourier 级数在某点发散——反例不是偶然现象，而是"普遍"现象。
  • 集合论与描述集合论：Baire 范畴方法是构造"病态"对象（处处不连续函数、处处不收敛级数）的标准非构造性工具，与 Lebesgue 测度论形成互补。

Baire 定理的深刻之处在于：它揭示了完备度量空间的内在"厚度"——完备空间不可能被可数个"碎片"（无处稠密闭集）拼出，总有大量剩余。

🔍 抽象化方法

Baire 定理的抽象框架建立在"集合的拓扑大小"概念上。与测度论用"零测 vs 满测"衡量大小不同，范畴论用"第一纲 vs 第二纲"来衡量——两个标尺独立且互补。

  • 第一纲集 (meagre)：可数个无处稠密集的并，拓扑意义上的"小"集。例：\(\mathbb{Q}\) 在 \(\mathbb{R}\) 中是第一纲的。
  • 第二纲集：非第一纲集。Baire 定理断言完备度量空间本身是第二纲的。
  • 测度 vs 范畴：两个概念不等价——存在满测的第一纲集（如适当的 Cantor 型构造），也存在零测的第二纲集。

残差集（comeager / residual set）——可数个稠密开集交所得的稠密 \(G_\delta\) 集——在范畴意义下扮演"几乎全部"的角色，正如满测集在测度论中的地位。

📐 公式推导 / 模型建立

1. 定理陈述

设 \((X, d)\) 为完备度量空间（或局部紧 Hausdorff 空间）。若 \(\{G_n\}_{n=1}^\infty\) 是可数个稠密开集，则

\[\bigcap_{n=1}^{\infty} G_n \;\text{在}\; X \;\text{中稠密}\]

2. 证明（嵌套闭球法）

设 \(B_0\) 为任意非空开球。\(G_1\) 稠密 \(\Rightarrow G_1 \cap B_0 \neq \emptyset\)。取闭球 \(\overline{B_1} \subseteq G_1 \cap B_0\)，\(\mathrm{diam}(B_1) < 1\)。

递归：选 \(\overline{B_{n+1}} \subseteq G_{n+1} \cap B_n\)，\(\mathrm{diam}(B_n) < 1/n\)。

\(\{\overline{B_n}\}\) 是嵌套闭集列，直径趋于 0。由完备性（Cantor 交集定理）：\(\bigcap \overline{B_n} = \{x^*\}\)。

\(x^* \in \overline{B_n} \subseteq G_n\) 对所有 \(n\)，且 \(x^* \in B_0\)。故 \(\bigcap G_n\) 与任意开球相交，即稠密。✔

3. 等价形式

  • 闭集版本：完备度量空间不是第一纲集——即不能写成可数个无处稠密闭集的并。

4. 经典推论

  • \(\mathbb{R}\) 不可数：若 \(\mathbb{R} = \{r_n\}\)，则 \(\mathbb{R} = \bigcup \{r_n\}\) 是可数个无处稠密集之并（第一纲），与 Baire 定理矛盾。
  • 稠密 \(G_\delta\) 的稳定性：可数个稠密 \(G_\delta\) 集的交仍是稠密 \(G_\delta\)——因为可数个可数交仍是可数交。
🌐 实际应用

泛函分析：一致有界原理（Banach-Steinhaus）——若 \(\{T_\alpha\}\) 是 Banach 空间上的有界线性算子族且逐点上界有限，则一致有界。证明将 \(\{x : \sup_\alpha \|T_\alpha x\| \leq n\}\) 写成闭集并，用 Baire 定理保证至少一个有内点。

实分析：\(C[0,1]\) 中处处可微函数集是第一纲集，因此残差集中的函数处处不可微——"大多数"连续函数处处不可微。这为 Weierstrass 函数的存在提供了更强的范畴论证。

调和分析：Carleson 定理之前，du Bois-Reymond 利用 Baire 方法证明存在连续函数其 Fourier 级数在某点发散——该发散点集恰好是第一纲集的补。

✏️ 例题分析与易错点

例题 1：利用 Baire 定理证明 \(\mathbb{R}\) 不可数

假设：\(\mathbb{R} = \{r_1, r_2, \ldots\}\) 可数。

第一步：每个单点集 \(\{r_n\}\) 是闭集，其补 \(G_n = \mathbb{R} \setminus \{r_n\}\) 是稠密开集（去掉一个点不影响稠密性）。

第二步：Baire 定理 \(\Rightarrow \bigcap_n G_n = \mathbb{R} \setminus \{r_1, r_2, \ldots\} = \emptyset\) 应稠密。

第三步：但 \(\emptyset\) 不稠密——矛盾！故 \(\mathbb{R}\) 不可数。✔

例题 2：\(\mathbb{Q}\) 是 \(\mathbb{R}\) 中的第一纲集

题目：证明 \(\mathbb{Q}\) 在 \(\mathbb{R}\) 中是第一纲集（meagre）。

第一步：枚举 \(\mathbb{Q} = \{q_1, q_2, \ldots\}\)，则 \(\mathbb{Q} = \bigcup_{n=1}^\infty \{q_n\}\)。

第二步：每个单点集 \(\{q_n\}\) 的闭包是自身，内部为空，故无处稠密。

第三步：\(\mathbb{Q}\) 是可数个无处稠密集的并，即第一纲集。由 Baire 定理，\(\mathbb{R}\) 本身不是第一纲的，所以 \(\mathbb{R} \setminus \mathbb{Q}\) 不能也是第一纲的（否则 \(\mathbb{R}\) 是两个第一纲集的并，仍为第一纲，矛盾）。✔

常见错误

  • ❌ 对 \(\mathbb{Q}\) 套用 Baire 定理得出矛盾。
    ✅ \(\mathbb{Q}\) 不完备，Baire 定理不适用。\(\mathbb{Q}\) 确实是第一纲集（在自身中可数个单点集的并）。
  • ❌ 混淆"无处稠密"与"不稠密"。
    ✅ \(\mathbb{Z}\) 在 \(\mathbb{R}\) 中不稠密但无处稠密（其闭包 = 自身没有内点）。\(\mathbb{Q}\) 在 \(\mathbb{R}\) 中稠密，不是无处稠密的。
  • ❌ 认为第一纲集 = 零测集。
    ✅ 两个概念独立。存在满测的第一纲集（如 Cantor 型构造），也存在零测的第二纲集。
⚡ 快速检测
在完备度量空间中，以下哪个是第一纲集？

04完备化构造

🎯 为什么研究这个问题

当一个度量空间不完备时，我们能否系统性地"补齐"它所缺的极限点，使所有 Cauchy 列都收敛？实数的诞生就是对这一问题最经典的回答——Cauchy 用有理数列的等价类"填充"了 \(\mathbb{Q}\) 的缺口，构造出 \(\mathbb{R}\)。完备化理论将这一思想推广到任意度量空间。

  • 实分析基础：\(\mathbb{R}\) 的 Cauchy 构造本质就是 \(\mathbb{Q}\) 的完备化，这是实数严格定义的两种经典方法之一（另一种是 Dedekind 分割）。
  • 泛函分析：赋范空间的完备化给出 Banach 空间，内积空间的完备化给出 Hilbert 空间。\(L^p\) 空间就是可积函数在 \(\|\cdot\|_p\) 下的完备化。
  • 数论：\(\mathbb{Q}\) 关于 \(p\)-adic 绝对值的完备化给出 \(\mathbb{Q}_p\)，它与 \(\mathbb{R}\) 一起穷尽了 \(\mathbb{Q}\) 上所有绝对值的完备化（Ostrowski 定理）。

完备化的万有性质保证了构造的唯一性与函数的可延拓性：任何到完备空间的一致连续映射都能唯一地"穿过"完备化延拓，使得不完备空间上的分析无缝传递到完备框架中。

🔍 抽象化方法

完备化的核心思想是：不完备空间中"缺失的极限点"可以用 Cauchy 列的等价类来代表。两个 Cauchy 列若尾部距离趋于零，就视为同一个"缺失点"。这与 Dedekind 分割、\(p\)-adic 数的构造在精神上完全一致。

  • 万有性质：完备化 \((\hat{X}, \iota)\) 由万有性质唯一确定——任何到完备空间的一致连续映射唯一地延拓过 \(\hat{X}\)。
  • 等价类构造：Cauchy 列 modulo "尾部趋零"等价关系给出新空间的"点"，原空间通过常数列嵌入，像是稠密子集。
  • 稠密嵌入：完备化 = "填补所有 Cauchy 列的极限"，原空间的每个点保持不动，新增点恰好是缺失的极限。

万有性质不仅证明完备化在等距意义下唯一，还保证了完备化上的代数结构（范数、内积）可以自然延拓，这正是 \(L^p\)、Hilbert 空间构造的理论基石。

📐 公式推导 / 模型建立

1. 等价关系与点的构造

设 \((X, d)\) 为度量空间。Cauchy 列上的等价关系：

\[(x_n) \sim (y_n) \;\iff\; \lim_{n \to \infty} d(x_n, y_n) = 0\]

令 \(\hat{X} = \{\text{Cauchy 列}\} / {\sim}\)，典型元素 \([(x_n)]\)。

2. 度量的定义

\[\hat{d}\big([(x_n)],\, [(y_n)]\big) = \lim_{n \to \infty} d(x_n, y_n)\]

极限存在性：\(|d(x_m, y_m) - d(x_n, y_n)| \leq d(x_m, x_n) + d(y_m, y_n) \to 0\)（两个 Cauchy 列）。良定义性（与代表元无关）由等价关系定义保证。

3. 三大性质

  • 完备性：\((\hat{X}, \hat{d})\) 是完备的（Cauchy 列的 Cauchy 列收敛到"对角线"取的列）。
  • 等距嵌入：\(\iota: X \to \hat{X}\)，\(\iota(x) = [(x, x, x, \ldots)]\)。下面 \(\hat{d}(\iota(x), \iota(y)) = d(x, y)\)。
  • 像稠密：\(\iota(X)\) 在 \(\hat{X}\) 中稠密。

4. 万有性质

若 \(f: X \to Y\) 一致连续，\(Y\) 完备，则 \(f\) 唯一延拓为 \(\hat{f}: \hat{X} \to Y\)：\(\hat{f}([(x_n)]) = \lim f(x_n)\)。

🌐 实际应用

实分析：\(\hat{\mathbb{Q}} = \mathbb{R}\)——这是实数严格定义的经典方法之一。每个实数对应一个有理数 Cauchy 列的等价类，加法和乘法通过代表元逐项运算定义。

泛函分析：可测函数空间在 \(\|f\|_p = (\int |f|^p)^{1/p}\) 下的完备化（商去零测等价后）给出 \(L^p\) 空间。Riesz-Fischer 定理即 \(L^p\) 的完备性。

数论：\(\mathbb{Q}\) 在 \(p\)-adic 绝对值下的完备化得到 \(\mathbb{Q}_p\)，其中 \(p\)-adic 整数环 \(\mathbb{Z}_p\) 是紧致、完全不连通的——与 \(\mathbb{R}\) 的拓扑截然不同，这使得 \(p\)-adic 分析成为独立于经典实分析的分支。

✏️ 例题分析与易错点

例题 1：\((0,1)\) 的完备化

题目：描述 \(((0,1), |\cdot|)\) 在标准度量下的完备化。

第一步 — 识别"缺失的极限"：\((0,1)\) 中的 Cauchy 列要么收敛到 \((0,1)\) 中的点，要么逼近端点 \(0\) 或 \(1\)。

第二步 — 完备化空间：添加两个端点得 \([0,1]\)。嵌入 \(\iota: (0,1) \hookrightarrow [0,1]\) 是等距的，像在 \([0,1]\) 中稠密。\([0,1]\) 是 \(\mathbb{R}\) 的闭子集，故完备。✔

结论：\(\widehat{(0,1)} \cong [0,1]\)（等距同构）。

例题 2：离散度量空间的完备化

题目：设 \(X\) 为任意集合，\(d(x,y) = 1\)（\(x \neq y\)）。描述 \((X, d)\) 的完备化。

第一步 — 分析 Cauchy 列：取 \(\varepsilon = 1/2\)，Cauchy 条件要求 \(d(x_m, x_n) < 1/2\)（\(m, n > N\)），但 \(d\) 只取 \(0\) 或 \(1\)，故 \(x_m = x_n\)——Cauchy 列最终是常数列。

第二步 — 结论：常数列已经收敛，无需添加新点。\(\hat{X} = X\)，离散度量空间本身就是完备的。✔

常见错误

  • ❌ 认为完备化只依赖于拓扑（同胚类型）。
    ✅ 完备化依赖于度量。\((0,1)\) 同胚于 \(\mathbb{R}\)，但标准度量下完备化为 \([0,1]\)，而存在等距于 \(\mathbb{R}\) 的度量使完备化为 \(\mathbb{R}\)。
  • ❌ 认为完备空间的完备化会产生更大的空间。
    ✅ 完备空间本身就是自己的完备化（\(\hat{X} = X\)），不添加新点。
  • ❌ 忘记验证等价关系的良定义性。
    ✅ 需验证 \(\hat{d}\) 不依赖于 Cauchy 列代表元的选取：若 \((x_n) \sim (x'_n)\)，\((y_n) \sim (y'_n)\)，则 \(\lim d(x_n, y_n) = \lim d(x'_n, y'_n)\)。
