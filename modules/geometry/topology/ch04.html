01子空间拓扑与性质遗传

🎯 为什么研究这个问题

当我们研究一个大空间中的子集时，自然希望子集"继承"环境空间的拓扑 结构。子空间拓扑（诱导拓扑）正是实现这一目的的标准方式。核心问题 是：哪些拓扑性质能遗传给子空间？Hausdorff 性可以遗传，但紧致性 只遗传给闭子集——这些细微差别在实际应用中至关重要。

🔍 抽象化方法

子空间拓扑是初始拓扑的特例：关于包含映射 \(\iota: A \hookrightarrow X\) 的初始拓扑。

📐 公式推导 / 模型建立

定义：\((X, \tau)\) 为拓扑空间，\(A \subseteq X\)。 子空间拓扑为 \(\tau_A = \{U \cap A : U \in \tau\}\)。 验证是拓扑： (T1) \(\emptyset = \emptyset \cap A\)，\(A = X \cap A\)。 (T2) \((U_1 \cap A) \cap (U_2 \cap A) = (U_1 \cap U_2) \cap A\)。 (T3) \(\bigcup (U_\alpha \cap A) = (\bigcup U_\alpha) \cap A\)。 性质遗传： - \(T_2\) 遗传：\(\forall A \subseteq X\)，\((A, \tau_A)\) 仍 Hausdorff。 - \(C_1, C_2\) 遗传。 - 紧致性不完全遗传：\((0,1) \subseteq [0,1]\) 但 \((0,1)\) 不紧。 - 紧致性遗传到闭子集：\(A\) 闭 + \(X\) 紧 \(\Rightarrow\) \(A\) 紧。 \[\text{闭子集 of 紧空间} \Rightarrow \text{紧}\]

🌐 实际应用

- 在数值分析中，约束优化的可行域通常是 \(\mathbb{R}^n\) 的闭子空间

✏️ 例题分析与易错点

例题：\(A = [0,1] \cap \mathbb{Q}\) 在 \(\mathbb{R}\) 的子空间拓扑中 是否紧致？ 解答：不紧致。\([0,1] \cap \mathbb{Q}\) 不是 \(\mathbb{R}\) 中的闭集 （\(\sqrt{2}/2 \in \overline{A}\) 但 \(\sqrt{2}/2 \notin A\)）。 具体开覆盖：对每个 \(q \in A\)，取不含 \(\sqrt{2}/2\) 的小开区间。 或者更直接：\(A\) 不是 \(\mathbb{R}\) 中有界闭集，所以在 \(\mathbb{R}\) 中不紧。子空间拓扑下 \(A\) 的紧致性与环境空间一致。 易错点：误以为 \(A \subseteq [0,1]\)（有界）就紧致。 Heine-Borel 要求"有界且闭"，\(A\) 不闭。

02积拓扑

🎯 为什么研究这个问题

如何在笛卡尔积 \(\prod X_\alpha\) 上定义"自然"的拓扑？最直觉的 box 拓扑（每个分量取开集的积）在无穷积时性质太差。Tychonoff 选择了积拓扑（只允许有限多个分量限制），使得投影连续且积空间具有 良好的性质——尤其是 Tychonoff 定理（紧空间的积仍紧）。

🔍 抽象化方法

积拓扑 = 关于所有投影映射 \(\pi_\alpha\) 的初始拓扑。 这是范畴积(categorical product)在 \(\mathbf{Top}\) 中的实现。

📐 公式推导 / 模型建立

有限积：\(X_1 \times X_2\) 上的积拓扑以 \(\{U_1 \times U_2 : U_i \in \tau_i\}\) 为基。 此时 box 拓扑 = 积拓扑。 无穷积：\(\prod_{\alpha \in A} X_\alpha\) 上的积拓扑以 \[\pi_{\alpha_1}^{-1}(U_1) \cap \cdots \cap \pi_{\alpha_n}^{-1}(U_n)\] （有限交）为基，其中 \(U_i \in \tau_{\alpha_i}\)。 即只有有限多个分量被"限制"。 box 拓扑 vs 积拓扑：\(\prod U_\alpha\)（所有分量开）为 box 基。 无穷积时 box 严格细于 product。 关键区别：在 box 拓扑下，\(f: Y \to \prod X_\alpha\) 连续要求 每个 \(\pi_\alpha \circ f\) 连续且更多；在积拓扑下， \[f \text{ 连续} \iff \forall \alpha,\; \pi_\alpha \circ f \text{ 连续}\]

🌐 实际应用

- 函数空间 \(Y^X = \prod_{x \in X} Y\) 的积拓扑 = 逐点收敛拓扑 - 概率论中无穷乘积测度空间的基础

✏️ 例题分析与易错点

例题：设 \(f_n: \mathbb{R} \to \mathbb{R}\)，\(f_n(x) = x/n\)。 证明 \(F = (f_1, f_2, \ldots): \mathbb{R} \to \mathbb{R}^{\mathbb{N}}\) 在积拓扑下连续，但在 box 拓扑下不连续。 解答：积拓扑：\(\pi_n \circ F = f_n\) 连续 \(\forall n\)，故 \(F\) 连续。 Box 拓扑：取 \(W = \prod_n (-1/n, 1/n)\)，\(W\) 在 box 拓扑中开。 \(F^{-1}(W) = \{x : |x/n| < 1/n,\; \forall n\} = \{x : |x| < 1\} = (-1,1)\)。 等等，这看似是开的。修改例子：取 \(f_n(x) = x\)，\(F = (x,x,\ldots)\)， \(W = \prod_n (-1/n, 1/n)\)，则 \(F^{-1}(W) = \bigcap_n (-1/n,1/n) = \{0\}\)， 不是开集。 易错点：box 拓扑在无穷积中使太多映射不连续，这正是不选它的原因。

03商拓扑与商映射

🎯 为什么研究这个问题

许多重要空间来自"粘合"操作：将正方形的对边粘合得环面、将圆盘 的边界粘成一点得球面。商拓扑将等价关系下的"粘合"严格化：商空 间 \(X/{\sim}\) 继承了使商映射连续的最强拓扑。这是构造流形、轨 道空间和 CW 复形的基本手段。

🔍 抽象化方法

商拓扑 = 关于商映射 \(q: X \to X/{\sim}\) 的终止拓扑。

📐 公式推导 / 模型建立

定义：设 \(\sim\) 为 \(X\) 上的等价关系，\(q: X \to X/{\sim}\) 为商映射 \(q(x) = [x]\)。商拓扑为 \[\tau_{X/\sim} = \{U \subseteq X/{\sim} : q^{-1}(U) \in \tau_X\}\] 即 \(U\) 开当且仅当其原像在 \(X\) 中开。 商映射的判定：\(f: X \to Y\) 满射。\(f\) 是商映射 \(\iff\) \(V \subseteq Y\) 开 \(\iff f^{-1}(V) \subseteq X\) 开。 充分条件：\(f\) 满射 + 开映射，或 \(f\) 满射 + 闭映射。 性质：商映射不一定是开映射也不一定是闭映射。 万有性质：\(g: X/{\sim} \to Z\) 连续 \(\iff g \circ q\) 连续。

🌐 实际应用

- 群作用的轨道空间 \(X/G\) 是商空间（如 \(\mathbb{R}/\mathbb{Z} \cong S^1\)） - CW 复形的构造基于粘合映射和商拓扑

✏️ 例题分析与易错点

例题：证明 \(S^1 \cong [0,1]/\{0 \sim 1\}\)。 解答：定义 \(f: [0,1] \to S^1\)，\(f(t) = e^{2\pi i t}\)。 \(f\) 连续、满射。\(f(0) = f(1) = 1\)，且 \(f\) 在 \((0,1)\) 上单射。 \(f\) 与等价关系 \(0 \sim 1\) 相容，诱导双射 \(\bar{f}: [0,1]/{\sim} \to S^1\)。 \(\bar{f}\) 连续（\(\bar{f} \circ q = f\) 连续，由万有性质）。 \([0,1]\) 紧 \(\Rightarrow [0,1]/{\sim}\) 紧（连续像）， \(S^1\) Hausdorff，因此紧到 Hausdorff 的连续双射是同胚。 易错点：忘记验证"紧到 Hausdorff"条件。不能直接说连续双射=同胚。

04典型构造实例与不交并

🎯 为什么研究这个问题

理论的价值在于其实例。莫比乌斯带、环面、射影空间等经典空间都 可以用子空间、积和商拓扑来严格构造。不交并（余积）则是"分离 地放在一起"的操作。这些例子既是几何直觉的训练场，也是后续 代数拓扑中计算不变量的主要对象。

📐 公式推导 / 模型建立

莫比乌斯带：\(M = [0,1] \times [-1,1] / {\sim}\)， \((0, y) \sim (1, -y)\)。粘合时翻转了第二坐标。 环面：\(T^2 = [0,1]^2 / {\sim}\)，\((x,0) \sim (x,1)\)， \((0,y) \sim (1,y)\)。同胚于 \(S^1 \times S^1\)。 射影空间：\(\mathbb{RP}^n = S^n / {\sim}\)，\(x \sim -x\) （对径点等价）。\(\mathbb{RP}^1 \cong S^1\)。 不交并 (Coproduct)：\(X \sqcup Y\) 的拓扑为 \(\tau_{X \sqcup Y} = \{U \cup V : U \in \tau_X, V \in \tau_Y\}\)。 等价地，\(A \subseteq X \sqcup Y\) 开 \(\iff\) \(A \cap X\) 在 \(X\) 中开且 \(A \cap Y\) 在 \(Y\) 中开。 性质： \[\pi_1(T^2) \cong \mathbb{Z}^2, \quad \pi_1(\mathbb{RP}^2) \cong \mathbb{Z}/2\mathbb{Z}\] \[\chi(T^2) = 0, \quad \chi(S^2) = 2, \quad \chi(\mathbb{RP}^2) = 1\]

🌐 实际应用

- 莫比乌斯带是纤维丛理论的入门例子（不可定向曲面） - 射影空间在计算机视觉中用于表示直线方向

✏️ 例题分析与易错点

例题：证明 \(\mathbb{RP}^1 \cong S^1\)。 解答：\(\mathbb{RP}^1 = S^1/{\sim}\)，\((x,y) \sim (-x,-y)\)。 定义 \(f: S^1 \to S^1\)，\(f(e^{i\theta}) = e^{2i\theta}\)。 则 \(f(e^{i\theta}) = f(e^{i(\theta+\pi)})\)，所以 \(f\) 与 \(\sim\) 相容。 诱导的 \(\bar{f}: \mathbb{RP}^1 \to S^1\) 是连续双射。 \(\mathbb{RP}^1\) 紧（\(S^1\) 紧的连续像），\(S^1\) Hausdorff， 故 \(\bar{f}\) 是同胚。 易错点：\(\mathbb{RP}^n \cong S^n\) 仅在 \(n=1\) 时成立！ \(n \geq 2\) 时 \(\mathbb{RP}^n \not\cong S^n\)（基本群不同）。
