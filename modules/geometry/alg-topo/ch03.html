单纯同调

01单纯复形与 \(\Delta\)-复形

🎯 为什么研究这个问题

要系统地研究拓扑空间中任意维度的"洞"，我们需要将空间离散化为可计算的组合对象。三角剖分——将空间分解为顶点、边、三角形等基本积木——正是实现这一目标的经典方法。

  • 检测高维洞：基本群只探测一维洞（回路），但空间可能有二维空腔（如球面）或更高维洞——需要系统方法处理所有维数。
  • 三角化 = 离散化：将空间拆解为顶点、边、三角形、四面体等"积木"（单纯形），把连续拓扑问题转化为有限组合问题。
  • Δ-复形的灵活性：传统单纯复形要求面互不粘合，\(\Delta\)-复形放宽限制——允许单纯形的面自我粘合，极大减少剖分所需的单纯形数量。

从单纯复形到 \(\Delta\)-复形，三角剖分为后续的链群和同调群的定义提供了组合基础，将连续拓扑问题转化为有限可算法化的代数问题。

🔍 抽象化方法
  • 组合化：单纯复形将拓扑空间转化为有限集合的集合（满足遗传性），用整数矩阵（边缘矩阵）的线性代数计算拓扑不变量。
  • 层级结构：\(\Delta^0 \subset \Delta^1 \subset \Delta^2 \subset \cdots\)——每个 \(n\)-单纯形有 \(n+1\) 个 \((n-1)\)-面，面映射 \(d_i\) 编码了组合结构。
📐 公式推导 / 模型建立

1. 标准 \(n\)-单纯形

\(\Delta^n = \{(t_0, \ldots, t_n) \in \mathbb{R}^{n+1} : \sum t_i = 1,\; t_i \geq 0\}\)。顶点 \(e_i = (0,\ldots,1,\ldots,0)\)，面 \([e_{i_0}, \ldots, e_{i_k}]\)。

2. 单纯复形

\(K\) 是集合 \(V\)（顶点集）的有限子集的集合，满足遗传性：\(\sigma \in K,\; \tau \subseteq \sigma \Rightarrow \tau \in K\)。几何实现 \(|K|\)：将每个 \(\sigma \in K\) 实现为 \(\Delta^{|\sigma|-1}\)，按面关系粘合。

3. \(\Delta\)-复形

更宽泛——允许面的自粘合（如三角形两条边粘成一条）。有向单纯形，每个 \(n\)-单纯形有 \(n+1\) 个面映射：

\[d_i[v_0, \ldots, v_n] = [v_0, \ldots, \hat{v}_i, \ldots, v_n]\]

\(S^1\) 可用 1 个顶点 + 1 条边构成 \(\Delta\)-复形（传统单纯复形至少需 3 顶点 3 边）。

🌐 实际应用

拓扑数据分析 (TDA)：给定点云数据，构造 Rips 复形或 Čech 复形，用持续同调追踪不同尺度下的拓扑特征（连通分量、环、空腔）。

计算几何：三角网格是计算机图形学中曲面表示的标准方法——Delaunay 三角剖分保证了良好的数值性质。

有限元法：偏微分方程的数值解法将区域剖分为单纯复形，在每个单纯形上用多项式逼近——同调保证了离散化的正确性。

✏️ 例题分析与易错点

例题：环面 \(T^2\) 的 \(\Delta\)-复形剖分

题目：给 \(T^2\) 一个最小的 \(\Delta\)-复形结构。

第一步 — 识别：取正方形 \([0,1]^2\)，对边识别（\((x,0) \sim (x,1)\)，\((0,y) \sim (1,y)\)）。所有四个角等价为同一点 \(v\)。

第二步 — 计数：最少需 1 个顶点（\(v\)）、3 条边（\(a\)=水平, \(b\)=垂直, \(c\)=对角线）、2 个三角形（上三角 \(\sigma_1\)、下三角 \(\sigma_2\)）。

第三步 — 面映射：\(\partial \sigma_1 = a + c - b\)，\(\partial \sigma_2 = b - c - a\)。验证 \(\partial(a) = \partial(b) = \partial(c) = v - v = 0\)。✔

常见错误

  • ❌ 混淆单纯复形和 \(\Delta\)-复形。
    ✅ 单纯复形不允许面自粘——\(T^2\) 的单纯复形至少需要 7 个顶点（远多于 \(\Delta\)-复形的 1 个）。
  • ❌ 忘记遗传性条件。
    ✅ 单纯复形中若三角形 \([v_0, v_1, v_2] \in K\)，则三条边和三个顶点必须也属于 \(K\)。
  • ❌ 对边识别时搞错方向。
    ✅ \(T^2\) 要求两对平行边同向识别。若一对反向，则得到 Klein 瓶。
⚡ 快速检测
标准 \(2\)-单纯形 \(\Delta^2\) 有多少个面（包括所有维数）？

02链复形与边缘算子

🎯 为什么研究这个问题

有了空间的三角剖分，下一步是将"边界"这一几何操作代数化。链群和边缘算子将"一个三角形的边界是三条边"这样的直觉翻译为线性代数运算，为定义同调群铺平了道路。

  • 代数化"洞"：链群 \(C_n\) 是 \(n\)-单纯形的形式整数组合，边缘算子 \(\partial_n\) 将每个单纯形映为其边界——这把几何概念翻译为线性代数。
  • \(\partial^2 = 0\) 产生同调：闭链（\(\ker \partial\)）包含边界链（\(\mathrm{im}\,\partial\)），商群 \(H_n = Z_n/B_n\) 精确刻画了"有洞但不是边界"的链。
  • 计算框架：链复形将拓扑计算化归为整数矩阵的秩与 Smith 标准形——完全可算法化。

链复形 \(\cdots \to C_n \xrightarrow{\partial} C_{n-1} \to \cdots\) 与核心性质 \(\partial^2 = 0\) 构成了同调论的代数骨架，使"洞"的概念从几何直觉升级为可精确计算的代数对象。

🔍 抽象化方法
  • 链复形范式：\(\cdots \xrightarrow{\partial_{n+1}} C_n \xrightarrow{\partial_n} C_{n-1} \to \cdots\) 是代数拓扑的核心结构，\(\partial^2=0\) 产生"cycles mod boundaries"。
  • 函子性：连续映射诱导链映射，链映射诱导同调映射——拓扑 → 代数的翻译保持结构。
📐 公式推导 / 模型建立

1. 链群

\(C_n(K) = \bigoplus_{\sigma \in K_n} \mathbb{Z} \cdot \sigma\)（有向 \(n\)-单纯形自由生成的交换群）。

2. 边缘算子

\[\partial_n[v_0, \ldots, v_n] = \sum_{i=0}^n (-1)^i [v_0, \ldots, \hat{v}_i, \ldots, v_n]\]

直觉：\(\partial\) 取边界，交替符号保证定向一致。

3. \(\partial^2 = 0\) 的证明

\(\partial_{n-1} \circ \partial_n\) 中，双重删除 \(\hat{v}_i\) 再 \(\hat{v}_j\)（\(i < j\)）出现两次，符号 \((-1)^{i+j}\) 和 \((-1)^{i+j-1}\) 恰好相消。

4. 链复形结构

\(\cdots \to C_2 \xrightarrow{\partial_2} C_1 \xrightarrow{\partial_1} C_0 \to 0\)。\(Z_n = \ker \partial_n\)（闭链），\(B_n = \mathrm{im}\,\partial_{n+1}\)（边界）。\(\partial^2=0\) 保证 \(B_n \subseteq Z_n\)。

🌐 实际应用

持续同调算法：TDA 中对点云按距离参数构造 filtration of simplicial complexes，追踪链复形的 \(\ker/\mathrm{im}\) 如何变化——高效算法基于边缘矩阵的列约化。

电路理论：Kirchhoff 电流定律 \(\sum I = 0\) 就是 \(\partial_1\) 的核条件；电压回路定律对应 \(\partial_1\) 的像——电路中的 mesh 分析本质是链复形。

✏️ 例题分析与易错点

例题：计算三角形的边缘算子

题目：计算 \(\partial_2[v_0, v_1, v_2]\) 并验证 \(\partial_1 \circ \partial_2 = 0\)。

第一步 — 边缘：\(\partial_2[v_0, v_1, v_2] = [v_1, v_2] - [v_0, v_2] + [v_0, v_1]\)（三条有向边）。

第二步 — 验证：\(\partial_1([v_1,v_2] - [v_0,v_2] + [v_0,v_1]) = (v_2-v_1) - (v_2-v_0) + (v_1-v_0) = 0\)。✔

常见错误

  • ❌ 符号交替 \((-1)^i\) 搞反。
    ✅ 删第 0 个顶点符号 \(+\)，删第 1 个符号 \(-\)，删第 2 个符号 \(+\)。建议画图辅助。
  • ❌ 忘记有向性——将 \([v_1, v_0]\) 误当作 \([v_0, v_1]\)。
    ✅ \([v_1, v_0] = -[v_0, v_1]\)，交换两个顶点改变符号。
  • ❌ 认为 \(\partial^2=0\) 是因为"边界没有边界"的直觉就够了。
    ✅ 严格证明需要追踪 \((-1)^i(-1)^j\) 的双重删除抵消——任何非交替的符号约定都不满足 \(\partial^2=0\)。

03单纯同调群的计算

🎯 为什么研究这个问题

链复形中的闭链（无边界的链）代表潜在的"洞"，但有些闭链本身就是边界——真正的"洞"是闭链中减去边界后剩余的部分。同调群 \(H_n = Z_n/B_n\) 正是对这一想法的精确表述。

  • 精确化"洞"的概念：\(H_n(K) = Z_n/B_n\) 将"闭链模边界"精确化——\(H_0\) 计数连通分量，\(H_1\) 检测环，\(H_2\) 检测空腔。
  • 可计算性：计算同调等价于整数矩阵的 Smith 标准形——完全可算法化，结果是有限生成交换群 \(\mathbb{Z}^r \oplus T\)。
  • Euler 特征：\(\chi = \sum (-1)^n \beta_n\) 统一了顶点-边-面计数和同调——古典公式的深刻推广。

同调群不仅给出了"洞"的精确代数描述，还通过 Smith 标准形将计算完全算法化——从 Euler 特征到 Betti 数，经典的拓扑量都在同调的框架中自然统一。

🔍 抽象化方法
  • Betti 数：\(\beta_n = \mathrm{rank}\,H_n\) 计数 \(n\) 维独立洞数；挠部分编码更精细的信息（如 \(\mathbb{R}P^2\) 的 \(H_1 = \mathbb{Z}/2\)）。
  • 整数线性代数：同调计算 = 边缘矩阵的 Smith 标准形 → 对角元素给出挠，零行/列给出自由部分。
📐 公式推导 / 模型建立

1. 同调群定义

\(H_n(K; \mathbb{Z}) = \ker \partial_n / \mathrm{im}\,\partial_{n+1}\)。

2. 计算步骤（Smith 标准形）

(1) 写出边缘矩阵 \([\partial_n]\)。(2) Smith 标准形化简：\([\partial_n] = UDV\)（\(D\) 对角）。(3) \(\mathrm{rank}(\partial_n) = r_n\)。(4) \(H_n \cong \mathbb{Z}^{z_n - r_{n+1}} \oplus \text{挠}\)，其中 \(z_n = \dim C_n - r_n\)。

3. 经典结果

\(H_n(S^k) = \begin{cases}\mathbb{Z} & n=0,k \\ 0 & \text{otherwise}\end{cases}\)。\(H_0(X) \cong \mathbb{Z}^c\)（\(c\) = 连通分量数）。\(H_*(T^2) = (\mathbb{Z}, \mathbb{Z}^2, \mathbb{Z})\)。

4. Euler 特征

\[\chi(K) = \sum_{n} (-1)^n |K_n| = \sum_{n} (-1)^n \beta_n\]
🌐 实际应用

持续同调与 TDA：追踪 Betti 数随 filtration 参数变化，生成 persistence diagram——每个点 \((b,d)\) 代表一个拓扑特征的"出生"和"死亡"时刻。

传感器网络：用 Rips 复形覆盖检测传感器网络中的覆盖空洞——\(H_1 \neq 0\) 意味着存在未覆盖区域。

图像分析：数字图像的 cubical complex 同调可检测物体内部空洞和连通性——用于医学影像中的结构分析。

✏️ 例题分析与易错点

例题：\(S^1\) 的同调（三角剖分）

题目：用 3 顶点 3 边的三角形边界 \(\Delta^2\) 的边界计算 \(H_*(S^1)\)。

第一步 — 链群：\(C_0 = \mathbb{Z}^3\)（顶点 \(v_0, v_1, v_2\)），\(C_1 = \mathbb{Z}^3\)（边 \(e_{01}, e_{12}, e_{02}\)），\(C_2 = 0\)。

第二步 — 边缘矩阵：\(\partial_1 = \begin{pmatrix}-1&0&-1\\1&-1&0\\0&1&1\end{pmatrix}\)，rank \(= 2\)。

第三步 — 同调：\(H_0 = \mathbb{Z}^{3}/\mathrm{im}\,\partial_1 = \mathbb{Z}^{3-2} = \mathbb{Z}\)。\(H_1 = \ker \partial_1 = \mathbb{Z}^{3-2} = \mathbb{Z}\)。✔

常见错误

  • ❌ 混淆同调的系数——\(H_n(X; \mathbb{Z})\) 和 \(H_n(X; \mathbb{Z}/2)\) 可以不同。
    ✅ \(H_1(\mathbb{R}P^2; \mathbb{Z}) = \mathbb{Z}/2\)，但 \(H_1(\mathbb{R}P^2; \mathbb{Z}/2) = \mathbb{Z}/2\)（结果巧合相同，但原因不同）。
  • ❌ 认为 \(H_1\) 总是自由的。
    ✅ \(H_1(\mathbb{R}P^2) = \mathbb{Z}/2\) 有挠——非可定向曲面产生挠。
  • ❌ 忽略 \(H_0\) 始终非零。
    ✅ 非空连通空间 \(H_0 = \mathbb{Z}\)，\(c\) 个连通分量则 \(H_0 = \mathbb{Z}^c\)。

04同调群的拓扑不变性

🎯 为什么研究这个问题

同调群依赖三角剖分的选择而定义，但拓扑不变量不应依赖于具体的剖分方式。同调群确实独立于三角剖分——更强地，它是同伦不变量：只要两个空间同伦等价，它们的同调群就相同。

  • 内在性：同调群不依赖于三角剖分的选择——不同剖分给出同构的同调群。同调是空间的内在性质，不是人工产物。
  • 同伦不变性：更强地，同伦等价的空间有相同的同调——\(f \simeq g \Rightarrow f_* = g_*\)。这使同调成为区分空间的强有力工具。
  • 链同伦机制：链映射 \(f_\#\) 和链同伦 \(D\) 提供了将连续信息翻译为代数信息的精确机制。

同伦不变性确立了同调群作为拓扑不变量的合法地位——链映射与链同伦的代数机制精确地保证了"连续变形不改变同调"，使同调成为区分空间的可靠工具。

🔍 抽象化方法
  • 函子性：链映射 \(f_\#\)（\(\partial \circ f_\# = f_\# \circ \partial\)）诱导 \(f_*: H_*(K) \to H_*(L)\)——"拓扑→代数"的翻译保持结构。
  • 链同伦 ⇒ 同调等价：\(f_\# - g_\# = \partial D + D\partial\) 蕴含 \(f_* = g_*\)——代数层面的"形变不改变结果"。
📐 公式推导 / 模型建立

1. 链映射

\(f_\#: C_n(K) \to C_n(L)\)，\(\partial^L \circ f_\# = f_\# \circ \partial^K\)。推论：\(f_\#(Z_n) \subseteq Z_n\)，\(f_\#(B_n) \subseteq B_n\)，诱导 \(f_*: H_n(K) \to H_n(L)\)。

2. 链同伦

\(D: C_n \to C_{n+1}\) 使得 \(f_\# - g_\# = \partial D + D\partial\)。对闭链 \(z \in Z_n\)：\(f_\#(z) - g_\#(z) = \partial D(z) \in B_n\)，故 \(f_* = g_*\)。

3. 同伦不变性定理

连续映射 \(f: |K| \to |L|\) 诱导链映射（通过单纯逼近定理），同伦映射诱导链同伦的链映射：

\[f \simeq g \;\Rightarrow\; f_* = g_*: H_*(K) \to H_*(L)\]

4. 推论

\(X \simeq Y \Rightarrow H_n(X) \cong H_n(Y)\)。不同三角剖分给出同构的同调群。可缩空间 \(H_n = 0\)（\(n \geq 1\)）。

🌐 实际应用

形状识别：计算机视觉中用同调群区分形状类——杯子和甜甜圈同伦等价（\(H_1 = \mathbb{Z}\)），但与球不同（球 \(H_1 = 0\)）。

网格简化：3D 模型简化时保持同调不变——确保简化后的网格拓扑正确（不产生新的洞或消除已有的洞）。

✏️ 例题分析与易错点

例题：证明 \(S^2 \not\simeq \mathbb{R}^3\)

题目：用同调证明 \(S^2\) 与 \(\mathbb{R}^3\) 不同伦等价。

第一步 — 计算：\(H_2(S^2) = \mathbb{Z}\)。\(H_2(\mathbb{R}^3) = 0\)（\(\mathbb{R}^3\) 可缩）。

第二步 — 结论：同伦等价保持同调群同构，\(\mathbb{Z} \not\cong 0\)，故 \(S^2 \not\simeq \mathbb{R}^3\)。✔

常见错误

  • ❌ 混淆"同伦不变"和"同胚不变"。
    ✅ 同伦不变更强——同胚蕴含同伦等价，但反之不然（\(\mathbb{R}^n \simeq \{*\}\) 但 \(\mathbb{R}^n \not\cong \{*\}\)）。
  • ❌ 认为同调能区分所有空间。
    ✅ 同调相同的空间可以不同伦——如 \(\mathbb{R}P^3\) 和 \(S^3 \times \mathbb{R}P^\infty\) 可有相同同调但不同伦。
  • ❌ 忘记单纯逼近定理的前提——映射需要连续。
    ✅ 不连续映射不诱导良好的链映射，同调的函子性依赖连续性。
