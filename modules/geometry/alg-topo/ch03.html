01单纯复形与 \(\Delta\)-复形

🎯 为什么研究这个问题

为了计算空间的"洞"——不仅仅是一维的（基本群），还有二维、三维 及更高维的——需要一种系统方法。单纯复形将空间"三角化"为顶点、 边、三角形、四面体等简单积木（单纯形）的组合。\(\Delta\)-复形是 更灵活的简化版本，允许面的粘合更自由。

🔍 抽象化方法

单纯复形 = 组合化的拓扑空间。将连续问题（拓扑）转化为有限 组合问题（整数矩阵的线性代数）。

📐 公式推导 / 模型建立

标准 \(n\)-单纯形： \(\Delta^n = \{(t_0, \ldots, t_n) \in \mathbb{R}^{n+1} : \sum t_i = 1, t_i \geq 0\}\)。 顶点 \(e_i\)，边 \([e_i, e_j]\)，面 \([e_{i_0}, \ldots, e_{i_k}]\)。 单纯复形：\(K\) 是集合 \(V\)（顶点集）的有限子集的集合， 满足遗传性：\(\sigma \in K, \tau \subseteq \sigma \Rightarrow \tau \in K\)。 几何实现 \(|K|\)：将每个 \(\sigma \in K\) 实现为 \(\Delta^{|\sigma|-1}\)，按面关系粘合。 \(\Delta\)-复形：更宽泛——允许面的恒同粘合（如三角形两条边 黏合为一条）。有向单纯形，每个 \(n\)-单纯形有 \(n+1\) 个面映射 \(d_i: \Delta^{n-1} \to \Delta^n\)（删去第 \(i\) 个顶点）。 \[d_i[v_0, \ldots, v_n] = [v_0, \ldots, \hat{v}_i, \ldots, v_n]\]

🌐 实际应用

- 拓扑数据分析 (TDA) 中的 Rips 复形和 Čech 复形用单纯复形 表示数据点云的形状

✏️ 例题分析与易错点

例题：将 \(T^2\)（环面）三角剖分为 \(\Delta\)-复形。 解答：用标准方法：取正方形 \([0,1]^2\)，对边识别。 最少需要 \(1\) 个顶点（所有角等价）、\(3\) 条边（\(a, b, c\)， 水平、垂直、对角）、\(2\) 个三角形。 面映射：\(\partial[012] = [12] - [02] + [01]\)（标准定向）。 易错点：\(\Delta\)-复形中允许一个单纯形多个面相同（如 \(S^1\) 可以 用 1 个顶点 + 1 条边构成），而传统单纯复形不允许。

⚡ 快速检测
标准 \(2\)-单纯形 \(\Delta^2\) 有多少个面（包括所有维数）？

02链复形与边缘算子

🎯 为什么研究这个问题

为了将"洞"代数化，定义链群 \(C_n\)（\(n\)-链 = \(n\)-单纯形的形式 整数线性组合）和边缘算子 \(\partial_n: C_n \to C_{n-1}\)（将每个单纯形 映为其边界之和）。关键性质 \(\partial^2 = 0\) 使得 \(\ker \partial\)（闭链） 包含 \(\mathrm{im}\,\partial\)（边界链），商群就是同调群。

🔍 抽象化方法

链复形 \(\cdots \xrightarrow{\partial_{n+1}} C_n \xrightarrow{\partial_n} C_{n-1} \xrightarrow{\partial_{n-1}} \cdots\) 是代数拓扑的核心代数结构。\(\partial^2 = 0\) 产生"cycles mod boundaries"。

📐 公式推导 / 模型建立

链群：\(C_n(K) = \bigoplus_{\sigma \in K_n} \mathbb{Z} \cdot \sigma\)（有向 \(n\)-单纯形自由生成的交换群）。 边缘算子： \[\partial_n[v_0, \ldots, v_n] = \sum_{i=0}^n (-1)^i [v_0, \ldots, \hat{v}_i, \ldots, v_n]\] \(\partial^2 = 0\)（关键引理）： \(\partial_{n-1} \circ \partial_n = 0\)。 证明：双重删除 \(\hat{v}_i\) 再 \(\hat{v}_j\) 出现两次（\(i < j\) 时 符号为 \((-1)^{i+j}\) 和 \((-1)^{i+j-1}\)），互相抵消。 链复形：\(\cdots \to C_2 \xrightarrow{\partial_2} C_1 \xrightarrow{\partial_1} C_0 \to 0\)。 \(Z_n = \ker \partial_n\)（\(n\)-闭链 / cycles）， \(B_n = \mathrm{im}\,\partial_{n+1}\)（\(n\)-边缘 / boundaries）。 \(B_n \subseteq Z_n\)（因为 \(\partial^2 = 0\)）。

✏️ 例题分析与易错点

例题：三角形 \([v_0, v_1, v_2]\)，计算 \(\partial_2[v_0, v_1, v_2]\)。 解答：\(\partial_2[v_0, v_1, v_2] = [v_1, v_2] - [v_0, v_2] + [v_0, v_1]\)。 直觉：三角形的边界 = 三条边（按定向）。 验证 \(\partial_1 \circ \partial_2\)： \(\partial_1([v_1,v_2] - [v_0,v_2] + [v_0,v_1])\) \(= (v_2-v_1) - (v_2-v_0) + (v_1-v_0) = 0\)。✓ 易错点：符号交替 \((-1)^i\) 容易搞错。建议画图辅助理解。

03单纯同调群的计算

🎯 为什么研究这个问题

同调群 \(H_n(K) = Z_n / B_n\) 将"闭链模边界"的概念精确化。 \(H_0\) 计数连通分量，\(H_1\) 检测一维洞（如圈），\(H_2\) 检测二维洞 （如空腔）。计算同调群就是做整数矩阵的行列变换（Smith 标准形）， 结果是有限生成交换群 \(\cong \mathbb{Z}^r \oplus T\)（自由部分+挠部分）。

🔍 抽象化方法

同调群 = 链复形的上同调。Betti 数 \(\beta_n = \mathrm{rank}\,H_n\) 计数 \(n\) 维独立的"洞"。

📐 公式推导 / 模型建立

同调群：\(H_n(K; \mathbb{Z}) = \ker \partial_n / \mathrm{im}\,\partial_{n+1}\)。 计算方法： (1) 写出边缘矩阵 \([\partial_n]\)（整数矩阵） (2) 用 Smith 标准形化简：\([\partial_n] = U D_n V\)，\(D_n\) 对角 (3) \(\mathrm{rank}(\partial_n) = r_n\) (4) \(H_n \cong \mathbb{Z}^{z_n - r_{n+1}} \oplus \text{挠部分}\) 其中 \(z_n = \dim C_n - r_n\) 经典结果： - \(H_n(S^k) = \begin{cases}\mathbb{Z} & n = 0, k \\ 0 & \text{otherwise}\end{cases}\) - \(H_0(X) \cong \mathbb{Z}^c\)（\(c\) = 连通分量数） - \(H_1(T^2) \cong \mathbb{Z}^2\), \(H_2(T^2) \cong \mathbb{Z}\) Euler 特征：\(\chi(K) = \sum (-1)^n |K_n| = \sum (-1)^n \beta_n\)。

🌐 实际应用

- 拓扑数据分析中持续同调 (persistent homology) 追踪 Betti 数 随尺度变化——检测数据中的拓扑特征

✏️ 例题分析与易错点

例题：计算 \(S^1\)（三角剖分为 3 顶点 3 边的三角形边界）的同调群。 解答：\(C_0 = \mathbb{Z}^3\)（\(v_0, v_1, v_2\)），\(C_1 = \mathbb{Z}^3\)（\(e_{01}, e_{12}, e_{02}\)），\(C_2 = 0\)。 \(\partial_1: e_{01} \mapsto v_1 - v_0\), \(e_{12} \mapsto v_2 - v_1\), \(e_{02} \mapsto v_2 - v_0\)。 矩阵 \(\partial_1 = \begin{pmatrix}-1&0&-1\\1&-1&0\\0&1&1\end{pmatrix}\)。\(\mathrm{rank} = 2\)。 \(H_0 = \mathbb{Z}^3/\mathrm{im}\,\partial_1 = \mathbb{Z}^{3-2} = \mathbb{Z}\)（1 个连通分量）。 \(H_1 = \ker \partial_1 / 0 = \mathbb{Z}^{3-2} = \mathbb{Z}\)（1 个一维洞）。 易错点：\(e_{01} + e_{12} - e_{02}\) 是 1-cycle（边界为零）且非边界 （\(C_2 = 0\)），它生成 \(H_1 = \mathbb{Z}\)。

04同调群的拓扑不变性

🎯 为什么研究这个问题

同调群的威力在于它是拓扑不变量——不依赖于具体的三角剖分。 不同的三角剖分给出同构的同调群！这意味着同调是空间的内在性质， 不是剖分方式的人工产物。证明这个定理需要"单纯逼近"和 链映射诱导的同调映射的理论。

🔍 抽象化方法

链映射 \(f_\#: C_*(K) \to C_*(L)\)（\(\partial \circ f_\# = f_\# \circ \partial\)） 诱导同调映射 \(f_*: H_*(K) \to H_*(L)\)。链同伦的映射诱导 相同的同调映射——这是同调的同伦不变性的代数机制。

📐 公式推导 / 模型建立

链映射：\(f_\#: C_n(K) \to C_n(L)\)（每个 \(n\)）， \(\partial^L \circ f_\# = f_\# \circ \partial^K\)。 则 \(f_\#(Z_n) \subseteq Z_n\)，\(f_\#(B_n) \subseteq B_n\)， 诱导 \(f_*: H_n(K) \to H_n(L)\)。 链同伦：\(f_\#, g_\#: C_*(K) \to C_*(L)\)， \(D: C_n \to C_{n+1}\) 使 \(f_\# - g_\# = \partial D + D\partial\)。 则 \(f_* = g_*: H_* \to H_*\)。 同伦不变性：连续映射 \(f: |K| \to |L|\) 诱导链映射 \(f_\#\) （通过单纯逼近定理）。同伦的映射诱导链同伦的链映射。 \[f \simeq g \Rightarrow f_* = g_*: H_*(K) \to H_*(L)\] 推论：\(X \simeq Y\) \(\Rightarrow\) \(H_n(X) \cong H_n(Y)\)。 特别地，不同三角剖分给出同构的同调群。

✏️ 例题分析与易错点

例题：证明 \(S^2\) 与 \(\mathbb{R}^3\) 不同伦等价。 解答：\(H_2(S^2) = \mathbb{Z} \neq 0 = H_2(\mathbb{R}^3)\) （\(\mathbb{R}^3\) 可缩，\(H_n = 0\) 对 \(n \geq 1\)）。 同伦等价保持同调，所以 \(S^2 \not\simeq \mathbb{R}^3\)。 易错点：同伦不变性比同胚不变性弱——它不能区分同伦等价 但不同胚的空间（如 \(\mathbb{R}^n\) 和 \(\{*\}\)）。
