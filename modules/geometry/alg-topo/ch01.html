01同伦与道路同伦

🎯 为什么研究这个问题

代数拓扑的核心思想是将拓扑空间中的"形变"关系代数化。两条道路 "同伦"意味着一条可以连续形变为另一条——这比要求道路完全相同 弱得多，但足以区分空间的"洞"。例如在平面去掉原点后，绕原点 的圈不能连续缩成一点——这就是非平凡同伦类的直觉。

🔍 抽象化方法

同伦是拓扑学中"等价"的正确概念。同伦等价将拓扑空间分类到 比同胚更粗的等价类中（只关注"形状"、不关注精确尺寸）。

📐 公式推导 / 模型建立

同伦：连续映射 \(f, g: X \to Y\) 同伦（\(f \simeq g\)）\(\iff\) 存在连续映射 \(H: X \times [0,1] \to Y\) 使 \(H(x,0) = f(x)\)，\(H(x,1) = g(x)\)。 道路同伦：道路 \(\gamma_0, \gamma_1: [0,1] \to X\) （\(\gamma_0(0) = \gamma_1(0) = a\)，\(\gamma_0(1) = \gamma_1(1) = b\)） 相对端点同伦 \(\iff\) 存在 \(H: [0,1]^2 \to X\)， \(H(s,0) = \gamma_0(s)\)，\(H(s,1) = \gamma_1(s)\)， \(H(0,t) = a\)，\(H(1,t) = b\)。 同伦类 \([\gamma]\) 在道路的连接运算下构成良定义的运算： \[[\gamma] \cdot [\delta] = [\gamma \cdot \delta]\] 其中 \((\gamma \cdot \delta)(s) = \begin{cases}\gamma(2s) & s \in [0,1/2]\\\delta(2s-1) & s \in [1/2,1]\end{cases}\)

✏️ 例题分析与易错点

例题：在 \(\mathbb{R}^2\) 中，两条从 \((0,0)\) 到 \((1,0)\) 的道路 \(\gamma_0(s) = (s, 0)\) 和 \(\gamma_1(s) = (s, s(1-s))\) 是否同伦？ 解答：是。\(H(s,t) = (s, ts(1-s))\) 是同伦： \(H(s,0) = (s,0) = \gamma_0(s)\)，\(H(s,1) = (s, s(1-s)) = \gamma_1(s)\)， \(H(0,t) = (0,0)\)，\(H(1,t) = (1,0)\)。 \(\mathbb{R}^2\) 是凸的，任何两条同端点道路都同伦。 易错点：\(\mathbb{R}^2 \setminus \{0\}\) 中不是所有同端点道路都同伦！ 绕原点和不绕原点的道路属不同同伦类。

02基本群的定义与性质

🎯 为什么研究这个问题

将基点处所有回路（闭道路）的同伦类收集起来，配上道路连接运算， 得到一个群——基本群 \(\pi_1(X, x_0)\)。这是空间"一维洞"的代数 编码：\(\pi_1(S^1) = \mathbb{Z}\)（绕不同圈数），\(\pi_1(\mathbb{R}^2) = 0\) （无洞可缩），\(\pi_1(T^2) = \mathbb{Z}^2\)（两个独立方向的圈）。

🔍 抽象化方法

基本群是从拓扑空间范畴（连续映射）到群范畴（同态）的函子。 连续映射 \(f: X \to Y\) 诱导群同态 \(f_*: \pi_1(X) \to \pi_1(Y)\)。

📐 公式推导 / 模型建立

定义：\(\pi_1(X, x_0) = \{[\gamma] : \gamma \text{ 是基于 } x_0 \text{ 的回路}\}\)。 群结构： - 乘法：\([\gamma][\delta] = [\gamma \cdot \delta]\) - 单位元：\([c_{x_0}]\)（常值回路） - 逆元：\([\gamma]^{-1} = [\bar{\gamma}]\)，\(\bar{\gamma}(s) = \gamma(1-s)\) 基点变更：若 \(X\) 道路连通，\(\pi_1(X, x_0) \cong \pi_1(X, x_1)\) （通过连接道路的共轭给出同构）。 函子性：\(f: (X, x_0) \to (Y, y_0)\) 连续， \(f_*: \pi_1(X, x_0) \to \pi_1(Y, y_0)\)，\(f_*([\gamma]) = [f \circ \gamma]\)。 \[f_*([\gamma][\delta]) = f_*([\gamma]) \cdot f_*([\delta])\] \((g \circ f)_* = g_* \circ f_*\)；\(\mathrm{id}_* = \mathrm{id}\)。

🌐 实际应用

- Brouwer 不动点定理可用 \(\pi_1\) 证明（\(n = 2\) 情况）

✏️ 例题分析与易错点

例题：计算 \(\pi_1(S^1, 1)\)。 解答（思路）：回路 \(\gamma: [0,1] \to S^1\) 的提升到 \(\mathbb{R}\)： \(\tilde{\gamma}(0) = 0\)，\(\tilde{\gamma}(1) = n \in \mathbb{Z}\)（绕圈数）。 \(n\) 不变于同伦，给出同构 \(\pi_1(S^1) \cong \mathbb{Z}\)。 生成元是"逆时针绕一圈"\([\omega_1]\)，\(\omega_1(s) = e^{2\pi is}\)。 易错点：\(\pi_1\) 中的"乘法"是同伦类的乘法，不是 \(S^1\) 中的点乘法。 \([\omega_n] = [\omega_1]^n\) 代表绕 \(n\) 圈。

03van Kampen 定理

🎯 为什么研究这个问题

如何从"局部"的基本群拼出"全局"的基本群？van Kampen 定理回答 了这个问题：如果空间 \(X = U_1 \cup U_2\)，\(U_1 \cap U_2\) 道路连通， 那么 \(\pi_1(X)\) 是 \(\pi_1(U_1)\) 和 \(\pi_1(U_2)\) 在 \(\pi_1(U_1 \cap U_2)\) 上 的自由积商——即"融合积 (amalgamated free product)"。

🔍 抽象化方法

van Kampen 定理是 \(\pi_1\) 函子的一个"余极限性质"——它将 基本群的计算转化为群论中的自由积与商运算。

📐 公式推导 / 模型建立

van Kampen 定理：\(X = U_1 \cup U_2\)，\(U_1, U_2, U_1 \cap U_2\) 开且道路连通，\(x_0 \in U_1 \cap U_2\)。含入 \(i_k: U_1 \cap U_2 \hookrightarrow U_k\) 诱导 \((i_k)_*: \pi_1(U_1 \cap U_2) \to \pi_1(U_k)\)。 \[\pi_1(X) \cong \pi_1(U_1) *_{\pi_1(U_1 \cap U_2)} \pi_1(U_2)\] 即 \(\pi_1(X) = (\pi_1(U_1) * \pi_1(U_2)) / N\)， \(N = \langle (i_1)_*(g) \cdot (i_2)_*(g)^{-1} : g \in \pi_1(U_1 \cap U_2) \rangle\)。 特例：\(U_1 \cap U_2\) 单连通（\(\pi_1 = 0\)）时 \(\pi_1(X) = \pi_1(U_1) * \pi_1(U_2)\)（自由积）。

🌐 实际应用

- 计算 knot group（结的余空间的基本群）——拓扑结理论的核心工具

✏️ 例题分析与易错点

例题：用 van Kampen 定理计算 \(\pi_1(S^1 \vee S^1)\)。 解答：\(S^1 \vee S^1\) = 两个圆在一点黏合（"花环"或"8"字形）。 取 \(U_1\) = 第一个圈加一小段第二个圈，\(U_2\) = 第二个圈加一小段第一个圈。 \(U_1 \simeq S^1\)，\(U_2 \simeq S^1\)，\(U_1 \cap U_2 \simeq \{*\}\)（可缩）。 \(\pi_1(U_1 \cap U_2) = 0\)。 \(\pi_1(S^1 \vee S^1) = \mathbb{Z} * \mathbb{Z}\)（自由群 \(F_2\)）。 易错点：\(\mathbb{Z} * \mathbb{Z} \neq \mathbb{Z} \times \mathbb{Z}\)！ 自由积中 \(ab \neq ba\)，是非交换群。\(\pi_1(T^2) = \mathbb{Z}^2\)， \(\pi_1(S^1 \vee S^1) = F_2\)。

⚡ 快速检测
\(\pi_1(S^2) = ?\)

04基本群与应用

🎯 为什么研究这个问题

基本群不仅是抽象的代数不变量——它有深刻的几何和拓扑应用。 代数基本定理可以用 \(\pi_1(S^1) = \mathbb{Z}\) 证明，Brouwer 不动点 定理也可以用基本群方法处理（2 维情况），Borsuk-Ulam 定理说明 \(S^2\) 到 \(\mathbb{R}^2\) 的映射必有对径点同值。

🔍 抽象化方法

基本群作为拓扑不变量，可以用来证明"不可能性"： 若 \(\pi_1(X) \not\cong \pi_1(Y)\) 则 \(X \not\simeq Y\)。

📐 公式推导 / 模型建立

Brouwer 不动点定理（\(n=2\)）：每个连续 \(f: D^2 \to D^2\) 有不动点。 证明：反证法。若 \(f(x) \neq x\)，定义收缩 \(r: D^2 \to S^1\)（从 \(f(x)\) 向 \(x\) 方向的射线与 \(S^1\) 的交点）。 \(r|_{S^1} = \mathrm{id}_{S^1}\)，即 \(r \circ i = \mathrm{id}\) （\(i: S^1 \hookrightarrow D^2\)）。 \(r_* \circ i_*: \pi_1(S^1) \to \pi_1(D^2) \to \pi_1(S^1) = \mathrm{id}: \mathbb{Z} \to \mathbb{Z}\)。 但 \(\pi_1(D^2) = 0\)，所以 \(r_* \circ i_*\) 必须通过 \(0\)， 不可能是 \(\mathbb{Z}\) 上的恒等映射。矛盾！ 应用：若 \(X \simeq Y\) 则 \(\pi_1(X) \cong \pi_1(Y)\)。 \(\pi_1(\mathbb{R}P^2) = \mathbb{Z}/2\mathbb{Z}\)， \(\pi_1(K) = \langle a, b | abab^{-1} \rangle\)（Klein 瓶）。

🌐 实际应用

- Brouwer 不动点定理在博弈论（Nash 均衡存在性）中是核心工具

✏️ 例题分析与易错点

例题：证明 \(\mathbb{R}^2\) 和 \(\mathbb{R}^3\) 不同胚。 解答：反证：若 \(f: \mathbb{R}^2 \to \mathbb{R}^3\) 是同胚。 去掉一点：\(f\) 限制为 \(\mathbb{R}^2 \setminus \{p\} \cong \mathbb{R}^3 \setminus \{f(p)\}\)。 \(\pi_1(\mathbb{R}^2 \setminus \{p\}) = \mathbb{Z}\)（\(\simeq S^1\)）， \(\pi_1(\mathbb{R}^3 \setminus \{q\}) = 0\)（\(\simeq S^2\)，单连通）。 \(\mathbb{Z} \neq 0\)，矛盾。 易错点：直接比较 \(\pi_1(\mathbb{R}^2) = \pi_1(\mathbb{R}^3) = 0\) 没有用（都是可缩空间）。必须去掉一点来制造区别。
