上同调与 Poincaré 对偶

01上同调群的定义与环结构

🎯 为什么研究这个问题

同调将空间分解为"洞"的信息，但上同调更进一步——它自带乘法 结构（杯积）！同调群只是交换群的序列，而上同调群在杯积下构成 一个分次交换环。这个额外的代数结构能区分同调群相同但上同调环 不同的空间（经典例子：\(\mathbb{C}P^2\) 和 \(S^2 \vee S^4\)）。

🔍 抽象化方法

上同调 = 链复形的对偶（\(\mathrm{Hom}\) 函子）。杯积来自链的 对角逼近（Alexander-Whitney 映射），给上同调以环结构。

📐 公式推导 / 模型建立

余链群：\(C^n(X; R) = \mathrm{Hom}(C_n(X), R)\)（\(R\) 为系数环）。 余边缘：\(\delta^n: C^n \to C^{n+1}\)，\((\delta \varphi)(\sigma) = \varphi(\partial \sigma)\)。 \(\delta^2 = 0\)（因为 \(\partial^2 = 0\)）。 上同调群：\(H^n(X; R) = \ker \delta^n / \mathrm{im}\,\delta^{n-1}\)。 杯积：\(\smile: H^p(X) \times H^q(X) \to H^{p+q}(X)\)。 定义在余链层面：\((\varphi \smile \psi)(\sigma) = \varphi(\sigma|_{[v_0,\ldots,v_p]}) \cdot \psi(\sigma|_{[v_p,\ldots,v_{p+q}]})\)。 \[\alpha \smile \beta = (-1)^{pq} \beta \smile \alpha\] （分次交换）。 上同调环：\(H^*(X; R) = \bigoplus_n H^n(X; R)\) 配合 \(\smile\) 构成分次交换环。

🌐 实际应用

- 杯积区分 \(\mathbb{C}P^2\) 和 \(S^2 \vee S^4\)（同调同构但杯积不同）

✏️ 例题分析与易错点

例题：\(H^*(\mathbb{C}P^2; \mathbb{Z})\) 的环结构。 解答：\(H^0 = \mathbb{Z}\), \(H^2 = \mathbb{Z}\), \(H^4 = \mathbb{Z}\)。 设 \(\alpha\) 生成 \(H^2\)。\(\alpha \smile \alpha\) 生成 \(H^4\)： \(H^*(\mathbb{C}P^2) \cong \mathbb{Z}[\alpha] / (\alpha^3)\)，\(\deg \alpha = 2\)。 比较 \(S^2 \vee S^4\)：\(H^2 = \mathbb{Z}\langle a \rangle\)，\(H^4 = \mathbb{Z}\langle b \rangle\)， \(a \smile a = 0 \neq b\)。 杯积区分了两个空间。 易错点：\(H_n\) 相同不等于空间同伦等价——上同调环是更精细的不变量。

02万有系数定理

🎯 为什么研究这个问题

用不同系数（\(\mathbb{Z}, \mathbb{Z}/2, \mathbb{Q}, \mathbb{R}\)）计算上同调 时结果不同——万有系数定理精确地描述了它们之间的关系。实际计算 中，\(\mathbb{Z}/2\) 系数消除了方向问题，\(\mathbb{Q}\) 系数消除了挠， \(\mathbb{R}\) 系数连接到 de Rham 上同调。

🔍 抽象化方法

万有系数定理说 \(H^n(X; G) \cong \mathrm{Hom}(H_n, G) \oplus \mathrm{Ext}^1(H_{n-1}, G)\) ——上同调由同调和 Ext 函子决定（自然但非典范分裂）。

📐 公式推导 / 模型建立

万有系数定理（上同调版）： \[0 \to \mathrm{Ext}^1_\mathbb{Z}(H_{n-1}(X), G) \to H^n(X; G) \to \mathrm{Hom}(H_n(X), G) \to 0\] 此序列分裂（但非自然）。 关键情况： - \(G = \mathbb{Q}\) 或 \(\mathbb{R}\)：\(\mathrm{Ext}\) 项消失（可除群）， \(H^n(X; \mathbb{R}) \cong \mathrm{Hom}(H_n(X), \mathbb{R})\)。 - \(G = \mathbb{Z}\)：\(\mathrm{Ext}^1(\mathbb{Z}/m, \mathbb{Z}) = \mathbb{Z}/m\)。 \(\mathrm{Ext}\) 项"提升"了挠信息。 万有系数定理（同调版）： \[0 \to H_n(X;\mathbb{Z}) \otimes G \to H_n(X; G) \to \mathrm{Tor}_1(H_{n-1}(X), G) \to 0\]

🌐 实际应用

- de Rham 定理：\(H^n_{dR}(M) \cong H^n(M; \mathbb{R})\)，将分析与拓扑连接

✏️ 例题分析与易错点

例题：\(H_1(\mathbb{R}P^2) = \mathbb{Z}/2\)，\(H_0 = \mathbb{Z}\)。 求 \(H^2(\mathbb{R}P^2; \mathbb{Z})\)。 解答：万有系数定理：\(H^2 \cong \mathrm{Hom}(H_2, \mathbb{Z}) \oplus \mathrm{Ext}^1(H_1, \mathbb{Z})\)。 \(H_2(\mathbb{R}P^2) = 0\)，\(\mathrm{Hom}(0, \mathbb{Z}) = 0\)。 \(\mathrm{Ext}^1(\mathbb{Z}/2, \mathbb{Z}) = \mathbb{Z}/2\)。 \(H^2(\mathbb{R}P^2; \mathbb{Z}) = \mathbb{Z}/2\)。 易错点：\(H^n \neq H_n\) 一般不成立！上同调可能有挠而同调没有（反之亦然）， 通过 \(\mathrm{Ext}\) 项联系。

若 \(H_n(X)\) 无挠（自由交换群），则 \(H^n(X;\mathbb{Z}) \cong ?\)

03Poincaré 对偶

🎯 为什么研究这个问题

闭定向 \(n\) 维流形有一个惊人的对称性：\(k\) 维的上同调与 \((n-k)\) 维 的同调同构！这就是 Poincaré 对偶——几何上对应于"\(k\) 维子流形 与 \((n-k)\) 维子流形横截相交"。对偶让高维上同调的计算可以 转化为低维同调的计算（或反之）。

🔍 抽象化方法

Poincaré 对偶 = 与基本类的杯积给出同构。 \([M] \in H_n(M)\) 是"整个流形"的同调类。

📐 公式推导 / 模型建立

Poincaré 对偶定理：\(M\) 是闭定向 \(n\) 维流形，\(R\) 为系数。 \[D: H^k(M; R) \xrightarrow{\cong} H_{n-k}(M; R)\] \(D(\alpha) = \alpha \frown [M]\)（帽积 with 基本类）。 等价表述：杯积配对 \[H^k(M; R) \otimes H^{n-k}(M; R) \xrightarrow{\smile} H^n(M; R) \cong R\] 是非退化的（当 \(R\) 是域时）。 推论： - Betti 数对称：\(\beta_k = \beta_{n-k}\) - \(\chi(M) = 0\) 若 \(n\) 为奇数 - 中间维上同调 \(H^{n/2}\) 上有非退化配对（交叉形式）

🌐 实际应用

- 4 维流形分类中，\(H^2\) 上的交叉形式是核心不变量（Donaldson 和 Freedman 的工作）

✏️ 例题分析与易错点

例题：\(M = S^2\)（\(n = 2\)）。验证 Poincaré 对偶。 解答：\(H^0 = \mathbb{Z}\)，\(H^1 = 0\)，\(H^2 = \mathbb{Z}\)。 \(H_0 = \mathbb{Z}\)，\(H_1 = 0\)，\(H_2 = \mathbb{Z}\)。 对偶：\(H^0 \cong H_2\)（\(\mathbb{Z} \cong \mathbb{Z}\)）， \(H^1 \cong H_1\)（\(0 \cong 0\)），\(H^2 \cong H_0\)（\(\mathbb{Z} \cong \mathbb{Z}\)）。✓ 易错点：Poincaré 对偶要求闭（无边界）+ 定向。莫比乌斯带（有边界且不可定向） 和 \(\mathbb{R}P^2\)（不可定向）不满足\(\mathbb{Z}\) 系数的 Poincaré 对偶。

04Künneth 公式与交叉积

🎯 为什么研究这个问题

积空间 \(X \times Y\) 的同调/上同调如何从因子的同调/上同调计算？ Künneth 公式给出答案：在系数为域时， \(H_n(X \times Y) \cong \bigoplus_{p+q=n} H_p(X) \otimes H_q(Y)\)。 交叉积 (cross product) 是实现这一同构的自然映射。

🔍 抽象化方法

Künneth 公式将同调理论中的"积"与代数中的"张量积"联系起来。 系数不是域时有修正项（Tor 项）。

📐 公式推导 / 模型建立

Künneth 公式（域系数 \(k\)）： \[H_n(X \times Y; k) \cong \bigoplus_{p+q=n} H_p(X; k) \otimes_k H_q(Y; k)\] 整系数 Künneth： \[0 \to \bigoplus_{p+q=n} H_p(X) \otimes H_q(Y) \to H_n(X \times Y) \to \bigoplus_{p+q=n-1} \mathrm{Tor}_1(H_p(X), H_q(Y)) \to 0\] 上同调中的交叉积： \(\times: H^p(X; R) \otimes H^q(Y; R) \to H^{p+q}(X \times Y; R)\)， \(\alpha \times \beta = \pi_X^* \alpha \smile \pi_Y^* \beta\)。 \[H^*(X \times Y; k) \cong H^*(X; k) \otimes_k H^*(Y; k) \quad (\text{分次环同构})\]

🌐 实际应用

- \(H^*(T^n; \mathbb{Z}) = \Lambda^*[\alpha_1, \ldots, \alpha_n]\)（\(\deg \alpha_i = 1\)， 外代数），因为 \(T^n = (S^1)^n\)

✏️ 例题分析与易错点

例题：计算 \(H_*(T^2; \mathbb{Z})\)。 解答：\(T^2 = S^1 \times S^1\)。\(H_0(S^1) = \mathbb{Z}\)，\(H_1(S^1) = \mathbb{Z}\)。 \(H_0(T^2) = \mathbb{Z} \otimes \mathbb{Z} = \mathbb{Z}\) \(H_1(T^2) = (H_0 \otimes H_1) \oplus (H_1 \otimes H_0) = \mathbb{Z}^2\) \(H_2(T^2) = H_1 \otimes H_1 = \mathbb{Z}\) 无 Tor 项（\(H_*(S^1)\) 自由）。 易错点：若因子有挠同调，Tor 项非零。例如 \(H_*(RP^2 \times RP^2)\) 需要 Tor 修正。
