Lie 代数表示入门

01Lie 代数的定义与例子

🎯 为什么研究这个问题

有限群表示处理的是离散对称性。连续对称性（旋转、Lorentz 变换）由 Lie 群描述。Lie 代数是 Lie 群的「线性近似」，它用满足 Jacobi 恒等式的 「括号」运算取代群乘法。Lie 代数是现代物理的数学语言。

🔍 抽象化方法

Lie 代数是向量空间 + 反对称双线性括号 + Jacobi 恒等式。 它是 Lie 群的切空间（无穷小元素的代数）。

📐 公式推导 / 模型建立

Lie 代数: 向量空间 \(\mathfrak{g}\) + 括号 \([\cdot, \cdot]: \mathfrak{g} \times \mathfrak{g} \to \mathfrak{g}\)。 1) 双线性 2) 反对称: \([X, Y] = -[Y, X]\) 3) Jacobi 恒等式: \([X, [Y, Z]] + [Y, [Z, X]] + [Z, [X, Y]] = 0\) 经典例子: - \(\mathfrak{gl}(n)\): \(n \times n\) 矩阵，\([A,B] = AB - BA\) - \(\mathfrak{sl}(n)\): 迹为 0 的矩阵 - \(\mathfrak{so}(n)\): 反对称矩阵 \[[X, Y] = XY - YX\]

🌐 实际应用

粒子物理: 规范对称性由 \(\mathfrak{su}(3)\) 描述（强相互作用）。 量子力学: 角动量算子满足 \([L_x, L_y] = i\hbar L_z\)（\(\mathfrak{so}(3)\)）。

✏️ 例题分析与易错点

例题: 证明 \([A, B] = AB - BA\) 满足 Jacobi 恒等式。 解答: \([A,[B,C]] = A(BC-CB) - (BC-CB)A = ABC - ACB - BCA + CBA\)。 循环求和: \([A,[B,C]] + [B,[C,A]] + [C,[A,B]]\)。 展开: 每个 \(XYZ\) 顺序出现 \(+1\) 和 \(-1\) 各一次，总和为 0。✓ 易错点: 以为 Lie 代数的括号满足结合律。 一般不满足: \([[A,B],C] \neq [A,[B,C]]\)。Jacobi 代替了结合律。

02\(\mathfrak{sl}(2)\) 的表示

🎯 为什么研究这个问题

\(\mathfrak{sl}(2, \mathbb{C})\) 是最小的非 Abel Lie 代数，它的表示论是所有 半单纯 Lie 代数表示论的基石。量子力学中角动量 \(j\) 的理论 恰好是 \(\mathfrak{sl}(2)\)（或 \(\mathfrak{su}(2)\)）的表示论。

🔍 抽象化方法

通过权空间分解（特征值分解）将表示完全分类。 每个有限维不可约表示由一个非负整数参数化。

📐 公式推导 / 模型建立

\(\mathfrak{sl}(2)\) 基: \(H = \begin{pmatrix}1&0\\0&-1\end{pmatrix}\)， \(E = \begin{pmatrix}0&1\\0&0\end{pmatrix}\)，\(F = \begin{pmatrix}0&0\\1&0\end{pmatrix}\)。 \([H, E] = 2E\)，\([H, F] = -2F\)，\([E, F] = H\)。 有限维不可约表示: 对每个 \(n \geq 0\)，存在唯一的 \(n+1\) 维 不可约表示 \(V_n\)。 基: \(\{v_0, v_1, \ldots, v_n\}\)，\(H v_k = (n - 2k)v_k\)。 \[H v_k = (n-2k)v_k, \quad E v_k = (n-k+1) v_{k-1}, \quad F v_k = (k+1) v_{k+1}\] 权（\(H\) 的特征值）: \(n, n-2, \ldots, -n\)。

🌐 实际应用

量子角动量: \(j = n/2\)，\(m = -j, -j+1, \ldots, j\)。 \(J_+, J_-\) 是升降算子，\(J_z\) 是对角可观测量。

✏️ 例题分析与易错点

例题: 写出 \(V_2\)（3 维）的矩阵表示。 解答: \(H = \mathrm{diag}(2, 0, -2)\)。 \(E = \begin{pmatrix}0&2&0\\0&0&1\\0&0&0\end{pmatrix}\)， \(F = \begin{pmatrix}0&0&0\\1&0&0\\0&2&0\end{pmatrix}\)。 验证: \([E, F] = H\)。✓ 易错点: \(E, F\) 的系数不是全 1，而是 \(k+1\) 和 \(n-k+1\)。

03Lie 代数的权空间分解

🎯 为什么研究这个问题

\(\mathfrak{sl}(2)\) 中 \(H\) 的特征值分解可以推广。对任意半单纯 Lie 代数， 选取最大交换子代数（Cartan 子代数 \(\mathfrak{h}\)）， 将表示空间按 \(\mathfrak{h}\) 的特征值（权）分解。 这是半单纯 Lie 代数表示论的核心框架。

🔍 抽象化方法

权空间分解推广了 \(\mathfrak{sl}(2)\) 的特征值理论。 根系统和 Dynkin 图对半单纯 Lie 代数给出完全分类。

📐 公式推导 / 模型建立

Cartan 子代数: \(\mathfrak{h} \subseteq \mathfrak{g}\)，最大交换子代数。 \(\mathrm{rank}(\mathfrak{g}) = \dim \mathfrak{h}\)。 根分解: \[\mathfrak{g} = \mathfrak{h} \oplus \bigoplus_{\alpha \in \Phi} \mathfrak{g}_\alpha\] \(\mathfrak{g}_\alpha = \{X : [H, X] = \alpha(H)X, \forall H \in \mathfrak{h}\}\)， \(\alpha \in \mathfrak{h}^*\) 称为根。 表示的权分解: \(V = \bigoplus_{\mu \in \mathfrak{h}^*} V_\mu\)， \(V_\mu = \{v : Hv = \mu(H)v\}\)。 不可约表示由最高权 \(\lambda\) 唯一确定。

✏️ 例题分析与易错点

例题: \(\mathfrak{sl}(3)\) 的根系统。 解答: \(\mathrm{rank} = 2\)，\(\mathfrak{h} = \{\mathrm{diag}(a,b,c) : a+b+c=0\}\)。 根: \(\pm(e_1-e_2), \pm(e_2-e_3), \pm(e_1-e_3)\)，共 6 个。 正根: \(\alpha_1 = e_1-e_2, \alpha_2 = e_2-e_3\)。 Dynkin 图: \(A_2\)（两个节点连一条线）。 易错点: 将根和权混淆。根是 Lie 代数自身的特征值， 权是表示空间的特征值。

04展望——等变神经网络与对称性

🎯 为什么研究这个问题

表示论在现代 AI 中复活: 等变神经网络将群对称性内建到神经网络架构中。 CNN 对平移等变，Graph NN 对置换等变，这些都是表示论的具体应用。 这是抽象代数与深度学习的前沿交叉。

🔍 抽象化方法

等变性 = 网络的输出与群作用「交换」: \(f(\rho(g)x) = \rho'(g)f(x)\)。 这正是表示论中 \(G\)-等变映射的概念。

📐 公式推导 / 模型建立

等变映射: \(f: V \to W\) 满足 \(f(\rho_V(g)v) = \rho_W(g)f(v)\)。 神经网络中的等变性: - CNN: \(G = (\mathbb{Z}^2, +)\)（平移群），卷积 = 等变线性层 - GNN: \(G = S_n\)（置换群），池化 = 不变层 一般框架: \(G\)-等变线性层的参数由 \(\mathrm{Hom}_G(\rho_{\mathrm{in}}, \rho_{\mathrm{out}})\) 描述。 Schur 引理限制了可学习的参数空间。 \[\text{等变线性层} \in \mathrm{Hom}_G(V_{\mathrm{in}}, V_{\mathrm{out}})\]

🌐 实际应用

分子性质预测（SE(3)-等变）、粒子物理模拟（Lorentz 等变）。

✏️ 例题分析与易错点

例题: 为什么 CNN 的卷积层是平移等变的？ 解答: 卷积: \((f * k)(x) = \sum_y k(y) f(x - y)\)。 平移 \(g\): \(T_g f(x) = f(x - g)\)。 \((T_g f * k)(x) = \sum_y k(y) f(x - g - y) = (f * k)(x - g) = T_g(f*k)(x)\)。 即 \(T_g \circ (\cdot * k) = (\cdot * k) \circ T_g\)。等变。 易错点: 混淆等变和不变。 等变: \(f(gx) = g \cdot f(x)\)（输出也变换）。 不变: \(f(gx) = f(x)\)（输出不变）。
