不可约表示与 Schur 引理

01不可约表示

🎯 为什么研究这个问题

就像素数是整数的基本构件，不可约表示是表示论的「原子」。 Maschke 定理保证每个表示可分解为不可约表示的直和，所以问题 归结为: 每个群有哪些不可约表示？

🔍 抽象化方法

不可约表示是「不可再分解」的表示，即无非平凡 \(G\)-不变子空间。

📐 公式推导 / 模型建立

不可约: \(V\) 无非平凡 \(G\)-不变子空间。 等价: \(V \neq 0\) 且唯一的 \(G\)-不变子空间是 \(0\) 和 \(V\)。 不可约表示个数: \(|G|\) 有限时，\(\mathbb{C}\) 上不可约表示的个数 = \(G\) 的共轭类个数。 分解唯一性: \[V \cong V_1^{\oplus n_1} \oplus \cdots \oplus V_r^{\oplus n_r}\] 其中 \(V_i\) 不可约，\(n_i\) 是重数，均唯一确定。

✏️ 例题分析与易错点

例题: 列出 \(S_3\) 的所有不可约复表示。 解答: \(S_3\) 有 3 个共轭类: \(\{e\}\)，\(\{(12),(13),(23)\}\)，\(\{(123),(132)\}\)。 故有 3 个不可约表示，维数 \(1, 1, 2\)（\(1+1+4=6\)）。 \(\rho_1\): 平凡。\(\rho_2\): 符号。\(\rho_3\): 2维标准。 易错点: 将共轭类个数和表示维数混淆。 共轭类个数 = 不可约表示个数，不是维数。

🌐 实际应用

光谱学：分子电子态按对称群的不可约表示分类。选择定则规定只有特定不可约表示之间的跃迁被允许——判断谱线是否出现，本质上是检查两个不可约表示的张量积是否含平凡分量。

密码学：有限群上的特征标和（character sums）提供密码安全性的数学保障。不可约表示的正交性使特征标和具有指数衰减性质，这是 Weil 界和差集、序列设计等密码学构造的核心工具。

化学：分子轨道按点群的不可约表示标记（如 \(A_1, B_2, E\)）。共价键类型（\(\sigma, \pi, \delta\)）对应不同不可约表示的组合，杂化轨道理论的数学基础就是将自然表示分解为不可约直和。

⚡ 快速检测
\(|G| = 12\) 且 \(G\) 有 4 个共轭类。可能的不可约表示维数为？

02Schur 引理

🎯 为什么研究这个问题

不可约表示之间的关系是什么？Schur 引理说: 不同不可约表示之间 的等变映射必为零，同一不可约表示的等变映射必为数量矩阵。 这是表示论中最基本也最强大的工具。

🔍 抽象化方法

Schur 引理是「不可约表示之间最大程度正交」的体现。

📐 公式推导 / 模型建立

Schur 引理: 设 \(\rho_1, \rho_2\) 不可约，\(T: V_1 \to V_2\) 为 \(G\)-等变映射。 则: - \(\rho_1 \not\cong \rho_2 \Rightarrow T = 0\) - \(\rho_1 = \rho_2\)，\(K = \mathbb{C} \Rightarrow T = \lambda I\) 证明: \(\ker T\) 和 \(\mathrm{im} T\) 都是 \(G\)-不变子空间。 不可约性 \(\Rightarrow\) \(\ker T = 0\) 或 \(V_1\)，\(\mathrm{im} T = 0\) 或 \(V_2\)。 若 \(\rho_1 \not\cong \rho_2\)，\(T\) 不可能是同构，故 \(T = 0\)。 若 \(V_1 = V_2\)，\(T - \lambda I\) 也是 \(G\)-等变，取特征值得 \(T = \lambda I\)。 \[T \in \mathrm{Hom}_G(V_i, V_j) \Rightarrow T = \begin{cases} 0 & i \neq j \\ \lambda I & i = j \end{cases}\]

✏️ 例题分析与易错点

例题: \(\mathbb{C}[S_3]\) 中，平凡表示 \(\rho_1\) 和符号表示 \(\rho_2\) 之间 的 \(G\)-等变映射是什么？ 解答: \(\rho_1 \not\cong \rho_2\)，由 Schur 引理，\(\mathrm{Hom}_G(\rho_1, \rho_2) = 0\)。 易错点: Schur 引理中 \(T = \lambda I\) 要求 \(K\) 代数闭。 \(\mathbb{R}\) 上可能有 \(T\) 不是数量矩阵（如旋转）。

🌐 实际应用

量子力学：Wigner-Eckart 定理是 Schur 引理的直接推论——对称群不可约表示空间之间的等变算子由一个约化矩阵元完全确定。这将角动量耦合中的矩阵元计算从 \(d^2\) 个独立量简化为 1 个。

机器学习：等变神经网络中 \(G\)-等变线性层的参数空间为 \(\mathrm{Hom}_G(\rho_{\mathrm{in}}, \rho_{\mathrm{out}})\)。Schur 引理表明：不同不可约表示之间无可学习参数，同一不可约表示仅有一个标量参数 \(\lambda I\)，极大压缩了参数空间。

调和分析：紧群上的 Peter-Weyl 定理将 \(L^2(G)\) 分解为不可约表示的矩阵系数。Schur 正交性保证了这些矩阵系数构成正交基，推广了经典 Fourier 分析到非交换群。

⚡ 快速检测
\(\dim \mathrm{Hom}_G(V, V) = ?\)（\(V\) 不可约，\(K = \mathbb{C}\)）

03正交关系与分解重数

🎯 为什么研究这个问题

给定一个表示 \(V\)，它分解为 \(V \cong \bigoplus n_i V_i\)。如何计算重数 \(n_i\)？ Schur 引理 + 内积给出: \(n_i = \dim \mathrm{Hom}_G(V_i, V)\)。 后续用特征标的内积可以直接计算。

🔍 抽象化方法

核心思路是将 Schur 引理从"映射维数"翻译为"函数内积"。用特征标 \(\chi_i(g) = \mathrm{tr}(\rho_i(g))\) 代替表示矩阵后，\(\mathrm{Hom}_G\) 的维数等于类函数的 Hermite 内积。

类函数空间 \(\mathrm{Cl}(G)\) 是 \(r\) 维的（\(r =\) 共轭类数），赋予内积 \(\langle f, g \rangle = \frac{1}{|G|}\sum \bar{f}g\) 后成为 Hilbert 空间。不可约特征标 \(\{\chi_1, \ldots, \chi_r\}\) 是其正交归一基。

任何表示的特征标可唯一展开为 \(\chi_V = \sum n_i \chi_i\)，系数 \(n_i = \langle \chi_V, \chi_i \rangle\) 即是不可约分量的重数。这将"分解表示"化约为"计算内积"。

📐 公式推导 / 模型建立

重数公式: \(V \cong \bigoplus n_i V_i\)，则 \[n_i = \dim \mathrm{Hom}_G(V_i, V)\] 证明: \(\mathrm{Hom}_G(V_i, V) \cong \mathrm{Hom}_G(V_i, \bigoplus n_j V_j) = \bigoplus n_j \mathrm{Hom}_G(V_i, V_j)\)。 由 Schur: \(\mathrm{Hom}_G(V_i, V_j) = \delta_{ij} \mathbb{C}\)。 故 \(\dim \mathrm{Hom}_G(V_i, V) = n_i\)。 等价地（用特征标）: \[n_i = \langle \chi_V, \chi_{V_i} \rangle = \frac{1}{|G|} \sum_{g \in G} \chi_V(g) \overline{\chi_{V_i}(g)}\]

✏️ 例题分析与易错点

例题: \(S_3\) 的正则表示 \(\mathbb{C}[S_3]\) 中各不可约表示的重数。 解答: \(\mathbb{C}[S_3] \cong \rho_1^{\oplus 1} \oplus \rho_2^{\oplus 1} \oplus \rho_3^{\oplus 2}\)。 重数 = 维数: \(n_i = d_i\)（正则表示的特殊性质）。 易错点: 重数与维数混淆。 一般表示中 \(n_i\) 和 \(d_i = \dim V_i\) 的关系无规律。

🌐 实际应用

分子光谱：判断某振动模式是否红外活性，等价于计算振动表示与平移表示的特征标内积 \(\langle \chi_{\mathrm{vib}}, \chi_T \rangle\)。若内积非零则该模式可被红外光激发，避免了逐一计算矩阵元。

数论：Burnside \(p^a q^b\) 定理用特征标正交性证明阶为 \(p^a q^b\) 的群可解。Dirichlet \(L\)-函数和素数在算术级数中的分布证明也依赖类函数的正交性。

统计数据分析：对称群特征标的正交性是非交换 Fourier 分析的基础。在排列数据（如投票排名）上做统计时，将频率函数按 \(S_n\) 的不可约特征标展开，各分量重数揭示数据的对称性结构。

⚡ 快速检测

04幂等元与投影公式

🎯 为什么研究这个问题

群代数 \(\mathbb{C}[G]\) 中存在特殊元素 \(e_i\)，它们是幂等元（\(e_i^2 = e_i\)）， 对应于各不可约表示的投影。这给出了分解的明确公式。

🔍 抽象化方法

中心幂等元将"从内积计算重数"上升为"在群代数中直接投影"。不必先求特征标再算内积，而是直接用代数元素 \(e_i\) 作用于表示空间，切出对应的不可约分量。

正交幂等元分解 \(1 = e_1 + e_2 + \cdots + e_r\) 对应了单位元的"频率分解"。每个 \(e_i\) 属于群代数的中心，在任何 \(\mathbb{C}[G]\)-模上都是投影算子，像空间恰好是 \(V_i\)-同型分量。

这给出了算法化的分解流程：给定任何表示 \(V\)，只需计算 \(e_i v\) 即可得到 \(V\) 在 \(V_i\) 方向上的投影，不再需要解特征方程或寻找不变子空间。

📐 公式推导 / 模型建立

中心幂等元: 定义 \[e_i = \frac{d_i}{|G|} \sum_{g \in G} \overline{\chi_i(g)} \cdot g \in \mathbb{C}[G]\] 性质: \(e_i^2 = e_i\)，\(e_i e_j = 0\)（\(i \neq j\)），\(\sum e_i = 1\)。 \(e_i\) 作用于任何 \(\mathbb{C}[G]\)-模 \(V\) 上给出 \(V_i\)-同型分量的投影。

✏️ 例题分析与易错点

例题: \(G = \mathbb{Z}/2\mathbb{Z} = \{e, g\}\)。求中心幂等元。 解答: 两个不可约表示: \(\chi_1(g) = 1\)，\(\chi_2(g) = -1\)。 \(e_1 = \frac{1}{2}(e + g)\)，\(e_2 = \frac{1}{2}(e - g)\)。 验证: \(e_1^2 = \frac{1}{4}(e + 2g + g^2) = \frac{1}{4}(e + 2g + e) = \frac{1}{2}(e + g) = e_1\)。✓ 易错点: 忘记取共轭（复表示时）。 \(e_i\) 中是 \(\overline{\chi_i(g)}\)，不是 \(\chi_i(g)\)。

🌐 实际应用

量子信息：量子纠错码的设计利用群代数幂等元投影。Knill-Laflamme 条件可用投影公式表述：码空间是群作用下特定不可约分量的像，\(e_i\) 直接给出编码投影算子。

信号滤波：循环群的中心幂等元就是 DFT 基向量，它们作用于信号相当于理想带通滤波器。推广到非交换群可设计对称性适配的滤波器组（如球谐滤波）。

晶体学：晶体场理论中，用点群的幂等元将电子轨道分解为对称性适配基组。投影算子 \(e_i\) 从原子轨道线性组合中提取属于特定不可约表示（如 \(t_{2g}, e_g\)）的分量。
