特征标理论

01特征标的定义与基本性质

🎯 为什么研究这个问题

矩阵表示依赖基的选取，但迹不依赖。特征标 \(\chi(g) = \mathrm{tr}(\rho(g))\) 抽取了表示的「基无关」信息，而且惊人地，它完全确定了表示 （在 \(\mathbb{C}\) 上）。这使得表示论可以用函数而非矩阵来研究。

🔍 抽象化方法

特征标是 \(G \to \mathbb{C}\) 的类函数（在共轭类上常值）， 它把表示理论化约为函数的内积理论。

📐 公式推导 / 模型建立

特征标: \(\chi_V(g) = \mathrm{tr}(\rho(g))\)。 \(\chi_V(e) = \dim V\)。 \(\chi_V(g^{-1}) = \overline{\chi_V(g)}\)（幺表示）。 \(\chi_V\) 是类函数: \(\chi_V(hgh^{-1}) = \chi_V(g)\)。 运算: \(\chi_{V \oplus W} = \chi_V + \chi_W\)， \(\chi_{V \otimes W} = \chi_V \cdot \chi_W\)， \(\chi_{V^*} = \overline{\chi_V}\)。 \[\chi_{V \otimes W}(g) = \chi_V(g) \chi_W(g)\]

🌐 实际应用

分子振动分析: 通过计算振动表示的特征标分解确定红外活性和拉曼活性。

✏️ 例题分析与易错点

例题: 计算 \(S_3\) 标准 2 维表示 \(\rho_3\) 的特征标。 解答: \(\rho_3(e) = I_2\)，\(\chi(e) = 2\)。 \((12)\): 交换前两个坐标，\(\mathrm{tr} = 0\)。 \((123)\): 循环置换，\(\mathrm{tr} = -1\)。 特征标表: \(\chi_3 = (2, 0, -1)\)。 易错点: 忘记特征标是类函数，逐元素计算而不是逐共轭类。 计算时只需对每个共轭类的代表元计算即可。

⚡ 快速检测

02特征标的正交关系

🎯 为什么研究这个问题

不可约表示的特征标组成了类函数空间的「正交基」。这些正交关系是 计算分解重数、判断不可约性的核心工具。

🔍 抽象化方法

类函数的内积将表示论变为线性代数，特征标是正交基。

📐 公式推导 / 模型建立

内积: 类函数 \(f, g: G \to \mathbb{C}\) 的内积: \[\langle f, g \rangle = \frac{1}{|G|} \sum_{g \in G} f(g) \overline{g(g)}\] 第一正交关系: \(\langle \chi_i, \chi_j \rangle = \delta_{ij}\)。 推论: - \(\chi_V\) 不可约 \(\Leftrightarrow\) \(\langle \chi_V, \chi_V \rangle = 1\) - 分解重数: \(n_i = \langle \chi_V, \chi_i \rangle\) \[\langle \chi_i, \chi_j \rangle = \delta_{ij}\]

✏️ 例题分析与易错点

例题: 验证 \(S_3\) 的 \(\chi_3 = (2, 0, -1)\) 是不可约的。 解答: \(\langle \chi_3, \chi_3 \rangle = \frac{1}{6}(1 \cdot 4 + 3 \cdot 0 + 2 \cdot 1) = \frac{6}{6} = 1\)。 内积为 1，故不可约。 易错点: 忘记乘以共轭类的大小。 \(\sum\) 中每个共轭类的贡献要乘以该类的元素个数。

⚡ 快速检测

03特征标表的构造

🎯 为什么研究这个问题

特征标表是一个 \(r \times r\) 的表格（\(r\) = 共轭类/不可约表示的个数）， 它完整编码了群的所有表示信息。构造特征标表是实际计算中的核心技能。

🔍 抽象化方法

特征标表是一个酉矩阵（适当加权后），行是不可约表示，列是共轭类。

📐 公式推导 / 模型建立

特征标表结构: 行标签 \(\chi_1, \ldots, \chi_r\)， 列标签 \(C_1, \ldots, C_r\)（共轭类）。 表顯 \(\chi_i(C_j)\)。 列正交关系: \(\sum_i \chi_i(C_j) \overline{\chi_i(C_k)} = \frac{|G|}{|C_j|} \delta_{jk}\)。 构造策略: 1) 确定 \(r\) 和 \(d_i\)（\(\sum d_i^2 = |G|\)） 2) 第一行全为 1（平凡表示） 3) 用正交关系确定其余表项 \[\sum_i h_j \chi_i(C_j) \overline{\chi_i(C_k)} = |G| \delta_{jk}\]

✏️ 例题分析与易错点

例题: \(S_3\) 的特征标表。 解答: | | \(e\) | \((12)\) | \((123)\) | |---|---|---|---| | \(\chi_1\) | 1 | 1 | 1 | | \(\chi_2\) | 1 | \(-1\) | 1 | | \(\chi_3\) | 2 | 0 | \(-1\) | 验证行正交: \(\langle \chi_1, \chi_2 \rangle = \frac{1}{6}(1-3+2) = 0\)。✓ 易错点: 忘记检核正交性来验证特征标表的正确性。

04不可约表示的维数关系

🎯 为什么研究这个问题

\(\sum d_i^2 = |G|\) 给出了维数的全局约束。更深入地，\(d_i \mid |G|\)。 这些数值约束大大限制了可能的特征标表，是确定表示的重要工具。

🔍 抽象化方法

维数条件体现了群阶与表示维数之间的深层感应。

📐 公式推导 / 模型建立

平方和公式: \(\sum_{i=1}^r d_i^2 = |G|\)。 来自: \(\dim \mathbb{C}[G] = |G|\)，\(\mathbb{C}[G] \cong \bigoplus M_{d_i}\)， \(\dim M_{d_i} = d_i^2\)。 整除性: \(d_i \mid |G|\)。 Abel 群特征: \(G\) Abel \(\Leftrightarrow\) 所有 \(d_i = 1\) \(\Leftrightarrow\) \(r = |G|\)。 \[\sum d_i^2 = |G|, \quad d_i \mid |G|\]

✏️ 例题分析与易错点

例题: \(|G| = 24\)，\(G\) 有 5 个共轭类。求可能的 \((d_1, \ldots, d_5)\)。 解答: \(\sum d_i^2 = 24\)，\(d_i \mid 24\)，且必有 \(d_1 = 1\)。 可能解: \((1, 1, 1, 3, 3)\)（\(S_4\) 的情况是 \((1,1,2,3,3)\)）。 \((1, 1, 2, 3, 3)\): \(1+1+4+9+9 = 24\)。✓ 易错点: 忘记 \(d_i \mid |G|\) 的整除条件。 \(d_i = 5\) 当 \(|G| = 24\) 时不可能（\(5 \nmid 24\)）。
