群表示基础

01表示的定义与基本概念

🎯 为什么研究这个问题

群是抽象的对称性，但线性代数是具体可计算的。表示论将群「嵌入」到 矩阵世界中: \(\rho: G \to GL(V)\)，用矩阵乘法实现群运算。 这把抽象群论问题变成线性代数问题，在物理（粒子对称性）和化学（分子振动）中 应用广泛。

🔍 抽象化方法

表示是群到线性算子的同态，本质上是 \(G\)-模的观点。

📐 公式推导 / 模型建立

表示: \(\rho: G \to GL(V)\) 是群同态。 \(\rho(g_1 g_2) = \rho(g_1)\rho(g_2)\)，\(\rho(e) = I\)。 \(\dim V\) 称为表示的维数/次数。 等价性: \(\rho_1 \sim \rho_2\) \(\Leftrightarrow\) \(\exists\) 可逆 \(T: V_1 \to V_2\)， \(T\rho_1(g) = \rho_2(g)T\)，\(\forall g\)。即矩阵表示同时相似。 常见表示: - 平凡表示: \(\rho(g) = I\)，\(\forall g\) - 正则表示: \(G \to GL(\mathbb{C}[G])\)，\(g \mapsto\) 左乘 \[\rho(g): \sum a_h h \mapsto \sum a_h (gh)\]

🌐 实际应用

分子对称性: 水分子 \(H_2O\) 的对称群 \(C_{2v}\) 的表示确定了分子振动模式。

✏️ 例题分析与易错点

例题: 给出 \(\mathbb{Z}/2\mathbb{Z}\) 的所有 1 维复表示。 解答: \(G = \{e, g\}\)，\(g^2 = e\)。 1 维表示: \(\rho(g) \in \mathbb{C}^\times\)，\(\rho(g)^2 = 1\)，故 \(\rho(g) = \pm 1\)。 两个: 平凡表示 \(\rho(g) = 1\)；符号表示 \(\rho(g) = -1\)。 易错点: 混淆「表示」和「作用」。 表示要求映射到 \(GL(V)\)（可逆线性算子），不仅仅是群对集合的作用。

⚡ 快速检测
群表示 \(\rho: G \to GL(V)\) 满足？

02子表示与完全可约性

🎯 为什么研究这个问题

给定一个表示，能否将它分解为更小的「积木」？子表示和直和分解就是 这种分解。对有限群的复表示，Maschke 定理保证了分解总能做到， 这种「完全可约性」是表示论得以系统化的基础。

🔍 抽象化方法

完全可约性意味着表示的「原子」是不可约表示，任何表示都是 不可约表示的直和。类似于向量空间的基分解。

📐 公式推导 / 模型建立

子表示: \(W \subseteq V\) 是 \(G\)-不变子空间 （\(\forall g, \rho(g)(W) \subseteq W\)）。 \(\rho|_W: G \to GL(W)\) 为子表示。 Maschke 定理: 若 \(G\) 有限，\(\mathrm{char}(K) \nmid |G|\)， 则每个 \(G\)-不变子空间都有 \(G\)-不变补空间。 即每个表示都是不可约表示的直和。 证明思路: 任取投影 \(\pi: V \to W\)，平均化: \[\tilde{\pi} = \frac{1}{|G|} \sum_{g \in G} \rho(g) \circ \pi \circ \rho(g)^{-1}\] \(\tilde{\pi}\) 是 \(G\)-等变投影，\(\ker \tilde{\pi}\) 是 \(G\)-不变补空间。

✏️ 例题分析与易错点

例题: \(S_3\) 的标准 2 维表示 \(V = \{(x,y,z) : x+y+z=0\}\) 是否可约？ 解答: \(\dim V = 2\)。\(S_3\) 的不可约表示维数为 \(1, 1, 2\)。 \(V\) 本身就是 2 维不可约表示（无非平凡 \(G\)-不变子空间）。 易错点: 忘记 Maschke 的条件 \(\mathrm{char}(K) \nmid |G|\)。 \(\mathrm{char}(K) \mid |G|\) 时 Maschke 失效（如 \(\mathbb{F}_2\) 上的 \(\mathbb{Z}/2\mathbb{Z}\)-表示）。

⚡ 快速检测
Maschke 定理的条件是？

03正则表示与群代数

🎯 为什么研究这个问题

正则表示 \(\mathbb{C}[G]\) 是群表示中最「大」的表示，它包含了所有不可约表示 的信息。群代数 \(\mathbb{C}[G]\) 是将群表示问题转化为环论问题的桥梁。

🔍 抽象化方法

群代数 \(\mathbb{C}[G]\) 是以群元为基的向量空间+环。 \(G\)-表示 \(\Leftrightarrow\) \(\mathbb{C}[G]\)-模。

📐 公式推导 / 模型建立

群代数: \(\mathbb{C}[G] = \{\sum_{g \in G} a_g g : a_g \in \mathbb{C}\}\)。 乘法由 \(G\) 的乘法线性扩录。\(\dim \mathbb{C}[G] = |G|\)。 正则表示: \(G\) 左乘作用于 \(\mathbb{C}[G]\)。 Artin-Wedderburn: 若 \(G\) 有限，则 \[\mathbb{C}[G] \cong \bigoplus_{i} M_{d_i}(\mathbb{C})\] 其中 \(d_i = \dim V_i\)（不可约表示维数）。 推论: \(\sum d_i^2 = |G|\)。

✏️ 例题分析与易错点

例题: \(S_3\) 的群代数分解。 解答: \(|S_3| = 6\)。不可约表示维数: \(1, 1, 2\)。 \(1^2 + 1^2 + 2^2 = 6 = |S_3|\)。✓ \(\mathbb{C}[S_3] \cong \mathbb{C} \oplus \mathbb{C} \oplus M_2(\mathbb{C})\)。 易错点: 以为 \(\sum d_i = |G|\)。 是平方和 \(\sum d_i^2 = |G|\)，不是维数和。

⚡ 快速检测
\(|G| = 8\)，不可约表示维数可能为？

04表示的直和、张量积与对偶

🎯 为什么研究这个问题

从已知表示构造新表示的三种基本操作: 直和、张量积、对偶。 张量积在物理中对应复合系统、耍宗量子数的耦合。

🔍 抽象化方法

表示的运算构成了一个「表示环」，其乘法为张量积，加法为直和。

📐 公式推导 / 模型建立

直和: \(\rho_1 \oplus \rho_2: G \to GL(V_1 \oplus V_2)\)， \((\rho_1 \oplus \rho_2)(g) = \rho_1(g) \oplus \rho_2(g)\)。 张量积: \(\rho_1 \otimes \rho_2: G \to GL(V_1 \otimes V_2)\)， \((\rho_1 \otimes \rho_2)(g) = \rho_1(g) \otimes \rho_2(g)\)。 \[\dim(\rho_1 \otimes \rho_2) = \dim \rho_1 \cdot \dim \rho_2\] 对偶表示: \(\rho^*: G \to GL(V^*)\)， \(\rho^*(g)(f) = f \circ \rho(g)^{-1}\)。 当 \(\rho\) 幺时，\(\rho^*(g) = \overline{\rho(g)}^T\)。

🌐 实际应用

粒子物理: 两个粒子的复合表示是各自表示的张量积， Clebsch-Gordan 分解将张量积分解为不可约表示的直和。

✏️ 例题分析与易错点

例题: \(\mathbb{Z}/2\mathbb{Z}\) 的符号表示 \(\varepsilon\)。计算 \(\varepsilon \otimes \varepsilon\)。 解答: \(\varepsilon(g) = -1\)。 \((\varepsilon \otimes \varepsilon)(g) = (-1)(-1) = 1\)。 \(\varepsilon \otimes \varepsilon \cong\) 平凡表示。 易错点: 张量积的维数不是加法。 \(\dim(\rho_1 \otimes \rho_2) = d_1 \cdot d_2\)，\(\dim(\rho_1 \oplus \rho_2) = d_1 + d_2\)。

⚡ 快速检测
\(\dim \rho_1 = 2, \dim \rho_2 = 3\)。\(\dim(\rho_1 \otimes \rho_2) = ?\)
