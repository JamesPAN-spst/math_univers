同态与同构

01群同态的定义、核与像

🎯 为什么研究这个问题

群同态是连接两个群的「结构保持映射」。就像两个国家的地图可能比 例不同但保持了地理关系一样，群同态 \(\varphi(ab)=\varphi(a)\varphi(b)\) 保持了群运算。核和像分别刻画了映射「丢失了什么信息」和「覆盖了 多少目标」。

🔍 抽象化方法

同态是范畴论中态射的原型——保持结构的映射。 核 \(\ker\varphi\) 度量信息损失，像 \(\mathrm{Im}(\varphi)\) 度量覆盖范围。

📐 公式推导 / 模型建立

定义: \(\varphi: G \to H\) 是群同态，若 \(\forall a,b \in G,\; \varphi(ab) = \varphi(a)\varphi(b)\)。 自动有: \(\varphi(e_G) = e_H\)，\(\varphi(a^{-1}) = \varphi(a)^{-1}\)。 核: \(\ker\varphi = \{g \in G : \varphi(g) = e_H\}\)。 \(\ker\varphi \trianglelefteq G\)（正规子群！）。 证: \(g \in G, n \in \ker\varphi \Rightarrow \varphi(gng^{-1}) = \varphi(g)e_H \varphi(g)^{-1} = e_H\)。 像: \(\mathrm{Im}(\varphi) = \{\varphi(g) : g \in G\} \leq H\)。 \(\varphi\) 单射 \(\Leftrightarrow \ker\varphi = \{e\}\)。 \(\varphi\) 满射 \(\Leftrightarrow \mathrm{Im}(\varphi) = H\)。 \[|\mathrm{Im}(\varphi)| = [G : \ker\varphi]\]

🌐 实际应用

行列式 \(\det: GL_n(K) \to K^*\) 是群同态，其核为 \(SL_n(K)\)； 符号映射 \(\mathrm{sgn}: S_n \to \{\pm 1\}\) 的核为 \(A_n\)。

✏️ 例题分析与易错点

例题: 设 \(\varphi: \mathbb{Z} \to \mathbb{Z}/5\mathbb{Z}\) 为自然投射 \(n \mapsto \overline{n}\)。求 \(\ker\varphi\) 和 \(\mathrm{Im}(\varphi)\)。 解答: \(\ker\varphi = 5\mathbb{Z} = \{\ldots,-5,0,5,10,\ldots\}\)。 \(\mathrm{Im}(\varphi) = \mathbb{Z}/5\mathbb{Z}\)（满射）。 \([\mathbb{Z} : 5\mathbb{Z}] = 5 = |\mathbb{Z}/5\mathbb{Z}|\)。✓ 易错点: 忘记验证 \(\varphi(e_G) = e_H\) 不是额外条件—— 它自动由 \(\varphi(ab)=\varphi(a)\varphi(b)\) 推出 （取 \(a = b = e\)）。

02第一同构定理

🎯 为什么研究这个问题

第一同构定理（也称基本同构定理）是群论中最核心的定理：它将 同态的核、像和商群三者精确联系起来。直觉上说，「商掉映射丢失的 信息（核），得到的恰好就是映射实际到达的部分（像）」。这个定理 几乎是后续所有结构定理的基石。

🔍 抽象化方法

\(G/\ker\varphi \cong \mathrm{Im}(\varphi)\) 体现了「商 = 像」的 范畴化原则——任何满同态都可以分解为自然投射后跟同构。

📐 公式推导 / 模型建立

第一同构定理: 设 \(\varphi: G \to H\) 为群同态，则 \[G / \ker\varphi \;\cong\; \mathrm{Im}(\varphi)\] 证明: 定义 \(\bar{\varphi}: G/\ker\varphi \to \mathrm{Im}(\varphi)\)， \(\bar{\varphi}(g \ker\varphi) = \varphi(g)\)。 - 良定义: \(g' \in g\ker\varphi \Rightarrow g' = gn,\; \varphi(g') = \varphi(g)\varphi(n) = \varphi(g)\)。 - 同态: \(\bar{\varphi}(g_1 N \cdot g_2 N) = \varphi(g_1 g_2) = \varphi(g_1)\varphi(g_2)\)。 - 单射: \(\bar{\varphi}(gN) = e \Rightarrow g \in \ker\varphi \Rightarrow gN = N\)。 - 满射: 由定义直接成立。

🌐 实际应用

\(\det: GL_n \to K^*\) 给出 \(GL_n / SL_n \cong K^*\)； \(\mathrm{sgn}: S_n \to \{\pm 1\}\) 给出 \(S_n / A_n \cong \mathbb{Z}/2\mathbb{Z}\)。

✏️ 例题分析与易错点

例题: 设 \(\varphi: \mathbb{Z} \to \mathbb{Z}/n\mathbb{Z}\) 为自然同态， 用第一同构定理确定 \(\mathbb{Z}/n\mathbb{Z}\) 的结构。 解答: \(\ker\varphi = n\mathbb{Z}\)，\(\mathrm{Im}(\varphi) = \mathbb{Z}/n\mathbb{Z}\)。 由第一同构定理 \(\mathbb{Z}/n\mathbb{Z} \cong \mathbb{Z}/n\mathbb{Z}\)。✓ 这看起来同义反复，但它确认了商群构造确实产生了正确的结构。 易错点: 使用第一同构定理时忘记验证 \(\varphi\) 是同态。 必须先确认映射保持运算，才能应用定理。

03第二、第三同构定理与对应定理

🎯 为什么研究这个问题

第一同构定理只是开始。第二定理（菱形定理）刻画两个子群的「交互」； 第三定理告诉我们「商的商还是商」；对应定理则建立了 \(G/N\) 的子群 与 \(G\) 中包含 \(N\) 的子群之间的完美双射。这些定理构成了群结构 分析的完整工具箱。

🔍 抽象化方法

对应定理是「格同构」的体现——商群的子群格与母群中包含核的 子群格保持包含关系和正规性。

📐 公式推导 / 模型建立

第二同构定理（菱形定理）: 设 \(H \leq G\)，\(N \trianglelefteq G\)， 则 \(HN \leq G\)，\(H \cap N \trianglelefteq H\)，且 \[H / (H \cap N) \cong HN / N\] 第三同构定理: 设 \(K \trianglelefteq N \trianglelefteq G\)， \(K \trianglelefteq G\)，则 \[(G/K) / (N/K) \cong G/N\] 对应定理: 设 \(N \trianglelefteq G\)，则 \(G/N\) 的子群 与 \(G\) 中包含 \(N\) 的子群之间存在保序双射： \(\overline{H} \leftrightarrow H\)（\(N \subseteq H \subseteq G\)）， 且 \(\overline{H} \trianglelefteq G/N \Leftrightarrow H \trianglelefteq G\)。

✏️ 例题分析与易错点

例题: 在 \(\mathbb{Z}\) 中，\(6\mathbb{Z} \subseteq 2\mathbb{Z}\)， 用第三同构定理描述 \((\mathbb{Z}/6\mathbb{Z}) / (2\mathbb{Z}/6\mathbb{Z})\)。 解答: 由第三同构定理: \((\mathbb{Z}/6\mathbb{Z}) / (2\mathbb{Z}/6\mathbb{Z}) \cong \mathbb{Z}/2\mathbb{Z}\)。 其中 \(2\mathbb{Z}/6\mathbb{Z} = \{\overline{0}, \overline{2}, \overline{4}\}\) 是 \(\mathbb{Z}/6\mathbb{Z}\) 的 3 阶子群，商群阶 = 6/3 = 2。✓ 易错点: 使用第三同构定理时忘记验证 \(K \trianglelefteq G\) （不仅仅是 \(K \trianglelefteq N\)）。

04Cayley 定理与自同构群

🎯 为什么研究这个问题

「每个群都可以看成置换群」——Cayley 定理说每个群都同构于某个 对称群的子群。这不仅给出了群的具体几何图像，也说明了对称群的 普遍性。自同构群 \(\mathrm{Aut}(G)\) 则刻画了群自身的「对称性」， 其中内自同构子群与中心的关系揭示了群结构的深层信息。

🔍 抽象化方法

Cayley 定理是群作用理论的特例（\(G\) 通过左乘作用于自身）。 \(\mathrm{Inn}(G) \cong G/Z(G)\) 将自同构群与中心联系起来。

📐 公式推导 / 模型建立

Cayley 定理: 每个群 \(G\) 同构于 \(S_G\)（\(G\) 上置换群）的子群。 证明: 定义 \(\lambda: G \to S_G,\; \lambda_g(x) = gx\)（左乘）。 \(\lambda_{gh}(x) = ghx = \lambda_g(\lambda_h(x))\)，故 \(\lambda\) 是同态。\(\lambda_g = \mathrm{id} \Rightarrow gx = x\) 对所有 \(x\)，故 \(g = e\)，即 \(\lambda\) 单射。 由此 \(G \cong \mathrm{Im}(\lambda) \leq S_G\)。 若 \(|G| = n\)，则 \(G\) 嵌入 \(S_n\)。 自同构群: \(\mathrm{Aut}(G)\) 是 \(G\) 到自身的所有同构组成的群。 内自同构: \(\mathrm{Inn}(G) = \{\varphi_g : x \mapsto gxg^{-1}\}\)。 \(\mathrm{Inn}(G) \trianglelefteq \mathrm{Aut}(G)\)，且 \[\mathrm{Inn}(G) \cong G / Z(G)\]

🌐 实际应用

Cayley 定理为有限群的计算机表示提供了基础—— 任何群都可用置换矩阵来存储和运算。

✏️ 例题分析与易错点

例题: 求 \(\mathrm{Aut}(\mathbb{Z}/6\mathbb{Z})\)。 解答: \(\mathbb{Z}/6\mathbb{Z}\) 的自同构由生成元的像决定。 生成元 \(\overline{1}\) 必须映到同阶元素（阶 6）， 即 \(\{\overline{1}, \overline{5}\}\)。 故 \(|\mathrm{Aut}(\mathbb{Z}/6\mathbb{Z})| = \varphi(6) = 2\)， \(\mathrm{Aut}(\mathbb{Z}/6\mathbb{Z}) \cong \mathbb{Z}/2\mathbb{Z}\)。 易错点: 以为 \(\overline{1}\) 可以映到任意非零元素。 自同构必须保持阶，故像也必须是生成元。
