01群作用的定义与轨道-稳定子定理

🎯 为什么研究这个问题

群的元素不仅能与彼此运算，还能「作用」于其他集合——如旋转群作用 于平面中的点，置换群作用于集合元素。群作用是群论最强大的工具： 它将群的抽象结构与具体的组合、几何问题联系起来。轨道-稳定子定理 是群作用的核心结果。

🔍 抽象化方法

群作用将群的抽象元素“实体化”为集合上的变换。 轨道分解是集合按对称性的「粗粒化」。

📐 公式推导 / 模型建立

定义: 群作用 \(G \times X \to X\)，\((g, x) \mapsto g \cdot x\)，满足: - \(e \cdot x = x\) - \((gh) \cdot x = g \cdot (h \cdot x)\) 轨道: \(\mathrm{Orb}(x) = \{g \cdot x : g \in G\}\)。 稳定子: \(\mathrm{Stab}(x) = G_x = \{g \in G : g \cdot x = x\}\)， \(G_x \leq G\)。 轨道-稳定子定理: \[|\mathrm{Orb}(x)| = [G : G_x] = |G| / |G_x|\] 证明: 建立双射 \(gG_x \leftrightarrow g \cdot x\)。 \(g_1 \cdot x = g_2 \cdot x \Leftrightarrow g_2^{-1}g_1 \in G_x \Leftrightarrow g_1 G_x = g_2 G_x\)。 轨道分解: \(X = \bigsqcup_{\text{orbits}} \mathrm{Orb}(x_i)\)， 故 \(|X| = \sum_i [G : G_{x_i}]\)。

🌐 实际应用

分子的空间构型由旋转群作用下的轨道结构确定； Rubik 魔方的状态空间是群作用的典型实例。

✏️ 例题分析与易错点

例题: \(D_3\) 作用于正三角形的顶点集 \(\{1,2,3\}\)。 求顶点 1 的轨道和稳定子。 解答: \(\mathrm{Orb}(1) = \{1,2,3\}\)（任何顶点可转到任何其他顶点）。 \(G_1 = \{e, s_1\}\) 其中 \(s_1\) 是关于顶点 1 的反射。 验证: \(|\mathrm{Orb}(1)| = 3 = |D_3|/|G_1| = 6/2\)。✓ 易错点: 混淆轨道与稳定子的角色——轨道是集合 \(X\) 的子集， 稳定子是群 \(G\) 的子群。

02类方程与 p-群的中心

🎯 为什么研究这个问题

当群 \(G\) 通过共轭作用于自身时，轨道就是共轭类。类方程将群的 阶分解为中心元素与非平凡共轭类的贡献，这是研究 \(p\)-群的关键工具， 直接推出 \(p\)-群的中心非平凡这一深刻结果。

🔍 抽象化方法

类方程是轨道分解公式应用于共轭作用的特殊情形。 它把群的规模信息转化为模算条件。

📐 公式推导 / 模型建立

\(G\) 通过共轭作用于自身: \(g \cdot x = gxg^{-1}\)。 不动点: \(g \cdot x = x \Leftrightarrow gx = xg\)，即不动点集 = \(Z(G)\)。 轨道 = 共轭类 \(\mathrm{Cl}(x)\)。 类方程: \[|G| = |Z(G)| + \sum_{i} [G : C_G(g_i)]\] 其中求和遍历所有大小 \(>1\) 的共轭类的代表元 \(g_i\)。 \(p\)-群的中心非平凡: 设 \(|G| = p^n\)，\(p\) 素数。 类方程中每个 \([G:C_G(g_i)]\) 整除 \(p^n\) 且 \(>1\)， 故 \([G:C_G(g_i)]\) 是 \(p\) 的倍数。由此 \(|Z(G)| = |G| - \sum [G:C_G(g_i)] \equiv 0 \pmod{p}\)， 且 \(|Z(G)| \geq 1\)（含 \(e\)），故 \(|Z(G)| \geq p\)。

🌐 实际应用

\(p\)-群中心非平凡的结论直接用于 Sylow 定理的证明； 也推出 \(|G| = p^2\) 的群必为交换群。

✏️ 例题分析与易错点

例题: 证明阶为 \(p^2\) 的群是交换群。 解答: 由类方程，\(|Z(G)| \geq p\)。 若 \(|Z(G)| = p^2\)，则 \(G = Z(G)\) 交换。 若 \(|Z(G)| = p\)，则 \(G/Z(G)\) 阶为 \(p\)，是循环群。 但 \(G/Z(G)\) 循环 \(\Rightarrow\) \(G\) 交换（标准结论）， 与 \(|Z(G)| = p < p^2 = |G|\) 矛盾。故 \(|Z(G)| = p^2\)。 易错点: 忘记「\(G/Z(G)\) 循环 \(\Rightarrow\) \(G\) 交换」这个关键 引理。它的证明：若 \(G/Z(G) = \langle \bar{a} \rangle\)， 则任意 \(g = a^i z\)（\(z \in Z(G)\)），从而 \(gh = hg\)。

03Burnside 引理与计数应用

🎯 为什么研究这个问题

用 \(n\) 种颜色涂染一条有 \(k\) 个珠子的项链，旋转后相同的方案算一种， 有多少种本质不同的染色？Burnside 引理用不动点计数解决这类 「在对称性下的计数」问题，广泛应用于组合学。

🔍 抽象化方法

Burnside 引理将「轨道个数」转化为「平均不动点数」， 把难以直接计算的“本质不同的配置”转为可管理的求和。

📐 公式推导 / 模型建立

Burnside 引理（也称 Cauchy-Frobenius）: 设 \(G\) 作用于有限集 \(X\)，则轨道个数为 \[|X/G| = \frac{1}{|G|} \sum_{g \in G} |\mathrm{Fix}(g)|\] 其中 \(\mathrm{Fix}(g) = \{x \in X : g \cdot x = x\}\)。 证明: 统计集合 \(\{(g,x) : g \cdot x = x\}\) 的大小。 按 \(g\) 求和: \(\sum_g |\mathrm{Fix}(g)|\)。 按 \(x\) 求和: \(\sum_x |G_x| = \sum_x |G|/|\mathrm{Orb}(x)|\)。 后者 \(= |G| \cdot\)(\u6bcf个轨道贡献 1) \(= |G| \cdot |X/G|\)。 两边相等即得引理。

🌐 实际应用

化学中分子式相同但空间结构不同的异构体计数； 编码理论中循环码的等价类计数。

✏️ 例题分析与易错点

例题: 用 2 种颜色涂染正方形的 4 个顶点，旋转下本质不同的 染色方案有多少种？ 解答: 旋转群 \(C_4 = \{e, r, r^2, r^3\}\)，\(|C_4| = 4\)。 \(|\mathrm{Fix}(e)| = 2^4 = 16\)，\(|\mathrm{Fix}(r)| = 2\)（全同色）， \(|\mathrm{Fix}(r^2)| = 2^2 = 4\)，\(|\mathrm{Fix}(r^3)| = 2\)。 \(|X/G| = (16+2+4+2)/4 = 6\)。 易错点: 忘记包含恒等变换 \(e\) 的贡献，或错误计算某个 轮转的不动点数。

04群作用的综合应用

🎯 为什么研究这个问题

群作用的视角统一了许多看似不相关的结果。Cayley 定理是 \(G\) 通过 左乘作用于自身的特例；类方程是共轭作用的轨道分解；Lagrange 定理可由轨道-稳定子定理推出。这个​视角体现了群作用的统一力量。

🔍 抽象化方法

不同类型的群作用（左乘、共轭、作用于陪集集）统一在群作用的 抽象框架下，每种作用提供不同的结构信息。

📐 公式推导 / 模型建立

Cayley 定理作为群作用: \(G\) 左乘作用于 \(X = G\)： \(g \cdot x = gx\)。轨道只有一个（\(X\) 本身）， \(G_x = \{e\}\)（忠实作用）。 由此 \(G \hookrightarrow S_G\)。 \(G\) 作用于陪集集: \(G\) 左乘作用于 \(G/H\)： \(g \cdot aH = gaH\)。左乘作用给出同态 \(\varphi: G \to S_{[G:H]}\)，\(\ker\varphi \subseteq H\)。 应用: 若 \([G:H] = n\)，则 \(G/\ker\varphi\) 嵌入 \(S_n\)， 由此 \(|G/\ker\varphi|\) 整除 \(n!\)。 特别地，若 \([G:H] = p\)（\(G\) 的最小素因子）， 则 \(\ker\varphi = H\)，即 \(H \trianglelefteq G\)。 \[[G:H] = p_{\min}(|G|) \Rightarrow H \trianglelefteq G\]

🌐 实际应用

“最小素因子指数的子群正规”用于 Sylow 分析中快速判断正规性。

✏️ 例题分析与易错点

例题: 设 \(|G| = 30\)，\(H \leq G\)，\(|H| = 15\)。证明 \(H \trianglelefteq G\)。 解答: \([G:H] = 2\)，2 是最小素因子（也是 \(|G|\) 的素因子）。 指数为 2 的子群必定正规。更一般地，指数等于最小 素因子时子群正规。 易错点: 以为“指数为任意素数的子群都正规”。 必须是 \(|G|\) 的最小素因子才行。
