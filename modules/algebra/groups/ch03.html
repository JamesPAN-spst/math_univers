正规子群与商群

01正规子群的定义与等价条件

🎯 为什么研究这个问题

在研究群的商结构时，我们需要一种特殊的子群，使得「陪集上的乘 法」能被良好定义。并非所有子群都允许这样做——只有当左陪集与右 陪集一致时，商集 G/H 才能继承群运算。这种特殊子群就是正规子 群。理解正规子群是打开同态基本定理和群分类理论的钥匙。

🔍 抽象化方法

核心抽象: 共轭不变性。\(N \trianglelefteq G\) 当且仅当对任意 \(g \in G\)，有 \(gNg^{-1} = N\)。这意味着 \(N\) 作为集合在群的 「内部对称变换」下不变——这是一种对称性的对称性。

📐 公式推导 / 模型建立

定义: \(N \trianglelefteq G\) iff \(\forall g \in G, gNg^{-1} = N\)。 等价条件: (a) \(\forall g \in G\), \(gN = Ng\)（左陪集 = 右陪集） (b) \(\forall g \in G, \forall n \in N\), \(gng^{-1} \in N\) (c) \(N\) 是某个群同态 \(\varphi: G \to H\) 的核 证明 (a)\(\Rightarrow\)(b): 若 \(gN = Ng\)，则对任意 \(n \in N\)， \(gn \in gN = Ng\)，故存在 \(n' \in N\) 使 \(gn = n'g\)， 即 \(gng^{-1} = n' \in N\)。 商群构造: 定义 \(G/N = \{gN : g \in G\}\)，运算 \((g_1 N)(g_2 N) = (g_1 g_2)N\)。 良定义性: 若 \(g_1 N = g_1' N\) 且 \(g_2 N = g_2' N\)， 则 \(g_1' = g_1 n_1\), \(g_2' = g_2 n_2\)， \(g_1' g_2' = g_1 n_1 g_2 n_2 = g_1 g_2 (g_2^{-1} n_1 g_2) n_2\)。 因为 \(N\) 正规，\(g_2^{-1} n_1 g_2 \in N\)，故 \(g_1'g_2' \in g_1 g_2 N\)。

🌐 实际应用

物理中正规子群对应「无法被外部破坏」的内禀对称性； 密码学中群的正规子群结构用于分析加密方案的代数安全性。

✏️ 例题分析与易错点

例题: 证明 \(S_3\) 中 \(A_3 = \{e, (123), (132)\}\) 是正规子群。 解答: \([S_3 : A_3] = 2\)，指数为 2 的子群一定正规。 因为只有两个陪集 \(A_3\) 和 \(S_3 \setminus A_3\)， 左右陪集必须相同。 易错点: 误以为「所有子群都是正规子群」。 反例: \(S_3\) 中 \(H = \{e, (12)\}\) 不是正规子群， 因为 \((13)(12)(13)^{-1} = (23) \notin H\)。

02商群 G/N 的构造与性质

🎯 为什么研究这个问题

正规子群的意义在于它允许我们构造「更小的群」——商群 \(G/N\)。 商群是将正规子群 \(N\) 中的元素「压缩为一点」后得到的群，它保留了 \(G\) 中被 \(N\) 忽略的结构。理解商群是建立同态基本定理的基石。

🔍 抽象化方法

商群是一种「粗粒化」——将群中关于 \(N\) 等价的元素归为一类。 这与实数中模运算 \(\mathbb{Z}/n\mathbb{Z}\) 的思想一致。

📐 公式推导 / 模型建立

商群 \(G/N\) 的元素是陪集 \(\{gN : g \in G\}\)，运算为 \((g_1 N)(g_2 N) = (g_1 g_2) N\)。 阶: \(|G/N| = [G:N] = |G|/|N|\)。 单位元: \(eN = N\)。逆元: \((gN)^{-1} = g^{-1}N\)。 自然投影: \(\pi: G \to G/N,\; g \mapsto gN\)， 是满射同态，\(\ker \pi = N\)。 关键性质: 交换群的子群都正规，故交换群的任何子群 都可构造商群。\(G/N\) 交换 \(\Leftrightarrow\) \([G,G] \subseteq N\)， 其中 \([G,G] = \langle ghg^{-1}h^{-1} \rangle\) 为换位子群。 \[|G/N| = [G : N]\]

✏️ 例题分析与易错点

例题: 描述 \(\mathbb{Z}/6\mathbb{Z}\) 中子群 \(H = \{\overline{0}, \overline{3}\}\) 的商群 \((\mathbb{Z}/6\mathbb{Z})/H\)。 解答: 陪集为 \(\overline{0}+H = \{\overline{0},\overline{3}\}\)， \(\overline{1}+H = \{\overline{1},\overline{4}\}\)， \(\overline{2}+H = \{\overline{2},\overline{5}\}\)。 共 3 个陪集，商群同构于 \(\mathbb{Z}/3\mathbb{Z}\)。 易错点: 在商群中运算时代入陪集的具体代表元不同， 以为会得到不同结果。商群运算与代表元选取无关（正规性保证）。

03共轭类与单群

🎯 为什么研究这个问题

共轭是群中最重要的等价关系：\(a \sim b\) 当且仅当存在 \(g\) 使 \(gag^{-1} = b\)。共轭类反映了元素的「本质相似性」。当一个群除了 平凡情形没有正规子群时，称其为单群——这是群论中的「素数」， 是有限群分类的基本构件。

🔍 抽象化方法

共轭是内自同构作用产生的等价关系。正规子群恰好是共轭类的并。 单群是指除 \(\{e\}\) 和 \(G\) 本身外没有正规子群的群——不可再 「分解」为更小的商群。

📐 公式推导 / 模型建立

共轭关系: \(a \sim_c b \Leftrightarrow \exists g \in G,\; gag^{-1} = b\)。 共轭类: \(\mathrm{Cl}(a) = \{gag^{-1} : g \in G\}\)。 由轨道-稳定子定理: \(|\mathrm{Cl}(a)| = [G : C_G(a)]\)。 \(N \trianglelefteq G \Leftrightarrow N\) 是若干共轭类的并。 单群定义: \(G\) 是单群，若 \(|G| > 1\) 且 \(G\) 的正规子群 只有 \(\{e\}\) 和 \(G\)。 例: 素阶群 \(\mathbb{Z}/p\mathbb{Z}\) 是（交换）单群。 \(A_5\)（60阶）是最小的非交换单群。 交换群的子群都正规，故交换单群只有 \(\mathbb{Z}/p\mathbb{Z}\)。

🌐 实际应用

有限单群分类是20世纪代数学最大的成就之一； \(A_5\) 的单性是五次方程不可根式求解的关键。

✏️ 例题分析与易错点

例题: 求 \(S_3\) 的所有共轭类。 解答: 按轮换类型分组: \(\{e\}\)（恒等）, \(\{(12),(13),(23)\}\)（对换）, \(\{(123),(132)\}\)（3-轮换）。共 3 个共轭类。 验证: \(A_3 = \{e,(123),(132)\}\) 恰好是两个共轭类之并， 故 \(A_3 \trianglelefteq S_3\)。✓ 易错点: 以为每个轮换类型只有一个共轭类。在 \(S_n\) 中确实如此， 但在 \(A_n\) 中可能分裂为两个类（如 \(A_5\) 中的 5-轮换）。

04合成列与可解群

🎯 为什么研究这个问题

如何系统地将群「拆解」为单群？合成列 \(G = G_0 \supsetneq G_1 \supsetneq \cdots \supsetneq G_k = \{e\}\) 给出这样的分解链，其中每个 \(G_i/G_{i+1}\) 都是单群。当这些单群都是交换的（即循环 \(p\)-群）， 群就被称为可解群——正是 Galois 用来判定方程可否根式求解的概念。

🔍 抽象化方法

合成列是群结构的「素因子分解」——正如整数分解为素数之积， 群分解为单群之「塔」。

📐 公式推导 / 模型建立

合成列: 子群链 \(G = G_0 \triangleright G_1 \triangleright \cdots \triangleright G_k = \{e\}\)，其中每个 \(G_i / G_{i+1}\) 是单群（称为合成因子）。 Jordan-Hölder 定理（陈述）: 任何两个合成列的合成因子 （作为多重集）相同（不计顺序）。即合成因子唯一确定。 可解群: \(G\) 是可解群，若存在子群链 \(G = G_0 \triangleright G_1 \triangleright \cdots \triangleright G_k = \{e\}\)，使得每个 \(G_i/G_{i+1}\) 是交换群。 等价地，合成因子全为素阶循环群 \(\mathbb{Z}/p\mathbb{Z}\)。 \[S_n \text{ 可解} \Leftrightarrow n \leq 4\] \(A_5\) 是单群 \(\Rightarrow\) \(S_5\) 不可解 \(\Rightarrow\) 五次一般方程不可根式求解。

🌐 实际应用

Galois 理论中，多项式可根式求解 \(\Leftrightarrow\) 其 Galois 群可解。 \(S_5\) 不可解是五次方程不可根式求解的直接原因。

✏️ 例题分析与易错点

例题: 验证 \(S_3\) 是可解群。 解答: \(S_3 \triangleright A_3 \triangleright \{e\}\)。 \(S_3/A_3 \cong \mathbb{Z}/2\mathbb{Z}\)（交换）， \(A_3/\{e\} \cong \mathbb{Z}/3\mathbb{Z}\)（交换）。 每步商群均为交换群，故 \(S_3\) 可解。✓ 易错点: 试图展开 Jordan-Hölder 定理的完整证明—— 该证明需要 Schreier 细化定理及 Zassenhaus 引理， 超出本章范围，只需掌握陈述。
