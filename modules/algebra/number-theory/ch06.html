丢番图方程

01线性丢番图方程

🎯 为什么研究这个问题

线性丢番图方程 \(ax + by = c\) 是最简单的整数方程，但元素的「整数性」 约束使得问题并不平凡。可解性完全由 \(\gcd(a,b) \mid c\) 决定， 解集是线性参数族。Bézout 等式是其完美工具。

📐 公式推导 / 模型建立

可解性: \(ax + by = c\) 有整数解 \(\Leftrightarrow \gcd(a,b) \mid c\)。 证: \(\Rightarrow\): \(\gcd(a,b) \mid a, \gcd(a,b) \mid b\)，故 \(\gcd(a,b) \mid c\)。 \(\Leftarrow\): 由 Bézout，\(\exists x_0,y_0\) 使 \(ax_0+by_0 = \gcd(a,b)\)。 乘以 \(c/\gcd(a,b)\) 即得特解。 通解: 若 \((x_0, y_0)\) 是特解，则通解为 \[x = x_0 + \frac{b}{d}t, \quad y = y_0 - \frac{a}{d}t, \quad t \in \mathbb{Z}\] 其中 \(d = \gcd(a,b)\)。

🌐 实际应用

钱币找零问题：用 5 元和 7 元的硬币能否凑出 \(c\) 元？

✏️ 例题分析与易错点

例题: 求 \(12x + 8y = 20\) 的所有整数解。 解答: \(\gcd(12,8) = 4\)，\(4 \mid 20\)，有解。 简化: \(3x + 2y = 5\)。特解: \(x_0 = 1, y_0 = 1\)。 通解: \(x = 1 + 2t, y = 1 - 3t\)。 易错点: 忘记简化（同除 \(\gcd\)），直接用原方程求解导致参数化错误。

⚡ 快速检测
\(15x + 10y = 7\) 是否有整数解？

02勾股数与参数化

🎯 为什么研究这个问题

\(a^2 + b^2 = c^2\) 是数学史上最古老的问题之一。什么样的整数三元组 满足勾股定理？完全参数化给出了优雅的答案，展示了数论中「参数化」 的强大方法。

🔍 抽象化方法

勾股数的参数化本质是单位圆 \(x^2+y^2=1\) 上有理点的分类—— 连接了数论与代数几何。

📐 公式推导 / 模型建立

定义: 勾股数 \((a,b,c) \in \mathbb{Z}^3\) 满足 \(a^2 + b^2 = c^2\)。 本原勾股数: \(\gcd(a,b,c) = 1\) 且 \(a,b,c > 0\)。 参数化定理: 所有本原勾股数（\(a\) 奇）为 \[a = m^2 - n^2, \quad b = 2mn, \quad c = m^2 + n^2\] 其中 \(m > n > 0\)，\(\gcd(m,n)=1\)，\(m-n\) 为奇。 证明思路: 由 \(a^2 = c^2 - b^2 = (c-b)(c+b)\)，利用 本原性推出 \(c-b\) 和 \(c+b\) 互素，从而各自为完全平方数。

🌐 实际应用

勾股数在信号处理中用于构造整数坐标的直角三角形。

✏️ 例题分析与易错点

例题: 列出所有 \(c \leq 25\) 的本原勾股数。 解答: \(m=2,n=1\): \((3,4,5)\)。\(m=3,n=2\): \((5,12,13)\)。 \(m=4,n=1\): \((15,8,17)\)。\(m=4,n=3\): \((7,24,25)\)。 \(m=3,n=1\)不符合（\(m-n=2\) 为偶）——等等，实际上 \(m=3,n=2\) 满足 \(m-n=1\) 奇。 易错点: 忘记条件 \(m-n\) 为奇，导致生成非本原的三元组。

⚡ 快速检测
\((m,n) = (5,2)\) 生成的勾股数是？

03Pell 方程

🎯 为什么研究这个问题

Pell 方程 \(x^2 - Dy^2 = 1\)（\(D\) 非完全平方数）是最重要的二次丢番图 方程。它有无穷多解，全部由一个「基本解」生成。连分数展开提供了 求基本解的精妙算法。

🔍 抽象化方法

Pell 方程的解集构成一个无穷循环群（通过范数 \(N(x+y\sqrt{D})= x^2-Dy^2\) 连接到二次域的单位群）。

📐 公式推导 / 模型建立

Pell 方程: \(x^2 - Dy^2 = 1\)，\(D > 0\) 非完全平方。 存在性: 存在无穷多组正整数解。 基本解: 最小正解 \((x_1, y_1)\)。 由 \(\sqrt{D}\) 的连分数展开找到: \(\sqrt{D}\) 的周期连分数的 渐进分数 \(p_k/q_k\) 使 \(p_k^2 - Dq_k^2 = \pm 1\)。 通解生成: 若 \((x_1, y_1)\) 是基本解，则所有解由 \[(x_n + y_n\sqrt{D}) = (x_1 + y_1\sqrt{D})^n, \quad n \in \mathbb{Z}\] 给出。

🌐 实际应用

早期印度数学家 Brahmagupta 用 Pell 方程求 \(\sqrt{D}\) 的精确有理逼近。

✏️ 例题分析与易错点

例题: 求 \(x^2 - 2y^2 = 1\) 的基本解。 解答: \(\sqrt{2} = [1; \overline{2}]\)。渐进分数: \(1/1, 3/2, 7/5, \ldots\) \(3^2 - 2 \times 2^2 = 9 - 8 = 1\)。基本解 \((3, 2)\)。 下一组解: \((3+2\sqrt{2})^2 = 17+12\sqrt{2}\)，即 \((17, 12)\)。 验证: \(17^2 - 2 \times 12^2 = 289 - 288 = 1\)。✓ 易错点: 将 \(x^2 - Dy^2 = -1\) 的解与 \(= 1\) 的解混淆。 \(x^2 - Dy^2 = -1\) 不一定有解（如 \(D=3\) 时无解）。

⚡ 快速检测
\(x^2 - 5y^2 = 1\) 的基本解是？

04费马大定理与无穷递降法

🎯 为什么研究这个问题

费马大定理——“\(x^n + y^n = z^n\)（\(n \geq 3\)）无正整数解”——是数学史上 最著名的问题，困扰了数学家 350 年。费马本人用「无穷递降法」 证明了 \(n=4\) 的情况，展示了一种优雅的证明方法。

🔍 抽象化方法

无穷递降法是反证法的特殊形式: 假设存在解，由其构造更小的解， 无穷重复得到无穷递减的正整数序列——矛盾。

📐 公式推导 / 模型建立

费马大定理（陈述）: \(\forall n \geq 3\)， \(x^n + y^n = z^n\) 无正整数解。 1995 年由 Andrew Wiles 证明（依赖椅圆曲线、模形式等深刻理论， 证明细节超出本课范围）。 \(n=4\) 的无穷递降法证明: 证明 \(x^4 + y^4 = z^2\) 无正整数解 （更强的结果）。 假设存在最小解 \((x,y,z)\)。由勾股数参数化， \(x^2 = m^2 - n^2,\; y^2 = 2mn,\; z = m^2 + n^2\)。 经过细致分析，构造出 \((x', y', z')\) 使 \(z' < z\)，矛盾。 \[x^4 + y^4 = z^2 \text{ 无正整数解} \Rightarrow x^4 + y^4 = z^4 \text{ 无正整数解}\]

🌐 实际应用

Wiles 的证明催生了模形式理论的重大发展，影响了整个算术几何。

✏️ 例题分析与易错点

例题: 用无穷递降法证明 \(\sqrt{2}\) 是无理数。 解答: 假设 \(\sqrt{2} = a/b\)，\(\gcd(a,b)=1\)，则 \(a^2 = 2b^2\)。 \(a^2\) 为偶，故 \(a\) 为偶，设 \(a = 2k\)，\(4k^2 = 2b^2\)，\(b^2 = 2k^2\)。 现在 \((b, k)\) 也满足相同关系且 \(b < a\)——无穷递降，矛盾。 易错点: 无穷递降法需要明确指出哪个量在递减（如 \(z\) 或 \(a\)）， 以及为什么它不能无限递减（正整数而言）。
