谱定理与矩阵分解

01正规算子与谱定理

🎯 为什么研究这个问题

不是所有矩阵都可对角化，但实对称矩阵和复正规矩阵总可以正交/酉对角化。 谱定理精确刻画了哪些矩阵可以由特征值「完全理解」， 是量子力学（可观测量）、统计（协方差矩阵）的数学基石。

🔍 抽象化方法

正规算子 = 与伴随算子可交换 (\(AA^* = A^*A\))。 谱定理: 正规 \(\Leftrightarrow\) 酉可对角化。

📐 公式推导 / 模型建立

正规算子: \(A \in M_n(\mathbb{C})\)，\(AA^* = A^*A\)。 包括: 自伴随（\(A = A^*\)）、酉（\(A^*A = I\)）、反自伴随等。 谱定理（有限维）: \(A\) 正规 \(\Leftrightarrow\) 存在酉矩阵 \(U\) 使 \[A = U \mathrm{diag}(\lambda_1, \ldots, \lambda_n) U^*\] 即 \(A\) 有一组标准正交特征基。 实对称情形: \(A = A^T\) 实矩阵 \(\Rightarrow\) 特征值全实， 存在正交矩阵 \(P\) 使 \(A = P D P^T\)。

🌐 实际应用

主成分分析（PCA）: 对协方差矩阵 \(\Sigma\)（实对称正半定）做谱分解。

✏️ 例题分析与易错点

例题: \(A = \begin{pmatrix}1&2\\2&4\end{pmatrix}\)。做正交对角化。 解答: \(\chi_A = x^2 - 5x = x(x-5)\)。特征值 0, 5。 \(\lambda=0\): \(v_1 = \frac{1}{\sqrt{5}}(-2, 1)^T\)。 \(\lambda=5\): \(v_2 = \frac{1}{\sqrt{5}}(1, 2)^T\)。 \(P = \frac{1}{\sqrt{5}}\begin{pmatrix}-2&1\\1&2\end{pmatrix}\)， \(A = P \mathrm{diag}(0, 5) P^T\)。 易错点: 特征向量不一定正交（除非矩阵是正规的）。 一般矩阵需用 Gram-Schmidt 正交化特征空间内的基。

02奇异值分解 (SVD)

🎯 为什么研究这个问题

谱定理只适用于正规矩阵（方阵且 \(AA^* = A^*A\)）。但任意矩阵（包括非方阵） 也有一种类似对角化的分解——奇异值分解 (SVD)。它是数据科学和数值计算中 最重要的矩阵分解，没有之一。

🔍 抽象化方法

SVD 揭示任意线性映射都可以分解为「旋转→缩放→旋转」。

📐 公式推导 / 模型建立

SVD 定理: 任意 \(A \in M_{m \times n}(\mathbb{R})\)，存在正交矩阵 \(U \in O(m)\)，\(V \in O(n)\) 和对角矩阵 \(\Sigma\) 使 \[A = U \Sigma V^T\] \(\Sigma\) 的对角元 \(\sigma_1 \geq \sigma_2 \geq \cdots \geq 0\) 是奇异值。 与谱的关系: \(\sigma_i = \sqrt{\lambda_i(A^T A)}\)。 \(V\) 的列是 \(A^T A\) 的特征向量，\(U\) 的列是 \(AA^T\) 的特征向量。 紧 SVD: \(A = \sum_{i=1}^r \sigma_i u_i v_i^T\)（\(r = \mathrm{rank} A\)）。

🌐 实际应用

推荐系统: Netflix 用 SVD 分解用户-电影评分矩阵做协同过滤。 图像压缩: 保留最大的 \(k\) 个奇异值近似矩阵。

✏️ 例题分析与易错点

例题: \(A = \begin{pmatrix}3&0\\0&2\\0&0\end{pmatrix}\)。求 SVD。 解答: \(A^T A = \mathrm{diag}(9, 4)\)，特征值 9, 4。 \(\sigma_1 = 3, \sigma_2 = 2\)。\(V = I_2\)。 \(u_1 = Av_1/3 = e_1, u_2 = Av_2/2 = e_2\)。取 \(u_3 = e_3\)。 \(U = I_3\)，\(\Sigma = \begin{pmatrix}3&0\\0&2\\0&0\end{pmatrix}\)。 \(A = I_3 \cdot \Sigma \cdot I_2^T\)。 易错点: 混淆特征值和奇异值。奇异值 ≥ 0（永远非负）， 特征值可以是负数或复数。

03低秩逼近与 Eckart-Young 定理

🎯 为什么研究这个问题

在数据科学中，高维矩阵常有「大部分信息集中在少数几个方向」的特性。 Eckart-Young 定理给出了截断 SVD 是最优低秩逼近的严格证明—— 它是 PCA、降维、数据压缩的理论基础。

🔍 抽象化方法

在 Frobenius 范数或谱范数下，截断 SVD 给出最优秩-\(k\) 逼近， 这是凸优化中的 elegant 结果。

📐 公式推导 / 模型建立

Eckart-Young 定理: 设 \(A = \sum_{i=1}^r \sigma_i u_i v_i^T\)， 则 \(A\) 的最优秩-\(k\) 逼近（\(k < r\)）为 \[A_k = \sum_{i=1}^k \sigma_i u_i v_i^T\] 即截断 SVD。误差: \[\|A - A_k\|_F = \sqrt{\sigma_{k+1}^2 + \cdots + \sigma_r^2}\] \[\|A - A_k\|_2 = \sigma_{k+1}\] 信息保留率: \(\frac{\sum_{i=1}^k \sigma_i^2}{\sum_{i=1}^r \sigma_i^2}\)。

🌐 实际应用

图像压缩: \(1000 \times 1000\) 图像保留前 50 个奇异值， 存储量从 \(10^6\) 降到 \(50 \times (1000 + 1000 + 1) \approx 10^5\)。

✏️ 例题分析与易错点

例题: \(\sigma_1 = 10, \sigma_2 = 3, \sigma_3 = 0.5\)。 秩-1 逼近的 Frobenius 误差和信息保留率？ 解答: 误差 \(= \sqrt{9 + 0.25} = \sqrt{9.25} \approx 3.04\)。 保留率 \(= 100 / (100 + 9 + 0.25) = 100/109.25 \approx 91.5\%\)。 易错点: 以为低秩逼近可以用任意 \(k\) 列选取。 Eckart-Young 说只有截断 SVD 是最优的。

04极分解与 Moore-Penrose 伪逆

🎯 为什么研究这个问题

复数 \(z = r e^{i\theta}\) 分解为模和旋转。矩阵也有类似的 「极分解」\(A = UP\)：正交（旋转）× 正半定（缩放）。 当 \(A\) 不可逆或非方阵时，伪逆 \(A^+\) 给出「最好的逆」—— 最小二乘解 \(x = A^+ b\)。

🔍 抽象化方法

极分解推广了复数的极坐标表示到矩阵。 Moore-Penrose 伪逆推广了逆矩阵到非方阵/奇异矩阵。

📐 公式推导 / 模型建立

极分解: \(A = UP\)，其中 \(U\) 酉（正交），\(P = \sqrt{A^*A}\) 正半定。 用 SVD: \(A = U_0 \Sigma V^T\)，则 \(P = V \Sigma V^T\)，\(U = U_0 V^T\)。 Moore-Penrose 伪逆: \(A^+\) 是满足四个条件的唯一矩阵: \(AA^+A = A\)，\(A^+AA^+ = A^+\)，\((AA^+)^* = AA^+\)，\((A^+A)^* = A^+A\)。 用 SVD: \(A = U \Sigma V^T\)，则 \[A^+ = V \Sigma^+ U^T, \quad \Sigma^+ = \mathrm{diag}(\sigma_1^{-1}, \ldots, \sigma_r^{-1}, 0, \ldots)\]

🌐 实际应用

最小二乘: \(\hat{x} = A^+ b\) 是 \(\|Ax - b\|^2\) 的最小范数解。

✏️ 例题分析与易错点

例题: \(A = \begin{pmatrix}1&0\\0&0\end{pmatrix}\)。求 \(A^+\)。 解答: SVD: \(A = I \cdot \mathrm{diag}(1,0) \cdot I\)。 \(\Sigma^+ = \mathrm{diag}(1, 0)\)。 \(A^+ = \mathrm{diag}(1, 0) = A\)。 易错点: 以为 \(A^+ = (A^T A)^{-1} A^T\) 总成立。 这只在 \(A\) 列满秩时成立。一般需用 SVD 公式。
