最小多项式与 Cayley-Hamilton

01最小多项式的定义与性质

🎯 为什么研究这个问题

矩阵 \(A\) 满足哪些多项式方程？最小多项式 \(m_A(x)\) 是零化 \(A\) 的 最低次多项式，它完全确定了 \(A\) 的「代数复杂度」。理解它与特征 多项式的关系是深入研究线性算子的关键。

🔍 抽象化方法

最小多项式是理想 \(\{p(x) : p(A) = 0\}\) 的生成元——反映了 \(K[x] \to \mathrm{End}(V)\) 这个环同态的核结构。

📐 公式推导 / 模型建立

定义: \(m_A(x)\) = 零化 \(A\) 的最低次首一多项式。 即 \(m_A(A) = 0\) 且 \(\deg m_A\) 最小。 性质: 1) \(p(A) = 0 \Rightarrow m_A \mid p\) 2) \(m_A \mid \chi_A\)（\(m_A\) 整除特征多项式） 3) \(m_A\) 与 \(\chi_A\) 有相同的不可约因子（即相同的根） \[m_A(A) = 0, \quad m_A \mid \chi_A\]

✏️ 例题分析与易错点

例题: \(A = \begin{pmatrix}2&0\\0&2\end{pmatrix}\)。 求 \(m_A\) 和 \(\chi_A\)。 解答: \(\chi_A(x) = (x-2)^2\)。 \(A - 2I = 0 \Rightarrow m_A(x) = x - 2\)。 验证: \(m_A \mid \chi_A\): \((x-2) \mid (x-2)^2\)。✓ 易错点: 以为 \(m_A = \chi_A\)。只有当每个特征值的几何重数 = 1 时才有 \(m_A = \chi_A\)。

⚡ 快速检测
\(A = I_n\)，则 \(m_A(x) = ?\)

02Cayley-Hamilton 定理

🎯 为什么研究这个问题

Cayley-Hamilton 定理说每个矩阵满足它自己的特征方程: \(\chi_A(A) = 0\)。 这是线性代数中最深刻的定理之一，将特征值的性质强制加到矩阵本身。 它提供了计算 \(A^{-1}\)、\(A^n\) 的强大工具。

🔍 抽象化方法

矩阵被它的特征多项式「零化」——这是线性算子的一种内在约束。

📐 公式推导 / 模型建立

Cayley-Hamilton: 设 \(A \in M_n(K)\)，\(\chi_A(x) = \det(xI - A)\)，则 \[\chi_A(A) = 0\] 证明策略（伴随矩阵法）: 设 \(B(x) = \mathrm{adj}(xI - A)\)（伴随矩阵）。 由 \((xI - A)B(x) = \chi_A(x) I\)。 \(B(x)\) 是 \(x\) 的多项式矩阵: \(B(x) = B_{n-1}x^{n-1} + \cdots + B_0\)。 展开并比较系数，将 \(x\) 替换为 \(A\)，级联求和得 \(\chi_A(A) = 0\)。 推论: \(m_A \mid \chi_A\)，\(\deg m_A \leq n\)。

🌐 实际应用

计算 \(A^{-1}\): 若 \(\chi_A(x) = x^n + c_{n-1}x^{n-1} + \cdots + c_0\)， 则 \(A^{-1} = -c_0^{-1}(A^{n-1} + c_{n-1}A^{n-2} + \cdots + c_1 I)\)。

✏️ 例题分析与易错点

例题: \(A = \begin{pmatrix}1&2\\3&4\end{pmatrix}\)，验证 \(\chi_A(A) = 0\)。 解答: \(\chi_A(x) = x^2 - 5x - 2\)。 \(A^2 = \begin{pmatrix}7&10\\15&22\end{pmatrix}\)。 \(A^2 - 5A - 2I = \begin{pmatrix}7-5-2&10-10-0\\15-15-0&22-20-2\end{pmatrix} = \begin{pmatrix}0&0\\0&0\end{pmatrix}\)。✓ 易错点: 将 \(\chi_A(A) = \det(A \cdot I - A) = \det(0) = 0\) 作为“证明”。 这是错误的——\(\chi_A(A)\) 是矩阵不是数。

⚡ 快速检测

03不变因子与有理标准形

🎯 为什么研究这个问题

最小多项式和特征多项式还不能完全描述矩阵的结构。不变因子（Smith 标准形）提供了更细致的信息，将多项式矩阵的等价类完全分类。 有理标准形是不需要扩展基域的标准形，在任意域上都可用。

🔍 抽象化方法

Smith 标准形是多项式环上的「对角化」——将 \(xI - A\) 的信息 编码为对角元列。

📐 公式推导 / 模型建立

Smith 标准形: 多项式矩阵 \(xI - A\) 可通过初等行列变换 化为对角形 \(\mathrm{diag}(d_1(x), \ldots, d_n(x))\)， 其中 \(d_i \mid d_{i+1}\)。\(d_i\) 称为不变因子。 \(\chi_A = \prod d_i\)，\(m_A = d_n\)（最后一个）。 有理标准形: 每个不变因子 \(d_i\) 对应一个尺同矩阵： \[C(d_i) = \begin{pmatrix} 0 & & -c_0 \\ 1 & \ddots & \vdots \\ & 1 & -c_{k-1} \end{pmatrix}\] \(A\) 相似于这些尺同矩阵的直和。

✏️ 例题分析与易错点

例题: \(\chi_A = (x-1)(x-2)^2\)，可能的不变因子列表是什么？ 解答: 两种可能: (1) \(d_1 = 1, d_2 = 1, d_3 = (x-1)(x-2)^2\)（\(m_A = \chi_A\)） (2) \(d_1 = 1, d_2 = (x-2), d_3 = (x-1)(x-2)\)（\(m_A = (x-1)(x-2)\)） 第二种对应 \(A\) 可对角化或 \(m_A\) 无重根。 易错点: 忘记 \(d_i \mid d_{i+1}\) 的整除条件。

⚡ 快速检测
Smith 标准形中最后一个不变因子 \(d_n\) 等于？

04最小多项式与对角化判定

🎯 为什么研究这个问题

什么时候矩阵可以对角化？特征值重数的分析是基本方法，而最小多项式 提供了简洁的等价条件: \(A\) 可对角化 \(\Leftrightarrow\) \(m_A\) 无重根。 这比逐个检查特征空间维数更高效。

🔍 抽象化方法

\(m_A\) 无重根 \(\Leftrightarrow\) 广义特征空间 = 普通特征空间， 这是 Jordan 标准形退化为对角形的条件。

📐 公式推导 / 模型建立

对角化充要条件: \(A\) 可对角化 \(\Leftrightarrow\) \(m_A(x)\) 在 \(K\) 上分解为不同一次因子之积: \[m_A(x) = (x - \lambda_1)(x - \lambda_2) \cdots (x - \lambda_r), \quad \lambda_i \text{ 两两不同}\] 证: \(m_A\) 无重根 \(\Rightarrow\) \(V = \bigoplus \ker(A - \lambda_i I)\) （中国剩余定理于多项式），即 \(V\) 是特征空间的直和。 计算方法: 用 \(\chi_A\) 的因子逐个检测: \((A - \lambda I) = 0\)？若否，\((A-\lambda I)^2 = 0\)？确定重数。

🌐 实际应用

对角化在计算 \(A^n\)、\(e^A\)、解微分方程等场景中至关重要。

✏️ 例题分析与易错点

例题: \(A = \begin{pmatrix}2&1\\0&2\end{pmatrix}\)，判断可否对角化。 解答: \(\chi_A = (x-2)^2\)。 \(A - 2I = \begin{pmatrix}0&1\\0&0\end{pmatrix} \neq 0\)。 故 \(m_A = (x-2)^2\)，有重根，不可对角化。 易错点: 只用特征值不同来判断。即使特征值有重复， 只要 \(m_A\) 无重根就可对角化（如 \(A = \mathrm{diag}(2,2)\)）。

⚡ 快速检测
\(m_A(x) = (x-1)^2(x-3)\)，\(A\) 能对角化吗？
