双线性形式与二次型

01双线性形式的定义与矩阵表示

🎯 为什么研究这个问题

向量的长度、角度、面积等几何量都依赖于两个向量之间的「配对」运算。 双线性形式 \(B(u,v)\) 是内积的推广——它对两个变量分别线性， 但不一定正定也不一定对称。理解它是掌握二次型、内积、辛形式的基础。

🔍 抽象化方法

双线性形式是 \(V \times V \to K\) 的双线性映射，等价于 \(V \to V^*\) 的线性映射。 这是对偶空间与线性形式理论的自然延伸。

📐 公式推导 / 模型建立

定义: \(B: V \times V \to K\) 满足对两个变量分别线性。 矩阵表示: 选基 \(\{e_1, \ldots, e_n\}\)，令 \(a_{ij} = B(e_i, e_j)\)。 则 \(B(u, v) = u^T A v\)，其中 \(A = (a_{ij})\)。 分类: - 对称: \(B(u,v) = B(v,u)\)，即 \(A = A^T\) - 反对称: \(B(u,v) = -B(v,u)\)，即 \(A = -A^T\) - 非退化: \(\ker(B) = 0\)，即 \(A\) 可逆 \[B(u,v) = u^T A v, \quad \ker(B) = \{v : B(v,w) = 0, \forall w\}\]

🌐 实际应用

辛几何中的辛形式（反对称非退化双线性形式）是哈密顿力学的数学基础。

✏️ 例题分析与易错点

例题: \(V = \mathbb{R}^2\)，\(B(u,v) = u_1 v_2 - u_2 v_1\)。判断类型。 解答: \(A = \begin{pmatrix}0&1\\-1&0\end{pmatrix}\)。 \(A^T = -A\)，故反对称。\(\det A = 1 \neq 0\)，非退化。 这是标准辛形式。 易错点: 混淆「对称双线性形式」和「内积」。 内积还需要正定性，对称双线性形式不需要。

⚡ 快速检测
\(B(u,v) = u^T A v\) 非退化意味着？

02二次型与标准化

🎯 为什么研究这个问题

二次型 \(Q(v) = B(v,v)\) 在几何中描述曲面（椭圆、双曲面等）， 在物理中描述能量。将二次型化为标准形（只有平方项）大大简化分析。 配方法和合同变换是实现标准化的核心工具。

📐 公式推导 / 模型建立

二次型与双线性形式的关系: \(Q(v) = B(v,v)\)。 反过来（\(\mathrm{char} K \neq 2\)）: \[B(u,v) = \frac{1}{2}[Q(u+v) - Q(u) - Q(v)]\] 这是极化恒等式。 标准化: 存在基使 \(Q = a_1 x_1^2 + \cdots + a_n x_n^2\)。 方法: - 配方法（逐项消去交叉项） - 矩阵: 找可逆 \(P\) 使 \(P^T A P = D\)（合同对角化） 合同变换保持秩和符号（实数域上）。

🌐 实际应用

曲面分类: \(ax^2 + bxy + cy^2 = 1\) 化为标准形后判断椭圆/双曲线。

✏️ 例题分析与易错点

例题: \(Q(x,y) = x^2 + 4xy + 3y^2\)。化为标准形。 解答: 配方: \(Q = (x+2y)^2 + 3y^2 - 4y^2 = (x+2y)^2 - y^2\)。 令 \(u = x+2y, v = y\)，则 \(Q = u^2 - v^2\)。 标准形为 \(u^2 - v^2\)（符号 \((1, -1)\)）。 易错点: 配方后忘记换元，写成 \((x+2y)^2 - y^2\) 却仍称为「标准形」。 标准形要求变量之间无交叉，应写新变量。

⚡ 快速检测

03Sylvester 惯性定理与正定性

🎯 为什么研究这个问题

配方法给出的标准形取决于配方顺序，不同方法得到的系数不同。 但 Sylvester 惯性定理保证正系数的个数和负系数的个数是不变量。 正定性（所有系数为正）在优化、统计中意味着极小值的存在。

🔍 抽象化方法

惯性定理说明实二次型的合同不变量只有 \((p, q)\)——正负惯性指数。 这是合同等价类的完全分类。

📐 公式推导 / 模型建立

Sylvester 惯性定理: 实对称矩阵 \(A\) 的任何合同对角化 \(P^T A P = \mathrm{diag}(d_1, \ldots, d_n)\) 中， 正 \(d_i\) 的个数 \(p\) 和负 \(d_i\) 的个数 \(q\) 不变。\((p, q)\) 称为符号。 证明思路: 假设两种标准化给出不同的 \((p,q)\)， 则存在子空间 \(W^+\)（\(Q > 0\)）和 \(W^-\)（\(Q \leq 0\)）， \(\dim W^+ + \dim W^- > n\)，故 \(W^+ \cap W^- \neq 0\)， 矛盾。 \[\text{符号 } (p, q) \text{ 是合同不变量, } p + q = \mathrm{rank}(A)\] 正定: \(p = n, q = 0\)。等价条件: 1) 所有特征值 > 0 2) 所有顺序主子式 > 0 3) \(Q(v) > 0\)，\(\forall v \neq 0\)

🌐 实际应用

机器学习中 Hessian 矩阵的正定性判定损失函数的局部极小。

✏️ 例题分析与易错点

例题: \(A = \begin{pmatrix}2&1\\1&3\end{pmatrix}\)。判断正定性。 解答: 顺序主子式: \(\Delta_1 = 2 > 0\)，\(\Delta_2 = 6-1 = 5 > 0\)。 所有顺序主子式为正，\(A\) 正定。 或特征值: \(\lambda = \frac{5 \pm \sqrt{5}}{2}\)，均为正。 易错点: 用「所有主子式 > 0」代替「顺序主子式 > 0」。 正定只需检查顺序主子式（左上角子矩阵的行列式）。

⚡ 快速检测
符号 \((2, 1)\) 的二次型是？

04内积空间与 Gram-Schmidt 正交化

🎯 为什么研究这个问题

内积把「长度」和「角度」引入向量空间。Gram-Schmidt 过程将任意基 转化为正交基，是数值计算（QR 分解）和理论分析的基石。 它在傅里叶分析、最小二乘等领域无处不在。

📐 公式推导 / 模型建立

内积: \(\langle \cdot, \cdot \rangle: V \times V \to K\) 满足 对称（共轭对称）、线性、正定。 Gram-Schmidt 过程: 给定 \(\{v_1, \ldots, v_n\}\)， \[u_1 = v_1, \quad u_k = v_k - \sum_{j=1}^{k-1} \frac{\langle v_k, u_j \rangle}{\langle u_j, u_j \rangle} u_j\] \(\{u_1, \ldots, u_n\}\) 两两正交。归一化得标准正交基。 QR 分解: \(A = QR\)，\(Q\) 正交，\(R\) 上三角。 本质就是对 \(A\) 的列向量做 Gram-Schmidt。

🌐 实际应用

最小二乘法: \(\hat{x} = (A^T A)^{-1} A^T b\) 的推导用到正交投影。

✏️ 例题分析与易错点

例题: 对 \(v_1 = (1,1,0)\)，\(v_2 = (1,0,1)\) 做 Gram-Schmidt。 解答: \(u_1 = (1,1,0)\)。 \(\langle v_2, u_1 \rangle = 1\)，\(\langle u_1, u_1 \rangle = 2\)。 \(u_2 = (1,0,1) - \frac{1}{2}(1,1,0) = (\frac{1}{2}, -\frac{1}{2}, 1)\)。 验证: \(\langle u_1, u_2 \rangle = \frac{1}{2} - \frac{1}{2} = 0\)。✓ 易错点: Gram-Schmidt 产生的向量只是正交，未必标准正交。 需额外除以 \(\|u_k\|\) 进行归一化。
