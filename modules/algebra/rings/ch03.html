多项式环

01多项式环的基本性质

🎯 为什么研究这个问题

\(\mathbb{Z}\) 和 \(K[x]\) 是环论中最重要的两个原型。\(K[x]\) 的带余除法、唯一因子 分解等性质与 \(\mathbb{Z}\) 几乎完全平行。事实上，\(K[x]\) 是欧几里得整环（ED）， 因此也是主理想整环（PID）和唯一分解整环（UFD）。

🔍 抽象化方法

多项式环是「自由交换代数」\(R[x]\) 的原型——用不定元的幂生成。

📐 公式推导 / 模型建立

多项式环 \(R[x]\): 元素为 \(\sum_{i=0}^n a_i x^i\)，\(a_i \in R\)。 \(\deg(fg) = \deg f + \deg g\)（当 \(R\) 是整环时）。 带余除法: 若 \(K\) 是域，\(f, g \in K[x], g \neq 0\)，则 \[f = qg + r, \quad \deg r < \deg g\] \(q, r\) 唯一。 推论: \(K[x]\) 是欧几里得整环（用次数作欧族函数）， \(\Rightarrow\) PID \(\Rightarrow\) UFD。 \(K[x]\) 中的理想都是主理想 \((f(x))\)。

🌐 实际应用

编码理论: 循环码基于 \(\mathbb{F}_q[x]/(g(x))\) 的理想结构。

✏️ 例题分析与易错点

例题: \(\mathbb{Q}[x]\) 中带余除: \(f = x^3 + 1\)，\(g = x + 1\)。 解答: \(x^3 + 1 = (x^2 - x + 1)(x+1) + 0\)。 \(q = x^2 - x + 1\)，\(r = 0\)。故 \((x+1) \mid (x^3+1)\)。 易错点: 带余除法要求首系数可逆。 \(\mathbb{Z}[x]\) 中带余除法不一定成立（如 \(x\) 除 \(2x-1\)）。

⚡ 快速检测
\(K[x]\) 中的理想都是？

02不可约性判定

🎯 为什么研究这个问题

判断多项式是否可因式分解是代数的基本问题。Eisenstein 判别法和模 \(p\) 归约是判断 \(\mathbb{Z}[x]\)（从而 \(\mathbb{Q}[x]\)）中不可约性的主要工具， 结合 Gauss 引理，完整解决了有理数域上的因式分解问题。

🔍 抽象化方法

Gauss 引理建立了 \(\mathbb{Z}[x]\) 和 \(\mathbb{Q}[x]\) 之间的桥梁: 整数系数上可约 \(\Leftrightarrow\) 有理系数上可约。

📐 公式推导 / 模型建立

Gauss 引理: \(f \in \mathbb{Z}[x]\) 本原，若 \(f = gh\) 在 \(\mathbb{Q}[x]\) 中，则可取 \(g, h \in \mathbb{Z}[x]\)（调整系数）。 Eisenstein 判别法: \(f = a_n x^n + \cdots + a_0 \in \mathbb{Z}[x]\)， 若存在素数 \(p\) 使: 1) \(p \nmid a_n\) 2) \(p \mid a_i\)，\(0 \leq i < n\) 3) \(p^2 \nmid a_0\) 则 \(f\) 在 \(\mathbb{Q}[x]\) 中不可约。 \[\text{Eisenstein: } p \nmid a_n,\; p \mid a_i (i

🌐 实际应用

\(x^{p-1} + x^{p-2} + \cdots + 1\) 的不可约性（代换 \(x \to x+1\) 后用 Eisenstein）。

✏️ 例题分析与易错点

例题: 证明 \(f = x^4 + 10x + 5\) 在 \(\mathbb{Q}[x]\) 上不可约。 解答: 取 \(p = 5\)。\(a_4 = 1\)，\(5 \nmid 1\)。 \(a_3 = 0, a_2 = 0, a_1 = 10, a_0 = 5\)，皆被 5 整除。 \(25 \nmid 5\)。满足 Eisenstein，故不可约。 易错点: Eisenstein 只发现不可约，不保证发现。 不满足 Eisenstein 不意味着可约。

⚡ 快速检测

03主理想整环与唯一分解整环

🎯 为什么研究这个问题

整数的唯一分解定理并非对所有环成立。主理想整环（PID）和唯一分解整环（UFD） 精确刻画了「什么时候有唯一分解」。理解它们之间的包含关系 （ED \(\Rightarrow\) PID \(\Rightarrow\) UFD）是代数数论的基础。

🔍 抽象化方法

ED/PID/UFD 是三个递减弱的「整数性质」抽象。 它们描述了环中「因式分解」行为的好坏程度。

📐 公式推导 / 模型建立

ED（欧几里得整环）: 存在欧族函数 \(d: R \setminus \{0\} \to \mathbb{N}\)， 使得带余除法成立。 例: \(\mathbb{Z}\)（\(d = |\cdot|\)），\(K[x]\)（\(d = \deg\)），\(\mathbb{Z}[i]\)（\(d = |\cdot|^2\)）。 PID: 每个理想由单元素生成: \(I = (a)\)。 UFD: 每个非零非可逆元能唯一分解为不可约元的乘积。 \[\text{ED} \Rightarrow \text{PID} \Rightarrow \text{UFD}\] 反例: \(\mathbb{Z}[\frac{1+\sqrt{-19}}{2}]\) 是 PID 但不是 ED。

🌐 实际应用

代数数论: \(\mathbb{Z}[\sqrt{-5}]\) 不是 UFD (\(6 = 2 \cdot 3 = (1+\sqrt{-5})(1-\sqrt{-5})\))， 这促使了理想的发明（Kummer/Dedekind）。

✏️ 例题分析与易错点

例题: 证 \(\mathbb{Z}[\sqrt{-5}]\) 不是 UFD。 解答: \(6 = 2 \times 3 = (1+\sqrt{-5})(1-\sqrt{-5})\)。 用范数 \(N(a+b\sqrt{-5}) = a^2 + 5b^2\) 证明 2, 3, \(1\pm\sqrt{-5}\) 均不可约（无元素范数为 2 或 3）。 两种分解不为相伴，唯一性失败。 易错点: 以为 \(\mathbb{Z}[\sqrt{d}]\) 对所有 \(d\) 都是 UFD。 实际上只有9 个负 \(d\) 值使 \(\mathbb{Z}[\sqrt{d}]\)（或其整数环）是 PID。

04多元多项式环与 Hilbert 基定理

🎯 为什么研究这个问题

多元多项式环 \(K[x_1, \ldots, x_n]\) 不再是 PID（当 \(n \geq 2\)）， 但仍有优良性质。Hilbert 基定理保证它是 Noetherian 环， 即每个理想都是有限生成的。这是代数几何和交换代数的基石。

🔍 抽象化方法

Noetherian 性质是「有限性」的代数抽象: 无穷上升理想链必定终止。

📐 公式推导 / 模型建立

Hilbert 基定理: 若 \(R\) 是 Noetherian 环，则 \(R[x]\) 也是。 推论: \(K[x_1, \ldots, x_n]\) 是 Noetherian（归纳）。 证明思路: 设 \(I \subseteq R[x]\) 是理想。 定义首系数理想 \(L_d = \{\text{I 中次数 } \leq d \text{ 的多项式的首系数}\} \cup \{0\}\)。 \(L_0 \subseteq L_1 \subseteq \cdots\) 是 \(R\) 中的理想上升链。 \(R\) 是 Noetherian \(\Rightarrow\) 链终止。从而构造 \(I\) 的有限生成元。 \[R \text{ Noetherian} \Rightarrow R[x] \text{ Noetherian}\]

🌐 实际应用

代数几何: 多项式理想描述代数簇，Noetherian 性质保证了 代数簇可用有限多个方程定义。

✏️ 例题分析与易错点

例题: \(K[x, y]\) 中 \((x, y)\) 是主理想吗？ 解答: 若 \((x, y) = (f)\)，则 \(f \mid x\) 且 \(f \mid y\)。 \(f \mid x \Rightarrow \deg_x f \leq 1, \deg_y f = 0\)，即 \(f = ax + b\)。 \(f \mid y \Rightarrow (ax+b) \mid y\)，不可能（除非 \(f\) 是常数）。 故 \((x, y) \neq (f)\)，不是主理想。\(K[x,y]\) 不是 PID。 易错点: 以为多元多项式环也是 PID。 \(n \geq 2\) 时 \(K[x_1,\ldots,x_n]\) 是 UFD 但不是 PID。
