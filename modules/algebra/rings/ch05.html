01Galois 扩张与 Galois 群

🎯 为什么研究这个问题

Galois 的天才洞察: 方程的可解性不取决于方程本身，而取决于其根的 「对称性」——即 Galois 群。\(\mathrm{Gal}(L/K)\) 是 \(L\) 中保持 \(K\) 不变的 自同构组成的群，它将域扩张的结构编码为群的结构。

🔍 抽象化方法

Galois 理论是「对称性决定结构」思想的典范: 域扩张的性质完全由其对称群决定。

📐 公式推导 / 模型建立

Galois 群: \(\mathrm{Gal}(L/K) = \{\sigma \in \mathrm{Aut}(L) : \sigma|_K = \mathrm{id}\}\)。 元素是 \(L\) 的自同构，固定 \(K\) 的每个元素。 Galois 扩张: \(L/K\) 满足 1) \(L/K\) 有限 2) \(L/K\) 可分 3) \(L/K\) 正规（\(\forall \sigma \in \mathrm{Aut}(\bar{K}/K)\)，\(\sigma(L) = L\)） 等价: \(L\) 是某个可分多项式的分裂域。 关键等式: \[|\mathrm{Gal}(L/K)| = [L:K]\]

✏️ 例题分析与易错点

例题: \(\mathrm{Gal}(\mathbb{Q}(\sqrt{2})/\mathbb{Q})\)。 解答: \([\mathbb{Q}(\sqrt{2}):\mathbb{Q}] = 2\)，故 \(|\mathrm{Gal}| = 2\)。 \(\sigma: \sqrt{2} \mapsto -\sqrt{2}\)（唯一非平凡自同构）。 \(\mathrm{Gal} \cong \mathbb{Z}/2\mathbb{Z}\)。 易错点: 以为 \(\mathrm{Gal}(\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}) \cong \mathbb{Z}/3\mathbb{Z}\)。 \(\mathbb{Q}(\sqrt[3]{2}) \subset \mathbb{R}\)，自同构保持序，故只有恒等。 \(\mathbb{Q}(\sqrt[3]{2})/\mathbb{Q}\) 不是 Galois 扩张（不正规）。

02Galois 基本定理

🎯 为什么研究这个问题

Galois 基本定理是数学中最美丽的对应之一: 它建立了中间域和子群之间 的一一对应，将域扩张问题完全转化为群论问题。 这是「用对称性理解结构」的极致体现。

🔍 抽象化方法

中间域格 \(\longleftrightarrow\) 子群格，反序对应。 这是 Galois 理论的核心思想。

📐 公式推导 / 模型建立

Galois 基本定理: 设 \(L/K\) 是 Galois 扩张，\(G = \mathrm{Gal}(L/K)\)。 则存在反序双射: \[\{K \subseteq M \subseteq L\} \longleftrightarrow \{H \leq G\}\] \(M \mapsto H = \mathrm{Gal}(L/M)\)，\(H \mapsto M = L^H = \{\alpha : \sigma(\alpha) = \alpha, \forall \sigma \in H\}\)。 细节: \([M:K] = [G:H]\)，\([L:M] = |H|\)。 \(M/K\) 正规 \(\Leftrightarrow\) \(H \trianglelefteq G\)，此时 \(\mathrm{Gal}(M/K) \cong G/H\)。

✏️ 例题分析与易错点

例题: \(L = \mathbb{Q}(\sqrt{2}, \sqrt{3})\)，求所有中间域。 解答: \(G = \mathrm{Gal}(L/\mathbb{Q}) \cong \mathbb{Z}/2\mathbb{Z} \times \mathbb{Z}/2\mathbb{Z}\)。 \(|G| = 4\)。子群: \(\{e\}\)，3 个阶 2 子群，\(G\)。 对应中间域: \(L\)，\(\mathbb{Q}(\sqrt{2}), \mathbb{Q}(\sqrt{3}), \mathbb{Q}(\sqrt{6})\)，\(\mathbb{Q}\)。 易错点: 忘记对应是反序的。 较大的子群对应较小的域。

⚡ 快速检测
若 \(H \trianglelefteq G\)，则对应的中间域 \(M = L^H\) 满足？

03五次方程不可解性

🎯 为什么研究这个问题

二、三、四次方程都有根式解，但 Abel 和 Galois 证明了一般五次方程 无法用根式解。这是 Galois 理论最著名的应用: 将根式可解性 翻译为群的可解性，而 \(S_5\) 不可解。

🔍 抽象化方法

「根式可解」\(\Leftrightarrow\) Galois 群可解。 \(S_n\)（\(n \geq 5\)）不可解（因 \(A_n\) 单纯）。

📐 公式推导 / 模型建立

根式可解: 方程的解可用 \(+, -, \times, \div, \sqrt[n]{\cdot}\) 表示。 等价: 分裂域可通过一系列循环扩张到达。 等价: Galois 群可解（存在合成列使商群均为循环群）。 \(A_5\) 单纯: \(A_5\) 是最小的非交换单纯群（\(|A_5| = 60\)）。 \(S_5\) 的合成列: \(S_5 \supset A_5 \supset \{e\}\)，\(A_5/\{e\} = A_5\) 不交换。 \[S_5 \text{ 不可解} \Rightarrow \text{一般五次方程不可根式解}\]

🌐 实际应用

具体多项式 \(x^5 - 4x + 2\) 的 Galois 群为 \(S_5\)，不可解。

✏️ 例题分析与易错点

例题: \(x^4 - 2\) 的 Galois 群是什么？可根式解吗？ 解答: Galois 群 \(\cong D_4\)（二面体群，阶 8）。 \(D_4\) 可解（合成列 \(D_4 \supset \mathbb{Z}/4 \supset \mathbb{Z}/2 \supset \{e\}\)）。 故可根式解，解为 \(\sqrt[4]{2}, i\sqrt[4]{2}\) 等。 易错点: 以为「五次方程无解」。 是无根式解，不是无解。五次方程在 \(\mathbb{C}\) 上总有 5 个根。

04Galois 理论的计算实例

🎯 为什么研究这个问题

推导定理后还要会算。对具体多项式计算 Galois 群、找中间域、判断可解性是 代数考试和研究中的核心技能。

📐 公式推导 / 模型建立

计算策略: 1) 确定分裂域 \(L\) 和 \([L:K]\) 2) 找所有 \(K\)-自同构（通过根的置换） 3) 验证 \(|\mathrm{Gal}| = [L:K]\) 4) 识别群结构 二次扩张 \(K(\sqrt{a})\): \(\mathrm{Gal} \cong \mathbb{Z}/2\mathbb{Z}\)， \(\sigma(\sqrt{a}) = -\sqrt{a}\)。 分圆域: \(\Phi_n(x)\) 的分裂域 \(\mathbb{Q}(\zeta_n)\)， \(\mathrm{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^\times\)。 \[\mathrm{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q}) \cong (\mathbb{Z}/n\mathbb{Z})^\times\]

🌐 实际应用

正 \(n\) 边形可尺规作图 \(\Leftrightarrow\) \(n = 2^a p_1 \cdots p_k\)（\(p_i\) 两两不同的费马素数）。

✏️ 例题分析与易错点

例题: 计算 \(\mathrm{Gal}(\mathbb{Q}(\zeta_7)/\mathbb{Q})\)。 解答: \([\mathbb{Q}(\zeta_7):\mathbb{Q}] = \varphi(7) = 6\)。 \(\mathrm{Gal} \cong (\mathbb{Z}/7\mathbb{Z})^\times \cong \mathbb{Z}/6\mathbb{Z}\)（循环群）。 生成元 \(\sigma: \zeta_7 \mapsto \zeta_7^3\)（\(3\) 是模 7 的原根）。 易错点: 将 \(\mathrm{Gal}(\mathbb{Q}(\zeta_n)/\mathbb{Q})\) 与 \(\mathbb{Z}/n\mathbb{Z}\) 混淆。 应是 \((\mathbb{Z}/n\mathbb{Z})^\times\)（乘法群），不是 \(\mathbb{Z}/n\mathbb{Z}\)（加法群）。
