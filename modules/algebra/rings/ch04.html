域扩张

01域扩张的基本概念

🎯 为什么研究这个问题

\(\mathbb{Q}\) 中无法解 \(x^2 - 2 = 0\)，但可以「扩充」得到 \(\mathbb{Q}(\sqrt{2})\)。 域扩张的理论精确描述了「添加新元素后域如何变大」， 是 Galois 理论的基础，也是理解「为什么五次方程无根式解」的关键。

🔍 抽象化方法

域扩张是「向域中添加新元素」的代数框架。 它将方程求解问题转化为理想和商环的结构问题。

📐 公式推导 / 模型建立

域扩张: \(K \subseteq L\) 为域的包含关系，记 \(L/K\)。 \(L\) 是 \(K\)-向量空间，\([L:K] = \dim_K L\) 称为扩张次数。 单扩张: \(K(\alpha) = K[\alpha] = K[x]/(m_\alpha(x))\)， 其中 \(m_\alpha\) 是 \(\alpha\) 在 \(K\) 上的最小多项式。 \[[K(\alpha):K] = \deg m_\alpha\] 塔定理: \([M:K] = [M:L][L:K]\)。

🌐 实际应用

尺规作图不可能性: 倽立方不可能因为 \([\mathbb{Q}(\sqrt[3]{2}):\mathbb{Q}] = 3\)， 而尺规作图只能构造次数为 2 的幂的扩张。

✏️ 例题分析与易错点

例题: 求 \([\mathbb{Q}(\sqrt{2}, \sqrt{3}):\mathbb{Q}]\)。 解答: \([\mathbb{Q}(\sqrt{2}):\mathbb{Q}] = 2\)（\(m_{\sqrt{2}} = x^2-2\)）。 \(\sqrt{3} \notin \mathbb{Q}(\sqrt{2})\)（可证），故 \([\mathbb{Q}(\sqrt{2}, \sqrt{3}):\mathbb{Q}(\sqrt{2})] = 2\)。 由塔定理: \([\mathbb{Q}(\sqrt{2}, \sqrt{3}):\mathbb{Q}] = 4\)。 基: \(\{1, \sqrt{2}, \sqrt{3}, \sqrt{6}\}\)。 易错点: 以为 \([\mathbb{Q}(\sqrt{2}, \sqrt{3}):\mathbb{Q}] = 2 + 2 = 4\)。 结果恰好正确，但推理错误。塔定理是乘法不是加法。

⚡ 快速检测
\([\mathbb{Q}(\sqrt[3]{5}):\mathbb{Q}] = ?\)

02代数元与超越元

🎯 为什么研究这个问题

\(\sqrt{2}\) 是 \(x^2-2\) 的根——它是代数元。\(\pi\) 和 \(e\) 不满足任何 有理系数多项式——它们是超越元。区分这两类元素是数论和域扩张 理论的基本问题，类似于区分「精确」和「混沌」。

🔍 抽象化方法

代数元 \(\Leftrightarrow K(\alpha)/K\) 有限 \(\Leftrightarrow K[\alpha]\) 已是域。 超越元 \(\Leftrightarrow K(\alpha) \cong K(x)\)（纯超越扩张）。

📐 公式推导 / 模型建立

代数元: \(\alpha\) 在 \(K\) 上代数 \(\Leftrightarrow\) \(\exists f \in K[x], f \neq 0: f(\alpha) = 0\)。 最小多项式 \(m_\alpha\) 不可约，\(K(\alpha) \cong K[x]/(m_\alpha)\)。 \([K(\alpha):K] = \deg m_\alpha < \infty\)。 超越元: \(\alpha\) 在 \(K\) 上超越 \(\Leftrightarrow\) 无非零多项式零化它。 \(K(\alpha) \cong K(x)\)（有理函数域），\([K(\alpha):K] = \infty\)。 代数闭包: 代数元的和、积、商仍是代数元。 \[\bar{\mathbb{Q}} = \{\alpha \in \mathbb{C} : \alpha \text{ 在 } \mathbb{Q} \text{ 上代数}\}\]

🌐 实际应用

Lindemann-Weierstrass 定理: \(e^\alpha\) 对代数数 \(\alpha \neq 0\) 是超越的。 推论: \(\pi\) 是超越的（因 \(e^{i\pi} = -1\)，若 \(\pi\) 代数则 \(e^{i\pi}\) 超越）。

✏️ 例题分析与易错点

例题: \(\alpha = \sqrt{2} + \sqrt{3}\)。求 \(m_\alpha\) over \(\mathbb{Q}\)。 解答: \(\alpha - \sqrt{2} = \sqrt{3}\)，\((\alpha - \sqrt{2})^2 = 3\)， \(\alpha^2 - 2\sqrt{2}\alpha + 2 = 3\)，\(\alpha^2 - 1 = 2\sqrt{2}\alpha\)， \((\alpha^2 - 1)^2 = 8\alpha^2\)，\(\alpha^4 - 2\alpha^2 + 1 = 8\alpha^2\)， \(m_\alpha = x^4 - 10x^2 + 1\)。 易错点: 以为 \((x^2-2)(x^2-3)\) 是最小多项式。 \(\sqrt{2}+\sqrt{3}\) 不是它的根。最小多项式必须不可约。

⚡ 快速检测
\([\mathbb{Q}(\pi):\mathbb{Q}] = ?\)

03分裂域与代数闭包

🎯 为什么研究这个问题

单扩张 \(K(\alpha)\) 只添加了一个根。但很多时候我们需要一个多项式的 所有根都在其中。分裂域是包含给定多项式所有根的最小域扩张， 是 Galois 理论的核心对象。

🔍 抽象化方法

分裂域是使多项式「完全分解」的最小环境。 不同于单扩张（只加一个根），分裂域加入了所有根。

📐 公式推导 / 模型建立

分裂域: \(f \in K[x]\) 的分裂域是最小的域 \(L \supseteq K\)， 使 \(f\) 在 \(L[x]\) 中完全分解为一次因子: \[f(x) = c(x - \alpha_1) \cdots (x - \alpha_n) \in L[x]\] \(L = K(\alpha_1, \ldots, \alpha_n)\)。 存在性与唯一性: 分裂域总存在，且在同构意义下唯一。 代数闭包: \(\bar{K}\) 是所有 \(K[x]\) 中多项式的分裂域的并。 \(\bar{K}\) 上每个非常多项式都有根。

✏️ 例题分析与易错点

例题: 求 \(x^4 - 2\) 在 \(\mathbb{Q}\) 上的分裂域。 解答: 根: \(\sqrt[4]{2}, -\sqrt[4]{2}, i\sqrt[4]{2}, -i\sqrt[4]{2}\)。 分裂域 \(= \mathbb{Q}(\sqrt[4]{2}, i)\)。 \([\mathbb{Q}(\sqrt[4]{2}):\mathbb{Q}] = 4\)，\([\mathbb{Q}(\sqrt[4]{2}, i):\mathbb{Q}(\sqrt[4]{2})] = 2\)。 \([\mathbb{Q}(\sqrt[4]{2}, i):\mathbb{Q}] = 8\)。 易错点: 以为 \(\mathbb{Q}(\sqrt[4]{2})\) 就是分裂域。 \(\mathbb{Q}(\sqrt[4]{2}) \subset \mathbb{R}\)，不含复数根。

⚡ 快速检测

04可分扩张与完美域

🎯 为什么研究这个问题

并非所有域扩张都行为良好。「可分」扩张是 Galois 理论成立的条件。 在特征 0 域（如 \(\mathbb{Q}, \mathbb{R}\)）中，每个代数扩张都是可分的。 但在特征 \(p\) 域中可能出现不可分性，需要完美域的概念。

🔍 抽象化方法

可分性确保了最小多项式无重根——这是 Galois 对应的前提。 完美域是「自动可分」的域。

📐 公式推导 / 模型建立

可分元: \(\alpha\) 在 \(K\) 上可分 \(\Leftrightarrow\) \(m_\alpha\) 无重根。 等价: \(\gcd(m_\alpha, m_\alpha') = 1\)。 可分扩张: \(L/K\) 可分 \(\Leftrightarrow\) \(L\) 中每个元素在 \(K\) 上可分。 等价: \(L\) 是某个可分多项式的分裂域。 完美域: 每个不可约多项式都可分。 包括: 特征 0 域、有限域。 \[\mathrm{char}(K) = 0 \text{ 或 } K \text{ 有限} \Rightarrow K \text{ 完美}\]

✏️ 例题分析与易错点

例题: \(\mathbb{F}_p(t)\) 中 \(x^p - t\) 的根是否可分？ 解答: \(x^p - t\) 在 \(\mathbb{F}_p(t)\) 上不可约。 设 \(\alpha^p = t\)，则 \(x^p - t = (x - \alpha)^p\)（特征 \(p\)）。 只有一个根 \(\alpha\)（重数 \(p\)），不可分。 \(\mathbb{F}_p(t)\) 不是完美域。 易错点: 以为所有域都是完美的。 \(\mathbb{F}_p(t)\)（无限域，特征 \(p\)）是不完美域的典型例子。

⚡ 快速检测
\(\mathbb{Q}\) 是完美域吗？
