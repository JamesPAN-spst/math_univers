01万能逼近定理

🎯 为什么研究这个问题

神经网络为什么能拟合如此复杂的函数？万能逼近定理(UAT)给出理论 保证：只要有足够多的神经元，单隐层网络即可逼近任意连续函数到 任意精度。这是神经网络表达能力的理论基础，但它没有告诉我们需要 多少神经元——深度在效率上远优于宽度。

🔍 抽象化方法

UAT 本质上是一个逼近论结果：神经网络作为函数类在 \(C(K)\) 中是 稠密的（在紧集上一致逼近）。

📐 公式推导 / 模型建立

Cybenko 定理(1989)：设 \(\sigma\) 是 sigmoid 类激活函数。 对任意 \(f \in C([0,1]^d)\) 和 \(\epsilon > 0\)，存在 \[g(x) = \sum_{j=1}^N \alpha_j \sigma(w_j^T x + b_j)\] 使 \(\|g-f\|_\infty < \epsilon\)。 Hornik 定理(1991)：对任何非多项式连续激活函数（包括 ReLU 的光滑近似）， 单隐层网络在 \(L^p\) 意义上稠密。 深度的优势：存在函数，\(L\) 层网络只需 \(O(d)\) 个神经元表示，但 单隐层网络需要 \(O(2^d)\) 个——指数级差距。例如锯齿函数的递归构造： 每层 ReLU 可将"折叠数"翻倍。

🌐 实际应用

UAT 解释了为什么 MLP 可以拟合任何数据，但实际中深度网络 （ResNet, Transformer）远比浅宽网络高效。

✏️ 例题分析与易错点

例题：用 2 个 ReLU 神经元逼近 \(f(x) = |x|\)（\(x \in [-1,1]\)）。 解答：\(\mathrm{ReLU}(x) = \max(0,x)\)。 \(|x| = \mathrm{ReLU}(x) + \mathrm{ReLU}(-x)\)。只需 2 个神经元精确表示。 易错点：UAT 保证存在性但不保证可学习性——梯度下降不一定能找到这个解。

02反向传播算法

🎯 为什么研究这个问题

训练神经网络需要计算损失关于所有参数的梯度。对百万参数的网络 逐个计算不可行。反向传播(backpropagation)利用链式法则和计算图 结构，以 \(O(1)\) 倍前向传播的代价高效计算所有梯度——这使得深度 学习在计算上可行。

🔍 抽象化方法

反向传播 = 计算图上的自动微分(AD)。前向传播计算函数值， 反向传播计算梯度，时间复杂度相同（常数因子差异）。

📐 公式推导 / 模型建立

计算图：设 \(z^{(l)} = W^{(l)}a^{(l-1)}+b^{(l)}\), \(a^{(l)} = \sigma(z^{(l)})\)。 反向传播的核心——误差信号 \(\delta^{(l)}\)： \[\delta^{(L)} = \nabla_{a^{(L)}} \ell \odot \sigma'(z^{(L)})\] \[\delta^{(l)} = (W^{(l+1)})^T \delta^{(l+1)} \odot \sigma'(z^{(l)})\] 梯度公式： \[\frac{\partial \ell}{\partial W^{(l)}} = \delta^{(l)} (a^{(l-1)})^T, \quad \frac{\partial \ell}{\partial b^{(l)}} = \delta^{(l)}\] 从输出层反向逐层计算，每个参数只需 \(O(1)\) 额外操作。

🌐 实际应用

PyTorch 的 autograd、TensorFlow 的 GradientTape 都是反向传播的工程实现。

✏️ 例题分析与易错点

例题：网络 \(f = \sigma(w_2 \cdot \sigma(w_1 x))\), \(\ell = (f-y)^2\)。 求 \(\partial\ell/\partial w_1\)。 解答：\(\partial\ell/\partial f = 2(f-y)\)。\(\partial f/\partial a_1 = w_2 \sigma'(w_2 a_1)\)。 \(\partial a_1/\partial w_1 = x \sigma'(w_1 x)\)。链式法则组合即可。 易错点：梯度消失——若 \(|\sigma'| < 1\) 且网络深，\(\delta^{(l)}\) 指数衰减。 ReLU (\(\sigma'=0\) 或 \(1\)) 缓解此问题。

03Neural Tangent Kernel (NTK)

🎯 为什么研究这个问题

过参数化神经网络（参数远多于数据量）为什么能训练到零训练误差？ NTK 理论给出了一个优雅的解释：在无穷宽极限下，神经网络的训练 动态等价于核回归——参数几乎不动（"lazy training"），核函数在 初始化时确定。这建立了核方法与深度学习之间的理论桥梁。

🔍 抽象化方法

NTK 将非线性的神经网络训练线性化：在参数空间中，参数变化很小 时网络近似线性，训练变为固定核的核回归。

📐 公式推导 / 模型建立

设网络 \(f_\theta(x)\), 参数 \(\theta \in \mathbb{R}^p\)。NTK 定义： \[\Theta_{NTK}(x,y) = \left\langle \frac{\partial f_\theta}{\partial \theta}(x),\, \frac{\partial f_\theta}{\partial \theta}(y) \right\rangle\] 梯度流动力学：\(\dot{\theta} = -\nabla_\theta \ell\)，推出 \[\frac{d}{dt} f_\theta(x_i) = -\sum_j \Theta_{NTK}(x_i, x_j)(f_\theta(x_j)-y_j)\] 无穷宽极限(Jacot 2018)：当宽度 \(\to \infty\) 时， (1) \(\Theta_{NTK}\) 在训练过程中保持不变 (2) 训练动态 = 关于固定核 \(\Theta_{NTK}\) 的核回归 (3) 若 \(\Theta_{NTK}\) 的最小特征值 \(>0\)，则线性收敛到零训练误差

🌐 实际应用

NTK 用于分析网络的泛化性、训练速度和初始化的影响。

✏️ 例题分析与易错点

例题：1D 网络 \(f(x) = \frac{1}{\sqrt{m}}\sum_{j=1}^m a_j \sigma(w_j x)\)，\(a_j\) 固定。 计算 NTK。解答：\(\partial f/\partial w_j = \frac{a_j}{\sqrt{m}} x \sigma'(w_j x)\)。 \(\Theta(x,y) = \frac{xy}{m}\sum_j a_j^2 \sigma'(w_j x)\sigma'(w_j y)\)。 \(m \to \infty\) 时 \(\to xy\, \mathbb{E}_{w}[a^2 \sigma'(wx)\sigma'(wy)]\)（确定性核）。 易错点：NTK 理论描述的是无穷宽极限，实际有限宽网络可能偏离"lazy"动态。

04过参数化与隐式正则化

🎯 为什么研究这个问题

传统统计学认为参数多于数据必导致过拟合，但深度学习中过参数化 模型反而泛化良好——这就是"双下降"现象。隐式正则化理论解释了 梯度下降如何偏好"简单"解（最小范数解、平坦极小值），无需显 式正则化即可避免过拟合。

📐 公式推导 / 模型建立

双下降(Double Descent)：测试误差随模型复杂度呈现"U+再下降"： (1) 欠参数化区间 (\(p < n\))：经典 U 型偏差-方差权衡 (2) 插值阈值 (\(p \approx n\))：测试误差剧增 (3) 过参数化区间 (\(p \gg n\))：测试误差再次下降 隐式偏好：线性模型中，梯度下降从 \(w_0=0\) 出发收敛到 \[w^* = X^T(XX^T)^{-1}y = \arg\min \|w\| \text{ s.t. } Xw = y\] 即最小 \(\ell_2\) 范数的插值解。 平坦极小值假说：SGD 的噪声使训练偏好 Hessian 特征值小的极小值 （平坦），而平坦极小值对扰动鲁棒，泛化更好。

🌐 实际应用

理解过参数化指导了大模型（GPT、LLM）的训练实践： "越大越好"（scaling law）有理论支撑。

✏️ 例题分析与易错点

例题：\(n=3\) 数据点，\(p=10\) 参数线性模型。\(X \in \mathbb{R}^{3 \times 10}\)。 GD 从 \(w=0\) 出发的解是什么？ 解答：\(w^*=X^T(XX^T)^{-1}y\)，是满足 \(Xw=y\) 的最小范数解。 \(\|w^*\|\) 小意味着模型"简单"（系数小），有利于泛化。 易错点：双下降不意味着可以无限增大模型——需要足够的数据和合理的训练。
