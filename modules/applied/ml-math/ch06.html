01PAC 学习与 VC 维

🎯 为什么研究这个问题

机器学习算法在训练集上表现好就够了吗？学习理论回答一个根本问 题：为什么从有限样本学到的模型能泛化到未见数据？PAC 学习框架 和 VC 维理论给出了第一个严格答案：学习的可能性取决于假设类的 "复杂度"与样本量的关系。

🔍 抽象化方法

从"学习一个概念"到"以高概率近似正确"(PAC)——将学习问题转 化为概率语言下的有限样本统计问题。

📐 公式推导 / 模型建立

PAC 学习：假设类 \(\mathcal{H}\) 是 PAC 可学的 iff 存在算法 \(A\)， \(\forall \epsilon, \delta > 0\), \(\forall P\)，当样本量 \[n \geq n_0(\epsilon, \delta, \mathcal{H})\] 时，\(\Pr[\mathrm{err}(A(S)) \leq \min_{h \in \mathcal{H}} \mathrm{err}(h) + \epsilon] \geq 1-\delta\)。 有限假设类：\(|\mathcal{H}| < \infty\) 时，union bound 给出 \(n = O(\frac{1}{\epsilon^2}(\log|\mathcal{H}|+\log(1/\delta)))\)。 VC 维：\(\mathrm{VC}(\mathcal{H})\) = \(\mathcal{H}\) 能打散(shatter)的最大点集大小。 VC 不等式：\(\Pr[\sup_{h}|R(h)-\hat{R}(h)| > \epsilon] \leq 4 \cdot S_{\mathcal{H}}(2n) \cdot e^{-n\epsilon^2/8}\)， 其中增长函数 \(S_{\mathcal{H}}(n) \leq (en/d)^d\)（\(d = \mathrm{VC}(\mathcal{H})\)）。

🌐 实际应用

VC 维决定了模型需要多少数据才能泛化——SVM 最大间隔原理可以 理解为控制"有效 VC 维"。

✏️ 例题分析与易错点

例题：\(\mathbb{R}^2\) 中线性分类器的 VC 维。 解答：3 个不共线点可被所有 \(2^3=8\) 种标签方式打散（用直线分类）。 4 个点中存在 XOR 配置不可打散。故 \(\mathrm{VC} = 3 = d+1\)。 一般 \(\mathbb{R}^d\) 中半空间的 VC 维 \(= d+1\)。 易错点：VC 维是最大可打散集合的大小，存在一个能打散的集合即可 （不要求所有大小为 \(d\) 的集合都可打散）。

02Rademacher 复杂度与泛化界

🎯 为什么研究这个问题

VC 维是分布无关的——可能过于悲观。Rademacher 复杂度是数据依赖 的复杂度度量，给出更紧的泛化界。它衡量假设类拟合随机噪声的能 力：能拟合噪声越多则复杂度越高，泛化越差。

🔍 抽象化方法

Rademacher 复杂度量化了"函数类与随机噪声的相关性"。

📐 公式推导 / 模型建立

经验 Rademacher 复杂度： \[\hat{\mathcal{R}}_n(\mathcal{H}) = \mathbb{E}_\sigma\left[\sup_{h \in \mathcal{H}} \frac{1}{n}\sum_{i=1}^n \sigma_i h(x_i)\right]\] 其中 \(\sigma_i \in \{+1,-1\}\) 均匀随机（Rademacher 变量）。 泛化界（对称化技巧）： \[\mathbb{E}[\sup_h (R(h)-\hat{R}(h))] \leq 2\mathcal{R}_n(\mathcal{H})\] 高概率版本（McDiarmid）： \[\Pr[R(h) \leq \hat{R}(h) + 2\mathcal{R}_n + \sqrt{\frac{\log(1/\delta)}{2n}}] \geq 1-\delta\] 与 VC 维的关系：线性分类器的 \(\mathcal{R}_n = O(\sqrt{d/n})\)。

🌐 实际应用

Rademacher 复杂度用于分析 RKHS 中核方法的泛化性： \(\mathcal{R}_n \leq \frac{B}{\sqrt{n}}\sqrt{\mathrm{tr}(K)/n}\)。

✏️ 例题分析与易错点

例题：\(\mathcal{H} = \{h(x) = \mathrm{sign}(w^Tx) \mid \|w\| \leq 1\}\)， \(x_i\) 在单位球上。估计 \(\hat{\mathcal{R}}_n\)。 解答：\(\hat{\mathcal{R}}_n = \mathbb{E}_\sigma[\sup_{\|w\|\leq 1} \frac{1}{n}|w^T\sum\sigma_i x_i|] = \frac{1}{n}\mathbb{E}[\|\sum \sigma_i x_i\|] \leq \frac{1}{n}\sqrt{n} = 1/\sqrt{n}\)。 易错点：Rademacher 复杂度是数据依赖的——不同数据集上的值不同。

03偏差-方差分解的理论视角

🎯 为什么研究这个问题

偏差-方差分解(Ch.01 引入)在学习理论中有更深层的理论含义：它连 接了模型选择、正则化和泛化界。理解经验过程理论视角下的偏差-方 差权衡，有助于设计更好的学习算法和正则化策略。

🔍 关键概念抽象

偏差-方差分解在学习理论框架下获得了更精确的数学表述。超额风险 \(R(\hat{f}_n) - R(f^*)\) 被分解为两个来源完全不同的项：逼近误差（假设类的结构限制）和估计误差（有限样本的随机波动）。这一分解揭示了学习问题的根本张力。

逼近误差衡量假设类 \(\mathcal{H}\) 的表达能力——如果目标函数 \(f^*\) 不在 \(\mathcal{H}\) 中，即使拥有无穷数据也无法完美学习。它只取决于 \(\mathcal{H}\) 的选择，与样本量 \(n\) 无关。增大 \(\mathcal{H}\)（如使用更高次多项式或更深网络）可以减小逼近误差。

估计误差则由假设类的"复杂度"和样本量共同决定。Rademacher 复杂度 \(\mathcal{R}_n(\mathcal{H})\) 精确刻画了 \(\mathcal{H}\) 拟合随机噪声的能力：复杂度越高，从有限样本中均匀逼近真实风险就越困难。一致收敛定理提供了估计误差的非渐近上界。

模型选择的本质就是在逼近误差和估计误差之间寻找最优权衡点。交叉验证、结构风险最小化（SRM）和信息准则分别从经验、理论和信息论的角度解决这一权衡问题。

📐 公式推导 / 模型建立

形式化：设最优函数 \(f^*\), 假设类中最佳 \(f_{\mathcal{H}}^* = \arg\min_{f \in \mathcal{H}} R(f)\), 学习算法输出 \(\hat{f}_n\)。 分解： \[R(\hat{f}_n) - R(f^*) = \underbrace{[R(f_\mathcal{H}^*) - R(f^*)]}_{\text{逼近误差(偏差)}} + \underbrace{[R(\hat{f}_n) - R(f_\mathcal{H}^*)]}_{\text{估计误差(方差)}}\] 逼近误差：假设类 \(\mathcal{H}\) 不含 \(f^*\) 时的"结构偏差"， \(\mathcal{H}\) 越大则越小。 估计误差：有限样本带来的随机波动，由 Rademacher 复杂度控制： \[R(\hat{f}_n) - R(f_\mathcal{H}^*) \leq 2 \sup_{f \in \mathcal{H}} |R(f)-\hat{R}(f)| \lesssim \mathcal{R}_n(\mathcal{H})\] 权衡：\(\mathcal{H}\) 越大 \(\Rightarrow\) 逼近误差↓、估计误差↑。

🌐 实际应用

模型选择（如交叉验证）本质上是在平衡逼近误差与估计误差。

✏️ 例题分析与易错点

例题：多项式假设类 \(\mathcal{H}_d\)（\(d\) 阶多项式）, 数据由 \(d^*=3\) 阶 多项式生成。分析 \(d=1, 5, 20\) 时的偏差-方差权衡。 解答：\(d=1\): 逼近误差大（不含 3 阶项），估计误差小。 \(d=5\): 逼近误差 \(=0\)（包含 3 阶），估计误差适中。 \(d=20\): 逼近误差 \(=0\)，但估计误差大（参数多、样本不足）。 易错点：逼近误差为零不意味着泛化好——还要看估计误差。

04因果推断初步

🎯 为什么研究这个问题

"相关不等于因果"——但 ML 模型只能捕捉相关。因果推断从数学上 区分因果效应与虚假关联，使我们能回答"如果干预 X 会怎样"。 结构因果模型(SCM)和 do-calculus 提供了严格的形式化框架，在 医学、政策评估和公平 ML 中至关重要。

🔍 抽象化方法

核心抽象：从观察分布 \(P(Y|X)\) 到干预分布 \(P(Y|do(X))\)。 \(do(X=x)\) 表示强制设定 \(X=x\)（而非观察 \(X=x\)）。

📐 公式推导 / 模型建立

结构因果模型：每个变量 \(X_i = f_i(\mathrm{Pa}(X_i), U_i)\)， \(U_i\) 是独立噪声。DAG 编码因果结构。 do-calculus：\(P(Y|do(X=x))\) 通过"删除指向 \(X\) 的箭头"后计算： \[P(Y|do(X=x)) = \sum_z P(Y|X=x, Z=z)P(Z=z)\] （后门调整公式，当 \(Z\) 满足后门准则时）。 后门准则：\(Z\) 阻断所有 \(X \to Y\) 的后门路径（通过 \(X\) 的因的路径）， 且 \(Z\) 中无 \(X\) 的后代。 前门准则：当后门不可用时，通过中间变量估计因果效应。

🌐 实际应用

A/B 测试 \(=\) 随机化干预，直接估计 \(P(Y|do(X))\)。 观察性研究中必须用后门/前门调整。

✏️ 例题分析与易错点

例题：\(X\) = 是否吸烟, \(Y\) = 肺癌, \(Z\) = 基因。DAG: \(Z \to X\), \(Z \to Y\), \(X \to Y\)。 \(P(Y|X=1) \neq P(Y|do(X=1))\) 因为 \(Z\) 是混杂因子。 后门调整：\(P(Y|do(X)) = \sum_z P(Y|X,Z=z)P(Z=z)\)。 易错点：\(P(Y|X)\) 混合了因果效应和选择偏差，不能直接用于因果推断。
