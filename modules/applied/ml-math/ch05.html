01贝叶斯网络与有向图模型

🎯 为什么研究这个问题

现实中的概率模型涉及大量随机变量，直接建模联合分布需要指数级 参数。贝叶斯网络(BN)用有向无环图(DAG)编码变量间的条件独立关 系，实现联合分布的因子分解，大幅降低参数量。BN 是因果推理、 医学诊断和专家系统的数学框架。

🔍 抽象化方法

DAG 编码因果/条件独立结构：每个节点条件于其父节点时与非后代节 点独立。图的拓扑 \(\leftrightarrow\) 条件独立约束。

📐 公式推导 / 模型建立

贝叶斯网络：DAG \(G = (V, E)\)，每个节点 \(X_i\) 配条件概率 \(p(X_i | \mathrm{Pa}(X_i))\)。 因子分解： \[p(X_1, \ldots, X_n) = \prod_{i=1}^n p(X_i | \mathrm{Pa}(X_i))\] d-分离判据：给定集合 \(C\)，若在 \(G\) 中 \(A\) 和 \(B\) 被 \(C\) d-分离，则 \(X_A \perp X_B \mid X_C\)。 三种基本结构：链(\(A \to C \to B\))、分叉(\(A \leftarrow C \to B\))、 对撞(\(A \to C \leftarrow B\)，条件于 \(C\) 后不独立）。

🌐 实际应用

医学诊断系统：症状-疾病-检验结果的 BN 模型（如 QMR-DT 网络）。

✏️ 例题分析与易错点

例题：3 变量 BN: \(A \to C\), \(B \to C\)（对撞）。\(A, B\) 独立。 问：给定 \(C\) 后 \(A, B\) 是否独立？ 解答：不独立。\(A \to C \leftarrow B\) 是"对撞"(v-structure)， 给定 \(C\) 后开启了 \(A, B\) 之间的依赖（"explain-away"效应）。 易错点：对撞节点未条件化时阻止信息流动，条件化后反而打开——与链和分叉相反。

⚡ 快速检测
在对撞结构 \(A \to C \leftarrow B\) 中，\(A \perp B\) 但条件于 \(C\) 后？

02马尔可夫随机场与无向图模型

🎯 为什么研究这个问题

并非所有依赖关系都是"有向"的。图像中相邻像素的关联、社交网 络中的同伴关系等更适合用无向图描述。马尔可夫随机场(MRF)用无 向图编码局部马尔可夫性质，通过势函数定义因子分解。MRF 是图像 分割、纠错编码和统计物理的核心工具。

🔍 抽象化方法

无向图中条件独立通过"路径被阻断"（分离集）判定，比 d-分离更直观。

📐 公式推导 / 模型建立

MRF 定义：无向图 \(G = (V, E)\)，局部马尔可夫性 \(X_i \perp X_{V \setminus \{i\} \setminus \mathrm{nb}(i)} \mid X_{\mathrm{nb}(i)}\)。 Hammersley-Clifford 定理：（在正性条件下）MRF 性质 \(\Leftrightarrow\) Gibbs 分布： \[p(x) = \frac{1}{Z} \prod_{C \in \mathcal{C}} \psi_C(x_C)\] 其中 \(\mathcal{C}\) 是图的团(cliques)，\(\psi_C\) 是势函数， \(Z = \sum_x \prod_C \psi_C(x_C)\) 是配分函数。 配分函数 \(Z\) 通常难以精确计算（#P-hard），需近似推断方法。

🌐 实际应用

Ising 模型（统计物理）、CRF 序列标注（NLP）、图像去噪（MRF prior）。

✏️ 例题分析与易错点

例题：链形 MRF: \(X_1 - X_2 - X_3\)，\(X_i \in \{0,1\}\)。 势函数 \(\psi_{12}(x_1,x_2) = e^{x_1 x_2}\), \(\psi_{23}\) 同理。 求 \(P(X_1=1|X_3=1)\)。 解答：\(p(x) \propto e^{x_1x_2+x_2x_3}\)。条件于 \(X_3=1\): 边缘化 \(X_2\): \(p(X_1=1|X_3=1) \propto \sum_{x_2} e^{x_2+x_2} = 1 + e^2\)。 归一化后 \(P = (1+e^2)/(2+e^0+e^2) = (1+e^2)/(3+e^2) \approx 0.757\)。 易错点：配分函数 \(Z\) 必须在归一化时精确计算，遗忘会导致概率不正确。

03精确推断——变量消元与信念传播

🎯 为什么研究这个问题

图模型建好后，核心计算任务是推断：计算边缘概率、条件概率或 MAP (最大后验)。朴素枚举指数级不可行，但利用图的结构可以高效 进行。变量消元和信念传播利用因子分解的局部性，在树形图上实现 精确推断。

🔍 抽象化方法

推断的核心是利用分布式因子化结构避免全局枚举——"局部计算+消息传递"。

📐 公式推导 / 模型建立

变量消元：计算边缘 \(p(X_q) = \sum_{x \setminus x_q} p(x)\)。 关键：利用因子分解将求和"推进"，先消元与 \(X_q\) 无关的变量。 复杂度由消元宽度(treewidth)决定： \[\text{复杂度} = O(n \cdot |\mathcal{X}|^{w+1})\] 其中 \(w\) 是消元顺序对应的树宽。 信念传播(BP)：在因子图上传递消息 \[m_{i \to a}(x_i) \propto \prod_{b \in \mathrm{nb}(i) \setminus a} m_{b \to i}(x_i)\] \[m_{a \to i}(x_i) \propto \sum_{x_{\mathrm{nb}(a) \setminus i}} f_a(x_{\mathrm{nb}(a)}) \prod_{j \in \mathrm{nb}(a) \setminus i} m_{j \to a}(x_j)\] 树形图上 BP 精确，且与变量消元等价。

🌐 实际应用

Turbo 码和 LDPC 码的解码算法本质上是因子图上的信念传播。

✏️ 例题分析与易错点

例题：链 \(X_1-X_2-X_3\)，\(p \propto \psi_{12}(x_1,x_2)\psi_{23}(x_2,x_3)\)。 消元 \(X_1\): \(\tau(x_2) = \sum_{x_1}\psi_{12}(x_1,x_2)\)（一条"消息"）。 再消元 \(X_2\): \(p(x_3) \propto \sum_{x_2}\tau(x_2)\psi_{23}(x_2,x_3)\)。 易错点：消元顺序影响复杂度。好的顺序保持中间结果维度低。

04近似推断——变分方法与采样

🎯 为什么研究这个问题

非树形图（含环）上精确推断通常是 NP-hard 的。两大近似推断策略： 变分推断(VI)将推断转化为优化问题；MCMC 用随机采样近似后验。 两者各有优劣：VI 快但有偏，MCMC 无偏但慢。了解两者是使用 现代概率模型（VAE, BNN）的基础。

📐 公式推导 / 模型建立

变分推断：用简单分布族 \(q_\phi(z)\) 近似后验 \(p(z|x)\)。 最小化 KL 散度：\(\min_\phi D_{\mathrm{KL}}(q_\phi \| p(\cdot|x))\)。 等价于最大化 ELBO： \[\mathcal{L}(\phi) = \mathbb{E}_{q_\phi}[\log p(x,z) - \log q_\phi(z)] \leq \log p(x)\] 平均场近似：\(q(z) = \prod_i q_i(z_i)\)，每个 \(q_i\) 独立更新。 MCMC（Gibbs 采样）：逐变量从条件分布采样 \(z_i^{(t+1)} \sim p(z_i | z_{-i}^{(t)}, x)\)。 逐步收敛到目标分布（遍历性定理），但 mixing time 可能很长。

🌐 实际应用

VAE 使用 VI（重参数化技巧）训练生成模型；贝叶斯神经网络用 MCMC 或 VI 做参数后验推断。

✏️ 例题分析与易错点

例题：单变量后验 \(p(z|x) \propto e^{-z^2/2-|z-2|}\)。 用 \(q = N(\mu, \sigma^2)\) 做 VI，写出 ELBO。 解答：\(\mathcal{L} = \mathbb{E}_q[\log p(x,z)] + H(q)\) \(= \mathbb{E}_q[-z^2/2 - |z-2|] + \frac{1}{2}\log(2\pi e\sigma^2)\)。 对 \(\mu, \sigma\) 优化（梯度下降或解析）。 易错点：VI 最小化 \(D_{\mathrm{KL}}(q\|p)\)（mode-seeking），倾向低估后验方差。

⚡ 快速检测
