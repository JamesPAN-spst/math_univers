01监督学习的数学框架与经验风险

🎯 为什么研究这个问题

机器学习的核心问题是从有限数据中学习一个泛化良好的预测函数。 将此问题形式化为期望风险最小化，然后用经验风险近似，是理解所 有 ML 算法的统一起点。偏差-方差分解揭示了模型复杂度的双刃剑 效应，指导模型选择。

🔍 抽象化方法

从"拟合数据"到"期望风险最小化"是关键抽象：我们优化的真正 目标是在未见数据上的表现，经验风险只是它的有限样本近似。

📐 公式推导 / 模型建立

期望风险：\(R(f) = \mathbb{E}_{(X,Y) \sim P}[\ell(f(X), Y)]\)。 经验风险：\(\hat{R}_n(f) = \frac{1}{n}\sum_{i=1}^n \ell(f(x_i), y_i)\)。 偏差-方差分解（平方损失）： \[\mathbb{E}[(Y-\hat{f}(X))^2] = \mathrm{Bias}^2 + \mathrm{Var} + \sigma^2\] 其中 \(\mathrm{Bias} = \mathbb{E}[\hat{f}(X)] - f^*(X)\)， \(\mathrm{Var} = \mathbb{E}[(\hat{f}(X)-\mathbb{E}[\hat{f}(X)])^2]\)。 偏差-方差权衡：模型复杂度增加 \(\Rightarrow\) 偏差下降、方差上升。 最优复杂度在两者平衡处。

🌐 实际应用

交叉验证(CV)是偏差-方差权衡的实践工具：\(k\)-fold CV 估计测试误差， 选择最优模型复杂度。

✏️ 例题分析与易错点

例题：多项式回归 \(y = \sum_{j=0}^d w_j x^j\)。\(d=1\) (线性), \(d=10\), \(d=100\) 在 \(n=20\) 个数据点上分别表现如何？ 解答：\(d=1\) 高偏差低方差（欠拟合），\(d=100\) 低偏差高方差（过拟合）， \(d \approx 3-5\) 通常最优。 易错点：训练误差总随模型复杂度下降，但测试误差先降后升（U 型曲线）。

02线性回归与最小二乘

🎯 为什么研究这个问题

线性回归是最简单的参数化模型，也是最重要的基线。它不仅自身应 用广泛，更是理解所有参数化方法（从 Ridge 到神经网络）的起点。 最小二乘的闭式解揭示了线性代数在 ML 中的核心地位。

🔍 关键概念抽象

线性回归在概念层面可拆解为三组互相关联的思想。首先是"参数化假设"——我们假设因变量 \(y\) 与自变量 \(x\) 之间存在线性关系 \(y \approx w^T x + b\)，将无穷的函数空间限制为有限维参数空间。这一假设看似简化，却为后续所有参数估计方法奠定了基础。

其次是"最小二乘准则"。在所有可能的衡量拟合好坏的方式中，平方误差 \(\sum(y_i - \hat{y}_i)^2\) 同时具备数学可操作性（凸、可微）和概率解释（高斯噪声下等价于最大似然）。正是这种双重优势使得最小二乘成为默认损失。

第三个关键抽象是"投影的几何直觉"。将向量 \(y\) 投影到矩阵 \(X\) 的列空间，残差正交于该子空间——这一几何图景将代数方程 \(X^TX w = X^T y\) 赋予了直观的空间含义，也揭示了帽子矩阵 \(H = X(X^TX)^{-1}X^T\) 的对称幂等性质。

理解这三层抽象后，线性回归不再是一个孤立的公式，而是"假设→优化→几何"思维链的典型范例，为理解正则化、核方法乃至深度学习提供了原始模板。

📐 公式推导 / 模型建立

模型：\(y = X w + \epsilon\)，\(X \in \mathbb{R}^{n \times d}\)。 最小二乘：\(\min_w \|Xw - y\|^2\)。 正规方程（令梯度为零）： \[X^TX w = X^Ty \quad \Rightarrow \quad w^* = (X^TX)^{-1}X^Ty\] （\(X^TX\) 可逆时）。 几何解释：\(\hat{y} = Xw^* = X(X^TX)^{-1}X^Ty\) 是 \(y\) 在 \(\mathrm{Col}(X)\) 上的正交投影。投影矩阵 \(H = X(X^TX)^{-1}X^T\)（帽子矩阵）。 概率解释：假设 \(\epsilon \sim N(0,\sigma^2I)\)，MLE \(= \) 最小二乘。

🌐 实际应用

经济学回归分析、工程系统辨识、信号处理中的参数估计。

✏️ 例题分析与易错点

例题：3 个数据点 \((1,2),(2,3),(3,5)\)，求线性拟合 \(y=w_0+w_1 x\)。 解答：\(X = \begin{pmatrix}1&1\\1&2\\1&3\end{pmatrix}\), \(y=(2,3,5)^T\)。 \(X^TX = \begin{pmatrix}3&6\\6&14\end{pmatrix}\), \(X^Ty = (10,24)^T\)。 \(w^* = (X^TX)^{-1}X^Ty = \frac{1}{6}\begin{pmatrix}14&-6\\-6&3\end{pmatrix}(10,24)^T = (-1/3, 3/2)^T\)。 拟合线：\(y = -1/3 + 3/2 x\)。 易错点：\(X^TX\) 不可逆（共线性）时正规方程无唯一解，需正则化。

03逻辑回归与广义线性模型

🎯 为什么研究这个问题

线性回归处理连续输出，但分类问题需要输出概率。逻辑回归通过 sigmoid 函数将线性组合映射到 \([0,1]\)，是二分类的基本模型。它 属于广义线性模型(GLM)家族，连接了线性模型与指数族分布。

🔍 抽象化方法

GLM 的统一框架：响应变量的分布属于指数族，链接函数(link function) 将均值参数与线性预测器联系：\(g(\mathbb{E}[Y]) = w^Tx\)。

📐 公式推导 / 模型建立

逻辑回归：\(P(Y=1|x) = \sigma(w^Tx+b)\)，其中 \(\sigma(z) = 1/(1+e^{-z})\)。 负对数似然（交叉熵损失）： \[\ell(w) = -\sum_{i=1}^n [y_i \log \hat{p}_i + (1-y_i)\log(1-\hat{p}_i)]\] 梯度：\(\nabla \ell = X^T(\hat{p} - y)\)（形式与线性回归相同）。 Hessian：\(\nabla^2 \ell = X^T \mathrm{diag}(\hat{p}_i(1-\hat{p}_i)) X \succeq 0\)， 故 \(\ell\) 凸，梯度下降保证全局最优。 没有闭式解，需迭代求解（梯度下降或 Newton-Raphson/IRLS）。

🌐 实际应用

医学诊断（疾病概率预测）、信用评分、广告点击率预测等。

✏️ 例题分析与易错点

例题：1D 数据，\(w^*=2, b^*=-3\)。对 \(x=2\) 预测 \(P(Y=1)\)。 解答：\(z = 2\times 2 - 3 = 1\)。\(P(Y=1|x=2) = \sigma(1) = e/(1+e) \approx 0.731\)。 易错点：逻辑回归的决策边界是线性的（\(w^Tx+b=0\)），模型本身不能 学习非线性边界。需配合特征工程或核方法。

04正则化与模型选择

🎯 为什么研究这个问题

当特征维度高于样本数（\(d > n\)）或特征间共线，最小二乘过拟合或 无唯一解。正则化通过惩罚模型复杂度来控制过拟合——\(L_2\) (Ridge) 收缩参数，\(L_1\) (LASSO) 选择特征。信息准则（AIC/BIC）提供了 不依赖验证集的模型选择方法。

🔍 关键概念抽象

正则化的核心思想是在"拟合数据"与"保持简单"之间寻找平衡。纯粹的经验风险最小化会被训练数据中的噪声所误导，而正则化项 \(\Omega(w)\) 为参数施加结构性约束，将搜索范围限制在更稳定的子集中。这种"约束即先验"的视角贯穿整个统计学习理论。

\(L_2\) 正则化（Ridge）和 \(L_1\) 正则化（LASSO）代表了两种截然不同的简化策略。Ridge 将所有参数均匀地向零收缩，保留所有特征但降低其影响力；LASSO 则倾向于产生稀疏解——直接将不重要的特征系数置零，起到自动特征选择的作用。两者在几何上对应圆形与菱形约束域。

模型选择准则（AIC、BIC、交叉验证）从另一个角度处理同样的问题：给定有限数据，如何在不同复杂度的模型间进行公平比较。AIC 和 BIC 通过惩罚有效参数数量来估计泛化误差，而交叉验证则直接模拟"未见数据"的表现，二者殊途同归。

将正则化理解为贝叶斯 MAP 估计（\(L_2\) 对应高斯先验，\(L_1\) 对应拉普拉斯先验），使得频率学派与贝叶斯学派的视角在此交汇，形成对过拟合控制的统一理解。

📐 公式推导 / 模型建立

Ridge 回归： \[\min_w \|Xw-y\|^2 + \lambda\|w\|^2 \quad \Rightarrow \quad w^* = (X^TX+\lambda I)^{-1}X^Ty\] \(\lambda > 0\) 保证可逆。偏差-方差视角：\(\lambda\) 增大 \(\Rightarrow\) 偏差↑方差↓。 贝叶斯解释：\(\lambda\|w\|^2\) 等价于先验 \(w \sim N(0, \sigma^2/\lambda \cdot I)\)， Ridge 解 \(=\) MAP 估计。 AIC/BIC：\(\mathrm{AIC} = 2k - 2\ln\hat{L}\)，\(\mathrm{BIC} = k\ln n - 2\ln\hat{L}\)。 BIC 对大样本惩罚更重，倾向选更简单模型。

🌐 实际应用

Elastic Net（\(\lambda_1\|w\|_1 + \lambda_2\|w\|^2\)）结合 LASSO 的稀疏性 和 Ridge 的稳定性，在基因组学(GWAS)中广泛使用。

✏️ 例题分析与易错点

例题：\(d=100\) 特征，\(n=50\) 样本。最小二乘 \(X^TX\) 不可逆。 Ridge（\(\lambda=1\)）解 \(w^*=(X^TX+I)^{-1}X^Ty\) 总存在。 验证：\(X^TX+I\) 最小特征值 \(\geq 1 > 0\)，必定可逆。 易错点：\(\lambda\) 的选择需要交叉验证，不能凭"感觉"设定。
