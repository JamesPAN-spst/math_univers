01统计流形与 Fisher 信息矩阵

🎯 为什么研究这个问题

概率分布族 \(\{p_\theta\}\) 可以看作一个"流形"，参数 \(\theta\) 是局部坐标。 不同的参数化给出不同的梯度——欧几里得梯度在统计问题上不是自然 的。Fisher 信息矩阵作为流形的黎曼度量，定义了参数空间上的"自 然距离"，使得优化在统计意义上是最优的。

🔍 抽象化方法

关键抽象：参数化分布族 \(=\) 统计流形，Fisher 信息 \(=\) 黎曼度量。 这将微分几何的工具引入统计学和机器学习。

📐 公式推导 / 模型建立

统计流形：\(\mathcal{M} = \{p_\theta \mid \theta \in \Theta \subseteq \mathbb{R}^d\}\)。 Fisher 信息矩阵： \[[F(\theta)]_{ij} = \mathbb{E}_{p_\theta}\left[\frac{\partial \log p_\theta}{\partial \theta_i} \cdot \frac{\partial \log p_\theta}{\partial \theta_j}\right]\] \(F\) 是半正定的（\(F = \mathrm{Cov}[\nabla_\theta \log p_\theta]\)）， 作为黎曼度量定义了切空间上的内积：\(\langle u, v\rangle_\theta = u^T F(\theta) v\)。 参数无关性：Fisher 度量在重参数化 \(\theta \to \eta(\theta)\) 下自然变换 （协变），而欧几里得度量不是。

🌐 实际应用

Cramér-Rao 下界：无偏估计的方差 \(\geq F^{-1}\)，Fisher 信息刻画 了参数可估性的根本极限。

✏️ 例题分析与易错点

例题：\(p_\theta(x) = \theta e^{-\theta x}\)（指数分布，\(x>0\)）。求 Fisher 信息。 解答：\(\log p_\theta = \log\theta - \theta x\)。\(\partial/\partial\theta = 1/\theta - x\)。 \(F(\theta) = \mathbb{E}[(1/\theta-x)^2] = \mathrm{Var}(x) = 1/\theta^2\)。 易错点：Fisher 信息依赖于参数化方式。\(\eta = \log\theta\) 时 \(F_\eta = 1\)（常数）。

02自然梯度下降

🎯 为什么研究这个问题

欧几里得空间中的梯度在参数重参数化下不保持不变——换个参数化梯 度方向就变了。自然梯度(Amari, 1998)用 Fisher 度量替代欧几里得度 量，得到的更新方向与参数化无关，在统计意义上是最陡下降方向。 Adam 优化器可视为自然梯度的廉价近似。

📐 公式推导 / 模型建立

欧几里得梯度：\(\nabla_\theta \ell\) 是在 \(\|d\theta\|_2^2\) 度量下的最陡方向。 自然梯度：在 Fisher 度量 \(\|d\theta\|_F^2 = d\theta^T F d\theta\) 下的最陡方向： \[\tilde{\nabla}_\theta \ell = F(\theta)^{-1} \nabla_\theta \ell\] 自然梯度下降：\(\theta_{t+1} = \theta_t - \alpha F(\theta_t)^{-1} \nabla \ell(\theta_t)\)。 与 KL 散度的联系：\(D_{\mathrm{KL}}(p_\theta \| p_{\theta+d\theta}) \approx \frac{1}{2} d\theta^T F(\theta) d\theta\)。 自然梯度在 KL 散度意义上做等步长更新。 连接 Adam：对角近似 \(F \approx \mathrm{diag}(\mathbb{E}[g^2])\)， \(F^{-1}g \approx g/\sqrt{\mathbb{E}[g^2]}\)——与 Adam 的自适应步长形式类似。

🌐 实际应用

TRPO/PPO 强化学习算法使用 Fisher 矩阵约束策略更新步长在 KL 球内。

✏️ 例题分析与易错点

例题：1D 指数分布 \(\theta e^{-\theta x}\), \(F=1/\theta^2\)。 损失 \(\ell(\theta)=-\log p_\theta(x_0)=-\log\theta+\theta x_0\)。 欧梯度：\(\nabla=-1/\theta+x_0\)。自然梯度：\(F^{-1}\nabla=\theta^2(-1/\theta+x_0) = -\theta+\theta^2 x_0\)。 易错点：\(F^{-1}\) 的计算/存储是 \(O(d^2)\)/\(O(d^3)\)，实际中需近似（K-FAC 等）。

03KL 散度与 Fisher 度量的关系

🎯 为什么研究这个问题

KL 散度是信息论和 ML 中最常用的分布距离度量，但它不是真正的 度量（不对称、不满足三角不等式）。信息几何揭示了 KL 散度的局 部结构：在参数微小变化时，KL 散度的二阶展开恰好给出 Fisher 度量。这统一了信息论、统计学和微分几何。

🔍 抽象化方法

KL 散度是统计流形上的"伪距离"，Fisher 度量是其无穷小版本。

📐 公式推导 / 模型建立

KL 散度定义： \[D_{\mathrm{KL}}(p_\theta \| p_{\theta'}) = \int p_\theta \log\frac{p_\theta}{p_{\theta'}} dx\] 局部展开（\(\theta' = \theta + \delta\)，\(\delta\) 小）： \(D_{\mathrm{KL}}(p_\theta \| p_{\theta+\delta}) = 0 + 0 + \frac{1}{2}\delta^T F(\theta)\delta + O(\|\delta\|^3)\)。 一阶项为零（因为 KL 在 \(\theta'=\theta\) 处取极小值 \(0\)）， 二阶项 \(= \frac{1}{2}\delta^T F(\theta)\delta\)。 即 Fisher 矩阵是 KL 散度的 Hessian： \[F(\theta) = \nabla^2_{\theta'} D_{\mathrm{KL}}(p_\theta \| p_{\theta'}) \big|_{\theta'=\theta}\]

🌐 实际应用

变分推断(VI)中最小化 KL 散度 \(D_{\mathrm{KL}}(q_\phi\|p)\)，其梯度计算 涉及 Fisher 信息，连接了推断与信息几何。

✏️ 例题分析与易错点

例题：两个高斯 \(N(\mu,1)\) 和 \(N(\mu+\delta,1)\)。 \(D_{\mathrm{KL}} = \frac{\delta^2}{2}\)。Fisher 信息 \(F=1\)。 验证：\(D_{\mathrm{KL}} = \frac{1}{2}\delta^T F \delta = \frac{\delta^2}{2}\) \(\checkmark\)。 易错点：KL 散度不对称：\(D_{\mathrm{KL}}(p\|q) \neq D_{\mathrm{KL}}(q\|p)\)。 两者对应不同的变分推断目标（mode-seeking vs mode-covering）。

04统计流形上的测地线与指数族

🎯 为什么研究这个问题

信息几何的强大之处在于它揭示了指数族分布的特殊结构：指数族上 的 Fisher 流形是平坦的（"对偶平坦"），KL 散度可以分解为 Bregman 散度。这不仅美观，还使得 EM 算法的收敛性分析变得自然。

🔍 抽象化方法

指数族 \(\leftrightarrow\) 对偶平坦流形 \(\leftrightarrow\) Bregman 散度。

📐 公式推导 / 模型建立

指数族：\(p_\theta(x) = h(x) \exp(\theta^T T(x) - A(\theta))\)， \(A(\theta) = \log \int h(x) e^{\theta^T T(x)} dx\)（对数配分函数）。 性质：\(\nabla_\theta A = \mathbb{E}[T(x)] = \mu\)（均值参数）， \(\nabla^2_\theta A = \mathrm{Cov}[T(x)] = F(\theta)\)（Fisher 信息 \(=\) Hessian of \(A\)）。 对偶坐标：\(\theta\)（自然参数）↔ \(\mu = \nabla A(\theta)\)（均值参数）。 对偶平坦：在 \(\theta\) 坐标下"e-平坦"，在 \(\mu\) 坐标下"m-平坦"。 KL 散度 \(=\) Bregman 散度： \[D_{\mathrm{KL}}(p_\mu \| p_{\mu'}) = B_A(\theta' \| \theta) = A(\theta') - A(\theta) - \nabla A(\theta)^T(\theta'-\theta)\]

🌐 实际应用

EM 算法的每步可解释为在 m-平坦子流形和 e-平坦子流形之间的 交替投影，保证收敛。

✏️ 例题分析与易错点

例题：伯努利分布 \(p_\theta(x) = \theta^x(1-\theta)^{1-x}\)，\(x \in \{0,1\}\)。 写成指数族形式。解答：\(p = \exp(x\log\theta+(1-x)\log(1-\theta)) = \exp(x\log(\theta/(1-\theta)) + \log(1-\theta))\)。 自然参数 \(\eta = \log(\theta/(1-\theta))\)（logit），\(A(\eta) = \log(1+e^\eta)\)。 易错点：混淆自然参数与均值参数。伯努利的均值参数 \(\mu=\theta\)， 自然参数 \(\eta = \mathrm{logit}(\theta)\)。
