01特征映射与核函数

🎯 为什么研究这个问题

线性模型太弱，无法处理非线性关系。一个想法是将数据映射到更高维 空间再做线性分类——但显式计算高维特征代价巨大。核函数提供了一 个优雅的解决方案：只需计算内积 \(k(x,y) = \langle\phi(x),\phi(y)\rangle\) 而 无需显式构造 \(\phi\)。这就是"核技巧"的本质。

🔍 抽象化方法

核心抽象：从"在特征空间中工作"到"只需计算内积"——任何只依赖 内积的算法都可以"核化"。

📐 公式推导 / 模型建立

特征映射：\(\phi: \mathcal{X} \to \mathcal{H}\)（将输入映到高维空间）。 核函数：\(k(x,y) = \langle\phi(x), \phi(y)\rangle_{\mathcal{H}}\)。 例：多项式核 \(k(x,y) = (x^Ty + 1)^2\) 对应 \(\phi(x) = (x_1^2, x_2^2, \sqrt{2}x_1x_2, \sqrt{2}x_1, \sqrt{2}x_2, 1)^T\)。 显式 \(\phi\) 将 2D 映到 6D，但核只需 \(O(d)\) 计算。 高斯核（RBF）：\(k(x,y) = \exp(-\|x-y\|^2/(2\sigma^2))\) 对应无穷维特征空间——无法显式计算 \(\phi\)，但核可以。

🌐 实际应用

核 SVM、核 PCA、核 Ridge 回归——所有线性方法都可核化。

✏️ 例题分析与易错点

例题：验证 \(k(x,y) = (x^Ty)^2\) 在 \(\mathbb{R}^2\) 上对应的 \(\phi\)。 解答：\((x^Ty)^2 = (x_1y_1+x_2y_2)^2 = x_1^2y_1^2 + x_2^2y_2^2 + 2x_1x_2y_1y_2\) \(= \langle(x_1^2, x_2^2, \sqrt{2}x_1x_2), (y_1^2, y_2^2, \sqrt{2}y_1y_2)\rangle\)。 故 \(\phi(x) = (x_1^2, x_2^2, \sqrt{2}x_1x_2)^T\)。 易错点：不是所有函数都是合法核——必须满足正定性（Mercer 条件）。

02正定核与 RKHS

🎯 为什么研究这个问题

什么函数可以作为核？Mercer 定理回答了这个问题：核函数必须是正 定的。正定核与再生核 Hilbert 空间(RKHS)一一对应——RKHS 为核方 法提供了严格的函数空间框架，保证了优化问题的良定义性。

🔍 抽象化方法

RKHS 是一个函数的 Hilbert 空间，其中"点评估"是连续线性泛函。 再生性质：\(f(x) = \langle f, k(\cdot, x)\rangle_{\mathcal{H}}\)。

📐 公式推导 / 模型建立

正定核：\(k: \mathcal{X} \times \mathcal{X} \to \mathbb{R}\) 正定 iff 对任意 \(n\) 个点 \(x_1,\ldots,x_n\)，Gram 矩阵 \(K_{ij} = k(x_i,x_j)\) 半正定。 Mercer 定理：\(k\) 正定 \(\Leftrightarrow\) 存在特征映射 \(\phi\) 使 \(k(x,y) = \langle\phi(x),\phi(y)\rangle\)。 RKHS 构造（Moore-Aronszajn）：每个正定核唯一确定一个 RKHS \(\mathcal{H}_k\)： \[\mathcal{H}_k = \overline{\mathrm{span}\{k(\cdot, x) \mid x \in \mathcal{X}\}}\] 再生性质：\(f(x) = \langle f, k(\cdot, x)\rangle_{\mathcal{H}_k}\)。 RKHS 范数 \(\|f\|_{\mathcal{H}_k}\) 控制函数的"光滑度"。

🌐 实际应用

RKHS 框架使核方法的泛化分析严格化——RKHS 范数作为复杂度度量 出现在泛化界中。

✏️ 例题分析与易错点

例题：验证 \(k(x,y)=\min(x,y)\)（\(x,y > 0\)）是正定核。 解答：取 \(x_1 < x_2\)，Gram 矩阵 \(K = \begin{pmatrix}x_1&x_1\\x_1&x_2\end{pmatrix}\)。 特征值 \((x_1+x_2 \pm \sqrt{(x_2-x_1)^2})/2 = x_2, 0\)（半正定）。 一般情形可通过 \(k(x,y)=\int_0^\infty \mathbb{1}_{[0,x]}(t)\mathbb{1}_{[0,y]}(t)dt\) 验证。 易错点：Gram 矩阵半正定 \(\neq\) 对称正定。半正定允许零特征值。

03表示定理与核 SVM

🎯 为什么研究这个问题

核化后在无穷维空间中优化，解是否有限表示？表示定理给出肯定回 答：RKHS 中正则化优化问题的解总可以写成训练数据核函数的线性 组合。这将无穷维问题化为有限维，使核方法在实践中可行。

🔍 关键概念抽象

表示定理是核方法从理论走向实践的关键桥梁。它回答了一个自然的疑问：在可能无穷维的 RKHS 中做优化，最优解是否可以有限地表示？答案是肯定的——最优解总可以写成训练数据点处核函数的线性组合 \(f^* = \sum_{i=1}^n \alpha_i k(\cdot, x_i)\)。

这一定理的深层含义在于"数据决定表示"。无论特征空间有多大（甚至无穷维），我们只需 \(n\) 个系数就能完全描述最优解。这将函数空间优化问题降维为有限维参数优化问题，同时保留了在高维空间中工作的所有好处。

核 SVM 是表示定理最经典的应用。通过对偶形式，SVM 的优化目标只依赖于数据点之间的核函数值 \(k(x_i, x_j)\)——即核矩阵 \(K\)。这使得我们无需显式计算高维映射 \(\phi(x)\)，只需选择一个合适的核函数即可在隐式特征空间中构建最大间隔分类器。

📐 公式推导 / 模型建立

表示定理(Representer Theorem)：对 \[\min_{f \in \mathcal{H}_k} \sum_{i=1}^n L(y_i, f(x_i)) + \lambda\|f\|_{\mathcal{H}_k}^2\] 最优解形如 \(f^*(x) = \sum_{i=1}^n \alpha_i k(x, x_i)\)。 证明思路：将 \(f\) 分解为 \(f = f_\parallel + f_\perp\)， 其中 \(f_\parallel \in \mathrm{span}\{k(\cdot,x_i)\}\)。 由再生性 \(f(x_i) = f_\parallel(x_i)\)，且 \(\|f\|^2 = \|f_\parallel\|^2 + \|f_\perp\|^2\)， 故 \(f_\perp = 0\) 最优。 核 SVM 对偶：\(\max_\alpha \sum \alpha_i - \frac{1}{2}\sum \alpha_i\alpha_j y_iy_j k(x_i,x_j)\)， s.t. \(0 \leq \alpha_i \leq C\), \(\sum \alpha_i y_i = 0\)。

🌐 实际应用

核 SVM 在小样本高维数据（如生物信息学、文本分类）中表现优异。

✏️ 例题分析与易错点

例题：核 Ridge 回归，\(K_{ij}=k(x_i,x_j)\)，求 \(\alpha^*\)。 解答：代入表示定理后，优化变为 \(\min_\alpha \|K\alpha-y\|^2+\lambda\alpha^TK\alpha\)。 梯度为零：\(2K(K\alpha-y)+2\lambda K\alpha=0\), \((K+\lambda I)\alpha=y\), \(\alpha^* = (K+\lambda I)^{-1}y\)。预测：\(f^*(x_{new}) = k^T_{new}(K+\lambda I)^{-1}y\)。 易错点：表示定理只保证最优解有限表示，不是所有 RKHS 函数都有。

04高斯过程与核 PCA

🎯 为什么研究这个问题

核方法不仅用于分类——高斯过程(GP)是核方法的贝叶斯版本，不仅 给出预测还给出不确定性估计。核 PCA 则将 PCA 推广到非线性降维。 这些应用展示了核方法框架的强大统一性。

🔍 抽象化方法

GP 的核心抽象：函数本身是随机变量，由核函数决定其协方差结构。

📐 公式推导 / 模型建立

高斯过程：\(f \sim \mathcal{GP}(m(\cdot), k(\cdot,\cdot))\)。 有限维投影：\((f(x_1),\ldots,f(x_n)) \sim N(\mu, K)\)，\(K_{ij}=k(x_i,x_j)\)。 GP 回归（后验预测）： \[f^*|X,y,x^* \sim N(k_*^T(K+\sigma^2I)^{-1}y,\; k_{**}-k_*^T(K+\sigma^2I)^{-1}k_*)\] 其中 \(k_*\) 是测试-训练核向量。均值 \(=\) 核 Ridge 回归的解。 核 PCA：在特征空间中做 PCA，等价于对中心化 Gram 矩阵 \(\tilde{K}\) 做特征分解。投影：\(f_j(x) = \sum_i \alpha_i^{(j)} k(x, x_i)\)。

🌐 实际应用

GP 在贝叶斯优化（超参数调优）中是标准代理模型；核 PCA 用于 流形学习和非线性降维。

✏️ 例题分析与易错点

例题：GP 回归，\(n=1\) 个观测 \((x_1, y_1)=(0, 1)\)，\(k(x,y)=e^{-(x-y)^2}\)， \(\sigma^2=0.1\)。预测 \(x^*=1\) 处的均值和方差。 解答：\(K=[1]\), \(k_*=[e^{-1}]\), \(k_{**}=1\)。 均值 \(= e^{-1} \cdot (1+0.1)^{-1} \cdot 1 \approx 0.334\)。 方差 \(= 1 - e^{-2}/(1.1) \approx 0.877\)。远离观测点不确定性增大。 易错点：GP 的计算瓶颈是 \(O(n^3)\) 的矩阵求逆，大规模需近似方法。
