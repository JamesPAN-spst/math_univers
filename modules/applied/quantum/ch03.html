量子纠缠与 Bell 不等式

01Bell 态与纠缠的数学描述

🎯 为什么研究这个问题

量子纠缠是量子力学最深刻的特性——纠缠粒子之间的关联无法用经典 概率解释。四个 Bell 态构成 2-qubit 最大纠缠态的完备基，是量子隐 形传态、超密编码和量子密钥分发的资源。数学上需要精确刻画"怎样 的态是纠缠的"以及"纠缠度有多少"。

🔍 抽象化方法

可分态 vs 纠缠态是张量积结构下的核心二分法。纠缠度量 （entropy of entanglement）给出了量化理论。

📐 公式推导 / 模型建立

四个 Bell 态（2-qubit 最大纠缠态的正交基）： \[|\Phi^{\pm}\rangle = \frac{1}{\sqrt{2}}(|00\rangle \pm |11\rangle), \quad |\Psi^{\pm}\rangle = \frac{1}{\sqrt{2}}(|01\rangle \pm |10\rangle)\] Schmidt 分解：任意 2-qubit 纯态可写为 \(|\psi\rangle = \sum_i \lambda_i |a_i\rangle|b_i\rangle\)，\(\lambda_i \geq 0\), \(\sum \lambda_i^2 = 1\)。 可分 \(\Leftrightarrow\) 只有一个非零 Schmidt 系数。 纠缠熵：\(E(|\psi\rangle) = S(\rho_A) = -\sum_i \lambda_i^2 \log \lambda_i^2\)。 Bell 态：\(\lambda_1 = \lambda_2 = 1/\sqrt{2}\)，\(E = \log 2 = 1\) ebit（最大纠缠）。

🌐 实际应用

量子隐形传态：Alice 利用 Bell 态和经典通信将未知量子态传给 Bob， 不违反不可克隆定理。超密编码：一个量子比特传递 2 bit 经典信息。

✏️ 例题分析与易错点

例题：\(|\psi\rangle = \frac{\sqrt{3}}{2}|00\rangle + \frac{1}{2}|11\rangle\) 的纠缠熵。 解答：Schmidt 系数 \(\lambda_1 = \sqrt{3}/2, \lambda_2 = 1/2\)。 \(E = -\frac{3}{4}\log_2\frac{3}{4} - \frac{1}{4}\log_2\frac{1}{4}\) \(= -\frac{3}{4}(\log_2 3 - 2) - \frac{1}{4}(-2) = \frac{3}{2} - \frac{3}{4}\log_2 3 + \frac{1}{2} = 2 - \frac{3}{4}\log_2 3 \approx 0.811\) ebit。 易错点：Shannon 熵用 \(\lambda_i^2\)（概率），不是 \(\lambda_i\)。

⚡ 快速检测
\(|00\rangle + |01\rangle\)（未归一化）是纠缠态吗？

02CHSH 不等式与非定域性

🎯 为什么研究这个问题

Einstein-Podolsky-Rosen(EPR)在 1935 年质疑量子力学完备性，提出 "隐变量"解释。Bell 在 1964 年证明：任何经典隐变量理论必须满足 Bell 不等式，而量子力学可以违反它。CHSH 不等式是 Bell 不等式最 常用的实验可检验形式，已被多次实验确认。

🔍 抽象化方法

CHSH 不等式将物理问题转化为代数不等式：经典相关函数有界 \(\leq 2\)， 量子力学允许 \(\leq 2\sqrt{2}\)（Tsirelson 界）。

📐 公式推导 / 模型建立

CHSH 设置：Alice 选测量 \(A_1\) 或 \(A_2\)（结果 \(\pm 1\)），Bob 选 \(B_1\) 或 \(B_2\)。 CHSH 量：\(S = \langle A_1 B_1 \rangle + \langle A_1 B_2 \rangle + \langle A_2 B_1 \rangle - \langle A_2 B_2 \rangle\)。 经典（隐变量）上界： \[|S_{\text{classical}}| \leq 2\] 量子上界（Tsirelson 界）：选择 \(|\Phi^+\rangle\) 和最优测量方向， \[S_{\text{quantum}} = 2\sqrt{2} \approx 2.828\] 证明：取 \(A_1 = Z, A_2 = X, B_1 = (Z+X)/\sqrt{2}, B_2 = (Z-X)/\sqrt{2}\)。 每项 \(\langle A_i B_j \rangle = \cos\theta_{ij}/\sqrt{2}\)，优化后得 \(2\sqrt{2}\)。

🌐 实际应用

设备无关量子密钥分发(DI-QKD)：Bell 不等式违反保证安全性，不依赖设备信任。

✏️ 例题分析与易错点

例题：Alice 测 \(Z\)，Bob 测 \((Z+X)/\sqrt{2}\)，共享 \(|\Phi^+\rangle\)。 求 \(\langle A_1 B_1 \rangle\)。 解答：\(\langle \Phi^+| Z \otimes \frac{Z+X}{\sqrt{2}} |\Phi^+\rangle = \frac{1}{\sqrt{2}}(\langle \Phi^+|Z \otimes Z|\Phi^+\rangle + \langle \Phi^+|Z \otimes X|\Phi^+\rangle)\)。 计算：\(\langle Z \otimes Z \rangle = 1\)（因 \(|\Phi^+\rangle\) 总给相同 \(Z\) 结果）， \(\langle Z \otimes X \rangle = 0\)（对称性）。结果 \(= 1/\sqrt{2}\)。 易错点：Bell 态上不同算子的关联需要逐项计算，不能直接用经典直觉。

03量子隐形传态与超密编码

🎯 为什么研究这个问题

量子隐形传态(teleportation)和超密编码(superdense coding)是纠缠 作为资源的两大标志性协议。传态用 1 ebit + 2 cbit 传送 1 qubit； 超密编码用 1 ebit + 1 qubit 传送 2 cbit。它们完美体现了量子信 息论中经典和量子资源的互换关系。

🔍 抽象化方法

量子隐形传态的本质：Bell 基测量 + 经典通信 + 酉纠正。不需要直 接传送量子比特——打破了"物理传输"的直觉。

📐 公式推导 / 模型建立

量子隐形传态协议：Alice 持 \(|\psi\rangle = \alpha|0\rangle + \beta|1\rangle\)， Alice 和 Bob 共享 \(|\Phi^+\rangle_{AB}\)。 初始态：\(|\psi\rangle_1 |\Phi^+\rangle_{23} = (\alpha|0\rangle+\beta|1\rangle)\frac{1}{\sqrt{2}}(|00\rangle+|11\rangle)\) Bell 基测量（qubit 1,2）后，展开为四个 Bell 态的叠加： \[\frac{1}{2}[|\Phi^+\rangle_{12}(\alpha|0\rangle+\beta|1\rangle)_3 + |\Phi^-\rangle_{12}(\alpha|0\rangle-\beta|1\rangle)_3\] \[+ |\Psi^+\rangle_{12}(\alpha|1\rangle+\beta|0\rangle)_3 + |\Psi^-\rangle_{12}(\alpha|1\rangle-\beta|0\rangle)_3]\] Alice 测量得到 2 bit 结果，告诉 Bob。Bob 对 qubit 3 施加相应 Pauli 纠正 (\(I, Z, X, XZ\)) 恢复 \(|\psi\rangle\)。

🌐 实际应用

量子中继器网络中传态是远距离量子通信的核心操作（结合纠缠蒸馏）。

✏️ 例题分析与易错点

例题：Alice 测量得到 \(|\Psi^+\rangle\)，Bob 的 qubit 状态和纠正操作？ 解答：Bob 的态为 \(\alpha|1\rangle + \beta|0\rangle = X(\alpha|0\rangle+\beta|1\rangle)\)。 纠正：\(X \cdot (\alpha|1\rangle+\beta|0\rangle) = \alpha|0\rangle + \beta|1\rangle = |\psi\rangle\)。 易错点：传态不违反不可克隆——原始量子态在 Alice 侧被测量破坏了。

04不可克隆定理与量子信息基本限制

🎯 为什么研究这个问题

不可克隆定理(No-Cloning Theorem)是量子信息论最基本的约束之一： 不存在通用的量子复制机。这看似限制实则保护——它是量子密码学安 全性的根基。同时还有不可删除定理、不可通信定理等一系列基本限制 构成量子信息的公理框架。

🔍 抽象化方法

不可克隆 \(\Leftrightarrow\) 线性性与酉性的直接推论。 量子信息论的基本限制来自 Hilbert 空间的线性结构。

📐 公式推导 / 模型建立

不可克隆定理证明：假设存在酉算子 \(U\) 使得对所有 \(|\psi\rangle\)： \(U(|\psi\rangle|0\rangle) = |\psi\rangle|\psi\rangle\)。 取 \(|\psi\rangle = |0\rangle\)：\(U(|0\rangle|0\rangle) = |0\rangle|0\rangle\)。 取 \(|\psi\rangle = |1\rangle\)：\(U(|1\rangle|0\rangle) = |1\rangle|1\rangle\)。 取 \(|\psi\rangle = (|0\rangle+|1\rangle)/\sqrt{2}\)： \[U(|+\rangle|0\rangle) = \frac{1}{\sqrt{2}}(U|00\rangle + U|10\rangle) = \frac{1}{\sqrt{2}}(|00\rangle+|11\rangle)\] 但要求 \(|+\rangle|+\rangle = \frac{1}{2}(|00\rangle+|01\rangle+|10\rangle+|11\rangle)\)。矛盾！

🌐 实际应用

QKD 安全性：窃听者无法完美复制传输中的量子态，窃听必留痕迹。

✏️ 例题分析与易错点

例题：能否克隆一组已知的正交态 \(\{|0\rangle, |1\rangle\}\)？ 解答：可以！\(\text{CNOT}(|b\rangle|0\rangle) = |b\rangle|b\rangle\) 对 \(b \in \{0,1\}\)。 不可克隆只禁止克隆任意未知态。 易错点：不可克隆说的是"不存在对所有态都适用的克隆机"，正交基可以复制。
