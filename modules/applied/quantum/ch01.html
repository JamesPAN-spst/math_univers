量子比特与量子态

01量子比特与 Dirac 记号

🎯 为什么研究这个问题

经典计算机的 bit 只能是 0 或 1，量子比特(qubit)可以同时处于两 者的叠加态——这是量子计算超越经典计算的根源。Dirac 记号 \(|0\rangle, |1\rangle\) 提供了优雅的数学语言。理解量子态的数学表 示（复 Hilbert 空间中的单位向量）是量子计算的绝对起点。

🔍 抽象化方法

核心抽象：量子态 \(=\) 复 Hilbert 空间 \(\mathbb{C}^2\) 中的单位向量 （全局相位不影响物理，故实际是射影空间 \(\mathbb{CP}^1\)）。

📐 公式推导 / 模型建立

量子比特：\(|\psi\rangle = \alpha|0\rangle + \beta|1\rangle\)， \(\alpha, \beta \in \mathbb{C}\), \(|\alpha|^2 + |\beta|^2 = 1\)。 计算基：\(|0\rangle = \begin{pmatrix}1\\0\end{pmatrix}\), \(|1\rangle = \begin{pmatrix}0\\1\end{pmatrix}\)。 Bloch 球表示：\(|\psi\rangle = \cos(\theta/2)|0\rangle + e^{i\phi}\sin(\theta/2)|1\rangle\)， 其中 \(\theta \in [0,\pi]\), \(\phi \in [0,2\pi)\)。纯态 \(\leftrightarrow\) 球面上的点。 测量公设（Born 规则）：测量 \(|\psi\rangle\) 在计算基下， \[\Pr(\text{结果}=0) = |\alpha|^2, \quad \Pr(\text{结果}=1) = |\beta|^2\] 测量后量子态坍缩到对应基态。

🌐 实际应用

量子密钥分发(QKD, BB84 协议)利用测量导致坍缩的性质实现窃听检测。

✏️ 例题分析与易错点

例题：\(|\psi\rangle = \frac{1}{\sqrt{3}}|0\rangle + \sqrt{\frac{2}{3}}|1\rangle\)。 测量 \(|0\rangle\) 的概率？ 解答：\(P(0) = |1/\sqrt{3}|^2 = 1/3\)。\(P(1) = 2/3\)。 易错点：概率是振幅的模方 \(|\alpha|^2\)，不是 \(\alpha\) 本身。复数振幅的相位影响干涉但不影响单量子比特测量概率。

⚡ 快速检测
量子比特 \(|\psi\rangle = \frac{1}{\sqrt{2}}(|0\rangle + i|1\rangle)\)

02多量子比特与张量积

🎯 为什么研究这个问题

单量子比特状态空间为 \(\mathbb{C}^2\)。\(n\) 个量子比特的联合态空 间是张量积 \((\mathbb{C}^2)^{\otimes n} = \mathbb{C}^{2^n}\)——维度指数 增长，这正是量子计算威力的来源。理解张量积结构和纠缠是量子计 算的核心数学基础。

🔍 抽象化方法

张量积 \(\leftrightarrow\) 复合系统的数学描述。\(|a\rangle \otimes |b\rangle\) 描述两个独立子系统，但并非所有 \(\mathbb{C}^4\) 中的向量都可分解。

📐 公式推导 / 模型建立

2-qubit 系统：\(|\psi\rangle \in \mathbb{C}^{2} \otimes \mathbb{C}^{2} = \mathbb{C}^{4}\)。 计算基：\(|00\rangle, |01\rangle, |10\rangle, |11\rangle\)。 一般态：\(|\psi\rangle = \sum_{i,j \in \{0,1\}} c_{ij}|ij\rangle\)，\(\sum |c_{ij}|^2 = 1\)。 可分态 vs 纠缠态：\(|\psi\rangle = |a\rangle \otimes |b\rangle\) 为可分态。 Bell 态 \(|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)\) 不可分解 为张量积——这就是纠缠。 \(n\)-qubit：\(2^n\) 维空间，存储一个一般态需要 \(2^n\) 个复数。 经典模拟器无法高效存储/操作——这是量子计算优势的根源。

🌐 实际应用

量子随机数生成利用多量子比特测量的随机性，提供真正的随机数。

✏️ 例题分析与易错点

例题：判断 \(|\psi\rangle = \frac{1}{\sqrt{2}}(|01\rangle + |10\rangle)\) 是否纠缠。 解答：假设 \(|\psi\rangle = (a|0\rangle+b|1\rangle)(c|0\rangle+d|1\rangle)\)。 则 \(ac=0, ad=1/\sqrt{2}, bc=1/\sqrt{2}, bd=0\)。 \(ac=0\) 意味 \(a=0\) 或 \(c=0\)；但 \(ad \neq 0\) 要求 \(a,d \neq 0\)， 故 \(c=0\)，但 \(bc \neq 0\) 矛盾。\(\Rightarrow\) 纠缠态。 易错点：判断纠缠需要尝试所有分解——不存在分解即为纠缠。 对 2-qubit 可用 Schmidt 分解或 concurrence。

03测量公设与 Born 规则

🎯 为什么研究这个问题

量子测量是量子力学最反直觉的部分：测量不仅获取信息，还不可逆 地改变量子态。Born 规则将量子态的振幅与测量结果的概率联系起来。 理解投影测量和一般 POVM 测量是设计量子算法和分析量子协议的基础。

🔍 抽象化方法

测量对应一组投影算子 \(\{P_m\}\)，\(\sum P_m = I\)，概率由 \(\langle\psi|P_m|\psi\rangle\) 给出。

📐 公式推导 / 模型建立

投影测量：正交基 \(\{|m\rangle\}\) 定义测量。对 \(|\psi\rangle = \sum_m c_m |m\rangle\)： \[\Pr(\text{结果}=m) = |c_m|^2 = |\langle m|\psi\rangle|^2\] 测量后态：\(|\psi\rangle \to |m\rangle\)（坍缩）。 非计算基测量：在基 \(\{|+\rangle, |-\rangle\}\) 下测量 （\(|+\rangle = (|0\rangle+|1\rangle)/\sqrt{2}\)），概率 \(= |\langle +|\psi\rangle|^2\)。 期望值：可观测量 \(M\)（厄米算子）的期望 \[\langle M \rangle = \langle\psi|M|\psi\rangle = \sum_m m \cdot \Pr(m)\]

🌐 实际应用

量子态层析(Quantum State Tomography)：通过多种基的测量重建密度矩阵。

✏️ 例题分析与易错点

例题：\(|+\rangle = (|0\rangle+|1\rangle)/\sqrt{2}\) 在计算基下测量。 解答：\(P(0) = |\langle 0|+\rangle|^2 = |1/\sqrt{2}|^2 = 1/2\)。同理 \(P(1)=1/2\)。 但在 \(\{|+\rangle,|-\rangle\}\) 基下测量：\(P(+) = 1, P(-) = 0\)。 易错点：测量基不同结果完全不同。量子态本身不变，是测量方式决定了概率分布。

04密度矩阵与混合态

🎯 为什么研究这个问题

纯态 \(|\psi\rangle\) 只描述完全已知的量子系统。实际中常面对统计 混合（如部分已知的系统或纠缠系统的子系统），需要密度矩阵 \(\rho\) 来描述。密度矩阵统一了纯态和混合态，是量子信息理论的 基本语言。

🔍 抽象化方法

密度矩阵 = 量子态的最一般描述。纯态是特例 \(\rho = |\psi\rangle\langle\psi|\)。

📐 公式推导 / 模型建立

密度矩阵定义：\(\rho\) 是半正定、迹为 1 的矩阵。 纯态：\(\rho = |\psi\rangle\langle\psi|\)，\(\mathrm{tr}(\rho^2) = 1\)。 混合态：\(\rho = \sum_i p_i |\psi_i\rangle\langle\psi_i|\)，\(\mathrm{tr}(\rho^2) < 1\)。 物理量期望：\(\langle M \rangle = \mathrm{tr}(M\rho)\)。 测量概率：\(\Pr(m) = \mathrm{tr}(P_m \rho)\)。 部分迹（约化密度矩阵）：对复合系统 \(\rho_{AB}\)， \[\rho_A = \mathrm{tr}_B(\rho_{AB}) = \sum_j (I_A \otimes \langle j|_B)\rho_{AB}(I_A \otimes |j\rangle_B)\] 纠缠态的约化密度矩阵是混合态——子系统信息不完全。

🌐 实际应用

量子通信中的信道容量分析需要密度矩阵语言；量子纠错也基于此。

✏️ 例题分析与易错点

例题：Bell 态 \(|\Phi^+\rangle = (|00\rangle+|11\rangle)/\sqrt{2}\) 的约化密度矩阵 \(\rho_A\)。 解答：\(\rho_{AB} = |\Phi^+\rangle\langle\Phi^+| = \frac{1}{2}(|00\rangle\langle 00|+|00\rangle\langle 11|+|11\rangle\langle 00|+|11\rangle\langle 11|)\)。 \(\rho_A = \mathrm{tr}_B(\rho_{AB}) = \frac{1}{2}(|0\rangle\langle 0|+|1\rangle\langle 1|) = \frac{I}{2}\)。 这是最大混合态——A 的局部信息完全随机。 易错点：整体纯态（\(\mathrm{tr}(\rho_{AB}^2)=1\)）的子系统可以是混合态。
