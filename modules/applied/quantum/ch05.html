量子错误纠正

01量子噪声与错误模型

🎯 为什么研究这个问题

真实量子硬件充满噪声——退相干、门误差、测量错误。如果不能纠错， 量子计算将无法扩展。与经典比特只有翻转(bit-flip)不同，量子比特 还有相位翻转(phase-flip)和两者的组合。量子纠错必须在不直接测量 （破坏）量子态的前提下诊断和修复错误。

🔍 抽象化方法

量子信道(Quantum Channel) \(=\) 完全正的保迹(CPTP)映射。 Kraus 表示：\(\mathcal{E}(\rho) = \sum_k E_k \rho E_k^\dagger\)。

📐 公式推导 / 模型建立

主要噪声模型： 1) 比特翻转信道：\(\mathcal{E}(\rho) = (1-p)\rho + pX\rho X\) 2) 相位翻转信道：\(\mathcal{E}(\rho) = (1-p)\rho + pZ\rho Z\) 3) 去极化信道：\(\mathcal{E}(\rho) = (1-p)\rho + \frac{p}{3}(X\rho X + Y\rho Y + Z\rho Z)\) 关键观察：任何单量子比特错误可分解为 Pauli 基 \(\{I, X, Y, Z\}\)： \[E = c_0 I + c_1 X + c_2 Y + c_3 Z\] 因此纠正 Pauli 错误即可纠正一般连续错误——数字化噪声！ 保真度(Fidelity)：\(F(\rho, \sigma) = \left(\mathrm{tr}\sqrt{\sqrt{\rho}\sigma\sqrt{\rho}}\right)^2\)， 衡量量子态经历噪声后的质量。

🌐 实际应用

IBM/Google 的量子处理器每个门错误率约 \(10^{-3}\)，需要纠错才能执行深线路。

✏️ 例题分析与易错点

例题：去极化信道 \(p=0.1\) 作用于 \(|0\rangle\)。输出密度矩阵？ 解答：\(\rho = |0\rangle\langle 0|\)。 \(X\rho X = |1\rangle\langle 1|\), \(Y\rho Y = |1\rangle\langle 1|\), \(Z\rho Z = |0\rangle\langle 0|\)。 \(\mathcal{E}(\rho) = 0.9|0\rangle\langle 0| + \frac{0.1}{3}(|1\rangle\langle 1|+|1\rangle\langle 1|+|0\rangle\langle 0|)\) \(= (0.9 + 0.1/3)|0\rangle\langle 0| + 2(0.1/3)|1\rangle\langle 1| \approx 0.933|0\rangle\langle 0| + 0.067|1\rangle\langle 1|\)。 易错点：\(Y\rho Y\) 中 \(Y = iXZ\)，计算时注意相位：\(Y|0\rangle = i|1\rangle\)，\(Y|0\rangle\langle 0|Y^\dagger = |1\rangle\langle 1|\)。

02经典纠错到量子：Shor 码与 Steane 码

🎯 为什么研究这个问题

Shor 在 1995 年提出第一个量子纠错码——9-qubit 码，用 9 个物理量 子比特保护 1 个逻辑量子比特。Steane 的 7-qubit 码基于经典 Hamming 码构造，是 CSS 码(Calderbank-Shor-Steane)的原型。这些 开创性工作证明了量子纠错的可行性。

🔍 抽象化方法

CSS 码 \(=\) 从两个嵌套经典码 \(C_2 \subset C_1\) 构建量子码。 \(X\)-错误用 \(C_1\) 纠正，\(Z\)-错误用 \(C_2^\perp\) 纠正——解耦处理。

📐 公式推导 / 模型建立

Shor 码 \([[9,1,3]]\)：编码 1 逻辑量子比特到 9 物理量子比特。 \(|0_L\rangle = \frac{1}{2\sqrt{2}}(|000\rangle+|111\rangle)^{\otimes 3}\) \(|1_L\rangle = \frac{1}{2\sqrt{2}}(|000\rangle-|111\rangle)^{\otimes 3}\) 结构：先用相位翻转码（3 个 block 保护 \(Z\) 错误）， 每个 block 内用比特翻转码（3 qubit 保护 \(X\) 错误）。 Steane 码 \([[7,1,3]]\)：基于经典 \([7,4,3]\) Hamming 码。 稳定子群由 6 个生成元构成（3 个 \(X\)-type + 3 个 \(Z\)-type）。 \[S_1 = IIIXXXX, \quad S_4 = IIIZZZZ\] \[S_2 = IXXIIXX, \quad S_5 = IZZIIZZ\] \[S_3 = XIXIXIX, \quad S_6 = ZIZIZIZ\]

🌐 实际应用

Steane 码支持横向(transversal) CNOT 门——这对容错逻辑门实现至关重要。

✏️ 例题分析与易错点

例题：Shor 码中第 2 个 qubit 发生 \(X\) 错误。如何检测和纠正？ 解答：第 1 个 block = \(|000\rangle+|111\rangle\) 变为 \(|010\rangle+|101\rangle\)。 测量稳定子 \(Z_1 Z_2\)（结果 \(-1\)）和 \(Z_2 Z_3\)（结果 \(-1\)），定位 qubit 2。 施加 \(X_2\) 纠正。其他 block 不受影响。 易错点：必须用稳定子测量而非直接测量量子比特——直接测量会破坏叠加！

03稳定子码与容错阈值

🎯 为什么研究这个问题

稳定子形式(Stabilizer Formalism)是描述量子纠错码的统一框架——用 Pauli 群的子群定义码空间。容错阈值定理(Threshold Theorem)则保证： 只要物理错误率低于某个阈值，逻辑错误率可以任意压低。这是大规模 量子计算可行性的理论基石。

🔍 抽象化方法

稳定子群 \(\mathcal{S} \subset \mathcal{P}_n\)（\(n\)-qubit Pauli 群）。 码空间 = \(\mathcal{S}\) 所有元素的 \(+1\) 特征空间。

📐 公式推导 / 模型建立

稳定子码：\(\mathcal{S} = \langle g_1, \ldots, g_{n-k}\rangle\) 是 Abel 群， \(-I \notin \mathcal{S}\)，编码 \(k\) 逻辑量子比特到 \(n\) 物理量子比特。 综合征(Syndrome)：错误 \(E\) 的综合征 = 向量 \((s_1, \ldots, s_{n-k})\)， \(s_i = 0\) 若 \([E, g_i] = 0\)，\(s_i = 1\) 若 \(\{E, g_i\} = 0\)。 不同错误对应不同综合征 \(\Rightarrow\) 可识别并纠正。 容错阈值定理：存在阈值 \(p_{\text{th}} > 0\)，若每个物理门/测量的错误率 \(p < p_{\text{th}}\)，则通过码级联(concatenation)或增大码距，逻辑错误率 \[p_L \sim \left(\frac{p}{p_{\text{th}}}\right)^{d/2}\] 可任意降低。表面码(Surface Code)的阈值 \(p_{\text{th}} \approx 1\%\)。

🌐 实际应用

Google 2023 年实验首次展示了增大表面码距可降低逻辑错误率——验证了阈值行为。

✏️ 例题分析与易错点

例题：3-qubit 比特翻转码的稳定子生成元和综合征表。 解答：\(g_1 = Z_1 Z_2, g_2 = Z_2 Z_3\)。 \(E = I\): syndrome \((0,0)\); \(E = X_1\): \((1,0)\); \(E = X_2\): \((1,1)\); \(E = X_3\): \((0,1)\)。 4 个不同综合征 \(\Rightarrow\) 可唯一定位并纠正单 qubit \(X\) 错误。 易错点：稳定子测量不破坏码空间内的信息——只提取错误信息。

04表面码与拓扑量子计算

🎯 为什么研究这个问题

表面码(Surface Code / Toric Code)是目前最有前景的量子纠错码—— 它只需最近邻耦合（适合平面芯片布局），阈值高（~1%），且码距可 通过增大阵列尺寸提升。拓扑量子计算则用拓扑保护替代主动纠错， 是长远的"终极"方案（如 Microsoft 的 Majorana 路线）。

🔍 抽象化方法

表面码的错误 \(\leftrightarrow\) 环面上的同调类。逻辑算子 \(=\) 不可缩回路径——拓扑保护！

📐 公式推导 / 模型建立

表面码定义：\(L \times L\) 方格上的量子比特（在边上）， 稳定子：面(plaquette)算子 \(B_p = \prod_{e \in \partial p} Z_e\)， 星(star)算子 \(A_s = \prod_{e \ni s} X_e\)。 码参数：\([[2L^2, 2, L]]\)（环面）或 \([[L^2 + (L-1)^2, 1, L]]\)（平面边界）。 错误 = 激发的准粒子对（face 激发 = \(e\) 粒子，vertex 激发 = \(m\) 粒子）。 纠错 = 将准粒子配对消除。最小权匹配(MWPM)解码： \[\text{找权最小的路径连接同类型准粒子对}\] 复杂度 \(O(n^3)\)，实际中 Union-Find 解码器接近线性。 逻辑门：横向 Hadamard 和 CNOT 可自然实现；\(T\) 门需魔术态蒸馏(Magic State Distillation)。

🌐 实际应用

Google Sycamore / IBM Eagle 的中期纠错路线图均以表面码为核心。

✏️ 例题分析与易错点

例题：\(L=3\) 的平面表面码能纠正多少个错误？ 解答：码距 \(d = L = 3\)，可纠正 \(\lfloor(3-1)/2\rfloor = 1\) 个错误。 物理量子比特数 \(= 3^2 + 2^2 = 13\)，编码 1 个逻辑量子比特。 开销比 \(13:1\)。增大 \(L\) 可改善但开销也增大。 易错点：表面码纠正 1 个错误不意味只有 1 个量子比特出错——而是错误的"path"不横穿整个格子。
