01Deutsch-Jozsa 与量子并行性

🎯 为什么研究这个问题

Deutsch-Jozsa 算法是具有指数加速的最早量子算法——判断函数 \(f:\{0,1\}^n \to \{0,1\}\) 是常值还是平衡只需 1 次查询（经典需 \(2^{n-1}+1\) 次）。虽然实际意义有限，但它完美展示了量子并行性 和干涉两大核心原理，是理解后续算法的必经之路。

🔍 抽象化方法

量子并行性：\(H^{\otimes n}|0\rangle^n\) 一次性准备所有输入的叠加； oracle 同时计算所有 \(f(x)\)；最终干涉抽取全局性质。

📐 公式推导 / 模型建立

Oracle：\(U_f|x\rangle|y\rangle = |x\rangle|y \oplus f(x)\rangle\)。 利用 \(|{-}\rangle\) 辅助位实现 phase kickback： \(U_f|x\rangle|{-}\rangle = (-1)^{f(x)}|x\rangle|{-}\rangle\)。 Deutsch-Jozsa 线路： \[|0\rangle^n|1\rangle \xrightarrow{H^{\otimes(n+1)}} \frac{1}{\sqrt{2^n}}\sum_x|x\rangle|{-}\rangle \xrightarrow{U_f} \frac{1}{\sqrt{2^n}}\sum_x(-1)^{f(x)}|x\rangle|{-}\rangle\] \[\xrightarrow{H^{\otimes n}} \sum_y \left[\frac{1}{2^n}\sum_x(-1)^{f(x)+x\cdot y}\right]|y\rangle|{-}\rangle\] 测量全 \(|0\rangle^n\) 的概率：\(\left|\frac{1}{2^n}\sum_x (-1)^{f(x)}\right|^2\)。 常值函数：概率 \(= 1\)；平衡函数：概率 \(= 0\)。一次测量完美区分。

🌐 实际应用

Phase kickback 技术是 Grover、Shor 等几乎所有量子算法的核心子程序。

✏️ 例题分析与易错点

例题：\(n=2\), \(f(x) = x_1 \oplus x_2\)（平衡函数）。验证测量 \(|00\rangle\) 概率为 0。 解答：\((-1)^{f(00)}=1, (-1)^{f(01)}=-1, (-1)^{f(10)}=-1, (-1)^{f(11)}=1\)。 \(\frac{1}{4}(1-1-1+1) = 0\)。确认无法得到 \(|00\rangle\)。 易错点：Phase kickback 要求辅助位在 \(|{-}\rangle\)，不是 \(|0\rangle\)。

02Grover 搜索算法

🎯 为什么研究这个问题

在无序数据库中搜索一个目标，经典算法需要 \(O(N)\) 次查询。Grover 算法只需 \(O(\sqrt{N})\) 次——二次加速。虽非指数式，但对 NP-hard 问题的暴力搜索有巨大实际意义。Grover 搜索还被证明是最优的 （BBBV 下界），展示了量子计算能力的真实边界。

🔍 抽象化方法

振幅放大(Amplitude Amplification)：通过迭代"标记→反射"操作，目标 态的振幅在每步增长 \(O(1/\sqrt{N})\)，约 \(\sqrt{N}\) 步达到峰值。

📐 公式推导 / 模型建立

设 \(N = 2^n\)，目标态 \(|w\rangle\)。初始均匀叠加 \(|s\rangle = H^{\otimes n}|0\rangle = \frac{1}{\sqrt{N}}\sum_x|x\rangle\)。 Grover 迭代(Grover operator) \(G = D \cdot O\)： - Oracle \(O\)：\(O|x\rangle = (-1)^{f(x)}|x\rangle\)（标记目标，翻转相位） - Diffusion \(D = 2|s\rangle\langle s| - I\)（关于 \(|s\rangle\) 的反射） 几何解释：在 \(\{|w\rangle, |w^{\perp}\rangle\}\) 平面内，\(G\) 是旋转角 \(2\theta\) 的操作，其中 \(\sin\theta = 1/\sqrt{N}\)。 \[\text{经过 } k \text{ 次迭代：} G^k|s\rangle = \sin((2k+1)\theta)|w\rangle + \cos((2k+1)\theta)|w^{\perp}\rangle\] 最优迭代次数：\((2k+1)\theta \approx \pi/2\)，即 \(k \approx \frac{\pi}{4}\sqrt{N}\)。

🌐 实际应用

SAT 求解加速、碰撞查找、AES 密钥搜索（128-bit 安全降为 64-bit 量子安全）。

✏️ 例题分析与易错点

例题：\(N=4\)（2 qubit），目标 \(|11\rangle\)。初始态 \(|s\rangle = \frac{1}{2}(|00\rangle+|01\rangle+|10\rangle+|11\rangle)\)。 一次 Grover 迭代后各振幅？ 解答：Oracle 后：\(\frac{1}{2}(|00\rangle+|01\rangle+|10\rangle-|11\rangle)\)。 Diffusion：均值 \(\bar{a} = (1/2+1/2+1/2-1/2)/4 = 1/4\)。 新振幅 \(= 2\bar{a} - a_x\)：非目标 \(= 2(1/4)-1/2 = 0\)，目标 \(= 2(1/4)+1/2 = 1\)。 一次迭代即精确找到 \(|11\rangle\)。（\(N=4\) 时 \(k=1\) 恰好精确。） 易错点：迭代太多次振幅会"转过头"——不是越多越好！

03量子 Fourier 变换与相位估计

🎯 为什么研究这个问题

量子 Fourier 变换(QFT)是经典 FFT 的量子版本，只需 \(O(n^2)\) 门（经典 FFT 需 \(O(N\log N) = O(n2^n)\) 算术操作）。QFT 是 Shor 算法的核心子程序，也是相位估计的基础——而相位估计连接 了量子模拟和量子化学等应用。

🔍 抽象化方法

QFT 将计算基态映射到频率基态，在量子版本中可以指数级高效实现。

📐 公式推导 / 模型建立

QFT 定义（\(N = 2^n\)）： \[\text{QFT}|j\rangle = \frac{1}{\sqrt{N}} \sum_{k=0}^{N-1} e^{2\pi i jk/N} |k\rangle\] 张量积分解（高效线路的关键）： \[\text{QFT}|j\rangle = \frac{1}{\sqrt{2^n}} \bigotimes_{l=1}^{n} \left(|0\rangle + e^{2\pi i j/2^l}|1\rangle\right)\] 每个量子比特独立（但相位依赖 \(j\) 的不同位）。 实现：\(n\) 个 Hadamard 门 + \(n(n-1)/2\) 个受控相位门 = \(O(n^2)\) 门。 相位估计(QPE)：给定酉算子 \(U\) 和特征态 \(|u\rangle\)（\(U|u\rangle = e^{2\pi i\phi}|u\rangle\)）， QPE 用 \(O(n)\) 次受控 \(U\) 操作 + 逆 QFT 得到 \(\phi\) 的 \(n\)-bit 近似。

🌐 实际应用

量子化学：QPE 估计分子 Hamilton 量的基态能量，实现指数加速。

✏️ 例题分析与易错点

例题：2-qubit QFT 作用于 \(|3\rangle = |11\rangle\)。 解答：\(\text{QFT}|3\rangle = \frac{1}{2}\sum_{k=0}^{3} e^{2\pi i \cdot 3k/4}|k\rangle = \frac{1}{2}(|0\rangle + e^{3\pi i/2}|1\rangle + e^{3\pi i}|2\rangle + e^{9\pi i/2}|3\rangle)\) \(= \frac{1}{2}(|0\rangle - i|1\rangle - |2\rangle + i|3\rangle)\)。 易错点：QFT 不是经典 DFT 的直接替代——QFT 输出态的振幅无法直接读取， 必须通过测量提取信息。

04Shor 因数分解算法

🎯 为什么研究这个问题

Peter Shor 在 1994 年发现量子计算机可以在多项式时间内分解大整数 ——这直接威胁 RSA 等基于大数分解困难性的公钥密码系统。Shor 算 法是量子计算最著名的"杀手级应用"，也是后量子密码学兴起的主要 驱动力。

🔍 抽象化方法

Shor 算法将分解问题归约为求周期问题(Order Finding)，而量子 Fourier 变换可以高效提取函数的周期——经典算法做不到这一点。

📐 公式推导 / 模型建立

分解 \(N\) 的步骤： 1) 随机选 \(a < N\)，若 \(\gcd(a,N) > 1\) 则直接得因子 2) 找 \(a \bmod N\) 的阶 \(r\)：最小正整数使 \(a^r \equiv 1 \pmod{N}\) 量子求阶(Order Finding)： - 准备 \(\frac{1}{\sqrt{2^m}}\sum_{x=0}^{2^m-1}|x\rangle|0\rangle\) - 计算 \(|x\rangle|a^x \bmod N\rangle\) - 对第一个寄存器做 QFT： \[\xrightarrow{\text{QFT}} \sum_s \frac{1}{\sqrt{r}}|s \cdot 2^m/r\rangle|\cdots\rangle\] 测量得到 \(s/r\) 的近似，连分数展开恢复 \(r\)。 3) 若 \(r\) 为偶数，\(\gcd(a^{r/2}\pm 1, N)\) 以高概率给出非平凡因子。 复杂度：\(O(n^3)\)（\(n = \log N\)），指数优于经典最佳的亚指数算法。

🌐 实际应用

后量子密码迁移：NIST 已标准化 Kyber/Dilithium 等抗量子算法以应对 Shor 威胁。

✏️ 例题分析与易错点

例题：用 Shor 算法分解 \(N=15\)，取 \(a=7\)。 解答：\(7^1 \equiv 7, 7^2 \equiv 4, 7^3 \equiv 13, 7^4 \equiv 1 \pmod{15}\)， 故 \(r = 4\)（偶数）。\(a^{r/2} = 7^2 = 49 \equiv 4\)。 \(\gcd(4-1, 15) = \gcd(3, 15) = 3\), \(\gcd(4+1, 15) = \gcd(5, 15) = 5\)。 因子：\(15 = 3 \times 5\)。 易错点：\(a\) 的选择可能不幸（如 \(a=6\), \(r=2\), \(a^{r/2}=6\), \(6+1=7\) 不整除 \(15\)，但 \(6-1=5\) 可以）。
