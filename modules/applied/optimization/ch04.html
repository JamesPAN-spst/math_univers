对偶理论

01Lagrange 对偶函数与弱对偶

🎯 为什么研究这个问题

对偶理论是优化中最深刻的思想之一：每个优化问题都有一个"镜像" 问题（对偶问题），它从不同角度逼近同一个最优值。弱对偶定理说 明对偶问题总提供原问题最优值的下界。对偶视角不仅是理论工具， 更是 SVM 核技巧、线性规划灵敏度分析等实际方法的基础。

🔍 抽象化方法

原问题 \(\rightarrow\) Lagrange 函数 \(\rightarrow\) 对偶函数（取下确界） \(\rightarrow\) 对偶问题（取上确界）。这构成一个系统的"松弛-逼近"框架。

📐 公式推导 / 模型建立

Lagrange 函数：\(\mathcal{L}(x,\lambda,\nu) = f(x) + \sum \lambda_j g_j(x) + \sum \nu_i h_i(x)\)。 对偶函数： \[g(\lambda,\nu) = \inf_x \mathcal{L}(x,\lambda,\nu)\] 性质：\(g\) 总是凹函数（即使原问题非凸），因为它是仿射函数关于 \((\lambda,\nu)\) 的逐点下确界。 弱对偶定理：\(\forall \lambda \geq 0\), \(g(\lambda,\nu) \leq p^*\)（原问题最优值）。 对偶问题：\(d^* = \sup_{\lambda \geq 0,\nu} g(\lambda,\nu) \leq p^*\)。 对偶间隙：\(p^* - d^* \geq 0\)（弱对偶保证）。

🌐 实际应用

弱对偶用于提供最优值下界：在分支定界法中，每个子问题的对偶界 用于剪枝。

✏️ 例题分析与易错点

例题：\(\min x^2\) s.t. \(x \geq 1\)。写出对偶问题并求 \(d^*\)。 解答：\(\mathcal{L}=x^2+\lambda(1-x)\)。 \(g(\lambda)=\inf_x(x^2-\lambda x+\lambda)\)。令 \(2x-\lambda=0\), \(x=\lambda/2\)。 \(g(\lambda)=\lambda^2/4-\lambda^2/2+\lambda=\lambda-\lambda^2/4\)。 \(\max_{\lambda \geq 0} g(\lambda)\): \(g'=1-\lambda/2=0\), \(\lambda^*=2\), \(d^* = 2-1 = 1 = p^*\)。此例强对偶成立。 易错点：对偶函数定义中的 \(\inf\) 可能为 \(-\infty\)，此时该 \(\lambda\) 值无用。

02强对偶与 Slater 条件

🎯 为什么研究这个问题

弱对偶给出下界，但我们更希望 \(d^* = p^*\)（强对偶），因为此时 可通过求解（可能更简单的）对偶问题获得原问题最优值。Slater 条 件是保证凸问题强对偶成立的最常用充分条件，也是 KKT 充要性的关键。

📐 公式推导 / 模型建立

强对偶：\(d^* = p^*\)（对偶间隙为零）。 Slater 条件：对凸问题（\(f,g_j\) 凸，\(h_i\) 仿射），若存在严格 可行点 \(\tilde{x}\): \(g_j(\tilde{x}) < 0\), \(h_i(\tilde{x}) = 0\)，则强对偶成立。 证明思路（不详述）：利用凸集分离定理。定义集合 \[\mathcal{A} = \{(u,v,t) \mid \exists x:\, g_j(x) \leq u_j,\, h_i(x) = v_i,\, f(x) \leq t\}\] Slater 条件保证 \((0,0,p^*)\) 在 \(\mathcal{A}\) 的边界上，存在分离超平面 即对偶变量。 推论：强对偶 + KKT \(\Leftrightarrow\) 原问题-对偶问题同时最优。

🌐 实际应用

几乎所有凸优化求解器都假设 Slater 条件成立，从而保证对偶间隙 为零，可用对偶残差作为终止准则。

✏️ 例题分析与易错点

例题：\(\min x_1+x_2\) s.t. \(x_1^2+x_2^2 \leq 1\)。Slater 条件是否满足？ 解答：取 \(\tilde{x}=(0,0)\)，\(0^2+0^2=0<1\)，严格可行。Slater 满足， 强对偶成立。对偶最优 \(d^*=p^*=-\sqrt{2}\)。 易错点：Slater 条件要求严格不等式 \(g_j<0\)，等式约束不需要严格。 线性不等式约束自动满足 Slater（若可行）。

03互补松弛与灵敏度分析

🎯 为什么研究这个问题

强对偶成立时，互补松弛条件 \(\lambda_j^* g_j(x^*) = 0\) 连接了原问 题与对偶问题的最优解。它不仅是理论结果，更有实际价值：乘子 \(\lambda_j^*\) 量化了约束对最优值的"影子价格"，在工程和经济决策 中指导资源分配。

📐 公式推导 / 模型建立

互补松弛：若强对偶成立且 \((x^*,\lambda^*,\nu^*)\) 分别为原-对偶最优，则 \[\lambda_j^* g_j(x^*) = 0, \quad \forall j\] 证明：\(f(x^*) = g(\lambda^*,\nu^*) = \inf_x \mathcal{L}(x,\lambda^*,\nu^*) \leq \mathcal{L}(x^*,\lambda^*,\nu^*) = f(x^*) + \sum \lambda_j^* g_j(x^*)\)。 因 \(\lambda_j^* \geq 0\), \(g_j(x^*) \leq 0\)，每项 \(\leq 0\)，但总和 \(= 0\)， 故每项 \(=0\)。 灵敏度解释：\(\lambda_j^* = -\partial p^*/\partial b_j\)，其中 \(b_j\) 是约束右端。 即放松约束 \(j\) 一个单位，最优值改善 \(\lambda_j^*\)。

🌐 实际应用

工厂资源配置：\(\lambda_j^*\) 告诉管理者哪种资源的边际价值最高， 应优先增加。

✏️ 例题分析与易错点

例题：\(\min -x\) s.t. \(x \leq 5\), \(x \leq 3\)。求乘子和灵敏度。 解答：\(x^*=3\)（第二约束紧）。\(\lambda_1^*=0\)（第一约束松弛）， \(\lambda_2^*=1\)。灵敏度：放松第二约束到 \(x \leq 3+\epsilon\)， 最优值从 \(-3\) 变为 \(-(3+\epsilon)\)，改善 \(\epsilon = \lambda_2^* \cdot \epsilon\)。 易错点：松弛的约束乘子一定为零，但紧约束的乘子不一定非零（退化情形）。

04SVM 的对偶形式与核技巧

🎯 为什么研究这个问题

支持向量机（SVM）是对偶理论最经典的应用：通过对偶化，高维原 问题变为仅依赖内积的对偶问题，从而允许"核技巧"——将数据隐 式映射到高维空间而无需显式计算。这一巧妙的理论-算法联系是机 器学习中数学最优美的范例之一。

🔍 抽象化方法

核心抽象：对偶形式中 \(x_i\) 仅通过内积 \(x_i^T x_j\) 出现， 可替换为核函数 \(k(x_i, x_j)\)，实现隐式高维映射。

📐 公式推导 / 模型建立

SVM 原问题：\(\min_{w,b} \frac{1}{2}\|w\|^2\) s.t. \(y_i(w^Tx_i+b) \geq 1\)。 Lagrange 函数：\(\mathcal{L} = \frac{1}{2}\|w\|^2 - \sum \alpha_i[y_i(w^Tx_i+b)-1]\)。 对 \(w,b\) 求驻点：\(w = \sum \alpha_i y_i x_i\), \(\sum \alpha_i y_i = 0\)。 对偶问题： \[\max_{\alpha \geq 0} \sum \alpha_i - \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j x_i^Tx_j\] s.t. \(\sum \alpha_i y_i = 0\)。 核技巧：将 \(x_i^T x_j\) 替换为 \(k(x_i,x_j) = \phi(x_i)^T\phi(x_j)\)， 如高斯核 \(k(x,y) = \exp(-\|x-y\|^2/(2\sigma^2))\)。

🌐 实际应用

核 SVM 在图像分类、文本分类和生物信息学中广泛使用， 核选择决定了特征空间的结构。

✏️ 例题分析与易错点

例题：2D 数据 \((1,0)\to +\), \((0,1)\to +\), \((-1,0)\to -\), \((0,-1)\to -\)。 原始空间线性不可分（XOR 变体），用核 \(k(x,y)=(\langle x,y \rangle+1)^2\)。 解答：特征映射 \(\phi(x) = (x_1^2, x_2^2, \sqrt{2}x_1 x_2, \sqrt{2}x_1, \sqrt{2}x_2, 1)\)。 在 6 维特征空间中数据变为线性可分，SVM 可找到分离超平面。 易错点：核必须是正定的（Mercer 条件），否则对偶问题非凸。
