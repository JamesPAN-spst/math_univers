约束优化与 KKT

01Lagrange 乘子法（等式约束）

🎯 为什么研究这个问题

大多数实际优化问题都有约束——预算限制、物理守恒律、资源容量。 等式约束优化是约束优化的起点：如何在曲面上找最值？Lagrange 乘 子法将约束问题转化为无约束的驻点方程组，几何直觉清晰：在极值 点处 \(\nabla f\) 平行于约束面的法方向。

🔍 抽象化方法

关键几何直觉：在约束面 \(\{h(x)=0\}\) 的极值点处，\(\nabla f\) 平行于 \(\nabla h\)——否则沿切方向仍可改善 \(f\)。

📐 公式推导 / 模型建立

等式约束问题：\(\min f(x)\) s.t. \(h_i(x) = 0\), \(i=1,\ldots,m\)。 Lagrange 函数： \[\mathcal{L}(x, \nu) = f(x) + \sum_{i=1}^m \nu_i h_i(x)\] 必要条件（正则点处）： \[\nabla_x \mathcal{L} = \nabla f + \sum \nu_i \nabla h_i = 0, \quad h_i(x) = 0\] 正则点条件（LICQ）：\(\nabla h_1, \ldots, \nabla h_m\) 在极值点处线性无关。 若不满足，Lagrange 必要条件可能失效。乘子 \(\nu_i\) 的含义： \(\nu_i = -\partial f^*/\partial b_i\)（约束右端值的灵敏度）。

🌐 实际应用

经济学效用最大化：在预算 \(p^Tx=B\) 下最大化 \(U(x)\)，Lagrange 乘子 \(\nu\) 是"边际预算的影子价格"。

✏️ 例题分析与易错点

例题：约束 \(x^2+y^2=1\) 下求 \(f(x,y)=x+y\) 的极值。 解答：\(\mathcal{L} = x+y+\nu(x^2+y^2-1)\)。 \(1+2\nu x=0\), \(1+2\nu y=0\) \(\Rightarrow\) \(x=y=-1/(2\nu)\)。 代入约束：\(2/(4\nu^2)=1\) \(\Rightarrow\) \(\nu=\pm 1/\sqrt{2}\)。 最大值 \(\sqrt{2}\)（\(x=y=1/\sqrt{2}\)），最小值 \(-\sqrt{2}\)。 易错点：忘记检查正则性条件。若约束梯度在候选点为零则方法可能失效。

02KKT 条件的完整框架

🎯 为什么研究这个问题

不等式约束是实际问题的常态——资源非负、生产上限、风险限额。 KKT 条件（Karush-Kuhn-Tucker）是处理不等式约束的核心工具，含 四组方程，其中互补松弛条件捕捉了"约束是否紧"的关键信息。KKT 是约束优化的统一框架，贯穿从线性规划到 SVM 的所有应用。

📐 公式推导 / 模型建立

问题：\(\min f(x)\) s.t. \(g_j(x) \leq 0\), \(h_i(x) = 0\)。 KKT 四组条件： (1) 梯度条件：\(\nabla f + \sum \lambda_j \nabla g_j + \sum \nu_i \nabla h_i = 0\) (2) 原始可行：\(g_j(x) \leq 0\), \(h_i(x) = 0\) (3) 对偶可行：\(\lambda_j \geq 0\) (4) 互补松弛： \[\lambda_j g_j(x) = 0, \quad \forall j\] 互补松弛含义：\(\lambda_j > 0 \Rightarrow g_j(x) = 0\)（约束紧）； \(g_j(x) < 0 \Rightarrow \lambda_j = 0\)（约束松弛则乘子为零）。

🌐 实际应用

SVM 中 KKT 使大部分样本 \(\lambda_i=0\)（非支持向量），仅约束紧 的样本（支持向量）\(\lambda_i>0\)，实现稀疏解。

✏️ 例题分析与易错点

例题：\(\min x^2\) s.t. \(x \geq 1\)（即 \(-x+1 \leq 0\)）。 解答：KKT：\(2x-\lambda=0\); \(-x+1 \leq 0\); \(\lambda \geq 0\); \(\lambda(-x+1)=0\)。若 \(\lambda=0\): \(x=0\) 不满足 \(x \geq 1\)。 若 \(-x+1=0\): \(x=1\), \(\lambda=2>0\) \(\checkmark\)。\(x^*=1\)。 易错点：互补松弛有两种情况须分别讨论，不可只检查一种。

03约束限定条件与 KKT 的必要/充分性

🎯 为什么研究这个问题

KKT 条件并非总是极值的必要条件——需要额外的约束限定条件(CQ)。 同时对凸问题，KKT 还是充分条件。理解何时 KKT 必要、何时充分， 是正确使用约束优化理论的关键，也是凸优化软件（CVX）的理论基础。

📐 公式推导 / 模型建立

约束限定条件（CQ）： - LICQ：活跃约束梯度线性无关 - Slater 条件（凸问题）：存在严格可行点 \(g_j(x)<0\) - MFCQ：Mangasarian-Fromovitz，较弱但更常用 必要性定理：\(x^*\) 局部极小 + CQ \(\Rightarrow\) 存在乘子使 KKT 成立： \[\nabla f(x^*) + \sum \lambda_j \nabla g_j(x^*) + \sum \nu_i \nabla h_i(x^*) = 0\] 充分性定理（凸问题）：\(f, g_j\) 凸，\(h_i\) 仿射，\((x^*,\lambda^*,\nu^*)\) 满足 KKT \(\Rightarrow\) \(x^*\) 全局最优。 证明：凸性保证 \(\mathcal{L}(\cdot,\lambda^*,\nu^*)\) 凸，KKT 给出 \(\nabla_x \mathcal{L}=0\) 即全局极小。

🌐 实际应用

凸优化求解器（CVXPY, Mosek）本质上在寻找 KKT 点，凸性保证 KKT 点即全局最优解。

✏️ 例题分析与易错点

例题：\(\min x^2\) s.t. \((x-1)^3 \leq 0\)。\(x^*=1\) 处 LICQ 是否满足？ 解答：\(g(x)=(x-1)^3\), \(g'(1)=0\)。活跃约束梯度为零，LICQ 不满足。 KKT 条件 \(2+\lambda \cdot 0=0\) 无解。CQ 失败导致 KKT 无法描述此最优点。 易错点：CQ 失败不代表点非最优，只表示 KKT 无法刻画该点。

04正则化的优化视角（LASSO 与 Ridge）

🎯 为什么研究这个问题

ML 中正则化可从约束优化理解：LASSO (\(L_1\)) 等价于 \(L_1\) 约束 优化，Ridge (\(L_2\)) 等价于 \(L_2\) 约束优化。这一视角将统计正则 化与 KKT 条件直接联系，解释了 LASSO 产生稀疏解的几何原因—— \(L_1\) 球的顶点在坐标轴上。

🔍 抽象化方法

正则化 \(\leftrightarrow\) 约束的等价性通过 Lagrange 对偶建立。

📐 公式推导 / 模型建立

Ridge：\(\min_w \|Xw-y\|^2 + \lambda\|w\|^2\)。KKT 闭式解： \[w^* = (X^TX + \lambda I)^{-1}X^Ty\] LASSO：\(\min_w \|Xw-y\|^2 + \lambda\|w\|_1\)。 等价约束：\(\min \|Xw-y\|^2\) s.t. \(\|w\|_1 \leq t\)。 \(L_1\) 约束域是菱形，极值倾向落在顶点（某些 \(w_i=0\)），产生稀疏解。 次梯度最优条件： \[2X^T(Xw-y) + \lambda \partial\|w\|_1 \ni 0\] 软阈值公式（正交设计）：\(w_j^* = \mathrm{sign}(\hat{w}_j)\max(|\hat{w}_j|-\lambda/2, 0)\)。

🌐 实际应用

LASSO 在基因组学中从上万基因中选出少量相关基因；Ridge 在多重 共线性数据中稳定估计。Elastic Net 结合两者优势。

✏️ 例题分析与易错点

例题：1D LASSO: \(\min_w (w-3)^2 + 2|w|\)。求 \(w^*\)。 解答：\(w>0\): \(2(w-3)+2=0 \Rightarrow w=2>0\) \(\checkmark\)。 \(w<0\): \(2(w-3)-2=0 \Rightarrow w=4\)，矛盾。 \(w=0\): 次梯度 \(-6+2s=0\), \(s=3 \notin [-1,1]\)，不满足。 \(\Rightarrow\) \(w^*=2\)，LASSO 将估计从 3 收缩到 2。 易错点：\(L_1\) 在零点不可微，必须用次梯度处理。
