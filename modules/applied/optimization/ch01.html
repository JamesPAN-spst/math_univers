凸集与凸函数

01凸集的定义与凸组合

🎯 为什么研究这个问题

在工程优化和经济学中，我们需要在约束集合上寻找最优方案。凸集 是最"友好"的约束——在凸集上优化凸函数，局部最优即全局最优， 这是优化理论的基石。凸集还出现在投资组合（概率单纯形）、控制 论（可达集）和机器学习（可行参数域）等核心领域。

🔍 抽象化方法

凸性的核心是"内插封闭性"：集合 \(C\) 是凸集当且仅当任意两点 的连线段完全在 \(C\) 内。这一几何直觉推广到凸组合 \(\sum \alpha_i x_i\)（\(\alpha_i \geq 0, \sum \alpha_i = 1\)）和凸包 \(\mathrm{conv}(S)\)。凸锥则结合了凸性与锥性。

📐 公式推导 / 模型建立

凸集定义：\(C \subseteq \mathbb{R}^n\) 是凸集 iff \[\forall x, y \in C,\; \forall \theta \in [0,1]:\; \theta x + (1-\theta) y \in C\] 凸组合：点 \(x_1, \ldots, x_k\) 的凸组合为 \(\sum_{i=1}^k \alpha_i x_i\)， 其中 \(\alpha_i \geq 0\), \(\sum \alpha_i = 1\)。凸包 \(\mathrm{conv}(S) = \{\sum \alpha_i x_i \mid x_i \in S,\, \alpha_i \geq 0,\, \sum \alpha_i = 1\}\) 是包含 \(S\) 的最小凸集。 凸锥：\(K\) 是凸锥 iff \(\forall x, y \in K,\, \forall \alpha, \beta \geq 0\): \(\alpha x + \beta y \in K\)。典型例子：半正定锥 \(S_+^n\)。 保凸运算：凸集的交仍是凸集；仿射映射保凸（\(f(C)\) 与 \(f^{-1}(C)\) 都保凸性）；投影保凸。

🌐 实际应用

在投资组合优化中，可行配置向量构成单纯形 \(\Delta_n = \{w \in \mathbb{R}^n \mid w_i \geq 0,\, \sum w_i = 1\}\)， 凸性保证了分散投资策略总是可行的。

✏️ 例题分析与易错点

例题：判断集合 \(S = \{(x,y) \in \mathbb{R}^2 \mid x^2 + y^2 \leq 1,\, x \geq 0\}\) 是否为凸集。 解答：\(S\) 是圆盘 \(\{x^2+y^2 \leq 1\}\)（凸集）与半平面 \(\{x \geq 0\}\)（凸集） 的交集。凸集的交仍凸，所以 \(S\) 是凸集。 易错点：不要混淆凸集和连通集。\(\{x \in \mathbb{R} \mid x^2 \geq 1\} = (-\infty,-1] \cup [1,+\infty)\) 不是凸集但每个连通分量是凸集。

02凸函数的定义与判定条件

🎯 为什么研究这个问题

凸函数是优化理论的核心对象。其最重要性质是"任何局部极小值都是 全局极小值"，使得凸最小化在理论和计算上都可控。掌握凸函数的多 种等价判定条件（零阶/一阶/二阶）是将实际问题建模为凸优化的关键。

🔍 抽象化方法

凸函数可通过上方图 \(\mathrm{epi}(f) = \{(x,t) \mid f(x) \leq t\}\) 理解 ——\(f\) 凸当且仅当 \(\mathrm{epi}(f)\) 是凸集，将函数性质转化为集合性质。

📐 公式推导 / 模型建立

零阶定义：\(f: C \to \mathbb{R}\) 是凸函数 iff \(\forall x,y \in C\), \(\forall \theta \in [0,1]\): \[f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y)\] 一阶条件（\(f\) 可微）：\(f\) 凸 iff \[f(y) \geq f(x) + \nabla f(x)^T(y - x)\] 几何含义：函数图像总在其切平面之上。 二阶条件（\(f\) 二次可微）：\(f\) 凸 iff \(\nabla^2 f(x) \succeq 0\) （Hessian 半正定）。严格凸的充分条件：\(\nabla^2 f(x) \succ 0\) （但非必要，如 \(f(x)=x^4\) 在 \(x=0\) 处 \(f''=0\) 仍严格凸）。

🌐 实际应用

在统计学中，指数族分布的负对数似然是凸函数，因此 MLE 有唯一解 可用凸优化高效求解。

✏️ 例题分析与易错点

例题：证明 \(f(x) = \log(1 + e^x)\)（softplus）是凸函数。 解答：\(f'(x) = \frac{e^x}{1+e^x} = \sigma(x)\)， \(f''(x) = \sigma(x)(1-\sigma(x))\)。因 \(0 < \sigma(x) < 1\)， 故 \(f''(x) > 0\)，函数严格凸。 易错点：\(\nabla^2 f \succ 0\) 是严格凸的充分非必要条件。 \(f(x)=x^4\) 在 \(x=0\) 处 \(f''(0)=0\)，但 \(f\) 仍严格凸。

03强凸函数与条件数

🎯 为什么研究这个问题

仅知函数凸还不够——我们需量化"有多凸"。强凸函数在最小点附近 提供二次下界，直接决定梯度下降的收敛速率。条件数 \(\kappa = L/\mu\) 是衡量优化问题"难度"的核心指标，贯穿算法设计与分析。

📐 公式推导 / 模型建立

强凸定义：\(f\) 是 \(\mu\)-强凸（\(\mu > 0\)）iff \[f(y) \geq f(x) + \nabla f(x)^T(y-x) + \frac{\mu}{2}\|y-x\|^2\] 等价：\(\nabla^2 f(x) \succeq \mu I\) 处处成立。 Lipschitz 梯度：\(\|\nabla f(x) - \nabla f(y)\| \leq L\|x-y\|\)， 等价 \(\nabla^2 f(x) \preceq LI\)。 条件数 \(\kappa = L/\mu\)：\(\kappa = 1\) 时等高线为圆，梯度下降一步到位； \(\kappa \gg 1\) 时等高线极扁，锯齿路径缓慢收敛。 梯度下降收敛： \[f(x_k) - f^* \leq \left(\frac{\kappa-1}{\kappa+1}\right)^{2k}(f(x_0)-f^*)\]

🌐 实际应用

\(L_2\) 正则化添加 \(\frac{\lambda}{2}\|w\|^2\) 使损失函数变为 \(\lambda\)-强凸， 条件数从可能无穷降至 \(L/\lambda\)，大幅加速训练。

✏️ 例题分析与易错点

例题：\(f(x) = \frac{1}{2}x^TAx\)，\(A\) 特征值 \(\lambda_1=1, \lambda_2=100\)， 求 \(\mu, L, \kappa\)。 解答：\(\mu = \lambda_{\min}(A) = 1\), \(L = \lambda_{\max}(A) = 100\), \(\kappa = 100\)。收敛因子 \((99/101)^2 \approx 0.96\)。 易错点：\(\mu\) 是所有点上 Hessian 最小特征值的下确界，不是某点值。

04Jensen 不等式与保凸运算

🎯 为什么研究这个问题

实际建模中，复杂函数的凸性不可能逐一从定义验证。保凸运算规则 提供了"组合积木"方法，从简单凸函数构造复杂凸函数。Jensen 不 等式则是凸性在概率论和信息论中的重要桥梁，是证明 KL 散度非负、 EM 算法收敛等关键结果的基础。

📐 公式推导 / 模型建立

Jensen 不等式：若 \(f\) 凸，\(X\) 是随机变量，则 \[f(\mathbb{E}[X]) \leq \mathbb{E}[f(X)]\] 直观：函数在平均值处的值 \(\leq\) 函数值的平均。 保凸运算： (1) 非负加权和：\(\alpha_1 f_1 + \alpha_2 f_2\)（\(\alpha_i \geq 0\)）保凸 (2) 仿射复合：\(f(Ax+b)\) 保凸 (3) 逐点上确界：\(f(x) = \sup_\alpha f_\alpha(x)\) 保凸 (4) 复合规则：\(h\) 凸递增且 \(g\) 凸 \(\Rightarrow\) \(h \circ g\) 凸 (5) 透视函数：\(f\) 凸 \(\Rightarrow\) \(g(x,t)=tf(x/t)\) 凸（\(t>0\)） 拟凸函数：\(f\) 拟凸 iff 下水平集 \(\{x \mid f(x) \leq \alpha\}\) 对所有 \(\alpha\) 是凸集。凸 \(\Rightarrow\) 拟凸，反之不成立。

🌐 实际应用

Jensen 不等式证明 KL 散度非负： \(D_{\mathrm{KL}}(P\|Q) = \mathbb{E}_P[\log(P/Q)] \geq -\log(\mathbb{E}_P[Q/P]) = 0\)。

✏️ 例题分析与易错点

例题：判断 \(f(x) = \max(x_1, \ldots, x_n)\) 是否凸。 解答：\(f(x) = \sup_i e_i^T x\) 是仿射函数的逐点上确界，故 \(f\) 凸。 易错点：逐点下确界不保凸。\(\inf_\alpha f_\alpha(x)\) 即使每个 \(f_\alpha\) 凸，结果也不一定凸。
