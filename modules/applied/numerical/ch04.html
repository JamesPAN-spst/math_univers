线性方程组的数值解

01Gauss 消元与 LU 分解

🎯 为什么研究这个问题

\(Ax = b\) 是科学计算中最频繁出现的问题——PDE 离散化、 最小二乘、优化子问题都归结为线性方程组。Gauss 消元法将 \(A\) 化为上三角矩阵后回代，LU 分解 \(A = LU\) 是其矩阵化表述。 \(O(n^3)\) 的成本对稠密矩阵是标准方法。

🔍 抽象化方法

Gauss 消元 = 左乘初等矩阵 \(L_k\) 使 \(A\) 上三角化。 \(L_n\cdots L_1 A = U\) → \(A = (L_1^{-1}\cdots L_n^{-1})U = LU\)。 每个 \(L_k^{-1}\) 是下三角单位矩阵，乘积仍是下三角（结构保持）。

📐 公式推导 / 模型建立

Gauss 消元（无选主元版）： 第 \(k\) 步：\(a_{ij}^{(k+1)} = a_{ij}^{(k)} - \frac{a_{ik}^{(k)}}{a_{kk}^{(k)}} a_{kj}^{(k)}\)，\(i > k\)。 乘数 \(l_{ik} = a_{ik}^{(k)}/a_{kk}^{(k)}\) 存入 \(L\) 的下三角部分。 LU 分解的存在条件：\(A\) 的所有顺序主子式不为零（\(\det A_{1:k,1:k} \neq 0\)）。 实际使用部分选主元（partial pivoting）：每步选取 \(|a_{ik}^{(k)}|\) 最大的行交换到第 \(k\) 行。 结果 \(PA = LU\)（\(P\) 是置换矩阵）。 复杂度：消元 \(\frac{2}{3}n^3 + O(n^2)\) flops，回代 \(n^2\) flops。 后向稳定性：Gauss 消元 + 部分选主元是后向稳定的（Wilkinson 定理）， \(\hat{x}\) 满足 \((A + \Delta A)\hat{x} = b\)，\(\|\Delta A\| \leq f(n)\epsilon_{\text{mach}}\|A\|\)。

🌐 实际应用

- LAPACK/BLAS 中的 `dgesv` 是 Gauss 消元 + 部分选主元的工业级实现 - 有限元/有限差分的稠密子问题用直接法求解

✏️ 例题分析与易错点

例题：对 \(A = \begin{pmatrix}2&1&1\\4&3&3\\8&7&9\end{pmatrix}\) 做 LU 分解。 解答：\(l_{21} = 4/2 = 2\)，\(l_{31} = 8/2 = 4\)。 第一步消元后 \(A^{(2)} = \begin{pmatrix}2&1&1\\0&1&1\\0&3&5\end{pmatrix}\)。 \(l_{32} = 3/1 = 3\)。\(U = \begin{pmatrix}2&1&1\\0&1&1\\0&0&2\end{pmatrix}\)。 \(L = \begin{pmatrix}1&0&0\\2&1&0\\4&3&1\end{pmatrix}\)。验证 \(LU = A\)。✓ 易错点：如果主元 \(a_{kk}^{(k)} = 0\)（或很小），不选主元的 Gauss 消元要么失败 要么数值不稳定。必须使用部分选主元。

02Cholesky 分解与对称正定系统

🎯 为什么研究这个问题

对称正定矩阵（SPD）在物理和工程中无处不在（刚度矩阵、 协方差矩阵、正规方程）。Cholesky 分解 \(A = LL^T\) 利用了 对称正定性，计算量仅为 LU 的一半，且无需选主元天然稳定。 它是 SPD 系统的首选方法。

🔍 抽象化方法

Cholesky 分解是「矩阵平方根」的三角化版本。\(L\) 的存在 本身就是 \(A\) 正定的判据——如果 Cholesky 分解过程中出现 非正对角元素，说明 \(A\) 不正定。

📐 公式推导 / 模型建立

Cholesky 分解：对 SPD 矩阵 \(A\)，存在唯一的下三角 \(L\)（对角元 \(>0\)）使 \(A = LL^T\)。 推导（按列/按行递推）： \(a_{jj} = l_{j1}^2 + \cdots + l_{jj}^2\) → \(l_{jj} = \sqrt{a_{jj} - \sum_{k=1}^{j-1}l_{jk}^2}\)（正定保证根号内 \(>0\)）。 \(a_{ij} = \sum_{k=1}^j l_{ik}l_{jk}\) → \(l_{ij} = (a_{ij}-\sum_{k=1}^{j-1}l_{ik}l_{jk})/l_{jj}\)（\(i > j\)）。 复杂度：\(\frac{1}{3}n^3 + O(n^2)\) flops（LU 的一半）。 稳定性：无需选主元，因为 \(|l_{ij}| \leq l_{ii} \leq \sqrt{a_{ii}}\)（增长因子为 1）。 不完全 Cholesky（预条件器）：\(A \approx \tilde{L}\tilde{L}^T\)， 丢弃某些非零元素使 \(\tilde{L}\) 稀疏，用作迭代法的预条件器。

🌐 实际应用

- 最小二乘正规方程 \(A^TAx = A^Tb\)（\(A^TA\) 是 SPD） - 统计中多元正态分布的采样 \(x = L\xi\)（\(\xi \sim N(0,I)\)）

✏️ 例题分析与易错点

例题：对 \(A = \begin{pmatrix}4&2\\2&5\end{pmatrix}\) 做 Cholesky 分解。 解答：\(l_{11} = \sqrt{4} = 2\)，\(l_{21} = 2/2 = 1\)， \(l_{22} = \sqrt{5-1^2} = 2\)。 \(L = \begin{pmatrix}2&0\\1&2\end{pmatrix}\)，\(LL^T = \begin{pmatrix}4&2\\2&5\end{pmatrix} = A\)。✓ 易错点：如果在计算中遇到 \(l_{jj}^2 < 0\)（平方根内为负）， 说明矩阵不正定，Cholesky 分解不存在。这可作为正定性的数值判据。

03迭代法——Jacobi、Gauss-Seidel、SOR

🎯 为什么研究这个问题

大型稀疏矩阵（如 PDE 离散化产生的 \(n \gg 10^4\) 的系统） 用直接法（\(O(n^3)\)）太昂贵。迭代法从初始猜测出发逐步逼近 精确解，每步只需 \(O(\text{nnz})\)（非零元素数）运算。Jacobi、 Gauss-Seidel 和 SOR 是经典的定常迭代法。

🔍 抽象化方法

迭代法将 \(Ax = b\) 化为不动点迭代 \(x^{(k+1)} = Bx^{(k)} + c\)。 谱半径 \(\rho(B) < 1\) 是收敛的充要条件——控制谱半径就是设计 迭代法的核心目标。

📐 公式推导 / 模型建立

矩阵分裂框架：\(A = M - N\)，\(Mx^{(k+1)} = Nx^{(k)} + b\)。 迭代矩阵 \(B = M^{-1}N\)，收敛 \(\iff \rho(B) < 1\)。 Jacobi 迭代：\(M = D\)（对角），\(x_i^{(k+1)} = \frac{1}{a_{ii}}(b_i - \sum_{j\neq i}a_{ij}x_j^{(k)})\)。 Gauss-Seidel 迭代：\(M = D + L\)（下三角），使用最新值： \(x_i^{(k+1)} = \frac{1}{a_{ii}}(b_i - \sum_{ji}a_{ij}x_j^{(k)})\)。 SOR（逐次超松弛）：引入参数 \(\omega\)： \[x^{(k+1)} = (1-\omega)x^{(k)} + \omega x^{(k+1)}_{\text{GS}}\] \(\omega \in (1,2)\) 超松弛加速，\(\omega = 1\) 为 Gauss-Seidel。 收敛判据（充分条件）：\(A\) 严格对角占优 \(|a_{ii}| > \sum_{j\neq i}|a_{ij}|\) → Jacobi 和 Gauss-Seidel 均收敛。\(A\) SPD → Gauss-Seidel 收敛。

🌐 实际应用

- Laplace 方程的五点差分 → Gauss-Seidel/SOR 是经典求解器 - 图像处理中的平滑滤波可用 Jacobi 迭代实现

✏️ 例题分析与易错点

例题：对 \(\begin{pmatrix}4&1\\1&3\end{pmatrix}x = \begin{pmatrix}5\\5\end{pmatrix}\) 做 2 步 Jacobi 迭代（\(x^{(0)}=(0,0)^T\)）。 解答：step 1：\(x_1^{(1)} = (5-0)/4 = 1.25\)，\(x_2^{(1)} = (5-0)/3 = 1.667\)。 step 2：\(x_1^{(2)} = (5-1.667)/4 = 0.833\)，\(x_2^{(2)} = (5-1.25)/3 = 1.25\)。 精确解 \((10/11, 15/11) \approx (0.909, 1.364)\)。收敛中。 易错点：SOR 的 \(\omega\) 选取至关重要。\(\omega\) 太大（\(>2\)）必定发散， \(\omega\) 太小收敛慢。最优 \(\omega\) 取决于 \(A\) 的谱结构。

04Krylov 子空间方法——共轭梯度与 GMRES

🎯 为什么研究这个问题

经典迭代法收敛慢（\(\rho(B)\) 接近 1 时需要很多步）。Krylov 子空间 方法在 \(\text{span}\{r_0, Ar_0, A^2r_0, \ldots\}\) 中寻找最优近似解， 收敛速度由 \(A\) 的特征值分布（而非谱半径）决定——配合预条件后， 通常远快于定常迭代。CG 和 GMRES 是现代大规模线性方程组的标准方法。

🔍 抽象化方法

Krylov 方法在逐步增长的子空间中最优化残差或误差。 共轭梯度 = SPD 矩阵上的能量范数最优化（等价于二次规划）。 GMRES = 一般矩阵上的残差 \(L^2\) 范数最小化。

📐 公式推导 / 模型建立

共轭梯度法（CG）（\(A\) SPD）： 在 \(x_0 + \mathcal{K}_k(A,r_0)\) 中找 \(x_k\) 使能量误差 \(\|x-x_k\|_A\) 最小。 迭代： \(r_k = b - Ax_k\)（残差），\(p_k\)（搜索方向，与之前所有 \(p_j\) 关于 \(A\) 共轭）。 \[\alpha_k = \frac{r_k^T r_k}{p_k^T A p_k}, \quad x_{k+1} = x_k + \alpha_k p_k\] \[r_{k+1} = r_k - \alpha_k Ap_k, \quad \beta_k = \frac{r_{k+1}^T r_{k+1}}{r_k^T r_k}\] \[p_{k+1} = r_{k+1} + \beta_k p_k\] 每步仅需一次矩阵-向量乘积 \(Ap_k\)（\(O(\text{nnz})\)）。 收敛速率：\(\|x-x_k\|_A \leq 2\left(\frac{\sqrt{\kappa}-1}{\sqrt{\kappa}+1}\right)^k\|x-x_0\|_A\)， 其中 \(\kappa = \kappa(A)\)。条件数越小收敛越快 → 预条件的重要性。 GMRES（一般矩阵）：在 \(\mathcal{K}_k\) 中最小化 \(\|b-Ax\|_2\)， 用 Arnoldi 过程构造正交基。通用但每步存储增长。

🌐 实际应用

- 有限元求解的核心线性代数步骤几乎都用预条件 CG 或 GMRES - 机器学习中大规模优化的 CG 变体

✏️ 例题分析与易错点

例题：对 \(A = \begin{pmatrix}2&-1\\-1&2\end{pmatrix}\)，\(b = (1,0)^T\) 做 CG 第一步。 解答：\(x_0 = (0,0)^T\)，\(r_0 = b = (1,0)^T\)，\(p_0 = r_0 = (1,0)^T\)。 \(Ap_0 = (2,-1)^T\)，\(\alpha_0 = r_0^Tr_0/(p_0^TAp_0) = 1/2\)。 \(x_1 = (0.5, 0)^T\)，\(r_1 = (1,0)-(1/2)(2,-1) = (0, 0.5)^T\)。 \(\beta_0 = 0.25/1 = 0.25\)，\(p_1 = (0,0.5) + 0.25(1,0) = (0.25, 0.5)^T\)。 易错点：CG 仅适用于 SPD 矩阵。对非对称或不定矩阵必须用 GMRES 或 BiCGSTAB。 对 SPD 矩阵误用 GMRES 会浪费存储。
