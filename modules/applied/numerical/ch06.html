PDE 数值方法

01有限差分法

🎯 为什么研究这个问题

将 PDE 中的微分算子用差分近似，在网格上将 PDE 转化为 代数方程组——这是最直观的 PDE 数值方法。五点差分格式 解 Laplace 方程、FTCS/BTCS 解热方程、Lax-Wendroff 解波方程 都是经典模板。有限差分法的理论框架（一致性+稳定性→收敛） 是理解所有网格方法的基础。

🔍 抽象化方法

有限差分在均匀网格上用 Taylor 展开推导差分格式。 Lax 等价定理是核心：对线性问题，一致性 + 稳定性 \(\iff\) 收敛。 von Neumann 稳定性分析用 Fourier 模式检测格式稳定性。

📐 公式推导 / 模型建立

空间差分（二阶中心差分）： \[u_{xx} \approx \frac{u_{j+1}-2u_j+u_{j-1}}{(\Delta x)^2} + O(\Delta x^2)\] 热方程 FTCS（前向时间中心空间）：\(u_t = \kappa u_{xx}\)。 \[u_j^{n+1} = u_j^n + r(u_{j+1}^n - 2u_j^n + u_{j-1}^n), \quad r = \kappa\Delta t/\Delta x^2\] von Neumann 分析：令 \(u_j^n = g^n e^{ij\xi}\)，代入得 \(g = 1 - 4r\sin^2(\xi/2)\)。\(|g| \leq 1\) 要求 \(r \leq 1/2\)。 即 \(\Delta t \leq \Delta x^2/(2\kappa)\)（CFL 条件）。 隐式 BTCS（后向时间中心空间）： \(u_j^{n+1} - r(u_{j+1}^{n+1}-2u_j^{n+1}+u_{j-1}^{n+1}) = u_j^n\)。 \(|g| = 1/(1+4r\sin^2(\xi/2)) \leq 1\)（对所有 \(r > 0\) 成立）→ 无条件稳定。 Crank-Nicolson（\(\theta = 1/2\)）：梯形法时间积分，\(O(\Delta t^2 + \Delta x^2)\)，无条件稳定。 Lax 等价定理：线性初值问题中，一致 + 稳定 \(\iff\) 收敛。

🌐 实际应用

- 天气预报中的有限差分模型 - 地球物理中的波动方程正演模拟

✏️ 例题分析与易错点

例题：验证热方程 FTCS 格式在 \(r = 1\)（\(\Delta t = \Delta x^2/\kappa\)）时不稳定。 解答：\(g = 1 - 4\sin^2(\xi/2)\)。当 \(\xi = \pi\) 时，\(g = 1 - 4 = -3\)，\(|g| = 3 > 1\)。 数值解将以因子 3 逐步振荡放大。实际计算中看到交替正负且幅度指数增长。 易错点：CFL 条件 \(r \leq 1/2\) 关系到 \(\Delta t\) 和 \(\Delta x^2\)（不是 \(\Delta x\)！）。 网格加密 \(\Delta x/2\) 时，\(\Delta t\) 要除以 4，计算量增大 8 倍。

02有限元方法基础

🎯 为什么研究这个问题

有限差分法在规则网格上简单高效，但难以处理复杂几何域和 自然边界条件。有限元方法（FEM）基于弱形式，使用不规则的 三角形/四面体网格逼近复杂域，是工程结构分析的标准工具。 FEM 有坚实的数学理论（Céa 引理、先验误差估计）保证收敛。

🔍 抽象化方法

FEM = Galerkin 方法 + 分段多项式试函数空间。弱形式 \(a(u,v) = L(v)\) 在有限维子空间 \(V_h \subset V\) 上求解， Céa 引理保证 \(\|u-u_h\| \leq C\inf_{v_h\in V_h}\|u-v_h\|\)（准最优逼近）。

📐 公式推导 / 模型建立

Galerkin 方法：弱形式 \(a(u,v) = L(v)\)（\(\forall v \in V\)）， 在有限维子空间 \(V_h\) 上：找 \(u_h \in V_h\) 使 \(a(u_h, v_h) = L(v_h)\)，\(\forall v_h \in V_h\)。 选基 \(\{\phi_1,\ldots,\phi_N\}\)，\(u_h = \sum c_j\phi_j\)，得线性系统 \[Kc = F, \quad K_{ij} = a(\phi_j, \phi_i), \; F_i = L(\phi_i)\] \(K\) 是刚度矩阵（对称正定当 \(a\) 对称强制时）。 分段线性有限元（\(P_1\) 元）：三角形网格上，每个基函数 \(\phi_i\) 在节点 \(i\) 取值 1，其余节点取值 0，各三角形内线性。\(K\) 稀疏。 Céa 引理：\(\|u-u_h\|_V \leq \frac{M}{\alpha}\inf_{v_h\in V_h}\|u-v_h\|_V\)。 结合插值估计 \(\inf\|u-v_h\| \leq Ch^k\|u\|_{H^{k+1}}\)（\(k\)=多项式次数）， 得先验误差估计： \[\|u-u_h\|_{H^1} \leq Ch^k\|u\|_{H^{k+1}}\]

🌐 实际应用

- 结构力学中的应力分析（桥梁、飞机、汽车） - 半导体器件模拟（drift-diffusion 方程）

✏️ 例题分析与易错点

例题：一维 \(-u'' = 1\)（\(u(0)=u(1)=0\)）用 2 个等距 \(P_1\) 元求解。 解答：节点 \(x_0=0, x_1=0.5, x_2=1\)，内部自由度仅 \(x_1\)。 \(\phi_1(x)\) 是以 \(x_1=0.5\) 为顶点的帽函数。 \(K_{11} = \int_0^1|\phi_1'|^2 = 2\int_0^{0.5}(2)^2 = 4\)，\(F_1 = \int_0^1\phi_1 = 0.5\)。 \(u_h(0.5) = F_1/K_{11} = 0.5/4 = 0.125\)。精确解 \(u(0.5) = 0.5\times0.5/2 = 0.125\)。精确！ 易错点：\(P_1\) 元对二次解可以精确还原（因为 \(u = x(1-x)/2\) 在每个单元内是二次， 但节点上恰好精确）。一般情况下不会如此巧合。

03有限体积法与守恒律

🎯 为什么研究这个问题

守恒律（如 Euler 方程、浅水方程）的解可能产生激波（间断面）。 有限差分法在间断处失效（Gibbs 振荡），有限元法需要特殊处理。 有限体积法（FVM）直接离散积分形式的守恒律，天然保证通量守恒， 是计算流体力学（CFD）的标准方法。

🔍 抽象化方法

FVM 在每个控制体 \(C_j\) 上积分守恒律 \(u_t + \text{div}(F) = 0\)， 将体积分化为面通量。关键是数值通量 \(\hat{F}_{j+1/2}\) 的设计： 它要满足一致性、守恒性和某种耗散（避免非物理振荡）。

📐 公式推导 / 模型建立

一维守恒律：\(u_t + f(u)_x = 0\)。 在 \([x_{j-1/2}, x_{j+1/2}]\) 上积分： \[\frac{d\bar{u}_j}{dt} = -\frac{f(u(x_{j+1/2})) - f(u(x_{j-1/2}))}{\Delta x}\] \(\bar{u}_j = \frac{1}{\Delta x}\int_{x_{j-1/2}}^{x_{j+1/2}}u\,dx\)（单元平均值）。 Godunov 格式：\(\hat{f}_{j+1/2} = f(u^*_{j+1/2})\)，\(u^*\) 是 Riemann 问题 \(u_L = u_j\)，\(u_R = u_{j+1}\) 的精确解在 \(x/t = 0\) 处的值。 精确但昂贵 → 用近似 Riemann 求解器（如 Roe、HLL）。 Lax-Friedrichs 格式：\(\hat{f}_{j+1/2} = \frac{1}{2}[f(u_j)+f(u_{j+1})] - \frac{\Delta x}{2\Delta t}(u_{j+1}-u_j)\)。 简单但耗散大（1 阶）。 TVD/WENO 高阶重构：用限制器或加权基本无振荡（WENO） 插值实现高阶精度同时抑制间断处振荡。 CFL 条件（双曲型）：\(\Delta t \leq \Delta x/\max|f'(u)|\)（信息传播速度限制）。

🌐 实际应用

- 航空航天中的可压缩流动模拟（跨音速绕流、激波管） - 天体物理中的磁流体力学（MHD）模拟

✏️ 例题分析与易错点

例题：对 Burgers 方程 \(u_t + (u^2/2)_x = 0\) 用 Lax-Friedrichs 格式。 初值 \(u(x,0) = \sin(\pi x)\)，\(\Delta x = 0.1\)，\(\Delta t = 0.05\)。 解答：\(\hat{f}_{j+1/2} = \frac{1}{2}(u_j^2/2+u_{j+1}^2/2)-\frac{0.1}{0.1}(u_{j+1}-u_j)/2\) \(= \frac{u_j^2+u_{j+1}^2}{4} - (u_{j+1}-u_j)\)。 然后 \(u_j^{n+1} = u_j^n - (\Delta t/\Delta x)(\hat{f}_{j+1/2}-\hat{f}_{j-1/2})\)。 CFL 数 \(\nu = \max|u|\cdot\Delta t/\Delta x = 1\times0.05/0.1 = 0.5 \leq 1\)。✓ 易错点：守恒格式（\(\hat{f}_{j+1/2}\) 形式）保证 Rankine-Hugoniot 条件 在离散层面自动满足。非守恒格式可能收敛到错误的弱解（Hou-LeFloch 定理）。

04谱方法与有限元高级话题

🎯 为什么研究这个问题

有限差分和低阶有限元的精度受限于多项式次数（\(O(h^k)\)）。 当解光滑时，谱方法使用全局高阶多项式（Fourier 或 Chebyshev）， 达到指数收敛（谱精度）——每增加一个自由度，误差下降一个数量级。 这是高精度科学计算的首选方法。

🔍 抽象化方法

谱方法 = Galerkin/配置方法 + 全局基函数。收敛速率由解的 光滑性决定：解析函数 → 指数收敛，\(C^k\) 函数 → \(O(N^{-k})\) 代数收敛。 hp-FEM 则结合网格细化（\(h\)）和多项式升阶（\(p\)），兼顾灵活性和精度。

📐 公式推导 / 模型建立

Fourier 谱方法（周期问题）：展开 \(u_N = \sum_{|k|\leq N/2}\hat{u}_k e^{ikx}\)。 对热方程 \(u_t = \kappa u_{xx}\)：\(\hat{u}_k' = -\kappa k^2\hat{u}_k\)（精确的模态方程！）。 空间误差由截断 Fourier 级数的尾部决定： \[\|u - u_N\| \leq C_s N^{-s}\|u\|_{H^s} \quad (\forall s, \text{if } u \in C^\infty)\] 即对 \(C^\infty\) 函数，收敛比任何 \(N^{-s}\) 都快（= 指数收敛）。 Chebyshev 谱方法（非周期问题）：在 Chebyshev-Gauss-Lobatto 点上配置。 微分矩阵 \(D_N\)：\((D_N u)_j = \sum_k D_{jk}u_k\) 离散化 \(d/dx\)。 稠密矩阵但可用 FFT 加速为 \(O(N\log N)\)。 hp-有限元：在解光滑区域用高阶多项式（\(p\) 方法）， 在奇异/间断附近加密网格（\(h\) 方法）。理论预测对角点奇异性 也能达到指数收敛（Babuška-Guo 定理）。 误差类型对比： - 有限差分 \(O(h^p)\)（固定 \(p\)，代数收敛） - 有限元 \(P_k\)：\(O(h^k)\) - 谱方法：\(O(e^{-cN})\)（指数收敛，\(c\) 取决于解的解析性）

🌐 实际应用

- 全球气候模型中的球谐谱方法 - DNS（直接数值模拟）湍流使用 Fourier 谱方法

✏️ 例题分析与易错点

例题：用 Fourier 谱方法求解 \(-u'' = f(x)\)（周期 \(2\pi\)），\(f = \sin 3x\)。 解答：\(\hat{f}_k = \delta_{k,3}/(2i)\)，\(\hat{u}_k = \hat{f}_k/k^2\)。 \(\hat{u}_3 = 1/(9\cdot2i)\)，\(u = \sin(3x)/9\)。 谱方法对这类问题可以给出精确解（有限个 Fourier 模式的源项）。 更一般地，\(f\) 有无穷多个 Fourier 模式时，谱方法截取 \(|k| \leq N/2\)， 高频尾部是误差。若 \(|\hat{f}_k|\) 指数衰减，则 \(N\) 稍大即可高精度。 易错点：谱方法对非光滑解（激波、间断）会产生 Gibbs 现象 （与高次插值的 Runge 现象类似），需要滤波或去混叠处理。
