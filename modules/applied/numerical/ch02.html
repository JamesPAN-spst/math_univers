01Lagrange 插值与 Newton 插值

🎯 为什么研究这个问题

给定 \(n+1\) 个数据点 \((x_i, y_i)\)，如何找到过这些点的多项式？ 多项式插值是数据拟合、数值微分、数值积分的基础工具。 Lagrange 形式直接写出答案，Newton 形式便于递推添加新点。 理解插值误差是控制精度的关键。

🔍 抽象化方法

插值是在多项式空间 \(\mathcal{P}_n\) 中求解线性方程组 \(p(x_i) = y_i\)。Vandermonde 矩阵的非奇异性（节点互不相同时） 保证了 \(n\) 次插值多项式的存在唯一性。

📐 公式推导 / 模型建立

Lagrange 插值公式：\(p(x) = \sum_{i=0}^n y_i L_i(x)\)，其中 \[L_i(x) = \prod_{j \neq i}\frac{x - x_j}{x_i - x_j}\] \(L_i(x_j) = \delta_{ij}\)，保证精确过点。 Newton 差商形式：\(p(x) = f[x_0] + f[x_0,x_1](x-x_0) + \cdots + f[x_0,\ldots,x_n]\prod_{k=0}^{n-1}(x-x_k)\)。 差商递推：\(f[x_i,\ldots,x_{i+k}] = \frac{f[x_{i+1},\ldots,x_{i+k}]-f[x_i,\ldots,x_{i+k-1}]}{x_{i+k}-x_i}\)。 插值误差：设 \(f \in C^{n+1}\)，则 \[f(x) - p(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}\prod_{i=0}^n(x-x_i) = \frac{f^{(n+1)}(\xi)}{(n+1)!}\omega_{n+1}(x)\] 其中 \(\xi \in [\min x_i, \max x_i]\)。误差由节点多项式 \(\omega_{n+1}(x)\) 和导数上界控制。

🌐 实际应用

- 数值积分公式（Newton-Cotes）是先插值再积分 - 有限差分格式的推导基于局部多项式插值

✏️ 例题分析与易错点

例题：用 Lagrange 插值求过 \((0,1),(1,3),(2,7)\) 的二次多项式。 解答：\(L_0 = \frac{(x-1)(x-2)}{(0-1)(0-2)} = \frac{(x-1)(x-2)}{2}\)， \(L_1 = \frac{x(x-2)}{1\cdot(-1)} = -x(x-2)\)， \(L_2 = \frac{x(x-1)}{2\cdot1} = \frac{x(x-1)}{2}\)。 \(p(x) = 1\cdot L_0 + 3\cdot L_1 + 7\cdot L_2 = x^2 + x + 1\)。 验证：\(p(0)=1\)，\(p(1)=3\)，\(p(2)=7\)。✓ 易错点：高次多项式插值可能出现 Runge 现象（节点间剧烈振荡）， 等距节点尤其危险。应使用 Chebyshev 节点或分段插值。

02Runge 现象与 Chebyshev 节点

🎯 为什么研究这个问题

等距节点高次插值的 Runge 现象（端点附近剧烈震荡）表明 「更多数据不一定更精确」。Chebyshev 节点是节点多项式 \(|\omega_{n+1}(x)|\) 最大值最小化的最优选择，从根本上消除 Runge 现象。这是逼近论中 minimax 思想的经典应用。

🔍 抽象化方法

Chebyshev 节点是 \([-1,1]\) 上 Chebyshev 多项式 \(T_{n+1}(x)\) 的 零点。\(T_{n+1}\) 在所有首一 \(n+1\) 次多项式中 \(\|p\|_\infty\) 最小—— 这是等振定理（equioscillation theorem）的推论。

📐 公式推导 / 模型建立

Runge 现象：\(f(x) = 1/(1+25x^2)\) 在 \([-1,1]\) 上等距插值， 误差 \(\max|f-p_n| \to \infty\)（\(n \to \infty\)）。原因：\(\omega_{n+1}\) 在端点附近 呈指数增长。 Chebyshev 节点：\(x_k = \cos\frac{(2k+1)\pi}{2(n+1)}\)，\(k=0,1,\ldots,n\)。 它们是 \(T_{n+1}(x) = \cos((n+1)\arccos x)\) 的零点。 最优性定理：在所有首一 \(n+1\) 次多项式 \(p\) 中， \[\min_p \max_{x\in[-1,1]}|p(x)| = \frac{1}{2^n} = \max_{x}|T_{n+1}(x)/2^n|\] 用 Chebyshev 节点时 \(\|\omega_{n+1}\|_\infty = 2^{-n}\)， 等距节点时 \(\|\omega_{n+1}\|_\infty\) 随 \(n\) 指数增长。 误差估计：\(\|f-p_n\|_\infty \leq \frac{\|f^{(n+1)}\|_\infty}{2^n(n+1)!}\)（Chebyshev 节点）。

🌐 实际应用

- 谱方法（Chebyshev 谱方法）使用 Chebyshev 节点作为配置点 - 数值积分中的 Clenshaw-Curtis 求积公式基于 Chebyshev 插值

✏️ 例题分析与易错点

例题：对 \(f(x) = 1/(1+25x^2)\)，比较 \(n = 10\) 时等距节点和 Chebyshev 节点插值的最大误差（估计量级）。 解答：等距节点：Runge 现象导致端点附近误差 \(\approx 1\)（100% 相对误差）。 Chebyshev 节点：\(\|f^{(11)}\| \lesssim C \cdot 25^{5.5}\)， \(\|\omega_{12}\| = 2^{-11}\)，误差 \(\approx 10^{-2}\)（好 2 个数量级）。 易错点：Chebyshev 节点必须映射到计算区间 \([a,b]\)： \(\tilde{x}_k = \frac{a+b}{2} + \frac{b-a}{2}x_k\)。忘记映射会导致节点出界。

03分段插值与样条

🎯 为什么研究这个问题

避免 Runge 现象的实用方法是分段低次插值。三次样条 （cubic spline）兼顾光滑性（\(C^2\) 连续）和低次多项式的稳定性， 是工程和计算机图形学中最广泛使用的插值工具。

🔍 抽象化方法

样条函数是分段多项式的全局优化：在所有 \(C^2\) 插值函数中， 自然三次样条使弯曲能量 \(\int|f''|^2\) 最小——它是变分问题的解。

📐 公式推导 / 模型建立

分段线性插值：每两个相邻节点间用直线连接。 \(C^0\) 连续，\(O(h^2)\) 逼近（\(h\) = 最大节点间距）。 三次样条：在每段 \([x_i, x_{i+1}]\) 上 \(S(x)\) 是三次多项式（\(4n\) 个未知数）， 约束： - 插值条件：\(S(x_i) = y_i\)（\(n+1\) 个） - \(C^2\) 连续：\(S\)，\(S'\)，\(S''\) 在内节点连续（\(3(n-1)\) 个） - 总条件 \(n+1+3(n-1) = 4n-2\)，还差 2 个 → 端点条件 自然样条：\(S''(x_0) = S''(x_n) = 0\)。 箝位样条：\(S'(x_0) = f'(x_0)\)，\(S'(x_n) = f'(x_n)\)。 最优性定理：在所有 \(C^2\) 插值函数中，自然三次样条使 \[\int_a^b |f''(x)|^2\,dx\] 最小。证明：\(\int(f''-S'')^2 = \int(f'')^2 - \int(S'')^2 \geq 0\)（利用分部积分和插值条件）。

🌐 实际应用

- CAD/CAM 中的曲线曲面设计（B 样条，NURBS） - 动画中的关键帧插值

✏️ 例题分析与易错点

例题：对数据 \((0,0),(1,1),(2,0)\) 构造自然三次样条。 解答：两段 \(S_0\)（\([0,1]\)），\(S_1\)（\([1,2]\)）。 设 \(M_i = S''(x_i)\)，自然条件 \(M_0 = M_2 = 0\)。 \(C^2\) 连续给出方程 \(M_0 + 4M_1 + M_2 = 6[(y_2-y_1)-(y_1-y_0)] = 6(-1-1) = -12\)。 \(M_1 = -3\)。 \(S_0(x) = 0 + (1+3/6)x + 0 - (3/6)x^3 = \frac{3}{2}x - \frac{1}{2}x^3\)。 验证：\(S_0(0) = 0\)，\(S_0(1) = 3/2-1/2 = 1\)。✓ 易错点：三次样条的端点条件（自然 vs 箝位 vs 周期）显著影响结果精度。 箝位样条（已知端点导数）精度为 \(O(h^4)\)，自然样条降为 \(O(h^2)\) 在端点附近。

04最佳逼近与最小二乘

🎯 为什么研究这个问题

插值要求过所有数据点，但数据有噪声时这不合理。最小二乘 拟合允许残差，找到最优的逼近函数。从最佳一致逼近（\(L^\infty\)） 到最佳均方逼近（\(L^2\)），不同范数下的最佳逼近理论给出了 逼近能力的理论极限。

🔍 抽象化方法

最佳逼近是 Hilbert 空间（\(L^2\)）或一般赋范空间中的 正交投影问题。正规方程 \(A^TAx = A^Tb\) 是投影的矩阵表示。

📐 公式推导 / 模型建立

最小二乘问题：给定 \(m\) 个数据点和 \(n\) 个基函数（\(m > n\)）， 找 \(c = (c_1,\ldots,c_n)^T\) 使 \(\|Ac - b\|_2^2\) 最小。 \(A_{ij} = \phi_j(x_i)\)，\(b_i = y_i\)。 正规方程：\(\nabla\|Ac-b\|^2 = 2A^T(Ac-b) = 0\)，即 \[A^TAc = A^Tb\] \(A^TA\) 对称正定（当 \(A\) 列满秩时）→ 唯一解 \(c = (A^TA)^{-1}A^Tb\)。 连续版本（\(L^2\) 最佳逼近）：在子空间 \(V_n \subset L^2\) 中找 \(p\) 使 \(\|f - p\|_{L^2}\) 最小。解是正交投影： \[\langle f - p, \phi_k\rangle = 0, \quad k=1,\ldots,n\] 若 \(\{\phi_k\}\) 正交（如 Legendre 多项式），\(c_k = \langle f,\phi_k\rangle/\|\phi_k\|^2\)（无需解方程组）。 Weierstrass 逼近定理：\([a,b]\) 上的连续函数可被多项式一致逼近。 即 \(\min_{p \in \mathcal{P}_n}\|f - p\|_\infty \to 0\)（\(n \to \infty\)）。

🌐 实际应用

- 数据拟合/回归分析的数学基础 - 正交多项式在信号处理中的应用（Legendre、Chebyshev）

✏️ 例题分析与易错点

例题：用最小二乘拟合直线 \(y = c_0 + c_1 x\) 过数据 \((0,1),(1,2),(2,4),(3,4)\)。 解答：\(A = \begin{pmatrix}1&0\\1&1\\1&2\\1&3\end{pmatrix}\)，\(b = (1,2,4,4)^T\)。 \(A^TA = \begin{pmatrix}4&6\\6&14\end{pmatrix}\)，\(A^Tb = \begin{pmatrix}11\\22\end{pmatrix}\)。 解 \(4c_0+6c_1=11\)，\(6c_0+14c_1=22\) → \(c_0 = 1.1\)，\(c_1 = 1.1\)。 拟合直线 \(y = 1.1 + 1.1x\)。 易错点：正规方程的条件数 \(\kappa(A^TA) = \kappa(A)^2\)， 直接求解可能不稳定。实际应使用 QR 分解求解最小二乘。
