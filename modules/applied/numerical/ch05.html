ODE 数值方法

01Euler 方法与基本概念

🎯 为什么研究这个问题

大多数 ODE 没有解析解，数值方法是唯一的出路。Euler 方法 是最简单的 ODE 数值解法——用切线代替曲线。虽然精度低（\(O(h)\)）， 但它包含了所有 ODE 数值方法的核心概念：离散化、局部截断误差、 全局误差、收敛性和稳定性。

🔍 抽象化方法

ODE 数值方法将连续动力系统 \(y' = f(t,y)\) 离散化为 差分递推 \(y_{n+1} = \Phi(y_n, t_n, h)\)。一步法和多步法 的区别在于 \(\Phi\) 是否用到前几步的值。

📐 公式推导 / 模型建立

前向 Euler：\(y_{n+1} = y_n + hf(t_n, y_n)\)。 几何意义：沿 \((t_n, y_n)\) 处的切线走一步。 局部截断误差（LTE）：\(\tau_{n+1} = y(t_{n+1}) - y(t_n) - hf(t_n, y(t_n))\)。 Taylor 展开：\(y(t_{n+1}) = y(t_n) + hy'(t_n) + \frac{h^2}{2}y''(\xi)\)， \(\tau_{n+1} = \frac{h^2}{2}y''(\xi) = O(h^2)\)。LTE 阶 = \(p+1 = 2\)，方法阶 \(p = 1\)。 全局误差：\(e_n = y(t_n) - y_n\)。 递推：\(e_{n+1} = e_n + h[f(t_n,y(t_n))-f(t_n,y_n)] + \tau_{n+1}\)。 Lipschitz 条件 → \(|e_{n+1}| \leq (1+Lh)|e_n| + Ch^2\)。 Grönwall 引理 → \(|e_n| \leq \frac{Ch}{L}(e^{L(t_n-t_0)}-1) = O(h)\)。 后向 Euler：\(y_{n+1} = y_n + hf(t_{n+1}, y_{n+1})\)（隐式，需解方程）。 稳定性远优于前向 Euler（见 KP3）。

🌐 实际应用

- ODE 数值方法是天体力学（轨道计算）、化学动力学（反应方程）的基本工具 - 机器学习中的 Neural ODE 将 ODE 数值方法与深度学习结合

✏️ 例题分析与易错点

例题：用前向 Euler（\(h=0.1\)）解 \(y' = -y\)，\(y(0) = 1\)，求 \(y(0.2)\)。 解答：\(y_1 = 1 + 0.1(-1) = 0.9\)，\(y_2 = 0.9 + 0.1(-0.9) = 0.81\)。 精确值 \(y(0.2) = e^{-0.2} \approx 0.8187\)。误差 \(\approx 0.009\)（\(O(h)\)）。 易错点：前向 Euler 对刚性方程（如 \(y' = -1000y\)）需要极小步长 \(h < 2/1000\)， 否则数值解振荡发散。刚性问题必须用隐式方法。

02Runge-Kutta 方法

🎯 为什么研究这个问题

Euler 方法精度太低。Runge-Kutta（RK）方法通过在每步内 评估多个斜率的加权平均来提高精度，无需计算高阶导数。 经典四阶 RK（RK4）以仅 4 次函数求值达到 \(O(h^4)\) 精度， 是一般非刚性 ODE 的标准方法。

🔍 抽象化方法

RK 方法用 Butcher 表 \((A,b,c)\) 统一描述。阶条件是一系列 关于 \(A,b,c\) 的代数方程，由 Taylor 展开与 B-series 理论推导。 \(s\) 级 RK 方法最高可达 \(s\) 阶（\(s \leq 4\)）或更低（\(s \geq 5\) 时有阶障碍）。

📐 公式推导 / 模型建立

\(s\) 级显式 RK 方法： \[k_i = f\left(t_n + c_i h,\; y_n + h\sum_{j=1}^{i-1}a_{ij}k_j\right), \quad i=1,\ldots,s\] \[y_{n+1} = y_n + h\sum_{i=1}^s b_i k_i\] 经典 RK4（最著名的 RK 方法）： \(k_1 = f(t_n, y_n)\)，\(k_2 = f(t_n+h/2, y_n+hk_1/2)\)， \(k_3 = f(t_n+h/2, y_n+hk_2/2)\)，\(k_4 = f(t_n+h, y_n+hk_3)\)。 \[y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4)\] LTE = \(O(h^5)\)，全局误差 \(O(h^4)\)。 阶条件（\(p=4\) 需要 8 个方程）： \(\sum b_i = 1\)，\(\sum b_i c_i = 1/2\)，\(\sum b_i c_i^2 = 1/3\)， \(\sum b_i a_{ij}c_j = 1/6\)，... 嵌入式 RK（Dormand-Prince）：两个不同阶的 RK 共享 \(k_i\)， 阶差提供误差估计 → 自适应步长控制。

🌐 实际应用

- MATLAB `ode45` = Dormand-Prince 4(5) 阶嵌入式 RK - 计算机游戏中的物理模拟（刚体运动）常用 RK4

✏️ 例题分析与易错点

例题：用 RK4（\(h=0.1\)）解 \(y' = -y\)，\(y(0)=1\)。求 \(y_1\)。 解答：\(k_1 = -1\)，\(k_2 = -(1-0.05) = -0.95\)， \(k_3 = -(1+0.1\times(-0.95)/2) = -(1-0.0475) = -0.9525\)， \(k_4 = -(1+0.1\times(-0.9525)) = -(1-0.09525) = -0.90475\)。 \(y_1 = 1 + 0.1/6(-1-1.9-1.905-0.90475) = 1 - 0.1 \times 0.95163 = 0.904837\)。 精确值 \(e^{-0.1} = 0.904837...\)。误差 \(< 10^{-6}\)（\(O(h^4)\) 非常精确！）。 易错点：RK4 需要 4 次函数求值/步，对高维系统（\(f\) 计算昂贵时） 成本不低。此时应考虑多步法或自适应步长方法。

03刚性问题与隐式方法

🎯 为什么研究这个问题

刚性（stiff）ODE 包含快慢两种时间尺度（如化学反应中的快速 平衡与慢速动力学）。显式方法被迫使用极小步长以保稳定， 效率极低。隐式方法（如后向 Euler、梯形法、隐式 RK）的稳定域 覆盖整个左半平面，天然适合刚性问题。

🔍 抽象化方法

稳定性分析在测试方程 \(y' = \lambda y\)（\(\text{Re}\,\lambda < 0\)）上进行。 方法对 \(z = h\lambda\) 的放大因子 \(R(z)\) 决定了稳定域 \(\{z : |R(z)| \leq 1\}\)。A-稳定 = 稳定域包含整个 \(\text{Re}\,z < 0\)。

📐 公式推导 / 模型建立

测试方程：\(y' = \lambda y\)，\(\lambda \in \mathbb{C}\)，\(\text{Re}\,\lambda < 0\)。 精确解 \(y = e^{\lambda t} \to 0\)，数值方法也应衰减。 前向 Euler：\(y_{n+1} = (1+z)y_n\)，\(R(z) = 1+z\)。 稳定域：\(|1+z| \leq 1\)（以 \(-1\) 为心的单位圆）。 \(\lambda\) 实负时需 \(|1+h\lambda| \leq 1\) → \(h < 2/|\lambda|\)。 后向 Euler：\(y_{n+1} = y_n + h\lambda y_{n+1}\) → \(R(z) = 1/(1-z)\)。 \(|R(z)| \leq 1 \iff |1-z| \geq 1\)，包含整个 \(\text{Re}\,z < 0\)。A-稳定！ 梯形法（Crank-Nicolson）：\(R(z) = (1+z/2)/(1-z/2)\)。 \(|R(z)| = 1\) 当 \(\text{Re}\,z = 0\)，\(|R(z)| < 1\) 当 \(\text{Re}\,z < 0\)。A-稳定且 2 阶。 Dahlquist 第二障碍定理：线性多步法中 A-稳定方法的最高阶是 2 （即梯形法已是理论极限）。隐式 RK 可以突破此限制。

🌐 实际应用

- 电路仿真（SPICE）中的电路方程是刚性 ODE - 大气化学模型中的反应-扩散方程

✏️ 例题分析与易错点

例题：\(y' = -100y\)，\(y(0) = 1\)。比较前向 Euler 和后向 Euler 在 \(h = 0.03\) 时的表现。 解答：\(z = h\lambda = -3\)。 前向 Euler：\(R(-3) = 1 + (-3) = -2\)，\(|R| = 2 > 1\) → 发散！\(y_1 = -2\)，\(y_2 = 4\)，\(y_3 = -8\)... 后向 Euler：\(R(-3) = 1/(1-(-3)) = 1/4\)，\(|R| = 0.25 < 1\) → 稳定衰减。 \(y_1 = 0.25\)，\(y_2 = 0.0625\)，...（单调衰减到 0）。 易错点：「刚性」不等于「解变化快」。刚性是指存在快速衰减的模态 约束了显式方法的步长，即使这些快速模态对解的贡献很小。

04多步法与自适应步长

🎯 为什么研究这个问题

RK 方法每步重新计算多个 \(k_i\)，不利用先前步的信息。 线性多步法利用 \(y_{n}, y_{n-1}, \ldots\) 的历史值，仅需每步 1 次 \(f\) 求值即可达到高阶——效率极高。自适应步长控制则根据 局部误差估计自动调整 \(h\)，兼顾精度和效率。

🔍 抽象化方法

线性多步法 = 差分方程，收敛性 \(\iff\) 一致性 + 零稳定性 （Dahlquist 等价定理）。这类似于 PDE 中的 Lax 等价定理， 是数值方法理论的核心结果。

📐 公式推导 / 模型建立

\(k\) 步线性多步法： \[\sum_{j=0}^k \alpha_j y_{n+j} = h\sum_{j=0}^k \beta_j f_{n+j}\] \(\alpha_k = 1\)（归一化）。\(\beta_k = 0\)（显式）或 \(\beta_k \neq 0\)（隐式）。 Adams-Bashforth（显式）：用 \(f_n, f_{n-1},\ldots\) 的插值多项式积分。 AB2：\(y_{n+2} = y_{n+1} + h(3f_{n+1}-f_n)/2\)（2 阶）。 AB4：4 阶，每步仅 1 次 \(f\) 求值。 Adams-Moulton（隐式）：同阶精度更高。 AM2（梯形法）：\(y_{n+1} = y_n + h(f_{n+1}+f_n)/2\)（2 阶 A-稳定）。 预测-校正：用 AB 预测，AM 校正（PECE 模式）。 Dahlquist 等价定理：多步法收敛 \(\iff\) 一致（\(p \geq 1\)）+ 零稳定 （第一特征多项式 \(\rho(\zeta) = \sum\alpha_j\zeta^j\) 的根满足根条件： \(|\zeta| \leq 1\)，且 \(|\zeta|=1\) 的根是单根）。 自适应步长：用嵌入公式或 Richardson 外推估计 LTE \(\approx \hat\tau\)。 若 \(|\hat\tau| > \text{tol}\)，缩小 \(h\)：\(h_{\text{new}} = h\cdot({\text{tol}}/|\hat\tau|)^{1/(p+1)}\)。 若 \(|\hat\tau| \ll \text{tol}\)，增大 \(h\) 以提高效率。

🌐 实际应用

- 天体力学中的长时间轨道积分使用高阶 Adams 方法 - MATLAB `ode113` = 变阶 Adams-Bashforth-Moulton PECE

✏️ 例题分析与易错点

例题：用 AB2 方法解 \(y' = -y\)（\(y_0 = 1\)，\(y_1 = e^{-h}\)，\(h = 0.1\)）。 解答：\(f_0 = -1\)，\(f_1 = -e^{-0.1} \approx -0.9048\)。 \(y_2 = y_1 + 0.1(3(-0.9048)-(-1))/2 = 0.9048 + 0.1(-2.7145+1)/2\) \(= 0.9048 + 0.1\times(-0.8572) = 0.9048 - 0.08572 = 0.8191\)。 精确值 \(e^{-0.2} \approx 0.8187\)。误差 \(\approx 4\times10^{-4}\)（\(O(h^2)\) ✓）。 易错点：多步法需要额外的起步值 \(y_1,\ldots,y_{k-1}\)（用 RK 方法计算）。 如果起步值精度低于多步法阶数，全局精度降阶。
