数值积分

01Newton-Cotes 公式

🎯 为什么研究这个问题

大量积分没有解析的原函数（如 \(\int e^{-x^2}dx\)），必须用数值方法。 Newton-Cotes 公式用等距节点上的插值多项式替代被积函数再积分， 是最直观的数值积分方法。梯形法则和 Simpson 法则是最常用的 低阶代表。

🔍 抽象化方法

数值积分 = 「用加权求和 \(\sum w_i f(x_i)\) 逼近积分」。 Newton-Cotes 用等距节点的 Lagrange 插值确定权重 \(w_i\)。 精度由「代数精度」衡量：对 \(\leq d\) 次多项式精确积分。

📐 公式推导 / 模型建立

梯形法则（\(n=1\)，两点）： \[\int_a^b f(x)\,dx \approx \frac{b-a}{2}(f(a)+f(b))\] 误差：\(-\frac{(b-a)^3}{12}f''(\xi)\)（代数精度 1）。 Simpson 法则（\(n=2\)，三点）： \[\int_a^b f(x)\,dx \approx \frac{b-a}{6}\left[f(a)+4f\!\left(\frac{a+b}{2}\right)+f(b)\right]\] 误差：\(-\frac{(b-a)^5}{2880}f^{(4)}(\xi)\)（代数精度 3——比预期多 1！）。 原因：Simpson 公式关于中点对称，奇次项误差自动消去。 复合公式：将 \([a,b]\) 分成 \(n\) 段，每段用简单公式。 复合梯形：\(T_n = h\left[\frac{f(a)}{2}+\sum_{k=1}^{n-1}f(a+kh)+\frac{f(b)}{2}\right]\)，误差 \(O(h^2)\)。 复合 Simpson：误差 \(O(h^4)\)。

🌐 实际应用

- 物理中的数值模拟（能量计算、面积/体积） - 概率密度函数的数值积分（正态分布表的计算）

✏️ 例题分析与易错点

例题：用复合梯形（\(n=4\)）计算 \(\int_0^1 e^x dx\)。 解答：\(h = 0.25\)，节点 \(x_k = 0, 0.25, 0.5, 0.75, 1\)。 \(T_4 = 0.25[e^0/2 + e^{0.25} + e^{0.5} + e^{0.75} + e^1/2]\) \(= 0.25[0.5 + 1.2840 + 1.6487 + 2.1170 + 1.3591] = 0.25 \times 6.9088 = 1.7272\)。 精确值 \(e-1 = 1.71828...\)，误差 \(\approx 0.009\)（\(O(h^2)\)，\(h^2 = 0.0625\)）。 易错点：高阶 Newton-Cotes（\(n \geq 8\)）的权重可能出现负值， 导致数值不稳定。实际中不使用高阶 Newton-Cotes，改用 Gauss 求积。

02Gauss 求积

🎯 为什么研究这个问题

Newton-Cotes 公式节点位置固定（等距），只优化权重。 如果同时优化节点位置和权重呢？\(n\) 个点的 Gauss 求积达到 代数精度 \(2n-1\)——几乎是 \(2n\) 个自由度的理论极限。这是 数值积分中精度最高的方法。

🔍 抽象化方法

Gauss 求积的节点是正交多项式的零点。权重由精确积分条件 唯一确定。这将数值积分问题与正交多项式理论深度连接。

📐 公式推导 / 模型建立

\(n\) 点 Gauss 求积：在 \([-1,1]\) 上 \[\int_{-1}^1 f(x)\,dx \approx \sum_{i=1}^n w_i f(x_i)\] 其中 \(x_1,\ldots,x_n\) 是 \(n\) 次 Legendre 多项式 \(P_n(x)\) 的零点， \(w_i = \int_{-1}^1 L_i(x)\,dx > 0\)。 代数精度定理：\(n\) 点 Gauss 求积对 \(\leq 2n-1\) 次多项式精确。 证明：任意 \(p \in \mathcal{P}_{2n-1}\)，做带余除法 \(p = qP_n + r\)（\(\deg q, r \leq n-1\)）。 \(\int p = \int qP_n + \int r\)。\(\int qP_n = 0\)（\(P_n\) 的正交性，\(\deg q \leq n-1\)）。 \(\sum w_i p(x_i) = \sum w_i[q(x_i)P_n(x_i) + r(x_i)] = \sum w_i r(x_i) = \int r\) （因 \(P_n(x_i)=0\) 且 \(\deg r \leq n-1\)，\(n\) 点对 \(r\) 精确）。故 \(\sum w_i p(x_i) = \int p\)。 误差：\(E_n(f) = \frac{f^{(2n)}(\xi)}{(2n)!}\frac{2^{2n+1}(n!)^4}{(2n+1)[(2n)!]^2}\)。

🌐 实际应用

- 有限元方法中的数值积分（Gauss 积分规则） - 光学和辐射传输中的角度积分

✏️ 例题分析与易错点

例题：用 2 点 Gauss 求积计算 \(\int_{-1}^1 e^x dx\)。 解答：\(P_2(x) = (3x^2-1)/2\)，零点 \(x_{1,2} = \pm1/\sqrt{3}\)，\(w_1 = w_2 = 1\)。 \(\int_{-1}^1 e^x dx \approx e^{-1/\sqrt{3}} + e^{1/\sqrt{3}} = 0.5615 + 1.7813 = 2.3428\)。 精确值 \(e - e^{-1} = 2.3504\)，误差 \(\approx 0.008\)。 2 点 Gauss 精度 \(2n-1=3\)，即对 \(\leq 3\) 次多项式精确。 易错点：Gauss 节点不包含端点 \(\pm 1\)。如果需要端点（如衔接条件）， 应使用 Gauss-Lobatto 求积（含端点，精度 \(2n-3\)）。

03自适应积分与 Romberg 方法

🎯 为什么研究这个问题

固定步长的复合积分可能在被积函数变化剧烈处精度不足， 在平坦处浪费计算量。自适应积分根据局部误差估计自动 细化网格，实现「精度目标驱动」的计算。Romberg 方法 则用 Richardson 外推系统地从低阶梯形法提升到高阶。

🔍 抽象化方法

自适应策略将全局误差分解为逐子区间的局部误差， 递归细化超过阈值的子区间。这是「分治 + 误差控制」 的算法范式。

📐 公式推导 / 模型建立

Romberg 积分：从复合梯形 \(T_h\) 出发，利用 Euler-Maclaurin 展开 \(T_h = I + c_1 h^2 + c_2 h^4 + \cdots\)，逐步消去低阶误差项。 Richardson 外推表格： \[R_{k,1} = T_{h/2^{k-1}}, \quad R_{k,j} = \frac{4^{j-1}R_{k,j-1}-R_{k-1,j-1}}{4^{j-1}-1}\] \(R_{k,2}\) = Simpson，\(R_{k,3}\) = Boole（\(O(h^6)\)），... 每列提升 2 阶精度。 自适应 Simpson（经典自适应算法）： 1. 在 \([a,b]\) 上计算 \(S_1 = S(a,b)\) 和 \(S_2 = S(a,m) + S(m,b)\)（\(m\) 中点） 2. 误差估计：\(|S_2 - S_1|/15\)（Richardson 外推思想） 3. 若 \(< \text{tol}\)，接受 \(S_2 + (S_2-S_1)/15\)；否则递归细分 Euler-Maclaurin 公式（Romberg 的理论基础）： \[\int_a^b f\,dx = T_h - \sum_{k=1}^p \frac{B_{2k}h^{2k}}{(2k)!}(f^{(2k-1)}(b)-f^{(2k-1)}(a)) + O(h^{2p+2})\]

🌐 实际应用

- 科学计算软件中的自适应积分（MATLAB `integral`，Python `scipy.integrate.quad`） - 量子化学中的电子积分使用自适应 Gauss 方法

✏️ 例题分析与易错点

例题：构造 Romberg 表格的前两列来计算 \(\int_0^1 e^x dx\)。 解答：\(R_{1,1} = T_1 = (1+e)/2 \approx 1.8591\)。 \(R_{2,1} = T_{1/2} = (1+e)/4 + e^{0.5}/2 \approx 1.7539\)。 \(R_{2,2} = (4R_{2,1}-R_{1,1})/3 = (4\times1.7539-1.8591)/3 = 1.7189\)。 精确值 \(1.71828\)。\(R_{2,2}\) 精度远好于 \(R_{2,1}\)（Simpson 阶）。 易错点：Romberg 方法要求 \(f\) 足够光滑（需要 Euler-Maclaurin 展开成立）。 对不光滑函数（如 \(|x|\)），误差展开含分数幂，外推失效。

04多重积分与 Monte Carlo 方法

🎯 为什么研究这个问题

高维积分 \(\int_{\mathbb{R}^d}f(x)\,dx\) 是科学计算的核心挑战。 确定性方法（如张量积 Gauss 求积）的计算量随维度指数增长 （维度灾难），而 Monte Carlo 方法的收敛速率 \(O(1/\sqrt{N})\) 与维度无关——这使它成为高维积分的唯一可行工具。

🔍 抽象化方法

Monte Carlo 将积分化为期望值 \(\int f = E[f(X)]\)， 用随机样本的算术平均来估计。中心极限定理保证 \(O(1/\sqrt{N})\) 的收敛速率。方差缩减技术在不增加样本量 的前提下提高精度。

📐 公式推导 / 模型建立

基本 Monte Carlo：\(I = \int_0^1 f(x)\,dx \approx \frac{1}{N}\sum_{i=1}^N f(X_i)\)， \(X_i \sim U(0,1)\) 独立同分布。 大数定律 → 一致收敛。误差（标准差）： \[\sigma_N = \frac{\sigma_f}{\sqrt{N}}, \quad \sigma_f^2 = \text{Var}(f(X))\] \(N\) 增大 4 倍，精度仅提高 2 倍。 与确定性方法对比（\(d\) 维）： - 张量积梯形/Gauss：\(N = n^d\) 个点，误差 \(O(n^{-p}) = O(N^{-p/d})\) - Monte Carlo：误差 \(O(N^{-1/2})\)（与 \(d\) 无关） - 当 \(d > 2p\) 时 Monte Carlo 胜出（高维必胜） 方差缩减： - 重要性采样：\(I = \int f/g \cdot g = E_{X\sim g}[f(X)/g(X)]\)（选 \(g \approx |f|\) 的分布） - 对偶变量法、控制变量法

🌐 实际应用

- 金融工程中的期权定价（高维随机过程的期望） - 粒子物理中的 Feynman 路径积分 - 机器学习中的随机梯度下降（本质上是 Monte Carlo 积分）

✏️ 例题分析与易错点

例题：用 Monte Carlo 估计 \(\pi\)：在 \([0,1]^2\) 中随机撒点， 落入单位圆内（\(x^2+y^2 \leq 1\)）的比例 \(\approx \pi/4\)。 解答：\(\hat\pi = 4 \cdot \frac{\#\{X_i^2+Y_i^2 \leq 1\}}{N}\)。 \(N = 10000\) 时典型精度 \(\approx \pm 0.02\)（\(1/\sqrt{10000} = 0.01\)）。 \(N = 10^6\) 时 \(\approx \pm 0.002\)。 每增加 2 位精度需要 \(10^4\) 倍样本量（效率低但通用）。 易错点：伪随机数的质量影响 Monte Carlo 结果。低质量的随机数生成器 可能在高维空间产生相关性，导致带偏估计。
