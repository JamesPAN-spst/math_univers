Wasserstein 距离

01\(W_p\) 距离的定义与度量空间结构

🎯 为什么研究这个问题

Wasserstein 距离 \(W_p\) 赋予概率测度空间 \(\mathcal{P}_p(X)\) 完备可 分度量空间结构。与 KL 散度（不对称、不满足三角不等式）和 TV 距 离（不考虑底空间几何）不同，\(W_p\) 是真正的"几何"距离——它感知 底空间中"多远"。这使 Wasserstein 空间成为现代概率论和机器学习 的核心几何对象。

🔍 抽象化方法

\((\mathcal{P}_p(X), W_p)\) 继承了 \((X, d)\) 的几何结构：测地线、曲率等。

📐 公式推导 / 模型建立

\(p\)-Wasserstein 距离（\(p \geq 1\)）： \[W_p(\mu,\nu) = \left(\inf_{\gamma \in \Pi(\mu,\nu)} \int_{X \times X} d(x,y)^p \, d\gamma(x,y)\right)^{1/p}\] \(\mathcal{P}_p(X) = \{\mu \in \mathcal{P}(X) : \int d(x, x_0)^p \, d\mu < \infty\}\)（有限 \(p\) 阶矩）。 完备性：若 \((X,d)\) 完备，则 \((\mathcal{P}_p(X), W_p)\) 完备。 可分性：若 \((X,d)\) 可分，则 \((\mathcal{P}_p(X), W_p)\) 也可分。 收敛拓扑（\(p=1\) 时）：\(W_1(\mu_n, \mu) \to 0\) 当且仅当 \(\mu_n \xrightarrow{w} \mu\)（弱收敛）且 \(\int d(x,x_0) \, d\mu_n \to \int d(x,x_0) \, d\mu\) （一阶矩收敛）。

🌐 实际应用

统计中的分布收敛速率分析：\(W_2\) 是最自然的度量。

✏️ 例题分析与易错点

例题：\(\mu_n = (1-1/n)\delta_0 + (1/n)\delta_n\), \(\mu = \delta_0\)（\(\mathbb{R}\) 上）。 \(W_1(\mu_n, \mu)\)？TV 距离？ 解答：\(W_1 = \inf_\gamma \int |x-y| d\gamma\)。最优传输：质量 \(1-1/n\) 留在 \(0\)， \(1/n\) 从 \(n\) 到 \(0\)，代价 \(= (1/n) \cdot n = 1\)。\(W_1 = 1\) \(\not\to 0\)！ 但 \(\text{TV} = 2/n \to 0\)。 \(\mu_n\) 弱收敛到 \(\mu\)，但一阶矩不收敛。 易错点：弱收敛不蕴含 \(W_p\) 收敛——还需矩条件。

⚡ 快速检测
以下哪个距离/散度满足三角不等式？

02\(W_p\) 与其他距离/散度的比较

🎯 为什么研究这个问题

实际使用中如何选择 \(W_p\)、KL、TV、MMD 等度量？需要理解它们各自 的优劣和适用场景。尤其是在生成模型训练中，\(W_1\) 相比 KL/JS 的 梯度稳定性是 WGAN 成功的关键。但 \(W_p\) 的计算成本也更高。

🔍 抽象化方法

不同度量对分布差异的敏感度不同——\(W_p\) 对平移敏感，TV 对支撑 差异敏感，KL 对密度比敏感。

📐 公式推导 / 模型建立

关键比较： 1) TV vs \(W_1\)：Pinsker 不等式 \(\text{TV}(\mu,\nu) \leq \sqrt{KL(\mu\|\nu)/2}\)。 但 \(\text{TV}\) 不感知底空间度量：\(\text{TV}(\delta_0, \delta_\varepsilon) = 2\) 对所有 \(\varepsilon > 0\)，而 \(W_1(\delta_0, \delta_\varepsilon) = \varepsilon \to 0\)。 2) KL vs \(W_p\)：KL 要求 \(\nu \ll \mu\)（绝对连续），否则 \(= +\infty\)。 \(W_p\) 对支撑不重叠的分布仍有限且有意义。 3) 不等式链：\(W_1(\mu,\nu) \leq C \cdot \text{TV}(\mu,\nu)\) 当 \(X\) 有界直径 \(C\)。反之无上界。 4) 维数灾难：经验分布 \(\hat{\mu}_n\) 的 \(W_p\) 收敛速率为 \(O(n^{-1/d})\) （\(d\) 维），而 KL 不受此限。

🌐 实际应用

文本生成中 exposure bias 的度量：\(W\) 距离比 BLEU 更好地反映语义距离。

✏️ 例题分析与易错点

例题：\(\mu_\theta = \delta_\theta\)（参数化点质量），\(\nu = \delta_0\)。 求 \(W_1, \text{TV}, KL\) 关于 \(\theta\) 的梯度。 解答：\(W_1 = |\theta|\)：\(\partial W_1 / \partial\theta = \text{sign}(\theta)\)。梯度常数——训练稳定！ \(\text{TV} = 2 \cdot \mathbf{1}_{\theta \neq 0}\)：不连续，梯度要么 0 要么不存在。 \(KL = +\infty\)（支撑不重叠）。 这就是 WGAN 成功的根本原因。 易错点：\(W_p\) 的梯度优势在分布支撑不重叠时最明显——这恰是 GAN 训练初期的情况。

⚡ 快速检测

03Wasserstein 空间上的测地线

🎯 为什么研究这个问题

Wasserstein 空间不仅是度量空间，还是一个具有丰富几何的测地空间。 \((\mathcal{P}_2(\mathbb{R}^d), W_2)\) 中的测地线对应 McCann 位移插 值——两个分布之间的"最自然"路径。这种几何视角引发了 Otto 微积 分学(Otto Calculus)和 gradient flow 理论，成为现代 PDE 和概率论 的核心。

🔍 抽象化方法

Wasserstein 测地线 \(=\) 位移插值 \(\mu_t = ((1-t)Id + tT)_\#\mu\)， 其中 \(T\) 是最优传输映射。

📐 公式推导 / 模型建立

McCann 位移插值：设 \(T\) 是 \(\mu\) 到 \(\nu\) 的最优映射（Brenier），定义 \[\mu_t = ((1-t)\text{Id} + tT)_\#\mu, \quad t \in [0,1]\] 则 \(\mu_0 = \mu, \mu_1 = \nu\)，且 \(W_2(\mu_s, \mu_t) = |s-t| \cdot W_2(\mu,\nu)\) ——等速测地线。 位移凸性(Displacement Convexity)：泛函 \(\mathcal{F}(\mu)\) 沿测地线是凸的： \[\mathcal{F}(\mu_t) \leq (1-t)\mathcal{F}(\mu_0) + t\mathcal{F}(\mu_1)\] 例如 Boltzmann 熵 \(H(\mu) = \int \rho \log \rho \, dx\)（\(d\mu = \rho \, dx\)）。 位移凸性 \(\Rightarrow\) 热方程解的唯一性和长时行为分析。

🌐 实际应用

Flow Matching / Rectified Flow：训练神经网络沿 McCann 插值学习速度场， 生成模型的最新范式。

✏️ 例题分析与易错点

例题：\(\mu = \mathcal{N}(0,1), \nu = \mathcal{N}(2,4)\)（一维）。\(\mu_{1/2}\) 的分布？ 解答：最优映射 \(T(x) = 2x + 2\)（分位数匹配）。 \(T_{1/2}(x) = (1-1/2)x + (1/2)(2x+2) = 3x/2 + 1\)。 \(\mu_{1/2} = (T_{1/2})_\#\mathcal{N}(0,1) = \mathcal{N}(1, 9/4)\)。 均值 \(= 1\)（\(0\) 和 \(2\) 的中点），标准差 \(= 3/2\)（\(1\) 和 \(2\) 的中点）。 易错点：位移插值不等于混合（\(\neq (1-t)\mu + t\nu\)）——它是"传输"中间态，保持单峰性。

04Wasserstein 重心与多边际传输

🎯 为什么研究这个问题

给定多个分布 \(\mu_1, \ldots, \mu_K\)，它们的 Wasserstein 重心 (barycenter)是使加权 \(W_2^2\) 和最小的分布。这推广了欧氏空间中 的 Fréchet 均值，在形状分析、纹理混合、联邦学习中有重要应用。 多边际最优传输(Multi-marginal OT)进一步推广到同时耦合多个分布。

🔍 抽象化方法

Wasserstein 重心 \(=\) 概率测度空间中的 Fréchet 均值。 多边际 OT \(=\) 多个分布的最优联合耦合。

📐 公式推导 / 模型建立

Wasserstein 重心： \[\bar{\mu} = \arg\min_{\mu} \sum_{k=1}^K \lambda_k W_2^2(\mu, \mu_k), \quad \lambda_k > 0, \sum \lambda_k = 1\] 离散情况（每个 \(\mu_k\) 有 \(n\) 个支撑点）的 LP 公式规模巨大—— 需要高效算法（Sinkhorn 重心或 free support 方法）。 多边际 OT： \[\inf_{\gamma \in \Pi(\mu_1,...,\mu_K)} \int c(x_1,...,x_K) \, d\gamma(x_1,...,x_K)\] 代价函数 \(c\) 作用于 \(K\) 个变量。密度泛函理论中的电子关联能等价于 特殊的多边际 OT 问题（Coulomb 代价）。

🌐 实际应用

联邦学习中模型聚合：将各客户端参数分布用 Wasserstein 重心融合。

✏️ 例题分析与易错点

例题：\(\mu_1 = \delta_0, \mu_2 = \delta_2\)，等权 \(\lambda = (1/2, 1/2)\)。 \(W_2\) 重心？ 解答：\(\bar{\mu} = \arg\min_\mu \frac{1}{2}W_2^2(\mu, \delta_0) + \frac{1}{2}W_2^2(\mu, \delta_2)\)。 对点质量 \(\mu = \delta_m\)：\(= \frac{1}{2}m^2 + \frac{1}{2}(m-2)^2\)。 导数 \(= m + (m-2) = 0 \Rightarrow m = 1\)。\(\bar{\mu} = \delta_1\)。 （重心是中点——直观！但对非点质量分布，结果更有趣。） 易错点：\(W_2\) 重心不同于逐点混合 \(\frac{1}{2}\delta_0 + \frac{1}{2}\delta_2\)， 后者是双峰的，前者集中在中间。
