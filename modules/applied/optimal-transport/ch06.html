01WGAN 与生成模型

🎯 为什么研究这个问题

Wasserstein GAN(Arjovsky et al. 2017)用 \(W_1\) 距离替代原始 GAN 的 JS 散度训练判别器，根本性解决了训练不稳定和模式坍缩问题。 WGAN 是 OT 在深度学习中最成功的应用，也直接推动了 OT 计算方法 和 Lipschitz 网络约束的研究热潮。

🔍 抽象化方法

GAN 训练 \(=\) 两个测度之间距离的极小极大优化。用 \(W_1\) 的 KR 对偶： critic 网络逼近最优势函数 \(\varphi\)。

📐 公式推导 / 模型建立

WGAN 目标： \[\min_G \max_{\|D\|_{\text{Lip}} \leq 1} \mathbb{E}_{x \sim p_{\text{data}}}[D(x)] - \mathbb{E}_{z \sim p_z}[D(G(z))]\] 由 KR 对偶，内层 max \(= W_1(p_{\text{data}}, p_G)\)。 外层 min 通过生成器 \(G\) 最小化 Wasserstein 距离。 Lipschitz 约束实现： 1) 权重裁剪（原始 WGAN）：\(\|W\|_\infty \leq c\)——粗糙但有效 2) 梯度惩罚(WGAN-GP)：\(\lambda \mathbb{E}_{\hat{x}}[(|\nabla D(\hat{x})| - 1)^2]\) 3) 谱归一化(SN)：\(W \leftarrow W / \sigma_1(W)\)——除以最大奇异值 训练稳定性的数学原因：\(W_1\) 对生成器参数的梯度连续（不像 JS 在支撑不重叠时梯度为 0）。

🌐 实际应用

高分辨率图像生成、药物分子生成、音乐合成。

✏️ 例题分析与易错点

例题：1D WGAN。\(p_{\text{data}} = \mathcal{N}(0,1)\)，\(p_G = \mathcal{N}(\theta, 1)\)。 最优 critic \(D^*\) 和 \(W_1\) 关于 \(\theta\) 的梯度？ 解答：\(W_1(\mathcal{N}(0,1), \mathcal{N}(\theta,1)) = |\theta|\)（相同方差时）。 最优 critic \(D^*(x) = \text{sign}(\theta) \cdot x\)（1-Lipschitz，线性）。 \(\partial W_1 / \partial\theta = \text{sign}(\theta)\)。梯度不随 \(|\theta|\) 衰减——训练稳定。 易错点：WGAN 的 critic 不是概率输出——是实值函数，不用 sigmoid。

02Flow Matching 与 OT 路径

🎯 为什么研究这个问题

Flow Matching(Lipman et al. 2023)是近年最热门的生成模型范式—— 直接用回归训练学习从噪声到数据的矢量场。当选择 OT 条件路径时 ，Flow Matching 产生最"直"的传输路径（McCann 插值），推理效率远 高于扩散模型(DDPM)。这是 OT 理论在 AI 前沿的直接应用。

🔍 抽象化方法

Flow Matching = 连续正规化流(CNF) + 条件矢量场回归。 OT 条件路径 \(=\) McCann 位移插值。

📐 公式推导 / 模型建立

连续正规化流：\(\frac{dx}{dt} = v_\theta(x, t)\)，\(t \in [0,1]\)。 \(p_0 = \mathcal{N}(0,I)\)（噪声），\(p_1 = p_{\text{data}}\)。 条件 Flow Matching 损失： \[\mathcal{L}(\theta) = \mathbb{E}_{t, x_1 \sim p_{\text{data}}} \|v_\theta(\mu_t(x_1), t) - \dot{\mu}_t(x_1)\|^2\] OT 条件路径（线性插值）： \(\mu_t(x_1) = (1-t) x_0 + t x_1\)，\(\dot{\mu}_t = x_1 - x_0\)。 这恰好是到 \(x_1\) 的 McCann 插值（\(\mathcal{N}(0,I)\) 到 \(\delta_{x_1}\)）。 与 DDPM 的对比：扩散模型使用弯曲的随机路径（正向 SDE + 逆向 SDE）， Flow Matching 使用直线——推理时需要更少的 ODE 求解步数（10-20 步 vs 1000 步）。

🌐 实际应用

Stable Diffusion 3、Meta Voicebox 等最新模型采用 Flow Matching 架构。

✏️ 例题分析与易错点

例题：1D Flow Matching。\(p_0 = \mathcal{N}(0,1)\)，\(x_1 = 3\)。 OT 条件路径和目标矢量场？ 解答：\(x_0 \sim \mathcal{N}(0,1)\)。\(\mu_t = (1-t)x_0 + 3t\)。 \(\dot{\mu}_t = -x_0 + 3 = 3 - x_0\)。 网络学习 \(v_\theta(\mu_t, t) \approx 3 - x_0\)。 因为 \(x_0 = (\mu_t - 3t)/(1-t)\)，所以 \(v_\theta\) 可以从 \((\mu_t, t)\) 推断 \(x_0\)。 易错点：OT 路径不等于 DDPM 的"去噪"路径——前者是直线，后者是曲线。

03分布鲁棒优化(DRO)

🎯 为什么研究这个问题

机器学习中训练分布 \(\neq\) 测试分布（分布偏移）。分布鲁棒优化 (Distributionally Robust Optimization)用 Wasserstein 球定义不确定 集，在最坏情况分布下优化——既不过于保守（如 minimax），也不过于 乐观（如 ERM）。这是 OT 在鲁棒统计和可靠 ML 中的核心应用。

🔍 抽象化方法

DRO \(=\) \(\min_\theta \sup_{\nu: W(\nu, \hat{\mu}_n) \leq \rho} \mathbb{E}_\nu[\ell(\theta; \cdot)]\)， 在训练数据附近的 Wasserstein 球内做最坏情况优化。

📐 公式推导 / 模型建立

Wasserstein DRO 问题： \[\min_\theta \sup_{\nu \in \mathcal{B}_\rho(\hat{\mu}_n)} \mathbb{E}_\nu[\ell(\theta; X, Y)]\] 其中 \(\mathcal{B}_\rho(\hat{\mu}_n) = \{\nu : W_p(\nu, \hat{\mu}_n) \leq \rho\}\)。 对偶形式（Blanchet-Murthy 2019）： \[= \min_\theta \inf_{\lambda \geq 0} \left\{\lambda \rho^p + \frac{1}{n}\sum_{i=1}^n \sup_z [\ell(\theta; z) - \lambda \cdot d(z, z_i)^p]\right\}\] 当 \(\ell\) 是 Lipschitz 的：内层 sup 有界，问题可高效求解。 \(\rho\) 的选取：置信区间方法，使真实分布以高概率在 \(\mathcal{B}_\rho\) 内。 与正则化的关系：当 \(\rho\) 小且 \(p=1\) 时，DRO \(\approx\) ERM + Lipschitz 正则化。

🌐 实际应用

金融风险管理(CVaR)、公平机器学习（对不同亚群鲁棒）。

✏️ 例题分析与易错点

例题：线性回归 \(\ell(\theta; x, y) = (y - \theta^\top x)^2\)。 \(n=1\) 个数据点 \((x_1, y_1) = (1, 1)\)，\(\rho = 0.5\), \(p=1\)。DRO 问题？ 解答：\(\sup_{\|(x,y)-(1,1)\|_1 \leq 0.5} (y-\theta x)^2\)。 对手选 \((x,y)\) 使损失最大。若 \(\theta > 0\)：减小 \(x\) 或 \(y\) 使 \(y - \theta x\) 尽量偏离 0。这给出一个关于 \(\theta\) 的非光滑但凸的目标。 易错点：DRO 的保守程度取决于 \(\rho\)——\(\rho\) 太大导致过于悲观，\(\rho = 0\) 退化为 ERM。

04OT 在科学前沿的应用

🎯 为什么研究这个问题

最优传输理论正在深刻影响多个科学领域：从单细胞生物学（追踪细胞 发育轨迹）到 Riemann 几何（Lott-Villani-Sturm 曲率定义）再到经济 学（匹配市场与合同设计）。理解这些前沿应用展示了 OT 作为数学 "通用语言"的力量。

🔍 抽象化方法

OT 提供了比较和插值概率分布的标准工具——任何涉及"分布之间关系" 的问题都可能受益。

📐 公式推导 / 模型建立

1) 单细胞生物学（Schiebinger et al. 2019, Waddington-OT）： 将不同时间点的细胞基因表达分布视为 \(\mu_t\)， 用 OT 推断细胞分化轨迹——哪些祖细胞分化成哪些子细胞。 \[\gamma^*_{t_1 \to t_2} = \arg\min \int c \, d\gamma + \varepsilon KL(\gamma\|\mu_{t_1} \otimes \mu_{t_2})\] 加上增殖/凋亡的不平衡修正。 2) 合成度量几何（Lott-Villani 2009, Sturm 2006）： 用 Wasserstein 空间中的位移凸性定义 Ricci 曲率下界 \[\text{Ric} \geq K \quad \Leftrightarrow \quad \text{熵沿 } W_2\text{-测地线是 } K\text{-凸的}\] 这给出了度量空间（不一定是光滑流形）上的曲率概念。 3) 域自适应(Domain Adaptation)：用 \(W\) 距离量化源域和目标域的差异， 最小化 \(\mathcal{L}_{\text{source}} + \lambda W(p_{\text{source}}, p_{\text{target}})\)。

🌐 实际应用

COVID-19 单细胞分析中追踪免疫细胞对病毒的响应轨迹。

✏️ 例题分析与易错点

例题：两时间点的单细胞数据（2D PCA）：\(t_1\) 有 100 个细胞，\(t_2\) 有 150 个细胞。 用 OT 推断命运映射。 解答：\(\mu_{t_1} = \frac{1}{100}\sum_i \delta_{x_i}\), \(\mu_{t_2} = \frac{1}{150}\sum_j \delta_{y_j}\)。 不平衡 OT（因为细胞增殖）：放松边缘约束，加 KL 惩罚。 求解后 \(\gamma^*_{ij}\) 大的 \((i,j)\) 对表示细胞 \(x_i\) 可能分化为 \(y_j\)。 易错点：细胞数量变化意味着传输不是平衡的——需要 unbalanced OT。
