Monge-Kantorovich 问题

01Monge 问题的历史与数学建模

🎯 为什么研究这个问题

1781 年 Gaspard Monge 提出一个看似简单的问题：如何将一堆土最省 力地搬到指定位置？这是最优传输理论的起点。虽然直觉上很自然，但 严格数学化需要测度论语言，而且 Monge 问题在一般情况下可能无解 ——这推动了 Kantorovich 松弛的诞生。

🔍 抽象化方法

质量分布 \(=\) 概率测度 \(\mu, \nu\)；搬运方案 \(=\) 推前映射 \(T\) 满足 \(T_\#\mu = \nu\)。

📐 公式推导 / 模型建立

Monge 问题的严格表述：给定概率空间 \((X, \mu)\) 和 \((Y, \nu)\)， 及代价函数 \(c: X \times Y \to \mathbb{R}_+\)，求 \[\inf_{T: T_\#\mu = \nu} \int_X c(x, T(x)) \, d\mu(x)\] 推前条件 \(T_\#\mu = \nu\)：对所有 Borel 集 \(B \subset Y\)， \(\mu(T^{-1}(B)) = \nu(B)\)。即 \(T\) 将 \(\mu\) 的质量精确转移到 \(\nu\)。 存在性问题：若 \(\mu = \delta_{x_0}\)（点质量），\(\nu = \frac{1}{2}\delta_{y_1} + \frac{1}{2}\delta_{y_2}\)， 则不存在映射 \(T\) 使 \(T_\#\mu = \nu\)——一个点无法映射到两个点！ Monge 问题可能无可行解。

🌐 实际应用

物流网络中的仓库-门店配送路径优化（每个仓库的货物只能发到一个目的地）。

✏️ 例题分析与易错点

例题：\(X = Y = \mathbb{R}\), \(\mu = \mathcal{U}[0,1]\)（均匀分布）， \(\nu = \mathcal{U}[1,2]\), \(c(x,y) = |x-y|^2\)。最优映射？ 解答：分位数映射 \(T(x) = F_\nu^{-1}(F_\mu(x)) = x + 1\)（平移）。 代价 \(= \int_0^1 |x - (x+1)|^2 dx = 1\)。 这是一维情况下的解——分位数映射总是最优的。 易错点：分位数映射公式只在一维有效。高维情况需要 Brenier 定理。

02Kantorovich 松弛与耦合

🎯 为什么研究这个问题

Leonid Kantorovich 在 1942 年提出了天才的松弛：不要求确定性映射， 而是允许随机传输方案（耦合）。每个源点的质量可以分配到多个目标。 这保证了最优解总是存在的。Kantorovich 因这一工作获得 1975 年诺 贝尔经济学奖（与 Koopmans 分享）。

🔍 抽象化方法

耦合 \(\gamma \in \Pi(\mu, \nu)\) \(=\) \(X \times Y\) 上的联合分布，边缘 为 \(\mu\) 和 \(\nu\)。松弛：映射 \(\to\) 联合分布。

📐 公式推导 / 模型建立

Kantorovich 问题： \[\inf_{\gamma \in \Pi(\mu,\nu)} \int_{X \times Y} c(x,y) \, d\gamma(x,y)\] 其中 \(\Pi(\mu,\nu) = \{\gamma \in \mathcal{P}(X \times Y) : (\pi_X)_\#\gamma = \mu, (\pi_Y)_\#\gamma = \nu\}\)。 存在性定理：若 \(c\) 下半连续且 \(X, Y\) 是波兰空间(Polish space)， 则 \(\Pi(\mu,\nu)\) 非空紧（弱拓扑），\(c\) 的积分下半连续， 由 Weierstrass 极值定理，最优 \(\gamma^*\) 存在。 与 Monge 的关系：若最优 \(\gamma^*\) 集中在图 \(\{(x, T(x))\}\) 上， 则 \(T\) 是 Monge 最优映射。Kantorovich \(\leq\) Monge（松弛更宽）。

🌐 实际应用

计算机视觉中图像之间的 Earth Mover's Distance 就是离散 Kantorovich 问题。

✏️ 例题分析与易错点

例题：\(\mu = \frac{1}{2}\delta_0 + \frac{1}{2}\delta_1\)，\(\nu = \delta_{1/2}\)，\(c = |x-y|^2\)。 Monge 解？Kantorovich 解？ 解答：Monge：\(T(0) = T(1) = 1/2\)。\(T_\#\mu = \delta_{1/2} = \nu\)。✓ 代价 \(= \frac{1}{2}(0-1/2)^2 + \frac{1}{2}(1-1/2)^2 = 1/4\)。 Kantorovich 也是 \(1/4\)（这里 Monge 解存在且是唯一耦合的支撑）。 反例：\(\mu = \delta_0, \nu = \frac{1}{2}\delta_{-1}+\frac{1}{2}\delta_1\)——Monge 无解， Kantorovich 解 \(\gamma = \frac{1}{2}\delta_{(0,-1)} + \frac{1}{2}\delta_{(0,1)}\)，代价 \(= 1\)。 易错点：Kantorovich 解总存在；Monge 解不一定存在。两者代价可以相等。

03离散最优传输与线性规划

🎯 为什么研究这个问题

当 \(\mu, \nu\) 都是有限离散测度时，Kantorovich 问题退化为标准线性 规划(LP)——传输问题(Transportation Problem)。这是运筹学的经典 问题，单纯形法可精确求解。理解离散版本有助于建立直觉，也是计算 方法的起点。

🔍 抽象化方法

耦合 \(\gamma\) \(\leftrightarrow\) 传输矩阵 \(P \in \mathbb{R}^{m \times n}_+\)； 约束 \(\leftrightarrow\) 行和列和；目标 \(\leftrightarrow\) Frobenius 内积 \(\langle C, P \rangle\)。

📐 公式推导 / 模型建立

离散设定：\(\mu = \sum_{i=1}^m a_i \delta_{x_i}\)，\(\nu = \sum_{j=1}^n b_j \delta_{y_j}\)， \(\sum a_i = \sum b_j = 1\)。代价矩阵 \(C_{ij} = c(x_i, y_j)\)。 LP 形式： \[\min_{P \geq 0} \sum_{i,j} C_{ij} P_{ij} \quad \text{s.t.} \quad P\mathbf{1} = a, \; P^\top\mathbf{1} = b\] \(P\) 有 \(mn\) 个变量，\(m+n-1\) 个独立约束。 解：传输单纯形法(Transportation Simplex)复杂度约 \(O(m^2 n \log m)\)。 Birkhoff-von Neumann 定理：当 \(m=n, a_i=b_j=1/n\) 时， 最优 \(P\) 是双随机矩阵集的顶点 \(=\) 置换矩阵——回到 Monge！

🌐 实际应用

Document distance（词频直方图之间的 Earth Mover's Distance）用于文本相似度。

✏️ 例题分析与易错点

例题：\(a = (1/2, 1/2)\)，\(b = (1/3, 2/3)\)。 \(C = \begin{pmatrix}1&2\\3&1\end{pmatrix}\)。求最优传输矩阵。 解答：\(P = \begin{pmatrix}p_{11}&p_{12}\\p_{21}&p_{22}\end{pmatrix}\)， \(p_{11}+p_{12}=1/2, p_{21}+p_{22}=1/2, p_{11}+p_{21}=1/3, p_{12}+p_{22}=2/3\)。 由约束：\(p_{11} = 1/3 - p_{21}\), \(p_{12} = 1/2 - p_{11} = 1/6 + p_{21}\), \(p_{22} = 1/2 - p_{21}\)。\(0 \leq p_{21} \leq 1/3\)。 目标：\((1/3-p_{21}) + 2(1/6+p_{21}) + 3p_{21} + (1/2-p_{21})\) \(= 1/3 + 1/3 + 1/2 + 4p_{21} = 7/6 + 4p_{21}\)。最小化取 \(p_{21}=0\)。 \(P^* = \begin{pmatrix}1/3&1/6\\0&1/2\end{pmatrix}\)，最优代价 \(= 7/6\)。 易错点：LP 最优解在顶点取到——传输矩阵的非零元素数 \(\leq m+n-1\)。

04最优传输的度量性质

🎯 为什么研究这个问题

最优传输代价赋予概率测度空间一个自然的距离结构——Wasserstein 距 离。这个距离比 KL 散度、TV 距离更"几何"：它考虑底空间的度量结 构。这种度量性质是最优传输在机器学习（WGAN）、统计和几何中广泛 应用的根基。

🔍 抽象化方法

Wasserstein 距离将底空间 \((X, d)\) 的距离结构提升到概率测度空间 \(\mathcal{P}(X)\) 上。

📐 公式推导 / 模型建立

\(p\)-Wasserstein 距离（\(p \geq 1\)）： \[W_p(\mu,\nu) = \left(\inf_{\gamma \in \Pi(\mu,\nu)} \int |x-y|^p \, d\gamma\right)^{1/p}\] 度量性质验证： - \(W_p(\mu,\nu) \geq 0\)，等号当且仅当 \(\mu = \nu\) - 对称性：\(W_p(\mu,\nu) = W_p(\nu,\mu)\) - 三角不等式：\(W_p(\mu,\sigma) \leq W_p(\mu,\nu) + W_p(\nu,\sigma)\) （通过耦合的"粘合"证明） 一维显式公式： \[W_p(\mu,\nu) = \left(\int_0^1 |F_\mu^{-1}(t) - F_\nu^{-1}(t)|^p \, dt\right)^{1/p}\] 其中 \(F^{-1}\) 是分位数函数。特别地， \(W_2^2(\mu,\nu) = \int_0^1 (F_\mu^{-1}(t) - F_\nu^{-1}(t))^2 \, dt\)。

🌐 实际应用

WGAN 用 \(W_1\) 距离替代 JS 散度训练 GAN，避免了梯度消失问题。

✏️ 例题分析与易错点

例题：\(\mu = \mathcal{N}(m_1, \sigma_1^2)\), \(\nu = \mathcal{N}(m_2, \sigma_2^2)\)。 求 \(W_2(\mu, \nu)\)。 解答：\(F_\mu^{-1}(t) = m_1 + \sigma_1 \Phi^{-1}(t)\), \(F_\nu^{-1}(t) = m_2 + \sigma_2 \Phi^{-1}(t)\)。 \(W_2^2 = \int_0^1 [(m_1-m_2) + (\sigma_1-\sigma_2)\Phi^{-1}(t)]^2 dt\) \(= (m_1-m_2)^2 + (\sigma_1-\sigma_2)^2\)（因 \(\int [\Phi^{-1}(t)]^2 dt = 1\)）。 \(W_2 = \sqrt{(m_1-m_2)^2 + (\sigma_1-\sigma_2)^2}\)。 易错点：高维高斯的 \(W_2\) 公式涉及协方差矩阵平方根： \(W_2^2 = \|m_1-m_2\|^2 + \mathrm{tr}(\Sigma_1+\Sigma_2-2(\Sigma_1^{1/2}\Sigma_2\Sigma_1^{1/2})^{1/2})\)。
