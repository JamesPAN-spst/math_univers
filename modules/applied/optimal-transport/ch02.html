Kantorovich 对偶

01线性规划对偶与 Kantorovich 对偶问题

🎯 为什么研究这个问题

每个线性规划都有对偶问题——Kantorovich 传输问题的对偶给出了最优 传输理论中最深刻的视角：传输代价 \(=\) 一对"势函数"的期望差。对 偶不仅提供理论洞察（KR 对偶、\(W_1\) 的 Lipschitz 刻画），也是 计算方法（Sinkhorn、神经网络求解）的理论基础。

🔍 抽象化方法

原问题（最小化传输代价）\(\leftrightarrow\) 对偶问题（最大化势函数的期望差）， 强对偶成立。

📐 公式推导 / 模型建立

离散 Kantorovich 对偶：原问题 \(\min_{P \geq 0} \langle C, P \rangle\), s.t. \(P\mathbf{1}=a, P^\top\mathbf{1}=b\)。 对偶：引入对偶变量 \(f \in \mathbb{R}^m, g \in \mathbb{R}^n\)： \[\max_{f,g} \langle f, a \rangle + \langle g, b \rangle \quad \text{s.t.} \quad f_i + g_j \leq C_{ij}, \; \forall i,j\] 强对偶(LP 对偶定理)：\(\min = \max\)。 连续版本： \[W_c(\mu,\nu) = \sup_{\varphi, \psi} \left\{\int \varphi \, d\mu + \int \psi \, d\nu : \varphi(x) + \psi(y) \leq c(x,y)\right\}\] 互补松弛：最优耦合 \(\gamma^*\) 的支撑 \(\subset \{(x,y) : \varphi(x)+\psi(y) = c(x,y)\}\)。

🌐 实际应用

WGAN 训练中的判别器(critic)正是 Kantorovich 对偶势函数 \(\varphi\)。

✏️ 例题分析与易错点

例题：\(\mu = \delta_0, \nu = \delta_1\), \(c(x,y) = |x-y|^2\)。 对偶最优解？ 解答：原问题代价 \(= |0-1|^2 = 1\)。 对偶：\(\max f(0) + g(1)\), s.t. \(f(0)+g(1) \leq 1\)。 最优 \(f(0)+g(1) = 1\)，如 \(f(0)=0, g(1)=1\)。强对偶验证：\(1=1\)。✓ 易错点：对偶变量 \((\varphi, \psi)\) 不唯一——可加减常数（\(\varphi + C, \psi - C\)）。

02c-变换与 c-凹函数

🎯 为什么研究这个问题

c-变换(c-transform)是最优传输对偶理论的核心工具：给定 \(\varphi\)， 其 c-变换 \(\varphi^c(y) = \inf_x [c(x,y) - \varphi(x)]\) 自动满足 对偶约束。c-凹函数(c-concave)是 c-变换的不动点——最优对偶解 \(\varphi\) 必为 c-凹的。这将无穷维对偶问题简化为单个函数的优化。

🔍 抽象化方法

c-变换 \(\leftrightarrow\) 凸共轭的推广。当 \(c = |x-y|^2/2\) 时， c-变换退化为 Legendre-Fenchel 变换。

📐 公式推导 / 模型建立

c-变换定义：\(\varphi^c(y) = \inf_x \{c(x,y) - \varphi(x)\}\)。 c-凹函数：\(\varphi\) 满足 \((\varphi^c)^c = \varphi\)，即 c-变换两次回到自身。 简化对偶：原对偶问题等价于 \[\sup_{\varphi \; c\text{-concave}} \left\{\int \varphi \, d\mu + \int \varphi^c \, d\nu\right\}\] 只需优化一个函数 \(\varphi\)（\(\psi = \varphi^c\) 自动确定）。 当 \(c(x,y) = \frac{1}{2}|x-y|^2\) 时： \(\varphi^c(y) = \inf_x \left\{\frac{|x-y|^2}{2} - \varphi(x)\right\} = \frac{|y|^2}{2} - \sup_x \left\{\langle x,y\rangle - \left(\frac{|x|^2}{2}-\varphi(x)\right)\right\}\) \(= \frac{|y|^2}{2} - \left(\frac{|\cdot|^2}{2}-\varphi\right)^*(y)\)。

🌐 实际应用

ICNN(Input Convex Neural Network)方法用神经网络参数化凸函数来近似 c-凹势函数。

✏️ 例题分析与易错点

例题：\(c(x,y) = |x-y|\)，\(\varphi(x) = x\) 在 \(\mathbb{R}\) 上。求 \(\varphi^c(y)\)。 解答：\(\varphi^c(y) = \inf_x \{|x-y| - x\}\)。 当 \(x \to -\infty\)：\(|x-y| - x \approx -x + y - x = y - 2x \to +\infty\)（若 \(x \ll y\)）。 当 \(x = y\)：\(0 - y = -y\)。当 \(x > y\)：\((x-y) - x = -y\)。 当 \(x < y\)：\((y-x) - x = y - 2x\)，最小化取 \(x \to y^-\) 得 \(-y\)。 故 \(\varphi^c(y) = -y\)。验证：\(\varphi(x)+\varphi^c(y) = x - y \leq |x-y|\)。✓ 易错点：c-变换的 inf 可能不在内部取到——需注意边界行为和存在性。

03Kantorovich-Rubinstein 对偶与 \(W_1\)

🎯 为什么研究这个问题

当代价 \(c = d\)（距离本身）时，Kantorovich 对偶获得一个极其优雅 的形式：\(W_1(\mu,\nu) = \sup_{\|\varphi\|_{\text{Lip}} \leq 1} \int \varphi \, d(\mu-\nu)\)。 这就是 Kantorovich-Rubinstein 对偶——\(W_1\) 距离等于 1-Lipschitz 函数类上的最大期望差。这个结果直接启发了 WGAN 的设计。

🔍 抽象化方法

\(W_1\) 的 KR 对偶将传输度量转化为算子范数： \(W_1 = \|\mu - \nu\|_{\text{Lip}^*}\)。

📐 公式推导 / 模型建立

Kantorovich-Rubinstein 定理：当 \(c(x,y) = d(x,y)\) 时， c-变换约束 \(\varphi(x) + \psi(y) \leq d(x,y)\) 等价于 \(\varphi(x) - \varphi(y) \leq d(x,y)\)（取 \(\psi = -\varphi\)），即 \(\varphi\) 是 1-Lipschitz。 \[W_1(\mu,\nu) = \sup_{\varphi : \text{Lip}(\varphi) \leq 1} \left\{\int \varphi \, d\mu - \int \varphi \, d\nu\right\}\] 这是 \((C_b(X), \|\cdot\|_\infty)\) 的对偶空间中 \(\mu - \nu\) 的一种范数。 与总变差的关系：\(\text{TV}(\mu,\nu) = \sup_{\|\varphi\|_\infty \leq 1} \int \varphi \, d(\mu-\nu)\)。 KR 对偶将 \(L^\infty\) 约束替换为 Lipschitz 约束——更"几何"。

🌐 实际应用

WGAN(Wasserstein GAN)：判别器网络被约束为 1-Lipschitz（梯度裁剪或梯度惩罚） 来近似 KR 对偶的最优势函数。

✏️ 例题分析与易错点

例题：\(\mu = \delta_0, \nu = \delta_a\)（\(a > 0\)）。 用 KR 对偶计算 \(W_1(\mu,\nu)\)。 解答：\(W_1 = \sup_{\text{Lip}(\varphi)\leq 1} [\varphi(0) - \varphi(a)]\)。 Lipschitz 约束：\(|\varphi(0)-\varphi(a)| \leq a\)。 取 \(\varphi(x) = -x\)（Lip = 1）：\(\varphi(0)-\varphi(a) = a\)。\(W_1 = a = d(0,a)\)。✓ 易错点：KR 对偶只适用于 \(c = d\)（即 \(p=1\)）。\(W_2\) 没有这种简洁的 Lipschitz 刻画。

04对偶的经济学与博弈论解释

🎯 为什么研究这个问题

Kantorovich 对偶不仅是数学技巧，还有深刻的经济学含义：势函数 \((\varphi, \psi)\) 可解释为"价格"——\(\varphi(x)\) 是源点 \(x\) 处的出价， \(\psi(y)\) 是目的地 \(y\) 处的出价，对偶约束保证没有套利机会。这种 解释连接了最优传输与市场均衡理论。

🔍 抽象化方法

对偶 \(\leftrightarrow\) 竞争性均衡。最优传输 \(=\) 无套利的价格-分配对。

📐 公式推导 / 模型建立

经济解释：运输公司提供从 \(x\) 到 \(y\) 的运输，收费 \(c(x,y)\)。 买家愿意支付 \(\psi(y)\)，卖家要价 \(\varphi(x)\)。 无套利条件：\(\varphi(x) + \psi(y) \leq c(x,y)\)（利润非负）。 最优对偶使总收入 \(\int \varphi \, d\mu + \int \psi \, d\nu\) 最大化—— 即定价使收入最大但不超过传输成本。 互补松弛：最优传输 \(\gamma^*\) 的支撑上 \(\varphi(x) + \psi(y) = c(x,y)\)——活跃的传输路线利润为零。 关联博弈论：传输问题等价于 Shapley-Shubik 匹配市场模型 中的核(core)分配——不存在联盟偏离。

🌐 实际应用

匹配市场(Matching Markets)：如住房分配、器官移植配对中的稳定匹配问题。

✏️ 例题分析与易错点

例题：2 个卖家、2 个买家。\(c = \begin{pmatrix}3&5\\4&2\end{pmatrix}\)。 \(a = b = (1/2, 1/2)\)。最优 \((\varphi, \psi)\)？ 解答：原问题最优：\(P^* = \begin{pmatrix}1/2&0\\0&1/2\end{pmatrix}\)，代价 \(= 5/2\)。 互补松弛：\(\varphi_1+\psi_1 = 3, \varphi_2+\psi_2 = 2\)， 且 \(\varphi_1+\psi_2 \leq 5, \varphi_2+\psi_1 \leq 4\)。 取 \(\varphi_1=0\)：\(\psi_1=3\), 则 \(\varphi_2 \leq 1\), \(\psi_2 = 2-\varphi_2\)。 取 \(\varphi_2 = 1, \psi_2 = 1\)：\(\varphi_1+\psi_2 = 1 \leq 5\) ✓, \(\varphi_2+\psi_1 = 4\) ✓。 易错点：对偶解不唯一（可平移常数），但 \(\varphi + \psi\) 在最优传输支撑上的值唯一。
